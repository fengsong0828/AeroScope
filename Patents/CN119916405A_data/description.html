Description

基于低轨北斗导航增强卫星的低空飞行器安全管理方法与 系统
技术领域
本发明涉及低空飞行器的安全管理领域，具体是一种低空飞行器利用低轨北斗导航增强卫星进行高精度精准定位和抗干扰的安全管理方法与系统。
背景技术
低空经济是以各种有人驾驶和无人驾驶航空器的各类低空飞行活动为牵引，辐射带动相关领域融合发展的综合性经济形态。低空领域的主要产品有备受关注的eVTOL(电动垂直起降飞行器)、无人机(消费级、工业级)、直升机、传统固定翼飞机等。从应用场景区分，低空经济可以划分为城市场景(城市空中交通)和非城市场景(偏远地区的工业、农业、林业、渔业和建筑业的作业飞行以及医疗救援、抢险救灾、气象探测、海洋监测、科学实验、教育训练、文化体育等方面的飞行活动)，2024年，低空经济已经纳入《政府工作报告》，成为新质生产力的重要推动引擎。
低空经济中载入和载货的无人机等低空飞行器在多个飞行器处在同一航道上时，面临着航道与航道之前碰撞或交叉接触的安全风险，需要实现各个飞行器之间的行驶间隔的有效监测和控制。同时，目前低空飞行器抗干扰能力差，地面干扰装备很容易对飞行器上的基准时钟信号和导航信号进行诱骗或压制，对飞行器进行干扰，形成重大安全隐患。
因此，实现低空飞行器的高精度定位管理和干扰监测与抗干扰管理，对低空飞行器安全行驶具备重大意义。
目前，国内部分低空飞行器配置了高精度定位管理模块，但都是基于地基基站和CORS系统，能够使飞行器的定位精度达到10cm量级，但是，这种系统的容易收到城市建筑、山体等结构的影响，稳定度不佳。此外，地面CORS系统设计主要针对地面高精度定位需求设备使用，系统信号上空空间传播和辐射能力较弱，这直接导致地面CORS系统用于低空飞行器领域效果不佳。同时，目前低空飞行器的干扰监测和抗干扰能力普遍性能不佳，以及基于地面CORS基站而部署的低空飞行器高精度定位及抗干扰设备整体造价偏高，不利用在低空经济领域进行广泛的产业化推广，因此亟需设计一种基于低轨北斗导航增强卫星的低空飞行器安全管理方法与系统用于解决上述问题。
发明内容
本发明的目的在于提供基于低轨北斗导航增强卫星的低空飞行器安全管理方法与系统，以解决上述背景技术中提出的目前低空飞行器的干扰监测和抗干扰能力普遍性能不佳，以及基于地面CORS基站而部署的低空飞行器高精度定位及抗干扰设备整体造价偏高，不利用在低空经济领域进行广泛的产业化推广的问题。
为实现上述目的，本发明提供如下技术方案：
基于低轨北斗导航增强卫星的低空飞行器安全管理系统，包括第0模块、第11模块、第12模块、第13模块、第14模块、第15模块、第16模块、第17模块、第18模块、第21模块、第22模块、第23模块、第24模块、第25模块、第26模块、第27模块、第28模块、第29模块、第291模块、第292模块、第31模块、第32模块、第33模块、第34模块、第35模块和第36模块，所述第0模块为卫星接收与分路模块，所述所述第11模块为卫星时钟基准信号提取模块，所述第12模块为本地时钟信号产生模块，第13模块为连续3次均方差计算模块，第14模块为微妙误差存储模块，第15模块为小于判读模块，第16模块为大于判读模块，第17模块为采用本地时钟信号模块，第18模块为采用卫星时钟信号模块，第21模块为伪随机信号提取模块，第22模块为本地伪随机信号产生模块，第23模块为相关性匹配模块，第24模块为检测成功判读模块，第25模块为检测不成功判读模块，第26模块为正常指示模块，第27模块为干扰报警指示，第28模块为启动信号增强模块，第29模块为检测成功判读模块，第291模块为检测不成功判读模块，未出现相关峰，持续进行信号增强，第292模块为工作正常指示模块，第31模块为低轨北斗导航增强定位信号提取模块，第32模块为工作状态正常监测模块，第33模块为工作状态正常监测模块，第34模块为工作状态不正常监测模块，第35模块为多星定位解算模块，第36模块为定位信息确定模块。
基于低轨北斗导航增强卫星的低空飞行器管理方法：其管理方法为：基于低轨北斗导航增强卫星，将中高轨北斗导航卫星信号的时频信息进行提取，进一步利用多中高轨北斗卫星的信号进行差分和误差校正，将校正后的高精度高基准度信号进行放大增强，广播至低空飞行器接收，广播信息中包含了高精度低延时的时钟基准、用于抗干扰的伪随机编码信息、误差校正后的北斗定位时差信息，进一步在低空飞行器导航定位模块和抗干扰模块的计算下，进行高精度位置信息的更新和干扰信息的监测及校准。
作为本发明再进一步的方案：其具体方法步骤如下：步骤一：首先根据低轨北斗导航卫星伪随机码与低空飞行器的解码单元的匹配，实现低轨北斗导航卫星身份特征的识别，每颗低轨北斗导航卫星释放的伪随机码不同。在接收信号编排方面，伪随机信号之后跟随的为高精度低延时的时钟基准、误差校正后的北斗定位时差信息；
步骤二：在伪随机码信息之后，跟随高精度低延时的时钟基准和误差校正后的北斗定位时差信息，高精度低时延的时钟基准信息用于和本地时钟进行比对，若发现偏离时钟基准，则及时进行综合纠偏，纠偏方法采用连续三次比对(t11、t12、t13)，均检测出偏差，则校正为当前连续三次接收到的低轨北斗导航卫星时钟基准作(t01、t02、t03)的平均值t0为低空飞行器的时钟基准，实现飞行器时钟基准的抗干扰；
步骤三：在步骤一中伪随机码接收过程中，接收信号路径为F1，如果连续三次该伪随机码均无法匹配，则启动低空飞行器接收模块的增益放大，并向控制系统发出报警，报警信息指示为收到干扰信息的压制或欺骗，信号连续放大直至匹配成功，则结束信号放大并维持现有信号功率。保持同时，向控制系统发出飞行器持续被干扰中，并对信号进行分路F2，分路后的信号与接收到的伪随机码进行匹配，若匹配成功，则说明干扰结束。信号结束放大，回归正常功率水平，报警解除；
步骤四：在步骤一中误差校正后的北斗定位时差信息中，根据飞行器接收到的卫星信号的数量，低空飞行器定位解算模块根据三角定位算法，进行北斗定位信息的解算和冗余计算，实现高精度的位置计算和信息输出。
作为本发明再进一步的方案：所述低空飞行器利用低轨北斗导航增强卫星进行信号抗干扰信息的监测，是基于对伪随机码信息的匹配，匹配成功说明信号未被干扰或压制；匹配不成功则说明信号受到干扰或压制。
作为本发明再进一步的方案：所述低空飞行器利用低轨北斗导航增强卫星进行时钟基准抗干扰信息的监测，进行连续偏差监测，出现连续时钟偏差，则说明时钟基准被干扰，正确时钟信息由低轨卫星传递时钟为基准进行纠偏。
作为本发明再进一步的方案：所述低空飞行器利用低轨北斗导航增强卫星进行高精度精准定位和抗干扰的安全管理系统，其特征在于：所述的低空飞行器利用低轨北斗导航增强卫星进行信号抗干扰信息的监测，需要采用自反馈机制进行恢复，采用了分路功能，一路进行放大增强，以实现工作正常；一路进行分路，实现信号干扰监测。两路同时匹配成功，则说明干扰解除，信号恢复单路工作状态。一旦触发干扰报警，则启动分路工作状态。
与现有技术相比，本发明的有益效果是：
1、本发明在确保时钟基准和信号正常情况下，进行定位信息的解算和相关信息提取，能够更好的避免因定位误差而可能造成的相关碰撞风险和诱骗风险；
2、本发明从充分发挥了低轨北斗导航卫星的高精度定位信号广播、低时延时空基准传递、抗干扰伪随机码匹配等优势，通过飞行器高效抗干扰解析方法，高可靠实现了低空飞行器的高精度定位和快速抗干扰监测，同时通过自反馈机制，实现干扰压制，确保了在第三方干扰情况，仍能正常工作；
3、本发明简单易行、容易实现、方法实现简单、消耗飞行器计算资源少，同时，处理干扰信息是从伪随机码频率信息维度进行，从根本上消除了其他频率干扰影响；采用分路策略和自反馈机制，通过先消除干扰，再进行定位的串行方法，从而保障了整个飞行路径从空域、时域和频域等多维度的干扰监测和定位精度，从而大大提高了飞行的安全性。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
请参阅图1，本发明实施例中，基于低轨北斗导航增强卫星的低空飞行器安全管理方法与系统，包括第0模块、第11模块、第12模块、第13模块、第14模块、第15模块、第16模块、第17模块、第18模块、第21模块、第22模块、第23模块、第24模块、第25模块、第26模块、第27模块、第28模块、第29模块、第291模块、第292模块、第31模块、第32模块、第33模块、第34模块、第35模块和第36模块，所述第0模块为卫星接收与分路模块，该模块实现低轨北斗导航增强卫星信号的接收与分路功能，对卫星信号接收后，进行识别和确认，分别提取出卫星信号中所包含的时钟基准信号，伪随机信号，定位信号，所述第11模块为卫星时钟基准信号提取模块，该模块实现对卫星时钟基准信号提取，后续模块会对该信号进行相应的处理和分析，所述第12模块为本地时钟信号产生模块，由本地时钟晶振产生本地时钟信号，第13模块为连续3次均方差计算模块，该模块功能为将本地时钟信号与卫星时钟信号进行比对，根据比对结果，选择是否继续使用本地时钟信号；
所述第14模块为微妙误差存储模块，该模块根据本地时钟的精度，确定微秒位具体的误差数值。第15模块为小于判读模块，如果微秒误差小于该数值，进入下一模块。第16模块为大于判读模块，如果微秒误差大于该数值，进入下一模块。第17模块为采用本地时钟信号模块，如果如果微秒误差小于该数值，则使用本地时钟信号作为定位基准信号，第18模块为采用卫星时钟信号模块，如果如果微秒误差大于该数值，则使用卫星时钟信号作为定位基准信号，第21模块为伪随机信号提取模块，对卫星信号中的伪随机信号进行提取，第22模块为本地伪随机信号产生模块，产生和卫星伪随机信号相匹配的本地伪随机信号，第23模块为相关性匹配模块，该模块将本地伪随机信号与卫星伪随机信号进行相关性匹配，匹配成功与否(成功的标识为产生相关峰)，进入下一个模块；
所述第24模块为检测成功判读模块，成功的标识为产生相关峰，进入下一模块，第25模块为检测不成功判读模块，未出现相关峰，进入下一模块，第26模块为正常指示模块，根据第24模块的相关性匹配结果，产生相关峰，则为正常指示，即为出现干扰或明显干扰，第27模块为干扰报警指示，根据第25模块的相关性匹配结果，未产生相关峰，则进行报警干指示扰，即为出现明显干扰，第28模块为启动信号增强模块，在受到干扰情况下，对接收到的信号进行持续放大，直至相关性匹配成功，第29模块为检测成功判读模块，成功的标识为产生相关峰，进入下一模块，第291模块为检测不成功判读模块，未出现相关峰，持续进行信号增强，第292模块为工作正常指示模块，根据第29模块的检测结果，成为进行信号匹配，进行工作正常指示，但表示为信号增强下的工作正常指示，第31模块为低轨北斗导航增强定位信号提取模块，对接收到的卫星信号提取出用于定位的北斗导航增强定位信号，第32模块为工作状态正常监测模块，该模块为26模块和292模块的信号接入，两个模块任一模块显示正常，均进入下一工作模块，第33模块为工作状态正常监测模块，正常工作状态下进入多星解算模块，第34模块为工作状态不正常监测模块，工作状态不正常则进入，进入自主定位模块，第35模块为多星定位解算模块，采用多星定位对位置信息进行解算，获取位置信息，第36模块为定位信息确定模块，获取位置信息后，对位置信息进行存储及更新。
卫星时钟基准信号经过提取后，与本地时钟产生模块产生的基准时钟，进行连续三次均方差计算，如果大于设定误差，则采用卫星时钟信号作为基准时钟。如果小于设定误差，则采用本地时钟信号作为基准时钟。伪随机信号提取后，与本地产生的伪随机信号进行相关性匹配，若能够获取相关峰，则进行工作正常指示，未受到干扰。若相关峰没有获取，则进行干扰报警，同时启动信号增强，直至匹配成功。这种情况下，工作正常为信号增强下的工作正常。此时，可以进行低轨导航增强卫星定位信息的解算和处理，包括多星定位解算以及定位信息的存储和更新。若工作状态不正常，则启动无源自主定位；
基于低轨北斗导航增强卫星，将中高轨北斗导航卫星信号的时频信息进行提取，利用多中高轨北斗卫星的信号进行差分和误差校正，将校正后的高精度高基准度信号进行放大增强，广播至低空飞行器接收，广播信息中包含了高精度低延时的时钟基准、用于抗干扰的伪随机编码信息、误差校正后的北斗定位时差信息，进一步在低空飞行器导航定位模块和抗干扰模块的计算下，进行高精度位置信息的更新和干扰信息的监测及校准，其具体步骤如下：
步骤一：首先根据低轨北斗导航卫星伪随机码与低空飞行器的解码单元的匹配，实现低轨北斗导航卫星身份特征的识别，每颗低轨北斗导航卫星释放的伪随机码不同。在接收信号编排方面，伪随机信号之后跟随的为高精度低延时的时钟基准、误差校正后的北斗定位时差信息；
步骤二：在伪随机码信息之后，跟随高精度低延时的时钟基准和误差校正后的北斗定位时差信息，高精度低时延的时钟基准信息用于和本地时钟进行比对，若发现偏离时钟基准，则及时进行综合纠偏，纠偏方法采用连续三次比对(t11、t12、t13)，均检测出偏差，则校正为当前连续三次接收到的低轨北斗导航卫星时钟基准作(t01、t02、t03)的平均值t0为低空飞行器的时钟基准，实现飞行器时钟基准的抗干扰；
步骤三：在步骤一中伪随机码接收过程中，接收信号路径为F1，如果连续三次该伪随机码均无法匹配，则启动低空飞行器接收模块的增益放大，并向控制系统发出报警，报警信息指示为收到干扰信息的压制或欺骗，信号连续放大直至匹配成功，则结束信号放大并维持现有信号功率。保持同时，向控制系统发出飞行器持续被干扰中，并对信号进行分路F2，分路后的信号与接收到的伪随机码进行匹配，若匹配成功，则说明干扰结束。信号结束放大，回归正常功率水平，报警解除；
步骤四：在步骤一中误差校正后的北斗定位时差信息中，根据飞行器接收到的卫星信号的数量，低空飞行器定位解算模块根据三角定位算法，进行北斗定位信息的解算和冗余计算，实现高精度的位置计算和信息输出；
低空飞行器利用低轨北斗导航增强卫星进行信号抗干扰信息的监测，是基于对伪随机码信息的匹配，匹配成功说明信号未被干扰或压制；匹配不成功则说明信号受到干扰或压制；
低空飞行器利用低轨北斗导航增强卫星进行时钟基准抗干扰信息的监测，进行连续偏差监测，出现连续时钟偏差，则说明时钟基准被干扰，正确时钟信息由低轨卫星传递时钟为基准进行纠偏；
低空飞行器利用低轨北斗导航增强卫星进行高精度精准定位和抗干扰的安全管理系统，其特征在于：所述的低空飞行器利用低轨北斗导航增强卫星进行信号抗干扰信息的监测，需要采用自反馈机制进行恢复，采用了分路功能，一路进行放大增强，以实现工作正常；一路进行分路，实现信号干扰监测。两路同时匹配成功，则说明干扰解除，信号恢复单路工作状态。一旦触发干扰报警，则启动分路工作状态；
本发明针对低空飞行器在飞行过程中的三个涉及飞行安全的关键环节，包括时钟基准的干扰、飞行器频率信号干扰、飞行器高精度定位等三个环节。首先是对飞行器频率信号(即正常工作信号)的干扰监测，这是基础，只要工作正常，则进行时钟基准干扰监测。若工作不正常，则持续启动信号放大，确保工作正常下，进行时钟基准干扰的监测和纠偏。时钟基准干扰的监测和工作信号的干扰监测均采用多次测量方式，以防止误检测，更加保证了干扰监测的准确性。然后，针对工作信号的干扰监测和压制，采用分路自反馈的方法，一方面能保证飞行器的正常工作，一方面通过持续反馈，能够实现干扰消失监测，双向机制能够更加安全低保护飞行器的正常行驶。然后，在确保时钟基准和信号正常情况下，进行定位信息的解算和相关信息提取，这样能够更好的避免因定位误差而可能造成的相关碰撞风险和诱骗风险。
