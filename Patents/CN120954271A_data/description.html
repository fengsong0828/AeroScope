Description

基于气象数据的低空飞行器混合起降场的红绿灯调整方法
技术领域
本发明涉及低空空域管理，具体涉及基于气象数据的低空飞行器混合起降场的红绿灯调整方法。
背景技术
随着低空空域开放的不断推进和无人机、通用航空器等低空飞行器数量的快速增长，低空飞行活动的密度与复杂性显著提升，对混合起降场的运行效率与安全管控提出了更高的要求。传统低空交通管理主要依赖人工调度与固定信号配时，难以适应动态空域负荷环境下的实时需求，尤其在复杂气象条件下（如低能见度、强对流、风切变等），飞行器起降冲突、空域资源分配不均等问题频发，导致航班延误、运行风险上升甚至出现事故。
现有技术中，低空飞行器混合起降场的信号控制主要存在以下局限性：
1）数据孤岛与感知滞后：空域状态监测通常独立依赖ADS-B（广播式自动相关监视）、雷达或气象站等单一数据源，缺乏对多源异构数据的融合分析，导致空域繁忙度评估片面，无法实时捕捉冲突热点与流量突变；
2）可视化与溯源能力不足：空域繁忙度通常以二维平面热力图呈现，缺乏三维网格空间下的多尺度动态映射，难以支持复杂地形（如山区、城市峡谷等）或立体交叉空域的历史流量回溯与机型占比分析，限制了调度策略的精细化制定；
3）决策链断裂与资源调配粗放：现有系统缺乏“感知—预判—调配”的闭环决策机制，气象数据仅用于事后风险警示，未能通过趋势预判提前优化红绿灯配时与跑道资源分配，导致管控措施滞后于风险演化；
4）静态配时与动态需求失配：现有红绿灯配时方案主要依赖人工调度与固定信号配时，未考虑气象风险变化（如突风导致起降间隔调整需求等）和历史流量周期性波动，难以适应动态空域负荷。
发明内容
（一）解决的技术问题
针对现有技术所存在的上述缺点，本发明提供了基于气象数据的低空飞行器混合起降场的红绿灯调整方法，能够有效克服现有技术所存在的难以在动态空域负荷环境下进行准确、实时的红绿灯配时的缺陷。
（二）技术方案
为实现以上目的，本发明通过以下技术方案予以实现：
基于气象数据的低空飞行器混合起降场的红绿灯调整方法，包括以下步骤：
S1、融合ADS-B轨迹数据、雷达数据、气象数据和空域状态信息，构建空域繁忙度计算模型，实时输出飞行器密度、冲突热点和服务负荷指数；
S2、将空域繁忙度计算结果映射至北斗三维网格空间，生成多尺度动态热力图，并支持任意空域剖面的历史流量回溯与机型占比分析；
S3、基于热力图关联气象数据和空域状态信息，建立“实时感知—趋势预判—资源调配”的闭环决策链优化机制；
S4、根据决策链指令，实时调整红绿灯配时。

1）计算飞行器密度：引入网格半径和指示函数，将飞行器分布从连续空间离散化为三维网格单元内的统计量，解决雷达数据因扫描间隔导致的密度计算误差；
2）检测冲突热点：引入Sigmoid函数将飞行器间距映射为冲突概率，避免传统阈值法的突变性，更符合实际飞行安全边界的模糊性；引入机型对权重量化不同机型组合的冲突风险差异，解决因起降场中机型动态特性不一致导致的冲突评估偏差；
3）计算服务负荷指数：将飞行器密度、冲突概率和气象风险进行加权聚合，突破传统流量统计仅依赖密度或冲突的单维度局限；

（三）有益效果
与现有技术相比，本发明所提供的基于气象数据的低空飞行器混合起降场的红绿灯调整方法，具有以下有益效果：
1）精准掌握空域态势，助力科学决策
融合多种数据构建空域繁忙度计算模型，实时输出关键指标，并将结果映射生成多尺度动态热力图，支持历史流量回溯与机型占比分析，使管理者能够精准、全面、动态地掌握空域实时状态和历史规律，打破传统信息局限，为制定科学合理的低空空域管理策略和红绿灯配时调整方案提供有力依据，提升决策的科学性和精准性；
2）智能预判与调配，提升运行效能
基于热力图关联气象数据和空域状态信息，建立“实时感知—趋势预判—资源调配”的闭环决策链优化机制，该机制不仅能实时感知空域情况，还能预判未来趋势，提前调整红绿灯配时，优化飞行器起降顺序和节奏，实现空域资源的高效利用，减少飞行器等待时间，提高低空飞行器混合起降场的整体运行效能；
3）灵活适应动态变化，保障运行安全
充分考虑气象数据、空域状态等多因素动态变化，以及历史流量周期性波动的基础上，实时调整红绿灯配时，在面对复杂多变的天气条件和空域突发状况时，能够迅速做出反应，灵活调整飞行器起降规则，避免因环境变化导致的冲突和危险情况，有效保障低空飞行器混合起降场的安全运行，降低安全风险。

上述技术方案，基于热力图关联气象数据和空域状态信息，建立“实时感知—趋势预判—资源调配”的闭环决策链优化机制，该机制不仅能实时感知空域情况，还能预判未来趋势，提前调整红绿灯配时，优化飞行器起降顺序和节奏，实现空域资源的高效利用，减少飞行器等待时间，提高低空飞行器混合起降场的整体运行效能。
此外，上述技术方案充分考虑气象数据、空域状态等多因素动态变化，以及历史流量周期性波动的基础上，实时调整红绿灯配时，在面对复杂多变的天气条件和空域突发状况时，能够迅速做出反应，灵活调整飞行器起降规则，避免因环境变化导致的冲突和危险情况，有效保障低空飞行器混合起降场的安全运行，降低安全风险。
