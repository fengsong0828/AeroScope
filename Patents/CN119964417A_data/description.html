Description

城市低空飞行器的空中交通管理方法、计算设备及系统
技术领域
本发明涉及空中交通管理技术领域，具体涉及一种城市低空飞行器的空中交通管理方法、计算设备及系统。
背景技术
随着低空经济的高速发展，城市低空物流和出行迎来了前所未有的机遇，同时也面临着新一轮的挑战。
与传统民航运输相比，低空空域飞行架次多，飞行距离短，有限空域内执飞的飞行器密度大。现有的基于管制员指挥的空中交通管理模式难以应对复杂的低空飞行器交通状况。
发明内容
为确保在城市低空空域中，高密度的飞行器能够安全高效地运行，本发明的目的在于针对现有技术的不足，提供一种城市低空飞行器空中交通管理方法、计算设备及系统。
为实现上述目的，第一方面，本发明实施例提供了一种城市低空飞行器的空中交通管理方法，包括：
获取城市低空飞行器的飞行计划数据；
基于所述飞行计划数据，在预先构建的城市低空圆径模型中规划最优航线；
当所述城市低空飞行器按照所述最优航线、在所述城市低空圆径模型中飞行时，实时调节多个所述城市低空飞行器之间的间距；
当出现紧急情况导致有冲突风险时，为相关飞行器制定冲突解脱方案，并使得相关飞行器按照所述冲突解脱方案飞行。
作为本申请的一种优选实现方式，获取城市低空飞行器的飞行计划数据之前，所述方法还包括构建城市低空圆径模型，具体为：
根据城市低空飞行器起降点的分布情况以及城市地面道路情况等因素，构建城市低空圆径模型，使得城市低空飞行器起降点均位于城市低空圆径模型的水平投影范围内；
其中，所述城市低空圆径模型包括两个圆环层和两个径向层；每个圆环层由多个半径各异的同心圆均匀分布组成，两个圆环层交错分布；每个径向层由多条均匀分布的径向路径组成，每条径向路径均穿过圆心，两个径向层交错分布。
作为本申请的一种具体实现方式，规划最优航线具体为：
解析所述飞行计划数据，获取城市低空飞行器的出发点S、目的地D；
根据出发点S、目的地D确定所述城市低空飞行器的圆周飞行模式、径向飞行模式；
基于所述圆周飞行模式、径向飞行模式以及空域流量情况规划城市低空飞行器的最优航线和速度。
作为本申请的一种具体实现方式，根据出发点S、目的地D确定所述城市低空飞行器的圆周飞行模式、径向飞行模式，具体为：
将出发点S、目的点D投影到二维水平面上，并以圆心投影点为原点构建极坐标系来判定飞行路径；在极坐标系下，出发点S投影坐标S'(ρs，θs)，目的点D投影坐标D'(ρd，θd)；
引入高度信息，出发点的坐标为S(ρs，θs，h1)，目的点的坐标为D(ρd，θd，h4)；
计算出发点和目的点之间的夹角θ＝θd-θs，并根据规则确定圆周飞行模式；
计算出发点和目的点极径的差值ρ＝ρd-ρs，并根据规则确定径向飞行模式。
第二方面，本发明实施例还提供了一种计算设备，适用于城市低空飞行器的空中交通管理，包括处理器、输入设备、输出设备和存储器，所述处理器、输入设备、输出设备和存储器相互连接，其中，所述存储器用于存储计算机程序，所述计算机程序包括程序指令，所述处理器被配置用于调用所述程序指令，执行上述第一方面的方法。

飞行计划处理模块，用于获取城市低空飞行器的飞行计划数据；
航线规划模块，用于基于所述飞行计划数据，在预先构建的城市低空圆径模型中规划最优航线；
间距控制模块，用于当所述城市低空飞行器按照所述最优航线、在所述城市低空圆径模型中飞行时，实时调节多个所述城市低空飞行器之间的间距；
冲突解脱模块，用于当出现紧急情况导致有冲突风险时，为相关飞行器制定冲突解脱方案，并使得相关飞行器按照所述冲突解脱方案飞行。
作为本申请的一种优选实现方式，所述系统还包括模型构建模块，用于构建城市低空圆径模型，具体为：
根据城市低空飞行器起降点的分布情况，构建城市低空圆径模型，使得城市低空飞行器起降点均位于城市低空圆径模型的水平投影范围内；
其中，城市低空圆径模型包括两个圆环层和两个径向层；每个圆环层由多个半径各异的同心圆均匀分布组成，两个圆环层交错分布；每个径向层由多条均匀分布的径向路径组成，每条径向路径均穿过圆心，两个径向层交错分布。
本发明实施例所提供的城市低空飞行器空中交通管理方案，通过构建圆径模型、整合飞行计划、规划最优航线、实时间距控制以及制定冲突解脱方案，显著降低了低空飞行器的碰撞风险。对于保障飞行安全、提升低空交通运行效率具有重要现实意义。

请参考图1，本发明实施例提供的城市低空飞行器的空中交通管理方法可以包括如下步骤：
S1，构建城市低空圆径模型。
具体地，城市低空圆径模型如图2所示，其具体构建过程包括：
根据城市低空飞行器起降点的分布情况以及城市地面道路情况等因素，构建城市低空圆径模型，使得飞行器起降点均位于圆径模型的水平投影范围内。
其中，圆径模型共四层，包括两个圆环层和两个径向层，分别分布于G类空域(高度范围为120米至300米)中的H1、H2、H3、H4四个高度。在本实施例中，具体设置H1＝120米为顺时针圆环层、H2＝160米为向心径向层、H3＝200米为向心径向层、H4＝240米为逆时针圆环层，250-300米范围为应急救援预留高度层，飞行器在此范围内可直飞；
如图3所示，每个圆环层由多个半径各异的同心圆(也可以是椭圆)均匀分布组成，为减少飞行器在飞行过程中的冲突，将两个圆环层设计为交错分布的形式，即它们在水平面上的投影不重叠。此处对于圆环层和径向层的定义为概念示意，具体的构建和组合方式应根据城市实际情况进行定制设计，即圆环层和径向层的具体形式可包括不规则部分，但总体需符合概念示意要求。所有符合上述概念示意的组合方式均涵盖在本发明的保护范围之内。在图3中，实线圆表示逆时针圆环层的投影，虚线圆为顺时针圆环层的投影，d1为相邻的顺时针圆与逆时针圆之间的间距，d2为相邻的同向圆之间的间距，rmin为圆环层最小半径，rmax为圆环层最大半径。
如图4所示，每个径向层由多条均匀分布的径向路径组成，每条路径的延长线都穿过圆心。随着径向路径接近圆心，相邻路径的水平距离缩短，导致发生飞行冲突的概率增加，因此在圆心周围设置禁飞区以确保安全。且与圆环层类似，径向层的设计也采用了交错分布的方式。在图4中，β1为相邻向心路径之间的夹角，β2为相邻离心路径之间的夹角，l为径向路径的长度。
S2，获取城市低空飞行器的飞行计划数据。
该行计划数据包括但不限于出发点、目的地、飞行器参数等。
S3，基于所述飞行计划数据，在预先构建的城市低空圆径模型中规划最优航线。
具体实现时，步骤S3包括：
S31、解析步骤S21中的飞行计划数据，获取飞行器的出发点S、目的点D；
S32、判定飞行器在圆周上的飞行模式，如图5所示，将出发点S、目的点D投影到二维水平面上，并以圆心投影点为原点构建极坐标系来判定飞行路径。在此坐标系下，出发点投影坐标S'(ρs，θs)，目的点投影坐标D'(ρd，θd)。进一步的，引入高度信息后，出发点的坐标为S(ρs，θs，h1)，目的点的坐标为D(ρd，θd，h4)。计算出发点和目的点之间的夹角θ＝θd-θs，并根据以下规则确定圆周飞行模式：
(1)当θ≥0且θ≤π，目的点位于出发点沿圆周逆时针方向的一侧，飞行器应逆时针飞行；
(2)当θ<0且|θ|≤π，目的点位于出发点沿圆周顺时针方向的一侧，飞行器应顺时针飞行；
(3)当θ>π，目的点位于出发点沿圆周顺时针方向超过半圈的位置，飞行器应顺时针飞行2π-θ；
(4)当θ<-π，目的点位于出发点沿圆周逆时针方向超过半圈的位置，飞行器应逆时针飞行-2π-θ；
S33、计算出发点和目的点极径的差值ρ＝ρd-ρs，根据以下规则确定径向飞行模式：
当ρ>0，即ρd>ρs，目的点相对出发点处于更外层的圆环，飞行器应在径向层离心飞行；
当ρ<0，即ρd<ρs，目的点相对出发点处于更内层的圆环，飞行器应在径向层向心飞行；
S34，航线规划模块基于步骤S32和S33所确定的圆周和径向飞行模式，以及空域流量情况，为飞行器规划最优航线和速度。
规划的结果是结合飞行的距离、空域的流量情况，做出如下判断：是先进进行圆周飞行还是径向飞行，以及圆周飞行中是顺时针飞行还是逆时针飞行。如图6所示，飞行器计划从低空飞行器起降点S飞行至起降点D。规划的最优航线为先从S垂直上升至H2高度的向心径向层的A点，向心飞行至B点，接着下降至H1高度的逆时针圆环层的C点，再逆时针飞行至E点，最后从E点垂直降落即可到达起降点D。
S4，当所述城市低空飞行器按照所述最优航线、在所述城市低空圆径模型中飞行时，实时调节多个所述城市低空飞行器之间的间距。
具体地，飞行器按照规划的最优航线和速度，在城市低空圆径模型内飞行。飞行过程中，间距控制模块实时调节飞行器之间的间距以保障飞行安全。
S5，当出现紧急情况导致有冲突风险时，为相关飞行器制定冲突解脱方案，并使得相关飞行器按照所述冲突解脱方案飞行。
