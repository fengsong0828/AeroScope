Description

一种应用于低空飞行器下的噪声消除装置
技术领域
本实用新型属于噪声消除技术领域，具体涉及一种应用于低空飞行器下的噪声消除装置。
背景技术
随着低空飞行器(如无人机、轻型直升机等)在军事侦察、物流配送、农业植保、测绘勘探等领域的广泛应用，其运行时产生的强噪声问题日益凸显，为了降低低空飞行器噪声的影响，一种应用于低空飞行器下的噪声消除装置应运而生，这种噪声消除装置通常包括jeston核心板、可充电电池、多个USB接口支持音频输入和输出、扬声器等部分，其基本工作流程为：通过音频输入设备采集受飞行器强噪声污染的含噪语音信号，将采集到的信号输入到jeston核心板进行语音增强(噪声消除)处理，处理后的干净语音信号再通过扬声器进行输出即可，在硬件组成方面，现有的噪声消除装置一般配备jeston核心板作为处理单元，该核心板具有强大的计算能力，能够实时对含噪语音信号进行处理；采用可充电电池为装置供电，保证了装置的便携性和续航能力；设置多个USB接口，支持音频输入和输出，方便与其他设备进行连接和数据传输。
然而现有噪声消除装置上扬声器的安装方式大多采用多个螺丝进行固定，在装置的前期组装阶段，安装人员需要使用工具逐个拧紧螺丝，操作过程繁琐，耗时较长，增加了组装成本和生产周期，而在后续使用过程中，若扬声器发生损坏，需要对其进行更换维护时，拆卸过程同样不方便，安装人员需要再次使用工具逐个拧松螺丝，不仅操作不便，而且在一些紧急情况下，无法快速将扬声器拆下，影响了设备的及时修复和正常使用，为此本实用新型提出一种应用于低空飞行器下的噪声消除装置。
实用新型内容
本实用新型的目的在于提供一种应用于低空飞行器下的噪声消除装置，以解决上述背景技术中提出的问题。
为实现上述目的，本实用新型提供如下技术方案：一种应用于低空飞行器下的噪声消除装置，包括壳体、安装在壳体内部的jeston核心板、设置在jeston核心板前端的多个USB接口、开设在壳体前表面且与USB接口相对应的连接缺口、安装在壳体上的扬声器组件，所述扬声器组件包括扬声器盖板和固定在扬声器盖板后表面的扬声器主体，所述壳体的前表面开设有供扬声器主体插入的矩形插口，且扬声器盖板贴合在壳体的前表面，还包括
设置在扬声器盖板和壳体之间的两个旋转式快拆装结构，所述旋转式快拆装结构包括旋转基条、轴块、圆柱形插块和矩形卡块，所述旋转基条和圆柱形插块分别处于扬声器盖板的前后表面，所述轴块转动装设在扬声器盖板的内壁，且轴块的两端分别与旋转基条和圆柱形插块固定，所述圆柱形插块的后端表面固定有矩形卡块，所述旋转式快拆装结构还包括开设在壳体表面供圆柱形插块贯穿的圆形插孔，所述圆形插孔与矩形插口之间开设有供矩形卡块贯穿的卡块进出口，所述圆柱形插块的后端贯穿至壳体的内侧，且矩形卡块卡在壳体的内侧表面。
优选的，所述旋转式快拆装结构还包括卡位结构，所述卡位结构设置在旋转基条和扬声器盖板之间。
优选的，所述卡位结构包括开设在扬声器盖板内部的内槽、装设在内槽内的弹簧和伸缩卡珠，所述旋转基条的后表面开设有与伸缩卡珠相对应的圆形卡槽，所述伸缩卡珠通过弹簧活动安装在内槽内，且伸缩卡珠的前端弹出至圆形卡槽内。
优选的，所述弹簧的前端与伸缩卡珠固定，所述弹簧的后端固定在内槽的内壁。
优选的，所述旋转式快拆装结构还包括定位结构，所述定位结构设置在旋转基条和扬声器盖板之间。
优选的，所述定位结构包括嵌入安装在扬声器盖板前表面的橡胶垫、设置在橡胶垫前表面的橡胶凸起、开设在旋转基条后表面且与橡胶凸起相对应的定位槽。
优选的，所述橡胶垫和橡胶凸起为一体式结构。
优选的，所述壳体的顶面和侧面均开设有多个条形孔。
与现有技术相比，本实用新型的有益效果是：本实用新型通过对现有技术中应用于低空飞行器下的噪声消除装置进行改进优化，在扬声器盖板上设计旋转式快拆装结构，有效解决了现有装置中扬声器安装和拆卸便捷性低的问题，不仅方便前期装置的整体组装，提高了生产效率，而且在后续扬声器损坏时，能够快速将其拆下进行更换维护，减少了装置后期的维修时间和成本，同时具有良好的稳定性，能够保证装置在日常使用过程中的可靠性，为低空飞行器的噪声消除带来了更加高效、稳定的支持。

实施例1
请参阅图1至图5，为本实用新型第一个实施例，该实施例提供了一种技术方案：一种应用于低空飞行器下的噪声消除装置，包括
壳体1、安装在壳体1内部的jeston核心板2、设置在jeston核心板2前端的多个USB接口21、开设在壳体1前表面且与USB接口21相对应的连接缺口11、安装在壳体1上的扬声器组件3，扬声器组件3包括扬声器盖板31和固定在扬声器盖板31后表面的扬声器主体32，壳体1的前表面开设有供扬声器主体32插入的矩形插口12，且扬声器盖板31贴合在壳体1的前表面，同时壳体1内还设置有用于日常运行供电的可充电电池，上述结构均为现有技术，具体结构原理在此不过多赘述，大致的工作流程就是操作人员用外接音频输入设备连接USB接口21，并将受飞行器强噪声污染的含噪语音信号输入jeston核心板2内，经过这个jeston核心板2的进行语音增强噪声消除处理，再输出原始的干净语音经过扬声器组件3输出；
还包括设置在扬声器盖板31和壳体1之间的两个旋转式快拆装结构5，旋转式快拆装结构包括旋转基条51、轴块57、圆柱形插块53和矩形卡块54，旋转基条51和圆柱形插块53分别处于扬声器盖板31的前后表面，轴块57转动装设在扬声器盖板31的内壁，且轴块57的两端分别与旋转基条51和圆柱形插块53固定，使得旋转基条51可以和圆柱形插块53一起旋转，圆柱形插块53的后端表面固定有矩形卡块54，旋转式快拆装结构5还包括开设在壳体1表面供圆柱形插块53贯穿的圆形插孔55，圆形插孔55与矩形插口12之间开设有供矩形卡块54贯穿的卡块进出口56，圆柱形插块53的后端贯穿至壳体1的内侧，且矩形卡块54卡在壳体1的内侧表面，可实现对扬声器盖板31和扬声器主体32的稳定限位，保证扬声器组件3的安装稳定性，后续若要将扬声器组件3快速拆下进行检修维护时，只需将旋转基条51旋转一百八十度，使得圆柱形插块53带着矩形卡块54旋转，致使矩形卡块54与卡块进出口56对齐，即可快速解除对扬声器盖板31和扬声器主体32的限位，此时将扬声器盖板31前拉，即可直接将扬声器主体32从壳体1的内侧抽出，快速完成扬声器组件3的拆装，方便后续扬声器组件3的拆卸检修维护，也方便前期装置组装时扬声器组件3与壳体1的快速组装。
本实施例中，优选的，旋转式快拆装结构5还包括卡位结构，卡位结构设置在旋转基条51和扬声器盖板31之间。
本实施例中，优选的，卡位结构包括开设在扬声器盖板31内部的内槽581、装设在内槽581内的弹簧582和伸缩卡珠583，旋转基条51的后表面开设有与伸缩卡珠583相对应的圆形卡槽584，伸缩卡珠583通过弹簧582活动安装在内槽581内，且伸缩卡珠583的前端弹出至圆形卡槽584内，可在日常使用过程中对旋转基条51进行卡合限位，保证旋转基条51和圆柱形插块53不会自行旋转，从而保证扬声器组件3的安装稳定性，保证装置的稳定运行，后续若要将扬声器组件3拆下时，只需用力的将旋转基条51推动，使得伸缩卡珠583的端部受到挤压并逐渐被挤入内槽581内，即可顺利的将旋转基条51从伸缩卡珠583处旋转移走，并顺利实现圆柱形插块53和矩形卡块54的旋转。
本实施例中，优选的，弹簧582的前端与伸缩卡珠583固定，弹簧582的后端固定在内槽581的内壁，保证弹簧582和伸缩卡珠583的稳定安装。
本实施例中，优选的，壳体1的顶面和侧面均开设有多个条形孔。
实施例2
请参阅图1至图5，为本实用新型第二个实施例，该实施例基于上一个实施例，不同的是，可通过设置的定位结构52，实现后续旋转基条51旋转一百八十度后的辅助限位，进一步方便扬声器组件3的拆装。
具体的，旋转式快拆装结构5还包括定位结构52，定位结构52设置在旋转基条51和扬声器盖板31之间。
本实施例中，优选的，定位结构52包括嵌入安装在扬声器盖板31前表面的橡胶垫521、设置在橡胶垫521前表面的橡胶凸起522、开设在旋转基条51后表面且与橡胶凸起522相对应的定位槽523，当后期操作人员将旋转基条51旋转一百八十度后，定位槽523会与橡胶凸起522对其，此时橡胶凸起522会被挤入定位槽523内，从而对旋转基条51和圆柱形插块53起到一定的辅助限位作用，防止此时旋转基条51和圆柱形插块53发生自旋，确保扬声器组件3在拆装过程中矩形卡块54始终与卡块进出口56对齐，从而有效保证扬声器组件3的拆装便捷性。
本实施例中，优选的，橡胶垫521和橡胶凸起522为一体式结构，二者均为橡胶材质制成，受到挤压时会发生弹性形变。
