Description

一种以微型涡轮喷气发动机为动力的低空单人飞行器
技术领域
本发明涉及载人飞行器技术领域，具体涉及一种以微型涡轮喷气发动机为动力的低空单人飞行器。
背景技术
微型涡轮喷气发动机驱动的低空单人飞行器，以其体积小巧和动力强劲的特点，为单人快速出行提供了新的可能性。这类飞行器不仅具备高速度上限和灵活性，还能在特殊场景下发挥重要作用，例如高山救援和海上救援。救援人员穿戴飞行器后，能够迅速定位和救助被困人员，提供紧急救援服务。此外，低空单人飞行器在军事领域也有着广泛的应用。以英国某公司的单人飞行器为例，它已被英国海军用于多次实验，证明其在海上战舰之间的快速转移和抢滩登陆战斗中的优越性能。
现以涡喷发动机为动力的单人飞行器为例。在以涡喷发动机为动力的单人飞行器的研究历史上，最经典的三个产品分别为喷气式飞机服、美国喷气飞行背包、飞行滑板，三种单人飞行器的设计理念完全不同，但都是通过试图在人身的某个部位放置微型涡喷发动机来提供推力。喷气式飞机服是其中操作性能最优的飞行器，同时它使用人体骨骼为支架来减少外部机械结构。
但目前的喷气式飞机服，主要是通过喷气式引擎的驱动下实现飞行，但由于现有喷气式飞机服为了保证飞行员的安全，采用了多种防护措施，例如：防护服等(刚制背包框架)，因此，其存在自身重量大，穿戴不便的弊端，而且由于飞行员是在涡喷发动机工作状态下进行位姿操作的，操作过程中会存在振动和推力(驾驶员同时独立操作多个发动机难以完全把握整个飞行器姿态的控制，很容易产生失误)，导致操作过程易失控，如果操作不当，涡喷发动机喷射出的流体还会对飞行员造成烧伤等意外伤害。
综上所述，现有的喷气式飞机服存在自身重量铰大、位姿调整不便和易对飞行员造成伤害的问题。
发明内容
本发明的目的是是为了解决现有的喷气式飞机服存在自身重量铰大、位姿调整不便和易对飞行员造成伤害的问题。进而提供一种以微型涡轮喷气发动机为动力的低空单人飞行器。
本发明的技术方案是：
一种以微型涡轮喷气发动机为动力的低空单人飞行器，它包括飞行背包和两个手臂推进装置；
飞行背包包括：安全带、壳体、供油系统、电气系统和背包微型涡轮发动机，供油系统安装在壳体内部，电气系统安装在壳体的上端面上，背包微型涡轮发动机安装在壳体的下部；
壳体包括背板和飞行背覆盖件，背板为板体，其中，背板的上部开设有多个减重孔，且背板采用铝合金制成，背板的四周开设有穿接孔，安全带的中部通过螺栓和鹰嘴钩固定安装在背板的中部，安全带的背带穿过穿接孔之后与飞行员的身体连接，飞行背覆盖件为壳体罩，飞行背覆盖件盖装在背板上，飞行背覆盖件的下部设有丝状网；
其中，两个手臂推进装置分别安装在飞行背包的壳体的左右两侧；
每个手臂推进装置均包括发动机支架、两个发动机、握把、扳机、手臂支架、挡板条和两组连杆组件，两组连杆组件的一端分别安装在手臂支架的左右两侧，两组连杆组件的另一端分别与一个发动机支架连接，每个发动机支架上安装有一个发动机，握把水平安装在手臂支架内，扳机安装在手臂支架上。
进一步地，壳体还包括织物层，织物层固定安装在背板上，且位于远离飞行背覆盖件一侧的背板上。
更进一步地，壳体还包括EVA板，EVA板包裹在织物层上。
进一步地，供油系统包括油箱、空气管路、加油管路、过滤器、油泵和球阀，空气管路和加油管路安装在油箱上，过滤器、油泵和球阀通过管路依次连接，过滤器的一端通过管路与油箱的内部连接，球阀的另一端通过管路与背包微型涡轮发动机连接，为背包微型涡轮发动机供油。
优选地，油箱包括方形箱体和油量传感器，油量传感器竖直并密封插装在方形箱体内部。
优选地，方形箱体包括箱体、多个横向稳流防波板、多个纵向稳流防波板和多个前后稳流防波板，多个横向稳流防波板水平等间距布置、纵向稳流防波板竖直安装在多个横向稳流防波板上，前后稳流防波板纵向安装并与纵向稳流防波板垂直布置，多个横向稳流防波板、纵向稳流防波板和前后稳流防波板组成防波板组件，多个防波板组件顺次安装在箱体内。
进一步地，多个横向稳流防波板、多个纵向稳流防波板和多个前后稳流防波板上均沿垂直于板体的方向开设有通油孔。
优选地，通油孔的形状为圆形孔、矩形孔、锥形孔或长条形孔。
进一步地，电气系统包括动力电池、第一电子控制单元、第二电子控制单元、飞行姿态控制器、电源管理单元和电气系统托盘，电气系统托盘与背板连接，动力电池和飞行姿态控制器前后安装电气系统托盘上，第一电子控制单元和第二电子控制单元分别安装在飞行姿态控制器的左右两侧，电源管理单元安装在飞行姿态控制器下部的电气系统托盘上。
更进一步地，飞行姿态控制器上设有上位机界面。
本发明与现有技术相比具有以下效果：
1、本发明提供的是一种以微型涡轮喷气发动机为动力的低空单人飞行器，该背包相对于现有技术来讲，通过在背板上开设有减重孔、并采用轻质材料进行3D打印而成，整体性好，大大降低了背板的重量，在其他构件的重量不能再减轻时，在保证背板的强度前提下，减轻了整个背包的重量。
2、本发明通过安全带牢靠的绑定在飞行员身上，由于安全带上带有快拆装置，当遇到危险或者特殊情况时，能够快速的与飞行背包进行分离，进而避免在飞行过程中对飞行员造成的伤害。
3、本发明采用了电气系统来实现对飞行过程中的位姿和控制，集成度高。操作简单，而且控制精度高。飞行背包装载大容量油箱、完整油路、电气系统及动力系统，可以为单人飞行器提供最大80KG的推力，维持额定续航时间10分钟，同时带有使用深度强化学习控制器的飞行姿态控制器，能够辅助驾驶员飞行。
4、本发明中主油箱容积非常大，且内置了纵横交错的防波板。防波板的形状、大小、开孔尺寸由动态仿真过程确定，能够有效减少在飞行过程中燃油的晃动，提高了飞行器整体的稳定性和可控性。
5、低空单人飞行器使用了六个两种型号的微型涡轮喷气发动机作为动力系统，每个发动机对于研究人员来讲都是一个黑箱。微型涡轮喷气发动机的ECU只能使用转速控制，这对于整体的控制十分不便，因而需要想办法实现整体的推力控制。首先对每个微型涡轮喷气发动机进行不同幅度的阶跃信号实验，得到控制信号与转速的控制模型，再得到转速与推力之间的控制模型，最终便可以实现单个发动机的推力控制曲线进而实现推力控制。低空单人飞行器的动力系统整体上分为三个部分两个手臂推进装置、飞行背包推进装置。如果单个推进装置中的两个发动机不能实现同步，则会在推进装置轴线上产生弯扭，这对于驾驶员和飞行器的稳定控制都是有害的，因而需要使用底层控制来完成每个推进装置的同步。本发明中的飞行背包利用系统辨识和底层控制技术实现了每组动力装置的同步，使得整体动力输出更加稳定可控，动力变化更加平滑。
6、完成低空单人飞行器的结构设计和算法设计后，设计并搭建可将驾驶员和装备吊起的试验平台，吊起高度应可调节。在该试验平台上完成常规飞行测试和高机动飞行测试。利用上位机实时采集飞行数据，建立数据集，并用新的数据集强化深度学习模型，得到更优秀的控制器，如此反复直到达到我们想要的控制效果。在完成地面试验和算法优化后，在更复杂的环境中，如高山、湖泊，进行测试实验，最终得到低空单人飞行器的性能参数。本发明中的单人飞行器飞行高度设定为3-5m，可以保证驾驶员在低空域畅通无阻；飞行速度最高达120KM/h,允许高速飞行；优化后的控制器保证了悬停稳定和机动型飞行。

具体实施方式
具体实施方式一：结合图1至图18说明本实施方式，本实施方式包括飞行背包和两个手臂推进装置7；
飞行背包包括：安全带1、壳体2、供油系统3、电气系统4和背包微型涡轮发动机5，供油系统3安装在壳体2内部，电气系统4安装在壳体2的上端面上，背包微型涡轮发动机5安装在壳体2的下部；
壳体2包括背板2-1和飞行背覆盖件2-2，背板2-1为板体，其中，背板2-1的上部开设有多个减重孔2-3，且背板2-1采用铝合金制成，背板2-1的四周开设有穿接孔2-4，安全带1的中部通过螺栓和鹰嘴钩固定安装在背板2-1的中部，安全带1的背带穿过穿接孔2-4之后与飞行员的身体连接，飞行背覆盖件2-2为壳体罩，飞行背覆盖件2-2盖装在背板2-1上，飞行背覆盖件2-2的下部设有丝状网2-5；
其中，两个手臂推进装置7分别安装在飞行背包的壳体2的左右两侧；
每个手臂推进装置7均包括发动机支架8、两个发动机9、握把10、扳机11、手臂支架13、挡板条14和两组连杆组件12，两组连杆组件12的一端分别安装在手臂支架13的左右两侧，两组连杆组件12的另一端分别与一个发动机支架8连接，每个发动机支架8上安装有一个发动机9，握把10水平安装在手臂支架13内，扳机11安装在手臂支架13上。
本发明的飞行背包是与低空单人飞行器配合使用的，以微型涡轮喷气发动机为动力的低空单人飞行器的完整组成为：手臂推进装置、飞行背包、飞行姿态控制器及各种电气设备。整体外形如图1所示，其中未显示将驾驶员牢靠固定在飞行器上的安全带。主要设计指标为飞行高度4m、飞行时长10min左右、额定载重100KG以上、可以在任意位置悬停、能完成机动性飞行。
本发明的飞行背包是以微型涡轮喷气发动机为动力的低空单人飞行器中最为复杂的组成部分。
该飞行背包中使用了两个40KG级涡喷，也就是成对并排使用的背包微型涡轮发动机5，其配套的供油系统通过结构的设计已经被合理的安装在背包中。单个发动机的供油系统如图15所示，除此之外，为了保证供油顺畅，油箱的安装位置不低于发动机。在该飞行背包中，供油系统实际使用时，能够同时为6台发动机供油，见图3，布置上每条油路符合供油顺畅的条件。
本发明在开始每个组件的设计之前，首先对飞行器的整体进行受力分析，确定飞行器平衡时各个手臂推进装置所需推力、飞行背包所需推力与三者之间夹角的关系，进而确定微型涡喷发动机的选型与布置(最终选定手臂推进器放置2个推力为30KG级发动机、飞行背包放置2个推力为40KG级发动机，共使用6个微型涡喷发动机，相关分析将在研究基础中说明)。
本实施方式中的手臂推进装置如图17和图18中所展示的那样，手臂推进装置中有两个关键参数，R和β。R为两发动机轴线交点到各自头部的距离、β为发动机轴线与手臂推进器轴线之间夹角，两者如图19中所表示的那样。R和β关系到手臂推进器的整体尺寸、手臂推进器所能产生的最大推力和驾驶员前臂能否保证绝对安全不被发动机喷出气所烫伤。
在通过仿真和实验选定了重要设计参数R-β后(R＝400Mmm,β＝20deg)，我们已经可以完整设计出手臂推进装置。护手和握把的外形根据中国成年人人体尺寸GB/T10000-2023中50％分位数的手臂和手掌相关尺寸设计，其外形如图18所示。护手的复杂外形可以由3D打印铝合金工艺完美复现，其靠近手肘末端的开口具有弹性，可以使用额外的夹紧工具使其贴合使用者的小臂。握把内置两个自复位线性传感器，实际使用时对他们加载反向电压可以获得两个反向的信号，如此不仅可以保证信号的准确性，也利用冗余传感器保证了可靠性。
发动机固定架应轻量化，同时不能在发动机运行时产生太多热变形，还应对发动机夹紧可靠。全速运行时发动机主体外壳温度不超过130℃，因而我们选用6061铝合金作为发动机固定架的材料。发动机固定架的内壁尺寸与发动机外壳过盈配合，两片固定架采用螺栓连接，

具体实施方式二：结合图7说明本实施方式，本实施方式壳体2还包括织物层2-6，织物层2-6固定安装在背板2-1上，且位于远离飞行背覆盖件2-2一侧的背板2-1上。
如此设置，本实施方式的织物层2-6在减轻整个壳体2的重量的前提下，与飞行员接触的部位更加舒适，减轻振动和冲击给飞行员背部带来的不适感。其它组成和连接关系与具体实施方式一相同。
具体实施方式三：结合图7说明本实施方式，本实施方式的壳体2还包括EVA板，EVA板包裹在织物层2-6上。
如此设置，EVA板对织物层2-6起到一个支撑作用，也为织物层2-6的连接提供一个载体，以便更好的与背板连接在一起。其它组成和连接关系与具体实施方式一或二相同。
人机工程上需要考虑背负的舒适性、飞行时对驾驶员固定的可靠性、发动机运行中飞行背包的安全性。背板采用铝合金板和EVA板相结合的形式，如图7所示。为了提高飞行背包的可靠性，除油路和线束等其他部件均需要与铝合金背板刚性连接。铝合金板利用拓扑优化做了较多的减重处理，这使得铝合金板与油箱贴合的部分出现了大片的空挡，为了提高安全性，我们在这部分空挡计划贴一些凯夫拉纤维。这种纤维拥有极好的阻燃隔热能力。铝合金板的另一侧是有织物包裹的EVA防护板，他们共同组成了飞行背包背板。有织物包裹的EVA防护板比较柔软。可以在上面贴气垫等来进一步提高驾驶员背负时背部的舒适度。

飞行背包的背带则可以使用六点式赛车安全带，末端使用鹰嘴勾、螺栓安装在飞行背包上。这种背带可以将背包牢靠的绑定在驾驶员身上。赛车安全带的背带在背负时可能会给驾驶员的肩膀带来比较大的压力，在背带上安装减压肩带能有效解决。赛车安全带的固定非常方便，其末端为鹰嘴勾带螺钉，只要拧在背板上就好。同时，赛车用安全带带有快拆装置，驾驶员在遇到突发问题时可以通过快拆装置迅速与飞行背包分离。该飞行背包可以选用赛诺SNK5D01-2六点式安全带，如图8所示。
飞行背包上还需设计挂载架，挂载架上安装手臂推进器支架和飞行背包支架。单人飞行器在未启动时，装备放在地面上应该能平稳的站立，让飞行背包主体以及手臂推进装置与地面保持安全距离。启动时，可以先点火热机，让六个发动机产生一定的推力来抵消飞行背包自身的重力。而后，驾驶员在半蹲状态下穿戴单人飞行器。站起后，驾驶员操作手臂推进装置的扳机就可以实现低空自由飞行。降落后，驾驶员松开扳机并将手臂推进装置放在手臂推进器支架上，此时各发动机进入怠速状态，驾驶员解下安全带离开，单人飞行器仍然可以平稳的站立在地面上。挂载架、手臂推进器支架、飞行背包支架如图9所示(覆盖件未显示)。

飞行背包的电气系统主要包括：动力电池、V3 ECU组、V4 ECU组、飞行姿态控制器、电源管理单元。我们选用了两块高容量10000mAh的4S LiPo动力电池，在不驱动发动机启动电机、油泵的情况下，两块动力电池可以维持电气系统正常工作48小时以上。电源管理单元负责通断和分发电源、监测动力电池电压状况、紧急切断电源。通断电源的功能实现起来非常简单，只需要安装开关即可。紧急切断电源实现起来也很容易，在电源总线上串联急停开关即可。监测动力电池电压我们采取了两种方式，在测试时我们可以查看电压测量模块的指示电压或容量百分比，在系统中则由飞行姿态控制器实时检测电压，当电源电压低时将在显示屏显示错误代码并蜂鸣提示。
