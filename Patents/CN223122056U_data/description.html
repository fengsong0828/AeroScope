Description

一种可调节式低空飞行器反制装置
技术领域
本实用新型涉及飞行器领域，尤其涉及一种可调节式低空飞行器反制装置。
背景技术
飞行器反制装置是针对无人机等低空可操作飞行器进行反制和防范的装置，它通过侦测、识别、干扰、诱骗、控制或摧毁等手段，确保飞行器在合法、安全、有序的环境中运行，防止其被用于非法活动或造成危害。
目前，在低空飞行器反制装置的使用过程中，往往需要先将反制装置放置在地面上，经检索，中国专利公告号：CN212776459U公开了一种用于核设施防御的无人飞行器反制装置，包括反制本体，所述反制本体底端固定连接有支撑块，所述支撑块内部转动连接有第一圆盘，所述支撑块底端设有稳定块，所述稳定块顶部固定连接第二圆盘，所述第二圆盘贯穿于支撑块内部。上述申请文件通过在稳定块底端设置第一支撑杆，第一支撑杆和滑套的加入，使固定块可以与地面相贴合时，大大增加该装置的稳定性，同时使固定块可以适应不平整的地面，第二滑槽直径大于滑杆直径，可以使固定块在与不平整的地面相贴合时，不会使滑杆与第二滑槽卡死，挡块的加入，使滑杆不会从凹槽内部滑出，卡合槽的截面形状设置为圆弧形，可以卡合杆脱离卡合槽时，更加省力。
上述申请文件虽然能够在一定程度上保证在不平整地面上与地面的贴合，但在实际使用的过程中，当地面为斜面时，可能导致设备整体处于倾斜状态，故可能使得反制装置的扫描范围出现偏差，而当地面较为水平，但地面凹凸不平时，则固定块仅能与地面的凸起处贴合，故贴合位置较少，可能导致设备在使用的过程中，为此提出一种可调节式低空飞行器反制装置来解决上述问题。
实用新型内容
为了弥补以上不足，本实用新型提供了一种可调节式低空飞行器反制装置，旨在改善了现有技术中安装反制装置时当地面不平整时不易调节的问题。
为了实现上述目的，本实用新型采用了如下技术方案：一种可调节式低空飞行器反制装置，包括反制装置本体，所述反制装置本体的底端转动连接有连接盘，所述连接盘的外壁固定连接有观察框，所述连接盘的底端设置有支撑机构，所述支撑机构包括安装杆，所述安装杆的底端开设有滑动槽，所述滑动槽的内壁滑动连接有滑杆，所述滑杆的底端固定连接有支板，所述安装杆的外壁铰接有支撑杆，所述支撑杆的内部设置有调节组件，所述调节组件包括螺纹槽，所述螺纹槽开设在支撑杆的内壁，所述支撑杆处于螺纹槽下方的内壁开设有移动槽，所述螺纹槽的内壁螺纹连接有螺纹杆，所述螺纹杆的底部转动连接有延伸杆，所述延伸杆的外壁与移动槽的内壁滑动连接，所述延伸杆的下部设置有贴地组件。

工作原理：在使用时，工作人员先拉出滑杆43，并将设备通过支板44暂时放置在需要安装的地面上，随后工作人员向外拉动支撑杆45，再拉出推拉块474，使得推拉块474带动插条475向外移动，从而离开插孔476。
当地面为平地时，则调整贴地块472的平面至与地面接触，当地面不平时，则调整贴地块472至移动条478与地面接触，并拉出插杆4712，当插杆4712离开插口4713后，推动控制块4711，从而使得控制块4711带动滑板479移动，从而使得气压仓477内部的气压增大，从而使得移动条478在气压的作用下向下移动，从而使得移动条478的底端与地面接触。
在调整完成贴地块472的位置后，工作人员拉出压板473，并在压板473的上方放置重物，从而使得贴地块472的位置得到固定。
当贴地块472固定完成后，工作人员向上移动滑杆43与支板44，并站在两个支撑杆45之间查看观察框3内的液面顶端与连接盘2上表面的差异，并根据实际情况，调整对应延伸杆464的长度。
在调整延伸杆464长度时，工作人员转动旋钮4611，从而使得旋钮4611带动转动杆469与锥齿轮二4610转动，从而使得锥齿轮二4610带动锥齿轮一468转动，从而使得转杆467转动。
在转杆467转动时，转杆467带动凸起块466转动，从而使得凸起块466在转动的过程中挤压控制槽465的内壁，从而使得螺纹杆463转动，从而使得螺纹杆463在转动的过程中带动延伸杆464移动，从而使得延伸杆464处于移动槽462内部的长度改变，从而使得延伸杆464与支撑杆45的总长度发生改变。
当改变完成后，工作人员重新观察液面与连接盘2上表面的平行情况，并根据情况再次进行调整，直至液面与连接盘2上表面完全平齐，此时能够确保反制装置本体1处于水平状态，从而保证使用设备时不容易出现死角。
