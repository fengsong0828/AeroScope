Description

低空飞行器的自动避障方法及系统
技术领域
本申请涉及自动避障技术领域，具体涉及低空飞行器的自动避障方法及系统。
背景技术
当前，低空飞行器(如多旋翼无人机、垂直起降固定翼飞行器等)在物流运输、电力巡检、农业监测、应急救援等领域得到广泛应用。然而，在复杂地形、城市低空、高速飞行等多变环境下，飞行器容易面临多类型障碍物(如建筑物、电线杆、树木、飞鸟等)的潜在碰撞威胁。传统避障技术大多依赖单一传感器(如超声波或视觉传感器)进行障碍物检测，受限于传感精度、视野范围及环境适应性，难以实现对动态障碍物的实时识别与有效规避。此外，现有系统往往将避障策略固化在飞控算法中，缺乏灵活性和智能性，无法根据不同飞行状态和障碍物特征动态调整应对措施，导致在突发风险情形下响应滞后、避障精度不足，存在较大飞行安全隐患。
发明内容
本申请通过提供了低空飞行器的自动避障方法及系统，旨在解决现有低空飞行器在复杂环境中避障响应滞后，难以实时精准规避障碍物的技术问题。
本申请公开的第一个方面，提供了低空飞行器的自动避障方法，所述方法包括：实时获取低空飞行器的飞行状态信息，所述飞行状态信息包括飞行速度、飞行高度、飞行姿态及实时位置信息；通过环境感知模块获取所述低空飞行器周围环境的障碍物信息，所述障碍物信息至少包括障碍物位置及障碍物尺寸；基于所述飞行状态信息及障碍物信息，计算所述低空飞行器与障碍物之间的相对位置及相对速度，确定潜在碰撞风险区域；若判断存在潜在碰撞风险，基于预设避障策略库，结合所述潜在碰撞风险区域检索并选择避障策略，生成避障控制指令；将所述避障控制指令传输至飞行控制器，控制低空飞行器执行自动避障动作。
本申请公开的另一个方面，提供了低空飞行器的自动避障系统，所述系统包括：飞行状态获取单元：实时获取低空飞行器的飞行状态信息，所述飞行状态信息包括飞行速度、飞行高度、飞行姿态及实时位置信息；环境感知单元：通过环境感知模块获取所述低空飞行器周围环境的障碍物信息，所述障碍物信息至少包括障碍物位置及障碍物尺寸；碰撞风险确定单元：基于所述飞行状态信息及障碍物信息，计算所述低空飞行器与障碍物之间的相对位置及相对速度，确定潜在碰撞风险区域；避障策略选择单元：若判断存在潜在碰撞风险，基于预设避障策略库，结合所述潜在碰撞风险区域检索并选择避障策略，生成避障控制指令；飞行控制单元：将所述避障控制指令传输至飞行控制器，控制低空飞行器执行自动避障动作。

实施例一，如图1所示，本申请提供了低空飞行器的自动避障方法，所述方法包括：
实时获取低空飞行器的飞行状态信息，所述飞行状态信息包括飞行速度、飞行高度、飞行姿态及实时位置信息。
在本申请实施例中，通过低空飞行器搭载的惯性测量单元(IMU)、全球定位系统(GPS)及高度计等传感组件，持续获取低空飞行器在飞行过程中的状态参数，包括当前的飞行速度(即飞行器单位时间内移动的距离，用于判断其运行快慢)、飞行高度(飞行器相对于地面的垂直距离，用于判断其所处的飞行层级)、飞行姿态(通常包含俯仰角、滚转角和偏航角，用于描述飞行器在空间中的方向与姿态变化)，以及实时位置信息(通过GPS或惯性导航系统获取，用于精准定位飞行器在三维空间中的地理坐标)。这些信息作为低空飞行器感知和决策的基础输入，为后续的障碍物检测、碰撞判断和避障路径规划提供动态数据支持。
通过环境感知模块获取所述低空飞行器周围环境的障碍物信息，所述障碍物信息至少包括障碍物位置及障碍物尺寸。
在一个实施例中，利用配置在低空飞行器上的环境感知模块，实时感知和获取飞行器周围环境中的障碍物信息。该模块通常集成多种传感器，如激光雷达、毫米波雷达、视觉摄像头等，通过扫描飞行器四周空间，识别出存在的障碍物。所获取的障碍物信息至少包含两个关键要素，一个是障碍物的位置，表示该障碍物在空间中的具体坐标，用于判断其相对飞行器的位置关系；另一个是障碍物的尺寸，描述障碍物在三维空间中的长宽高范围，用于了解障碍物的边界，这些信息为后续的碰撞风险判断和避障路径计算提供了必要的环境基础。
进一步，本申请提供了通过环境感知模块获取低空飞行器周围环境的障碍物信息，包括：
激活所述环境感知模块，所述环境感知模块包括雷达传感器、激光雷达传感器和视觉传感器；基于所述环境感知模块进行障碍物识别，获得多组障碍物识别信息；对所述多组障碍物识别信息中的位置信息进行融合修正，得到所述障碍物位置。
优选的，首先，激活环境感知模块，该模块集成有多种传感器类型，包括雷达传感器(如毫米波雷达)、激光雷达传感器(如LiDAR)以及视觉传感器(如RGB摄像头或深度摄像头)，各类传感器同时工作，分别从不同角度和分辨率获取飞行器周围障碍物的原始感知数据。随后，基于这些传感器获取的数据执行障碍物识别操作，获取多组障碍物识别信息，每组障碍物识别信息对应一个传感器，其中，雷达可快速检测大体轮廓及运动障碍物，激光雷达提供精细的空间点云分布，视觉传感器则辅助确认物体边缘与类别。之后，从多组障碍物识别信息中提取障碍物的位置信息，具体来说，对于雷达传感器采集的障碍物识别信息，通过计算雷达发射与接收信号的时间差，获得飞行器到障碍物的直线距离，通过多天线阵列的波束形成算法计算障碍物相对飞行器的水平角度位置，再将距离和方位角投影至飞行器本体坐标系，通过三维极坐标转化公式的计算得到障碍物在水平面上的一个雷达粗略位置。对于激光雷达传感器采集的障碍物识别信息，对原始点云数据进行体素下采样或稀疏点筛选，降低计算负担，再通过DBSCAN、Euclidean Cluster Extraction等聚类算法分离独立障碍物，通过计算聚簇的质心坐标获得激光雷达粗略位置。对于视觉传感器采集的障碍物识别信息，应用如YOLO、Faster R-CNN等目标检测算法，识别图像中障碍物并获取其二维图像坐标中心点，再结合图像像素点与对应深度值，进行像素反投影，将图像中心点映射为三维空间坐标，从而得到视觉粗略位置。然后，将每个传感器标记的测量精度的反比作为每个传感器观测的权重，对获得的雷达粗略位置、激光雷达粗略位置和视觉粗略位置进行加权融合，得到障碍物位置，这个障碍物位置消除了因视角不同、分辨率差异或同步误差导致的识别偏差，为后续风险分析与避障决策提供精准的位置输入。
进一步，本申请提供了通过环境感知模块获取低空飞行器周围环境的障碍物信息，包括：
对所述多组障碍物识别信息进行分解，获得激光雷达障碍物识别信息；采用密度聚类对所述激光雷达障碍物识别信息中的点云数据进行障碍物实体分割，提取独立障碍物点云簇；计算所述独立障碍物点云簇在三维坐标系中的极值点坐标，生成障碍物边界框尺寸参数；对所述障碍物边界框尺寸参数进行三维尺寸计算，获得所述障碍物尺寸；将所述障碍物尺寸与所述障碍物位置进行映射，构建所述障碍物信息。
可选的，首先，对由多种传感器联合输出的多组障碍物识别信息进行分解处理，提取出其中由激光雷达传感器生成的障碍物识别数据，该数据主要包含三维点云信息，即由激光扫描所获得的大量空间散点，每个点具有空间坐标(x,y,z)，反映障碍物表面在三维空间中的分布形态。随后，采用密度聚类算法(如DBSCAN)对激光雷达点云数据进行处理，该算法可根据点云中各点的局部密度，结合距离阈值和最小聚类点数，自动识别出属于同一实体的点集，排除离群点，从而将原始的连续点云划分为若干个独立障碍物点云簇，每个点云簇对应一个实际障碍物。之后，对于每一个独立障碍物点云簇，在三维坐标系下计算其极值点坐标，即提取该点云簇在x、y、z轴方向上的最小值和最大值，这些极值点用于构建一个包围障碍物的最小边界框(Bounding Box)，形成该障碍物的边界框尺寸参数。然后，根据上述边界框的三个维度(长、宽、高)进行三维尺寸计算，使用x轴方向上的最大值减去最小值，得到障碍物的长度，使用y轴方向上的最大值减去最小值，得到障碍物的宽度，使用z轴方向上的最大值减去最小值，得到障碍物的高度，通过将这些尺寸数据进行汇总，得到障碍物在实际物理空间中的尺寸数据。最后，将三维障碍物尺寸与前述通过多传感器融合得到的障碍物空间位置进行映射整合，将位置坐标与边界框尺寸结合，完成该障碍物在环境模型中的结构化表示，输出为完整的障碍物信息，包括其在空间中的位置、尺寸和几何形态，作为飞行避障判断的输入依据。
基于所述飞行状态信息及障碍物信息，计算所述低空飞行器与障碍物之间的相对位置及相对速度，确定潜在碰撞风险区域。
在一个实施例中，在获取到低空飞行器当前的飞行状态信息以及周围障碍物信息后，将障碍物的位置坐标转换或映射到与飞行器统一的坐标系中，以确保两者的空间位置具有可比性。随后，根据低空飞行器的实时位置信息和障碍物位置，计算障碍物与低空飞行器之间的直线距离，这个直线距离用于表示障碍物与低空飞行器的相对位置关系。之后，根据激光雷达传感器采集的空间点云数据，计算障碍物速度，其中，如果障碍物为静止障碍物，障碍物速度会直接设为0，再将低空飞行器的速度和障碍物速度进行分解计算，得到两者的相对速度。在得到相对位置和相对速度后，基于预设的安全距离阈值，预测飞行器在未来一段时间内是否可能进入危险接近状态，如果预测结果显示飞行器可能在一定时间内与障碍物接近至小于安全距离，就会判定存在潜在碰撞风险，并根据预测轨迹构建出一个潜在碰撞风险区域，作为后续避障策略选择和控制指令生成的核心依据。这个区域可视为飞行路径上的高风险段，系统将在此基础上提前介入，进行路径调整或主动规避，以提高低空飞行器的自主避障成功率和飞行安全性。
进一步，本申请提供了基于所述飞行状态信息及障碍物信息，计算所述低空飞行器与障碍物之间的相对位置及相对速度，确定潜在碰撞风险区域，包括：
将障碍物信息映射至飞行坐标系，计算所述障碍物与低空飞行器之间的直线距离及相对速度；基于预设安全距离阈值及相对速度，动态预测潜在碰撞轨迹，确定潜在碰撞风险区域。
可选的，将通过环境感知模块获取到的障碍物位置信息转换到与飞行器使用的统一飞行坐标系中，这一过程通常采用坐标转换算法，例如，利用传感器安装位置的外参矩阵(即传感器相对飞行器机体的位置和姿态)将障碍物在传感器坐标系中的位置数据转换为飞行器坐标系下的坐标数据，这样可以确保飞行器自身的位置和障碍物位置在同一空间参考框架下进行计算。随后，在统一坐标系下，根据低空飞行器的实时位置信息和障碍物位置，通过欧氏距离计算障碍物与低空飞行器之间的直线距离，这个直线距离用于表示障碍物与低空飞行器的相对位置关系。同时，根据激光雷达传感器采集的空间点云数据，计算障碍物速度，其中，如果障碍物为静止障碍物，障碍物速度会直接设为0，不进行障碍物速度的计算。之后，通过将低空飞行器的速度和障碍物速度进行分解，得到在X、Y、Z轴方向的速度分量，再使用障碍物的每个方向的速度分量减去飞行器在相应方向上的速度分量得到，障碍物相对于飞行器的速度差值，用于表示两者相对速度方向和分量的向量，再通过模运算对这个相对速度向量进行计算，得到两者的相对速度。然后，根据一个预设的安全距离阈值(即为了确保飞行安全而设定的最小允许距离)，预测低空飞行器在未来一段时间内是否会进入该距离范围，预测过程中会利用相对速度估算飞行器的未来位置轨迹，并判断障碍物是否也处于相对接近状态。如果在预设时间窗口内存在距离小于安全阈值的情况，则说明存在潜在碰撞风险，此时，系统会将这些存在风险的轨迹点集定义为潜在碰撞风险区域，可以作为后续避障策略决策的核心依据，以避免进入高风险区域，影响低空飞行器的飞行安全。
进一步，本申请提供了在计算相对速度前，还包括：
利用所述激光雷达传感器以设定频率连续采集所述障碍物的空间点云数据，构建障碍物时序点云帧序列；对所述障碍物时序点云帧序列进行空间位置匹配，获取所述障碍物的初始位置移动向量，并通过所述飞行状态信息进行飞行器运动补偿处理，生成实际位置移动向量；对所述实际位置移动向量进行速度矢量计算，获得障碍物速度。
可选的，利用激光雷达传感器以设定的采样频率(例如每秒10帧或20帧)持续采集障碍物的空间点云数据，每一帧点云数据都是障碍物在某一时刻的三维表面形态，多个连续时间点采集的点云帧按照时间顺序组合起来，构建成一个障碍物时序点云帧序列，反映障碍物随时间的空间变化过程。随后，对这些连续帧进行空间位置匹配处理，即通过点云配准算法(如ICP算法，Iterative Closest Point)对相邻两帧点云数据进行对齐，识别障碍物在连续时间帧中的位置变化。通过对比不同时刻的点云位置变化，可计算得到障碍物的初始位置移动向量，即在飞行器静止的情况下，障碍物在某时间间隔内移动的方向和距离，其中，如果初始位置移动向量为0或趋近于0(与0的偏差在允许范围内)，说明障碍物为静态障碍物，此时，会将障碍物速度设置为0，不再进行后续的障碍物速度计算。然而，由于飞行器本身也在运动，因此仅根据点云帧对比计算的移动向量会包含飞行器自身运动引入的偏差。为此，需要引入飞行状态信息(如飞行器自身的速度、姿态和位移数据)，对障碍物的移动向量进行补偿处理，排除飞行器运动带来的影响。具体来说，从低空飞行器的飞行状态信息中实时提取加速度、角速度信息、实时位置信息和飞行姿态，构建出一个完整的飞行器运动变换矩阵，该矩阵包含飞行器在一定时间(Δt)内的旋转信息(以旋转矩阵表示)和位移信息(以平移向量表示)，用于描述飞行器自身的刚性运动轨迹，再从激光雷达模块中提取连续两个时刻的点云帧数据，分别为t时刻的源点云帧和t+Δt时刻的目标点云帧。由于飞行器在此期间处于移动状态，导致激光雷达的视角也随之发生变化，从而使得两帧点云在空间上的位置并不一致。为解决这一问题，采用点云配准技术(例如ICP或NDT算法)对两帧点云进行刚性配准处理，以ICP为例，以t时刻的点云帧为源帧，t+Δt时刻的点云帧为目标帧，通过迭代优化，使源帧中每一个点与目标帧中最接近的点之间的欧几里得距离最小，从而求解出最佳刚性变换矩阵，并将该矩阵的逆变换应用于目标帧的点云数据，实现飞行器运动引起的坐标偏移消除。之后，将经过补偿处理后的目标帧中障碍物点云区域与源帧中的对应障碍物区域进行配对匹配，恢复出障碍物在世界坐标系中的真实空间位置变化，也就是说，障碍物的实际位移向量可由其在补偿后的t+Δt时刻位置减去t时刻的位置坐标得到，从而准确反映障碍物自身在飞行器运动影响被剔除情况下的实际位置移动向量，这个实际位移向量再除以时间间隔Δt，即可进一步计算得到障碍物的速度矢量，作为后续相对速度计算和碰撞风险判断的关键输入参数。该过程确保了障碍物速度估计的准确性，有效提升系统对动态目标的识别能力和预测精度，符合飞行器在复杂环境中安全自主飞行的技术要求。
