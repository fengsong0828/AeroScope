Description

一种适用于中低空飞行器航拍测量的图像矫正装置
技术领域
本实用新型涉及农业技术领域，具体为一种适用于中低空飞行器航拍测量的图像矫正装置。
背景技术
航拍测量，又称航空摄影测量，指的是在飞行器上用航摄仪器对地面连续摄取像片，结合地面控制点测量、调绘和立体测绘等步骤，绘制出地形图的作业。航空摄影测量可用于野外微地貌研究、林地考察、考古、文物修缮等领域。
传统的飞行器无法方便的对拍摄的角度进行调节，这样导致拍摄的画面不够理想，且摄像头的表面会聚集异物，影响拍摄的效果。
实用新型内容
(一)解决的技术问题
针对现有技术的不足，本实用新型提供了一种适用于中低空飞行器航拍测量的图像矫正装置，解决了上述背景技术中提出的问题。
(二)技术方案
为实现以上目的，本实用新型通过以下技术方案予以实现：一种适用于中低空飞行器航拍测量的图像矫正装置，包括飞行器本体，所述飞行器本体的上侧壁固定连接有第一电动机，所述第一电动机的输出端贯穿飞行器本体并固定连接有转盘，且转盘的下侧壁固定连接有固定架，且固定架的侧壁固定连接有第二电动机，且第二电动机的输出端贯穿固定架并固定连接有固定框，且固定框内部通过螺栓固定安装有摄像头，所述固定框的下侧壁固定连接有清洁组件。
可选的，所述清洁组件包括固定连接在固定框下侧壁的固定块，且固定块的后端固定连接有第三电动机，且第三电动机的另一端贯穿固定块并固定连接有清洁条。
可选的，所述清洁条与摄像头位置对应，所述清洁条靠近摄像头的一端固定连接有清洁布，且清洁布与摄像头的前端接触。
可选的，所述飞行器本体的下侧壁固定连接有两个对称设置的支撑架。
可选的，所述飞行器本体的前端固定连接有两个对称设置的探照灯。
可选的，所述飞行器本体的前端固定连接有两个对称设置的探照灯。
(三)有益效果
本实用新型提供了一种适用于中低空飞行器航拍测量的图像矫正装置，具备以下有益效果：
1、该适用于中低空飞行器航拍测量的图像矫正装置，通过设置第一电动机和第二电动机，启动第一电动机，带动转盘转动，这样实现该摄像头水平角度的调节，启动第二电动机，带动固定框转动，实现摄像头竖直方向的角度调节，便于对准拍摄物，效果更理想。
2、该适用于中低空飞行器航拍测量的图像矫正装置，通过设置清洁组件，启动第三电动机，带动清洁条转动，这样清洁布能够对该摄像头的表面进行清洁，从而保证拍摄的效果。
