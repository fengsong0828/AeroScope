Description

一种低空飞行器专用电机故障检测方法及系统
技术领域
本发明涉及电机故障检测技术领域，特别是指一种低空飞行器专用电机故障检测方法及系统。
背景技术
随着无人机技术的迅速发展，低空飞行器在民用、商业、科研等多个领域的应用愈加广泛。作为低空飞行器的核心部件之一，专用电机承担着飞行器的动力输出、稳定控制以及任务执行等关键功能，因此对电机的稳定性和可靠性提出了更高的要求。
然而，在复杂的飞行环境中，电机的故障可能对低空飞行器的安全性和稳定性产生重大影响，甚至可能导致飞行器的坠落。因此，低空飞行器电机的实时监控与故障检测显得尤为重要。
在现有技术中，传统的电机故障检测方法大多依赖于单一传感器的数据进行监测，无法充分利用多传感器数据的优势，在面对复杂环境和多故障模式时，单一传感器方法往往难以做到准确和全面地检测电机的潜在故障，且容易受到噪声的干扰，导致误判或漏判。
发明内容
为了解决现有技术中传统的电机故障检测方法大多依赖于单一传感器的数据进行监测，无法充分利用多传感器数据的优势，在面对复杂环境和多故障模式时，单一传感器方法往往难以做到准确和全面地检测电机的潜在故障，且容易受到噪声的干扰，导致误判或漏判的技术问题，本发明提供了一种低空飞行器专用电机故障检测方法及系统。

S1：获取待检测低空飞行器专用电机在工作状态下的多种传感器数据。
可选地，传感器数据包括：多个时间采样点下的电机温度、电机转速、电机电流、电机电压、电机位置角度、电机振动以及电机扭矩。
需要说明的是，电机温度指的是电机内部或外部部件的温度，反映了电机的工作负荷和热损耗情况。
需要说明的是，电机转速指的是电机转子每分钟的旋转次数，反映了电机输出的运动速度。
需要说明的是，电机电流指的是电机运行时消耗的电流大小，反映了电机的负载情况和功率需求。
需要说明的是，电机电压指的是施加在电机上的电压大小，反映了电机的供电状态。
需要说明的是，电机位置角度指的是电机转子的当前位置，反映了电机在执行特定任务时的位置精度。
需要说明的是，电机振动指的是电机在运行过程中产生的振动幅度，反映了电机运行的平衡性。
需要说明的是，电机扭矩指的是电机输出的旋转力矩，反映了电机的工作能力和负载能力。

通过温度传感器、转速传感器、电流传感器、电压传感器、绝对式光电编码器、振动传感器以及扭矩传感器，依次获取待检测低空飞行器专用电机在工作状态下的电机温度、电机转速、电机电流、电机电压、电机位置角度、电机振动以及电机扭矩。

在本发明实施例中，通过多传感器数据采集的方式，能够全面、精准地监控电机的各项关键指标，提升故障诊断的准确性、可靠性和鲁棒性。在复杂环境下工作时，多传感器数据的融合能够有效地弥补单一传感器的不足，提供更全面的信息，有助于在故障发生前及时采取措施，避免设备损坏和停机，确保低空飞行器的安全性和稳定性。
S2：根据各种传感器数据，对待检测低空飞行器专用电机进行初步检测，初步判断待检测低空飞行器专用电机是否处于故障状态。若是，进入S3，否则，返回S1。
可选地，初步检测的过程包括两种可能的实施方式：一种是基于电机温度的统计方法，通过计算电机温度的均值和标准差来判断是否超过设定的温度阈值。另一种是基于MSET算法的状态估计方法，通过构建记忆矩阵并计算状态残差来判断电机是否处于故障状态。在实际应用过程中，可以根据具体需求选择其一。
需要说明的是，MSET（多元状态估计技术，Multivariate State EstimationTechnique）是一种基于多维时间序列数据的非线性分析方法，旨在通过对设备的正常运行数据进行建模，预测和估计系统在不同工作状态下的行为。MSET算法通过构建“记忆矩阵”来捕捉设备的正常运行特征，并利用这些特征来估算设备在实际运行中的状态。该方法通常应用于状态监测和故障检测，通过比较估算状态与实际状态的残差，来判断设备是否存在故障或异常。MSET算法具有较强的抗噪能力，能够处理复杂的非线性系统，并且能有效地识别出设备的潜在故障。
需要说明的是，记忆矩阵（Memory Matrix）是一种用于存储和表示系统历史状态数据的矩阵结构，通常用于多元状态估计（MSET）算法中。在记忆矩阵中，每一行表示系统在不同时间点下的一个传感器数据样本，每一列表示一个特定的传感器特征。通过将设备在正常工作状态下的历史数据组织成矩阵，记忆矩阵能够捕捉系统的动态行为和特征，为后续的状态估计和故障检测提供基础。记忆矩阵的作用是通过学习设备的正常运行模式，为设备状态的预测和异常检测提供参考，从而帮助准确地判断设备是否存在故障。
在本发明实施例中，通过采用两种不同的检测方式（电机温度统计方法和MSET算法），本发明能够提供更高的检测灵活性、准确性和鲁棒性。基于温度的统计方法简单且快速，适用于常规故障检测，而MSET算法则提供了一种综合的、多维数据的分析方法，能够应对复杂环境中的故障检测任务。两者相结合，能够在不同应用场景下有效识别故障、提高诊断精度，并减少误报，提高系统的稳定性和可靠性。

需要说明的是，图数据结构（Graph Data Structure）是一种用于表示对象及其关系的数学结构，由节点（Vertex）和边（Edge）组成，其中节点代表个体对象，边则描述节点之间的关系或关联强度。与传统的线性或矩阵数据不同，图数据能够灵活建模复杂的非欧式空间关系，如社交网络、传感器网络、交通系统等。在图数据结构中，节点可以包含多维特征，边可以带有权重信息，用以量化节点间的相似性、距离或交互强度。通过单层图、多层图或超图（如超邻接矩阵）扩展，图数据结构能够在多粒度、多尺度上捕捉数据的局部与全局关系，成为处理复杂关联型数据的重要工具，广泛应用于图神经网络、知识图谱、结构化数据分析等领域。
在本发明实施例中，采用图数据结构能够全面提升多传感器数据处理的深度和灵活性，充分发挥图神经网络（GCN）等技术的优势，增强低空飞行器电机故障检测系统的准确性、鲁棒性和可扩展性，尤其在复杂环境和数据模式下表现出色。

S4：构建电机故障检测模型。
可选地，S4具体为：结合一维卷积神经网络和图卷积网络，构建电机故障检测模型。
需要说明的是，一维卷积神经网络（1D-CNN，1-Dimensional ConvolutionalNeural Network）是一种针对一维数据（如时间序列、语音信号、传感器数据等）设计的深度神经网络。其核心思想是通过在时间轴上滑动卷积核（滤波器），提取数据中的局部特征和模式。1D-CNN能够有效捕捉信号的局部依赖性与周期性变化，具有参数共享、特征自动提取、计算高效等优点，广泛应用于时序分析、故障检测、金融预测等领域。相比传统手工特征工程，1D-CNN能够通过端到端学习方式自动从原始数据中提取有用特征，提升模型性能和泛化能力。
需要说明的是，图卷积网络（GCN，Graph Convolutional Network）是一种针对图结构数据进行深度学习的神经网络。与传统神经网络处理规则网格数据（如图像、时间序列）不同，GCN能够直接在图中节点及其邻居结构上定义卷积操作，捕捉节点间复杂的关系和拓扑信息。通过在节点及其邻居间聚合、更新特征，GCN可以有效提取局部与全局结构特征，适用于社交网络分析、分子结构预测、交通流量预测等任务。在电机故障检测等场景中，GCN能够充分利用多传感器数据之间的内在关联性，提升检测模型对复杂关系的建模能力和鲁棒性。
在本发明实施例中，通过结合一维卷积神经网络和图卷积网络，实现了时间序列特征与结构化关系特征的深度融合，不仅提升了低空飞行器专用电机故障检测的准确性、鲁棒性与实时性，还增强了模型的适应性和扩展能力，具有显著的工程应用价值和产业推广潜力。
