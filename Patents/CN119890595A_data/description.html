Description

一种低空无人飞行器用半固态电池包
技术领域
本发明涉及电池包技术领域，具体为一种低空无人飞行器用半固态电池包。
背景技术
无人飞行器在飞行过程中，很多时间都在进行直线飞行，以快速到达目标位置，由于低空无人飞行器通常飞行高度并不高，在直线快速飞行过程中，遇到垂直方向的建筑物反射气流时，垂直上升气流会使无人飞行器瞬间获得向上的额外升力，导致机头向上抬起，改变俯仰姿态，对飞行器的稳定性产生的影响较大；并且由于电池技术的发展，具有更高能量密度的半固态电池包也开始应用在无人飞行器中，在无人飞行器直线飞行时，往往速度最快，一旦发生碰撞，冲击力很容易使得半固态电池包中的固态电解质破碎，增加电池热失控风险，存在较大的安全隐患。
发明内容
本发明的目的在于提供一种低空无人飞行器用半固态电池包，以解决上述背景技术中提出的问题。
为实现上述目的，本发明提供如下技术方案：一种低空无人飞行器用半固态电池包，包括轨道壳、活动设置在轨道壳内部的半固态电芯组、与轨道壳固定的压缩气瓶，所述半固态电芯组的一侧设置有前端主簧，所述半固态电芯组的另一侧设置有控制缸，所述压缩气瓶中灌充有压缩气体，所述压缩气瓶用于对控制缸进行驱动，使得控制缸推动半固态电芯组直线移动，所述前端主簧用于推动半固态电芯组复位；
当所述控制缸推动半固态电芯组直线移动时，无人飞行器的重心向着直线飞行的方向前移。
所述控制缸与所述轨道壳固定安装，所述控制缸的端部固定设置有直角托板，所述直角托板中开设有内通气路，所述内通气路与所述控制缸的内腔连通，所述内通气路和所述压缩气瓶之间连通设置有主干气管。
所述主干气管上设置有调压阀，所述主干气管上连通设置有分支气管，所述分支气管处于调压阀远离压缩气瓶所在位置的一侧，所述分支气管上连通设置有放气电磁阀。
所述控制缸的内部设置有活塞体，所述活塞体与所述控制缸之间气密接触，所述活塞体上固定设置有驱动顶轴，所述驱动顶轴上固定设置有端部推盘，所述驱动顶轴通过端部推盘与半固态电芯组接触，推动半固态电芯组移动，所述控制缸的内壁表面固定设置有限位凸部，所述限位凸部用于对活塞体进行限位。
所述轨道壳的内壁表面设置有凸罩仓，所述凸罩仓处于半固态电芯组远离控制缸所在位置的一侧，所述半固态电芯组与所述轨道壳的内壁表面之间气密接触，所述凸罩仓的内壁表面开设有气窗，凸罩仓通过气窗与外界大气连通。
所述凸罩仓的内壁表面上固定设置有检测短管，所述检测短管的两端开口，且检测短管的其中一端与轨道壳的内腔连通，所述检测短管中固定设置有封闭环檐，所述封闭环檐远离半固态电芯组的一侧设置有状态检测盘，所述状态检测盘与所述检测短管的内壁表面气密接触，所述状态检测盘远离封闭环檐的一侧设置有盘侧压簧。
所述状态检测盘上贯穿开设有束流排气孔，所述检测短管上贯穿开设有快速排气槽，所述状态检测盘上固定设置有主动锁扣，所述检测短管上固定设置有配合锁扣，当所述状态检测盘轴向移动经过快速排气槽所在位置时，主动锁扣会与配合锁扣接触扣合，从而将状态检测盘限位。
所述凸罩仓的内壁表面固定设置有气控凸座，所述气控凸座和所述轨道壳的侧壁中开设有气体流道，所述气体流道的一端与压缩气瓶连通，所述气控凸座的内部开设有横截圆柱腔，所述横截圆柱腔将气体流道横截分断。
所述横截圆柱腔中设置有气封动轴，所述气封动轴与所述横截圆柱腔的内壁表面气密接触，通过气封动轴将气体流道封堵，所述气控凸座中开设有轴端槽，所述轴端槽中设置有弹簧压盘，所述弹簧压盘与所述气封动轴固定安装。
所述轴端槽中螺旋安装有螺纹塞，所述螺纹塞与所述弹簧压盘之间设置有动轴弹簧，所述气封动轴上开设有环形气槽。
所述气控凸座上嵌设有气笛，所述气笛与气体流道连通；
当所述状态检测盘轴向移动经过快速排气槽所在位置时，状态检测盘会与气封动轴的端部挤压接触，推动气封动轴轴向移动，使得环形气槽移动到气体流道中，此时压缩气瓶通过气体流道与气笛连通。
所述半固态电芯组上设置有电池插头，所述轨道壳上开设有插头活动窗，所述电池插头与所述插头活动窗位置对应，半固态电芯组通过电池插头与无人飞行器供电连接；
所述压缩气瓶上设置有充压气嘴，通过充压气嘴能够往压缩气瓶中补充压缩气体。
所述轨道壳中设置有泡沫垫，所述泡沫垫上开设有主簧定位孔和仓室槽，所述前端主簧限位设置在主簧定位孔中，所述凸罩仓处于仓室槽中。
与现有技术相比，本发明的有益效果是：
本发明低空无人飞行器用半固态电池包，能够在低空无人飞行器直线快速飞行时，通过调整半固态电芯组的位置来改变无人飞行器的重心，使得无人飞行器重心前移，当无人飞行器重心前移后，由于机头惯性增加，不容易被气流干扰，从而提高快速直线飞行时的稳定性；而在非快速直线飞行时，能够控制重心复位，回到初始状态，保证无人飞行器的动态操控性。
同时通过上述结构的设置，使得无人飞行器直线快速飞行过程中发生撞击时，半固态电芯组能够通过前端主簧，相对于轨道壳进一步前移缓冲，此时半固态电芯组的动能部分转化为前端主簧的弹性势能，能够降低无人飞行器直线快速飞行过程中的碰撞冲击力峰值，进而降低损坏程度，更大程度的对半固态电芯组进行保护，降低半固态电芯组中的固态电解质破碎概率。
本发明通过设置的检测短管、状态检测盘和气封动轴等结构配合，能够自动区分半固态电芯组的缓慢调节移动和碰撞冲击移动，当半固态电芯组在碰撞下冲击移动时，上述结构能够自动将压缩气瓶与气笛连通，使得无人飞行器发出持续笛音，从而在碰撞导致断电的情况下，能够通过笛音对无人飞行器进行定位寻找。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
请参阅图1至图8，本发明提供一种技术方案：一种低空无人飞行器用半固态电池包，如图8中所示，包括轨道壳1、活动设置在轨道壳1内部的半固态电芯组2、与轨道壳1固定的压缩气瓶4，轨道壳1采用铝制成，提高半固态电池包的轻量化，半固态电芯组2由多块半固态电池串并联构成，半固态电芯组2的一侧设置有前端主簧3，半固态电芯组2的另一侧设置有控制缸5，压缩气瓶4中灌充有压缩气体，压缩气瓶4用于对控制缸5进行驱动，使得控制缸5推动半固态电芯组2直线移动，前端主簧3用于推动半固态电芯组2复位；当控制缸5推动半固态电芯组2直线移动时，无人飞行器的重心向着直线飞行的方向前移。
控制缸5与轨道壳1固定安装，控制缸5的端部固定设置有直角托板501，直角托板501在起到气体连通作用的同时，能够对控制缸5的尾端进行稳定支撑，直角托板501中开设有内通气路502，内通气路502与控制缸5的内腔连通，内通气路502和压缩气瓶4之间连通设置有主干气管503。主干气管503上设置有调压阀504，调压阀504是一种现有技术中的压力调节阀门，用于控制压缩气瓶4的输出气压。
主干气管503上连通设置有分支气管505，分支气管505处于调压阀504远离压缩气瓶4所在位置的一侧，分支气管505上连通设置有放气电磁阀506。控制缸5的内部设置有活塞体507，活塞体507与控制缸5之间气密接触，活塞体507上固定设置有驱动顶轴508，驱动顶轴508上固定设置有端部推盘509，驱动顶轴508通过端部推盘509与半固态电芯组2接触，推动半固态电芯组2移动，控制缸5的内壁表面固定设置有限位凸部510，限位凸部510用于对活塞体507进行限位。通过调压阀504改变作用在活塞体507表面的气压，使得活塞体507产生不同大小的推力，与前端主簧3的弹力在不同位置达到平衡，从而实现对半固态电芯组2的驱动位置调整。轨道壳1的内部可以设置位置传感器，对半固态电芯组2处于轨道壳1的位置进行闭环反馈。
轨道壳1的内壁表面设置有凸罩仓101，凸罩仓101处于半固态电芯组2远离控制缸5所在位置的一侧，半固态电芯组2与轨道壳1的内壁表面之间气密接触，凸罩仓101的内壁表面开设有气窗102，凸罩仓101通过气窗102与外界大气连通。凸罩仓101的内壁表面上固定设置有检测短管103，检测短管103的两端开口，且检测短管103的其中一端与轨道壳1的内腔连通，检测短管103中固定设置有封闭环檐104，封闭环檐104远离半固态电芯组2的一侧设置有状态检测盘105，状态检测盘105与检测短管103的内壁表面气密接触，状态检测盘105远离封闭环檐104的一侧设置有盘侧压簧106。
状态检测盘105上贯穿开设有束流排气孔107，检测短管103上贯穿开设有快速排气槽108，状态检测盘105上固定设置有主动锁扣109，检测短管103上固定设置有配合锁扣110，当状态检测盘105轴向移动经过快速排气槽108所在位置时，主动锁扣109会与配合锁扣110接触扣合，从而将状态检测盘105限位。
凸罩仓101的内壁表面固定设置有气控凸座111，气控凸座111和轨道壳1的侧壁中开设有气体流道112，气体流道112的一端与压缩气瓶4连通，如图3和图6中所示，气体流道112开设在气控凸座111和轨道壳1的侧壁中，如图3中所示，在压缩气瓶4的底部设置有管道，通过管道与轨道壳1侧壁中的气体流道112连通。
气控凸座111的内部开设有横截圆柱腔113，横截圆柱腔113将气体流道112横截分断。横截圆柱腔113中设置有气封动轴114，气封动轴114与横截圆柱腔113的内壁表面气密接触，通过气封动轴114将气体流道112封堵，气控凸座111中开设有轴端槽115，轴端槽115中设置有弹簧压盘116，弹簧压盘116与气封动轴114固定安装。轴端槽115中螺旋安装有螺纹塞117，螺纹塞117与弹簧压盘116之间设置有动轴弹簧118，气封动轴114上开设有环形气槽119。
气控凸座111上嵌设有气笛120，气笛120与气体流道112连通，当状态检测盘105轴向移动经过快速排气槽108所在位置时，状态检测盘105会与气封动轴114的端部挤压接触，推动气封动轴114轴向移动，使得环形气槽119移动到气体流道112中，此时压缩气瓶4通过气体流道112与气笛120连通。
半固态电芯组2上设置有电池插头201，轨道壳1上开设有插头活动窗202，电池插头201与插头活动窗202位置对应，半固态电芯组2通过电池插头201与无人飞行器供电连接，无人飞行器中设置有能够与电池插头201公母插接配合的插头，并通过线排连接，不影响半固态电芯组2的活动；压缩气瓶4上设置有充压气嘴401，通过充压气嘴401能够往压缩气瓶4中补充压缩气体，可以在压缩气瓶4中设置压力传感器，当压缩气瓶4中气压不足时进行提醒。
轨道壳1中设置有泡沫垫301，泡沫垫301采用防火材料更加安全，泡沫垫301上开设有主簧定位孔302和仓室槽303，前端主簧3限位设置在主簧定位孔302中，凸罩仓101处于仓室槽303中。
本发明低空无人飞行器用半固态电池包在安装时，需要使得半固态电芯组2的调节移动方向平行于无人飞行器的直线飞行方向；若当电池包应用在能够多方向直线飞行的无人飞行器中时，可以在轨道壳1的外部设置旋转舵机，旋转舵机为现有技术中能够进行旋转驱动，并具有自锁结构的装置，本申请中不再赘述，通过控制轨道壳1转动，使得半固态电芯组2的调节移动方向始终平行于无人飞行器的直线飞行方向。并且使得前端主簧3处于飞行方向的前部，在直线飞行过程中发生撞击时，前端主簧3能够对半固态电芯组2进行弹性支撑缓冲。
初始状态如图4中所示，此时在前端主簧3的弹性推力下，半固态电芯组2被挤压到最右侧；在此初始状态下，调整无人飞行器的配重，使得无人飞行器的重心居中，在重心居中时，无人飞行器通常具有较高的操控性。也可以根据特殊需要进行特调。
当无人飞行器进行直线方向的高速飞行时，控制调压阀504开启，通过调压阀504释放压缩气瓶4中的压缩气体，压缩气体通过主干气管503和内通气路502进入控制缸5中，推动活塞体507和驱动顶轴508移动，使得端部推盘509推动半固态电芯组2调节移动，将前端主簧3进行压缩。当半固态电芯组2向着前端主簧3移动时，由于前端主簧3处于飞行方向的前端，此时会使得无人飞行器的重心沿着飞行方向前移。靠前的重心会使无人飞行器头部惯性较大，更不容易因气流干扰而改变姿态，从而增强了俯仰稳定性。当受到垂直方向干扰气流使无人飞行器姿态发生改变时，重心靠前会产生更有效的恢复力矩。例如，当气流使无人飞行器机头向上抬起时，靠前的重心会形成一个较大的向下的重力分量，该重力分量相对重心产生的力矩能够促使机头向下恢复到初始水平姿态。就像钟摆，重心靠前类似于钟摆的重心更靠近摆动轴，受到外力偏离平衡位置后，更容易回到初始位置。
调压阀504具有压力调节作用，能够控制压缩气瓶4的输出气体压力，通过控制气压，来控制半固态电芯组2的平衡位置，实现对半固态电芯组2位置的调整。值得注意的是，调整半固态电芯组2的移动位置时，半固态电芯组2向着泡沫垫301所在方向移动调整到最大程度，也要保证半固态电芯组2和泡沫垫301之间存在一定的缓冲间隙。无人飞行器直线飞行时，往往速度最快，当无人飞行器在直线飞行过程中发生碰撞时，碰撞冲击往往平行于半固态电芯组2的调节移动方向，此时半固态电芯组2会相对于轨道壳1前移缓冲，将部分动能转化为前端主簧3的弹性势能，降低无人飞行器直线快速飞行过程中的碰撞冲击力峰值。
在无人飞行器非直线飞行时，控制调压阀504关闭，控制放气电磁阀506开启，此时控制缸5中的压缩气体通过放气电磁阀506排出，在前端主簧3的弹力下，半固态电芯组2复位到初始状态，提高无人飞行器的操控性。
上述过程中，如图7中所示，状态检测盘105能够对半固态电芯组2的移动状态进行检测，当半固态电芯组2被控制缸5推动低速调节移动时，此时半固态电芯组2在轨道壳1的内部相当于活塞，推动轨道壳1中的气体通过束流排气孔107排出到外界，由于半固态电芯组2的移速较低，轨道壳1中的气体有充足的时间通过束流排气孔107，不会使得状态检测盘105发生移动。而在直线飞行过程中发生碰撞时，半固态电芯组2向着前端主簧3快速冲击移动，使得气体来不及通过束流排气孔107排出，气压会推动状态检测盘105后移，使得状态检测盘105经过快速排气槽108位置，通过快速排气槽108对外排气。
当状态检测盘105经过快速排气槽108位置后，主动锁扣109会与配合锁扣110接触锁定，此时气封动轴114被状态检测盘105挤压轴移并保持住，使得环形气槽119处于气体流道112中，由于气体流道112与压缩气瓶4连通，压缩气瓶4中的压缩气体，经过气体流道112和环形气槽119进入到气笛120中，使得气笛120发出笛音；若无人飞行器在碰撞过程中电路损坏，失去定位和供电时，通过上述物理笛音能够更加方便的寻找无人飞行器，通过控制气体流道112的直径，使得压缩气瓶4中的压缩气体释放流量足够小，保证充足的释放时间。
