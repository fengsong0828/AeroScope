Description

一种适用于低空经济的飞行器工作管理平台及方法
技术领域
本发明涉及低空经济管理领域，尤其涉及一种适用于低空经济的飞行器工作管理平台及方法。
背景技术
由于低空经济应用场景涵盖范围非常广泛，如载人空中交通、物流运输、应急救援、农林植保、旅游观光及个人航拍等，未来必然会出现大量低空飞行器同时工作的场景，如果不提前进行飞行器工作的管理规划，那必然导致整个空中网络的混乱，甚至出现安全隐患。
发明内容
针对现有技术的不足，本发明提出一种适用于低空经济的飞行器工作管理平台及方法，对不同应用场景的低空飞行器进行飞行工作管理，保障大量低空飞行器有序安全高效的工作，有助于促进低空经济的发展。
具体技术方案如下：
一种适用于低空经济的飞行器工作管理平台，包括：云端数据库、总控中心、终端低空飞行器；所述云端数据库包括：低空飞行器信息管理数据库、三维地形图数据库、低空飞行器实时动态数据库；
所述总控中心用于根据终端低空飞行器发出的工作任务请求信息，结合空域实时信息进行路径规划，生成的飞行计划信息发送至终端低空飞行器；所述空域实时信息根据三维地形图数据库和低空飞行器实时动态数据库得到；所述飞行计划信息包括飞行高度区间和最优路径，所述最优路径为同时满足两个条件的最短距离路径，两个条件分别为：权限级别高的飞行器优先、飞行器安全边界不重叠。
进一步地，所述工作任务请求信息包括：工作任务类型、飞行器起降位置、负载重量、任务时间。
进一步地，所述低空飞行器信息管理数据库根据低空飞行器的标签信息进行分类存储，标签信息包括：工作任务类型、型号、载重、飞行高度、飞行速度、权限级别；
所述工作任务类型包括：载人空中交通、大型运输、小型物流、应急救援、农林植保、旅游观光、个人航拍。
进一步地，所述终端低空飞行器的飞行高度范围为0～3000米，按照每500米高度设置一个区间，共6个区间；不同工作任务类型的终端低空飞行器分配的飞行高度区间根据要求设定。
进一步地，所述终端低空飞行器的安全边界设置原则与终端低空飞行器的工作任务类型关联，小型物流、应急救援、农林植保、个人航拍不设安全边界，载人空中交通、大型运输、旅游观光的安全边界空间为其对应前、后、左、右、上、下边界所构成的长方体空间。
一种适用于低空经济的飞行器工作管理方法，根据所述的适用于低空经济的飞行器工作管理平台实现，包括如下步骤：
S1：终端低空飞行器向总控中心发出飞行器工作任务请求，总控中心向低空飞行器信息管理数据库查询该终端低空飞行器的信息是否入库，若已入库则接受该工作任务请求，否则退回该请求并要求终端低空飞行器进行注册并完成信息入库；
S2：总控中心确认接受终端低空飞行器工作任务请求后，与该终端低空飞行器保持连接，同步向低空飞行器信息管理数据库调取该终端低空飞行器的全部飞行器信息，并访问低空飞行器实时动态数据库与三维地形图数据库获取其协同生成的当前空域实时信息；
S3：总控中心根据飞行工作任务，结合当前空域的实时信息，通过内置路径规划智能算法，生成飞行计划信息并发送到终端低空飞行器；飞行计划信息包括：飞行高度区间及最优路径，最优路径为满足权限级别高的飞行器优先且各飞行器安全边界不重叠条件下的最短距离路径；
S4：终端低空飞行器按照总控中心发送的飞行计划信息进行飞行工作，飞行工作期间通过总控中心向低空飞行器实时动态数据库传输实时状态及位置信息，以及保持接受总控中心的实时飞行计划信息变更；
S5：终端低空飞行器完成全部飞行工作任务后，向总控中心发送任务结束信息，总控中心断开与该终端低空飞行器的连接。
本发明的有益效果是：
本发明提供一种适用于低空经济的飞行器工作管理平台及方法，该方法根据不同应用场景的低空飞行器的工作任务请求信息以及飞行器详细信息，为低空飞行器分配合适的飞行高度区间及最优路径，保障低空飞行器有序安全高效的工作，促进低空经济行业的发展。

如图1所示，一种适用于低空经济的飞行器工作管理平台，包括：云端数据库、总控中心、终端低空飞行器。云端数据库包括：低空飞行器信息管理数据库、三维地形图数据库、低空飞行器实时动态数据库。终端低空飞行器所适用的低空经济高度范围为0～3000m。
总控中心用于根据终端低空飞行器发出的工作任务请求信息，结合低空飞行器信息管理数据库中的飞行器信息、三维地形图数据库与低空飞行器实时动态数据库协同生成的空域实时信息，通过总控中心内置的路径规划智能算法，生成飞行计划信息并发送到终端低空飞行器。总控中心在低空飞行器工作期间始终保持与终端低空飞行器的实时连接，具备对终端低空飞行器飞行计划临时调整的功能。工作任务请求信息包括：工作任务类型、飞行器起降位置、负载重量、任务时间等。
低空飞行器信息管理数据库根据低空飞行器的标签信息进行分类存储，标签信息包括：工作任务类型、型号、载重、飞行高度、飞行速度、权限级别等，典型的低空飞行器标签信息实例如表1所示。工作任务类型包括：载人空中交通、大型运输、小型物流、应急救援、农林植保、旅游观光、个人航拍。优选的，权限级别由高到低包括S、A、B、C四个级别，与工作任务类型匹配，具体见表2。低空飞行器实时动态数据库存储有终端低空飞行器的实时状态及位置信息，在终端低空飞行器工作期间通过总控中心与低空飞行器实时连接。
