Description

一种机场低空飞行器监视方法及系统
技术领域
本发明涉及航空航天监控技术领域，特别是一种机场低空飞行器监视方法及系统。
背景技术
在现代机场运营中，低空飞行器的监视和管理成了一个重要的安全问题。随着无人机和其他低空飞行器的广泛应用，传统的雷达和通信系统在监控低空飞行器方面的能力有限，无法满足对飞行器高精度、高实时性监控的需求。现有技术通常依赖于有限的监测手段，如地面雷达、固定传感器和手动监控，这些方法难以有效处理密集的低空飞行器流量，导致监测盲区和响应延迟，从而影响到机场及其周边空域的安全。
此外，传统的无线电信号处理技术在应对复杂信号环境和多信号重叠的情况下也显得力不从心。现有系统在处理低空飞行器发出的无线电信号时，存在信号干扰、噪声影响以及实时分析能力不足等问题。这些技术限制使得对飞行器的分类、定位和行为预测难以做到精确和及时，因此亟需一种更为先进的监视方法和系统，以提升低空飞行器监控的整体效果和可靠性。
发明内容
鉴于现有的低空飞行器监视系统在精确度、实时性和信号处理能力上存在不足，导致对低空飞行器的监测和管理不够有效，提出了本发明。
因此，本发明所要解决的问题在于如何高效、精确地监视和定位低空飞行器的问题，从而有效保障机场及其周边空域的安全。
为解决上述技术问题，本发明提供如下技术方案：
第一方面，本发明实施例提供了一种机场低空飞行器监视方法，其包括在机场周边预设区域内布置多个软件定义无线电接收器；利用软件定义无线电接收器扫描并捕获低空区域的无线电信号，生成待传输数据包；将待传输数据包通过边缘网络传输至中央处理服务器，在中央处理服务器中对待传输数据包进行解析和特征提取，生成信号特征信息；基于信号特征信息，采用多源融合定位算法，结合多个软件定义无线电接收器的信号时间差和频率差，定位低空飞行器并生成综合定位信息；结合综合定位信息和信号特征信息，使用图神经网络构建动态飞行行为模型，实时评估潜在威胁；当检测到威胁时，启动定向天线阵列，向目标区域发射与低空飞行器相同频率的仿真控制信号，对低空飞行器进行安全引导或接管控制。
作为本发明所述机场低空飞行器监视方法的一种优选方案，其中：生成待传输数据包包括以下步骤：设置软件定义无线电接收器的扫描参数，适应低空区域的信号特征；利用可编程射频前端和高速模数转换器，接收并数字化低空区域的无线电信号，得到数字化无线电信号；使用数字信号处理器，对数字化无线电信号执行多级小波变换降噪；利用信号分析软件模块，对降噪后的数字化无线电信号进行时频分析，生成时频图谱；启动信号分类模块，基于时频图谱对无线电信号进行分类；控制多元天线阵列执行波束形成，针对各类信号生成可调向波束信号；对可调向波束信号执行快速傅里叶变换，得到信号频谱特征；将信号频谱特征与对应的信号类型标签组合，生成待传输数据包。

作为本发明所述机场低空飞行器监视方法的一种优选方案，其中：基于时频图谱对无线电信号进行分类包括以下步骤：对时频图谱进行图像预处理；从预处理后的时频图谱中提取特征向量，所述特征向量包括频率带宽、时间持续性和能量分布；利用预训练的卷积神经网络模型对特征向量进行初次分类；基于初次分类结果，应用集成学习方法进行细化分类；对细化分类结果应用时序一致性检查，消除瞬时误分类；利用频谱占用分析方法，识别时频图谱中的多信号重叠区域；对多信号重叠区域应用盲源分离算法，分离出独立的信号成分；将分离出的信号成分重新输入分类流程，进行二次分类；综合初次分类结果和二次分类结果，生成信号类型标签；将信号类型标签与原始时频图谱关联，形成标记后的时频图谱。
作为本发明所述机场低空飞行器监视方法的一种优选方案，其中：针对各类信号生成可调向波束信号包括以下步骤：获取多元天线阵列的几何布局参数和每个天线单元的相位中心位置；基于各类信号的频率特性，计算每种信号对应的波长；对每种识别的信号类型，确定期望的主波束方向和旁瓣抑制级别；利用自适应波束形成算法，计算主波束方向和旁瓣抑制级别所需的每个天线单元的幅度和相位权值；将计算得到的幅度和相位权值应用到多元天线阵列的每个天线单元；通过调整天线单元的数字移相器，进行相位控制；利用可变增益放大器调节每个天线单元的信号幅度，与相位控制相配合；执行实时信号合成，将各天线单元接收到的信号按照设定的幅度和相位进行叠加；对合成后的信号进行数字滤波，去除非目标频段的干扰；生成可调向波束信号，并将其与对应的信号类型标识关联存储。
作为本发明所述机场低空飞行器监视方法的一种优选方案，其中：定位低空飞行器并生成综合定位信息包括以下步骤：根据信号类型标签和关键参数，动态选择定位算法组合利用时间相关性和频率相关性，计算多个软件定义无线电接收器之间的信号时间差和频率差；将信号时间差和频率差输入选定的定位算法组合，计算低空飞行器的三维坐标和速度矢量；基于时间相关性和频率相关性的变化趋势，构建低空飞行器运动模型，计算预测轨迹；根据关键参数的时变特性，实时调整定位算法组合中各算法的权重系数；利用多个软件定义无线电接收器的空间分布信息，构建协同定位网络，提高定位精度并降低单点故障风险；基于定位结果和预测轨迹，动态调整软件定义无线电接收器的参数，包括波束方向和增益；计算定位结果的可信度指数，评估定位精度；生成综合定位信息，将综合定位信息与对应的信号特征信息关联，更新数据库中的记录。
作为本发明所述机场低空飞行器监视方法的一种优选方案，其中：动态选择定位算法组合包括以下步骤：构建定位算法库；基于信号类型标签，从定位算法库中初步筛选适用的定位算法子集；利用关键参数中的中心频率和带宽，计算信号的时频分辨率指标，根据时频分辨率指标对定位算法子集中的算法进行优先级排序；基于关键参数中的峰值功率，估算信号的信噪比，根据信噪比从排序后的定位算法子集中选择前N个算法；构建定位算法组合评分模型，评分模型考虑算法复杂度、计算资源需求和历史定位精度；利用定位算法组合评分模型，对选择的N个算法进行组合优化，得到最优定位算法组合。
第二方面，本发明实施例提供了机场低空飞行器监视系统，其包括接收器布置模块，用于在机场周边预设区域内布置多个软件定义无线电接收器；数据包生成模块，用于利用软件定义无线电接收器扫描并捕获低空区域的无线电信号，生成待传输数据包；特征提取模块，用于将待传输数据包通过边缘网络传输至中央处理服务器，在中央处理服务器中对待传输数据包进行解析和特征提取，生成信号特征信息；定位模块，用于基于信号特征信息，采用多源融合定位算法，结合多个软件定义无线电接收器的信号时间差和频率差，定位低空飞行器并生成综合定位信息；威胁评估模块，用于结合综合定位信息和信号特征信息，使用图神经网络构建动态飞行行为模型，实时评估潜在威胁；控制模块，用于当检测到威胁时，启动定向天线阵列，向目标区域发射与低空飞行器相同频率的仿真控制信号，对低空飞行器进行安全引导或接管控制。

第四方面，本发明实施例提供了一种计算机可读存储介质，其上存储有计算机程序，其中：所述计算机程序指令被处理器执行时实现如本发明第一方面所述的机场低空飞行器监视方法的步骤。
本发明有益效果为：本发明通过利用高精度的软件定义无线电接收器，结合多级小波变换降噪和先进的信号分类算法，实现了对低空区域无线电信号的精确捕获、处理和分类。采用多源融合定位算法，结合信号时间差和频率差，高效地实现了低空飞行器的实时定位，并动态调整定位算法和参数以提高定位精度。通过图神经网络实时评估飞行行为，早期预警潜在威胁，并在检测到威胁时，利用定向天线阵列发射仿真控制信号，实现对目标飞行器的安全引导或接管控制，有效保障机场周边空域的安全。

具体的，对时频图谱进行图像预处理，包括去噪和对比度增强；从预处理后的时频图谱中提取特征向量，包括频率带宽、时间持续性和能量分布；利用预训练的卷积神经网络模型对特征向量进行初次分类；基于初次分类结果，应用集成学习方法(如随机森林或梯度提升树)进行细化分类；对细化分类结果应用时序一致性检查，消除瞬时误分类；利用频谱占用分析方法，识别时频图谱中的多信号重叠区域；对多信号重叠区域应用盲源分离算法，分离出独立的信号成分；将分离出的信号成分重新输入分类流程，进行二次分类；综合初次分类结果和二次分类结果，生成最终的信号类型标签；将信号类型标签与原始时频图谱关联，形成标记后的时频图谱。
S2.6：控制多元天线阵列执行波束形成，针对各类信号生成可调向波束信号。
具体的，获取多元天线阵列的几何布局参数和每个天线单元的相位中心位置；基于各类信号的频率特性，计算每种信号对应的波长；对每种识别的信号类型，确定期望的主波束方向和旁瓣抑制级别；利用自适应波束形成算法，计算主波束方向和旁瓣抑制级别所需的每个天线单元的幅度和相位权值；将计算得到的幅度和相位权值应用到多元天线阵列的每个天线单元；通过调整天线单元的数字移相器，进行相位控制；利用可变增益放大器调节每个天线单元的信号幅度，与相位控制相配合；执行实时信号合成，将各天线单元接收到的信号按照设定的幅度和相位进行叠加；对合成后的信号进行数字滤波，去除非目标频段的干扰；生成可调向波束信号，并将其与对应的信号类型标识关联存储。
S2.7：对可调向波束信号执行快速傅里叶变换，得到信号频谱特征。
S2.8：将信号频谱特征与对应的信号类型标签组合，生成待传输数据包。
优选的，在数据包中添加时间戳、地理位置信息(如有)等元数据，以便后续处理和分析。
S3：将待传输数据包通过边缘网络传输至中央处理服务器，在中央处理服务器中对待传输数据包进行解析和特征提取，生成信号特征信息。
具体的，通过边缘网络将待传输数据包传输至中央处理服务器；在中央处理服务器中，解析待传输数据包，提取信号频谱特征和对应的信号类型标签；对信号频谱特征进行归一化处理，消除不同接收器间的增益差异；基于信号类型标签，选择相应的特征提取算法；从归一化后的信号频谱特征中提取关键参数，包括中心频率、带宽和峰值功率；计算信号频谱特征的时间相关性和频率相关性；结合关键参数、时间相关性和频率相关性，生成信号特征信息；将信号特征信息与原始信号类型标签关联，更新数据库记录。
S4：基于信号特征信息，采用多源融合定位算法，结合多个软件定义无线电接收器的信号时间差和频率差，定位低空飞行器并生成综合定位信息。
具体的，包括以下步骤：
S4.1：从信号特征信息中提取信号类型标签、关键参数、时间相关性和频率相关性。
S4.2：根据信号类型标签和关键参数，动态选择定位算法组合。
优选的，构建定位算法库，定位算法库包含时差到达、频率差到达、到达角、多普勒频移等相关定位算法；基于信号类型标签，从定位算法库中初步筛选适用的定位算法子集；利用关键参数中的中心频率和带宽，计算信号的时频分辨率指标；根据时频分辨率指标，对定位算法子集中的算法进行优先级排序；基于关键参数中的峰值功率，估算信号的信噪比；根据信噪比，从排序后的定位算法子集中选择前N个算法，其中N为预设阈值；构建定位算法组合评分模型，评分模型考虑算法复杂度、计算资源需求和历史定位精度；利用定位算法组合评分模型，对选择的N个算法进行组合优化，得到最优定位算法组合。
S4.3：利用时间相关性和频率相关性，计算多个软件定义无线电接收器之间的信号时间差和频率差。
S4.4：将信号时间差和频率差输入选定的定位算法组合，计算低空飞行器的三维坐标和速度矢量。
S4.5：基于时间相关性和频率相关性的变化趋势，构建低空飞行器运动模型，计算预测轨迹。
优选的，获取时间相关性和频率相关性的历史数据序列，对历史数据序列进行时间窗口分段处理，得到多个时间窗口的数据子序列；对多个时间窗口的数据子序列分别执行快速傅里叶变换，提取每个时间窗口内的主要频率分量及其幅值；基于主要频率分量及其幅值，构建时变自回归移动平均模型，时变自回归移动平均模型的参数随时间窗口动态调整；利用时变自回归移动平均模型，计算时间相关性和频率相关性的短期预测值；将短期预测值输入预先训练的长短期记忆神经网络，长短期记忆神经网络用于捕捉时间相关性和频率相关性的长期变化特征；基于长短期记忆神经网络的输出，构建低空飞行器的非线性状态空间模型，非线性状态空间模型包含位置、速度和加速度等状态变量；对非线性状态空间模型应用无迹卡尔曼滤波算法，获得低空飞行器状态的最优估计；基于最优估计的状态，利用龙格-库塔法对非线性状态空间模型进行数值积分，得到低空飞行器未来一段时间内的预测轨迹点集。
S4.6：根据关键参数的时变特性，实时调整定位算法组合中各算法的权重系数。
S4.7：利用多个软件定义无线电接收器的空间分布信息，构建协同定位网络，提高定位精度并降低单点故障风险。
S4.8：基于定位结果和预测轨迹，动态调整软件定义无线电接收器的参数，包括波束方向和增益。
S4.9：计算定位结果的可信度指数，考虑因素包括关键参数中的峰值功率、参与定位的接收器数量和几何分布，评估定位精度。
S4.10：生成综合定位信息，将综合定位信息与对应的信号特征信息关联，更新数据库中的记录。
需要说明的是，综合定位信息包括低空飞行器的三维坐标、低空飞行器的三维坐标、预测轨迹点集、定位精度估计值以及可信度指数。
优选的，该步骤通过采用多源融合定位算法，实现了对低空飞行器的高精度、实时定位和轨迹预测。通过动态选择定位算法组合，系统能够根据不同信号特征灵活调整定位策略，提高了适应性和准确性。利用时间相关性和频率相关性，结合多个接收器的信息，不仅提高了定位精度，还增强了系统的抗干扰能力。构建的低空飞行器运动模型和预测轨迹功能，为及时识别潜在威胁提供了关键支持。实时调整定位算法权重和接收器参数，进一步优化了系统性能。协同定位网络的构建提高了系统可靠性，而可信度指数的计算则为定位结果提供了质量保证。
S5：结合综合定位信息和信号特征信息，使用图神经网络构建动态飞行行为模型，实时评估潜在威胁。
具体的，从综合定位信息中提取低空飞行器的三维坐标、速度矢量和预测轨迹点集，从信号特征信息中提取信号类型标签、关键参数、时间相关性和频率相关性；构建时空图结构，其中图节点表示低空飞行器在不同时间点的状态，图边表示相邻时间点之间的状态转移；为图节点分配特征向量，特征向量包含三维坐标、速度矢量、信号类型标签和关键参数；为图边分配权重，权重基于时间相关性、频率相关性和相邻时间点之间的状态变化计算得出；利用时间卷积网络对特征向量进行局部特征提取，得到节点的局部时间特征；将局部时间特征输入图注意力网络，进行全局空间特征聚合，得到节点的全局时空特征；利用门控循环单元网络对全局时空特征进行序列建模，捕捉长期依赖关系，输出动态飞行行为特征；将动态飞行行为特征与预定义的威胁模式库进行相似度计算，得到威胁相似度得分；基于威胁相似度得分和预设阈值，判断低空飞行器是否构成潜在威胁。
S6：当检测到威胁时，启动定向天线阵列，向目标区域发射与低空飞行器相同频率的仿真控制信号，对低空飞行器进行安全引导或接管控制。
需要说明的是，目标区域是指低空飞行器所在区域。

实施例2，参照图1～图4，为本发明第二个实施例，该实施例提供了一种机场低空飞行器监视方法，为了验证本发明的有益效果，通过经济效益计算和仿真实验进行科学论证。
在机场周边5公里范围内布置了8个软件定义无线电接收器，形成一个均匀分布的监测网络。每个接收器配备了4元天线阵列，可编程射频前端的频率范围设置为400MHz-6GHz，采样率为20MHz，可调增益范围为0-60dB。
实验过程中模拟了3种不同类型的低空飞行器：一架民用四旋翼无人机、一架固定翼无人机和一架微型飞艇。这些飞行器在机场周边1-3公里的空域内以不同轨迹和速度飞行，高度在50-300米之间变化。飞行持续时间为30分钟。
首先，软件定义无线电接收器成功捕获了这些飞行器发射的各类无线电信号。通过多级小波变换降噪和时频分析，得到了清晰的时频图谱。信号分类模块正确识别出了遥控信号(2.4GHz)、数传信号(915MHz)和图传信号(5.8GHz)。
随后，多源融合定位算法基于不同接收器捕获的信号时间差和频率差，成功计算出了飞行器的三维坐标。定位精度在水平方向优于5米，垂直方向优于10米。预测轨迹与实际飞行轨迹的平均偏差不超过20米。
接下来，图神经网络构建的动态飞行行为模型对飞行器的行为进行了实时评估。在实验过程中，模拟了一次潜在威胁场景：固定翼无人机突然改变飞行方向，向跑道快速接近。系统在10秒内成功识别出这一异常行为，威胁相似度得分达到0.85(阈值为0.7)。
最后，系统启动定向天线阵列，向目标区域发射了与该无人机相同频率的仿真控制信号。信号功率逐步调整，最终在距离目标300米处成功接管了无人机的控制权，将其安全引导至指定区域降落。
整个实验过程中，系统的平均信号处理延迟为50毫秒，定位更新频率为10Hz，威胁评估周期为1秒。实验结果表明，该机场低空飞行器监视方法及系统能够有效识别、定位和应对潜在的低空飞行器威胁，为机场安全管理提供了有力支持。
