Description

一种低空飞行器的飞行环境识别监测方法及系统
技术领域
本发明涉及低空飞行器技术领域，具体涉及一种低空飞行器的飞行环境识别监测方法及系统。
背景技术
飞行器是利用无线电遥控设备和自备的程序控制装置操纵的不载人飞机，或者由车载计算机完全地或间歇地自主地操作。与有人驾驶飞机相比，低空飞行器往往更适合那些太“愚钝，肮脏或危险”的任务。低空飞行器按应用领域，可分为军用与民用。军用方面，低空飞行器分为侦察机和靶机。民用方面，低空飞行器+行业应用，是低空飞行器真正的刚需；目前在航拍、农业、植保、微型自拍、快递运输、灾难救援、观察野生动物、监控传染病、测绘、新闻报道、电力巡检、救灾、影视拍摄、制造浪漫等等领域的应用，大大的拓展了低空飞行器本身的用途，发达国家也在积极扩展行业应用与发展低空飞行器技术。
由于低空飞行器的飞行环境为低空飞行，在飞行过程中，会出现与鸟类或其他移动物体障碍物体出现碰撞的情况，同时飞行器容易受外界环境的影响导致无法正常飞行，特别是在天气信息、飞行路线区域信息与路障信息方面。
为此提出一种低空飞行器的飞行环境识别监测方法及系统。
发明内容
本发明的目的在于：为解决由于低空飞行器的飞行环境为低空飞行，在飞行过程中，会出现与鸟类或其他移动物体障碍物体出现碰撞的情况，同时飞行器容易受外界环境的影响导致无法正常飞行，特别是在天气信息、飞行路线区域信息与路障信息方面的问题，本发明提供了一种低空飞行器的飞行环境识别监测方法及系统。
本发明为了实现上述目的具体采用以下技术方案：
一种低空飞行器的飞行环境识别监测方法及系统，包括视觉识别模块、监控魔窟家、自动避让模块、主控制器模块、辅控制器模块、气压传感器模块、超声波传感器模块、温度传感器模块、风速传感器模块和湿度传感器模块，所述视觉识别模块，用于识别设定行驶路线上的移动物和障碍物并发送至主控制器模块；所述监控模块用于对设定行驶路线进行视频监控，获取监控视频，在接收到主控制器模块的调取信号后将监控视频发送至控制终端，所述自动避让模块用于避让视觉识别模块识别到的行驶路线上的移动物和障碍物，所述自动避让模块用于配合监控模块将飞行器避让至无障碍物区域，所述主控制器模块用于对移动物信息或障碍物信息进行分析，所述移动物信息包括移动物的高清图片；所述障碍物信息包括障碍物的高清图片，当移动物信息或障碍物信息位于飞行器设定飞行路线时，获取移动物信息或障碍物信息其他方位的监控视频，并生成避障信号发送至自动避让模块，所述温度传感器模块、风速传感器模块和湿度传感器模块用于监测限定飞行高度内的实时天气信息，所述天气信息包括：降水、风力和温湿度，其监测结果会被发送到辅控制器模块中，所述气压传感器模块和超声波传感器模块用于监测飞行器的实时飞行高度，并将采集到的飞行高度信息传递到辅控制器模块中。
进一步地，所述自动避让模块的避让信息生成包括：在监控模块识别规定路线上的障碍物为非移动物体时，分析监控视频中在障碍物侧方向空间距离，并生成避让信号进行避让：当监控模块识别规定路线上的障碍物为移动物体时，辅控制器模块分析并获取移动物的移动方向，当移动物的方向与飞行器的行驶方向相同时，生成避让信号进行避让，当移动物与飞行器相对行驶时，获取移动物到达障碍物前方设定距离的时间T，然后获取飞行器的避障时间t，当T大于等于t时，生成一次避让信号，当T小于t时，生成减速信号至辅控制器模块。
进一步地，所述风速传感器模块判定天气信息为刮风天时，将风力信息的风力级别信息与风向信息提取并发送至辅控制器模块，当风力级别大于预设级别时，即生成大风预警信息，此时大风预警信息的内容为风力级别与风向。
进一步地，所述主控制器模块和辅控制器模块会将接收到的信息转化为控制指令并将控制指令发送至工作人员的智能移动终端。
所述飞行路线环境识别监测信息如下：
步骤一：定位已知飞行高度以及飞行路线，监控模块模块驱动飞行器搭载的高清摄像头捕捉飞行器飞行路线上的移动物或障碍物图像信息；
步骤二：当移动物或障碍物处于飞行器的规划飞行路线时，移动物或障碍物的图像信息发送至主控器模块，生成避让信号发送至自动避让模块；
步骤三：自动避让模块驱动飞行器对移动物或障碍物进行避让处理，避让完毕后，监控模块判定移动物或障碍物后方飞行路线正常，飞行器复位规划的行驶路线；
所述飞行状态环境识别监测信息如下：
步骤一：定位已知飞行高度以及飞行路线，辅控制器模块接收气压传感器模块、超声波传感器模块、温度传感器模块、风速传感器模块和湿度传感器模块接收的各组检测信息数据，并进行合并整理；
步骤二：整理合并后的信息数据进行对比分析，判断飞行器飞行区域的环境状态是否处于强风强雨的恶劣条件中；
步骤三：判断结果为否时，飞行器按规划路线正常飞行：判断结果为处于恶劣天气中时，则辅控制器模块发送信息至驱动模块，调整飞行器的飞行姿态或重新规划飞行路线。
本发明的有益效果如下：
1、本发明通过设置了视觉识别模块配合监控模块以及多种用于信息采集的传感器，从而实现对飞行状态下飞行信息的多种监测，有效的为飞行器的飞行环境提供了强有力的飞行保护，能够通过监测飞行区域的环境状况，避免飞行器受外界环境的影响出现飞行事故，有效的保证飞行器的飞行安全，满足用户的使用需求，值得推广使用。
2、本发明通过设置视觉识别模块和监控模块的使用，有效的保证了飞行器在飞行路途中，对突然出现的鸟类或其他移动物体能够进行紧急的避让，避免出现飞行器与鸟类出现碰撞导致的飞行事故的发生，大大提高了飞行器的飞行安全性。
