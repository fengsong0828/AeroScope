Description

一种低空飞行器空中交通管理方法及系统
技术领域
本发明属于低空飞行器技术领域，特别涉及低空飞行器空中交通管理方法及系统。
背景技术
随着低空飞行器(如飞行汽车等)的不断发展和普及，越来越多的低空飞行器开始出现在城市和人口密集区域。然而，低空飞行器的数量的增加也带来了空中交通管理的挑战，包括碰撞风险、空中拥堵等问题。传统的空中交通管理系统难以应对复杂的低空飞行器交通状况。
现有的一些低空飞行器空中交通管理系统尝试利用卫星导航和无线通信技术，以实现位置定位和通信，但在实际应用中仍然存在一些问题。例如，由于通信延迟和数据处理能力的限制，这些系统可能无法及时检测到飞行器之间的碰撞风险，从而导致安全隐患。此外，传统的空中交通管理系统通常依赖于地面基础设施，可能受到地形限制，难以适应复杂多变的低空飞行环境。
发明内容
针对上述问题，本发明提供一种低空飞行器空中交通管理方法，所述方法包括：
收集飞行器的飞行数据；
以飞行数据为依据规划最优航线并将最优航线发送至飞行器；
飞行器按照对应的最优航线航行，航行中实时检测是否存在碰撞风险；
存在碰撞风险时，规划避让航线并让飞行器按照避让航线航行。
优选地，所述飞行数据包括自身数据和环境数据；
其中，所述自身数据包括飞行器的位置、速度、姿态、高度和航行意图；
环境数据为飞行器周围环境的感知数据；
所述自身数据通过5G通信和低轨道卫星通信传输。
优选地，所述以飞行数据为依据规划最优航线之前还需要递交航线申请并判断申请是否通过，在航线申请通过时，再执行最优航线的规划。
优选地，所述以飞行数据为依据规划最优航线，包括：
接收飞行器的飞行数据；
将飞行数据与系统预设的航线规则和空中交通流量情况相结合，规划飞行器的航线和速度。
优选地，所述飞行器按照对应的最优航线航行时，向飞行器发布空中交通限制区的警告和限制信息。
优选地，所述方法还包括：
实时记录和存储飞行器的飞行数据。
本发明还提出一种低空飞行器空中交通管理系统，所述系统包括收集模块、规划模块、检测模块和避让模块；
收集模块用于收集飞行器的飞行数据；
规划模块用于以飞行数据为依据规划最优航线并将最优航线发送至飞行器；
检测模块用于实时检测航行中是否存在碰撞风险；
避让模块用于规划避让航线并让飞行器按照避让航线航行
优选地，所述系统还包括记录存储模块；
记录存储模块用于实时记录和存储飞行器的飞行数据。
本发明还提出一种电子设备，包括：
处理器以及存储器；
所述处理器调用所述存储器存储的计算机程序，以执行根据上述任意一项所述的低空飞行器空中交通管理方法。
本发明还提出一种计算机可读存储介质，
所述计算机可读存储介质中存储有计算机程序，当所述计算机程序被处理器执行时，使得所述处理器能够执行上述任一项所述的低空飞行器空中交通管理方法。
本发明具有以下有益效果：
(1)本发明利用5G通信技术和低轨道卫星通信技术，实现低空飞行器之间的高效通信，实时传输位置和速度等数据，有效避免碰撞风险，提高低空飞行器的空中交通安全性；
(2)本发明中的飞行汽车配备了自动航线申请和规划功能，减轻驾驶员的负担，使飞行汽车的操作更加简单和智能化；
(3)本发明采用低轨道卫星通信技术，实现对低空飞行器的全球范围内的空中交通管理，适用于复杂多变的飞行环境，飞行器在全球各地都能够稳定通信，不受地形限制；
(4)本发明适用于各类低空飞行器，包括飞行汽车、个人无人机等，为未来低空飞行器的发展和商业化应用提供了重要支持。

收集模块10用于收集飞行器的飞行数据；

具体地，低空飞行器空中交通管理系统的工作流程如下：
1.数据收集和传输阶段：
低空飞行器配备5G通信模块和低轨道卫星通信设备，实时收集飞行器的位置、速度、高度、航行意图等飞行数据。
飞行器通过5G通信模块与其他飞行器和地面调度系统进行实时数据通信，同时通过低轨道卫星通信设备与地面通信基站保持稳定通信。
2.数据处理和航线规划阶段：
飞行器将实时采集的数据传输到云端调度系统，云端数据处理单元利用高性能处理器和智能算法对数据进行实时处理和分析。
航线规划和调度模块根据飞行器的位置、目的地、预定航线和交通流量等信息，智能规划最优航线。
3.碰撞避免和预警阶段：
云端调度系统中的碰撞避免和预警系统负责监测飞行器之间的距离、相对速度等信息，并预测潜在的碰撞风险。
如果系统检测到潜在碰撞风险，将立即发出预警信号，通知相关飞行器进行航线调整或速度变更，避免碰撞事故的发生。
4.空中交通限制阶段：
云端调度系统中的空中交通限制模块监测特定空域的交通流量，并发布相应的空中交通限制区。
飞行器通过5G通信模块或低轨道卫星通信设备接收空中交通限制区的信息，并根据指示遵守相应的飞行规定。
5.自动化控制阶段：
飞行器配备自动化飞行控制系统，根据云端调度系统的航线规划和碰撞避免预警的指示，自主进行航线调整和避让操作。
驾驶员在需要时可以进行手动干预，但系统优先考虑自动化航行，以确保飞行的高效和安全。
6.数据记录和分析阶段：
云端调度系统对飞行器的飞行数据进行实时记录和存储，包括航线信息、碰撞预警记录等。
飞行数据用于日志记录、事故调查和交通流量分析等方面，以进一步提高低空飞行器空中交通管理的安全性和效率。
