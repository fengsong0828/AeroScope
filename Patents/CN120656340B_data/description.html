Description

适用于低空多飞行器混合起降场的立体化调度方法及系统
技术领域
本发明涉及飞行器调度，具体涉及适用于低空多飞行器混合起降场的立体化调度方法及系统。
背景技术
随着低空空域逐步开放及无人机、通用航空器在应急救援、物流配送、城市巡检等领域的规模化应用，低空多飞行器混合起降场景中的空域资源竞争与冲突风险日益突出。传统空域调度方法主要依赖二维平面划分与静态规则管理，存在以下技术局限：
1）三维空域利用不足：现有技术多采用二维电子围栏或固定航路，未充分利用垂直维度空间资源，导致起降场周边空域利用率低，难以适应低空复杂地形与动态障碍环境；
2）动态调度能力薄弱：缺乏对三维网格单元的实时状态感知与动态划分机制，无法根据空域占用、气象变化等因素实时调整网格通行权限，易引发局部拥堵或资源浪费；
3）冲突预测与协同避让效率低：低空飞行器轨迹受风场、建筑物扰动等影响，不确定性高，传统几何冲突检测方法难以准确预测四维时空（三维空间+时间）冲突风险，且多飞行器协同避让路径规划易陷入局部最优。
现有技术未有效融合北斗网格码的三维编码优势与量子优化的全局搜索能力，无法实现低空空域的精细化建模、动态通行权限分配和高效冲突消解。因此，亟需一种适用于低空多飞行器混合起降场的立体化调度方法及系统，通过三维空域剖分、四维时空冲突预测及改进量子近似优化算法，提升低空多飞行器混合起降场景的安全性与运行效率。
发明内容
针对现有技术所存在的上述缺点，本发明提供了适用于低空多飞行器混合起降场的立体化调度方法及系统，能够有效克服现有技术所存在的难以对多飞行器避让路径进行有效规划，无法对多飞行器冲突进行高效消解的缺陷。
为实现以上目的，本发明通过以下技术方案予以实现：
适用于低空多飞行器混合起降场的立体化调度方法，包括以下步骤：
S1、基于北斗网格码对空域进行三维立体剖分，并对三维网格单元进行空域状态实时映射；
S2、根据网格状态对三维网格单元进行划分；
S3、根据飞行器的任务类型设定优先级权重，结合网格状态实时生成通行权限；
S4、将时间维度引入北斗网格码，将空域剖分为多个四维时空网格单元，并对所有四维时空网格单元进行冲突风险预测；
S5、根据冲突风险预测结果，采用改进量子近似优化算法为存在冲突风险的飞行器生成避让路径；
其中，在改进量子近似优化算法中，构造横向场哈密顿量时，一方面根据多飞行器冲突避让问题的特性调整权重，对存在冲突风险的四维时空网格单元增加横向场强度；另一方面，引入双量子比特混合项，增强量子态间的纠缠，提升全局搜索能力，避免算法陷入局部最优。
