Description

一种基于穿戴式低空飞行器的悬浮动态穿戴方法
技术领域
本发明属于多旋翼航空飞行器技术领域，特别是涉及一种基于穿戴式低空飞行器的悬浮动态穿戴方法。
背景技术
随着低空经济的兴起，涌现出了众多面向载人用途的低空飞行器。其中，存在一类载人用途的穿戴式低空载人飞行器。
相比开放式座舱和封闭式座舱的低空载人飞行器，穿戴式低空载人飞行器需要乘客将飞行器与身体通过某些装置进行绑定，再开展载人飞行。这种飞行器可看作衣服或者穿戴式装备，因此称为穿戴式载人飞行器。穿戴式载人飞行器具有灵活穿戴，快速机动的特点。
穿戴式飞行器普遍需要飞行器在未启动状态下，完成乘客对其的穿戴。然而，由于不同的乘客身高可能相差悬殊，固定高度的穿戴式飞行器，难以在穿戴环节适应不同的乘客身高，为了适应不同乘客的身高，穿戴式飞行器需要设计高度调节机构，这就增加了结构复杂度，也增加了整机质量，进而影响续航时间。不论穿戴式飞行器在未启动时是固定高度还是可调高度，都会影响穿戴式飞行器的易用性和灵活性，降低使用体验。同时，穿戴式载人飞行器普遍较重，穿戴过程对乘客的体力也有一定要求。
发明内容
本发明目的在于提供一种基于穿戴式低空飞行器的悬浮动态穿戴方法，提高了飞行器的穿戴适应性，降低了对操作员体能的要求。
为了实现本发明目的，本发明提供了一种基于穿戴式低空飞行器的悬浮动态穿戴方法，包括
所述飞行器设有一键起降开关，用于完成飞行器的一键起飞、一键降落；
所述飞行器设有操纵手柄，且所述操纵手柄设置回弹式按钮用于牵拉随动模式与定点飞行模式之间的切换；
所述飞行器设有飞控模块，用以稳定控制单人飞行器飞行；
所述飞控模块内嵌牵拉随动模式：当所述飞行器检测到外力牵拉，所述飞行器进入0速度控制模式，此时所述飞行器随外力牵引而改变悬停位置，进入牵拉随动模式；
所述悬浮动态穿戴方法基于牵拉随动模式包括在以下过程中的穿戴：
起飞过程穿戴：当所述飞行器位于地面静止状态，具备定点起飞的条件时，进入起飞过程穿戴；
降落或复飞过程穿戴：所述飞行器的操作员在空中执行飞行任务，当要进行降落时所述飞行器进入降落或复飞过程穿戴。
所述飞行器的牵拉随动模式内部指令具体包括以下步骤：
步骤1、所述飞控模块自行判断所述飞行器进入定点模式；
步骤2、所述飞控模块判断牵拉随动模式开关是否关闭；当牵拉随动模式开关关闭则退出牵拉随动模式，转到步骤六；当牵拉随动模式开关打开则继续牵拉随动模式，转到步骤三；
步骤3、所述飞控模块实时检测当前是否存在大于等于阈值的外部拉力；当所述外部拉力大于等于阈值时，则转到步骤四；当所述外部拉力小于阈值时，转到步骤一；
步骤4、当所述飞行器进入0速度模式，在所述0速度模式下飞行器在三维空间中默认期望保持0速度飞行，但通过操纵杆能够改变其状态，此时外力牵引所述飞行器，所述飞行器能够随动，转到步骤五；
步骤5、所述飞控模块实时检测外部拉力是否小于阈值；当检测到外部拉力小于阈值时，转到步骤一；当检测到外部拉力大于等于阈值时，转到步骤四；
步骤6、退出牵拉随动模式。
所述起飞过程穿戴包括以下步骤：
步骤1、所述飞行器具备一键起飞功能，当所述飞行器以定点飞行模式起飞时，转到步骤二；
步骤2、所述飞行器从空中飞至距离地面指定高度时，转到步骤三；
步骤3、所述飞行器进入所述牵拉随动模式；此时操作员根据自身身高牵拉飞行器使其适合自身身高，并完成飞行器穿戴，转到步骤四；
步骤4、所述操作员操作退出所述牵拉随动模式，并推动油门开始飞行任务，同时起飞过程穿戴结束。
所述降落或复飞过程穿戴包括以下步骤：
步骤1、所述飞行器降落至距离地面小于米的高度；
步骤2、操作员降落至双脚着地站稳后，切换到所述牵拉随动模式，完成飞行操作员与飞行器的脱离；
步骤3、所述飞行器进入所述牵拉随动模式；
步骤4、所述操作员根据是否降落选择下一个步骤，当选择降落时，则转到步骤五；当选择复飞时，转到步骤六；
步骤5、所述操作员与飞行器分开后，控制飞行器降落，则降落过程穿戴结束；
步骤6、所述操作员根据自身身高牵拉飞行器使其适合自身身高，并完成复飞过程穿戴；
步骤7、所述操作员操作退出所述牵拉随动模式，推动油门开始飞行任务，则复飞过程穿戴结束。
本发明还包括一种基于穿戴式低空飞行器的悬浮动态穿戴方法的飞行器操作方法，包括以下步骤：
在起飞阶段，设置所述飞行器起飞至距离地面1米高度，随后所述飞行器进入所述牵拉随动模式，所述操作员拉扯所述飞行器适应自身身高，并完成穿戴操作，后按下手柄上的回弹按钮以换到定点模式，完成飞行任务；
在降落阶段，所述操作员在所述飞行器降落至1.5米高度，并双脚着地后，松开操纵手柄同时也松开回弹式按钮，所述飞行器进入所述牵拉随动模式，所述操作员解除与所述飞行器的连接，结束飞行任务或更换另一名操作员重复穿戴操作，并使用操纵手柄继续执行飞行任务，在当前操作员解除与飞行器连接后按下一键起降开关完成飞行器降落。
与现有技术相比，本发明的显著进步在于：本发明所提出的动态穿戴方法可以适应不同身高人员的穿戴要求。飞行操作人员可在飞行器起飞后手动调节飞行器的高度适合自身身高，快速完成对飞行器的穿戴动作。
