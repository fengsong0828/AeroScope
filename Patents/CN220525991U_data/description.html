Description

一种用于低空飞行器探测的隐蔽式防务雷达
技术领域
本实用新型涉及雷达技术领域，具体为一种用于低空飞行器探测的隐蔽式防务雷达。
背景技术
现代飞行器的发展，得益于19世纪工业革命带来的科学和技术的巨大飞跃，飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。随着飞行器普及，个人飞行器必然是未来发展方向，轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向，对于这类飞机主要使用雷达进行检测，进而截获无线遥控信号，可见光，干扰GPS
防务雷达根据实际需要，设置雷达系统的参数，通过选择合适的扫描模式，例如机械扫描或电子扫描，开始扫描空域以探测目标，分析雷达返回的信号，对目标进行识别和分类，使用目标追踪功能跟踪目标的位置、速度和轨迹，可以使用自动跟踪功能来持续追踪目标，根据雷达提供的情报，作出相应的决策和应对措施。这可能涉及到指挥官的判断、任务分配、目标打击，但是在对飞行器进行检测环境可能会比较差，地面不平整，可能会使雷达发生偏移，进而可能导致雷达探测区域出现误差，为此，我们提出一种用于低空飞行器探测的隐蔽式防务雷达。
实用新型内容
本实用新型的目的在于提供一种用于低空飞行器探测的隐蔽式防务雷达，以解决上述背景技术中提出的问题。
为实现上述目的，本实用新型提供如下技术方案：一种用于低空飞行器探测的隐蔽式防务雷达，包括：支撑柱、支撑板和雷达板，
支撑板固定连接在支撑柱顶端表面，支撑柱用于固定支撑板，雷达板设在支撑板两侧，雷达板用于检测飞行器；
支撑柱侧面设有若干个固定组件，固定组件用于固定支撑柱，固定组件和支撑柱之间设有滑动组件，滑动组件用于安放固定组件，支撑板和雷达板之间设有收纳组件，收纳组件用于收纳雷达板。
优选地，固定组件包括固定板和调节件，固定板设在支撑柱侧面，且固定板呈L形，调节件设在固定板上，调节件用于调节支撑柱。
优选地，调节件包括丝杠、把手、调节板和弹簧，丝杠与固定板一端螺纹连接，丝杠用于调节固定板高度，把手固定连接在丝杠一端，调节板转动连接在丝杠的另一端，弹簧套设在丝杠外壁上，且弹簧设在固定板和调节板之间，弹簧用于支撑固定板。
优选地，滑动组件包括滑块和滑槽，滑块固定连接在固定板侧面，滑槽开设在支撑柱侧面上，且滑块与滑槽滑动连接，滑槽和滑块用于取出固定板和安放固定板。
优选地，收纳组件包括侧槽、转动轴、转动板、转动件和收缩件，侧槽开设在支撑板上，转动轴两端固定连接在侧槽内壁上，转动板与转动轴转动连接，转动件设在支撑板侧面，收缩件与转动件相连接，收缩件用于收缩雷达板。
优选地，转动件包括矩形槽和转轴，矩形槽开设在支撑板两侧，转轴两端固定连接在矩形槽内壁上，转轴用于转动收缩件。
优选地，收缩件包括转板、气缸和矩形板，转板与转轴转动连接，气缸一端与转板固定连接，矩形板一侧面固定连接在气缸另一端，矩形板另一侧与雷达板固定连接。
本实用新型至少具备以下有益效果：通过设置丝杠与固定板一端螺纹连接，丝杠用于调节固定板高度，把手固定连接在丝杠一端，调节板转动连接在丝杠的另一端，弹簧套设在丝杠外壁上，且弹簧设在固定板和调节板之间，弹簧用于支撑固定板，根据环境不同，转动把手，进而带动丝杠转动，使固定板在丝杠上竖向移动，调节固定板高度，进而调节雷达板，使雷达板保持平行，防止雷达检测结果出现误差。

实施例
一种用于低空飞行器探测的隐蔽式防务雷达，包括：
支撑柱1、支撑板3和雷达板2，
支撑板3固定连接在支撑柱1顶端表面，支撑柱1用于固定支撑板3，雷达板2设在支撑板3两侧，雷达板2用于检测飞行器；
支撑柱1侧面设有若干个固定组件4，固定组件4用于固定支撑柱1，根据不同环境地面调节雷达板2，使雷达板2保持水平状态，防止雷达检测出现误差，固定组件4和支撑柱1之间设有滑动组件，滑动组件用于安放固定组件4，支撑板3和雷达板2之间设有收纳组件5，收纳组件5用于收纳雷达板2。
固定组件4包括固定板41和调节件，固定板41设在支撑柱1侧面，且固定板41呈L形，调节件设在固定板41上，调节件用于调节支撑柱1，使固定板41能够固定雷达板2。
调节件包括丝杠43、把手42、调节板45和弹簧44，丝杠43与固定板41一端螺纹连接，丝杠43用于调节固定板41高度，把手42固定连接在丝杠43一端，调节板45转动连接在丝杠43的另一端，弹簧44套设在丝杠43外壁上，且弹簧44设在固定板41和调节板45之间，弹簧44用于支撑固定板41，根据环境不同，转动把手42，进而带动丝杠43转动，使固定板41在丝杠43上竖向移动，调节固定板41高度，进而调节雷达板2，使雷达板2保持平行，防止雷达检测结果出现误差。
滑动组件包括滑块10和滑槽9，滑块10固定连接在固定板41侧面，滑槽9开设在支撑柱1侧面上，且滑块10与滑槽9滑动连接，滑槽9和滑块10用于取出固定板41和安放固定板41，通过设置滑槽9和滑块10，方便对固定板41进行拆卸，方便对其进行收纳。
收纳组件5包括侧槽6、转动轴8、转动板7、转动件和收缩件，侧槽6开设在支撑板3上，转动轴8两端固定连接在侧槽6内壁上，转动板7与转动轴8转动连接，转动件设在支撑板3侧面，收缩件与转动件相连接，收缩件用于收缩雷达板2，雷达板2通过转动板7在转动轴8上进行转动，进而可以对雷达板2进行收纳，方便运行。
转动件包括矩形槽和转轴53，矩形槽开设在支撑板3两侧，转轴53两端固定连接在矩形槽内壁上，转轴53用于转动收缩件，通过设置转轴53，可以对雷达板2进行收纳。
收缩件包括转板54、气缸55和矩形板51，转板54与转轴53转动连接，气缸55一端与转板54固定连接，矩形板51一侧面固定连接在气缸55另一端，矩形板51另一侧与雷达板2固定连接，气缸55收缩转动可以对雷达板2进行收纳，进而雷达板2通过转动板7在转动轴8上进行转动，进而可以对雷达板2进行收纳，方便运行。
