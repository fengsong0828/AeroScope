Description

一种用于低空飞行器快速充电的液冷超级充电桩
技术领域
本发明涉及充电桩技术领域，具体为一种用于低空飞行器快速充电的液冷超级充电桩。
背景技术
随着低空飞行器市场的不断扩大，充电桩的需求量也将持续增长，而充电桩是接入市电或其他电源，以提供充电过程中所需的电能，在充电桩内部，电力电子器件如整流器、滤波器等发挥着关键作用，对于交流充电桩，这些器件将输入的交流电转换为直流电，因为低空飞行器的电池系统通常需要使用直流电进行充电，低空飞行器通过充电接口与充电桩相连，建立电能传输的通道；缺陷：然而，市面上的充电桩的散热效果差，大大降低了用户的体验；为了解决上述缺陷一，现有技术一(申请号：202021958406.8，授权公告日为2021-04-13的中国专利)公开了一种具备能源控制功能的充电桩，能够根据充电车辆的数量来控制输出功率，同时各个器件间隔设置，使得散热效果好，大大提升了用户的体验；缺陷二：在拉簧拉力作用下，拉簧拉动牵拉绳从拉绳卷绕辊上展开，牵拉绳带动拉绳卷绕辊、旋转轴和电缆卷绕辊反方向旋转，电缆卷绕辊将充电电缆卷绕在电缆卷绕辊上，实现充电电缆的自动收卷，但该装置中的收线拉簧在长时间的使用后，拉簧的拉力会受到减小，使用一段时间后充电电缆会无法正常收回，为了解决缺陷二，现有技术二(申请号：202320795717.4，授权公告日为2023-08-29的中国专利)公开了自动收线式充电桩，当充电完成之后，将充电头从车上拔出，充电头不在给电源线施加拉力，位于充电桩本体内部的重物向下运动，利用重物向下运动带动滚筒转动，滚筒转动能够对电源线进行收线，由重物向下运动带动滚筒转动对电源线进行收线；转动第一把手，第一把手能够带动螺纹杆向下运动，利用螺纹杆向下移动能够带动移动轮线下运动，使移动轮与地面接触，通过推动充电桩本体带动移动轮转动。
现有技术是通过利用重物向下运动带动滚筒转动，能够对电源线进行收线，但用户每次拉出一截电源线时，无法对其进行固定，使得用户一旦脱手，电源线就会被收回，导致使用起来不够方便，同时由于充电桩大多是放置在户外使用的，使得散热网的表面会附着大量的灰尘和杂质，继而堵塞住散热网，导致充电桩内部的热量无法排出，并影响充电桩的散热，所以我们提出了一种用于低空飞行器快速充电的液冷超级充电桩，以便于解决上述中提出的问题。
发明内容
本发明的目的在于提供一种用于低空飞行器快速充电的液冷超级充电桩，以解决上述背景技术提出的用户每次拉出一截电源线时，无法对其进行固定，使得用户一旦脱手，电源线就会被收回，导致使用起来不够方便，同时由于充电桩大多是放置在户外使用的，使得散热网的表面会附着大量的灰尘和杂质，继而堵塞住散热网，导致充电桩内部的热量无法排出，并影响充电桩的散热的问题。
为实现上述目的，本发明提供如下技术方案：一种用于低空飞行器快速充电的液冷超级充电桩，包括壳体，所述壳体的正面设置有控制面板，且壳体左右两侧的上方均插合连接有充电枪，所述壳体的两侧均固定连接有收线壳，所述壳体内部的左右两侧对称安装有马达；所述收线壳的内部连接有转动杆，且转动杆的外侧套设有收卷筒，所述收卷筒与转动杆之间安装有棘轮组件，且收卷筒的两端固定有限位盘；所述壳体两侧的下方均开设固定有散热网格，所述壳体两侧位于散热网格的上方均设置有从动齿轮，且壳体两侧位于从动齿轮的下方均连接有清洁刷，且清洁刷与从动齿轮之间通过连接板相连接，所述转动杆一端的外侧均套设固定有主动齿轮，所述壳体的右侧设置有循环管，所述壳体的顶部固定有储液箱。
优选的，两个所述转动杆的相对端均伸入壳体的内部，且转动杆的一端马达的输出端固定连接，所述转动杆与壳体和收线壳之间为轴承连接。
通过采用上述结构设置，使得马达驱动转动杆和收卷筒转动并对电缆进行自动收卷。
优选的，所述充电枪上的电缆缠绕在收卷筒的外侧，所述收卷筒通过棘轮组件构成单向转动结构。
通过采用上述结构设置，使得棘轮组件只能让收卷筒正向旋转。
优选的，两个所述收线壳的相反面均转动连接有活动壳，且活动壳顶部的一侧均转动连接有毛刷辊，且两个毛刷辊的外侧与充电枪上电缆的表面相接触。
通过采用上述结构设置，使得电缆在被收卷时，能够利用毛刷辊将其表面的灰尘和杂质给清理掉。
优选的，所述从动齿轮通过轴转动连接在壳体的侧面，且从动齿轮的外侧伸入收线壳的内部，并且从动齿轮与主动齿轮之间为啮合连接，所述连接板的两端分别与从动齿轮的侧面和清洁刷的顶部之间均为铰接连接，所述清洁刷滑动设置在壳体的侧面，且清洁刷的毛刷位置与散热网格的表面相接触。
通过采用上述结构设置，使得连接板带动清洁刷对散热网格进行清理，防止其堵塞而影响壳体的散热效果。
优选的，所述壳体两侧位于充电枪的上方位置固定连接有防雨罩，且防雨罩呈半圆弧形状设置。
通过采用上述结构设置，能够对充电枪和壳体上的插座进行挡雨，防止发生电气短路。
优选的，所述壳体内部的后端安装有冷却管，且壳体内壁底部的右侧螺栓连接有循环泵。
通过采用上述结构设置，能够让冷却管内的冷却液循环使用。
优选的，所述冷却管的两端分别与储液箱的底部和循环泵的左端固定连接，所述循环管的两端分别与储液箱的右侧和循环泵的右端固定连接，且储液箱、冷却管、循环泵和循环管内部互相连通，所述冷却管和循环管上端的内部均安装有单向阀。
通过采用上述结构设置，能够防止液体发生乱流。
优选的，所述壳体右侧的上方固定有排气通道，且排气通道的内部设置有传动杆，所述传动杆的右端固定连接有风扇，所述排气通道内部的右侧螺栓安装有防尘网，所述循环管的内部通过轴转动连接有叶轮，所述叶轮的轴端与传动杆的左端之间通过锥齿轮组传动连接，且锥齿轮组的外侧安装有保护壳，所述储液箱的顶部固定连接有防护罩，且防护罩的左侧面等间距开设有进气孔，所述排气通道与防护罩之间固定连接有排气管，所述储液箱内壁的顶部等间距贯穿固定有散热片。
通过采用上述结构设置，能够对使用完的冷却液进行快速降温，保证下次冷却使用。
优选的，多个所述散热片的顶部均伸入防护罩的内部，所述传动杆的左端伸入循环管的内部，所述传动杆与排气通道和循环管之间为轴承连接，所述排气通道与防护罩之间通过排气管相连通。
通过采用上述结构设置，使得防护罩内部的热量能够从排气管和排气通道排出。
与现有技术相比，本发明的有益效果是：该用于低空飞行器快速充电的液冷超级充电桩，采用新型结构设计，其具体内容如下所示：(1)当用户拿取充电枪并给低空飞行器充电时，此时充电枪上电缆被拉出一截，使得收卷筒转动，并利用棘轮组件的作用下对收卷筒自动限位，防止拉出的电缆收回，而后当充电完成后，用户将充电枪放回原处；进一步，通过马达带动转动杆反向转动，使得转动杆利用棘轮组件带动收卷筒反向旋转，并对拉出的电缆进行自动收卷，从而无需用户手动整理电缆，提高了充电的便捷性，且防止电缆被踩踏、碾压或拉扯造成的损伤，延长了电缆的使用寿命。
(2)当马达带动转动杆反向旋转的同时，主动齿轮带动相啮合的从动齿轮转动，使得连接板带动清洁刷上下往复移动，并对散热网格进行清理，从而避免散热网格被灰尘堵塞而影响壳体内部热量的排出。
(3)储液箱内部的冷却液进入到冷却管内，能够吸收电池在充电过程中产生的热量，确保电池和充电系统的温度始终控制在安全且高效的范围内，同时吸收热量的冷却，通过循环泵和循环管重新回到储液箱内部，而后冷却液温度降低后再次回到冷却管中进行冷却，从而实现循环液冷，提高冷却效率。
(4)冷却液中的热量会传递到多个散热片上，然后外界的空气从左侧的多个进气孔进入到防护罩内部，并流过多个散热片时，热量会从散热片传递到空气中，从而实现散热，且利用冷却液流动的势能驱动叶轮转动，使得叶轮利用锥齿轮组带动传动杆进行转动，同时传动杆带动风扇旋转，并配合排气管能够将防护罩内部的热量通过排气通道排出，进而保证防护罩内部的空气流通，并提高对冷却液的散热效果。

下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。

实施例一：为了解决现有技术中用户每次拉出一截电源线时，无法对其进行固定，使得用户一旦脱手，电源线就会被收回，导致使用起来不够方便，同时由于充电桩大多是放置在户外使用的，使得散热网的表面会附着大量的灰尘和杂质，继而堵塞住散热网，导致充电桩内部的热量无法排出，并影响充电桩的散热的问题，因此公开了如下方案，具体参考图1-图3、图5、图7、和图8所示，包括壳体1，壳体1的正面设置有控制面板，且壳体1左右两侧的上方均插合连接有充电枪2，壳体1的两侧均固定连接有收线壳7，壳体1内部的左右两侧对称安装有马达；收线壳7的内部连接有转动杆9，两个转动杆9的相对端均伸入壳体1的内部，且转动杆9的一端马达的输出端固定连接，充电枪2上的电缆缠绕在收卷筒10的外侧，收卷筒10通过棘轮组件14构成单向转动结构，转动杆9与壳体1和收线壳7之间为轴承连接，且转动杆9的外侧套设有收卷筒10，收卷筒10与转动杆9之间安装有棘轮组件14，且收卷筒10的两端固定有限位盘；壳体1两侧的下方均开设固定有散热网格24，壳体1两侧位于散热网格24的上方均设置有从动齿轮12，且壳体1两侧位于从动齿轮12的下方均连接有清洁刷22，且清洁刷22与从动齿轮12之间通过连接板23相连接，从动齿轮12通过轴转动连接在壳体1的侧面，且从动齿轮12的外侧伸入收线壳7的内部，并且从动齿轮12与主动齿轮11之间为啮合连接，连接板23的两端分别与从动齿轮12的侧面和清洁刷22的顶部之间均为铰接连接，清洁刷22滑动设置在壳体1的侧面，且清洁刷22的毛刷位置与散热网格24的表面相接触，转动杆9一端的外侧均套设固定有主动齿轮11，两个收线壳7的相反面均转动连接有活动壳25，且活动壳25顶部的一侧均转动连接有毛刷辊13，且两个毛刷辊13的外侧与充电枪2上电缆的表面相接触，壳体1两侧位于充电枪2的上方位置固定连接有防雨罩3，且防雨罩3呈半圆弧形状设置。
在使用时，用户可通过壳体1正面的显示屏和操作按钮来选择充电模式、启动充电、设置参数等，之后用户将左侧或右侧的充电枪2从壳体1上的插座拔下，并将充电枪2插入到低空飞行器上的充电口内进行充电利用壳体1内部的充电模块，将输入的交流电转换为适合低空飞行器充电的直流电，同时充电枪2在被拔下并拉出一截电缆时，活动壳25随着充电枪2的移动方向转动，使得充电枪2上的电缆方便被拉出，此时收卷筒10转动，并利用棘轮组件14的作用下对收卷筒10自动限位，防止拉出的电缆自动收回，而后当充电完成后，用户将左侧或右侧的充电枪2插回到壳体1上的插座，并通过控制面板启动左侧或右侧的马达，马达带动转动杆9反向转动，使得转动杆9利用棘轮组件14带动收卷筒10反向旋转，并对拉出的电缆进行自动收卷，从而无需用户手动整理电缆，提高了充电的便捷性，且防止电缆被踩踏、碾压或拉扯造成的损伤，延长了电缆的使用寿命，同时在电缆被收卷时，利用设置的两个毛刷辊13能够自动对电缆表面的灰尘和杂质进行清理，进而减少电缆的磨损和老化，从而延长其使用寿命，然后当转动杆9反向旋转的同时，主动齿轮11转动并带动相啮合的从动齿轮12进行转动，继而使得连接板23带动清洁刷22上下往复移动，并对散热网格24进行清理，从而避免散热网格24被灰尘堵塞而影响壳体1内部热量的排出，且利用设置的防雨罩3能够对充电枪2进行挡雨，防止充电枪2和壳体1上的插座被雨水渗透而导致电气短路。
实施例二：与实施例一不同的是，本实施例利用冷却管5内部的冷却液能够对壳体1内部的电池模块进行液冷降温，保证充电桩的使用性能，具体参考图1和图4所示，壳体1的顶部固定有储液箱4，壳体1内部的后端安装有冷却管5，且壳体1内壁底部的右侧螺栓连接有循环泵6，壳体1的右侧设置有循环管8，冷却管5的两端分别与储液箱4的底部和循环泵6的左端固定连接，循环管8的两端分别与储液箱4的右侧和循环泵6的右端固定连接，且储液箱4、冷却管5、循环泵6和循环管8内部互相连通，冷却管5和循环管8上端的内部均安装有单向阀。
储液箱4内部的冷却液进入到冷却管5内，能够吸收电池在充电过程中产生的热量，确保电池和充电系统的温度始终控制在安全且高效的范围内，同时吸收热量的冷却，然后启动循环泵6，将冷却管5内部使用后的冷却液重新回到储液箱4内部，之后冷却液温度降低后再次回到冷却管5中进行冷却，从而实现循环液冷，提高冷却效率。
实施例三：与实施例二不同的是，本实施例利用冷却液流动的势能，能够驱动风扇旋转，并配合散热片18能够对使用后的冷却液进行快速降温，保证下次冷却使用，具体参考图1、图6和图9所示，壳体1右侧的上方固定有排气通道17，且排气通道17的内部设置有传动杆16，传动杆16的右端固定连接有风扇，排气通道17内部的右侧螺栓安装有防尘网19，循环管8的内部通过轴转动连接有叶轮15，叶轮15的轴端与传动杆16的左端之间通过锥齿轮组传动连接，且锥齿轮组的外侧安装有保护壳21，储液箱4的顶部固定连接有防护罩26，且防护罩26的左侧面等间距开设有进气孔，排气通道17与防护罩26之间固定连接有排气管20，储液箱4内壁的顶部等间距贯穿固定有散热片18，多个散热片18的顶部均伸入防护罩26的内部，传动杆16的左端伸入循环管8的内部，传动杆16与排气通道17和循环管8之间为轴承连接，排气通道17与防护罩26之间通过排气管20相连通。
冷却液在循环泵6的作用下被泵入储液箱4内部时，冷却液中的热量会传递到多个散热片18上，然后外界的空气从左侧的多个进气孔进入到防护罩26内部，并流经多个散热片18，散热片18热量会传递到空气中，通过加快防护罩26内部的空气流通，可以持续的将散热片18热量带走，这样可以间接的对冷却液实现散热，保证了冷却液的散热效果；
利用冷却液在循环管8内部流动的势能驱动叶轮15转动，使得叶轮15利用锥齿轮组带动传动杆16进行转动，同时传动杆16带动风扇旋转并配合排气管20能够将防护罩26内部冷却液吸收的热量通过排气通道17排出，进而保证防护罩26内部的空气流通，结构合理。
