Description

一种应用于低空飞行器的防撞装置
技术领域
本发明涉及飞行器防护技术领域，尤其涉及一种应用于低空飞行器的防撞装置。
背景技术
低空飞行器作为现代航空技术的重要组成部分，用于农业植保、物流配送、地形测绘、应急救援等领域，现有低空飞行器通常装备有防撞装置，主要关注于飞行器底部支架与地面接触时的缓冲保护，以减轻着陆冲击，然而，低空飞行器在有风天气进行着陆时，飞行器降落过程中会发生的侧向晃动，而飞行器的不稳定运动会导致位于飞行器周向的扇叶（如多旋翼无人机的螺旋桨）与停机坪地面碰撞，引发扇叶损伤乃至破坏，严重时会造成飞行器失去动力平衡，发生失控，不仅危及机载人员的生命安全，而且会缩短飞行器的使用寿命，增加维修成本，限制了飞行器的可靠性和任务执行的能力，因此，需要研发一种具有在有风天气下稳定降落保护飞行器扇叶的低空飞行器防撞装置。
发明内容
为了解决飞行器在有风天气下落地易发生晃动使扇叶损坏的问题，本发明提供了一种应用于低空飞行器的防撞装置。
具体方案为：一种应用于低空飞行器的防撞装置，包括有机体，所述机体的下侧转动连接有矩形阵列分布的转动壳，所述转动壳内滑动连接有滑动直杆，所述滑动直杆的下侧转动连接有转动支板，所述转动壳靠近所述机体的一侧固接有固定块，所述固定块设置有与对应所述转动壳连通的通孔，所述固定块内滑动连接有滑动板，所述滑动板设置有第一通孔和第二通孔，所述滑动板上的第一通孔和第二通孔均与对应所述固定块的通孔连通配合，所述固定块固接有与其通孔连通的储液筒，所述储液筒内滑动连接有活塞直杆，所述机体的下侧设置有用于控制矩形阵列分布所述滑动直杆移动速度的锁定机构，所述滑动直杆的侧壁上设置有用于检测其与对应所述转动壳相对移动距离的状态检测机构。
作为本发明的优选，所述滑动板内第一通孔设置为锥形，所述滑动板内第一通孔的最大直径小于所述滑动板内第二通孔的直径，用于限定对应所述转动壳内对应所述滑动直杆的移动速度。
作为本发明的优选，所述锁定机构包括有矩形阵列分布的固定连杆，矩形阵列分布的所述固定连杆均固接于所述机体的下侧，所述固定连杆的下侧固接有第一固定筒，所述第一固定筒内滑动连接有活塞支杆，所述第一固定筒与对应所述活塞支杆之间固接有拉簧，所述活塞支杆固接有挤压块，所述挤压块靠近对应所述转动壳的一侧设置有斜面，所述挤压块的斜面与对应所述转动壳挤压配合，所述机体的下侧固接有第二固定筒，所述第二固定筒的一侧连通有第一连通管，矩形阵列分布的所述第一固定筒均与所述第一连通管连通，所述第二固定筒内滑动连接有滑动壳，所述第二固定筒与所述滑动壳之间固接有弹簧，所述第二固定筒的侧壁上设置有用于控制矩形阵列分布所述滑动板移动的换位组件，所述储液筒的侧壁上设置有用于控制对应所述转动壳转动的辅助转动组件。
作为本发明的优选，所述换位组件包括有第二连通管，所述第二连通管连通于所述第二固定筒的另一侧，所述第二固定筒的侧壁上滑动连接有限位销，所述第二固定筒与所述限位销之间固接有弹簧，所述滑动壳的侧壁上设置有与所述限位销限位配合的盲孔，所述滑动壳内密封滑动连接有滑动片，所述滑动壳与所述滑动片之间固接有弹簧，所述转动壳的侧壁上固接有与所述第二连通管连通的第一液压缸，所述第一液压缸的伸缩端与对应所述滑动板固接。
作为本发明的优选，所述辅助转动组件包括有第二液压缸，所述第二液压缸固接于对应所述储液筒的侧壁上，所述第二液压缸的伸缩端与对应所述活塞直杆固接，所述滑动直杆与对应所述转动支板之间固接有弹簧，所述活塞直杆与对应所述储液筒之间固接有弹簧，所述第二液压缸连通有导液管，所述机体靠近对应所述转动壳的一侧固接有与所述导液管连通的弧形筒，所述弧形筒内密封滑动连接有与对应所述转动壳固接的弧形活塞杆，所述弧形筒与对应所述弧形活塞杆之间固接有拉簧。
作为本发明的优选，所述状态检测机构包括有固定套，所述固定套固接于对应所述转动壳的侧壁上，所述固定套内滑动连接有与对应所述滑动直杆挤压配合的滑动杆，所述固定套与对应所述滑动杆之间固接有弹簧，所述滑动板远离对应所述第一液压缸的一侧设置有用于控制其内第一通孔连通面积的限速组件。
作为本发明的优选，所述限速组件包括有第三液压缸，所述第三液压缸固接于对应所述滑动板远离对应所述第一液压缸的一侧，所述第三液压缸连通有与对应所述固定套远离对应所述滑动直杆一侧连通的导流管，所述第三液压缸的伸缩端固接有与对应所述滑动板滑动连接的封堵板，所述封堵板与对应所述滑动板第一通孔封堵配合。
作为本发明的优选，所述封堵板封堵对应所述滑动板第一通孔的最大面积仅为一半，用于限定矩形阵列分布的所述转动壳和对应所述滑动直杆的相对移动。
作为本发明的优选，还包括有矩形阵列分布的缓冲限位机构，矩形阵列分布的所述缓冲限位机构分别设置于对应所述转动支板的下侧，所述缓冲限位机构用于缓冲对应所述转动支板受到的挤压，所述缓冲限位机构包括有缓冲板，所述缓冲板滑动连接于对应所述转动支板的下侧，所述缓冲板与对应所述转动支板之间固接有拉簧，所述转动支板的上侧固接有连接杆，所述连接杆上侧限位滑动连接有连接滑杆，所述连接滑杆远离对应所述连接杆的一侧设置有单向轮，所述单向轮与对应所述连接杆之间固接有弹簧，所述缓冲板固接有与对应所述单向轮啮合的齿条，所述齿条穿透对应所述转动支板，所述转动支板的上侧设置有用于控制对应所述单向轮移动的复位组件。
作为本发明的优选，所述复位组件包括有固定支杆，所述固定支杆固接于对应所述转动支板的上侧，所述缓冲板和对应所述转动支板共同滑动连接有限位滑杆，所述限位滑杆的下侧固接有与对应所述缓冲板滑动连接的挤压板，所述固定支杆滑动连接有滑动弯板，所述滑动弯板上设置有与对应所述限位滑杆限位滑动的滑槽，所述滑动弯板与对应所述单向轮固接。
与现有技术相比，本发明的有益效果为：1、本发明通过固定块内滑动板的移动，配合转动壳和滑动直杆相对移动，达到对机体预固定支撑的目的，避免了有风天气下机体晃动使其上扇叶与停机坪碰撞，保证了机体下落过程中的稳定性和安全性。
2、通过锁定机构中挤压块与转动壳的挤压配合，保证本装置四个转动壳对机体支撑的稳定性和平衡性，降低了机体滑动对其降落定位的影响。
3、通过限速组件中封堵板的移动，达到控制滑动直杆向转动壳移动速度的目的，使机体稳定缓慢的放置于停机坪上，保证了机体的安全性和结构的稳定性，间接延长了机体的使用寿命。
4、通过缓冲限位机构中单向轮与齿条的啮合，使转动支板与停机坪之间稳定缓冲，保证了本装置内零件的安全性和使用寿命，降低了机体降落碰撞对停机坪损坏的可能性。

具体实施方式
下面结合附图1-图12，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
实施例1：现有飞行器在有风的天气下进行落地操作时，飞行器容易出现侧向晃动的情况，而飞行器机体1在晃动状态下降落会导致飞行器周向的扇叶（如多旋翼无人机的螺旋桨）与停机坪碰撞，引发扇叶损伤乃至破坏，进而导致飞行器失去动力平衡，发生失控，不仅危及机载人员的生命安全，而且会缩短飞行器的使用寿命。
一种应用于低空飞行器的防撞装置，参照图1-图5所示，包括有机体1，机体1的下侧转动连接有矩形阵列分布的四个转动壳2，转动壳2内滑动连接有滑动直杆3，滑动直杆3的下侧转动连接有转动支板4，通过四对转动壳2、滑动直杆3和转动支板4对机体1进行支撑固定，转动壳2的上侧固接有固定块5，固定块5设置有通孔，固定块5的通孔与对应转动壳2连通，固定块5内滑动连接有滑动板6，滑动板6设置有第一通孔和第二通孔，以右前侧的滑动板6为例，滑动板6上第一通孔的内径由右至左逐渐减小，滑动板6上的第一通孔和第二通孔均与对应固定块5的通孔连通配合，通过滑动板6的移动，滑动板6上两个通孔分别与对应固定块5的通孔连通，滑动板6内第一通孔设置为锥形，滑动板6内第一通孔的最大直径小于滑动板6内第二通孔的直径，用于限定对应转动壳2内对应滑动直杆3的移动速度，通过滑动板6的移动控制对应固定块5通孔内液体的流量，固定块5固接有储液筒7，储液筒7与对应固定块5的通孔连通，储液筒7内滑动连接有活塞直杆8，转动壳2和储液筒7内对应活塞直杆8靠近对应固定块5的一侧均注有液压油，滑动直杆3向对应转动壳2内移动时，转动壳2内液压油流入储液筒7内蓄压，便于后续液压油反向流动复位，机体1的下侧设置有用于控制矩形阵列分布滑动直杆3移动速度的锁定机构，滑动直杆3的侧壁上设置有用于检测其与对应转动壳2相对移动距离的状态检测机构，通过滑动板6的移动，使滑动板6与固定块5通孔连通的通孔切换，达到控制转动壳2内液压油流入储液筒7内的速度，进而使转动壳2、滑动直杆3和转动支板4对机体1进行支撑固定，避免了机体1在落地前由于风的吹动晃动使其扇叶与停机坪碰撞，保证了机体1的安全性和降落的稳定性。
参照图2和图6-图8所示，锁定机构包括有矩形阵列分布的四个固定连杆201，四个固定连杆201均固接于机体1的下侧，以右前侧的固定连杆201为例，固定连杆201的下侧固接有第一固定筒202，第一固定筒202内滑动连接有活塞支杆203，第一固定筒202与对应活塞支杆203之间固接有拉簧，该拉簧初始时处于拉伸状态，且挤压块204与对应转动壳2初始时为挤压状态，活塞支杆203固接有挤压块204，挤压块204的前侧设置有斜面，该斜面由前至后向右倾斜，挤压块204的斜面与转动壳2挤压配合，用于使转动壳2转动复位时挤压挤压块204，挤压块204移动复位，机体1的下侧固接有第二固定筒205，第二固定筒205的右侧连通有第一连通管206，四个第一固定筒202均与第一连通管206连通，第二固定筒205内滑动连接有滑动壳207，第二固定筒205与滑动壳207之间固接有弹簧，用于推动滑动壳207移动复位，第二固定筒205的侧壁上设置有用于控制四个滑动板6移动的换位组件，储液筒7的侧壁上设置有用于控制对应转动壳2转动的辅助转动组件，第二固定筒205、第一连通管206和第一固定筒202内对应活塞支杆203靠近对应挤压块204的一侧均注有液压油，通过挤压块204与转动壳2挤压配合，使本装置能够准确判断四组转动壳2和滑动直杆3是否对机体1进行支撑，便于后续控制机体1上扇叶停止转动，稳定控制机体1降落至停机坪。

在将本装置安装在飞行器的下侧后，在有风的天气条件下，当飞行器晃动的情况下向下移动进行降落时，机体1带动四个转动壳2和与其相连的零件同步向下移动，前侧的两个转动支板4先接触并挤压地面，以右前侧的转动壳2为例，随着机体1通过转动壳2和滑动直杆3带动转动支板4向下移动，转动支板4在滑动直杆3上顺时针（由前至后看）转动并压缩与其相连的弹簧，直至转动支板4的下侧面与停机坪贴合后，转动支板4停止运动。
当转动支板4与停机坪贴合后，随着机体1向下移动进行降落，四个转动壳2和滑动直杆3逐步对机体1进行支撑，滑动直杆3向转动壳2内移动，转动壳2内的液压油通过固定块5的通孔和滑动板6第二通孔进入储液筒7内，储液筒7内液压油增加，活塞直杆8向左移动并压缩与其相连的弹簧，在滑动直杆3向转动壳2内移动的过程中，转动支板4与停机坪贴合后不进行移动，活塞直杆8带动第二液压缸401的伸缩端向左移动，第二液压缸401的伸缩端伸出，第二液压缸401内液压油通过导液管402流入弧形筒403内弧形活塞杆404下侧，弧形活塞杆404拉伸与其相连的拉簧，最终使转动壳2相对于机体1逆时针（由前至后看）转动，通过弧形筒403内弧形活塞杆404移动，使滑动直杆3和转动壳2相对移动的过程整体摆动，达到了转动支板4与停机坪稳定贴合的目的，避免了转动支板4在停机坪上滑动影响本装置对机体1支撑的稳定性。
