Description

结合气象数据的低空飞行器智能涡轮发电控制方法
技术领域
本发明涉及飞行器能源管理技术领域，尤其涉及结合气象数据的低空飞行器智能涡轮发电控制方法。
背景技术
随着低空飞行器在巡航、物流运输、环境监测等领域的广泛应用，对其续航能力和能源管理的要求日益提高。传统飞行器多依赖电池或燃料驱动，而这些能源形式在复杂的气象环境中续航时间有限，且能源利用效率较低。近年来，涡轮发电技术逐渐应用于低空飞行器中，通过风能发电补充飞行器的电力需求。然而，如何根据实时气象数据和飞行任务需求，动态调整发电策略，使得飞行器在不同环境下都能高效地利用风能，成为现有技术中的一个关键挑战。
现有技术中，涡轮发电系统多依赖固定的气象条件判断和预设的发电模式，无法灵活应对复杂和多变的气象条件，导致了发电效率低下以及能源浪费等问题。此外，现有系统缺乏实时的反馈机制，无法根据实时变化的气象条件对发电功率进行动态调整，容易出现风速过高或不足时发电功率不匹配的情况，影响了飞行器的续航能力和运行效率。因此，迫切需要一种能够结合实时气象数据，并动态调整发电功率的智能控制方法，解决上述问题。
发明内容
基于上述目的，本发明提供了结合气象数据的低空飞行器智能涡轮发电控制方法。
结合气象数据的低空飞行器智能涡轮发电控制方法，包括以下步骤：
S1：在飞行器上部署气象传感器阵列，用于实时获取飞行器周边的风速、风向、温湿度和气压的数据，并通过无线通信技术将获取的数据传输至地面控制中心；
S2：地面控制中心对接收到的数据进行分析，结合历史气象数据和预设的气象模型，预测未来短时间内的气象条件；
S3：根据S2分析的结果，判断是否具备启动涡轮发电的条件；
S4：若S3中判断具备启动条件时，飞行器将根据当前的飞行任务和电力需求，选择对应的发电模式；所述发电模式包括全功率发电模式、部分功率发电模式和待机模式；
S5：在发电过程中，根据飞行器的实时电量、飞行速度以及气象条件的变化，动态调整涡轮的发电功率，使发电输出与飞行器的电力需求相匹配；
S6：飞行器持续获取气象数据，并将实时数据反馈到地面控制中心，控制中心将根据最新的气象数据调整发电策略，以应对气象条件的变化。
可选的，所述S1具体包括：
S11：所述气象传感器阵列包括风速传感器、风向传感器、温湿度传感器和气压传感器，所有传感器均布置在飞行器的外部结构表面；其中，风速传感器安装在飞行器前部机头位置；风向传感器安装在飞行器机翼的两侧；温湿度传感器安装在飞行器底部的保护罩内；气压传感器则安装在飞行器尾部上方位置；
S12：所述风速传感器用于实时测量飞行器周围的空气流动速度，风向传感器用于实时检测飞行器前方和侧面的风向变化，温湿度传感器用于实时测量环境温度和湿度，气压传感器用于检测飞行器所在高度的气压值；每个传感器均通过数据接口与飞行器的中央数据处理单元相连；
S13：所有传感器获取的数据将通过数据处理单元进行初步处理，排除由飞行器振动或气流波动引起的高频噪声和瞬时异常数据，确保气象数据的平滑性和准确性；处理后的数据通过无线通信模块传输至地面控制中心，所述无线通信模块采用基于IEEE 802.11协议的无线局域网技术，确保数据在复杂环境下的传输速率和稳定性。
可选的，所述S2具体包括：
S21：地面控制中心接收到传输的气象数据后，通过数据清洗算法对数据进行处理，去除噪声数据和异常值，确保数据的准确性；
S22：将清洗后的实时气象数据与历史气象数据进行比对分析，所述历史气象数据包括过去相同区域内、相同季节和气象条件下的风速、风向、温湿度及气压数据；具体比对过程使用动态时间规整算法，用于捕捉实时数据与历史数据之间的时间序列相似性；
S23：地面控制中心基于比对分析后的数据调用预设的气象模型，结合区域气象特点预先训练的贝叶斯网络算法，通过输入当前实时数据与历史数据的分析结果，直接预测未来短时间内的气象变化趋势，尤其针对风速和风向的变化；
S24：结合模型预测结果，生成未来气象条件的变化趋势图，用于后续涡轮发电启动条件的判断。
可选的，所述S22具体包括：
S221：将实时气象数据和历史气象数据分别表示为两个时间序列，并为每个时间点的数据赋值；
S222：计算实时气象数据序列中每个时间点与历史气象数据序列中每个时间点之间的差值，并生成一个表示两个序列之间差异的矩阵，用于确定数据之间的匹配程度；
S223：基于差值矩阵，构建累积差异值矩阵，通过比较不同路径的累积值，找到从第一个数据点开始到每个数据点的最小差异值，从而确定两个序列之间的最优匹配路径；
S224：根据累积差异值的结果，确定实时气象数据与历史气象数据在每个时间点的最佳匹配点，并构建一个包含所有匹配点的路径；
S225：根据最优匹配路径，计算两个时间序列之间的总差异值，总差异值用于表示两个序列之间的整体相似度。
可选的，所述S23具体包括：
S231：地面控制中心接收到实时数据与历史数据的比对分析结果后，先将数据输入到预设的贝叶斯网络模型中，所述贝叶斯网络模型是基于区域气象特点预先训练的，模型中的节点表示气象变量，包括风速、风向、温度和气压，边表示变量之间的条件概率关系；
S232：根据贝叶斯网络的条件概率公式，计算出当前气象条件下各个气象变量的联合概率分布；
S233：利用当前实时气象数据作为观测证据，更新贝叶斯网络中未观测气象变量的后验概率分布；
S234：根据更新后的后验概率分布，预测未来短时间内的风速和风向变化趋势，具体先提取后验概率分布中风速和风向的概率值，得到其在未来各个取值下的概率分布；然后根据概率分布，确定风速和风向的变化范围，生成未来短时间内的风速和风向预测结果。
可选的，所述S3具体包括：
S31：根据步骤S2分析的结果，提取当前的风速和风向，并将风速与预设的启动风速阈值进行比较，若当前风速，则风速满足启动条件；
S32：在风速条件满足的前提下，分析通过贝叶斯网络预测得到的未来短时间内的风速和风向变化趋势，若预测的未来风速仍然满足启动风速阈值，且预测风向与飞行器前进方向的夹角在允许的最大偏差角度之内，则认为未来气象条件适合启动发电；
S33：结合当前气象条件和预测的短期气象变化，若当前和未来风速、风向均满足发电启动条件，地面控制中心则判定具备启动涡轮发电的条件，并发出启动指令。
可选的，所述S4具体包括：
S41：在判断具备启动涡轮发电的条件后，先评估飞行器当前的飞行任务类型，包括飞行器是否处于巡航模式、低速飞行模式或悬停模式，并结合飞行器的电力需求，确定发电功率的需求等级；
S42：当飞行器处于高电力需求的任务状态时，选择全功率发电模式；
S43：若飞行任务处于中等电力需求状态，选择部分功率发电模式；
S44：当飞行器处于低电力需求状态或即将完成任务时，选择待机模式。
可选的，所述S5具体包括
S51：实时监测飞行器的电量状态，并计算剩余电量与任务电力需求之间的差异；若，则说明剩余电量不足，系统需要提高发电功率；若，则电量充足，系统减少发电功率；
S52：根据飞行速度的实时变化，评估飞行速度对电力需求的影响，当飞行速度增大时，负载功率增加，发电系统相应提高功率输出；当速度减小时，负载功率降低，发电功率也需相应减少；
S53：结合气象条件，包括实时风速和实时风向的变化，评估涡轮发电效率；当风速较高时，涡轮发电的功率输出提高，风速较低或风向不利时，则需要降低发电功率；
S54：综合电量状态、飞行速度和气象条件，，采用反馈控制算法动态调整涡轮发电功率，使发电输出与飞行器的电力需求相匹配。

可选的，所述S6具体包括：
S61：飞行器通过部署的气象传感器，持续实时采集周围环境的气象数据，包括风速、风向、温湿度和气压，并将这些数据通过无线通信系统发送至地面控制中心；
S62：地面控制中心接收实时传输的气象数据后，对其进行数据清洗和预处理，以剔除噪声和异常数据；处理后的数据将结合历史气象数据和预设的气象模型进行分析，预测短期内气象条件的变化趋势；
S63：地面控制中心对当前的发电策略进行调整；当预测到风速增加且风向与飞行器前进方向有利时，控制中心会发出指令，增加发电功率；当预测到风速减弱或风向不利时，系统将降低发电功率；
S64：地面控制中心将调整后的发电策略通过无线通信技术反馈给飞行器，飞行器根据收到的指令，自动调节涡轮发电机的输出功率，以应对当前气象条件的变化。
本发明的有益效果：
本发明，通过结合实时气象数据分析和预测技术，能够根据风速、风向等气象条件的变化，动态调整涡轮发电的功率输出，通过实时采集和反馈气象数据，地面控制中心可以及时优化发电策略，确保在风能条件有利时提高发电效率，而在风速不足或风向不利时减少功率输出，从而有效提升飞行器的能源利用效率和续航能力。
本发明，通过采用了基于历史气象数据和预设模型的智能预测技术，通过对短期气象变化趋势的预判，提前做出发电策略的调整，避免了气象条件突变对发电系统的影响，这种技术手段提高了系统的响应速度和灵活性，确保飞行器能够在复杂的气象环境下稳定、高效地运行，显著改善了现有技术中存在的能源浪费和续航不足问题。

S63：地面控制中心对当前的发电策略进行调整；当预测到风速增加且风向与飞行器前进方向有利时，控制中心会发出指令，增加发电功率；当预测到风速减弱或风向不利时，系统将降低发电功率，确保发电系统在风能利用较低的情况下仍能稳定运行；
