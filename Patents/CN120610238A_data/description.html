Description

一种分布式低空飞行器声探测阵列的在线校准方法
技术领域
本发明属于分布式声探测阵列校准技术领域，具体涉及一种分布式低空飞行器声探测阵列的在线校准方法。
背景技术
低空飞行器通常指飞行高度在1000米以下的航空器，常见类型包括无人机、直升机和电动垂直起降飞行器(eVTOL)等载人飞行器，具有高机动性、垂直起降能力、低空域适应性和智能化操控等特点，结合电动化、轻量化设计，可灵活执行短距运输、精准作业等任务。
低空飞行器在低空经济中应用场景多元：物流领域依托无人机和eVTOL实现城市空中配送与偏远地区物资投送；城市交通探索“飞行汽车”试点，缓解地面拥堵；农业植保、电力巡检通过高精度传感器提升作业效率；应急救援则利用快速响应优势开辟“空中生命通道”。军事领域，低空飞行器兼具战术与战略价值：小型无人机执行侦察监视、电子干扰及集群打击任务；重型无人货运平台保障前线物资补给；隐身型飞行器渗透敌后实施突袭，而有人-无人协同系统进一步扩展战场信息感知与协同作战能力。随着空域管理数字化与能源动力技术突破，低空飞行器正加速推动经济业态升级与军事作战模式革新，成为未来城市发展与国防智能化的关键载体。
随着低空飞行器的快速普及技术升级，使其逐渐成为新型空中威胁，其探测和反制技术成为亟待解决的问题。低空飞行器探测技术手段包括雷达、光电成像、声学等，单一技术无法同时满足复杂环境、远距离、全天候、实时、高精度识别等应用需求，通常采用多技术融合的方法，声学技术可广域探测目标方位，而光学技术可实现近距离高精度定位与识别。
声探测技术因其阵型布放灵活、低空定位精度高、全天候实时监控等优势，已成为雷达、光电成像探测的有效“补盲”技术手段。声探测技术采用传声器阵列的指向性实现声源目标的位置定向，多阵列定向的交汇实现动态声源目标的定位和轨迹跟踪，根据声源目标的信号特征和应用场景对探测装备的尺寸、功耗和重量要求，标量或矢量传声器及其阵列阵型均可灵活配置。声探测技术的主要局限是其探测距离有限，通常采用分布式部署予以弥补。传声器阵元数量多，几百乃至上千节点，且多采用低成本传声器；具有广域、全天候、实时监测功能，对探测阵列的定位及目标识别性能有长期稳定性的要求。因此，分布式声探测阵列需要周期性在线校准，以保障定位和目标识别特性的性能保持。考虑其广域、全天候、实时监测的特点，分布式低空飞行器声探测系统不具备拆卸送校实验室的条件，亟需提出在线、原位校准方法，用于核查其长期运行过程中定位精度、目标特性识别的性能保持情况。
发明内容
本发明的目的在于：提供一种分布式低空飞行器声探测阵列的在线校准方法，解决现有分布式低空飞行器声探测阵列长期运行可能出现的性能退化，需要进行在线校准的问题。
本发明采用的技术方案如下：
一种分布式低空飞行器声探测阵列的在线校准方法，包括以下步骤：
(1)将机载目标模拟源悬挂在无人机上，机载目标模拟源控制器和无人机地面遥控器放置于地面；
(2)无人机地面遥控器控制携带机载目标模拟源的无人机的飞行轨迹和运动姿态，机载目标模拟源控制器获取机载目标模拟源的空间位置坐标；
(3)携带机载目标模拟源的无人机在预设的空间位置处悬停，机载目标模拟源控制器启动机载目标模拟源，调整功率放大器的增益挡位，依次播放不同目标特征的标准声纹数据，同步采集待校准的分布式低空飞行器声探测阵列的定位结果及目标特性识别结果，与机载目标模拟源控制器接收的预设空间位置坐标和已标注的标准声纹特征信息进行比较，实现分布式低空飞行器声探测阵列的在线校准；
(4)携带机载目标模拟源的无人机依次悬停在不同的空间位置处，实现多点位、多目标特征的分布式低空飞行器声探测阵列的在线校准，校准参数包括定位精度和目标特性识别的准确度；
(5)校准程序完成后，机载目标模拟源控制器关闭机载模拟源，无人机地面遥控器控制无人机返航。
进一步地，所述机载目标模拟源控制器通过无线通信模块与机载目标模拟源信号连接。
进一步地，所述机载目标模拟源包括动圈扬声器、功率放大器、电源、北斗定位模块、无线通信模块。
进一步地，所述机载目标模拟源通过支杆悬挂于无人机下方。
进一步地，所述机载目标模拟源设置有多个，多个机载目标模拟源均通过支杆悬挂于无人机下方。
综上所述，由于采用了上述技术方案，本发明的有益效果是：
1、本发明中，无人机携带机载目标模拟源按预设的飞行轨迹和校准位置悬停，位于地面的机载目标模拟源控制器通过无线通信模块获取机载目标模拟源内置的北斗定位模块返回的空间位置坐标，采用已知空间位置坐标的机载目标模拟源实现分布式低空飞行器声探测阵列定位精度的在线校准。
2、本发明中，位于地面的机载目标模拟源控制器通过无线通信模块远程控制机载目标模拟源播放不同目标特征的标准声纹数据集，模拟不同低空飞行器通过目标区域，实现分布式低空飞行器声探测阵列目标特性识别的在线校准。
3、本发明中，机载目标模拟源可直接悬挂于无人机下方，也可通过支杆悬挂，使得机载目标模拟源和无人机保持一定间距，避免悬停状态的无人机声纹干扰分布式低空飞行器声探测阵列对目标模拟声纹的识别。此外，多个机载目标模拟源可通过一定的支杆连接结构同时悬挂于无人机下方，实现分布式低空飞行器声探测阵列对多个目标声纹的定位和识别校准。
