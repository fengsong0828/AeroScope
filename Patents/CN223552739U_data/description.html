Description

低空飞行器轻量化连接器
技术领域
本实用新型涉及航空航天技术领域，具体为低空飞行器轻量化连接器。
背景技术
随着低空飞行器广泛应用于物流运输、农业监测、灾害救援等领域，对其轻量化设计的需求日益增加，因此需要低空飞行器轻量化连接器。
现有的低空飞行器连接器重量较大，多采用传统金属材质进行设计，从而增大了飞行器负载，限制了飞行性能，且多采用插拔式以方便安装与拆卸，但在实际的使用过程中缺少对连接处的固定结构，可能导致连接不紧密，进而造成松动，稳定性欠佳。
实用新型内容
本实用新型的目的在于提供低空飞行器轻量化连接器，以解决上述背景技术中提出的现有的低空飞行器连接器重量较大，多采用传统金属材质进行设计，从而增大了飞行器负载，限制了飞行性能，且多采用插拔式以方便安装与拆卸，但在实际的使用过程中缺少对连接处的固定结构，可能导致连接不紧密，进而造成松动，稳定性欠佳。
为实现上述目的，本实用新型提供如下技术方案：低空飞行器轻量化连接器，包括连接器本体，所述连接器本体的顶部固定安装有固定圈，所述固定圈的顶部转动安装有丝杆，所述丝杆的底部贯穿固定圈并延伸至固定圈的下方，所述丝杆的底部固定安装有密封圈，所述连接器本体、固定圈、密封圈的材质为碳纤维，所述丝杆的顶部固定安装有转把，所述固定圈的两端皆固定安装有固定板，所述固定板的内部皆开设有卡槽，所述密封圈的两端皆固定安装有移动板，所述移动板的顶部皆固定安装有卡块，且固定板、移动板、卡块的材质为聚碳酸酯。
优选的，所述固定圈的顶部固定安装有竖管，所述竖管的一端开设有竖槽，所述竖槽的内部活动安装有圆轴，所述密封圈的顶部固定安装有竖杆，所述竖管、圆轴与竖杆的材质为钛合金，所述竖杆的顶部贯穿固定圈并延伸至竖管的内部，且竖杆的一端与圆轴的一端固定连接。
优选的，所述连接器本体的表面皆固定安装有固定杆，所述固定杆的材质为铝合金，且固定杆的内部皆开设有条形槽。
优选的，所述条形槽的内部皆固定安装有圆杆，所述圆杆的表面两端皆活动套接有滑块，所述滑块与条形槽之间皆固定安装有缓冲弹簧，所述缓冲弹簧的内表面与圆杆的外表面活动连接，所述滑块的底部皆固定安装有固定块，且圆杆、滑块与固定块的材质为铝合金。
优选的，所述连接器本体的表面两侧皆固定安装有圆管，所述圆管的材质为镁合金，所述连接器本体的表面两侧皆固定安装有位于圆管内部的复位弹簧，且复位弹簧的外表面与圆管的内表面活动连接。
优选的，所述复位弹簧的底部固定安装有防护壳，所述防护壳的顶部皆固定安装有挤压块，且防护壳与挤压块的材质为镁合金。
与现有技术相比，本实用新型的有益效果是：
该低空飞行器轻量化连接器，在进行日常使用的过程中，连接器本体、固定圈、密封圈的材质为碳纤维，其强度高，具有出色的耐腐蚀性，固定板、移动板、卡块的材质为聚碳酸酯，其具有优异的耐冲击性、良好的绝缘性和抗老化性，竖管、圆轴与竖杆的材质为钛合金，其耐热性好、强度高，固定杆、圆杆、滑块与固定块的材质为铝合金，其强度高，圆管、防护壳与挤压块的材质为镁合金，其密度低并具有良好的减震性能，以上材料不仅强度高性能优异且质轻，从而减少连接器的整体重量。
该低空飞行器轻量化连接器，在进行日常使用的过程中，操作人员将连接器本体进行连接后，随之旋转转把，转把在旋转时会带动丝杆发生转动，此时丝杆会带动密封圈发生滑动，随之密封圈会带动竖杆在竖管的内部发生滑动，随后竖杆会带动圆轴在竖槽的内部发生滑动，同时密封圈会带动移动板与卡块发生滑动，直至卡块置于卡槽的内部，即可通过固定圈与密封圈对连接处进行密封，即可提高连接的紧密性，从而减少松动。

具体实施方式
下面将结合本实用新型实施例中的附图，对本实用新型实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本实用新型一部分实施例，而不是全部的实施例。基于本实用新型中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本实用新型保护的范围。
请参阅图1-6，本实用新型提供一种技术方案：低空飞行器轻量化连接器，包括连接器本体1，连接器本体1的顶部固定安装有固定圈2，固定圈2与密封圈4的尺寸相同，固定圈2的顶部转动安装有丝杆3，丝杆3的底部贯穿固定圈2并延伸至固定圈2的下方，固定圈2与丝杆3通过螺纹进行连接，丝杆3的底部固定安装有密封圈4，丝杆3在旋转时会带动密封圈4发生滑动，连接器本体1、固定圈2、密封圈4的材质为碳纤维，其是由经环氧涂层处理和石墨压织的碳化纤维制成的，不仅强度高而且具有出色的耐腐蚀性，丝杆3的顶部固定安装有转把5，由于转把5的设计可方便对丝杆3进行旋转，固定圈2的两端皆固定安装有固定板6，固定板6的内部皆开设有卡槽7，卡槽7与移动板8相适配，密封圈4的两端皆固定安装有移动板8，移动板8的顶部皆固定安装有卡块9，密封圈4在滑动时会带动移动板8与卡块9发生滑动，且固定板6、移动板8、卡块9的材质为聚碳酸酯，其是分子链中含有碳酸酯基的高分子聚合物，并具有优异的耐冲击性、良好的绝缘性和抗老化性，固定圈2的顶部固定安装有竖管10，竖管10的一端开设有竖槽11，竖槽11的内部活动安装有圆轴12，竖槽11的内表面与圆轴12的外表面皆光滑，密封圈4的顶部固定安装有竖杆13，竖管10、圆轴12与竖杆13的材质为钛合金，指的是多种用钛与其他金属制成的合金金属，其耐热性好、强度高，竖杆13的顶部贯穿固定圈2并延伸至竖管10的内部，且竖杆13的一端与圆轴12的一端固定连接，竖杆13在滑动时会带动圆轴12在竖槽11的内部发生滑动。
连接器本体1的表面皆固定安装有固定杆14，固定杆14的数量为两个，且两个固定杆14的尺寸相同，固定杆14的材质为铝合金，且固定杆14的内部皆开设有条形槽15，条形槽15的内部皆固定安装有圆杆16，圆杆16的表面两端皆活动套接有滑块17，滑块17的内表面与圆杆16的外表面皆光滑，可使滑块17在圆杆16的表面滑动的更加流畅，减少出现卡顿的情况，滑块17与条形槽15之间皆固定安装有缓冲弹簧18，缓冲弹簧18的内表面与圆杆16的外表面活动连接，滑块17在滑动时会对缓冲弹簧18进行挤压并压缩，且滑块17的底部皆固定安装有固定块19，固定块19在滑动时会带动滑块17在条形槽15的内部及圆杆16的表面发生滑动，且圆杆16、滑块17与固定块19的材质为铝合金，其是以铝为基添加一定量其他合金化元素的合金，其强度高，连接器本体1的表面两侧皆固定安装有圆管20，连接器本体1的表面两侧皆固定安装有位于圆管20内部的复位弹簧21，且复位弹簧21的外表面与圆管20的内表面活动连接，复位弹簧21的底部固定安装有防护壳22，防护壳22在滑动时会对复位弹簧21进行挤压并压缩，同时防护壳22可对连接器本体1进行防护，防护壳22的顶部皆固定安装有挤压块23，防护壳22在滑动时会带动挤压块23发生滑动，且防护壳22与挤压块23的材质为镁合金，其是以镁为基础加入其他元素组成的合金，其密度低并具有良好的减震性能。
工作原理：首先操作人员将连接器本体1进行连接后，随之旋转转把5，转把5在旋转时会带动丝杆3发生转动，此时丝杆3会带动密封圈4发生滑动，随之密封圈4会带动竖杆13在竖管10的内部发生滑动，随后竖杆13会带动圆轴12在竖槽11的内部发生滑动，同时密封圈4会带动移动板8与卡块9发生滑动，直至卡块9置于卡槽7的内部，即可通过固定圈2与密封圈4对连接处进行密封，其次当防护壳22受到冲击时，随之防护壳22会带动挤压块23发生滑动，此时挤压块23的外表面会不断挤压固定块19的外表面，随之固定块19会带动滑块17在条形槽15的内部及圆杆16的表面发生滑动，随后滑块17会对缓冲弹簧18进行挤压并压缩，同时防护壳22会对复位弹簧21进行挤压并压缩，即可对冲击力进行缓冲。
