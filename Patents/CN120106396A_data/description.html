Description

基于机器学习的低空飞行器风险管理方法及系统
技术领域
本发明涉及低空飞行技术领域，尤其涉及一种基于机器学习的低空飞行器风险管理方法及系统。
背景技术
随着低空空域逐步开放以及无人机技术的快速发展，低空飞行器在城市物流、农业植保、电力巡检和应急救援领域的应用日益广泛，然而，低空飞行器在复杂环境中运行时面临诸多潜在风险因素，如突发气象变化、地形干扰、通信延迟及飞行器自身性能波动，容易引发飞行偏航、姿态失稳、轨迹异常飞行安全问题，因此，如何实现对低空飞行器飞行状态的有效监控与风险评估，成为当前低空飞行安全管理亟待解决的重要课题。
现有的低空飞行器风险管理技术主要依赖规则库判定和传统模型评估，系统多以静态风险指标为基础，通过预设规则判断飞行轨迹是否越界或存在异常点，但存在明显不足：一方面，静态规则难以适应飞行过程中不断变化的动态特征，无法有效识别轨迹复杂性与飞行参数之间的交互风险；另一方面，当前常用的风险评估模型通常依赖人工设定特征或单尺度分析，缺乏对飞行轨迹局部变化与空间复杂结构的精细建模能力，导致对异常行为或潜在风险识别精度不高，此外，部分系统虽然集成了基础的飞行轨迹监控模块，但缺乏对多维动态飞行参数与轨迹形态变化的联合分析能力，容易出现误判或漏判的情况，严重影响风险响应的时效性和准确性。
综上所述，亟需一种能够融合多尺度轨迹信息与飞行动态特征，并具备智能评估能力的全流程风险管理方法加以改进。
发明内容
本发明的一个目的在于提出一种基于机器学习的低空飞行器风险管理方法及系统，本发明精准捕捉轨迹中具有高转弯率或速度剧变高风险特征段，极大提高了局部风险事件的识别准确性。
根据本发明实施例的一种基于机器学习的低空飞行器风险管理方法，包括如下步骤：
S1.实时获取低空飞行器在指定低空区域内的三维飞行轨迹原始数据集并进行预处理，生成预处理后的飞行轨迹数据集；
S2.对预处理后的飞行轨迹数据集实施多尺度切分处理，形成分尺度飞行轨迹数据集，对分尺度飞行轨迹数据集应用分形维数分析方法提取飞行轨迹的复杂性描述向量；
S3.将飞行轨迹复杂性描述向量与低空飞行器运行过程中的动态飞行参数数据进行数据融合，构建多维风险特征数据集；
S4.基于最小描述长度原则对多维风险特征数据集实施风险评价模型的候选模型筛选，通过评估各候选模型的模型编码长度及数据编码长度，自动选取描述长度最小且数据拟合精度最佳的最优风险评价模型；
S5.利用所选最优风险评价模型对新采集的飞行轨迹数据集及其对应的多维风险特征数据集进行实时风险评估，生成飞行轨迹风险评价输出；
S6.将飞行轨迹风险评价输出反馈至飞行控制系统及监控终端，实现动态风险监控与飞行路径调整，并为低空飞行器风险管理提供决策依据。

可选的，所述风险等级划分规则包括：
当分形维数特征值大于设定的上限阈值，且瞬时飞行速度变化值或加速度变化值超过对应飞行器安全容限范围时，判定该轨迹分段为高风险分段，输出高风险等级；
当分形维数特征值处于中间范围，且飞行姿态角变化值在正常波动范围内，但存在单一参数异常波动时，判定该轨迹分段为中风险分段，输出中风险等级；
当各项动态飞行参数变化值均在安全阈值范围内，分形维数特征值处于设定下限范围以下时，判定该轨迹分段为低风险分段，输出低风险等级。
一种基于机器学习的低空飞行器风险管理系统，用于执行所述的一种基于机器学习的低空飞行器风险管理方法，包括如下模块：
轨迹数据采集模块，用于采集飞行器在指定区域内的三维飞行轨迹数据集；
数据预处理模块，用于对飞行轨迹数据集进行时空对齐、异常点剔除与滤波处理，生成预处理后的飞行轨迹数据集；
多尺度特征提取模块，用于基于自适应尺度切分飞行轨迹并提取局部分形维数，形成飞行轨迹复杂性描述向量；
动态参数融合模块，用于提取各分段飞行轨迹的速度变化值、加速度变化值与姿态角变化值，并与复杂性描述向量融合生成多维风险特征数据集；
模型选择模块，用于基于最小描述长度原则从候选模型中选取描述长度最小的最优风险评价模型；
风险评估模块，用于基于所选模型对飞行轨迹进行风险等级预测与预警信息输出；
反馈模块，用于将风险评价结果反馈至飞行器控制系统与监控终端，实现飞行路径动态调整与风险预警联动。
本发明的有益效果是：
（1）本发明将飞行轨迹的局部分形维数作为核心描述子引入风险管理体系，通过多尺度切分后的轨迹分段提取分形维数特征，有效表征飞行轨迹在不同尺度下的局部复杂度与不规则性，此外，与瞬时速度变化值、加速度变化值与姿态角变化值动态飞行参数进行向量级联融合，构成多维风险特征向量，显著增强了模型对微小轨迹扰动和潜在风险征兆的敏感性，提升了高风险轨迹段的准确识别率。
（2）本发明突破传统依赖人工设定模型结构或固定模板的方式，采用信息论中的最小描述长度原理对候选模型集进行编码复杂度与数据误差联合优化评价，综合考虑模型结构复杂性与预测误差带来的信息损失，在众多候选模型中自动筛选出既具有紧凑结构又具备高风险判别能力的最优模型，确保了所选模型在不同飞行环境与轨迹模式下具备稳定可靠的风险评估性能。
（3）本发明设计了基于飞行轨迹点空间曲率变化率与飞行速度偏离度联合驱动的自适应划分函数，实现对飞行轨迹的非等距动态切分，不仅有效避免了固定时间窗口划分带来的特征丢失或冗余问题，还能精准捕捉轨迹中具有高转弯率或速度剧变高风险特征段，极大提高了局部风险事件的识别准确性。

在07时41分12秒，巡航至任务段第8航段（位置GPS坐标：xxx.260138°E，xx.876322°N）时，地面管控系统接收到异常提示：飞行器在短时间内发生了三次连续的非指令性偏航操作，其中第二次偏航幅度达到29.6度，远高于该型号机体允许的紧急航向调整上限（22度），与此同时，飞控回传数据显示瞬时速度Δv为5.91m/s，加速度Δa为7.24m/s²，姿态角变化Δψ为16.7°，并伴随3秒内分形维数飙升至1.894。
