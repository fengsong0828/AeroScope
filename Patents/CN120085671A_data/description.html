Description

低空智能动态监测飞行器的自主避障方法及系统
技术领域
本申请涉及低空飞行器避障技术领域，尤其涉及一种低空智能动态监测飞行器的自主避障方法及系统。
背景技术
低空飞行器集群技术在巡逻等领域得到广泛应用。然而，复杂环境下的低空飞行器集群自主避障和巡航空域规划仍面临诸多挑战，特别是在未知时变风扰的影响下，传统的路径规划和避障方法难以保证飞行安全和任务效果。
目前的低空飞行器集群控制方法主要采用集中式架构，这种方式在面对大规模集群协同任务时存在通信压力大、实时性差等问题。同时，现有的避障算法多基于确定性环境假设，难以应对动态变化的风扰环境，且缺乏对多机协同性能的整体考虑，容易导致局部最优解。
发明内容
本申请提供了一种低空智能动态监测飞行器的自主避障方法及系统，进而实现了低空飞行器集群的全局优化控制，确保了低空飞行器集群在未知风扰环境下的稳定运行。
本申请第一方面提供了一种低空智能动态监测飞行器的自主避障方法，所述低空智能动态监测飞行器的自主避障方法包括：
对低空飞行器集群的位置坐标、飞行速度、航向角进行状态向量构建，并结合障碍物位置数据和风速分量进行运动学建模，得到低空飞行器集群状态空间；
对所述低空飞行器集群状态空间中的控制点序列进行贝塞尔曲线计算和遗传优化，得到最优探测航线参数；
根据所述最优探测航线的航路点序列进行几何距离计算和双点移除处理，得到巡逻空域方程；
将所述低空飞行器集群状态空间、所述最优探测航线参数和所述巡逻空域方程输入分层强化学习框架进行多层次奖励函数计算，得到高低层决策网络；
基于所述高低层决策网络进行时序差分学习和策略梯度优化，得到分布式控制策略。
本申请第二方面提供了一种低空智能动态监测飞行器的自主避障系统，所述低空智能动态监测飞行器的自主避障系统包括：
建模模块，用于对低空飞行器集群的位置坐标、飞行速度、航向角进行状态向量构建，并结合障碍物位置数据和风速分量进行运动学建模，得到低空飞行器集群状态空间；
计算模块，用于对所述低空飞行器集群状态空间中的控制点序列进行贝塞尔曲线计算和遗传优化，得到最优探测航线参数；
处理模块，用于根据所述最优探测航线的航路点序列进行几何距离计算和双点移除处理，得到巡逻空域方程；
强化学习模块，用于将所述低空飞行器集群状态空间、所述最优探测航线参数和所述巡逻空域方程输入分层强化学习框架进行多层次奖励函数计算，得到高低层决策网络；
优化模块，用于基于所述高低层决策网络进行时序差分学习和策略梯度优化，得到分布式控制策略。
与现有技术相比，本申请具有以下有益效果：通过构建分层强化学习框架，将低空飞行器集群控制问题分解为高层任务分配和低层轨迹执行两个层次，降低了系统复杂度，提升了算法的可扩展性和计算效率。采用贝塞尔曲线结合遗传算法优化的方式生成最优探测航线，并通过椭圆拟合方法确定巡逻空域，使规划结果更符合实际飞行需求。设计了基于时序差分学习和策略梯度优化的分布式控制策略，实现了多机协同性能与单机控制性能的统一优化。通过引入自适应补偿控制器和卡尔曼滤波器，解决了未知时变风扰下的路径跟随问题，保证了系统的稳定性。采用基于几何距离的双点移除算法，有效避免了异常航路点对巡逻空域规划的影响，提高了空域划分的合理性。通过多层次奖励函数的设计，将碰撞避免、任务完成度和能量效率等多个目标进行统一考虑，实现了低空飞行器集群的全局优化控制。基于李雅普诺夫稳定性理论设计风扰补偿控制器，结合全局协同指标进行参数调整，确保了集群系统在未知风扰环境下的稳定运行。采用分布式部署方式实现控制策略，减少了通信负担，提高了系统的实时性和鲁棒性。

步骤100、对低空飞行器集群的位置坐标、飞行速度、航向角进行状态向量构建，并结合障碍物位置数据和风速分量进行运动学建模，得到低空飞行器集群状态空间；
可以理解的是，本申请的执行主体可以为低空智能动态监测飞行器的自主避障系统，还可以是终端或者服务器，具体此处不做限定。本申请实施例以服务器为执行主体为例进行说明。
具体的，将集群中的单架低空飞行器的二维平面坐标值、飞行速度值和航向角值进行组合，形成单机状态向量。每架低空飞行器的单机状态向量以矩阵的形式顺序排列，并对这些状态向量进行拼接，生成一个全面描述低空飞行器集群动态状态的集群状态矩阵，反映集群中所有低空飞行器的空间分布、运动特性和航向信息。对低空飞行器避障环境中的障碍物信息进行处理，以构建静态障碍物集合。提取每个障碍物的空间位置坐标和范围参数，利用这些数据对障碍物进行集合分析，构建包含m个障碍物的静态障碍物集合，描述任务区域内障碍物的空间分布和大小。同时，对当前时刻的水平方向风速分量和垂直方向风速分量进行提取，并将其组合成一个二维向量，构建时变风扰向量。为了确保低空飞行器的任务执行边界清晰，提取指定任务区域的水平和垂直方向的最小坐标值与最大坐标值，生成任务空域边界向量。将集群状态矩阵、时变风扰向量代入微分方程进行数值求解，以建立低空飞行器集群的运动学状态转移方程。该微分方程描述低空飞行器在外部环境影响下的动态变化，并提供时间序列上的状态预测能力。通过数值求解方法，获得低空飞行器集群的状态随时间的演化规律。对集群状态矩阵中的任意两架低空飞行器的位置数据进行欧氏距离计算，并与预设的风险阈值进行比较，得到碰撞风险度量函数，实时量化低空飞行器之间的碰撞风险，为动态调整飞行策略提供直接依据。同时，为了确保低空飞行器能够安全避开障碍物，针对集群状态矩阵中的每架低空飞行器与静态障碍物集合中的每个障碍物，分别进行最小距离计算，并结合安全裕度进行评估，得到障碍物风险度量函数。基于上述运动学状态转移方程、碰撞风险度量函数和障碍物风险度量函数，创建低空飞行器集群状态空间。
步骤200、对低空飞行器集群状态空间中的控制点序列进行贝塞尔曲线计算和遗传优化，得到最优探测航线参数；
具体的，对低空飞行器集群状态空间中的运动学状态转移方程进行数值积分运算。通过离散化状态转移方程并利用数值积分方法（如梯形法或龙格-库塔法）进行求解，获取低空飞行器在飞行过程中的运动轨迹信息，并提取出初始控制点集合。控制点集合在空间中构成一条初始路径。根据风险评估机制对初始控制点集合进行筛选和调整。通过结合碰撞风险度量函数和障碍物风险度量函数，评估每个控制点在实际路径中的安全性。例如，对于每个控制点，评估其与邻近低空飞行器以及环境障碍物之间的距离，如果发现某些控制点存在潜在的碰撞风险或偏离安全裕度的可能性，则对其进行标记并剔除，以确保剩余的控制点集合构成候选控制点序列。基于候选控制点序列，构建贝塞尔曲线的参数方程。对于每个控制点，计算与其相关的二项式系数。通过对这些基函数进行矩阵化处理，构造出一个n阶贝塞尔曲线的基函数矩阵。将基函数矩阵与控制点序列进行矩阵乘法运算，生成对应的贝塞尔曲线参数方程，描述低空飞行器在路径上的运动特性。贝塞尔曲线的优势在于其天然的平滑性和灵活性，通过控制点的调整来改变路径的形状，从而实现对飞行航线的动态优化。对贝塞尔曲线进行评价与优化。通过对航线长度、曲线平滑度和探测效率进行加权求和计算，构造目标优化函数，该函数综合考虑路径规划的多个重要指标。例如，航线长度越短，飞行效率越高；曲线越平滑，低空飞行器的运动控制越稳定；探测效率的高低直接影响任务完成的时间与效果。通过调整加权系数，根据实际任务需求对不同指标赋予不同的优先级。对构建的控制点序列进行染色体编码，每个染色体表示一个控制点的坐标值，并对这些染色体进行随机初始化，生成初始种群。初始种群作为算法的起点，为整个优化过程提供多样化的解空间。在优化过程中，采用遗传算法中的经典操作，包括轮盘赌选择、均匀交叉和高斯变异运算。轮盘赌选择根据个体的适应度值进行概率性选择，确保高适应度的个体有更大的生存和繁殖几率；均匀交叉通过随机交换染色体的基因片段来产生新的个体，增加种群的多样性；高斯变异通过对个体的基因进行微小随机调整，探索解空间中的局部区域，从而避免陷入局部最优解。通过多次迭代优化，种群中的个体逐步向全局最优解收敛，生成一个优化后的控制点序列。将优化后的控制点序列代入贝塞尔曲线参数方程，进行路径的重构与生成，得到最终的最优探测航线参数。
步骤300、根据最优探测航线的航路点序列进行几何距离计算和双点移除处理，得到巡逻空域方程；
需要说明的是，通过对最优探测航线参数进行参数离散化处理和坐标计算，得到初始航路点集合。这些航路点通过均匀离散化航线参数生成，每个点的坐标精确对应探测航线在离散化区间中的位置。在此基础上，对初始航路点集合中的相邻航路点进行曲率计算，以评估路径的几何变化特性。曲率值能够量化航线的弯曲程度，进而对这些点进行阈值判断，筛选出满足任务要求的航路点序列。对航路点的坐标数据进行均值计算和离差矩阵运算，提取航路点的统计特征参数，包括中心位置和分布范围，反映航路点在空间中的整体分布趋势。同时，为了量化航路点之间的几何关系，将航路点序列输入欧氏距离计算模块，对每两点之间的距离进行计算，生成航路点间的距离矩阵。距离矩阵中包含所有航路点之间的相对位置关系。对距离矩阵进行统计分析以判断异常点的分布。通过基于k倍标准差的原理，动态计算航路点之间的距离阈值，得到异常点判别准则。该准则依据航路点的正常分布特性设定，当两个航路点的距离偏离正常分布范围（超过k倍标准差）时，即被判定为异常点对。基于该判别准则，对航路点序列中的异常点进行迭代筛选和递归移除，对于不满足准则的点对，通过逐步剔除其中的异常点，使得优化后的航路点序列能够更加平滑，并避免冗余或误差点的干扰。在优化完成后，对航路点的几何特性进行建模。通过对优化后的航路点序列应用主成分分析，提取其在空间中的主导特性。主成分分析通过特征值分解，确定航路点分布的主轴方向和长短轴比例，这一结果描述航路点的分布椭圆的基本形状和方向。结合椭圆基本参数和航路点的统计特征参数，代入二次曲线的标准方程，构建巡逻空域方程。二次曲线方程通过对航路点坐标进行变换和系数计算，将这些点的分布特性以数学形式表示，并为低空飞行器的飞行任务提供明确的巡逻空域范围。得到巡逻空域方程。
