Description

一种用于低空飞行器的高压喷气装置及其喷气发动机
技术领域
本实用新型属于低空飞行器的飞行动力领域，具体地说，涉及一种用于低空飞行器的高压喷气装置及其喷气发动机。
背景技术
飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。随着飞行器普及，低速、低空飞行器将是个人飞行器的发展方向，尤其是为了缓解现有交通压力，已经有越来越多的车企在研究飞行汽车，用以作为汽车的前进方向之一。
现有飞行汽车基本均为概念器，其要解决的技术问题包括车体本身的重量以及供给汽车的飞行动力，现有的解决方案一是借鉴无人机的方式，采用螺旋桨向下鼓风提供动力，二是学习现有飞机采用涡轮式喷气发动机，但前者为了有足够的动力，需要有足够大的空间和足够大或足够多的螺旋桨来提供动力，对整体重量较大的飞行汽车来说，为了使其能够克服重力作用而浮在空中，则需要至少大于飞行汽车两倍以上的空间来满足安装螺旋桨的要求，而后者成本过高且动力过于强劲，用于低空飞行过于浪费。
发明内容
本实用新型的所要解决的技术问题在于提供一种空间占用相对较小、且不会过度造成动力浪费、结构合理的用于低空飞行器的高压喷气装置及其喷气发动机。
本实用新型解决上述技术问题的技术方案为：
一种用于低空飞行器的高压喷气装置，包括呈阵列排布的至少一组高压发生机构，所述高压发生机构包括有高压发生仓、用于气密性连接所述高压发生仓两端的连接件以及置于所述高压发生仓内的高压发生部件，所述连接件中设有相互隔离的进气仓与出气仓，所述进气仓、所述出气仓分别与所述高压发生仓相互导通，所述进气仓、所述出气仓与外界相通处均设有单向气阀，所述高压发生部件由活塞以及贯穿所述活塞的滑竿构成，所述滑竿的两端分别贯穿位于所述高压发生仓两侧的所述连接件，所述活塞与所述高压发生仓相匹配并能在所述高压发生仓内由所述滑竿带动进行活塞运动。
具体的，所述高压发生机构为至少两列设置，相邻的所述高压发生机构的所述连接件为一体结构，所述连接件上设有与列数数量相同的气仓槽，所述高压发生仓的端部气密性嵌入在所述气仓槽中，所述出气仓、所述进气仓与所述高压发生仓的连通处设置在所述气仓槽中。
进一步的，每列所述高压发生机构为至少两组设置，相邻的两组所述高压发生机构相互靠近的所述连接件呈镜像相靠设置，每组所述高压发生仓内均设有所述活塞，所述滑竿直接贯穿整列的所述高压发生机构并带动本列中所有的所述活塞。
具体的，所述连接件的侧壁上设有至少一条贯穿所述连接件使所述进气仓或所述出气仓与外界的气道，所述单向气阀设置在所述气道中并与所述气道的数量相匹配。
优选的，所述单向气阀包括固定件、密封件、弹簧件与连接杆，所述气道的进气口处设有用于放置所述固定件的凹槽，所述固定件固定在所述凹槽中，所述弹簧件固定在所述固定件上方，所述密封件置于所述气道的出气方向处并能封堵住所述气道的出气口，所述连接杆贯穿所述固定件且两端分别连接所述弹簧件与所述密封件。
具体的，镜像相靠设置的两个所述连接件中间设有隔板，单独设置的所述连接件远离所述高压发生仓的一面设有顶板，所述顶板与所述隔板均用于将所述进气仓与所述出气仓封闭使单个所述连接件中的进气仓与所述出气仓相对独立。
优选的，还包括有高压气仓与过滤仓，所述高压气仓与所述出气仓相对应并包裹所有贯通所述出气仓的所述气道，所述过滤仓与所述进气仓相对应并包裹所有贯穿所述进气仓的所述气道，所述隔板、所述顶板、所述高压发生机构、所述高压气仓与所述过滤仓均通过若干根贯穿式的螺杆进行固定。
本实用新型解决上述技术问题的另一技术方案为：
一种用于低空飞行器的喷气发动机，包括驱动源与固定架，还包括如上所述高压喷气装置，所述高压喷气装置至少两台，两台所述高压喷气装置呈镜像固定在所述固定架上，位于两台所述高压喷气装置的中间处设有同时带动两台所述高压喷气装置进行同步镜像运动的驱动机构，所述驱动源驱动所述驱动机构。
具体的，所述驱动机构包括驱动曲轴与驱动连杆，所述驱动连杆的两端用于铰接所述驱动曲轴与所述滑竿，所述驱动曲轴上至少设有一组方向相反且幅度一致的凸轴，用于带动两侧两个对称的所述滑竿的所述驱动连杆分别与互为一组的所述凸轴铰接。
进一步的，所述固定架上位于所述高压喷气装置的前后两侧均设有承载板，所述承载板上设有数量与所述高压发生机构列数一致的滑轨，所述滑轨上设有滑块与滑盖，所述滑块上方设有容纳所述滑竿的固定槽以及用于放置与所述驱动连接铰接的连接杆的连接槽，所述滑盖扣合在所述滑块上并将所述滑竿的端部以及所述连接杆固定在所述滑块上，所述滑块下方设有与所述滑轨相匹配的滑槽，所述滑槽中设有若干滑珠，所述滑槽滑设与所述滑轨上并通过所述滑珠减少摩擦力。
本实用新型具有以下有益效果：
（1）利用高压喷气装置中产生的高压空气，形成类似于蜗轮式喷气发动机的喷气效果，且可以通过控制高压发生机构的数量有效调节喷气的流量与压力，从而在提供充足的动力的前提下，能够有效的减少能源的浪费；
（2）通过单列多组的设置方式，使每个高压发生机构的高压发生仓长度缩小，进而减少滑竿的实际行程，在保障足够的高压空气的供给前提下，有效的减少了整个高压喷气装置的长度，并且较短的实际行程也能够有效的减少带动高压喷气装置的驱动距离。
（3）通过高压气仓对同一个高压喷气装置制造出来的高压空气进行收集后统一供给，在提高高压空气的利用度的前提下，还能够避免过多管道设置导致高压空气的不必要损耗以及结构的复杂程度增加。
（4）通过曲轴同时带动镜像设置的高压喷气装置，能够使整台发动机的运作更有规律，能够有效的避免发动机发生抖动的现象。
（5）通过设置滑轨与滑块的放置，有效减少滑竿在滑行时受到的摩擦力，有利于更好的将机械能转换成空气能，提高了能量的转化效率。

实施例：
本实用新型实施例的一种用于低空飞行器的喷气发动机，包括驱动源8与固定架4，还包括如上所述高压喷气装置1，所述高压喷气装置1至少两台，两台所述高压喷气装置1呈镜像固定在所述固定架4上，位于两台所述高压喷气装置1的中间处设有同时带动两台所述高压喷气装置1进行同步镜像运动的驱动机构，所述驱动源8驱动所述驱动机构。
其中，所述高压喷气装置1包括呈阵列排布的至少一组高压发生机构，所述高压发生机构包括有高压发生仓15、用于气密性连接所述高压发生仓15两端的连接件13以及置于所述高压发生仓15内的高压发生部件，所述连接件13中设有相互隔离的进气仓131与出气仓132，所述进气仓131、所述出气仓132分别与所述高压发生仓15相互导通，所述进气仓131、所述出气仓132与外界相通处均设有单向气阀14，所述高压发生部件由活塞16以及贯穿所述活塞16的滑竿17构成，所述滑竿17的两端分别贯穿位于所述高压发生仓15两侧的所述连接件13，所述活塞16与所述高压发生仓15相匹配并能在所述高压发生仓15内由所述滑竿17带动进行活塞16运动。通过活塞16对高压发生仓15内的空气进行挤压，在朝任何一个方向运动时，均能使高压空气从活塞16前进方向朝向的出气仓132灌出，并使外界空气从活塞16前进方向背面的进气仓131灌入，从而使活塞16的双向运动均在做工，有效提高了压缩空气的流量。
具体的，所述高压发生机构为至少两列设置，相邻的所述高压发生机构的所述连接件13为一体结构，所述连接件13上设有与列数数量相同的气仓槽133，所述高压发生仓15的端部气密性嵌入在所述气仓槽133中，所述出气仓132、所述进气仓131与所述高压发生仓15的连通处设置在所述气仓槽133中。采用多列设置的高压发生机构，能够大幅度提高单一一个高压喷气装置1的总喷出的高压空气流量，从而使高压喷气机构具备足够的动力进行运作。
进一步的，每列所述高压发生机构为至少两组设置，相邻的两组所述高压发生机构相互靠近的所述连接件13呈镜像相靠设置，每组所述高压发生仓15内均设有所述活塞16，所述滑竿17直接贯穿整列的所述高压发生机构并带动本列中所有的所述活塞16。采用多组成列的方式，使高压发生仓15的长度能够有效的控制，同时也能保证滑竿17的实际行程得到有效的控制，保证滑竿17的实际行程较短，从而能够使整个高压喷气装置1的体积有效减少但依旧能够具有大气缸所造成的高压空气流量。
具体的，所述连接件13的侧壁上设有至少一条贯穿所述连接件13使所述进气仓131或所述出气仓132与外界的气道134，所述单向气阀14设置在所述气道134中并与所述气道134的数量相匹配。优选的，所述单向气阀14包括固定件141、密封件142、弹簧件143与连接杆7144，所述气道134的进气口处设有用于放置所述固定件141的凹槽135，所述固定件141固定在所述凹槽135中，所述弹簧件143固定在所述固定件141上方，所述密封件142置于所述气道134的出气方向处并能封堵住所述气道134的出气口，所述连接杆7144贯穿所述固定件141且两端分别连接所述弹簧件143与所述密封件142。单向气阀14主要是为了在活塞16进行运动时，能够将活塞16前进方向前方的进气仓131以及活塞16前进方向后方的出气仓132对应的气道134密封，从而避免无用功的出现。优选的，气道134采用多条设计，能够有效避免单向气阀14的面积，从而使空气在推动单向气阀14时更为省力，减少空气动力的损耗。
具体的，镜像相靠设置的两个所述连接件13中间设有隔板12，单独设置的所述连接件13远离所述高压发生仓15的一面设有顶板11，所述顶板11与所述隔板12均用于将所述进气仓131与所述出气仓132封闭使单个所述连接件13中的进气仓131与所述出气仓132相对独立。连接件13正常采用铸件形式浇筑成型，为了方便生产，进气仓131与出气仓132会从连接件13原理气仓槽133的背面处裸露出来，因此，需要通过隔板12或者顶板11对其进行密封，从而保障进气仓131与出气仓132的密封性，同时也降低了其生产的难度，能够有利于产品量产。
优选的，还包括有高压气仓3与过滤仓2，所述高压气仓3与所述出气仓132相对应并包裹所有贯通所述出气仓132的所述气道134，所述过滤仓2与所述进气仓131相对应并包裹所有贯穿所述进气仓131的所述气道134，所述隔板12、所述顶板11、所述高压发生机构、所述高压气仓3与所述过滤仓2均通过若干根贯穿式的螺杆18进行固定。高压气仓3上有额外开设的管口31，用于与外界管道相连，供应高压空气，而过滤仓2的顶部设有过滤网，能够有效的过滤外界空气，避免外界杂质进入到高压发生机构中，此外，连接件13侧面还设有若干螺孔，所述螺孔用于将高压喷气装置1稳定的固定在固定架4上。
具体的，所述驱动机构包括驱动曲轴5与驱动连杆6，所述驱动连杆6的两端用于铰接所述驱动曲轴5与所述滑竿17，所述驱动曲轴5上至少设有一组方向相反且幅度一致的凸轴51，用于带动两侧两个对称的所述滑竿17的所述驱动连杆6分别与互为一组的所述凸轴51铰接。固定架4用于安装驱动曲轴5的位置处均设有轴承41，方便驱动曲轴5的安装与转动，同时，采用相互外扩镜像设置的凸轴51，能够对两侧的高压喷气装置1进行同步带动，从而减少整个喷气发动机运行过程中的抖动现象。
进一步的，所述固定架4上位于所述高压喷气装置1的前后两侧均设有承载板42，所述承载板42上设有数量与所述高压发生机构列数一致的滑轨43，所述滑轨43上设有滑块44与滑盖45，所述滑块44上方设有容纳所述滑竿17的固定槽441以及用于放置与所述驱动连接铰接的连接杆7144的连接槽442，所述滑盖45扣合在所述滑块44上并将所述滑竿17的端部以及所述连接杆7144固定在所述滑块44上，所述滑块44下方设有与所述滑轨43相匹配的滑槽444，所述滑槽444中设有若干滑珠445，所述滑槽444滑设与所述滑轨43上并通过所述滑珠445减少摩擦力。优选的，固定槽441与连接槽442相互融合，并呈两组镜像设置在滑块44上方，从而在安装时无需特定限制方向，方便安装，同时也方便将高压喷气装置1也采用阵列排布的方式进行布置，同列的高压喷气装置1前后相邻的两个之间也能直接通过该滑块44进行相连，并通过滑块44上设置的两根固定杆451，分别贯穿滑竿17端部的固定孔171以及连接杆7144中部的连接孔71后，嵌套在固定槽441与连接槽442融合后的插孔443中，再将滑盖45用螺丝锁紧在滑块44上，使整体连接稳固可靠，保障发动机的整体运行。
