Description

一种掠水面超低空飞行器
技术领域
本发明属于飞行器设计领域，具体涉及一种掠水面超低空飞行器的设计方法。该设计利用地面效应、喷水矢量与水翼联合控制技术在掠水面飞行器的力系平衡与操纵性设计中，另辟蹊径寻找攻克传统地效飞行器某些技术瓶颈的创新设计。
背景技术
目前，地效飞行器主要有如下几种：水上飞机、地效飞行器、地效翼船、冲翼艇等。
现有公开的技术文献“地效飞行器的发展现状及军事应用研究”(航空科学技术，2006年第一期)中阐述了地效飞行器是一种利用地面效应原理飞行的新型运载工具，该文以俄罗斯为主要背景，介绍了国外、国内地效飞行器的发展简史及现状，总结了地效飞行器的优越性能；文献“地效翼船气动仿真及横向静稳定性分析”(无限互联科技，2015年3月第六期)针对地效翼船横向稳定性问题，对飞行高度、攻角、展弦比等因素进行分析，介绍了地效翼船的主要设计特点；在文献“国内外地效飞行器实验研究”(战术导弹技术，2010年5月，32-36)中，对国内外地效飞行器实验数据结果进行了收集、整理,对目前的实验方法进行了分类、总结,得到了地效飞行器气动特性的一般规律。
早在四十多年前地效飞行器就被认为是海上运输与作战的极佳选择，并且在研制和使用方面取得了丰富的经验，人们对它的了解也比较深刻，但其推广与应用情况仍距人们的期望甚远。以技术领先的俄罗斯地效飞行器设计特点为例，为了消除波浪对地效力的影响以提高飞行安全性，设计了安装在高大垂直尾翼顶端的大面积稳定翼，而这种结构使得侧滑干扰有可能导致机翼顶端结构触水变成致命的安全问题；再如为了提高地面效应、减少机翼触水几率，采用小翼展的宽弦比机翼设计，加之高置的大面积稳定水平尾翼等结构设计，都不同程度的增加了飞行器的重量与飞行阻力，使得其飞行效率与经济性变差。
但是，由于地效飞行器至今仍存在诸多的技术障碍尚未突破，其应用推广仍然距人们的期望甚远。这些技术障碍主要体现在地效飞行器设计理论尚不成熟，在总体气动布局、动力增升、水动布局设计及地效飞行器的操纵性等关键技术方面仍有待突破；此外，地效飞行器在结构重量增加、地面效应利用困难、气动布局局限等方面也存在关键技术难题没能有效攻克[2]，导致地效飞行器经济性欠佳，难以得到广泛的推广应用。总之，正如前节所述，地效飞行器在总体气、水动布局与飞行器操纵性等方面的技术障碍成为其发展进程中的瓶颈。
同时，俄罗斯地效飞行器专家提出了利用涡扇发动机矢量推力技术提高飞行器的飞行轨迹精度与操纵性、意大利提出了利用浸水水翼技术为航行中的地效飞行器提供稳定的操纵力矩等观点以消除或减小地效力的不利影响。喷水推进技术虽然被提出也是在年代久远以前，但随着喷水推进器专业制造技术的不断进步，其技术的优越性仍然使之成为船舶领域重要的研究与应用对象，特别是近年来取得了长足的进展与广泛的应用。2010年前后，喷水飞行器jetovator的问世更是拓展了喷水推进装置的应用领域，也展示了喷水推进技术的先进性。
本发明也正是受到了上述观点的启发，提出了一种将喷水矢量技术、浸水水翼控制技术与地效飞行技术有效结合的掠水面超低空飞行器设计方法。
发明内容
本发明所要解决的技术问题是，为了避免现有技术存在的不足，本发明提出一种掠水面超低空飞行器，借鉴水上摩托与喷水推进的设计思想，提出一种利用常规航空动力推进、基于喷水矢量与浸水水翼联合控制实现力系平衡与操纵性的近水面超低空飞行器力学原理模型的总体设计方案。
本发明的技术方案是：包括艇体2、螺旋浆1，驱动装置、水泵5、出水管3、浸水水翼4、和汲水管7，所述浸水水翼4外形为NACA对称翼型，且垂直于水平面布置；浸水水翼4位于艇体2下方且始终处于水面之下；汲水管7、舵机6和水泵5位于浸水水翼4内；浸水水翼4位于艇体2下方且始终处于水面之下；汲水管7、驱动装置和水泵5位于浸水水翼4内；汲水管7一端伸出浸水水翼4远离艇体2的一面，另一端连接水泵5的进水口；水泵5出水口与两根出水管3一端连接；出水管3管体部分位于艇体2内，另一端对称伸出艇体2外且管口均向下；驱动装置用于改变浸水水翼4方向。
本发明进一步的技术方案是，汲水管7、水泵5进水口出水口两端、出水管3 下端同轴安装。
本发明进一步的技术方案是，所述艇体2采用NACA翼型回转体低阻流线型气动外形设计。
本发明进一步的技术方案是，所述驱动装置为舵机6。
本发明进一步的技术方案是，所述艇体2与浸水水翼4比例为3:1。
发明效果
本发明的技术效果在于：提出的掠水面超低空飞行器设计方案的优点主要体现在以下几个方面：(1)完全或主要依靠喷水矢量力用以实现飞行器掠水面飞行时纵平面的力系平衡，可以对地效翼进行优化或变体设计，从而实现飞行器的低阻流线型气动外形设计，提高飞行器的飞行效率；(2)水翼外形的汲水管道，其沾湿控制力较气动舵控制力更加稳定与高效，可以提高飞行器的飞行稳定性与操纵性；(3)喷水矢量控制力与水击力方案，与传统的地效飞行器相比，大大减小了海况影响，飞行器适应性更强，运动更平稳。如海况引起的水面剧烈波动导致地效力难于控制与利用，海浪对低置宽弦比地效翼的冲击威胁等状况，都会因为水击力方案的优越性而避免发生。

具体实施方式
下面结合具体实施实例，对本发明技术方案进一步说明。
1、参见图1到图3，本发明的设计思路在于：掠水面超低空飞行器是由艇体、出水管、浸水水翼、水泵、汲水管、驱动装置与螺旋桨等部分组成。本设计主要分为上部艇体部分和下部浸水水翼部分。艇体部分在水面上，而浸水水翼部分在水中(浸水水翼部分始终在水里，浸水水翼的设计是为了借助水的流体动力为整个航行器提供稳定面和操纵面的作用)。艇体部分与浸水水翼部分比例为3:1,浸水水翼部分设计的较小可以减少飞行器在水下的阻力以提高飞行速度。艇体主体设计为NACA翼型回转体低阻流线型气动外形设计(设计选用NACA翼型回转体是为了使航行器艇体主体部分流体动力特性达到最佳状态)，可以为飞行器飞行减少阻力并产生地效力；出水管包在艇体内，并将两个出水口末端伸出艇体外，两个出水口对称布置(对称布置可使航行器不会产生前后倾覆)在艇体重心前后，作业时出水口向下喷水平衡重力使其达到平衡状态；螺旋桨布置于艇体前部，为飞行器提供前进动力。水泵、汲水管、驱动装置包在浸水水翼内部，浸水水翼竖直于水平面布置，防止飞行器侧翻，具有安定面与控制面作用，浸水水翼设计为流线型(设计为流线型的浸水水翼是为了使航行器流体动力特性达到最佳状态)可以减少飞行器在水下的阻力；水泵的进水口与进水管上端连接，出水口与出水管下端连接；汲水管底部汲水并由水泵增压从出水管喷出；浸水水翼内布置的驱动装置为舵机，在舵机的驱动下可使浸水水翼改变方向，从而可改变航行器运动方向。汲水管、水泵进水口出水口两端、出水管下端同轴安装。本实施例掠水面超低空飞行器由螺旋桨1、艇体2、出水管3、浸水水翼4、水泵5、驱动装置6和汲水管7组成。掠水面超低空飞行器外形结构图见图1，在艇体2前侧与螺旋桨1连接；艇体2与浸水水翼4通过出水管3连接；浸水水翼底部设计有汲水管7.掠水面超低空飞行器剖视图见图2，电机与舵机6用螺钉连接在汲水管7上；汲水管7与水泵5用螺栓连接；汲水管7、驱动装置6、水泵5和出水管4下端都被安装在浸水水翼4内；水泵5与出水管3用螺栓连接；出水管3安装在艇体2内。
本实施例中，飞行器飞行时艇体部分在水面以上，浸水水翼部分在水面以下。水泵5运作通过汲水管道7将水抽入管内，并经过水泵5对水增压将其通过出水管3前后两个出水口喷出。水流高速喷出对水面产生击水力并且对飞行器产生升力。艇体2前侧的螺旋桨1为飞行器运动提供动力。水面以下的浸水水翼4垂直布置于水面可保证飞行器在运动时不会发生侧翻。设置于浸水水翼4中的驱动装置6可以在飞行器运动过程中改变运动方向。
本装置在开始工作时，在水泵5运作下将水通过汲水管道7抽入，并经过水泵5对水增压将其通过出水管3前后两个出水管喷出为航行器提供升力。艇体2前侧的螺旋桨1在电机的驱动下转动为航行器提供动力。水面以下的浸水水翼4在驱动装置6的驱动下改变浸水水翼方向来改变航行器的运动方向。
