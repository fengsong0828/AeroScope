Description

一种用于低空飞行器航行引导的路径规划方法及系统
技术领域
本发明涉及低空飞行器飞行控制技术领域，特别是涉及一种用于低空飞行器航行引导的路径规划方法及系统。
背景技术
低空飞行器路径规划是为低空飞行器规划出一条从起点到终点的最优或安全路径，以确保低空飞行器能够安全、高效地完成任务。在相关低空飞行器路径规划技术中，规划的路径与实际航行路径偏差显著。并且规划路径易贴近障碍物，具有低空飞行器碰撞障碍物的风险。
发明内容
本发明提供一种用于低空飞行器航行引导的路径规划方法及系统，旨在至少解决现有技术中存在的技术问题之一。
本发明的技术方案为一种用于低空飞行器航行引导的路径规划方法，包括：
获取目标区域内低空飞行器的先验飞行数据，对所述先验飞行数据进行预处理；
基于核密度估计，对预处理后的所述先验飞行数据计算概率密度值，对所述概率密度值进行归一化处理，将归一化处理后的所述概率密度值映射到栅格化环境模型的栅格图中，生成飞行线路安全经验分布图；
基于安全距离约束矩阵扩展障碍物区域边界，生成包含安全缓冲区的限制区域矩阵；
根据双向A*-KDE算法代价函数、所述飞行线路安全经验分布图和所述限制区域矩阵生成初始路径；
对所述初始路径去除冗余节点和进行平滑处理，得到飞行航线。

在一个具体的实施例中，获取目标区域的实际地图数据，采用灰度图像处理技术对目标区域的实际地图数据进行处理，生成灰度处理后的地图，再栅格化处理灰度处理后的地图，将栅格化处理后的地图中的障碍物信息映射至栅格化环境模型中。

步骤103：基于安全距离约束矩阵扩展障碍物区域边界，生成包含安全缓冲区的限制区域矩阵。
在一个具体的实施例中，安全距离约束矩阵Ann是一个大小为n×n的矩阵，旨在对航线中的障碍区域进行扩展，形成一个包含安全缓冲区的限制区域矩阵。具体地，通过预设的安全距离dsafe调整安全距离约束矩阵Ann的大小，以确保在节点扩展过程中，仅选择那些与障碍物保持有预设的安全距离的节点进行扩展。安全距离约束矩阵Ann和预设的安全距离dsafe的关系表示为：
Ann,n＝2·dsafe+1，
当dsafe取1时，障碍物周围的栅格单位将不满足安全距离约束。
具体地，目标区域空间栅格化后，形成多个立体方块。障碍物不会被选择作为路径扩展，当dsafe取1时，紧靠障碍物的一个立体方块也不会作为路径选择的节点，使生成的路径和障碍物有一定的距离。
