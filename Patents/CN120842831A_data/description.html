Description

一种适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材及其制 备方法
技术领域
本发明涉及高分子材料技术领域，具体为一种适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材及其制备方法。
背景技术
直升机与固定翼飞机相比，其飞行高度一般在6000m 以下，甚至在超低空 15~30m之间，是一种中低空、低速飞行器。旋翼是直升机的重要部件，它为直升机提供了主要的升力和操纵力。早期直升机旋翼主要采用铝合金、不锈钢、钛合金等金属制造。随着材料技术的不断发展与提高，目前直升机旋翼大多采用玻璃纤维和碳纤维的复合材料。由于直升机机动灵活，可随时垂直起降，因此直升机经常服役于湿/热、干/寒、砂尘/雨淋以及海洋等恶劣环境下。在旋翼高速旋转的情况下，砂石、雨水将会以较大的作用力击打在复合材料表面，造成旋翼表面的磨损，在严重的情况下还会导致复合材料中纤维断裂或界面分层，使材料提前失效。
在直升机旋翼上贴上具有弹性的保护膜可以对砂蚀和雨蚀起到一定的防护作用，聚氨酯弹性体是目前研究较多的弹性膜材，在直升机旋翼的防护中有较大的应用前景，但依旧存在防护效果有限，容易老化和更换频率较高的问题。本发明对现有技术进行改进，赋予聚氨酯弹性体膜材优良的耐磨损、耐雨蚀、抗老化和自愈能力，综合提升保护膜的耐久性，减少保护膜的更换频率。
发明内容
本发明的目的在于提供一种适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材及其制备方法，以解决现有技术中存在的问题。
为了解决上述技术问题，本发明提供如下技术方案：
一种适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材，所述适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材是将烯基芴单体、9-乙烯基咔唑聚合包覆在二氧化硅表面制得改性二氧化硅；将聚氨酯、水合肼反应制得改性聚氨酯；将改性聚氨酯、硼酸酯交联剂、改性二氧化硅、N,N-二甲基甲酰胺混合均匀固化后制得；
所述烯基芴单体是将9,9-二甲基-9H-芴-2-醇、3-丁酰氯反应制得；
所述硼酸酯交联剂是将4-甲酰基苯硼酸、3,4-二羟基丁醛反应制得；
所述聚氨酯是将聚酯二元醇、紫外吸收单体、双羟基封端聚二甲基硅氧烷与异佛尔酮二异氰酸酯聚合，并用N-(4-硝基苯基)二乙醇胺扩链制得；
所述紫外吸收单体是将氰乙酸乙酯、4,4'-二羟甲基二苯甲酮反应制得。
一种适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材的制备方法，所述适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材的制备方法包括以下制备步骤：
（1）按质量比为1:(2~3):(2~3):(0.10~0.12)的比例称取二氧化硅、烯基芴单体、9-乙烯基咔唑、过氧化苯甲酰；将烯基芴单体、9-乙烯基咔唑加入到烯基芴单体质量8~10倍的甲苯中混合均匀，配制成反应液；将过氧化苯甲酰、甲苯按质量比为1:(6~8)混合均匀，配制成引发剂溶液；将二氧化硅、1/4份的反应液、1/4份的引发剂溶液混合，在氮气保护下，70~80℃，200~300r/min搅拌反应20~30min，在30min内匀速滴加剩余的反应液和引发剂溶液，滴加完毕后继续搅拌反应10~12h，过滤，用无水乙醇洗涤3次，在真空条件下，50~60℃干燥13~15h，制得改性二氧化硅；
（2）将4-甲酰基苯硼酸、3,4-二羟基丁醛按摩尔比为1:1加入到4-甲酰基苯硼酸质量13~15倍的二氯甲烷中，加入4-甲酰基苯硼酸质量0.1~0.2倍的无水硫酸镁，在50~60℃，300~400r/min搅拌反应8~9h，过滤，将滤液在真空条件下，40~50℃干燥10~12h，制得硼酸酯交联剂；
（3）将聚酯二元醇、紫外吸收单体、双羟基封端聚二甲基硅氧烷、二月桂酸二丁基锡、间二甲苯按质量比为1:(0.4~0.5):(0.5~0.6):(0.01~0.02):(8~10)混合均匀，在200~300r/min搅拌条件下，以5℃/min的速度升温至138~140℃，并搅拌回流2~3h，降温至78~82℃，加入聚酯二元醇质量2.3~2.5倍的异佛尔酮二异氰酸酯，继续搅拌反应3~4h，加入异佛尔酮二异氰酸酯质量0.2~0.3倍的N-(4-硝基苯基)二乙醇胺，继续搅拌反应30~40min，在真空条件下，80~90℃干燥13~15h，制得聚氨酯；将聚氨酯、钯碳、N,N-二甲基甲酰胺按质量比为1:(0.05~0.06):(10~12)混合均匀，在90~94℃，200~300r/min搅拌20~30min，降温至60~62℃，在20min内匀速滴加聚氨酯质量0.7~0.8倍的水合肼，滴加完毕后升温至90~94℃，继续搅拌反应8~9h，在真空条件下，70~80℃干燥13~15h，制得改性聚氨酯；
（4）将改性聚氨酯、硼酸酯交联剂、改性二氧化硅、N,N-二甲基甲酰胺按质量比为1:(0.07~0.08):(0.04~0.05):(2~2.2)混合均匀，在55~65℃，100~120r/min搅拌10~12min，倒入模具中，在真空条件下，70~80℃干燥13~15h，冷却至室温后脱模，制得适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材。

与现有技术相比，本发明所达到的有益效果是：
本发明在制备适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材时，首先，将9,9-二甲基-9H-芴-2-醇、3-丁酰氯反应制得烯基芴单体；将烯基芴单体、9-乙烯基咔唑聚合包覆在二氧化硅表面制得改性二氧化硅；芴和咔唑分子结构中含有大量的共轭π电子结构，具有转光性能，可以吸收紫外光，将紫外光能转化为荧光的形式发出，从而提升聚氨酯弹性体膜材的抗老化性能。
其次，将4-甲酰基苯硼酸、3,4-二羟基丁醛反应制得硼酸酯交联剂；硼酸酯交联剂上的醛基可以与改性聚氨酯分子侧链上引入的氨基发生席夫碱反应，形成交联网络结构，抑制聚氨酯分子链间的相对滑移，使材料不容易被外界剪切力撕裂，提升了聚氨酯弹性体膜材的耐磨损能力和耐雨蚀性能，硼酸酯交联剂上的硼酸酯键属于动态共价键，硼酸酯键遇水会发生分解反应，去除掉水后又能重新生成硼酸酯键，这种可逆的水解反应赋予了聚氨酯弹性体膜材自修复的能力。
其次，将氰乙酸乙酯、4,4'-二羟甲基二苯甲酮进行脑文格缩合反应制得紫外吸收单体；将聚酯二元醇、紫外吸收单体、双羟基封端聚二甲基硅氧烷与异佛尔酮二异氰酸酯聚合，并用N-(4-硝基苯基)二乙醇胺扩链制得聚氨酯；在聚氨酯分子主链上引入了奥克立林结构和硅氧烷结构，侧链上引入硝基；奥克立林结构属于肉桂酸类化合物，可以吸收紫外光并且将紫外光能转化为热能的形式释放，进一步提升聚氨酯弹性体膜材的抗老化性能；聚氨酯主链上引入的硅氧烷结构使材料憎水性提升，雨滴撞击到材料表面后，润湿效果变差，从而减少了水分子与材料总体的接触面积和接触数量，进而提高了耐雨蚀性能。
最后，将聚氨酯分子侧链上引入的硝基用水合肼还原成氨基，制得改性聚氨酯，改性聚氨酯分子侧链上引入的氨基与硼酸酯交联剂发生席夫碱反应，形成交联网络结构，使材料不容易被外界剪切力撕裂，提升了聚氨酯弹性体膜材的耐磨损能力和耐雨蚀性能。

从表2中实施例1~3和对比例1~6的实验数据比较可发现，本发明制得的适用于低空飞行器旋翼桨叶的聚氨酯弹性体膜材具有良好的自修复和抗老化性能。
通过对比，实施例1~3的自修复率大于对比例2~3的自修复率，说明将4-甲酰基苯硼酸、3,4-二羟基丁醛反应制得硼酸酯交联剂；硼酸酯交联剂上的醛基可以与改性聚氨酯分子侧链上引入的氨基发生席夫碱反应，形成交联网络结构，硼酸酯交联剂上的硼酸酯键属于动态共价键，硼酸酯键遇水会发生分解反应，去除掉水后又能重新生成硼酸酯键，这种可逆的水解反应赋予了聚氨酯弹性体膜材自修复的能力。
通过对比，实施例1~3的自修复率大于对比例6和对比例的自修复率，说明将聚氨酯分子侧链上引入的硝基用水合肼还原成氨基，制得改性聚氨酯，改性聚氨酯分子侧链上引入的氨基与硼酸酯交联剂发生席夫碱反应，形成动态交联网络结构，当聚氨酯侧链上不含有氨基时，即使加了硼酸酯交联剂也不能形成交联网络结构，无法发挥自修复功能。
通过对比，实施例1~3的性能下降率小于对比例1的性能下降率，说明将9,9-二甲基-9H-芴-2-醇、3-丁酰氯反应制得烯基芴单体；将烯基芴单体、9-乙烯基咔唑聚合包覆在二氧化硅表面制得改性二氧化硅；芴和咔唑分子结构中含有大量的共轭π电子结构，具有转光性能，可以吸收紫外光，将紫外光能转化为荧光的形式发出，从而提升聚氨酯弹性体膜材的抗老化性能。
通过对比，实施例1~3的性能下降率小于对比例4的性能下降率，说明将氰乙酸乙酯、4,4'-二羟甲基二苯甲酮进行脑文格缩合反应制得紫外吸收单体；将聚酯二元醇、紫外吸收单体、双羟基封端聚二甲基硅氧烷与异佛尔酮二异氰酸酯聚合，并用N-(4-硝基苯基)二乙醇胺扩链制得聚氨酯；在聚氨酯分子主链上引入了奥克立林结构；奥克立林结构属于肉桂酸类化合物，可以吸收紫外光并且将紫外光能转化为热能的形式释放，进一步提升聚氨酯弹性体膜材的抗老化性能。
