Description

一种基于自适应控制的低空飞行器稳定性增强系统
技术领域
本发明属于飞行器低空控制技术领域，具体的说是一种基于自适应控制的低空飞行器稳定性增强系统。
背景技术
低空飞行器（如无人机、直升机、小型飞机等）在执行任务时，通常面临复杂的飞行环境。与高空飞行相比，低空飞行器更容易受到地形、气流、天气、风速、气流湍流等因素的影响；这些因素往往表现为高频扰动、非线性变化及环境不确定性，严重影响飞行器的飞行稳定性、控制精度以及任务完成效率。在低空飞行过程中，飞行器需要快速应对突发的气流变化、气候条件变化及周围环境的扰动。
传统的低空飞行器控制方法，如PID控制、LQR控制等，通常基于飞行器的线性动力学模型进行设计。这些方法在飞行器受扰动较小、飞行环境较为稳定的情况下能表现出较好的控制效果，但当飞行器遇到复杂的气流扰动、地形变化以及非线性因素时，这些传统方法的控制效果往往难以满足要求。尤其是在复杂、动态、非线性的低空飞行环境中，传统控制方法容易出现反应迟钝、精度低，甚至系统失稳的现象。
因此，随着飞行器技术的不断发展，特别是在低空飞行的应用场景日益增多（如环境监测、农业喷洒、城市空中运输等），如何提升低空飞行器在复杂飞行环境中的稳定性和控制精度，成为了当前飞行器控制领域的研究热点。现有技术多依赖于扰动补偿、模型参考控制等方式来处理低空飞行中的复杂环境，但这些方法仍然存在一定的局限性，难以应对飞行器动态性能的变化和复杂非线性扰动的实时处理。
自适应控制技术能够实时调节控制系统参数，适应系统动态变化和不确定性，在低空飞行器控制中，自适应控制能够根据飞行器在复杂环境中遇到的不同扰动和变化进行参数调整，从而增强系统的鲁棒性和稳定性。然而，现有的自适应控制方法仍然面临诸多挑战，如何快速响应突发扰动、如何处理非线性飞行器动态特性、如何实现多模式飞行器控制等，这些问题亟待解决。
发明内容
针对现有技术的不足，本发明提出了一种基于自适应控制的低空飞行器稳定性增强系统，通过设计自适应控制器来实时调整控制策略，从而提升低空飞行器的稳定性，能够有效应对飞行器在不同飞行阶段、不同环境条件下的动态变化，实现对飞行器在低空环境中的精准控制和高效稳定。
为实现上述目的，本发明提供如下技术方案：
一种基于自适应控制的低空飞行器稳定性增强系统，包括：数据采集模块、数据处理模块、自适应控制模块和执行模块；
所述数据采集模块，用于实时采集低空飞行器的飞行数据，并对采集的低空飞行器的飞行数据进行预处理；
所述数据处理模块，用于接收来自数据采集模块预处理后的低空飞行器的飞行数据，并对其接收的来自数据采集模块的飞行数据进行处理和分析；
所述自适应控制模块，用于运用自适应控制算法实时调整低空飞行器的控制参数，输出控制指令；
所述执行模块，用于接收来自自适应控制模块的控制指令，并将控制指令转化为实际的动作，调整低空飞行器的飞行姿态和轨迹。
具体的，所述数据处理模块包括：扰动感知与分类单元和扰动模型建立单元；
所述扰动感知与分类单元，用于对外部环境扰动进行感知，并对外部扰动进行分类；
所述扰动模型建立单元，用于对飞行器的状态进行估计，并根据外部环境扰动建立扰动源模型，结合飞行器的实时状态，对飞行器飞行风险进行评估。
具体的，所述对外部环境扰动进行感知，并对外部扰动进行分类，包括：
根据飞行器所处环境的风速和风向，结合地面风速数据和飞行器的实时飞行轨迹，使用风场建模技术对飞行器周围的风速分布进行实时计算；
对飞行器前方和侧方的气流进行定向感知；
识别地形障碍物和变化的地貌，感知飞行器周围的地形特征。
具体的，所述对飞行器的状态进行估计，并根据外部环境扰动建立扰动源模型，结合飞行器的实时状态，对飞行器飞行风险进行评估，包括：
基于预处理后的低空飞行器的飞行数据，使用滤波算法对飞行器的动态状态进行计算，得到飞行器的实时状态；
基于风速感知、气流扰动和地形影响，通过分析历史数据和实时监测的扰动信息，对外部扰动源进行实时建模和识别；
基于建立的扰动源模型，结合飞行器的实时状态，采用基于自适应采样的仿真方法评估外部扰动对飞行器造成的影响。
具体的，所述对外部扰动源进行实时建模和识别，包括：
提取扰动数据特征，包括时域特征、频域特征、非线性动态特征和统计学特征；
基于飞行器的实时状态和外部环境的实时数据，采用数据驱动的机器学习方法对扰动源进行建模，得到扰动源模型；
将提取的扰动数据特征输入至扰动源模型中，进行识别与分类。
具体的，所述基于建立的扰动源模型，结合飞行器的实时状态，采用基于自适应采样的仿真方法评估外部扰动对飞行器造成的影响，包括：
对飞行器的飞行状态进行灵敏度分析，评估不同扰动源对飞行器状态的敏感程度；
基于对扰动源的识别和分类，采用基于自适应采样的仿真方法对飞行器在不同环境扰动下的行为进行多场景与多模式仿真，评估外部扰动对飞行器造成的影响范围，所述自适应采样的过程中包含采样数据和校验帧，所述校验帧包括采样帧和数据校验帧；
基于飞行器的状态响应、扰动源的强度与持续时间因素，构建多维度的风险评估指标，包括：失稳概率、能量消耗增量和操控性降低指数；
根据评估结果，通过实时计算的风险评估指标，自动触发相应的警告或应急措施。
具体的，所述自适应控制模块，包括：模型与期望输出单元、非线性控制与补偿单元和多模式自适应控制单元；
所述模型与期望输出单元，用于建立飞行器的运动模型，并基于需求设计期望输出；
所述非线性控制与补偿单元，用于对飞行器的非线性特征进行控制和补偿；
所述多模式自适应控制单元，用于在不同飞行条件和模式情况下，自适应控制飞行器。
具体的，所述建立飞行器的运动模型，并基于需求设计期望输出，包括：
建立飞行器的线性和非线性运动方程，通过飞行器的质量、惯性矩、空气动力学特性等参数，推导出飞行器的运动模型；
定义容忍范围，根据不同的飞行任务设计期望输出。
具体的，所述在不同飞行条件和模式情况下，自适应控制飞行器，包括：
通过比较飞行器的实际状态和模型的期望输出，计算控制误差，并对控制误差的动态变化进行分析；
采用自适应增益调节算法，实时调整控制器增益，并自适应修正误差；
基于预处理后的低空飞行器的飞行数据，实时检测飞行器当前的飞行模式，根据不同模式下的飞行任务，自动切换适合的控制策略。
与现有技术相比，本发明的有益效果是：
1.本发明提出一种基于自适应控制的低空飞行器稳定性增强系统，自适应控制能够根据飞行器的实时状态以及外部扰动动态调整控制参数，确保飞行器在复杂、动态、非线性的低空飞行环境中保持稳定性。
2.本发明提出一种基于自适应控制的低空飞行器稳定性增强系统，通过综合多种传感器数据，实时识别外部扰动的类型、强度以及可能的影响，能够根据扰动源的变化，自动调整控制策略，补偿飞行器因气流扰动、地形影响等因素造成的偏差，使飞行器在整个飞行过程中始终处于稳定的控制状态。
3.本发明提出一种基于自适应控制的低空飞行器稳定性增强系统，通过非线性控制和补偿，设计了适应飞行器非线性动态特性的控制策略，处理飞行器在遇到强气流扰动或快速机动时的非线性响应，此外，能够根据飞行任务的不同，自动切换飞行模式（如巡航、机动、下降等），确保飞行器在各种飞行阶段下都能够保持良好的稳定性和控制精度。

使用采样帧可以有效节省系统资源，减少数据传输次数，使用数据校验帧对飞行阶段的校验功能极大地增强了容错性，若因外界干扰、传感器故障等原因导致飞行阶段判断失误，校验帧可及时发现并纠正，避免后续采样策略的错误执行。
在本实施例中，本申请提出了自适应采样方法，通过动态调整采样策略，提高仿真效率并增强扰动模拟的精度，包括：重要性采样，根据扰动源的概率分布和飞行器的动态响应模型，采用重要性采样方法提高高风险扰动（例如风速剧烈变化、气流湍流等）发生时的采样频率，通过加大对高影响扰动源的采样密度，避免低概率但高风险的情况被忽略，确保仿真结果的代表性；自适应重采样，在每一轮仿真过程中，根据前一轮仿真得到的扰动影响评估结果，实时调整采样策略，如果某些扰动源的影响较大，加大对扰动源的采样，反之，如果某些扰动源对飞行器影响较小，则会降低其采样密度，通过自适应调整，仿真效率得到了显著提高，且能够在局部“高风险区域”提供更为精准的模拟。
多场景与多模式仿真包括：多飞行场景模拟，模拟飞行器在多种飞行环境和任务条件下的表现，包括城市区域、山区、森林区域等复杂地形，以及不同气象条件（如大风、暴雨、雷电等），每种场景下，飞行器所受扰动的性质和强度不同，因此需要分别进行仿真，并对每种环境下的飞行器响应进行统计分析；任务模式切换仿真，飞行器的任务模式通常包括爬升、巡航、下降、机动等多个阶段，每个阶段的飞行环境和扰动特性都有所不同，通过设定不同的任务模式和飞行阶段，能够在不同的飞行状态下进行扰动影响预测和评估，确保仿真覆盖各种飞行情况；极端场景仿真，为保障飞行安全，设计极端扰动场景仿真，例如，在飞行器遇到强气流、突发雷暴或极端风速时的表现，虽然发生概率较低，但对飞行器造成较大的威胁，因此需要进行充分的仿真，评估飞行器的应对能力。
步骤B33：基于飞行器的状态响应、扰动源的强度与持续时间等因素，构建一套多维度的风险评估指标，包括：
失稳概率：根据扰动对飞行器的影响，计算飞行器在特定环境下失稳或偏离轨迹的概率；
能量消耗增量：评估扰动对飞行器能量消耗的影响，尤其是在高强度气流变化时飞行器需要的额外功率；
操控性降低指数：评估扰动对飞行器控制系统精度和响应速度的影响，给出操控性下降的量化指标；
步骤B34：根据评估结果，通过实时计算的风险评估指标，自动触发相应的警告或应急措施。
当风险高时，飞行器可根据控制策略调整飞行路径或高度，避免进入高风险区域；当风险达到预设阈值时，系统会自动调整飞行器的控制参数，或建议飞行员采取应急措施。
所述自适应控制模块，用于运用自适应控制算法实时调整飞行器的控制参数，输出控制指令；
所述自适应控制模块，包括：模型与期望输出单元、非线性控制与补偿单元和多模式自适应控制单元；
所述模型与期望输出单元，用于建立飞行器的运动模型，并基于需求设计期望输出；
所述建立飞行器的运动模型，并基于需求设计期望输出，包括：
建立飞行器的线性和非线性运动方程，通过飞行器的质量、惯性矩、空气动力学特性等参数，推导出飞行器的运动模型；
在本实施例中，在低空飞行时，考虑到气流干扰、地形效应等因素，模型的设计要能够应对高度变化、飞行速度和姿态的剧烈变化；
定义一个容忍范围，根据不同的飞行任务设计期望输出。
例如，在巡航飞行模式下，模型的输出应保持平稳的飞行状态；在执行机动飞行或应急避障任务时，模型则应提供更加灵活的姿态控制需求，通过设定不同飞行模式的参考轨迹，控制器能够自适应地调整飞行器的飞行行为，容忍范围，用于处理实际飞行过程中出现的小幅度偏差，保证系统的鲁棒性和适应性。
所述非线性控制与补偿单元，用于对飞行器的非线性特征进行控制和补偿；
在低空飞行中，飞行器的动力学模型常常呈现明显的非线性特征，如升力和攻角的非线性关系、风速与飞行轨迹的非线性影响等，非线性因素需要通过专门的非线性补偿策略进行处理，常见的非线性补偿方法是通过反馈线性化技术，将飞行器的非线性动力学模型通过适当的控制输入进行线性化处理，采用滑模控制技术对飞行器的控制误差进行补偿或神经网络作为补偿；
所述多模式自适应控制单元，用于在不同飞行条件和模式情况下，自适应控制飞行器。
所述在不同飞行条件和模式情况下，自适应控制飞行器，包括：
通过比较飞行器的实际状态和模型的期望输出，计算控制误差，并对控制误差的动态变化进行分析；
例如，可以计算位置误差、速度误差、姿态误差等多个维度的误差，以综合评估飞行器的偏差；
采用自适应增益调节算法，如L1自适应控制、增益调度方法等，实时调整控制器增益，并自适应修正误差，确保飞行器在受到扰动或动态环境变化时依然能够快速响应；
例如，在强烈横风或气流变化下，系统会增加俯仰角控制增益，提升飞行器的稳定性；而在平稳飞行时，控制增益则可以适当降低，以减少能量消耗；
当检测到较大的误差或不稳定趋势时，自适应机制会启动修正过程。该过程通过调整飞行器的控制指令（如舵面角度、推力等），以逐步减小误差并恢复飞行器到期望状态，在遇到突发性扰动时，系统会迅速调整控制策略，以最小化扰动带来的不良影响。
基于预处理后的低空飞行器的飞行数据，实时检测飞行器当前的飞行模式，根据不同模式下的飞行任务，自动切换适合的控制策略，输出控制指令。
例如，在飞行器执行复杂机动任务时，控制器会激活高增益控制模式，而在巡航过程中，则采用较低增益的稳定控制模式，在突发气流扰动下，控制器需要自动调整飞行路径，以保持飞行稳定；在应急避障任务中，飞行器的控制器会迅速进入较为激进的机动模式，以避开障碍物。
所述执行模块，用于接收来自自适应控制模块的控制指令，并将控制指令转化为实际的动作，调整低空飞行器的飞行姿态和轨迹。
