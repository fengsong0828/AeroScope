Description

一种基于多源数据融合的低空飞行器安全评估方法及系统
技术领域
本发明涉及飞行安全技术领域，特别是涉及一种基于多源数据融合的低空飞行器安全评估方法及系统。
背景技术
随着低空经济的迅猛崛起，无人机物流、城市空中交通（UAM）、应急救援、农业植保等应用场景正从概念加速走向现实。低空空域，这片曾经静谧的区域，正演变为一个高度动态、密集、复杂的数字化新兴领域。然而，低空飞行器，特别是无人驾驶飞行器，其大规模商业化应用面临的最核心瓶颈，即是安全问题。
但是，传统的安全评估方法已显得力不从心，因为这些方法往往依赖于单一或少量数据源（如GPS位置）和预设的静态规则（如地理围栏），其本质是被动式的响应和告警，缺乏对潜在风险的预测能力，这无法满足低空飞行器对实时性、自主性和高可靠性的苛刻安全要求。
发明内容
为了解决上述技术问题，本发明提供了一种基于多源数据融合的低空飞行器安全评估方法及系统，包括：
获取低空飞行器的实时飞行数据，并对实时飞行数据进行分析，确定与飞行器安全相关的关键飞行参数；
确定各关键飞行参数的数据特征，基于各关键飞行参数及数据特征对低空飞行器的安全状态进行评估，得到飞行安全状态评估值；
获取历史飞行环境数据，并基于历史飞行环境数据和预设神经网络模型构建飞行环境预测模型进行预测，得到飞行环境预测数据；
分析飞行环境预测数据，确定环境状态系数，并确定环境对飞行安全的影响系数，基于环境状态系数和影响系数确定环境影响系数；
基于环境影响系数对飞行安全状态评估值进行修正，得到综合安全评估值，并基于综合安全评估值确定低空飞行器的安全等级。
进一步的，所述获取低空飞行器的实时飞行数据，并对实时飞行数据进行分析，确定与飞行器安全相关的关键飞行参数，包括：
获取低空飞行器的实时飞行数据，并按照参数类型将实时飞行数据划分为多个飞行参数数据组；
分别确定各飞行参数数据组与飞行器安全的相关度，并将相关度大于预设相关度阈值的飞行参数数据组对应的参数确定为与飞行器安全相关的关键飞行参数。
进一步的，所述确定各关键飞行参数的数据特征，基于各关键飞行参数及数据特征对低空飞行器的安全状态进行评估，得到飞行安全状态评估值，包括：
确定各关键飞行参数对应的飞行参数数据组，并基于各飞行参数数据组分别构建时间进度的数据变化曲线；
确定各数据变化曲线中预设数据统计指标的数值，并将超出对应预设阈值的预设数据统计指标选取出来，确定为各关键飞行参数的数据特征；
计算各数据特征的数值与对应预设阈值之间的差值，并对该差值进行评估取值，得到各数据特征的偏差评估值；
对各关键飞行参数对应的所有数据特征的偏差评估值进行标准化处理，并将标准化处理后的偏差评估值进行相加计算，得到各关键飞行参数的综合偏差评估值；
确定各关键飞行参数的预设权重，并基于各关键飞行参数的预设权重和综合偏差评估值进行计算，得到低空飞行器的飞行安全状态评估值。

进一步的，所述获取历史飞行环境数据，并基于历史飞行环境数据和预设神经网络模型构建飞行环境预测模型进行预测，得到飞行环境预测数据，包括：
获取历史飞行环境数据，历史飞行环境数据包括历史气象数据组、历史空域交通数据组和历史地理数据组，并提取各数据组的数据特征；
基于各数据组和对应关键数据特征分别构建数据集，并将数据集输入至对应的预设神经网络模型中，分别构建对应的预测初始模型；
将各数据集按照预设比例划分为训练集和测试集，并将训练集和测试集输入至对应的预测初始模型中；
对各预测初始模型进行训练和测试，直至各预测初始模型满足预设的收敛条件，得到各预测模型；
将各预测模型进行融合，得到飞行环境预测模型，并获取实时飞行环境数据，将实时飞行环境数据输入至飞行环境预测模型中进行预测，得到未来一段时间内的飞行环境预测数据。
进一步的，所述分析飞行环境预测数据，确定环境状态系数，包括：
从飞行环境预测数据中确定气象预测数据组、空域交通预测数据组和地理预测数据组；
确定气象预测数据组中的各气象参数数据组，并计算各气象参数数据组的平均值；
确定各气象参数数据组的基准平均值，并计算各气象参数数据组的平均值与对应基准平均值的差值，得到各气象参数数据组的数据偏差；
确定各气象参数数据组的预设系数，将各气象参数数据组的数据偏差与预设系数进行相乘计算，得到气象预测数据组的第一状态系数；
基于空域交通预测数据组确定预测入侵目标，并确定各预测入侵目标与该低空飞行器之间预测轨迹的最接近点距离和到达最接近点时间；
基于最接近点距离和最接近点时间进行计算，得到各预测入侵目标与该低空飞行器之间的子安全系数，子安全系数的计算公式为：
R=α*(1/A)+β*(1/B)，
其中，R为子安全系数，α为第一预设权重，A为最接近点距离，β为第二预设权重，B为最接近点时间；
将各预测入侵目标与该低空飞行器之间的子安全系数进行相加计算，得到空域交通预测数据组的第二状态系数；
基于地理预测数据组确定障碍物，并计算该低空飞行器与各障碍物之间的距离；
确定该低空飞行器与各障碍物之间距离中的最大距离，并将最大距离与预设安全转换系数进行相乘计算，得到地理预测数据组的第三状态系数；
对气象预测数据组的第一状态系数、空域交通预测数据组的第二状态系数和地理预测数据组的第三状态系数分别进行归一化处理，并将归一化处理的结果进行相加计算，得到环境状态系数。
进一步的，所述确定环境对飞行安全的影响系数，包括：
确定历史飞行环境数据和历史飞行安全数据，并按照参数类型将历史飞行环境数据划分为多个历史飞行环境参数数据组；
针对第i个历史飞行环境参数数据组，确定在其他飞行环境参数相同时的历史飞行安全数据，并确定第i个历史飞行环境参数数据组的第一变化量以及对应历史飞行安全数据的第二变化量；
计算第二变化量与第一变化量的比值，得到各飞行环境参数对飞行安全的子影响系数，并将各飞行环境参数对飞行安全的子影响系数进行相加计算，得到环境对飞行安全的影响系数。
进一步的，所述基于环境状态系数和影响系数确定环境影响系数，包括：
将环境状态系数与影响系数进行相乘计算，得到环境影响系数。
进一步的，所述基于环境影响系数对飞行安全状态评估值进行修正，得到综合安全评估值，并基于综合安全评估值确定低空飞行器的安全等级，包括：
将环境影响系数与飞行安全状态评估值进行相乘计算，得到低空飞行器的综合安全评估值；
预先设定预设安全等级-综合安全评估值区间对应关系，预设安全等级-综合安全评估值区间对应关系针对每一综合安全评估值区间，均关联有对应的预设安全等级；
获取低空飞行器的综合安全评估值，并基于该综合安全评估值所属的综合安全评估值区间在预设安全等级-综合安全评估值区间对应关系内的映射关系，选取与综合安全评估值区间对应的预设安全等级确定为低空飞行器的安全等级。
本发明还提供了一种基于多源数据融合的低空飞行器安全评估系统，包括：
获取模块，用于获取低空飞行器的实时飞行数据，并对实时飞行数据进行分析，确定与飞行器安全相关的关键飞行参数；
评估模块，用于确定各关键飞行参数的数据特征，基于各关键飞行参数及数据特征对低空飞行器的安全状态进行评估，得到飞行安全状态评估值；
预测模块，用于获取历史飞行环境数据，并基于历史飞行环境数据和预设神经网络模型构建飞行环境预测模型进行预测，得到飞行环境预测数据；
计算模块，用于分析飞行环境预测数据，确定环境状态系数，并确定环境对飞行安全的影响系数，基于环境状态系数和影响系数确定环境影响系数；
确定模块，用于基于环境影响系数对飞行安全状态评估值进行修正，得到综合安全评估值，并基于综合安全评估值确定低空飞行器的安全等级。
本发明实施例一种基于多源数据融合的低空飞行器安全评估方法及系统与现有技术相比，其有益效果在于：
本发明通过获取低空飞行器的实时飞行数据并对其进行分析，确定各关键飞行参数的数据特征并进行分析，建立起这些参数之间的关联性和规律性，可以更准确地评估飞行器的状态，并为后续的安全评估提供基础；
本发明通过获取低空飞行器的实时飞行数据并对其进行分析，确定各关键飞行参数的数据特征，并基于关键飞行参数及其数据特征对飞行器的安全状态进行评估，得到一个综合的飞行安全状态评估值，可以更准确地评估飞行器的状态，及时了解飞行器的安全状况；
本发明利用历史飞行环境数据和预设的神经网络模型构建飞行环境预测模型，可以预测未来的飞行环境状况，并确定环境状态系数和环境对飞行安全的影响系数，量化环境因素对飞行安全的影响程度；
本发明基于环境影响系数对飞行安全状态评估值进行修正，得到综合的安全评估值，可以更全面地考虑环境因素对飞行安全的影响，可以更准确地评估飞行器的整体安全性；
综上所述，本发明通过融合了实时飞行数据分析和环境预测数据，并通过飞行安全状态评估、确定环境影响系数和安全等级评定等多个技术手段来进行低空飞行器安全状态的评估，全面提高了对低空飞行器安全状态的监测和评估的能力，有助于精确的确定飞行器的安全状态，以提升飞行器的飞行安全水平，降低飞行风险。

具体而言，将代表飞行器内部健康状态的飞行安全状态评估值与代表外部环境实际威胁的环境影响系数进行相乘计算；预置一个预设安全等级-综合安全评估值区间对应关系查询表，这是一个将连续的综合安全评估值离散化为几个明确安全等级的规则；例如：[0.9,1.0] ->安全（绿色），[0.7,0.9] ->注意（黄色），[0.3,0.7] ->警告（橙色），[0.0,0.3] ->危险（红色），计算出综合安全评估值后，将其与这个查询表进行比对，确定其所属的区间，并最终输出该区间所关联的预设安全等级。该步骤实现内外部风险的深度融合与精准量化，避免了简单加权平均可能带来的风险掩盖问题，确保了任何一方存在高风险都会在最终结果中得到显著体现，使得评估结果极其敏感和可靠，能够真实反映“一个特定健康状况的飞行器，在一个特定环境下面临的整体风险”；最终输出的不是一个需要进一步解读的复杂数字，而是一个明确的、国际通用的安全等级标签，降低了人机交互的认知负荷，无论是飞行员、地面站操作员还是自动化系统，都能在瞬间理解当前的安全态势，并触发预设的标准化响应流程，为应对紧急情况节省了宝贵时间。
