Description

一种低空飞行器监测用控制结构
技术领域
本实用新型属于飞行器技术领域，特别是涉及一种低空飞行器监测用控制结构。
背景技术
无人机是指不完全依靠人为控制并能够执行相对任务的空中飞行器。无人机属于低空飞行器的一种，其飞行方式表现为：自主控制飞行姿态，在手动飞行、地面电台控制或者自主记忆情况下来完成所需飞行的任务。
在现有技术中，低空飞行器在进行飞行时不能够很好的对其进行监测控制，在长时间飞行后可能会产生高温，导致其损坏，因此，本申请提出了一种低空飞行器监测用控制结构来解决上述问题。
实用新型内容
本实用新型的目的在于提供一种低空飞行器监测用控制结构，通过温度监测触发装置和散热控制装置，解决了现有的问题。
为解决上述技术问题，本实用新型是通过以下技术方案实现的：
本实用新型为一种低空飞行器监测用控制结构，包括飞行器主体，所述飞行器主体的底部固定连接有支撑架，所述飞行器主体的顶部固定且电性连接有控制模块，所述控制模块上设置有温度监测触发装置和散热控制装置；
所述温度监测触发装置包括连接槽，所述连接槽的内部固定连接有导热块，所述导热块的顶部固定连接有受热膨胀气囊，所述受热膨胀气囊的顶部固定连接有连接杆，所述连接杆的顶部固定连接有推板，所述推板的上方设置有电源，所述电源靠近推板的一侧固定连接有限位块。
进一步地，所述控制模块的顶部开设有连接槽，所述控制模块的顶部固定连接有导热块，控制模块能够有效控制飞行器主体进行飞行。
进一步地，所述受热膨胀气囊、连接杆和推板均设置在限位块的内侧，受热膨胀气囊在吸收热量后会进行膨胀。
进一步地，所述限位块远离电源的一侧与控制模块的顶部固定连接，所述限位块的内部设置为空心状态，限位块能够有效对受热膨胀气囊进行限位。
进一步地，所述散热控制装置包括连接线，所述连接线的侧边固定且电性连接有电机，所述电机的侧边转动且电性连接有电动转杆，所述电动转杆远离电机的一侧固定连接有散热扇，所述电机远离电动转杆的一侧固定连接有保护壳，所述保护壳上开设有散热槽，散热控制装置能够有效对控制模块进行散热降温处理，提高控制模块的控制性能。
进一步地，所述连接线远离电机的一侧与电源固定且电性连接，所述电源远离限位块的一侧与保护壳的内壁固定连接，连接线能够有效的将电源内的电流导入到电机的内部。
进一步地，所述散热扇设置在控制模块的两侧，所述散热槽设置在控制模块没有散热扇的两侧，散热扇能够有效对控制模块进行散热。
本实用新型具有以下有益效果：
1、本实用新型通过温度监测触发装置，能够在飞行器主体长时间飞行后，控制模块会产生较高的热量，导热块会将热量导入到受热膨胀气囊处，受热膨胀气囊在吸收热量后会进行膨胀，在受热膨胀气囊膨胀的同时会带动连接杆和推板进行运动，推板在运动到固定位置后会挤压电源的开关，开启散热控制装置，能够有效对控制模块的温度进行监测，实用性较强。
2、本实用新型通过散热控制装置，能够在电源开启后，通过连接线将电源内部的电流输送到电机内部，电机会带动电动转杆和散热扇进行转动，散热扇能够有效对控制模块进行散热降温，能够有效对控制模块进行保护，防止飞行器主体损坏，进一步加强了整个装置的实用性。
当然，实施本实用新型的任一产品并不一定需要同时达到以上所述的所有优点。

请参阅图1-5所示，本实用新型为一种低空飞行器监测用控制结构，包括飞行器主体1，飞行器主体1的底部固定连接有支撑架2，飞行器主体1的顶部固定且电性连接有控制模块3，控制模块3上设置有温度监测触发装置4和散热控制装置5；
温度监测触发装置4包括连接槽41，连接槽41的内部固定连接有导热块42，导热块42的顶部固定连接有受热膨胀气囊43，受热膨胀气囊43的顶部固定连接有连接杆44，连接杆44的顶部固定连接有推板45，推板45的上方设置有电源46，电源46靠近推板45的一侧固定连接有限位块47。
控制模块3的顶部开设有连接槽41，控制模块3的顶部固定连接有导热块42，控制模块3能够有效控制飞行器主体1进行飞行。
受热膨胀气囊43、连接杆44和推板45均设置在限位块47的内侧，受热膨胀气囊43在吸收热量后会进行膨胀。
限位块47远离电源46的一侧与控制模块3的顶部固定连接，限位块47的内部设置为空心状态，限位块47能够有效对受热膨胀气囊43进行限位。
散热控制装置5包括连接线51，连接线51的侧边固定且电性连接有电机52，电机52的侧边转动且电性连接有电动转杆53，电动转杆53远离电机52的一侧固定连接有散热扇54，电机52远离电动转杆53的一侧固定连接有保护壳55，保护壳55上开设有散热槽56，散热控制装置5能够有效对控制模块3进行散热降温处理，提高控制模块3的控制性能。
连接线51远离电机52的一侧与电源46固定且电性连接，电源46远离限位块47的一侧与保护壳55的内壁固定连接，连接线51能够有效的将电源46内的电流导入到电机52的内部。
散热扇54设置在控制模块3的两侧，散热槽56设置在控制模块3没有散热扇54的两侧，散热扇54能够有效对控制模块3进行散热。
本实用新型通过温度监测触发装置4，能够在飞行器主体1长时间飞行后，控制模块3会产生较高的热量，导热块42会将热量导入到受热膨胀气囊43处，受热膨胀气囊43在吸收热量后会进行膨胀，在受热膨胀气囊43膨胀的同时会带动连接杆44和推板45进行运动，推板45在运动到固定位置后会挤压电源46的开关，开启散热控制装置5，能够有效对控制模块3的温度进行监测，实用性较强。本实用新型通过散热控制装置5，能够在电源46开启后，通过连接线51将电源46内部的电流输送到电机52内部，电机52会带动电动转杆53和散热扇54进行转动，散热扇54能够有效对控制模块3进行散热降温，能够有效对控制模块3进行保护，防止飞行器主体1损坏，进一步加强了整个装置的实用性。
本实施例的一个具体应用为：在飞行器主体1长时间飞行后，控制模块3会产生较高的温度，导热块42会将控制模块3处的热量导入到受热膨胀气囊43的内部，受热膨胀气囊43在吸收热量过后会进行膨胀，在受热膨胀气囊43膨胀的同时会带动连接杆44和推板45进行运动，推板45会挤压电源46的开关，电源46会开启，开启散热控制装置5，能够有效对控制模块3处的热量进行监测，在控制模块3处的温度处于正常状态后，导热块42不会继续传导热量，受热膨胀气囊43内部的气体在没有高温的驱动下也会恢复原状，受热膨胀气囊43也会恢复原状，连接杆44和推板45也会恢复原位，电源46的开关不会继续被挤压，电源46会关闭，便于下一次进行使用，实用性较强，在电源46开启后，连接线51会将电源46处的电流输送到电机52的内部，电机52会带动电动转杆53和散热扇54进行转动，散热扇54能够有效对控制模块3进行散热处理，加强控制模块3的控制性能，在控制模块3处的温度恢复正常后，电源46会关闭，电机52也会关闭，散热扇54会停止转动，便于下一次继续进行使用，进一步加强了整个装置的实用性。
