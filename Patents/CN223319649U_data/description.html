Description

一种低空飞行器防砂、耐腐组合换热结构
技术领域
本实用新型涉及航空飞机及发动机配套换热技术领域，特别是一种低空飞行器防砂、耐腐组合换热结构。
背景技术
在航空领域，飞行器在对流层中飞行时必须应对各种恶劣的环境条件，其中包括高浓度的砂尘和颗粒物。这些粉状物质有可能以高速撞击飞行器的关键组件，如空气换热装置。传统的解决方案是采用铝合金材料，并通过表面钝化、阳极氧化和喷漆等方式来提高其耐腐蚀性能。然而，这些防护措施在遭受机械冲击(如砂尘和颗粒物的撞击)后可能会受损，从而导致铝合金基材暴露于腐蚀环境中，进而发生点蚀等腐蚀现象。
若采用钛合金或不锈钢替代铝合金，虽然其具有优异的耐腐蚀性能，但这些材料的密度较大，从而会显著增加飞行器的重量。对于航空器而言，重量的增加将直接导致燃料消耗的增加以及运营成本的上升。因此，寻求一种既能抵抗极端环境下的腐蚀，又能保持较低重量的材料或技术方案是当前航空领域的一个重要研究方向。
实用新型内容
本实用新型的目的是：提供了一种低空飞行器防砂、耐腐组合换热结构。本实用新型能够能够防止航空飞行器在飞行过程中砂尘、颗粒物高速进入换热装置，导致换热装置翅片、隔板等表面碰伤。
本实用新型的技术方案是：一种低空飞行器防砂、耐腐组合换热结构，包括铝合金散热芯体，铝合金散热芯体的迎风端连接有钛合金散热芯体，铝合金散热芯体的迎风侧翅片A的流道与钛合金散热芯体迎风侧翅片B的流道对应连通。
前述的低空飞行器防砂、耐腐组合换热结构中，所述的迎风侧翅片A与迎风侧翅片B经铜扩散焊层连接，或者所述的迎风侧翅片A与迎风侧翅片B由3D打印构成一体式结构。
前述的低空飞行器防砂、耐腐组合换热结构中，所述的铜扩散焊层的厚度为0.7～1mm。
前述的低空飞行器防砂、耐腐组合换热结构中，所述的迎风侧翅片B表面附有陶瓷涂层。
前述的低空飞行器防砂、耐腐组合换热结构中，所述的迎风侧翅片B沿空气流动方向呈波纹结构。
前述的低空飞行器防砂、耐腐组合换热结构中，所述的迎风侧翅片A在空气流动方向上的长度为70～80mm。
本实用新型的优点是：
本实用新型通过在铝合金散热芯体的迎风端经铜扩散焊层或3D打印连接钛合金散热芯体，成功实现了两种散热芯体结构之间的无缝对接。该结构，保护了铝合金免受高速冲压风中的砂砾等颗粒物的侵蚀，这些颗粒物有可能破坏铝合金表面的钝化层、阳极化层及漆层。通过结合钛合金散热芯体的卓越防腐性能，本实用新型显著提升了换热装置的防腐能力，延长了设备的使用寿命，并提高了在恶劣环境下的工作稳定性。通过在铝合金散热芯体前增加钛合金散热芯体，形成一道防护屏障，拦截高速气流中的颗粒物，避免它们直接接触到铝合金表面，从而保护铝合金不受磨损和腐蚀。利用钛合金散热芯体的天然防腐特性和铝合金散热芯体的热传导优势，构建了一个高性能的复合换热系统。这不仅保证了优良的防腐性能，同时也确保了高效的热量传递。通过铜扩散焊层连接两个散热芯体，使得铝合金与钛合金之间实现微观上的冶金结合，既牢固又可靠，为极端环境下的换热应用提供了新的解决方案。本实用新型通过巧妙的设计和先进的制造工艺，提供了一种高性能、长寿命的换热装置，具有重要的工程应用价值和市场前景。
本实用新型的迎风侧翅片B采用了独特的波纹翅片设计，波纹形状的翅片能够有效地将高速湍流的空气转换为速度较低的稳定气流，这对于保护铝合金翅片至关重要。同时波纹形状有助于改变气流方向，从而降低气流速度，减少气流对换热装置的冲击，尤其在高速冲压风的情况下，这种设计能有效减少颗粒物的携带和沉积。颗粒物拦截：当含有砂砾等颗粒的气流通过波纹翅片时，颗粒会因为碰撞而附着在钛合金翅片(迎风侧翅片B)上，而不是继续前进到后面的铝合金部分，这大大减少了对铝合金翅片的磨损和腐蚀。易于清洁：由于颗粒物主要集中在钛合金翅片上，这使得清洁和维护变得更加集中和简便，进而提高了整个换热装置的维护效率。
本实用新型中，钛合金的迎风侧翅片B具备更高的耐温性能，能够承受高达450℃的温度，而铝合金的迎风侧翅片A耐温能力一般在250℃以下。此外，钛合金的密度为4.5g/cm3，而铝合金的密度较低，为2.7g/cm3。基于这些物理特性，本实用新型构成了分级换热的模型，即首先利用钛合金散热芯体处理高温空气，然后再将其导入铝合金散热芯体进行后续冷却。温度管理：通过先使用钛合金散热芯体处理高温空气，可以有效地将空气温度降至铝合金能够承受的范围内，这样不仅保护了铝合金芯体免受过高的温度损伤，同时也提高了整体换热系统的安全性和效率。重量优化：由于钛合金和铝合金的密度不同，本实用新型能通过调整钛合金芯体的长度，在保证耐温性能的同时，显著减轻产品的总重量。这对于移动式或航空领域的应用尤为重要，因为它可以减少能耗并提高机动性。因此，本实用新型实现了高效且重量优化的换热方案。这种设计不仅考虑到了材料的耐温性能，还兼顾了产品的实用性和便携性，进一步提升了产品在多种应用场景下的性能表现
本实用新型在钛合金翅片的表面添加陶瓷涂层，这一改进大大提升了翅片的耐冲击性和防腐性能。陶瓷涂层具有较高的硬度和抗冲击性，因此涂层可以有效抵抗高速气流携带的砂砾对翅片的冲击，减少翅片损伤，延长其使用寿命。陶瓷涂层不活泼，不易与其他物质发生化学反应，这可以进一步增强钛合金的防腐蚀性能，尤其是在恶劣的工业或自然环境中，这一特性显得尤为重要。陶瓷涂层具有良好的耐磨性，这意味着即使在长时间使用后，也不容易出现磨损，从而降低了维护频率和成本。因此，本实用新型不仅提高了钛合金翅片的物理性能，还进一步拓宽了该换热装置的应用领域，特别是在需要应对极端环境挑战的场合，如化工、航空航天等领域，这一改进有望成为关键的技术突破
本实用新型可基于空气流速对波纹翅片进行优化设计，使得随空气进入的砂砾在迎风侧翅片B的入口处滞留，不再进入芯体的内部，防止它们进一步侵热装置的核心部件，即芯体。由于颗粒物主要集中在入口处，这使得清洁和维护工作变得更加集中和简便。在产品维护过程中，仅需要清理该端面即可，无需深入到换热装置的内部进行清理。通过减少颗粒物对芯体的磨损和堵塞，间接地延长了换热装置的整体使用寿命。因此本实用新型通过一个简单而有效的设计改进，不仅提高了换热装置的清洁效率，还大幅延长了设备的使用寿命。这对于保障设备长期运行效率和减少运营成本具有重要价值，尤其适用于需要频繁清洁和维护的环境。
传统铝合金和钛合金之间的连接通常需要通过添加过渡段的方式来实现，这往往会导致产品体积和重量的增加。本实用新型经铜扩散焊层连接或3D打印一体式结构，实现了两者的直接连接，省去了不必要的过渡段，从而有效地减小了整个换热装置的体积和重量，这对于便携式设备或者对重量有严格要求的应用场景尤其重要。减少了材料的使用，同时也就减少了热能在这部分材料中的损耗，使得整体换热效率得到提升。
综上所述，本实用新型提供了一种低空飞行器防砂、耐腐组合换热结构，适用于多种工业和民用领域。同时，通过优化结构，提高了材料的利用率，降低了生产成本。

参考图1，一种低空飞行器防砂、耐腐组合换热结构包括：铝合金散热芯体1、钛合金散热芯体2、铜扩散焊层5组成。其中，铝合金散热芯体1包含由迎风侧翅片A3，钛合金散热芯体2包含迎风侧翅片B4。
钛合金散热芯体2采用了高性能的钛合金材料，这种材料强度、韧性和耐腐蚀性高。钛合金散热芯体2迎风侧翅片B4可以优化空气动力学性能并提高散热效率。翅片的角度通过精确的计算和风速得出。通过调节风速与翅片角度的配合，可以减少砂尘等污染物进入后续结构，保护后续结构表面处理层不受损伤，延长整个设备的使用寿命。
迎风侧翅片B4设置陶瓷涂层增加翅片表面的硬度，提高其抗冲击和耐磨损能力。陶瓷涂层可以减小由于高速气流带来的动态载荷对翅片结构的损害，确保散热器在长期运行过程中的稳定性和可靠性。
在一些实施例中，为提升产品的耐高温特性，把钛合金替换为高温合金或不锈钢材料。高温合金或不锈钢材料具有更高的热稳定性和抗氧化性，在高温环境下展现出优异的机械性能和耐腐蚀性能。通过这种材料的转换，产品能够承受更高的温度，确保在极端热环境中保持结构完整性和功能性。
同时，针对不同位置的组件可能面临不同的温度条件，后段铝合金等材料被安排在温度较低的环境中工作。既充分利用了铝合金等材料良好的热传导性能和较轻的重量，又避免了因过高的温度而导致性能下降的风险，提高整体系统的效能和可靠性。通过智能地选择和分配不同性能的材料，本实用新型实现了对产品耐高温能力的有效提升，并且确保了各个部分在各自的工作环境中都能发挥最佳效能。
