Description

一种低空飞行器用降落伞快展安全装置
技术领域
本发明涉及低空飞行器降落伞展开装置技术领域，具体为一种低空飞行器用降落伞快展安全装置。
背景技术
低空飞行器为人造飞行器，飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物，其中无人机为低空飞行器的一种，小型无人机在降落的时候可以弹出降落伞用于辅助降落，降落伞通过固定在无人机上的展开装置对降落伞进行展开：
在中国专利申请号为202411087206.2的一种低空飞行器专用降落伞安全装置，此发明包括有外壳，所述外壳内安装有高度感应器，所述外壳内固接有周向均匀分布的伞绳壳体，周向均匀分布的所述伞绳壳体共同通过伞绳固接有位于所述外壳内的降落伞，所述外壳上设置有对称分布的腔体，所述外壳的腔体内转动连接有转轴，所述转轴上固接并绕设有第一柔性绳，所述第一柔性绳绕过滑轮组并固接有重块，所述外壳内固接有对称分布的电磁块，所述外壳内固接有对称分布的限位架。本发明通过重块夹紧并带动降落伞抛出，模拟人手抛出降落伞的动作，加快降落伞打开的速度，从而增加低空飞行器坠落时的缓冲力度，进而更好的保护飞行器与人体安。
现今低空飞行器降落伞展开装置的降落伞和连接降落伞的绳子直接安装在降落伞展开装置外壳的内部，比较杂乱，安装的时候不方便，且降落伞在展开的时候绳子容易缠绕，展开较慢，影响降落伞的展开的速度。
发明内容
本发明的目的在于提供一种低空飞行器用降落伞快展安全装置，以解决上述背景技术中提出的问题。
为实现上述目的，本发明提供如下技术方案：
一种低空飞行器用降落伞快展安全装置，包括底座、安装筒、第一安装板和降落伞，所述底座通过螺丝固定安装于无人机机身顶部，所述安装筒通过螺丝固定安装于所述底座顶端，所述第一安装板通过螺丝固定安装于所述安装筒顶端，所述降落伞设置于所述安装筒上方；
限位机构；所述限位机构设置于所述第一安装板上，所述限位机构通过其中的支撑框和移动框对所述降落伞上的连接绳进行限位；
若干个收集机构，所述收集机构设置于所述安装筒内壁底端，所述收集机构通过其中的收集框对所述降落伞上的连接绳进行收纳；
若干个辅助展开机构，所述辅助展开机构设置于所述安装筒内壁底端，所述辅助展开机构通过其中的喷头辅助所述降落伞展开；
所述限位机构包括安装架、支撑框、滑槽、移动框、安装块、挡板、若干个第一弹簧、若干转动架、滚轮、固定孔、第一转动槽、内齿环、若干个连杆、固定柱、卡接件和卡板，所述安装架固定安装于所述第一安装板顶端，所述支撑框通过扭簧转动安装于所述安装架一侧，所述滑槽开设于所述支撑框两侧，所述安装块固定安装于所述移动框内壁两侧，所述安装块滑动安装于对应所述滑槽槽壁，所述移动框通过其内部的所述安装块滑动安装于所述支撑框外壁，所述挡板固定安装于所述移动框内壁，若干所述第一弹簧固定安装于所述挡板一侧，所述第一弹簧另一端固定安装于所述支撑框一端，若干所述转动架分别固定安装于所述支撑框和移动框内壁，所述滚轮转动安装于所述转动架外壁。
可选的，所述固定孔开设于所述支撑框一侧，所述第一转动槽开设于所述第一安装板顶端，所述第一转动槽呈环形，所述内齿环底部转动安装于所述第一转动槽槽壁，若干所述连杆固定安装于所述内齿环外壁，所述固定柱固定安装于所述连杆一侧，所述固定柱于对应所述固定孔相卡合，所述卡接件固定安装于所述移动框一侧，所述卡板固定安装于所述第一安装板顶端，且所述卡板与所述卡接件位置相对应，所述卡接件与对应所述卡板相卡接。
可选的，所述第一安装板顶端开设有若干第一开口，所述第一安装板顶端通过螺丝固定安装有第二安装板，所述第二安装板顶端开设有第二开口，所述第一开口与所述第二开口位置相对应，所述安装架位于对应所述第一开口相对应。
可选的，所述第一安装板顶端固定安装有固定架，所述固定架顶端固定安装有第一电机，所述第一电机输出端延伸至所述固定架内部，所述第一电机输出端固定安装有齿轮，所述齿轮与所述内齿环内壁的齿相啮合。
可选的，所述收集机构包括若干收集框、转动杆、第一锥齿轮、固定筒、连接柱、第二锥齿轮和旋钮，若干所述收集框固定安装于所述安装筒内壁底端，所述转动杆转动安装于所述收集框内壁相对的两侧，所述转动杆一端延伸至所述收集框一侧外部，所述第一锥齿轮固定安装于所述转动杆延伸至所述收集框外部的一端，所述固定筒固定安装于所述第一安装板顶端中心，所述连接柱转动安装于所述固定筒内壁，所述第二锥齿轮固定安装于所述连接柱底端，所述旋钮固定安装于所述连接柱顶端，所述第二锥齿轮与所述第一锥齿轮相啮合，所述收集框位置于所述第一开口相对应。
可选的，所述连接绳一端固定安装于对应所述转动杆外壁，所述连接绳另一端穿过所述支撑框和移动框延伸至所述支撑框一侧外部，所述降落伞固定安装于所述连接绳延伸至所述支撑框一侧外部的一端。
可选的，所述固定筒内壁开设有第二转动槽，所述第二转动槽呈环形，所述第二转动槽槽壁转动安装有安装环，所述安装环一端固定安装有若干移动柱，所述连接柱外壁开设有若干移动槽，所述移动柱滑动安装于对应所述移动槽槽壁，所述连接柱通过其上的移动槽滑动安装于所述移动柱上，所述安装环一端固定安装有若干第二弹簧，所述第二弹簧另一端固定安装于所述连接柱外壁。
可选的，所述第二安装板顶端边缘位置固定安装有转动框，所述转动框一侧固定安装有转动电机，所述转动电机输出端延伸至所述转动框内部，所述转动框内壁转动安装有上盖，所述上盖通过外壁的凸块转动安装于所述转动框内壁，且所述上盖外壁的凸块固定安装于所述转动电机输出端。
可选的，所述辅助展开机构包括若干个固定座、高压气罐、固定环、电磁阀、连接管和若干个喷头，若干所述固定座固定安装于所述安装筒内壁底端，所述固定环固定安装于所述高压气罐外壁，所述高压气罐设置于所述固定座内壁，所述固定环通过螺丝固定安装于所述固定座顶端，所述安装筒内壁固定安装有若干支架，所述电磁阀固定安装于对应所述支架上，若干所述喷头固定安装于所述第一安装板顶端，且所述喷头底端延伸至所述第一安装板底部，所述电磁阀一端通过管道与对应所述高压气罐相连接，所述连接管固定安装于对应所述电磁阀一端，所述连接管另一端固定安装于对应所述喷头底端。
可选的，所述第二安装板顶端开设有若干个通孔，所述喷头顶端穿过对应所述通孔延伸至所述第二安装板顶端。
本发明至少具备以下有益效果：
(1)本方案通过设置限位机构，在降落伞展开的时候，内齿环转动，通过连杆带动固定柱脱离支撑框一侧的固定孔，支撑框于安装架连接处设置的扭簧回弹，使支撑框从平行与第一安装板端面的状态逆时针转动，直到支撑框一侧被第一开口内壁挡住，支撑框带动内部的连接绳调整角度，直到移动框上的卡接件卡在对应的卡板上，使几个连接绳之间张开，使连接绳可以更好的张开，减少连接绳缠绕的情况，使降落伞可以更快的张开；
(2)本方案在安装连接绳和降落伞时，转动连接柱带动第二锥齿轮转动，第二锥齿轮转动带动第一锥齿轮转动，第一锥齿轮转动带动与其相连接的转动杆转动，使连接绳缠绕在转动杆上，方便对连接绳进行安装和收纳；
(3)本方案在用手按压连接柱，可使第二锥齿轮啮合在第一锥齿轮上，可以对连接绳进行收纳，松开对连接柱的按压的时，第二弹簧回弹支撑连接柱向上移动，使第二锥齿轮与第一锥齿轮脱离，可以减少转动杆转动时受到的阻力，使连接绳顺利展开；
(4)本方案通过设置高压气罐，在上盖打开的时候，电磁阀同步打开，通过喷头朝向降落伞处喷出高压气流，可以辅助降落伞更好的进行展开。
