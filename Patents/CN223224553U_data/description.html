Description

一种收纳机构及低空飞行器
技术领域
本实用新型涉及低空飞行器技术领域，特别是一种收纳机构及低空飞行器。
背景技术
无人驾驶飞机简称“无人机”(“UAV”)，是利用无线电遥控设备和自备的程序控制装置操纵的不载人飞行器，无人机实际上是无人驾驶飞行器的统称，从技术角度定义可以分为：无人固定翼飞机、无人垂直起降飞机、无人飞艇、无人直升机、无人多旋翼飞行器、无人伞翼机等，其中无人多旋翼飞行器最为常见，而部分无人多旋翼飞行器用于竞技飞行，此举对无人多旋翼飞行器的发展颇有益处。
在无人机竞赛中，速度和效率是取得胜利的关键因素之一。竞赛飞机由于需要长途飞行，需要落地进行电池的更换，同时在落地时需要尽量节省更换电池的时间，因此，设计一种能够实现快速电池更换的无人机显得尤为重要。实用新型内容
本部分的目的在于概述本实用新型的实施例的一些方面以及简要介绍一些较佳实施例。在本部分以及本申请的说明书摘要和实用新型名称中可能会做些简化或省略以避免使本部分、说明书摘要和实用新型名称的目的模糊，而这种简化或省略不能用于限制本实用新型的范围。
鉴于上述现有技术中存在需要能够快速拆卸电池的低空飞行器，提出了本实用新型。
为解决上述技术问题，本实用新型提供如下技术方案：一种收纳机构，其包括承载壳；以及，支撑组件，包括有设置于所述承载壳底部的横架、设置于所述横架底部的套筒以及设置于所述套筒内壁的支脚；以及，电池仓组件，包括有设置于所述承载壳侧面的容置仓、设置于所述容置仓内壁的摩擦板、设置于所述容置仓底部的撑板以及设置于所述撑板顶部的配重块。
作为本实用新型收纳机构的一种优选方案，其中：所述支脚的外壁设置有滑筒，所述套筒的外壁开设有滑槽，所述滑槽的内壁设置有连杆，所述连杆的一端与所述滑筒相连接，所述连杆远离所述滑筒的一端设置有集中盘，所述集中盘的顶部设置有顶柱。
作为本实用新型收纳机构的一种优选方案，其中：所述支脚的顶部设置有垫片，所述垫片的顶部与所述横架相连接。
作为本实用新型收纳机构的一种优选方案，其中：所述顶柱的轴线与所述撑板的质心位于同一铅垂线上，所述顶柱的顶部与所述撑板的底部相连接。
作为本实用新型收纳机构的一种优选方案，其中：所述配重块的顶部开设有弧面，所述弧面的侧面设置有绞绳，所述绞绳的侧面设置有第一导向轮，所述第一导向轮的侧面设置有第一转架。
作为本实用新型收纳机构的一种优选方案，其中：所述第一转架的顶部设置有第二转架，所述第二转架的侧面设置有第二导向轮，所述绞绳位于所述第二导向轮上方的一端设置有卡柱，所述第二导向轮的顶部母线与所述卡柱的轴线处于同一水平线。
作为本实用新型收纳机构的一种优选方案，其中：所述卡柱的侧面设置有滑片，所述滑片的侧面设置有第二弹性件，所述第二弹性件的内部设置有滑杆。
作为本实用新型收纳机构的一种优选方案，其中：所述滑片的侧面设置有仓盖构件。
作为本实用新型收纳机构的一种优选方案，其中：所述仓盖构件包括设置于所述容置仓顶部的翻盖，所述翻盖的底部设置有锁块，所述锁块的侧面开设有供所述卡柱滑动的槽；
其中，所述翻盖的顶部设置有弧形板，所述翻盖与所述容置仓连接处设置有扭转弹簧。
提供了一种低空飞行器，包括收纳机构，还包括飞控系统、电机以及螺旋桨。
本实用新型的收纳机构的有益效果为：在飞机降落时自动打开仓盖构件，方便取下电池进行更换，同时利用地面对低空飞行器的冲击力移动配重块，可以减少降落时对低空飞行器电池的冲击。

实施例1，参照图1～图3，为本实用新型第一个实施例，该实施例提供了一种收纳机构，包括承载壳100；以及，支撑组件200，包括有设置于承载壳100底部的横架201、设置于横架201底部的套筒202以及设置于套筒202内壁的支脚203，低空飞行器降落时，支脚203与地面接触，会受到地面的反作用力；以及，电池仓组件300，包括有设置于承载壳100侧面的容置仓301、设置于容置仓301内壁的摩擦板304、设置于容置仓301底部的撑板302以及设置于撑板302顶部的配重块303，电池存放于容置仓301中，电池仓组件300在低空飞行器飞行时，对电池进行保护，低空飞行器降落时，支撑组件200会将地面传来的反作用力传递到配重块303，利用配重块303开启电池仓组件300，使得操作人员能够快速拆卸电池，节省时间。
使用过程：电池仓组件300在低空飞行器飞行时，对电池进行保护，低空飞行器降落时，支脚203与地面接触，会受到地面的反作用力，支撑组件200会将地面传来的反作用力传递到配重块303，利用配重块303开启电池仓组件300，使得操作人员能够快速拆卸电池，节省时间，同时将地面对飞机施加的冲量部分转化为开启电池仓组件300所需的能量，加强缓冲作用，更好的保护飞机。
实施例2，参照图1～图5，为本实用新型第二个实施例，与上一个实施例不同的是，支脚203的外壁设置有滑筒205，套筒202的外壁开设有滑槽206，滑槽206的内壁设置有连杆207，连杆207可以在滑槽206内滑动，连杆207的一端与滑筒205相连接，滑筒205对套筒202与外界起到封闭阻隔作用，防止套筒202里的零件被污染，连杆207远离滑筒205的一端设置有集中盘208，集中盘208的顶部设置有顶柱209，支脚203的顶部设置有垫片204，垫片204的顶部与横架201相连接，顶柱209的轴线与撑板302的质心位于同一铅垂线上，顶柱209的顶部与撑板302的底部相连接，当低空飞行器降落，低空飞行器的速度在支脚203接触地面时变为零，此时低空飞行器收到地面的冲击力，支脚203会向上移动，带动滑筒205以及连杆207向上移动，进而带动集中盘208上移，最终会带动撑板302向上移动。
进一步的，配重块303的顶部开设有弧面305，弧面305的侧面设置有绞绳306，绞绳306的侧面设置有第一导向轮307，第一导向轮307的侧面设置有第一转架308，，第一导向轮307引导者绞绳306向上延伸，第一转架308的顶部设置有第二转架309，第二转架309的侧面设置有第二导向轮310，绞绳306位于第二导向轮310上方的一端设置有卡柱311，第二导向轮310的顶部母线与卡柱311的轴线处于同一水平线，在低空飞行器落地时，地面的冲击力带动撑板302向上移动后，撑板302会带动配重块303上移，配重块303会带着底端的绞绳306上移，于是与卡柱311连接的绞绳306端会向着第二转架309的方向移动。
进一步的，卡柱311的侧面设置有滑片312，滑片312的侧面设置有第二弹性件314，第二弹性件314的内部设置有滑杆313，滑片312上开设有与滑杆313外径相同的槽，滑片312可以和卡柱311一同沿着滑杆313方向滑动，当与卡柱311连接的绞绳306端会向着第二转架309的方向移动，会带动卡柱311一同滑动。
进一步的，滑片312的侧面设置有仓盖构件315，仓盖构件315包括设置于容置仓301顶部的翻盖315a，翻盖315a的底部设置有锁块315b，锁块315b的侧面开设有供卡柱311滑动的槽，当卡柱311处于锁块315b侧面的槽中时，翻盖315a被锁住，对容置仓301中的电池起到保护作用。
使用过程：在低空飞行器正常飞行过程中，翻盖315a盖在容置仓301上方，锁块315b被卡柱311锁住，无法打开，对电池起到保护作用，当低空飞行器降落，低空飞行器的速度在支脚203接触地面时变为零，此时低空飞行器收到地面的冲击力，支脚203会向上移动，带动滑筒205以及连杆207向上移动，进而带动集中盘208上移，最终会带动撑板302向上移动，撑板302会带动配重块303上移，配重块303会带着底端的绞绳306上移，于是与卡柱311连接的绞绳306端会向着第二转架309的方向移动，带动卡柱311一同滑动，脱离锁块315b侧面的槽，此时翻盖315a解锁，操作人员可以轻松打开翻盖315a，取出电池，对电池进行更换，节省时间，而且利用地面对低空飞行器的冲击力移动配重块303，可以减少降落时对电池的冲击，更好的保护电池。
实施例3，参照图1～图5，为本实用新型第三个实施例，与上一个实施例不同的是，翻盖315a的顶部设置有弧形板315c，翻盖315a与容置仓301连接处设置有扭转弹簧315d，当低空飞行器落地翻板被解锁，弧形板315c能方便工作人员打开翻盖315a，当电池更换完成后，放下翻盖315a，扭转弹簧315d会使翻盖315a自动回位，。
使用过程：当电池更换完成，翻盖315a在扭转弹簧315d的作用下回到容置仓301上方，由于在容置仓301的内壁设置有摩擦板304，配重块303会相对于低空飞行器缓慢下移，此时绞绳306被放松，于是第二弹性件314会将滑片312与卡柱311一同朝着锁块315b滑动，直至卡柱311伸入锁块315b中，将翻盖315a锁住，在低空飞行器飞行中继续保护电池。
实施例4，参照图1～图5，为本实用新型第四个实施例，该实施例基于上一个实施例，不同的是提供一种低空飞行器，包括收纳机构，还包括飞控系统400、电机500以及螺旋桨600。
飞控系统400接收地面遥控信号并控制飞行，电机500为螺旋桨600提供动力带动低空飞行器飞行。
