Description

具有散热结构的驱动系统和低空飞行器
技术领域
本实用新型涉及低空飞行设备技术领域，具体而言，涉及一种具有散热结构的驱动系统和低空飞行器。
背景技术
随着低空经济的热度越来越高，电动垂直起降飞行器(eVTOL)因其安全、智能、经济、环保等特点而广受大众的青睐，电动垂直起降飞行器(eVTOL)使得人和货物能够以无缝且经济的方式在城市的低空快速流动以及灵活作业，有利于快速开发城市的低空空域资源，使得电动垂直起降飞行器(eVTOL)逐渐成为城市空中交通的主流发展方向。
现有的电动垂直起降飞行器(eVTOL)主要采用电机直驱，即，通过外转子带动螺旋桨旋转，从而驱动电动垂直起降飞行器(eVTOL)进行作业，这种采用电机直驱方式的电动垂直起降飞行器(eVTOL)存在以下技术问题：电机的定子和/或电机内部的轴承没有润滑油的润滑和冷却，导致轴承长期使用过程中，轴承只能通过消耗自身的润滑脂来实现润滑，由于轴承自身的润滑脂的消耗导致轴承的使用寿命降低，严重影响电动垂直起降飞行器(eVTOL)的正常作业，存在飞行失效的风险。
现有的部分电动垂直起降飞行器(eVTOL)通过设备内部的润滑油来实现对定子和轴承的润滑和冷却，但是，上述经过换热后的润滑油往往处于高温状态，无法有效带走定子和/或轴承上的热量。
实用新型内容
本实用新型的主要目的在于提供一种具有散热结构的驱动系统和低空飞行器，以解决现有技术中经过换热后的润滑油往往处于高温状态，无法有效带走定子和/或轴承上的热量的问题。
为了实现上述目的，根据本实用新型的一个方面，提供了一种具有散热结构的驱动系统，用于与低空飞行器的螺旋桨驱动连接，驱动系统包括电机壳体、电机组件、泵体结构和散热结构，其中，电机壳体具有容纳腔以及与容纳腔连通的第一避让过孔；电机壳体还具有通油油道；电机组件包括驱动部和传动部，驱动部的至少部分设置在容纳腔内，驱动部与传动部驱动连接，传动部远离驱动部的一端穿过第一避让过孔以用于与螺旋桨驱动连接；泵体结构设置在电机壳体背离第一避让过孔的一端，泵体结构具有相连通的储油腔和回油腔，储油腔与通油油道的油道入口连通；其中，位于第一避让过孔内的传动部的部分外周面与第一避让过孔的部分孔壁面之间形成第一环形空间，第一环形空间用于容纳第一轴承结构，通油油道的第一油道出口与第一环形空间连通，以对第一轴承结构进行润滑和冷却，第一环形空间与回油腔连通；和/或，驱动部包括定子结构，容纳腔具有第一子环腔，定子结构位于第一子环腔内，通油油道的第二油道出口与第一子环腔连通，以对定子结构进行冷却，第一子环腔与回油腔连通；散热结构设置在泵体结构背离电机壳体的轴向一端，以用于对储油腔和/或回油腔内的润滑油进行散热。
进一步地，散热结构与泵体结构可拆卸地连接。
进一步地，电机壳体与电机组件同心设置；和/或，电机组件与泵体结构同心设置；和/或，散热结构与泵体结构同心设置。
进一步地，散热结构包括散热壳体，散热壳体具有容纳槽，且容纳槽的槽口朝向泵体结构一侧，以使容纳槽的槽壁面与泵体结构的转子泵壳体背离电机壳体一端的端部表面之间形成安装空间；驱动系统还包括电机控制器，电机控制器设置在安装空间内，并与驱动部控制连接。
进一步地，散热壳体背离泵体结构一端的端面上具有第一安装区域和第二安装区域，且第二安装区域位于第一安装区域的外周侧；散热结构还包括风扇和散热翅片，其中，风扇可转动地设置在第一安装区域处；散热翅片为多个，多个散热翅片呈放射状设置在第二安装区域处。
进一步地，风扇的扇叶的数量小于散热翅片的数量。
进一步地，散热翅片在散热壳体的径向上的长度大于风扇的扇叶在散热壳体的径向上的长度。
进一步地，驱动部包括电机驱动轴、转子结构和定子结构，其中，电机驱动轴的部分可转动地设置在容纳腔内，电机驱动轴的另一部分伸入第一避让过孔内设置；转子结构套设在电机驱动轴的外周侧；定子结构套设在转子结构的外周侧，定子结构具有定子间隙，定子间隙与第一子环腔连通。
进一步地，电机壳体背离第一避让过孔的一端开设有第二避让过孔；泵体结构包括转子泵壳体和转子泵驱动轴，其中，转子泵壳体具有储油腔和回油腔，转子泵壳体设置在电机壳体背离第一避让过孔的一端，转子泵壳体背离电机壳体的一端具有第三避让过孔；转子泵驱动轴包括相连接的装配轴段和传动轴段，且装配轴段的轴径大于传动轴段的轴径，以使装配轴段与传动轴段的连接处形成止挡环面，装配轴段伸入电机驱动轴的中心孔内以与电机驱动轴连接，且止挡环面和电机驱动轴背离法兰连接轴一端的端面平齐；第一安装区域与电机控制器的第四避让过孔相对的位置处开设有第五避让过孔；传动轴段背离装配轴段的一端依次穿过第二避让过孔、第三避让过孔、第四避让过孔、第五避让过孔并与风扇驱动连接。
进一步地，电机壳体的外周面设置有多个第一散热片，多个第一散热片呈放射状绕电机壳体的周向间隔设置；和/或，泵体结构的转子泵壳体的外周面设置有多个第二散热片，多个第二散热片呈放射状绕转子泵壳体的周向间隔设置。
根据本实用新型的另一方面，提供了一种低空飞行器，包括驱动系统和螺旋桨，驱动系统与螺旋桨驱动连接，驱动系统为上述的驱动系统。
应用本实用新型的技术方案，提供了一种具有散热结构的驱动系统，驱动系统包括电机壳体、电机组件、泵体结构和散热结构，其中，电机壳体具有容纳腔以及与容纳腔连通的第一避让过孔，电机壳体还具有通油油道；电机组件包括驱动部和传动部，驱动部的至少部分设置在容纳腔内，驱动部与传动部驱动连接，传动部远离驱动部的一端穿过第一避让过孔以用于与螺旋桨驱动连接；泵体结构设置在电机壳体背离第一避让过孔的一端，泵体结构具有相连通的储油腔和回油腔，储油腔与通油油道的油道入口连通；散热结构设置在泵体结构背离电机壳体的轴向一端，以用于对储油腔和/或回油腔内的润滑油进行散热。
进一步地，位于第一避让过孔内的传动部的部分外周面与第一避让过孔的部分孔壁面之间形成第一环形空间，第一环形空间用于容纳第一轴承结构，通油油道的第一油道出口与第一环形空间连通，以对第一轴承结构进行润滑和冷却，第一环形空间与回油腔连通；和/或，驱动部包括定子结构，容纳腔具有第一子环腔，定子结构位于第一子环腔内，通油油道的第二油道出口与第一子环腔连通，以对定子结构进行冷却，第一子环腔与回油腔连通。
本申请提供的驱动系统，通过设置泵体结构，同时在电机壳体设置通油油道，使得泵体结构可以将储油腔内的润滑油通过通油油道泵送至第一油道出口处，使得润滑油流入第一环形空间内，从而实现对第一环形空间内的第一轴承结构进行润滑和冷却，最后经过润滑和换热后的润滑油再流回至回油腔内。
当然，泵体结构还可以将储油腔内的润滑油通过通油油道泵送至第二油道出口处，使得润滑油流入第一子环腔内，从而实现对第一子环腔内的定子结构进行冷却的目的，最后经过换热后的润滑油再流回至回油腔内。
此外，本申请通过设置散热结构，散热结构起到了对储油腔和/或回油腔内的润滑油进行散热的目的，从而确保储油腔和/或回油腔内的润滑油能够始终保持预设的温度，以确保其的润滑和换热效果。
