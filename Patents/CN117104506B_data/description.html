Description

一种土壤修复用修复剂低空喷洒飞行器
技术领域
本发明涉及土壤修复技术领域，具体为一种土壤修复用修复剂低空喷洒飞行器。
背景技术
土壤修复是使遭受污染的土壤恢复正常功能的技术措施。在土壤修复行业，已有的土壤修复技术达到一百多种，常用技术也有十多种，大致可分为物理、化学和生物三种方法。在土壤修复中，利用喷洒修复剂的方式对目标土壤进行处理为常用方法，喷洒方法有人工式、车载式等，近年新兴的喷洒方式还有利用无人机飞行器进行修复剂的喷洒，飞行器喷洒效率极高，为近年经常采用的方式，采用的无人机为专用的药剂喷洒无人机，但是目前的这类无人机普遍存在药剂喷洒均匀度不足的问题；对此，授权公告号为CN113548184A的中国发明专利公开了一种用于植物保护的无人机药剂喷洒装置，包括框体、喷头、调节机构、旋转机构、供液系统和支撑机构，调节机构设置于框体中，用于在无人机喷洒的过程中调整喷头位置；旋转机构安装于调节机构上，用于对喷头转动角度进行调整，该无人机药剂喷洒装置通过设置调节机构以及旋转机构在对药剂进行喷洒时，能够根据无人机飞行的高度不断的对喷头的摆动角度以及旋转幅度进行调整，保证均匀度，但是在喷洒方面仍有以下不足：
这类无人机飞行器在进行土壤修复剂喷洒工作时，会出现修复液不断晃动，导致液面不平的情况，这样就会影响无人机飞行器的重心位置，进而导致出现无人机颠簸的情况，影响飞行安全；并且土壤表面大部分情况下会出现高低差，这就导致喷洒高度与土壤表面距离发生改变，仅依靠调整角度的方式，无法解决喷洒距离不一致的问题，还是会导致喷洒不均的问题。
为此我们提出一种土壤修复用修复剂低空喷洒飞行器用于解决上述问题。
发明内容
本发明的目的在于提供一种土壤修复用修复剂低空喷洒飞行器，以解决上述背景技术中提出的问题。
为实现上述目的，本发明提供如下技术方案：一种土壤修复用修复剂低空喷洒飞行器，包括飞行器结构，所述飞行器结构包括机体，所述机体周侧水平固接四个翼臂，所述翼臂远离机体一端设置有旋翼，所述机体底面固接机壳，四个所述翼臂底面固接四个支臂，所述支臂远离机体一端底面固接高度调节组件，所述高度调节组件底面固接喷头结构，所述支臂底面固接接头，所述接头连通波纹软管一端，所述波纹软管另一端连通喷头结构；

与现有技术相比，本发明的有益效果是：
本发明在液箱内设置了稳面结构，稳面结构内的棘齿弹片与棘轮相互配合，使得锁紧齿轮只能单向转动，这样就可以使得锁紧板只能在液箱内部向下运动，无法向上，在喷洒过程中，液箱内修复剂不断减少，此时锁紧板及浮板均贴合在修复剂液面上，浮板会阻止液面出现晃动，保证了整体重心，避免在飞行时出现颠簸的情况，保证安全飞行；本发明在喷头结构上设置了高度调节组件，可以调节喷洒的高度位置，配合激光测距仪进行调整，保证喷洒时喷头结构与地面距离始终一致，保证喷洒均匀度。
