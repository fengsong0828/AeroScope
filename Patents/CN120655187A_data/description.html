Description

基于低空飞行器的物资配送规划方法、装置及存储介质
技术领域
本发明涉及物资配送技术领域，尤其涉及一种基于低空飞行器的物资配送规划方法、装置及存储介质。
背景技术
近年来，低空飞行器的技术快速发展，尤其是在定位、导航、躲避障碍物和续航能力等方面取得了显著进展。低空飞行器的灵活性、速度和高效性，使其在紧急情况下能够迅速抵达灾后区域，直接向需要救助的受灾群众投送物资，极大提高了救援效率。此外，通过无人机进行空中配送，不仅能够绕过地面交通障碍，还能有效减轻人力资源的压力。
中国专利公开号CN110428136A公开了一种灾后城镇物资协同配送方案生成方法，该方法包括以下步骤：S1：确定城镇受灾点信息；所述受灾点信息包括受灾点编号、受灾点空间位置和对物资A的需求量信息；S2：获取本地储备点信息；S3：计算城镇所有受灾点对物资A的总需求量、本地所有储备点对特定物资A的总储备量；S4：计算城镇受灾点受本地储备点救援配送的平均满足率；S5：制定本地救援配送方案；S6：制定外部协同救援配送方案；S7：综合得到灾后城镇物资协同配送总体方案。通过采用本发明方法，可对城镇应急管理部门快速调度应急资源实施灾后救援以及请求外部协同救援合作，提供有效的辅助决策依据；由此可见，该方案在对配送方案进行分析时，仅针对物资需求进行分析，存在物资配送效率低的问题。
发明内容
本发明的目的在于提供一种基于低空飞行器的物资配送规划方法、装置及存储介质，以解决现有技术存在的问题中的至少一个。
为实现上述目的，本发明采用了如下技术方案：
一种基于低空飞行器的物资配送规划方法，包括:
根据配送区域人数、配送区域的物资需求与配送区域类型计算配送区域的优先系数；
对充电站与配送点的距离进行计算，并根据计算结果、配送区域的优先系数和低空飞行器续航对配送区域与低空飞行器进行匹配；
根据获取的环境温度和空气密度分析配送环境状态，并根据配送环境状态处理配送区域与低空飞行器的匹配过程，根据低空飞行器电池充放电循环次处理配送环境状态的分析过程；
根据监测周期内低空飞行器的等待时间与配送环境状态对下一监测周期低空飞行器的飞行速度进行管理。
可选地，获取配送区域信息、低空飞行器信息与环境信息；
构建配送区域坐标、配送点坐标与充电站坐标；
根据配送区域坐标与配送点坐标划分配送区域类型；
将配送点坐标设为(0,0)，将第i个配送区域坐标设为(xi,yi)，将充电站坐标设为(xc,yc)；根据第i个配送区域坐标(xi,yi)与配送点坐标(0,0)对第i个配送区域和配送点的距离Li进行计算；
根据第i个配送区域和配送点的距离Li与预设距离L0对配送区域类型进行划分，其中：
若Li≤L0，判定第i个配送区域类型为一类区域，反之，判定第i个配送区域类型为二类区域。
可选地，将第i个配送区域的优先系数设为Yi，设定w1为人数权重，w2为需求权重，w1+w2＝1，mi为第i个配送区域人数，qi为第i个配送区域的物资需求，Si为第i个配送区域的类型系数。
可选地，充电站与配送点的距离的表达式如下：J＝(xc2+yc2)0.5；式中，J为充电站与配送点的距离；
将配送区域的优先系数进行降序排序，并将排序第一的配送区域作为待选区域，将待选区域与配电点的距离Ly与第k个低空飞行器续航Hk对待选区域和第k个低空飞行器进行匹配，其中：
若Ly≤α×Hk/2，将待选区域与该低空飞行器的匹配指数设为Pk1，设定Pk1＝1-ln{|α×Hk/2-Ly|/(α×Hk/2)+1}；
若Ly＞α×Hk/2且α×Hk/2＞J，将待选区域与该低空飞行器的匹配指数设为Pk2，设定Pk2＝1-β；
若Ly＞α×Hk/2且α×Hk/2≤J，将待选区域与该低空飞行器的匹配指数设为Pk3，设定Pk3＝0；其中，α为预设调整系数。
可选地，将待选区域与各低空飞行器的匹配系数进行降序排序，并将待选区域与低空飞行器匹配系数最大的低空飞行器作为匹配飞行器，若待选区域与低空飞行器匹配系数的最大值为0，则将排序第二的配送区域作为待选区域，重复上述过程，直至待选区域与低空飞行器匹配系数的最大值不为0。
可选地，根据获取的环境温度t0与预设温度t1对温度状态进行分析，若t0＜t1，判定温度状态为异常，并将温度异常系数设为WY1，设定WY1＝(t1-t0)/t1；若t0≥t1，判定温度状态为正常，并将温度异常系数设为WY2，设定WY2＝0；
根据获取的空气密度d0与预设空气密度d1对空气密度状态进行分析，若d0＜d1，判定空气密度状态为异常，并将空气密度异常系数设为KY1，设定KY1＝exp[3×(d1-d0)/d1-3]；若d0≥d1，判定空气密度状态为正常，并将空气密度异常系数设为KY2，设定KY2＝0；
根据温度异常系数与空气密度异常系数构建环境状态指数HZ，设定HZ＝r1×温度异常系数+r2×空气密度异常系数，r1为温度权重，r2为空气密度权重，r1+r2＝1；
根据环境状态指数HZ与预设状态指数HZ0对配送环境状态进行分析，若HZ≤HZ0，判定配送环境状态为正常；若HZ＞HZ0，判定配送环境状态为异常。
在配送环境状态为异常时对待选区域与低空飞行器的匹配过程进行处理，将处理后的预设调整系数设为α1。
可选地，根据第k个低空飞行器电池充放电循环次数fk0与预设次数f1对第k个低空飞行器的老化状态进行分析，若fk0≤f1，判定第k个低空飞行器的老化状态为正常；若fk0＞f1，判定第k个低空飞行器的老化状态为异常；
在第k个低空飞行器的老化状态为异常时对第k个低空飞行器配送环境状态的分析过程进行处理，并将处理后的预设状态指数设为HZ1，设定HZ1＝HZ0×{1-lg[5×(fk0-f1)/(fk0+f1)+1]/lg6}。
可选地，对下一监测周期低空飞行器的飞行速度进行管理过程如下：
当tw/T≤γ1时，若td/T≤γ2，判定当前监测周期低空飞行器的等待时间正常，不对下一监测周期低空飞行器的飞行速度进行管理，若td/T＞γ2，判定当前监测周期低空飞行器的等待时间异常，并将下一监测周期低空飞行器的飞行速度设为V1，设定V1＝V0×{1-η×(td/T-γ2)}；
当tw/T＞γ1时，若td/T≤γ3，判定当前监测周期低空飞行器的等待时间正常，不对下一监测周期低空飞行器的飞行速度进行管理，若td/T＞γ3，判定当前监测周期低空飞行器的等待时间异常，并将下一监测周期低空飞行器的飞行速度设为V2，设定V2＝V0×{1-η×(td/T-γ3)}；
将下一监测周期低空飞行器的飞行速度的管理结果向用户进行输出；
其中，tw为监测周期内各低空飞行器配送环境状态为异常的时长的平均值，T为监测周期的时长，td为监测周期内低空飞行器的等待时长，V0为当前监测周期低空飞行器的飞行速度，γ1为第一预设异常占比，γ2为第二预设异常占比，γ3为第三预设异常占比，η为预设调整比例系数。

规划模块，用以对充电站与配送点的距离进行计算，并根据计算结果、配送区域的优先系数和低空飞行器续航对配送区域与低空飞行器进行匹配；

速度管理模块，用以根据监测周期内低空飞行器的等待时间与配送环境状态对下一监测周期低空飞行器的飞行速度进行管理。

本发明的有益效果如下：过信息获取、坐标构建、类型划分、优先分析、规划等多个模块，有效提升了在灾后紧急情况下物资配送的效率，能够全面获取配送区域的物资需求、低空飞行器的技术参数及环境信息，确保数据基础的全面性与实时性，通过精确的坐标构建和类型划分，能够快速判断各配送区域的优先级，确保资源优化分配，优先满足高需求和近距离的区域，此外，通过智能匹配机制，将优先级区域与适合的低空飞行器进行高效匹配，大幅提高了物资的投送准确性，在环境分析与状态监测中，结合实时温度和空气密度的数据分析，能够灵活调整飞行策略，确保在不利环境条件下仍然保证配送效率，通过对飞行器速度的管理，实现了灾后物资配送的高效、智能与安全。

具体的，待选区域和第k个低空飞行器的匹配过程如下：
将配送区域的优先系数进行降序排序，并将排序第一的配送区域作为待选区域，将待选区域与配电点的距离Ly与第k个低空飞行器续航Hk对待选区域和第k个低空飞行器进行匹配，其中：
若Ly≤α×Hk/2，将待选区域与该低空飞行器的匹配指数设为Pk1，设定Pk1＝1-ln{|α×Hk/2-Ly|/(α×Hk/2)+1}；

其中，α为预设调整系数，β为预设比例系数。
具体的，通过分析优先系数、飞行器续航和充电点坐标，将高优先级区域与续航能力合适的飞行器进行匹配，提高配送效率，确保物资能够按时送到；对待选区域进行降序排序，保证首选区域优先得到配送，同时降低资源浪费，为紧急救援提供优先支持。
