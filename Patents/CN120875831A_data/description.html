Description

基于人工智能的低空飞行器运维管理系统及方法
技术领域
本发明涉及低空飞行器技术领域，具体为基于人工智能的低空飞行器运维管理系统及方法。
背景技术
低空飞行器是指在垂直高度1000米以下(部分区域延伸至3000米)空域运行的航空器，涵盖有人驾驶和无人驾驶类型，主要用于载人、载货或特殊作业。其核心类型包括电动垂直起降飞行器(eVTOL)、无人机、直升机和轻型固定翼飞机等。eVTOL通过电动推进实现垂直起降，适合城市空中交通；无人机则以体积小、成本低见长，广泛应用于航拍、农业植保等领域；直升机具备悬停能力，适应复杂地形；轻型固定翼飞机速度快、航程远，但需跑道起降。关键技术包括导航避障(激光雷达、视觉传感器)、飞控系统(自动化控制与气象监测)及动力系统(电池续航优化)。低空飞行器在农业、物流、救援等领域发挥重要作用，例如无人机精准喷洒农药、医疗物资快速配送。
低空飞行器的电池问题缺陷是影响其安全性和续航能力的关键因素，锂电池作为主要动力源，在高负荷运行或极端温度下易出现电量骤降、过热失控甚至起火爆炸等风险，现有的低空飞行器运维管理系统缺乏对电池的有效管理监测，可能造成因电池故障导致空中失控或坠毁。
发明内容
(一)解决的技术问题针对现有技术的不足，本发明提供了基于人工智能的低空飞行器运维管理系统及方法，具备对电池进行全面综合评估监测，实现电池问题的早期发现和预防性维护，避免电池长时间处于高温状态，减少热量对电池的损害，延长电池使用寿命，通过自动化监测和故障诊断，减少人工干预需求，提高运维管理的效率，同时，在故障下帮助系统判断飞行器是否具备安全返航的条件，帮助系统动态调整飞行计划，减少了电池故障情况下可能造成的无人机坠毁的问题等优点。
(二)技术方案
为实现上述目的，本发明提供如下技术方案：基于人工智能的低空飞行器运维管理系统，包括电池状态监测模块、数据分析管理模块、故障诊断模块模块、规划处理模块；
所述电池状态监测模块通过网络连接数据分析管理模块，所述数据分析管理模块通过网络连接故障诊断模块模块，所述故障诊断模块模块通过网络连接规划处理模块；
所述电池状态监测模块用于对飞行器进行状态监测，并获取监测数据进行记录保存；
所述数据分析管理模块用于根据监测数据进行计算电池散热效率指数和健康状态指数；
所述故障诊断模块模块用于根据电池散热效率指数以及健康状态指数对飞行器设备进行故障诊断；

基于人工智能的低空飞行器运维管理方法，包括以下步骤：
步骤一、对飞行器进行状态监测，并获取监测数据组成温度监测数据集、电压监测数据集、电流监测数据集进行记录保存；
步骤二、根据温度监测数据集、电压监测数据集、电流监测数据集进行计算电池散热效率指数和健康状态指数；
步骤三、根据电池散热效率指数进行判断电池的散热效果，在当前的电池散热效果不佳时，发送散热信号给散热系统进行散热优化；
步骤四、在散热优化后对电池散热效率指数持续监测，判断是否发生故障；步骤五、根据健康状态指数对飞行器设备进行判断是否存在电池健康状态不佳的情况；
步骤六、在判断电池故障、电池健康状态不佳任一情况下，对当前的电池返航能力进行评估；
步骤七、根据电池返航能力做出应急处理。
与现有技术相比，本发明提供了基于人工智能的低空飞行器运维管理系统及方法，具备以下有益效果：
1、本发明通过温度、电压、电流等多维度数据，对电池进行全面综合评估监测，实现电池问题的早期发现和预防性维护，持续监测电池散热效率指数，能够实时检测散热系统的工作状态。当电池散热效率指数超过阈值时，表明散热效果不佳，系统会立即触发散热优化措施，避免电池因散热不良导致温度过高，进而引发热失控、容量衰减或起火爆炸等严重问题，通过及时散热，避免电池长时间处于高温状态，减少热量对电池的损害，延长电池使用寿命，同时散热优化措施能排除临时性问题，提升系统判断精度，减少对飞行计划的干扰，实时掌握健康状态指数，实现自动化监测和故障诊断，减少人工干预需求，提高运维管理的效率。
2、本发明综合考虑了能耗、返航距离、散热效率和健康状态，能够较为全面地反映电池的返航能力，同时体现了散热效率和健康状态对返航能力的负面影响，当系统收到电池故障信号或健康状态不佳信号时，可以快速计算电池的返航能力，帮助系统判断飞行器是否具备安全返航的条件，帮助系统动态调整飞行计划，减少了电池故障情况下可能造成的无人机坠毁的问题。

上述基于人工智能的低空飞行器运维管理系统具体方法如下：
步骤一、对飞行器进行状态监测，并获取监测数据组成温度监测数据集、电压监测数据集、电流监测数据集进行记录保存；
步骤二、根据温度监测数据集、电压监测数据集、电流监测数据集进行计算电池散热效率指数和健康状态指数；
步骤三、根据电池散热效率指数进行判断电池的散热效果，在当前的电池散热效果不佳时，发送散热信号给散热系统进行散热优化；
步骤四、在散热优化后对电池散热效率指数持续监测，判断是否发生故障；步骤五、根据健康状态指数对飞行器设备进行判断是否存在电池健康状态不佳的情况；
步骤六、在判断电池故障、电池健康状态不佳任一情况下，对当前的电池返航能力进行评估；
步骤七、根据电池返航能力做出应急处理。
