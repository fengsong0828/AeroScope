Description

基于数字孪生的低空飞行器导航误差补偿系统及方法
技术领域
本发明涉及无人机导航技术领域，具体为基于数字孪生的低空飞行器导航误差补偿系统及方法。
背景技术
飞行器导航误差补偿是通过技术手段修正导航系统误差的过程。常见误差来源包括传感器噪声、环境干扰等。补偿方法有滤波算法抑制随机误差，误差模型建模修正系统误差，还可利用多传感器融合、卫星差分等技术提升精度，保障飞行器导航的可靠性与准确性。
申请号为201711312957.X的发明专利申请中公开了一种超高速飞行器自主导航系统的模型误差补偿方法，该申请旨在解决“在超高速飞行器机动的过程中,受发动机推力等作用的影响,初始的动力学模型与实际运动情况不符,而模型误差会直接影响滤波精度;超高速飞行器在跨域飞行边界由于激波形成包围飞行器的高温等离子体壳层,会导致无线电导航信息间断,因此依赖于卫星导航的接收机会存在大范围、长弧段误差,严重影响超高速飞行器的导航精度,甚至导致发散”的问题。
然而，针对于每次均按相同路线执行无人机巡检的技术场景，无人机分析偏航会直接导致无人机采集影像视角出现偏差，以至于采集的影像无法更加精准的反映巡检目标；
为此，我们提出了基于数字孪生的低空飞行器导航误差补偿系统及方法。
发明内容
针对现有技术所存在的上述缺点，本发明提供了基于数字孪生的低空飞行器导航误差补偿系统及方法，能够有效地解决现有技术的问题。
为实现以上目的，本发明通过以下技术方案予以实现；
本发明公开了基于数字孪生的低空飞行器导航误差补偿系统，系统应用于每次飞行均按相同预定路线飞行以执行视觉巡检的无人机，包括：
拾取模块，用于实时接收无人机搭载摄像头采集的影像数据，在影像数据中拾取关键图像；标注模块，用于在拾取模块中拾取的各关键图像中标注关键点及各关键点到关键图像来源时间无人机位置信息的距离；分析模块，用于在无人机再次执行巡检时，拾取模块与标注模块再次运行后运行，基于当前拾取关键图像与最早拾取的关键图像分析无人机偏航程度；决策模块，用于设定决策阈值，接收分析模块中无人机偏航程度分析结果，应用分析结果与决策阈值比对，在分析结果大于等于决策阈值时，控制协调模块运行，反之，控制跳转模块运行；协调模块，用于协调无人机回归预定路径；跳转模块，用于刷新系统运行跳转，跳转至拾取模块运行阶段；
所述系统运行阶段，基于数字孪生技术构建包含预定路线的三维虚拟场景，通过特征点云配准算法将实时采集影像与虚拟场景进行时空对齐，形成物理空间与虚拟空间的映射矩阵，并实时更新虚拟场景中的关键点坐标与距离参数。
更进一步地，所述拾取模块内部设置有储存单元，所述储存单元用于接收拾取模块运行拾取的关键图像，对关键图像进行储存；
所述无人机按预定路线飞行过程中，在预定路线中预定位置悬停飞行，并同步执行采集图像操作时，采集的图像记作关键图像；
当前影像数据采集过程无人机未执行悬停飞行并采集图像时，在影像数据中分割获取画面帧，分析各画面帧拾取倾向，基于画面帧拾取倾向分析结果，拾取对应拾取倾向最高的画面帧作为关键图像；
其中，拾取模块每次运行仅拾取一幅关键图像，且同步以关键图像在拾取时无人机自身位置信息标记于拾取的关键图像。

采用本发明提供的技术方案，与已知的现有技术相比，具有如下有益效果：
本发明提供一种基于数字孪生的低空飞行器导航误差补偿系统及方法，该系统及方法在运行过程中，能够在无人机飞行中，智能拾取关键图像并标记位置信息，通过分析不同时期关键图像，精准判断无人机偏航程度，并通过独特的图像分析逻辑，将图像转换为灰度图提取轮廓，综合轮廓复杂度、密度等多维度计算拾取倾向分值，还通过指标均衡性惩罚因子优化分析，确保关键图像选取的科学性，标注关键点时结合灰度值与角点，提升分析基准的可靠性；
分析偏航程度时，依据对应关键点距离变化并赋予不同权重计算综合表征值，使判断更精准，当偏航超过阈值，系统能控制无人机在垂直平面灵活移动归位，未超阈值则刷新系统继续运行。整套机制实现了从图像采集、分析到误差补偿的全流程智能化、精准化，有效提升无人机导航精度与巡检可靠性，并以数字孪生技术呈现无人机实时飞行状态，以便于无人机后台管理人员实时监管。

系统运行阶段，基于数字孪生技术构建包含预定路线的三维虚拟场景，通过特征点云配准算法将实时采集影像与虚拟场景进行时空对齐，形成物理空间与虚拟空间的映射矩阵，并实时更新虚拟场景中的关键点坐标与距离参数。
在本实施例中，拾取模块运行实时接收无人机搭载摄像头采集的影像数据，在影像数据中拾取关键图像；储存单元同步接收拾取模块运行拾取的关键图像，对关键图像进行储存，标注模块后置运行在拾取模块中拾取的各关键图像中标注关键点及各关键点到关键图像来源时间无人机位置信息的距离，再由分析模块在无人机再次执行巡检时，拾取模块与标注模块再次运行后运行，基于当前拾取关键图像与最早拾取的关键图像分析无人机偏航程度，进一步通过决策模块设定决策阈值，接收分析模块中无人机偏航程度分析结果，应用分析结果与决策阈值比对，在分析结果大于等于决策阈值时，控制协调模块运行，反之，控制跳转模块运行，应用协调模块协调无人机回归预定路径，或应用跳转模块刷新系统运行跳转，跳转至拾取模块运行阶段。
通过上述实施例中系统运行，为按相同预定路线飞行以执行视觉巡检的无人机提供以导航实时误差校准，以实现误差补偿，保障无人机每次飞行均能够按照预定路线精准飞行，从而准确的完成视觉巡检任务。
实施例2
在具体实施层面，在实施例1的基础上，本实施例参照图2对实施例1中基于数字孪生的低空飞行器导航误差补偿系统做进一步具体说明：
基于数字孪生的低空飞行器导航误差补偿方法，包括：
步骤1：拾取无人机搭载摄像头实时采集的影像数据，在影像数据中拾取关键图像；
步骤2：在关键图像中标注关键点，并获取各关键点到关键图像采集时间戳无人机所在位置信息的距离，并将获取的距离标注于关键图像中各对应关键点；
步骤3：在无人机再次搭载摄像头按预定路线飞行再次采集影像数据时，再次获取关键图像，以获取的关键图像与最早一次拾取的关键图像中对应图像分析无人机偏航程度；
步骤4：设定决策阈值，应用决策阈值与无人机偏航程度分析结果比对，决策无人机当前位置是否需要协调；
步骤5：决策结果为是，则控制无人机在与前进方向垂直的自身位置信息所在的平面上移动，以完成协调；决策结果为否，则跳转步骤执行初始阶段，刷新执行。
综上而言，上述实施例中系统及方法在运行过程中，能够在无人机飞行中，智能拾取关键图像并标记位置信息，通过分析不同时期关键图像，精准判断无人机偏航程度，并通过独特的图像分析逻辑，将图像转换为灰度图提取轮廓，综合轮廓复杂度、密度等多维度计算拾取倾向分值，还通过指标均衡性惩罚因子优化分析，确保关键图像选取的科学性，标注关键点时结合灰度值与角点，提升分析基准的可靠性，分析偏航程度时，依据对应关键点距离变化并赋予不同权重计算综合表征值，使判断更精准，当偏航超过阈值，系统能控制无人机在垂直平面灵活移动归位，未超阈值则刷新系统继续运行。整套机制实现了从图像采集、分析到误差补偿的全流程智能化、精准化，有效提升无人机导航精度与巡检可靠性，并以数字孪生技术呈现无人机实时飞行状态，以便于无人机后台管理人员实时监管。
