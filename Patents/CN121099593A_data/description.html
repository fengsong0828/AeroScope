CN121099593A

Description

一种应用于低空飞行器的抗干扰吸波铜箔及其制备方法
技术领域
本发明属于电磁屏蔽材料技术领域，具体涉及一种应用于低空飞行器的抗干扰吸波铜箔及其制备方法。
背景技术
低空飞行器(如无人机、轻型直升机等)在民用测绘、电力巡检、军事侦察等领域的应用日益广泛，但其工作环境中存在复杂的电磁干扰(如地面通信信号、导航频段干扰、电磁辐射等)，易导致控制系统失灵、通信中断等问题，严重影响飞行安全。现有低空飞行器的电磁防护多采用单一铜箔作为屏蔽材料，利用其高导电性反射电磁波，但存在以下缺陷：一是仅依赖反射作用，对电磁波的吸收能力弱，抗干扰带宽窄；二是铜箔与基材结合力差，在振动、高低温循环等工况下易分层；三是轻量化不足，增加飞行器能耗。
MXene材料因二维层状结构和高电导率，兼具电磁波吸收与反射性能，被视为理想的电磁屏蔽增强体。将其与碳纤维复合可形成高强度、高屏蔽效能的复合材料。碳纤维作为高强度轻量化材料，密度1.7-2.0g/cm3，抗拉强度>3GPa，常被用作载体，是良好的构建复合结构材料。然而，现有MXene/碳纤维、铜箔复合技术仍存在以下问题，一是MXene与碳纤维的界面相容性差，未经改性的MXene因表面能差异难以均匀分散，导致复合材料内部形成导电孤岛，削弱屏蔽网络连续性；二是铜箔与复合屏蔽层的粘结方式不合理，传统物理压合或普通树脂粘结易因界面电阻过大降低屏蔽效能，且耐温性不足(玻璃化转变温度<100℃)，无法适应低空飞行器的机舱热环境；三是复合工艺缺乏针对性设计，未考虑低空飞行器对材料厚度，通常要求<500μm)、柔韧性及耐候性的特殊需求，导致实际应用中性能衰减过快。
现有技术中关于电磁屏蔽材料的研究多聚焦于单一性能提升，如高屏蔽效能，而忽视屏蔽、力学、轻量化的协同优化，例如，中国专利CN119913582A公开了一种银灰色电磁屏蔽铜箔，虽通过合金镀层提升了屏蔽性能，但未引入吸波机制，抗干扰带宽有限；CN209619969U采用多层橡胶垫与铜箔复合，虽改善了柔韧性，但密度增加显著，且高温下易老化。
因此，开发一种以铜箔为反射层，MXene/碳纤维为吸波层，通过耐高温粘结剂实现稳定复合的抗干扰材料，成为解决低空飞行器电磁兼容问题的迫切需求。
发明内容
发明目的：
本发明的目的是提供一种应用于低空飞行器的抗干扰吸波铜箔及其制备方法，解决现有屏蔽材料抗干扰效果差、结构稳定性不足、轻量化欠佳等问题，实现高屏蔽效能、强结合力、轻量化的协同优化。
本发明的技术方案：
本发明有提供一种应用于低空飞行器的抗干扰吸波铜箔，所述应用于低空飞行器的抗干扰吸波铜箔包括两层铜箔层和一层电磁屏蔽层，所述电磁屏蔽层夹设于两层铜箔层之间；所述电磁屏蔽层与铜箔层之间通过耐高温树脂实现粘结固定。
进一步的，所述电磁屏蔽层由碳纤维接枝碳化钛MXene制成。
进一步的，所述电磁屏蔽层的制备方法，包括以下步骤：
S1：将碳化钛MXene加入无水乙醇中，使用醋酸溶液调节pH至4-5，缓慢加入3-氨丙基三乙氧基硅烷，在超声条件下60-70℃反应4-6小时，反应结束后使用无水乙醇和去离子水冲洗赶紧，沉淀真空干燥得到碳化钛MXene-NH2；
S2：在室温下，将碳纤维浸泡在丙酮溶液中，64-72h后使用去离子水清洗赶紧，真空干燥后得到预处理碳纤维，然后使用硝酸浸泡预处理碳纤维，加热至80-90℃，反应3-5h，然后用去离子水洗净，真空干燥得到氧化碳纤维；
S3：在DMF溶剂中加入步骤S1制备的碳化钛MXene-NH2和HATU粉末，超声处理直至分散均匀，再加入步骤S2制备的氧化碳纤维完全浸润并超声，在60-70℃条件下反应4-6h，反应结束后用去离子水清洗，真空干燥得到碳纤维接枝碳化钛MXene，即所述电磁屏蔽层。
进一步的，所述碳化钛MXene与无水乙醇的固液比为1g：100-150ml。
进一步的，所述硝酸的浓度为65-70wt％，所述硝酸与预处理碳纤维的液固比为10-20mL：1g。
进一步的，所述氧化碳纤维与碳化钛MXene-NH2的质量比为1：0.05-0.2。
本发明还提供一种应用于低空飞行器的抗干扰吸波铜箔的制备方法，其特征在于，包括以下步骤：
(1)预处理：对铜箔表面进行粗化处理；
(2)叠层：在电磁屏蔽层与铜箔层之间涂抹耐高温树脂，然后按铜箔层-电磁屏蔽层-铜箔层的顺序叠放，得到叠层结构；
(3)热压复合：将叠层结构置于层压机中，施加100-300℃的温度和1-10MPa的压力，使层间紧密结合并排除气泡，制得抗干扰吸波铜箔板。
进一步的，所述铜箔表面粗化处理采用电化学粗化法，粗化后铜箔表面的粗糙度Ra为0.5-2.0μm；步骤(3)中，热压复合的保温保压时间为30-60min。
进一步的，所述耐高温树脂为双酚A型氰酸酯、邻甲酚醛环氧树脂、环氧树脂F-51中的一种；所述耐高温树脂的涂抹厚度为50-300μm。
进一步的，所述铜箔层的厚度为10-50μm，所述电磁屏蔽层的厚度为50-200μm。
外层铜箔(高导电性)通过反射作用将大部分入射电磁波(尤其是高频段)折返，减少穿透；中间电磁屏蔽层中，碳纤维接枝的碳化钛MXene凭借二维层状结构形成连续导电网络，通过界面极化及涡流损耗(高电导率引发)吸收残余电磁波；内层铜箔进一步反射未被吸收的电磁波，形成“反射-吸收-再反射”的闭环衰减，显著提升屏蔽效能。电磁屏蔽层采用低密度碳纤维(1.7-2.0g/cm3)为基体，复合MXene后密度仍低于3.0g/cm3，配合薄型化设计(总厚度≤500μm)，显著降低整体质量；粘结树脂选用高Tg(＞100℃)品种，其分子链刚性与耐温性适配低空飞行器的机舱热环境，避免高温下粘结失效导致的屏蔽性能衰减。
有益效果：
本发明的一种应用于低空飞行器的抗干扰吸波铜箔及其制备方法，采用双层铜箔+碳纤维接枝碳化钛MXene屏蔽层结构，实现X波段45-62dB高屏蔽效能，依托“反射-吸收-再反射”机制高效抗干扰，电化学粗化与高Tg粘结树脂结合，界面强度高，1000次冷热循环后性能保持率超95％，轻量化设计(密度≤3.5g/cm3)适配飞行器需求，高温稳定性优异，200℃下屏蔽效能保持率超95％，工艺兼容现有生产体系，应用价值突出。
具体实施方式
以下将结合具体实施方案来说明本发明。需要说明的是，下面的实施例为本发明的示例，仅用来说明本发明，而不用来限制本发明。在不偏离本发明主旨或范围的情况下，可进行本发明构思内的其他组合和各种改良。
本发明所使用的化学试剂若无特殊说明，均为普通市售分析纯；其中，MAX相Ti3AlC2：400目，佛山市新烯科技有限公司；无水氯化锂(纯度99％)：分析纯，麦克林生化科技有限公司；O-(7-氮杂苯并三氮唑-1-基)-N,N,N',N'-四甲基脲鎓六氟磷酸酯(HATU，纯度99％)：上海阿拉丁生化科技股份有限公司；碳纤维T700；；铜箔厚度30μm；层压机，型号LD-300；电化学粗化设备，型号EC-200。
碳化钛MXene的制备：
S1：在100mL的聚四氟乙烯烧杯中依次加入20mL氢氟酸、20mL盐酸和10mL去离子水制备混合酸溶液，加热至35℃后，缓慢加入2g MAX相Ti3AlC2粉末，在600r/min转速下持续反应24h，反应结束，使用去离子水通过离心进行洗酸处理，直至上层清液pH≥6，留下底部沉淀；
S2：在40mL去离子水中溶解2g氯化锂粉末，加入步骤S1底部沉淀充分摇匀，室温搅拌24h后，使用去离子水通过离心处理清除氯化锂，过滤、干燥，制得碳化钛MXene。
电磁屏蔽层A的制备：
S1：将5g碳化钛MXene加入750ml无水乙醇中，使用醋酸溶液调节pH至4，缓慢加入8g 3-氨丙基三乙氧基硅烷，在超声条件下60℃反应4小时，反应结束后使用无水乙醇和去离子水冲洗赶紧，沉淀于60℃真空干燥8h，得到碳化钛MXene-NH2；
S2：在室温下，取20g碳纤维浸泡于500mL丙酮溶液中72h，用去离子水清洗至滤液无丙酮残留，60℃真空干燥6h得到预处理碳纤维，取10g预处理碳纤维，加入150mL质量浓度68％的硝酸(液固比15mL:1g)，加热至80℃反应4h，用去离子水洗净至滤液pH＝7，60℃真空干燥6h，得到氧化碳纤维；
S3：在500ml DMF溶剂中加入步骤S1制备的1g碳化钛MXene-NH2和1g HATU粉末，超声分散30min至均匀，再加入步骤S2制备的10g氧化碳纤维，完全浸润后继续超声30min，在60℃条件下反应4h，反应结束后用去离子水清洗，60℃真空干燥8h，得到碳纤维接枝碳化钛MXene，即所述电磁屏蔽层A。

实施例1
(1)预处理：对铜箔表面进行粗化处理，电化学粗化至Ra 1.0μm；
(2)叠层：在电磁屏蔽层A与铜箔层之间涂抹双酚A型氰酸酯150μm，铜箔层30μm，电磁屏蔽层100μm，铜箔层30μm，然后按铜箔层-电磁屏蔽层-铜箔层的顺序叠放，得到叠层结构；
