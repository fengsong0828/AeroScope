Description

一种低空载人飞行器的节能控制方法及系统
技术领域
本发明涉及飞行器总体技术领域，尤其涉及一种低空载人飞行器的节能控制方法及系统。
背景技术
目前，在临近空间运行的飞行器中，小型吸气式发动机或冲压发动机常被用作飞行动力装置。这类发动机由于其结构紧凑、尺寸小、推力有限，被用于满足飞行器在远程飞行和机动性能方面的需求。然而，由于发动机本身工作时间较短，燃料消耗较大，导致飞行器在执行任务时续航能力受到限制，难以实现高效长时间飞行。
在一种现有技术中，飞行器的控制方法主要针对固定翼飞机或直升机进行设计。例如，固定翼飞行器通过动力系统提供推力，在升力和重力的平衡下实现稳定飞行；而直升机则通过旋翼系统的转速和迎角调节，完成飞行方向和姿态的控制。这些控制方法通过调整动力输出和空气动力参数，实现飞行器在爬升、平飞和降落阶段的飞行任务。然而，这些技术的设计思路主要基于常规飞行器，针对低空载人飞行器的特殊工作模式和动力需求未进行有效适配，尤其是新型飞行器所采用的小型吸气式发动机或多喷管式发动机。
现有技术中存在控制环节之间相互独立，飞行器的节能性能较差，功耗较高的问题。
发明内容
本发明提供了一种低空载人飞行器的节能控制方法及系统，以实现了飞行器在不同飞行模式下的闭环节能控制，降低飞行器功耗。
第一方面，为了解决上述技术问题，本发明提供了一种低空载人飞行器的节能控制方法，包括：
获取大气特征数据和飞行器特征数据；其中，所述大气特征数据包括标准大气压、标准大气压的特征高度、海平面标准温度、环境温度递减率、标准大气密度和比气体常数；所述飞行器特征数据包括飞行器高度、实时速度、实时质量、升力系数和阻力系数；
根据所述大气特征数据和所述飞行器特征数据，进行环境特征计算，得到实时大气压和实时温度；
根据所述大气特征数据、所述飞行器特征数据、所述实时大气压和所述实时温度，进行密度计算，得到实时密度；
根据所述实时密度和所述飞行器特征数据，进行面积计算，得到迎风面积；
根据所述迎风面积、所述实时密度和所述飞行器特征数据，进行质量计算，得到调整质量；
根据所述飞行器特征数据、所述调整质量、所述迎风面积和所述实时密度，进行速度计算，得到最佳速度；
根据所述最佳速度、所述实时密度、所述迎风面积和所述飞行器特征数据，进行计算，得到实时功率和飞行器升力，并根据所述实时功率和所述飞行器升力进行节能控制操作。

相比于现有技术，本发明提供了一种低空载人飞行器的节能控制方法及系统，通过优化飞行器的能量管理和控制策略，降低飞行器的功耗，提高飞行器的续航能力和飞行效率。本发明通过获取大气特征数据和飞行器特征数据，包括标准大气压、标准温度、实时高度、实时速度、实时质量、升力系数和阻力系数等关键参数，进行动态环境特征计算，实时获得飞行高度下的大气压和温度。随后，基于大气特征数据和飞行器数据，通过密度计算得到实时大气密度，为进一步的能量计算提供基础。在此基础上，本发明通过迎风面积计算和质量调整计算，根据实时大气密度和飞行器速度动态调整迎风面积和飞行器质量，确保飞行器在不同飞行模式(爬升、平飞、降落)下能够维持最佳的空气动力特性。通过最佳速度计算，本发明确定了飞行器在不同工况下的最佳飞行速度，减少空气阻力和能量消耗。此外，本发明根据最佳速度、实时密度和迎风面积，通过功率和升力计算，评估实时所需功率和飞行器产生的升力，并在不同飞行模式下进行节能控制操作。在爬升模式下，若飞行器升力过高或功率超出阈值，系统自动降低升力系数和发动机推力输出，减少能量浪费；在平飞模式下，通过动态调整升力系数，确保升力与重力平衡，维持稳定平飞状态；在降落模式下，通过控制升力系数和发动机推力输出，降低能耗，确保飞行器平稳降落。
通过上述方法，本发明实现了飞行器在不同飞行模式下的闭环节能控制，降低飞行器功耗，提高低空载人飞行器的续航能力和飞行效率。

在一个具体的实施例中，标准大气压是国际标准大气模型中定义的海平面处的标准压力，取固定值：
P0＝101325Pa
需要说明的是，标准大气压作为计算实时大气压的基准，通过结合飞行器高度和特征长度，利用大气压随高度呈指数衰减的关系推算飞行高度下的实时大气压。
在一个具体的实施例中，标准大气压的特征高度是大气压力随高度变化的特征常数，由标准大气模型给出，取固定值：
H0＝8434.5m
需要说明的是，标准大气压的特征高度用于计算实时大气压，确定不同飞行高度下的大气压值。它描述了大气压变化的速度，保证了大气特征数据的准确性。
在一个具体的实施例中，海平面标准温度是标准大气模型中定义的海平面温度，取固定值：
T0＝288.15K(15°C)
需要说明的是，海平面标准温度结合环境温度递减率和飞行器高度，用于计算实时温度。温度是影响空气密度的重要因素，后续空气动力学参数(如升力、阻力)都依赖于温度计算结果。
在一个具体的实施例中，环境温度递减率是大气温度随高度线性递减的速率，取固定值：
Γ＝0.0065K/m
需要说明的是，环境温度递减率与海平面标准温度和飞行器高度共同决定实时温度。
在一个具体的实施例中，标准大气密度是海平面处标准状态下的空气密度，取固定值：
ρ0＝1.225kg/m3
需要说明的是，标准大气密度是计算飞行高度下实时密度的基准值，结合实时温度和实时大气压，通过公式计算得到实时密度。实时密度是空气动力学参数(如迎风面积、升力、阻力)计算的核心数据。
在一个具体的实施例中，比气体常数是空气的理想气体常数，取固定值：
R＝287J/(kg·K)
需要说明的是，比气体常数是结合实时温度和实时大气压通过理想气体状态方程计算空气的实时密度。空气密度对飞行器的升力和阻力特性具有直接影响。
在一个具体的实施例中，飞行器高度通过高度传感器(如气压高度计、激光测距仪或GPS系统)实时测量。
需要说明的是，飞行器高度是计算实时大气压、实时温度和实时密度的关键参数。高度变化直接影响大气环境特性，进而影响空气动力学参数和飞行器功率需求。
在一个具体的实施例中，实时速度通过飞行器上的空速管、GPS速度传感器或惯性导航系统实时获取飞行器的速度。
需要说明的是，实时速度是迎风面积、飞行器功率和飞行器升力计算的输入参数。速度的大小直接决定空气动力特性和飞行能量消耗。
在一个具体的实施例中，实时质量是通过飞行器载荷监测系统(如燃油流量计、负载传感器)实时估算飞行器的当前质量。
需要说明的是，实时质量是计算迎风面积、飞行器功率和最佳速度的输入参数，影响飞行器在不同模式(爬升、平飞、降落)下的能量平衡和控制策略。
在一个具体的实施例中，升力系数主要与飞行器的翼型、迎角(攻角)以及飞行状态相关。初始值由飞行器的气动设计参数确定，基于风洞试验或数值仿真分析获得特定飞行状态下的升力系数。在实际飞行过程中，飞行器配备的姿态传感器(如迎角传感器、角速度传感器)可以实时监测飞行器的迎角变化，结合飞行器飞行状态(如爬升、平飞或降落模式)动态调整升力系数。通过飞控系统的反馈调整，确保飞行器在不同飞行模式下维持所需的升力，满足飞行稳定性和能量最优控制要求。例如，在平飞模式下，维持适中迎角以降低阻力，同时满足升力与重力平衡。
需要说明的是，升力系数的初始值在飞行器设计阶段通过风洞试验或计算流体力学仿真(CFD)获得。例如，在风洞测试中，通过改变飞行器的迎角(攻角)，测得不同迎角下的升力系数，并绘制出升力系数与迎角的关系曲线，作为飞行器气动设计的参考。在实际飞行过程中，飞行器利用迎角传感器和姿态传感器实时监测当前迎角，通过飞控系统动态查找升力系数数据库，调整升力系数以满足当前飞行状态。例如，在爬升模式中，系统会通过调整升降舵角度增大迎角，使升力系数达到目标值，提供足够的升力克服重力，确保稳定爬升。
在一个具体的实施例中，阻力系数主要由飞行器的气动外形、迎风面积、表面光滑度及飞行状态决定。初始设计值通过风洞试验或数值仿真分析得到，具体针对飞行器在不同速度和攻角下的阻力特性。实际飞行中，阻力系数可以通过飞行器的空气动力反馈系统动态调整，传感器实时监测飞行速度、气流扰动及飞行器姿态变化，从而计算瞬时阻力大小并修正阻力系数。在节能控制过程中，通过优化迎风面积和飞行姿态，尽量减少阻力系数带来的空气阻力。例如，在平飞模式下，降低迎角和优化姿态可以有效减少阻力系数，从而降低功率消耗，提高飞行器的能效。
需要说明的是，阻力系数的初始值通过飞行器设计阶段的风洞试验或仿真分析确定，测量飞行器在不同速度和迎角下的空气阻力特性。在实际飞行中，飞行器配备的速度传感器和加速度传感器可实时测量飞行速度和阻力，通过空气阻力公式反向计算出当前的阻力系数。系统还可通过优化飞行姿态和迎角来动态降低阻力系数。例如，在平飞模式下，系统检测到迎角过大导致阻力系数增加，便自动调整机翼姿态，减小迎角，从而降低空气阻力，提升飞行器的节能性能。

需要说明的是，爬升功率阈值的具体设置根据飞行器的最大允许功率、飞行任务需求和安全性能限制进行动态设定，由设计阶段的理论计算和实际飞行测试相结合来确定。系统会预设一个安全阈值，例如飞行器发动机的最大输出功率为100kW，为了避免发动机长期处于高负荷状态引起过热或寿命缩短，爬升功率阈值可设置为0.8倍的最大输出功率，即80kW。在实际飞行中，系统通过实时监测飞行器的功率消耗，与设定的阈值进行比较，当实时功率超过80kW时，系统会自动降低发动机推力输出，并通过优化升力系数和迎角，减少不必要的升力，确保功率消耗处于安全范围内。例如，在某次爬升过程中，系统检测到功率达到85kW，超过设定的阈值，此时系统动态调整发动机推力输出至78kW，同时微调迎角，平衡升力与功耗，从而实现爬升过程中的节能与安全控制。
需要说明的是，在爬升模式中，飞行器需要克服重力产生足够的升力实现上升。当系统检测到升力不足时，系统通过增大迎角来提高升力系数，从而增加空气动力升力。具体操作是通过控制升降舵向下偏转，使飞行器机头略微抬高，从而增大迎角。增大迎角使机翼上下表面的气流速度差增大，从而提高升力，帮助飞行器顺利爬升。但为了避免迎角过大引起气流分离和失速，系统会将迎角控制在合理范围(如不超过15°)。
在一个具体的实施例中，在平飞模式中，飞行器需要维持稳定的水平飞行状态，保证升力与重力平衡，同时减少能量消耗。系统具体执行以下控制操作，当所述飞行器升力与飞行器重力不平衡时，系统动态调整升力系数，使升力与重力相等。通过姿态控制，保持飞行器在最优迎角下稳定飞行。例如，若检测到飞行器升力小于飞行器重力，系统增加迎角以提高升力系数，确保升力恢复到平衡状态；当实时功率超过平飞功率阈值时，系统根据实际情况动态调整飞行速度和发动机推力输出，以将功率消耗控制在最小值。通过优化速度与推力匹配关系，系统实现平飞状态下的最低能耗。
需要说明的是，平飞功率阈值的具体设置主要依据飞行器的巡航功率需求、设计飞行速度和安全余量进行确定，确保飞行器在平飞状态下维持稳定飞行且功耗最小。例如，假设飞行器在设计最佳巡航速度50m/s下，其功率需求通过理论计算为60kW。为了留出一定的安全余量，系统将平飞功率阈值设定为1.1倍的功率需求，即66kW。在实际飞行过程中，系统通过实时监测飞行器的功率消耗，当实时功率超过阈值66kW时，系统会动态调整飞行速度和发动机推力输出，降低功率消耗。例如，若系统检测到飞行器功率达到70kW，超过设定的平飞功率阈值，系统将适度降低飞行速度至48m/s，同时优化推力输出，确保功率回落至65kW范围内，从而实现节能控制，维持稳定的平飞状态。
需要说明的是，在平飞模式中，飞行器的升力需要与重力保持平衡，同时减少空气阻力以实现节能飞行。此时，系统通过微调迎角，会略微减小迎角，以维持飞行器的最优气动效率。具体操作是通过控制升降舵微小上调，使迎角保持在2-5°的最佳范围，确保升力与重力相等，同时空气阻力最小。
在一个具体的实施例中，在降落模式中，飞行器需要逐渐降低高度并平稳着陆，系统通过以下操作实现安全降落与能量优化，当所述飞行器升力大于飞行器重力时，系统进行降低升力系数和调整迎角操作，逐步减小升力，确保飞行器平稳下降，避免出现升力过大导致降落不稳定；当所述实时功率大于降落风阻功率阈值时，系统通过降低发动机推力输出，减少能量消耗，确保飞行器以合适的速度完成降落过程。系统同时配合飞行姿态调整，优化着陆滑行距离，提升降落的安全性与稳定性。
需要说明的是，降落风阻功率阈值的设置主要基于飞行器在降落过程中所需的减速功率需求、飞行速度和安全着陆要求进行确定，确保飞行器在降落阶段能够平稳减速且功率消耗保持在安全范围内。例如，假设飞行器的设计着陆速度30m/s，此时的风阻功率通过计算得出为40kW。为了保证安全余量，系统将降落风阻功率阈值设定为1.2倍风阻功率，即48kW。在实际降落过程中，系统实时监测飞行器的功率消耗，当功率超过设定阈值48kW时，系统会动态降低发动机推力输出，并通过调整迎角和升力系数，减小飞行器的速度和风阻。例如，当系统检测到功率达到50kW，系统立即降低发动机推力至45kW，并适度调整迎角，确保飞行器平稳减速，功率回落至设定阈值内，最终实现安全着陆和能量消耗最小化。
需要说明的是，在降落模式中，飞行器需要降低升力，实现高度和速度的平稳下降，确保安全着陆。系统通过减小迎角，即控制升降舵向上偏转，使飞行器机头下压，迎角减小，从而降低升力系数。减小迎角会使飞行器的升力逐渐小于重力，帮助飞行器平稳下降，同时减少不必要的空气阻力，优化减速效果。
综上所述，本发明提供了一种低空载人飞行器的节能控制方法及系统，通过优化飞行器的能量管理和控制策略，降低飞行器的功耗，提高飞行器的续航能力和飞行效率。本发明通过获取大气特征数据和飞行器特征数据，包括标准大气压、标准温度、实时高度、实时速度、实时质量、升力系数和阻力系数等关键参数，进行动态环境特征计算，实时获得飞行高度下的大气压和温度。随后，基于大气特征数据和飞行器数据，通过密度计算得到实时大气密度，为进一步的能量计算提供基础。在此基础上，本发明通过迎风面积计算和质量调整计算，根据实时大气密度和飞行器速度动态调整迎风面积和飞行器质量，确保飞行器在不同飞行模式(爬升、平飞、降落)下能够维持最佳的空气动力特性。通过最佳速度计算，本发明确定了飞行器在不同工况下的最佳飞行速度，减少空气阻力和能量消耗。此外，本发明根据最佳速度、实时密度和迎风面积，通过功率和升力计算，评估实时所需功率和飞行器产生的升力，并在不同飞行模式下进行节能控制操作。在爬升模式下，若飞行器升力过高或功率超出阈值，系统自动降低升力系数和发动机推力输出，减少能量浪费；在平飞模式下，通过动态调整升力系数，确保升力与重力平衡，维持稳定平飞状态；在降落模式下，通过控制升力系数和发动机推力输出，降低能耗，确保飞行器平稳降落。
通过上述方法，本发明实现了飞行器在不同飞行模式下的闭环节能控制，降低飞行器功耗，提高低空载人飞行器的续航能力和飞行效率。
