CN107571993A

Description

一种低空垃圾回收用飞行器
技术领域
本发明涉及垃圾清理设备领域，特别是涉及一种低空垃圾回收用飞行器。
背景技术
随着人类的不断发展，垃圾产生越来越多，并且很多人素质不高，会把很多可以回收的垃圾进行丢弃，尤其是饮料瓶和饮料罐，传统是通过环卫工人进行处理，这样环卫工人会非常辛苦，并且很多地方环卫工人往往不方便进去，随着无人技术不断进步，可以考虑设计一种低空垃圾回收用飞行器，来对其进行相应的收集。
发明内容
为了解决上述存在的问题，本发明提供一种低空垃圾回收用飞行器，可以根据需要飞行至对应垃圾边上再通过机械臂对其进行回收，使用时候可以设计固定的巡检路线来替代人工进行收集，为达此目的，本发明提供一种低空垃圾回收用飞行器，包括旋翼、旋翼电机、安装架、柔性保护环、连接架、无人机主体、支撑块、钢丝架、信号发射接收器、控制箱、主旋转电机、角度控制电机、机械臂、机械支架、升降气缸、安装板、感应器、双头气缸和夹紧板，所述无人机主体两侧各有一组旋翼电机，所述旋翼有2-4个，所述旋翼的端部安装在旋翼电机的转轴的外壁，所述旋翼等角度绕旋翼电机的转轴中心线一周，所述旋翼电机外套有柔性保护环，所述旋翼电机上下两侧通过安装架与柔性保护环相连，所述连接架为T型，所述连接架将相邻的两个柔性保护环相连后固定在无人机主体外侧，所述无人机主体为盒体，所述无人机主体上有开口，所述无人机主体的前侧和后侧各有一对信号发射接收器，所述无人机主体的前后端内有控制箱，所述控制箱朝无人机主体中部一侧有支撑块，所述钢丝架放置在支撑块上，所述控制箱中部上方有主旋转电机，所述机械臂一端两侧各有一个角度控制电机一通过安装轴固定在主旋转电机的转轴上，所述机械臂另一端两侧各有一个角度控制电机二，所述角度控制电机二的转轴与机械支架的底部支架相连，所述升降气缸的底部与机械支架相连，所述升降气缸的伸缩杆的端部安装有安装板，所述安装板的四个角各安装有一个感应器，所述双头气缸安装在安装板上，所述双头气缸的两个伸缩杆的端部安装有夹紧板。
本发明的进一步改进，所述无人机主体一侧的旋翼电机有2-4个，本发明可根据实际需要设置相应的旋翼电机。
本发明的进一步改进，所述钢丝架的前后两侧支架上有把手，在钢丝架上设置把手可以便于钢丝架取出清理。
本发明的进一步改进，所述无人机主体底部的前后两侧有排液龙头，若主体内液体较多可以通过龙头排出。
本发明的进一步改进，所述无人机主体底部沿边有一圈环形气囊，设置气囊后若发生意外无人机掉落可以对机体起保护作用。
本发明的进一步改进，所述夹紧板朝中间一侧有防滑纹路，设置防滑纹路可以便于夹紧可回收垃圾。
本发明一种低空垃圾回收用飞行器，采用六个旋翼，可以低空飞行，前后设置有机械臂可以同时进行作业灵活性强，运动通过通过安装板的感应器寻找可以回收的垃圾，飞行器底部有环形气囊，可以防止降落后造成损害，机械臂端部有挤压夹紧结构可以将金属压扁减少体积，无人机主体内有金属网可以对液体进行分离，待液体到一定程度通过一侧的龙头排出即可。
