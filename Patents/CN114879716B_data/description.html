Description

反制低空空域飞行器的执法无人机路径规划方法
技术领域
本发明涉及反制飞行器低空飞行技术领域，尤其涉及一种反制低空空域飞行器的执法无人机路径规划方法。
背景技术
当前无人机等低空空域飞行器相关技术发展迅速，被广泛应用于国民生产的各行各业。然而，部分不法分子有意或无意的非法使用低空空域飞行器，“黑飞”飞行器的出现加大了低空空域飞行的安全威胁，也给民用航空、军事禁区、安防监控、个人隐私等带来了极大的威胁和挑战。为了及时反制“黑飞”飞行器，并尽可能减少反制过程引入二次威胁，利用携带电磁发射器、捕捉网发射器等的执法无人机实施抵近反制或打击成为高效、安全的反制手段。因此，如何在综合考虑油耗、威胁、到达时间及飞行区域等因素的前提下，为执法无人机规划出最优或最满意的飞行路径以保证圆满地完成反制任务成为亟待解决的问题。
因此，有必要开发一种反制低空空域飞行器的执法无人机路径规划方法，能够为携带电磁发射器、捕捉网发射器等的执法无人机快速规划最优路径，及时准确地飞抵低空空域“黑飞”飞行器实施精准反制或打击。

其中，原本的粒子群方法用的名称就是适应度fitness，在本实施例中这个适应度函数指的就是目标函数J；A*算法是一种搜索式算法，可以理解成对每个节点向外扩展搜索，然后由若干节点连接成路径，而粒子群方法中的粒子是要随机初始化得到的，利用这些粒子不断迭代获得路径，由于本申请已经实现预规划了全局路径，因此无需在粒子群方法中随机初始化粒子，而是在预规划的全局粒径附近选取一些粒子(点)进行初始化粒子，提高效率。
需要说明的是，在全局规划路径中，由于模拟环境地图中的障碍物(威胁源)是已知且固定的，但在执法无人机实际飞行过程中，可能会出现障碍物动态变化的问题，这时就需要执法无人机通过自身传感器获取的实时环境信息规避动态障碍物，最终获得安全路径。其中，利用改进的A*算法得到的主路径是由若干散点连接而成的折线，所以在主路径上的散点附近随机选取若干数量的散点作为初始数据，对粒子群优化算法进行初始化，使其获得更多的初始信息，可以加快收敛速度，保证执法无人机飞行安全性，同时，为保证粒子群优化方法的局规划部路径是朝向黑飞飞行器飞行的，将改进的A*算法规划出的路径在局部环境中的边界点作为粒子群优化算法在局部规划中的目标点。
粒子群优化方法需要调整的参数少，收敛精度高，使规划路径具有收敛速度快、路径长度短、威胁代价小等优点，能够及时准确地飞抵低空空域黑飞飞行器实施精准反制或打击。
进一步地，步骤4包括：
在执法无人机飞行过程中，若目标跟踪与定位系统检测到黑飞飞行器的位置变化超过设定阈值，则回到全局规划路径阶段，以执法无人机当前所在位置为起始位置，重新规划执法无人机的路径；
其中，执法无人机在打击黑飞飞行器的时候，如果黑飞飞行器的位置跟规划路径时检测到的位置发生变化了，其实在一定范围内还是可以打击到的，这个设定阈值还要根据特定的打击方式，如发射电磁的作用范围进行设定。
