Description

一种适用于中低空飞行器航拍测量的地面定位标志装置
技术领域
本实用新型属于飞行器航拍测量的技术领域，更具体的，本实用新型涉及一种适用于中低空飞行器航拍测量定位装置的技术领域。
背景技术
航拍测量，又称航空摄影测量，指的是在飞行器上用航摄仪器对地面连续摄取像片，结合地面控制点测量、调绘和立体测绘等步骤，绘制出地形图的作业。航空摄影测量可用于野外微地貌研究、林地考察、考古、文物修缮等领域。航空摄影测量的作业涉及到像片控制点联测，像片控制点一般是航摄前在地面上布设的标志点，也可选用如道路交叉点等像片上明显地物点，用测角交会、测距导线、等外水准、高程导线等普通测量方法测定其平面坐标和高程。
在城市、乡村等地点进行航空摄影测量作业时，飞行器容易找到显著建筑物作为标志点；但在森林、草原、原始森林等地点进行航空摄影测量作业时，存在不容易找寻显著标志点的问题，影响航空摄影测量作业的精度，因此，亟需一种合适的飞行器航拍测量定位装置。
发明内容
针对现有技术现状的不足及存在的问题，特别是针对现有中低空飞行器航拍测量时不易找寻标志点的技术缺陷，本实用新型提供了一种适用于中低空飞行器航拍测量的地面定位标志装置，与现有技术相比，本实用新型提供的一种适用于中低空飞行器航拍测量的地面定位标志装置结构简单，使用方便，在航拍中清晰可见，易于识别，提高了航测地形图的精度，通过采用多个本地面定位标志装置的中心点作为飞行器航拍测量的相片控制点测出控制点的平面坐标与高度，可得到被测物体的精确细腻的三维立体形态，具有广泛的实用性。
为了解决上述问题，本实用新型采用的技术方案如下：
本实用新型提供的一种适用于中低空飞行器航拍测量的地面定位标志装置，包括支架和定位盘，支架包括支腿和安装环，安装环下方与支腿活动连接，支腿呈折叠梯状，定位盘包括盘体、包边和定位芯，盘体呈圆柱状，包边呈圆环状，包边覆盖盘体一周，定位芯设置在盘体中心，盘体下方设置安装槽，定位盘的安装槽与支架上的安装环通过卡槽连接。
本实用新型中，盘体的最大直径为40mm,盘体上表面设置黑色油漆包覆层。
本实用新型中，包边覆盖盘体的径向长度为3mm,包边外表面设置锡箔纸层。
本实用新型中，安装槽间设置固定绳，固定绳采用橡胶绳。
本实用新型中，盘体下方在安装槽与盘体边中心处设置驱虫盒。
本实用新型中，定位芯的最大直径为10mm，定位芯采用银色扁式图钉。
本实用新型中，支腿与安装环间中间水平设置隔板。
本实用新型中，隔板呈圆盘状，隔板上设置水平仪。
本实用新型采用的黑色油漆、锡箔纸、橡胶绳、驱虫盒、银色扁式图钉和水平仪均可通过公共渠道购买。
