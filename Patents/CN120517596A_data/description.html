Description

一种低空配送飞行器
技术领域
本发明涉及飞行器技术领域，具体是指一种低空配送飞行器。
背景技术
近年来，随着航空技术的不断进步，飞行器在货物配送领域的应用日益广泛。无人驾驶飞行器(例如无人机)因其高效、灵活以及能够覆盖传统交通工具难以到达的区域等优势，已被广泛用于物流运输、紧急物资投送以及偏远地区的货物配送。飞行器通过预设航线或实时导航，能够快速将货物从起始点运送至目标地点，为现代物流体系提供了全新的解决方案。然而，随着配送需求的增加，如何进一步优化飞行器的货物配送流程成为亟待解决的问题。
在现有的飞行器货物配送过程中，飞行器通常需要降落至地面，以便操作人员或接收者取出货物。然而，由于配送地点地面环境的复杂性，例如崎岖的地形、障碍物的存在或缺乏平坦的着陆区域，飞行器有时难以安全降落。这种情况下，若让飞行器悬浮于空中以避免降落，又面临缺乏有效货物释放手段的困境。现有的技术中，部分飞行器依赖简单的投放机制，但这种方式往往无法保证货物的精准送达或安全性，特别是在需要将货物送至特定位置或避免货物损坏时。因此，亟需一种能够在复杂环境下实现高效、安全货物配送的技术方案。

如图1至图15所示，本发明提供了一种低空配送飞行器，旨在解决现有飞行器货物配送中因地面环境复杂导致降落困难，以及缺乏高效货物释放手段的问题。所述低空配送飞行器包括飞行器本体1、两个支撑架2、货仓3、收放机构4以及吊绳5。
所述飞行器本体1为整个配送飞行器的飞行主体，其包括机体11和旋翼12。所述旋翼12与所述机体11固定连接，所述机体11内部配置有电动机，所述电动机与所述旋翼12传动连接，用于驱动所述旋翼12旋转。通过所述旋翼12的高速旋转，为所述飞行器本体1提供升力，从而使所述飞行器本体1能够在空中飞行或悬停。所述机体11内还可配置有控制模块和电源模块，用于控制飞行路径及供电，具体电路结构和控制方式可采用现有技术，在此不再赘述。
两个所述支撑架2分别与所述飞行器本体1的机体11固定连接，两个所述支撑架2对称设置于所述机体11的下方或两侧，用于支撑所述货仓3。所述货仓3设置于两个所述支撑架2之间，其为用于容纳货物的储物空间，所述货仓3可以是具有开口的箱体结构，开口朝下设置，以便于货物的装载和取出。所述收放机构4安装于所述货仓3的内部，具体地，所述收放机构4可以是卷扬机或电动绞盘等具有收卷和释放功能的装置，其通过电机驱动实现对吊绳5的收放控制。所述吊绳5的一端与所述收放机构4连接，另一端为自由端，用于绑定货物。所述吊绳5的材质可以是高强度纤维绳或钢丝绳，以确保在牵引货物时的可靠性。
所述低空配送飞行器的使用方法如下：在货物装载阶段，首先控制所述飞行器本体1通过旋翼12升空至一定高度，例如距离地面1-2米的低空区域，以避开地面杂物或不平整地形对飞行器的影响。随后，操作所述收放机构4释放所述吊绳5，使所述吊绳5的自由端下垂至接近地面的位置，用户将待配送的货物牢固绑定于所述吊绳5的自由端上。货物绑定完成后，控制所述收放机构4收卷所述吊绳5，所述吊绳5在收卷过程中带动货物向上移动，直至货物被牵引至所述货仓3内部并稳固存放。随后，控制所述飞行器本体1沿预设航线飞行至指定配送地点。
当所述飞行器本体1到达取货用户所在位置时，所述飞行器本体1保持在低空悬停状态，无需降落至地面。操作所述收放机构4再次释放所述吊绳5，所述吊绳5带动货物向下移动至接近地面的高度，用户即可方便地从所述吊绳5上取下货物。货物取下后，可通过所述收放机构4将所述吊绳5收卷回所述货仓3内部，完成配送任务。所述货物的类型可以包括但不限于食品、生活用品、电子产品或药品等，根据实际需求灵活选择。
本发明通过设置所述收放机构4和所述吊绳5，所述飞行器本体1在货物装载和取用过程中无需降落至地面，避免了因地面环境复杂(如存在杂物、凹凸不平或缺乏平坦区域)导致的降落受限问题。同时，所述吊绳5能够灵活调节货物的高度，便于用户在地面进行货物的绑定和取下操作，提高了配送效率和适用性。此外，所述货仓3为货物提供了临时存放空间，确保货物在运输过程中的稳定性，避免了直接投放可能导致的货物损坏或定位不准问题。
在本实施方式中，为进一步优化货物装载和释放的便捷性，所述货仓3的底壁设置有两个可开启的仓门31，所述货仓3的内部设置有第二驱动组件6，所述第二驱动组件6用于驱动两个所述仓门31实现开启或关闭动作。所述第二驱动组件6被配置为驱动所述仓门31向下开启，以形成供货物进出所述货仓3的开口。
具体地，所述第二驱动组件6包括电动推杆61和驱动件62。所述电动推杆61固定安装于所述货仓3的内部，其输出端与所述驱动件62连接，用于驱动所述驱动件62沿上下方向移动。所述驱动件62上开设有第一驱动槽621，每个所述仓门31上固定设置有第一传动臂32，所述第一传动臂32远离所述仓门31的一端通过轴体与所述第一驱动槽621滑动连接。两个所述仓门31通过现有的铰接结构(如铰链或转轴)与所述货仓3的底壁铰接连接，且两个所述第一传动臂32优选位于两个铰接轴体之间，以确保传动稳定性。
所述第二驱动组件6的工作原理如下：当所述电动推杆61的输出端伸出，推动所述驱动件62向下移动时，所述第一驱动槽621随之带动两个所述第一传动臂32滑动并翻转。由于所述第一传动臂32与所述仓门31固定连接，所述第一传动臂32的翻转动作驱动两个所述仓门31绕铰接轴体向下翻转，从而实现所述仓门31的开启。反之，当所述电动推杆61的输出端缩回，带动所述驱动件62向上移动时，所述第一驱动槽621驱动两个所述第一传动臂32反向翻转，进而使两个所述仓门31向上翻转闭合，恢复至关闭状态。
