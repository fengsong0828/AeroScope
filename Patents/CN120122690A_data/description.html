Description

一种重载低空飞行器的高效自主三维路径规划方法和系统
技术领域
本发明涉及无人机技术领域，特别涉及一种重载低空飞行器的高效自主三维路径规划方法和系统。
背景技术
随着无人机技术的飞速发展，低空飞行器在各个领域的应用越来越广泛，尤其是在重载运输、巡检、环境监测、测绘等任务中。然而，低空飞行器在执行任务时面临着复杂的飞行环境，包括障碍物的存在、飞行空间的限制以及需要尽量减少的飞行时间和能耗等问题。这些因素使得低空飞行器的路径规划变得更加复杂和具有挑战性。
目前，低空飞行器的路径规划方法主要包括基于图算法、基于采样的方法以及基于优化的算法等。其中，A*算法作为经典的图搜索算法，在路径规划中得到了广泛应用。A*算法通过结合启发式估计函数和实际路径代价，能够有效找到从起点到终点的最短路径。然而，传统A*算法在处理复杂环境时，存在以下问题：
计算量大：传统A*算法在路径搜索过程中，需要对整个空间进行全面的节点扩展和搜索，即使是在障碍物较少的区域，也会进行大量的无效计算，导致计算量大，效率低下。
路径质量不足：传统A*算法在计算路径时，仅考虑路径的总代价，忽略了飞行器的实际运动能耗和动态变化因素，导致得到的路径可能不是最优的能耗路径，无法完全满足飞行器节能的需求。
路径冗余：在飞行器路径规划过程中，特别是在复杂环境中，传统A*算法可能生成大量冗余转向节点，导致路径的折线过多、距离过长，影响飞行器的飞行效率。
动态避障能力差：传统A*算法在处理动态环境中的障碍物时，往往无法快速适应环境的变化，导致路径规划的灵活性和实时性差。
针对上述问题，尽管已有不少研究在优化路径规划算法方面做出了尝试，但仍然存在以下不足：现有的改进算法通常侧重于某一方面(如优化路径质量或计算效率)，难以兼顾多个目标，特别是在重载低空飞行器的三维路径规划中，既要满足避障需求，又要考虑能耗、路径平滑性和实时性等多种因素。因此，如何在保证计算效率的同时，优化路径的质量，尤其是能耗和冗余转向问题，仍然是一个亟待解决的技术难题。
发明内容
本发明针对现有技术的缺陷，提供了一种重载低空飞行器的高效自主三维路径规划方法和系统。结合飞行器运动模型和能耗预测，通过分段搜索、避障区域定义、关键路径节点筛选等方式，提高路径规划的效率、质量和飞行器的能效，适应重载低空飞行器的实际需求。
为了实现以上发明目的，本发明采取的技术方案如下：
一种重载低空飞行器的高效自主三维路径规划方法，通过改进的A*算法进行路径规划，该方法包括以下步骤：
步骤1：规划区域定义，首先将目标区域进行栅格化，并输入起点与终点。根据理想最优路径(起点与终点连线)标注路径所穿越的障碍物中间区域，并将该区域定义为避障区域。仅在避障区域内进行节点的搜索与成本值计算，非避障区域无需计算节点成本。
步骤2：节点成本值计算，设计分段搜索函数，依照步骤1中定义的避障区域，逐段搜索路径节点并计算每个节点的成本值，最终选择最小总代价路径。节点成本值计算公式为：
F(n)＝G(n)+H(n)+E(n)
其中，G(n)为已知运行距离，H(n)为未知运行距离的启发式估计，E(n)为运动能耗成本。
步骤3：路径后处理，对规划出的路径进行冗余转向节点筛选，通过关键路径节点的筛选机制去除不必要的转向，最终获得更短且高效的路径。
进一步地，避障区域的定义基于理想最优路径穿越的障碍物中间区域，并通过设置权重因子，控制避障区域的精度。
进一步地，所述分段搜索函数用于对避障区域进行分段计算，计算的总代价包括起点到节点的实际代价、节点到终点的启发式代价和运动能耗代价。每个节点的选择依赖于最小总代价准则。

进一步地，步骤2中的启发式估计函数H(n)采用欧几里得距离计算，表示从当前节点到终点的最短路径距离。
进一步地，步骤2中的能耗计算E(n)基于飞行器的动力学模型和运动姿态分析，采用非线性映射关系量化飞行器的运动能耗，并通过物理实验数据提高能耗估计精度。具体计算为：
E(nn)＝F[E(nn),N(nn)]+ΔE(nn)
其中，F[E(nn),N(nn)]表示能耗与飞行驱动电机转速之间的非线性映射关系，ΔE(nn)为估计能耗与实际能耗之间的误差。
进一步地，步骤3中的关键路径节点筛选机制，通过对路径中连续节点进行空间关系判断，若相邻节点间无障碍物，则合并为一条直线，从而筛选出关键节点，减少冗余转向节点，提高路径质量。
进一步地，步骤2中的节点成本值计算过程中，使用对径向基函数神经网络(RBFNN)对飞行器在不同节点的运动能耗进行预测，并通过非结构剪枝优化方法提高预测精度和计算效率。

与现有技术相比，本发明的优点在于：
1.通过对A*算法的改进，利用理想最优路径作为参考，仅在避障区域内进行节点搜索与成本值计算，显著减少了无关区域的计算量，从而提高了路径搜索的效率。
2.引入了分段搜索函数和节点成本值计算机制，综合考虑已知运行距离、未知运行距离的启发式估计以及运动能耗，能够有效规划出总代价最小的路径，确保飞行器能够在复杂环境中高效、准确地避开障碍物。
3.通过路径后处理模块中的关键路径节点筛选机制，去除不必要的转向，减少冗余路径节点，确保飞行器的飞行路径更加平滑和直观，从而提升飞行效率，减少能耗。
4.本发明通过引入运动能耗成本计算，基于飞行器的运动模型和能耗数据，准确量化能耗变化规律，为低空飞行器的飞行规划提供了更加实际的能耗预估，优化了飞行器的能效。
5.本发明能够自主进行路径规划、避障以及后续路径优化，减少了人工干预，提高了飞行器在复杂和动态环境中的适应能力和自主决策能力。

结果显示模块，用于显示规划出的最优路径，展示路径上的关键节点及路径的相关信息，提供用户直观的路径规划结果，并可根据需要进行进一步的路径调整。
本发明再一个实施例中，提供了一种终端设备，该终端设备包括处理器以及存储器，所述存储器用于存储计算机程序，所述计算机程序包括程序指令，所述处理器用于执行所述计算机存储介质存储的程序指令。处理器可能是中央处理单元(Central ProcessingUnit，CPU)，还可以是其他通用处理器、数字信号处理器(Digital Signal Processor、DSP)、专用集成电路(Application Specific Integrated Circ uit，ASIC)、现成可编程门阵列(Field-Programmable GateArray，FPGA)或者其他可编程逻辑器件、分立门或者晶体管逻辑器件、分立硬件组件等，其是终端的计算核心以及控制核心，其适于实现一条或一条以上指令，具体适于加载并执行一条或一条以上指令从而实现相应方法流程或相应功能；本发明实施例所述的处理器可以用于重载低空飞行器的高效自主三维路径规划方法的操作。
本发明再一个实施例中，本发明还提供了一种存储介质，具体为计算机可读存储介质(Memory)，所述计算机可读存储介质是终端设备中的记忆设备，用于存放程序和数据。可以理解的是，此处的计算机可读存储介质既可以包括终端设备中的内置存储介质，当然也可以包括终端设备所支持的扩展存储介质。计算机可读存储介质提供存储空间，该存储空间存储了终端的操作系统。并且，在该存储空间中还存放了适于被处理器加载并执行的一条或一条以上的指令，这些指令可以是一个或一个以上的计算机程序(包括程序代码)。需要说明的是，此处的计算机可读存储介质可以是高速RAM存储器，也可以是非不稳定的存储器(non-volati le memory)，例如至少一个磁盘存储器。
可由处理器加载并执行计算机可读存储介质中存放的一条或一条以上指令，以实现上述实施例中有关重载低空飞行器的高效自主三维路径规划方法的相应步骤；计算机可读存储介质中的一条或一条以上指令由处理器加载并执行。
