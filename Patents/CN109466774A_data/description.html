Description

一种农药喷洒的智能低空飞行器
技术领域
本发明涉及农业机械领域，具体涉及一种农药喷洒的智能低空飞行器。
背景技术
我国是一个地大物博的国家，拥有很多土地，需要更多人力资源和水资源去维护所需要保护的植被。随着农业现代化的进程加快，在农业机械领域出现了很多方便实用的机械，虽然在农药喷洒技术中出现了无人机自动化喷洒、履带农药喷洒机械等，但是出现了只限于向下喷洒，不能多方位喷洒等缺点，且喷水范围小，当需要大范围喷洒时难度高，人力和物力需求量大。
发明内容
为了解决现有技术中存在的某种或某些技术问题，本发明提供一种农药喷洒的智能低空飞行器，能够解决现有农药喷洒飞行器不能多方位喷洒的问题，喷洒范围大，能够有效的节省人力和物力，生产成本低，工作效率高。
为解决上述现有的技术问题，本发明采用如下方案：
一种农药喷洒的智能低空飞行器，包括机架、设在机架底部的支撑架、设在所述机架左右两侧的飞行装置、设在所述机架上的蓄水箱、与所述蓄水箱连通的喷洒装置，所述喷洒装置包括水泵、吸水管、输水管和用于将农药喷洒出去的喷水装置，所述输水管和喷水装置之间设有缓冲箱、所述缓冲箱的底部设有压力变送装置，所述压力变送装置的底部设有一个与所述机架连接的装置箱，所述装置箱内设有旋转装置，所述喷水装置通过所述旋转装置进行旋转，所述吸水管的两端分别连接所述蓄水箱和所述水泵，所述输水管的两端分别连接所述水泵和所述缓冲箱，所述机架内设有终端控制器、对所述飞行装置的飞行方向及速度进行控制的飞行控制器、用于供电的蓄电池、用于检测飞行环境的测距仪、用于接收电信号并转发给所述终端控制器的信号接收器、与所述终端控制器连接用于报警的蜂鸣警报器、以及分别设在所述机架前后两端的前置摄像头和后置摄像头，所述测距仪在飞行过程中对障碍物的距离检测并将检测到的信息传递到终端控制器，当理障碍物过近时通过所述终端控制器控制蜂鸣警报器发出警报，所述前置摄像头和后置摄像头用于对采集外部图像信息并且通过所述信号接收器收发信号，从而对所述飞行装置的飞行状态进行控制。
进一步地，所述机架的左右两侧各设有一块护板，所述蓄水箱采用两个，两个所述蓄水箱分别设置在所述机架顶部两侧并连着护板内两侧，所述蓄水箱的顶部中间设有支撑板，所述水泵固定在所述支撑板上方，所述蓄水箱前面设有进水口和水位测量仪。
进一步地，所述飞行装置包括与所述机架连接的螺旋装置、设在所述螺旋装置一端的飞行机翼、设在所述飞行机翼内的的电动机和通过所述电动机带动旋转的螺旋桨，所述螺旋桨和电动机至少采用两组且对称设置在所述螺旋装置的两侧，所述螺旋装置和所述电动机均通过电路与所述飞行控制器电性连接。
进一步地，所述飞行机翼上设有对所述蓄电池进行供电的太阳能电池板，所述太阳能电池板设在所述飞行机翼的中间。
进一步地，所述机架内设有通过电路与所述太阳能电池板是否对所述蓄电池进行充电的太阳能控制器，所述太阳能电池板的采光面为弧形结构。
进一步地，所述机架上设有与所述终端控制器电性连接对所述蓄电池进行充电控制的充电芯片、与所述充电芯片连接用于充电的充电口、与所述充电芯片电性连接用于显示所述蓄电池充电情况的充电信号灯，所述充电口接入电源时通过所述终端控制器自动控制所述太阳能控制器切断充电功能。
进一步地，所述机架内还设有对所述蓄电池的电量进行监控的电量检测器，所述电量检测器通过电路与所述终端控制器电性连接，所述充电信号灯通过电路与所述电量检测器电性连接，所述电量检测器检测到所述蓄电池电量过低时，将低电量信号发送给所述终端控制器对所述蜂鸣警报器发出报警信号。
进一步地，所述喷水装置包括设在所述缓冲箱上的喷柱、设是所述喷柱顶端的第一喷头、至少三根环设在所述喷柱或第一喷头上的延伸管道、设在所述延伸管道顶端的第二喷头，所述第一喷头和所述第二喷头的喷水口均朝上设置。
进一步地，第一喷头和所述第二喷头的喷水面为半椭圆型，且所述第一喷头和所述第二喷头的外部喷孔大于内部喷孔。
进一步地，所述缓冲箱与所述喷柱之间设有伸缩管道，所述伸缩管道的接口处分别设有橡胶圈。
进一步地，所述充电芯片通过电路与所述飞行控制器电性连接。
进一步地，所述机架上设有对所述水泵的状态进行显示的开关信号灯，所述开关信号灯通过电路与所述终端控制器电性连接。
进一步地，所述蓄水箱、水泵吸水管、输水管、装置箱、压力变送器、缓冲箱、伸缩管道和喷柱的材料均为不锈钢材料。
相比现有技术，本发明的有益效果在于：
在工作时，可以根据所需要求，调整飞行器的角度，使药水可以洒到所需位置；结果简单，采用半椭圆不同大小孔眼向上的喷头，在通过旋转装置高速旋转喷柱使喷出的药水呈旋涡状均匀横向耍出，有效的弥补了无人机喷洒只能向下喷洒的弊端；通过在机架前侧设置的测距仪、蜂鸣警报器和接受信号装置，以及内部连接蓄电池的电量检测器，当底板前侧距离障碍物的距离低于设置值，则测距仪将信息传递到控制器，控制器控制蜂鸣警报器发出警报，当蓄电池电量低于设定电量，则将信息传递到控制器，控制器控制蜂鸣警报器发出警报，可有效避免事故，通过前置和后置摄像头控制视野，通过接受信号灯传递信息，然后调整飞行器倾斜方向，生成飞行指令传送到飞行控制器，飞行控制器控制各个电动机功率来实现设备的飞行方向，该飞行器可以在复杂地形飞行，可以避免对土地的挤压。它可以不受区域限制，可以调节不同角度方便喷洒，优于以往的无人机和其它农药喷洒技术，可以大范围使用，节省了人力和物力。

在实际使用过程中，支撑架1安装在机架2的底部，用于在着陆或者未起飞时起到支撑的效果，从而防止机架2直接与地面进行接触，有效的避免了环境对飞行器的影响，使飞行器使用寿命更长，飞行装置39安装在机架2的两侧，主要用于起到飞行和变向的目的，在飞行过程中平稳性能够更高，农药存储在蓄水箱17内，在飞行过程中通过喷洒装置40将蓄水箱17内的农药向外界喷洒，在机架2内安装有终端控制器32、飞行控制器33、蓄电池30、测距仪12、信号接收器9、蜂鸣警报器14、前置摄像头13和后置摄像头34，其中蓄电池30用于对所有需要用电的部件进行供电处理，而终端控制器32为主控制器，用于接收、处理和发送各种信号指令，飞行控制器33用于单独对飞行装置39的飞行状态进行控制，测距仪12用于检测飞行器在飞行过程中与其他物体之间的距离，而前置摄像头13和后置摄像头34则分别用于采集飞行器前后两端的周围环境，信号接收器9用于在飞信器和外部遥控设备之间的信号收发，当需要起飞时，外部遥控设备直接发出信号给信号接收器9，通过信号接收器9将信号再传输到终端控制器32货飞行控制器33内对飞信器进行控制，当飞信器在飞行过程中，可以根据所需要求，通过飞行控制器33调整飞行装置39的位置从而实现飞行器的角度，使药水可以洒到所需位置，结果简单，当机架2前侧距离障碍物的距离低于设置值，则测距仪12将信息传递到终端控制器32，通过终端控制器32控制蜂鸣警报器14发出警报，从而使操作者能够及时的对飞行位置进行调整，可有效避免事故发生，通过前置摄像头13和后置摄像头34对飞行视野进行控制并将采集的信息通过信号接收器9传送到外部遥控设备上，从而方便操作者远程对飞行器进行飞行位置进角度的调整，使该飞行器可以在复杂地形飞行，可以避免对土地的挤压，在飞行过程中通过喷洒装置40进行农药的喷洒时，吸水管20的一端伸入蓄水箱17，农药通过水泵19从蓄水箱17内经过吸水管20吸出到输水管21内，然后在进入缓冲箱24，再通过缓冲箱24的底部的压力变送装置23对水泵19打出的弄呀进行增压处理，从而使农药通过喷水装置41喷洒出去时，压力更大且范围更广，动力更足，喷洒的距离控制更加方便，喷水装置41的底部通过装置箱22与机架2连接，稳定性更高，且装置箱22内设有对喷水装置41进行控制的旋转装置37，通过旋转装置37能够实现对喷水装置41的旋转，使喷水更加均匀，旋转装置37与喷水装置41之间采用电性连接，测距仪12可为红外线测距仪12或者声波测距仪12。
作为优选，在机架2的左右两侧各设有一块护板4，且蓄水箱17采用两个，两个蓄水箱17分别设置在所述机架2顶部两侧并连着护板4内两侧，两个蓄水箱17内的农药通过各自的水泵19汇集到缓冲箱24内，再进行增压打出，在喷洒时效率更高，供药效果更好，且蓄水箱17在飞行过程中稳定性更好，在飞行时整个设备的平稳性更好。在蓄水箱17的顶部中间设有支撑板18，所述水泵19固定在所述支撑板18上方，能够使水泵19的安装更加方便，稳定性更高，且吸水管20和输水管21的安装更加方便简单，在蓄水箱17前面设有进水口15和水位测量仪16，对于蓄水箱17内的药液量能够直接读取并通过进水口15进行加药处理，结构更加符合人们的生活习惯。
作为优选，飞行装置39由螺旋装置5、飞行机翼3、电动机8和螺旋桨7组成，通过飞行控制器33能够对螺旋装置5进行转动，从而使飞行器实现各种方向及角度的飞行控制和机架2的调整，而飞行机翼3内的螺旋桨7和电动机8至少采用两组且对称设置在所述螺旋装置5的两侧，在飞行时能够更加稳定，飞行动力更足，飞行控制器33可选用普通的四旋翼飞行控制器33，终端控制器32为CPU，内设置运行程序，可以通过接受信号器接收所需调整的倾斜角度，发出信号给飞行控制器33对螺旋装置5进行调节。
进一步的，在飞行机翼3上设有对蓄电池30进行供电的太阳能电池板6，且太阳能电池板6设在所述飞行机翼3的中间，采光效果更好，空间利用率更高，且在飞行的过程中通过太阳能电池板6能够持续的对蓄电池30进行充电电，从而使飞行器的飞行时间更长，延长设备的续航能力。同时，为了防止飞行在通过电源进行直接充电时太阳能电池板6也同时进行充电，从而使电池充电功率不好控制，在机架2内安装了一个通过电路与终端控制器32连接对太阳能电池板6是否对蓄电池30进行充电的太阳能控制器31，当蓄电池30直接通过电源进行充电时，可以通过太阳能控制器31对太阳能电池板6进行断电处理，当电源拔出后，再切换到充电模式，太阳能充电的控制更加方便，同时，通过信号接收器9也能够直接由外部控制设备进行是否充电的控制，控制更加灵活，而太阳能电池板6的采光面为弧形结构，采光面积能够更大，充电效率更高。
更进一步的，蓄电池30更换电池和在采用太阳能充电两种方式外，还有采用电源直接充电的方式，同感电源充电时，直接将电源数据线掺入充电口11内，电源通过充电口11对蓄电池30进行充电，同时，在蓄电池30和充电口11之间还设有一个充电芯片36，通过充电芯片36能够对充电情况进行控制，在配合与所述充电芯片36电性连接用于显示所述蓄电池30充电情况的充电信号灯10，当充电接口在充电时，充电信号灯10为红色，当充满时，充电信号灯10为绿色，同时，采用太阳能电池板6和充电口11进行充电时，均通过充电芯片36进行控制，当充电口11接入电源时，通充电制芯片发出信号给所述终端控制器32，通过终端控制器32再自动控制所述太阳能控制器31切断充电功能，能够有效的实现两者之间的快速精确切换。
再进一步的，在架内还安装有对所述蓄电池30的电量进行监控的电量检测器35，且电量检测器35通过电路与终端控制器32电性连接，当电量检测器35到蓄电池30的电量低于设定值时，通过终端控制器32控制蜂鸣警报器14发出报警信号。
