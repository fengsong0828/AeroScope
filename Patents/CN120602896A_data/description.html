Description

低空飞行器动态轨迹数据压缩传输与预测方法
技术领域
本发明涉及低空飞行器无线通信技术领域，更具体地说，涉及低空飞行器动态轨迹数据压缩传输与预测方法。
背景技术
随着低空飞行器应用的普及，除了传统的应用领域得到进一步拓展外，私人与公务低空飞行器的需求不断增长，同时低空飞行器在火灾救援、海上急救、缉私缉毒、消防、医疗救助等公务方面的应用也在很快展开。
目前低空飞行器与控制中心通信通常采用甚高频通信、高频通信和无线网络通信相结合的方式，其中，甚高频通信距离太近且受飞行高度影响较大，高频通信因短波信号不稳定、电台间相互干扰，通信质量差，无线网络受地域制约，在森林、山区等移动网络无法覆盖的地方无法使用。
而随着北斗二号系统开通运行后，虽能保证低空飞行器通信的连续性，为使用部门提供更直接有效的监控和指挥调度，但目前民用北斗卡频度较高，短报文容量有限，传输数据量受到严重制约，在这种情况下，如何高效压缩低空飞行器动态轨迹数据并进行可靠传输，同时对轨迹进行准确预测成为急待解决的问题，鉴于此，我们提出一种低空飞行器动态轨迹数据压缩传输与预测方法。
发明内容
本发明的目的在于提供一种低空飞行器动态轨迹数据压缩传输与预测方法，旨在解决现有技术难以高效压缩低空飞行器动态轨迹数据并进行可靠传输，同时对轨迹进行准确预测的问题。
为解决上述技术问题，本发明提供如下技术方案：一种低空飞行器动态轨迹数据压缩传输与预测方法，该方法包括以下步骤；
S1、对低空飞行器的动态轨迹数据进行特征提取和压缩处理，所述压缩处理采用基于时空相关性的压缩方法去除冗余数据；
S2、通过指挥调度中心配置北斗指挥型用户机、低空飞行器配置下属北斗用户机的中继方式实现利用北斗卫星系统将压缩后的数据传输至指挥调度中心；
S3、然后通过结合北斗系统精准授时功能，利用实时采集的飞行数据及历史轨迹数据，采用线性外推预测模型分析飞行器运动趋势并预测未来轨迹。
优选的，在上述步骤S1中，所述特征提取包括提取飞行高度、速度、加速度、经纬度坐标信息。
优选的，在上述步骤S1中，通过判断公式是否成立来确定时空相关性，式中，为轨迹点与的空间距离，为误差系数，为平均飞行速度，当满足该条件时，表明两点具有时空相关性，可进行压缩处理，从而减少数据冗余。
优选的，在上述步骤S2中，所述数据传输过程中，采用单机多卡轮换连续定位的方式，将定位频度缩小至10秒之内，使定位点数增加，误差缩小。
优选的，在上述步骤S2中，所述北斗指挥型用户机可容纳100-1000个用户终端，实现位置监控、通播、授时、轨迹回放及数据管理功能。
优选的，在上述步骤S3中，所述预测模型公式为，式中，为预测的时刻位置，为当前位置，为当前速度，为当前加速度，为预测时间间隔。
优选的，在上述步骤S2中，所述压缩方法还包括以下步骤：
S1.1、采用自回归模型对时间上相邻的轨迹点数据进行分析，利用历史轨迹点数据来预测当前轨迹点数据值，对于具有较强自相关性的轨迹数据时间序列，通过计算预测值与实际值之间的残差，并对残差进行编码存储，从而减少时间维度上的冗余信息；
S1.2、将飞行器飞行区域划分为多个空间块，当判断相邻轨迹点在同一空间块或相邻空间块内，且满足时空相关性条件时，利用空间相关性进一步减少数据冗余；
S1.3、构建轨迹数据特征字典，将常见的轨迹数据特征模式存储在字典中，在压缩过程中，对于符合字典中模式的轨迹数据部分，直接引用字典中的索引，以此减少数据存储量。
优选的，在上述步骤S1.1中，当前轨迹点为，其自回归预测模型可表示为，式中，为自回归系数，为过去个时间步的轨迹点数据，为预测残差 。
与现有技术相比，本发明的有益效果是：
1、本发明通过基于时空相关性的压缩方法去除冗余数据，有效减少了数据量，提升了传输效率，同时结合北斗精准授时功能及线性外推预测模型，实现了对飞行器未来轨迹的准确预测，满足了低空飞行器在通信传输和轨迹预测方面的迫切需求。
2、本发明在数据压缩处理中，采用自回归模型分析时间相邻轨迹点数据，通过计算残差并编码存储，减少时间维度冗余，将飞行区域划分为空间块，利用空间相关性进一步降低数据冗余，构建轨迹数据特征字典，引用字典索引减少数据存储量，多维度压缩方式显著提高了数据压缩效率。
3、本发明在数据传输过程中，采用单机多卡轮换连续定位方式，将定位频度缩小至10秒之内，增加定位点数，缩小误差，同时北斗指挥型用户机容纳100-1000个用户终端，实现位置监控、通播、授时、轨迹回放及数据管理等功能，保障了数据传输的可靠性和定位的精度。

S2、通过指挥调度中心配置北斗指挥型用户机、低空飞行器配置下属北斗用户机的中继方式实现利用北斗卫星系统将压缩后的数据传输至指挥调度中心，以借助北斗卫星系统的中继传输，在低空环境中实现稳定的数据传输，尤其适用于信号覆盖较差的区域，保障数据传输的可靠性；
