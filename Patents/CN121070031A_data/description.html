CN121070031A

Description

一种低空垂直起降飞行器动态安全间隔设置方法、设备、介质 及产品
技术领域
本申请涉及飞行器安全技术领域，特别是涉及一种低空垂直起降飞行器动态安全间隔设置方法、设备、介质及产品。
背景技术
低空垂直起降飞行器是指能在低空(通常1000米以下空域)无需长跑道就能实现垂直起飞和降落的飞行器。现有的低空垂直起降飞行器，其安全区域是固定的，不能变化。
为了应对现有三维间隔保护区域技术的局限性，本领域亟需一种能够变化的安全区域，以最大限度地利用空域资源。本研究提出了一种动态三维间隔保护区域设计方案。该方案能够实时调整三维间隔保护区域的大小，确保飞行器在飞行过程中始终处于安全状态，并且最大限度地利用空域资源，从而满足未来民航领域对飞行器安全监管的需求。
发明内容
本申请的目的是提供一种低空垂直起降飞行器动态安全间隔设置方法、设备、介质及产品，可依据飞行器性能的不同，使得飞行器周围的安全区域发生实时变化，并可以最大限度地利用空域资源。

现有技术中的安全区域是固定的，不能随CNS(CommunicationNavigationSurveillance)变化而变化。本申请依据CNS能力的不同，可以使得飞行器周围的安全区域发生实时变化。例如在导航精度发生波动时，可以通过计算来使得安全区域也随之改变。

在一个示例性的实施例中，如图1所示，提供了一种低空垂直起降飞行器动态安全间隔设置方法，包括以下步骤S1至步骤S7。其中：
S1、根据飞行器的机身尺寸与安全冗余系数，确定每个飞行器的保护区内层。
本实施例中，建立垂直起降飞行器的保护区模型；以得到飞行器安全区域，即保护区的确定。
基于飞行器动力学特性的双层保护区，如下图2和图3所示，其中，保护区外层为冲突告警层，通过模拟入侵机不同的运动模式下的飞行轨迹，确定其航迹可达范围；保护区内层为碰撞层，参照飞行器规格参数及机载设备性能，满足位置不确定情况下的机身防撞需要。
飞行器避撞过程分为3个阶段：态势感知、冲突预测和避撞机动。
基于上述阶段，设计飞行器双层保护区来模拟避撞过程，利用外层保护区进行避撞决策判断，防止间距继续缩小至内层保护区。针对飞行器飞行速度与方向随机、避让灵活性大等特点，设计飞行器双层保护区，具体包括表征航迹可达区域的外层冲突告警层和表征飞行器位置不确定性的内层碰撞层。当飞行器在探测范围内感知得到入侵机实时飞行状态信息(通过机载传感器等可信系统获取位置、航向、速度等)后，将对入侵机飞行航迹进行跟踪，在入侵机与飞行器进一步接近到一定距离(即保护区外层)时，飞行器将采飞行措施，防止两机距离进一步缩小(即保护区内层)造成机身碰撞。
外层保护区由预测入侵机航迹可达区域确定。入侵机的航迹可达区域表征了给定时间内入侵飞行器可能覆盖的威胁范围，随其机动性能(如转弯速度、倾斜角度)等动态变化。通过合理预测入侵机可达范围，确保飞行器触及保护区外层时及时触发避让动作，能够降低后续碰撞发生的可能性。

Rinner＝1.1Rspan；
式中，Rinner为保护区内层半径，Rspan为飞行器的机身尺寸半径。式中，基准飞行器机身尺寸的系数设为1.1(安全冗余系数)，旨在为气象因素、操作裕度和其他误差因素提供额外的安全裕度。

本实施例相较于传统CNS和碰撞方式有以下三方面的提升:
①安全性提高：
通过根据飞行状态和环境动态调整三维间隔保护区域的大小，飞行器能够始终处于安全状态。相比传统的固定三维间隔保护区域，该方案能实时应对复杂和变化的飞行环境，避免了空域资源浪费。当飞行在空旷区域时，三维间隔保护区域缩小，提高空域利用率；而在遇到潜在风险时，及时扩大三维间隔保护区域的范围，有效保护飞行安全。此外，方案引入了基于4D轨迹的飞行方式和性能评估标(PBC/PBN/PBS/PBA)，确保飞行器在飞行过程中实现轨迹一致性和安全分离，减少碰撞风险。
②飞行容量增大：
该方案通过动态调整风险体积，实现碰撞风险的量化。风险区域出现时，能够立即触发避碰措施，确保飞行任务的精确度和可靠性。在空旷区域时，三维间隔保护区域缩小，提高空域利用率，从而增加飞行容量。同时，考虑到通信延迟、环境干扰等因素，方案能够更精确地反映飞行中的不确定性，并为碰撞风险评估提供可靠依据。
③适应性增强：
方案将飞行器的传感器、导航系统和避碰功能有机结合，能够根据传感器反馈的信息实时调整飞行策略，增强了飞行器在复杂环境中的适应能力。在面对不同飞行环境和潜在风险时，能够灵活调整飞行路径和避碰策略，提高了飞行控制的主动性和精准性。
