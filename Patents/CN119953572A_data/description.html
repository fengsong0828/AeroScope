Description

低空飞行器传动系统
技术领域
本发明涉及到飞行器技术领域，具体涉及到一种低空飞行器传动系统。
背景技术
随着低空经济的蓬勃发展，对低空飞行器的应用场景不断涌现，已渗透到国民经济的各行各业。目前市面上常用的低空飞行器的传统系统带有传动减速器，传动效率低，传递的最大扭矩或最大功率小，传动比不稳定且质量大。不符合目前低空飞行器发展的高使用需求。
因此，开发一款具有高可靠性、高效率、轻量化的低空飞行器传动系统是低空飞行器领域的迫切需求。
发明内容
本发明的目的是针对现有技术对应的不足，提供一种低空飞行器传动系统。这种低空飞行器传动系统满足轻量化的同时具有高功率密度、高耐久性。
本发明的目的是采用下述方案实现的：一种低空飞行器传动系统，包括两平行设置的一级传动系统，所述一级传动系统的顶端设置二级传动系统，两二级传动系统通过互联轴连接，前端的二级传动系统与前旋翼传动系统连接固定，后端的二级传动系统与后旋翼传动系统连接固定，所述一级传动系统包括一级减速器，一级减速器的输出轴与一级传动轴的下端连接固定，所述一级传动轴的上端与二级传动系统中的二级减速器的输入轴连接固定，两二级减速器通过互联轴连接，所述二级减速器的输出轴通过二级传动轴分别与前旋翼传动系统中的前旋翼减速器以及后旋翼传动系统中的后旋翼减速器连接固定，所述前旋翼减速器与前旋翼杆连接固定，所述后旋翼减速器与后旋翼杆连接固定。
所述一级减速器的输出轴通过膜片联轴器一级传动轴连接，一级传动轴通过膜片联轴器与二级减速器的输入轴连接。
所述二级减速器与互联轴以及二级传动轴通过膜片联轴器连接，二级传动轴通过膜片联轴器分别与前旋翼减速器以及后旋翼减速器连接。
所述一级传动轴、二级传动轴、互联轴、前旋翼杆以及后旋翼杆为中空结构。
所述二级减速器中设置单向超越离合器。
所述一级减速器的输入轴通过联轴器与发动机或电机连接。
所述二级减速器、前旋翼减速器以及后旋翼减速器处于同一轴线上。
本发明的优点在于，在满足轻量化的同时具有高功率密度、高耐久性等特性。动力由发动机或电动机输出，经过各级减速器后，将动力传递到螺旋桨，实现机械能和螺旋桨动能的转换，推动飞行器上升和前进。

具体实施方式
如图1所示，一种低空飞行器传动系统，包括两平行设置的一级传动系统a，所述一级传动系统a的顶端设置二级传动系统b，两二级传动系统b通过互联轴11连接，前端的二级传动系统b与前旋翼传动系统c连接固定，后端的二级传动系统b与后旋翼传动系统d连接固定，所述一级传动系统a包括一级减速器1，所述一级减速器1的输入轴通过联轴器8与发动机或电机连接。一级减速器1的输出轴与一级传动轴3的下端连接固定，所述一级减速器1的输出轴通过膜片联轴器2一级传动轴3连接，所述一级传动轴3的上端与二级传动系统b中的二级减速器4的输入轴连接固定，一级传动轴3通过膜片联轴器2与二级减速器4的输入轴连接。两二级减速器4通过互联轴11连接，所述二级减速器4的输出轴通过二级传动轴5分别与前旋翼传动系统c中的前旋翼减速器9以及后旋翼传动系统d中的后旋翼减速器6连接固定，所述二级减速器4、前旋翼减速器9以及后旋翼减速器6处于同一轴线上。所述二级减速器4与互联轴11以及二级传动轴5通过膜片联轴器2连接，二级传动轴5通过膜片联轴器2分别与前旋翼减速器9以及后旋翼减速器6连接。所述二级减速器4中设置单向超越离合器。所述前旋翼减速器9与前旋翼杆10连接固定，所述后旋翼减速器6与后旋翼杆7连接固定。所述一级传动轴3、二级传动轴5、互联轴11、前旋翼杆10以及后旋翼杆7为中空结构。
本发明传动系统的动力由并联的前旋翼减速器和后旋翼减速器实现，一级减速器的动力由发动机或电动机输入，一级减速器的动力经过联轴器和传动轴传递到二级减速器，二级减速器将动力同步输出给前旋翼减速器以及后旋翼减速器，前旋翼减速器与后旋翼减速器将动力分别通过前旋翼杆以及后旋翼杆输出给螺旋桨。在二级减速器中配置有单向超越离合器，防止任何一级螺旋桨动力终端时，动力反向输入。
