Description

一种反低空飞行器抓捕弹
技术领域
本发明涉及一种反低空飞行器抓捕弹，属于无人机技术领域。
背景技术
虽然现在的反无人机的技术正在飞速的发展，但是当前的一些技术任然存在这一些问题，例如检测控制的方法(对私人编制的无人机作用有限)，直接摧毁类(危险性高，要求精度、耗费、耗能高)，干扰阻挠类(作用有限)。随着技术的进步，无人机的普及度越来越高，使用门槛也越来越低，几百元到几千元不等的平民化价格，更促使无人机成为了人们生活中的“玩具”。然而人们在领略高空美景的同时，一不小心就可能给航空飞行带来重大的安全隐患，特别是在机场禁空区内，飞机起降过程中如果与无人机发生碰撞，后果将不堪设想。消费级无人机与工业级无人机均有一定的载荷与续航能力，如果被别有用心的人在重要敏感场所用来偷拍、恐袭、投毒、抛洒传单等，将会对社会治安与公共安全造成重大威胁。无人机探测与反制技术全球发展失衡，常规雷达探测目标主要为导弹、飞机等高空、快速、大目标(简称高大快目标)，与之相反的无人机则属于低空、慢速、小目标(简称低慢小目标)，体积越小、速度越慢、飞行高度越低，探测难度越大。低慢小目标的反射截面积非常小，所以传统雷达无法有效探测，容易侵入禁飞区造成重大安全隐患。
因此，针对上述问题提出一种反低空飞行器抓捕弹。
发明内容
本发明的目的就在于为了解决上述问题而提供一种反低空飞行器抓捕弹。
本发明通过以下技术方案来实现上述目的：
一种反低空飞行器抓捕弹，包括主弹体、二级弹体、抓捕网、第一发射部、第二发射部和质量块；
所述主弹体内设置有主腔体，所述抓捕网设置在所述主腔体内，
所述二级弹体设置在所述主弹体的下方，所述二级弹体和所述主弹体的连接端设置有相互配合的凸台和凹槽，所述凸台和凹槽过盈配合；
所述二级弹体的底部设置有第一发射部，所述第一发射部的开口朝下；
所述二级弹体的侧面设置有第二发射部；
所述质量块与所述抓捕网的四角通过牵引绳固定连接，所述质量块设置在所述第二发射部内。
更进一步的，所述第二发射部的开口倾斜向上。
更进一步的，所述质量块的数量为四个，四个所述质量块分别通过牵引绳分别连接所述抓捕网的四角。
更进一步的，所述第二发射部包括四个发射管，四个所述质量块分别设置在四个所述发射管内。
更进一步的，所述二级弹体的四周设置有导向槽。
更进一步的，所述导向槽的数量为四个。
更进一步的，四个所述导向槽分别设置在四个所述发射管的正上方。
更进一步的，所述质量块的底部设置有挡块。
更进一步的，还包括胶塞，所述胶塞的一端设置有堵头，另一端与所述主弹体连接，所述堵头设置在所述第二发射部的开口处。
发明原理：本发明的反低空飞行器抓捕弹使用时，首先发射主弹体和二级弹体，在主弹体和二级弹体发射之后，在预定的时间到达的时候发射质量块，由四个质量块牵引展开抓捕网，抓捕网抓捕目标飞行器。
有益效果：本发明的反低空飞行器抓捕弹结构简单，设置合理，采用两级分离发射，利用质量块的牵引力带动抓捕网张开，从而达到抓捕无人机的目的，能够有效打击低空无人机。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
请参阅图1-6所示，本发明的反低空飞行器抓捕弹，包括主弹体1、二级弹体2、抓捕网、第一发射部3、第二发射部4和质量块；
主弹体1内设置有主腔体11，抓捕网13设置在主腔体11内，
二级弹体2设置在主弹体1的下方，二级弹体2和主弹体1的连接端设置有相互配合的凸台21和凹槽12，凸台21和凹槽12过盈配合；
二级弹体2的底部设置有第一发射部3，第一发射部3的开口朝下；
二级弹体2的侧面设置有第二发射部4；
质量块与抓捕网的四角通过牵引绳固定连接，质量块设置在第二发射部4内。
优选的，第二发射部4的开口倾斜向上。用于发射时，利用发射管的反作用力使上下两个弹体的结构分离，便于抓捕网的顺利展开，实现抓捕功能。
优选的，质量块的数量为四个，四个质量块分别通过牵引绳分别连接抓捕网的四角。
优选的，第二发射部4包括四个发射管，四个质量块分别设置在四个发射管内。利用四个发射管的反作用力使上下两个弹体的结构分离，便于抓捕网的顺利展开，实现抓捕功能。
优选的，二级弹体2的四周设置有导向槽22。
优选的，导向槽22的数量为四个。
优选的，四个导向槽22分别设置在四个发射管的正上方。
优选的，质量块的底部设置有挡块。
优选的，还包括胶塞5，胶塞5的一端设置有堵头，另一端与主弹体1连接，堵头设置在第二发射部4的开口处。胶塞既可以对第二发射部的开口进行封堵，又可以利用其对主弹体和二级弹体起到一定的连接作用。
具体的，
一、抓捕弹的主腔体11：主腔体11主要是用来储存所要发射的抓捕网；
二、抓捕弹的连接部位：在主弹体1发射之后，要完成主弹体1与二级弹体2的分离才能使抓捕网顺利的展开并实施抓捕作用，在主弹体1和二级弹体2的连接部位，如图1和图2所示：在主弹体1和二级弹体2之间，只要采用凸台21和凹槽12的过盈配合实现主弹体1和二级弹体2的周向定位，以此使得弹体不发生周向的扭动。在轴向定位上为了使弹体的上下两部分既能够紧密连接而且要便于后期的二级分离，我们采用胶塞5，在质量块的发射口和上弹体之间实现连接，用于牵引抓捕网的四角展开，实现抓捕的功能。
如图所示：将二级弹体2的上端与主弹体1的下部连接，二者之间通过套接方式连接，利用发射时所产生的冲击力是上下两部分弹体分离。与此同时，利用四个质量块的牵引力，使抓捕网张开，从而达到抓捕无人机的目的。
三、弹的发射部位：对于弹体的发射问题，采用了两级分离发射的原理，首先使主弹体发射，带动整个弹体发射至相应的高度，在发射之后，再采用二次发射的原理使二级弹体侧面的发射管中的四个相应的质量块发射，以质量块为牵引，使抓捕网张开，从而达到抓捕的效果。
如图2、图3和图4所示：在第一发射部3的弹药填充位31放置相应的填充物，发射带动整个弹体的发射。在第二发射部4的发射管41中放置火药以及发射物，带动四个质量块的发射，从而牵引抓捕网在空中的张开，以此达到相应的抓捕功能。
由于质量块与牵引绳相连接，为了保证发射时的足够的冲击力，在质量块的底部放置相应的圆形挡块，使得发射时拥有足够的推力保证质量块的高速发射。
发明原理：本发明的反低空飞行器抓捕弹使用时，首先发射主弹体和二级弹体，在主弹体和二级弹体发射之后，在预定的时间到达的时候发射质量块，由四个质量块牵引展开抓捕网，抓捕网抓捕目标飞行器。
本发明的反低空飞行器抓捕弹结构简单，设置合理，采用两级分离发射，利用质量块的牵引力带动抓捕网张开，从而达到抓捕无人机的目的，能够有效打击低空无人机。
