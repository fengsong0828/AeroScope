Description

一种低空飞行器低空空域精准管控系统及方法
技术领域
本发明涉及空域管制技术领域，具体为一种低空飞行器低空空域精准管控系统及方法。
背景技术
随着通用航空和无人机技术的发展，低空空域的使用需求增加，随着低空飞行器的不断普及，低空飞行器给人们日常生活带来了较多的便利，具有体积小、速度快、隐蔽性强的特点。
目前，由于低空空域环境存在多重动态干扰因素，在进行飞行器实时监测时，部署的多模态传感器网络采集低空环境数据，无法实时识别监测信号是否受到电磁干扰及气象畸变，当采集的传感数据出现波形失真，会导致空域态势感知误差扩大，影响数字孪生模型的构建精度。
因此，现提出一种低空飞行器低空空域精准管控系统及方法解决上述问题。
发明内容
（一）解决的技术问题
针对现有技术的不足，本发明提供了一种低空飞行器低空空域精准管控系统及方法,解决了上述背景技术中提出的问题。
（二）技术方案
为实现以上目的，本发明提供如下技术方案予以实现：一种低空飞行器低空空域精准管控系统及方法，所述方法包括以下步骤：
S1、通过多模态传感器网络采集低空环境动态数据，包括气象参数、地理信息、飞行器运行状态及实时碳排数据，生成低空多源异构数据集；
S2、构建数字孪生空域模型，基于毫米级精度的三维地理信息数据和厘米级定位数据，通过时空融合算法生成动态空域数字孪生体；
S3、执行空域资源动态分配：基于数字孪生体模拟低空交通流，采用强化学习算法预测空域冲突节点，生成三维网格化空域分配方案；
S4、实施飞行器精准导航控制：通过5G-A通感基站组网建立飞行器间通信链路，结合北斗差分定位数据生成自适应飞行剖面，动态调整飞行器航迹及速度参数；
S5、融合碳排数据进行空域优化：实时采集飞行器能源消耗数据，结合碳排因子库计算动态碳排强度，在空域分配方案中叠加碳排约束条件；
S6、建立分级预警响应机制：通过雷达与光电复合监测系统识别异常飞行状态，当偏离阈值大于5%时触发三级预警，同步启动应急航路再规划；
S7、生成空域管控数字资产：将空域状态数据、飞行器控制指令及碳排指标封装为区块链可追溯数据包，实时同步至低空经济智算中心。

（三）有益效果
与现有技术相比，本发明提供了一种低空飞行器低空空域精准管控系统及方法，具备以下有益效果：
1.本发明中，在构建低空数字孪生空域时，系统通过部署多源环境干扰过滤算法，实时分析传感器数据波形特征，动态识别并消除电磁干扰与气象畸变对监测信号的影响，能够消解空域态势感知中的信号失真问题，保证环境动态数据采集的真实性与可靠性，提升数字孪生模型的构建精度，进一步保障空域管控决策的准确性。
2.本发明中，在实施飞行器航迹动态监控时，通过建立三维姿态实时反馈机制，持续解算飞行器实际航向与规划路径的空间矢量夹角，使得系统能够及时感知位姿偏移状态，且在检测到位姿偏转异常时，自主生成航迹协同修正指令，实时优化网格化空域匹配策略，避免因突发气流导致的空域资源调度失准，进一步增强低空密集交通流的鲁棒控制能力。
3.本发明中，在优化全域碳排管理时，通过构建机型-工况差异化建模引擎，自动识别飞行器类型及其所处飞行阶段特征，包括垂直起降、过渡、巡航，按需加载适配的能耗计算规则，使得系统能够准确量化不同机动状态下的碳排强度，避免统一碳排系数导致的统计偏差，提升空域资源分配的环保效能与能耗调度公平性，进一步推动低空绿色交通体系的可持续发展。

该系统的运行步骤如下：
多模态空域感知与数据融合：
通过分布式传感器网络实时采集低空环境多维度动态数据，包括气象梯度：温度、气压、风切变、地理信息：地形高程、障碍物坐标、飞行器运行状态：位置、速度、姿态及能源消耗参数。气象监测无人机群以百米网格精度扫描大气扰动，激光雷达测绘车构建厘米级实景底图，机载多模芯片回传实时动力学数据。多源异构数据经时空编码引擎融合，生成动态空域数字孪生体的基础输入矩阵，实现物理环境与交通流的全息映射。
动态数字孪生体构建与冲突预测：
基于毫米级三维地理信息与实时交通流数据，采用时空融合算法构建数字孪生空域模型。通过图卷积神经网络解析空域时空编码矩阵的关联特征，解耦环境约束与交通行为的耦合关系。利用LSTM时序预测模块滚动推演未来空域状态，输出包含冲突热力图、湍流风险区及碳排密度分布的动态孪生体，为资源调度提供超前决策依据。
三维网格化空域资源智能分配：
将目标空域划分为50m×50m×10m的立体网格单元，基于孪生体模拟结果识别高冲突节点。采用多目标优化算法计算空域分配方案，同步优化飞行时效性、路径精准度及环保指标。通过密度自适应突变机制动态调整优化策略，增强随机探索能力生成多模态备选方案。
碳约束航迹精准控制与动态纠偏：
依托5G-A通感基站组网建立飞行器间高可靠通信链路，将空域分配方案转化为飞行控制指令。通过自适应PD控制算法动态解算航迹跟踪误差及其变化率，结合实时风场数据生成三维控制矢量，驱动飞行器按优化后的速度剖面和航向角飞行。当检测到位姿偏离大于5%及碳排强度超标时，自主触发分级修正策略：轻偏时通过航向微调回归网格中心点，重偏时重规划500米缓冲航路。
闭环验证与反制保障：
通过雷达-光电复合系统持续验证飞行状态与数字孪生体的吻合度，建立“监测-预测-控制-校验”闭环。异常场景下启动三级应急机制：一级告警通过声光提示人工介入；二级偏差启动冗余导航链路；三级威胁激活自适应电磁干扰波束，以双曲正切函数调控干扰频率压制非法信号。全流程数据封装为区块链可追溯资产，同步至低空经济智算中心支撑策略迭代。
空域资源分配策略智能优化：
基于实时交通密度与预设临界阈值=30架/立方公里的动态关系，通过概率调控模型驱动空域分配策略自适应演化。当监测到空域交通密度突破阈值时，系统自动提升方案突变概率，触发全局探索机制：利用蒙特卡洛随机采样生成备选空域分配方案池，结合梯度下降算法进行多目标寻优。当检测到局部网格碳排浓度超标＞0.2kgCO2/m3，则调节非线性因子β增强突变灵敏度，生成绕行高碳排区域的节能航路。该过程实现空域资源分配策略在运行效率与环保效能间的动态平衡。
全流程数据可信存证与策略演进：
基于区块链技术将孪生模型输出、空域分配方案及飞行控制指令封装为可追溯数据包，同步至低空经济智算中心。通过构建包含时空维度、碳排强度、交通状态的三维验证矩阵，持续比对实际飞行数据与数字孪生体预测结果。当航迹跟踪误差持续率大于8%及碳排统计偏差大于15%时，自动启动模型校准协议：利用历史数据回灌训练LSTM时空预测网络，更新能源-碳排转换因子库，动态优化密度突变概率函数参数β、，形成闭环自我演进机制。
