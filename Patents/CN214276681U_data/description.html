Description

一种低空小型无人飞行器的拦截装置
技术领域
本实用新型涉及拦截装置技术领域，尤其涉及一种低空小型无人飞行器的拦截装置。
背景技术
现有技术公开了申请号为CN201520386700.9的一种低空小型无人飞行器的拦截装置，包括发射器和拦截器；发射器包括压气活塞、压气气缸、单向阀、手柄、高压气缸、启动阀、填弹阀门、触发扳机、发射管、瞄准器、连接筋、拦截器和发射口挡板；拦截器包括外壳、拦截物质、密封罩、外壳定位筋和氧化脱落层。本实用新型的拦截装置适合拦截低空快速飞行的小型无人飞行器，成本低、使用安全。
上述实用中，发射器在发射过程中有工作人员手持发射，由于惯性产生较大的后坐力，使得工作人员在发射的过程中，可能会不小心被发射器撞击造成受伤，安全性较低，且由于拦截器的体积较小，多数的拦截器不便于户外携带取用，因此提出一种低空小型无人飞行器的拦截装置。
实用新型内容
本实用新型的目的是为了解决现有技术中存在的缺点，而提出的一种无人机拦截器释放机构，不仅可以通过连接板的移动使得能够对发射器发射产生的后坐力进行缓冲，且通过两个滑块的移动带动两个第二磁块分别移动至与两个第一磁块相近，使得能够保证发射器的缓冲效果更好，同时通过转动旋转轴，使得滑动套栓的移动带动滑动门滑动打开，从而使得能够便于对拦截器的存放拿取。
为了实现上述目的，本实用新型采用了如下技术方案：
一种低空小型无人飞行器的拦截装置，包括主体，所述主体上设有第一滑槽，所述第一滑槽上滑动连接有底板，所述底板的一侧固定连接有托板，所述底板的另一侧固定连接有伸缩管，所述伸缩管的内部设有第一弹簧，所述伸缩管的另一端固定连接有连接板，所述连接板的两端均转动连接有撑杆，所述底板的两端均设有第二滑槽，两个所述第二滑槽中均固定连接有固定杆，两个所述固定杆上均滑动连接有滑块，两个所述滑块的一端均固定连接有第二弹簧，两个所述第二弹簧的另一端分别与第二滑槽相连接，两个第二滑槽的相背的一端均固定连接有第一磁块，两个所述滑块上均固定连接有第二磁块，所述主体的下端固定连接有壳体，所述壳体的一侧设有滑动门，所述壳体的内部转动连接有旋转轴，所述旋转轴上连接有滑动套栓，所述滑动套栓与滑动门固定连接，所述壳体的内部设有多个放置装置。
优选地，所述伸缩管是由第一套管与第二套管组成，所述第一套管与第二套管滑动连接，所述第一弹簧的一端与第二套管相连接，所述第一弹簧的另一端与第一套管的内壁相连接。
优选地，两个所述第一磁块与两个第二磁块磁性相同。
优选地，所述壳体的一侧设有限位槽，所述滑动门与限位槽滑动连接。
优选地，所述旋转轴为螺纹杆，所述滑动套栓与旋转轴螺纹连接。
优选地，所述放置装置是由放置座、两个伸缩杆、两个第三弹簧、两个夹板共同组成，所述放置座的两端均设有竖板，两个所述伸缩杆的一端分别与两个竖板固定连接，两个所述伸缩杆的另一端分别与两个夹板固定连接，两个所述第三弹簧的一端分别与两个竖板相连接，两个所述第三弹簧的另一端分别与两个夹板相连接。
本实用新型与现有技术相比，其有益效果为：
1、本实用新型，通过主体、第一滑槽、底板、托板、伸缩管、第一弹簧、连接板、两个撑杆、两个第二滑槽、两个固定杆、两个滑块、两个第二弹簧，发射产生的后坐力带动主体后移，主体的移动带动伸缩管的第一套管与第二套管滑动调节相对距离，压缩第一弹簧，主体的移动同时带动连接板的移动，连接板的移动带动两个撑杆的移动，使得两个撑杆的另一端分别带动两个滑块在两个固定杆上相背滑动，两个滑块的移动分别压缩两个第二弹簧，从而使得能够对发射器发射产生的后坐力进行缓冲。
2、本实用新型，通过两个滑块、两个第一磁块、两个第二磁块，当两个滑块的移动带动两个第二磁块分别移动至与两个第一磁块相近，使得能够保证发射器的缓冲效果更好。
3、本实用新型，通过壳体、滑动门、旋转轴、滑动套栓，转动旋转轴，使得带动滑动套栓的移动，滑动套栓的移动带动滑动门滑动打开，从而使得能够便于对拦截器的存放拿取。
4、本实用新型，通过放置装置，使得能够便于拦截器在户外携带存放时，十分稳定。
综上所述，本实用新型结构合理，不仅可以通过连接板的移动使得能够对发射器发射产生的后坐力进行缓冲，且通过两个滑块的移动带动两个第二磁块分别移动至与两个第一磁块相近，使得能够保证发射器的缓冲效果更好，同时通过转动旋转轴，使得滑动套栓的移动带动滑动门滑动打开，从而使得能够便于对拦截器的存放拿取。

具体实施方式
下面将结合本实用新型实施例中的附图，对本实用新型实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本实用新型一部分实施例，而不是全部的实施例。
参照图1-4，一种低空小型无人飞行器的拦截装置，包括主体1，主体1上设有第一滑槽2，第一滑槽2上滑动连接有底板3，底板3的一侧固定连接有托板4，底板3的另一侧固定连接有伸缩管5，伸缩管5的内部设有第一弹簧6，伸缩管5是由第一套管与第二套管组成，第一套管与第二套管滑动连接，第一弹簧6的一端与第二套管相连接，第一弹簧6的另一端与第一套管的内壁相连接,伸缩管5的另一端固定连接有连接板7，连接板7的两端均转动连接有撑杆8，底板3的两端均设有第二滑槽9，两个第二滑槽9中均固定连接有固定杆10，两个固定杆10上均滑动连接有滑块11，两个滑块11的一端均固定连接有第二弹簧12，两个第二弹簧12的另一端分别与第二滑槽9相连接，发射产生的后坐力带动主体1后移，主体1的移动带动压缩第一弹簧6，主体1的移动同时带动连接板7的移动，使得带动两个撑杆8的另一端分别带动两个滑块11在两个固定杆10上相背滑动，两个滑块11的移动分别压缩两个第二弹簧12，从而使得能够对发射器发射产生的后坐力进行缓冲。
两个第二滑槽9的相背的一端均固定连接有第一磁块13，两个滑块11上均固定连接有第二磁块14，两个第一磁块13与两个第二磁块14磁性相同，当两个滑块11的移动带动两个第二磁块14分别移动至与两个第一磁块13相近，使得能够保证发射器的缓冲效果更好。
主体1的下端固定连接有壳体15，壳体15的一侧设有滑动门16，壳体15的一侧设有限位槽，滑动门16与限位槽滑动连接，壳体15的内部转动连接有旋转轴17，旋转轴17上连接有滑动套栓18，旋转轴17为螺纹杆，滑动套栓18与旋转轴17螺纹连接，滑动套栓18与滑动门16固定连接，转动旋转轴17，使得滑动套栓18的移动带动滑动门16滑动打开，从而使得能够便于对拦截器的存放拿取。
壳体15的内部设有多个放置装置，放置装置是由放置座19、两个伸缩杆20、两个第三弹簧21、两个夹板22共同组成，放置座19的两端均设有竖板，两个伸缩杆20的一端分别与两个竖板固定连接，两个伸缩杆20的另一端分别与两个夹板22固定连接，两个第三弹簧21的一端分别与两个竖板相连接，两个第三弹簧21的另一端分别与两个夹板22相连接，通过放置装置，使得能够便于拦截器在户外携带存放时，十分稳定。
本实用新型中，发射产生的后坐力带动主体1后移，由于托板4与人体相抵，且主体1的移动带动第一滑槽2与底板3相对滑动，主体1的移动带动伸缩管5的第一套管与第二套管滑动调节相对距离，压缩第一弹簧6，主体1的移动同时带动连接板7的移动，连接板7的移动带动两个撑杆8的移动，使得两个撑杆8的另一端分别带动两个滑块11在两个固定杆10上相背滑动，两个滑块11的移动分别压缩两个第二弹簧12，从而使得能够对发射器发射产生的后坐力进行缓冲。
当两个滑块11在两个固定杆10上相背滑动时，使得两个滑块11的移动带动两个第二磁块14分别移动至与两个第一磁块13相近，由于两个第一磁块13与两个第二磁块14磁性相同，使得两个第一磁块13与两个第二磁块14之间产生磁性，从而能够保证发射器的缓冲效果更好。
当需要取用拦截器时，转动旋转轴17，由于旋转轴17为螺纹杆，使得旋转轴17的移动带动滑动套栓18移动，滑动套栓18的移动带动滑动门16在位于壳体15上的限位槽处滑动，使得滑动门16能够滑动打开，从而实现能够便于对拦截器的存放拿取。
需要户外携带拦截器时，同时推动两个夹板22，两个夹板22的移动同时压缩两个第三弹簧21，使得两个夹板22的移动带动两个伸缩杆20滑动调节相对距离，将拦截器放置在放置座19上的两个竖板之间，同时松开两个夹板22，两个第三弹簧21均恢复形变，使得带动两个伸缩杆20再次滑动调节相对距离，使得通过通过放置装置，使得能够便于拦截器在户外携带存放时，十分稳定。
