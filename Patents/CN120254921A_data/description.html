Description

低空飞行器航路气象数据采集处理方法
技术领域
本发明涉及航路气象数据采集技术领域，具体涉及低空飞行器航路气象数据采集处理方法。
背景技术
低空飞行器航路气象数据采集处理指的是通过在低空飞行器上安装各种气象采集设备，实时监测飞行航路上的气象条件。这些设备通常包括温度传感器、湿度传感器、风速计、气压计等，能够收集到如风速、气温、湿度、气压等多种气象参数的数据。采集到的数据需要经过处理和分析，以确保飞行器能够根据当前的气象条件做出适当的航向调整，优化飞行安全性与效率。 低空飞行器航路气象数据采集处理的核心目的是提供对飞行器飞行路线的气象信息支持，以便于预测可能的气象变化，避免飞行过程中出现不利天气对飞行器造成的影响。通过数据处理，能够将原始采集的数据转换为易于理解的气象状态信息，供飞行员或自动化系统做出决策。
现有技术存在以下不足：
在低空飞行器航路气象数据采集过程中，基于卫星定位的GPS导航系统在地形复杂区域（例如高楼林立的城市峡谷、地形突变的悬崖边缘或大型金属构筑物附近）容易受到信号多径效应的干扰。该效应源于来自导航卫星的定位信号在到达接收端之前被周围建筑物或地貌反射，导致接收设备无法准确区分直达信号与反射信号的路径差异，进而引起定位误差和空间位置配准偏移。由于航路气象数据的采集与处理依赖于高精度的卫星定位结果进行时空关联，上述多径干扰所导致的定位偏差将直接引发气象参数与实际航迹位置的不一致，造成数据空间映射错位，进而影响后续气象预测模型的输入准确性和路径规划的合理性，严重削弱飞行器飞控系统与航路气象预警机制的可靠性与响应能力。

发明内容
本发明的目的是提供低空飞行器航路气象数据采集处理方法，通过引入了基于载波相位变化行为的扰动特征提取机制，并结合深度学习模型对多径风险进行自适应预测，实现了对多径污染卫星的动态解算权重调整，从而在不完全剔除观测源的前提下，增强了系统对环境变化的抗扰能力和数据处理的鲁棒性，不仅显著降低了多径引起的空间错配和气象数据误判风险，还保障了飞行器在城市峡谷、山区边缘等高反射风险区域中的持续定位精度和航迹感知可靠性，为高精度航路气象感知提供了稳定的数据基础与技术支撑，以解决上述背景技术中的问题。
为了实现上述目的，本发明提供如下技术方案：低空飞行器航路气象数据采集处理方法，包括以下步骤：
从GNSS接收机中获取目标卫星的载波相位观测值，并记录对应的时间戳；
对获取的载波相位观测值进行原始周转模糊度消解，确保载波相位变化的连续性；
对同一颗卫星在连续两个时间点的载波相位观测值进行一阶差分处理，构建出反映相位变化速率的瞬时载波相位变化量序列，并基于该瞬时载波相位变化量序列；应用特征工程技术提取出能够表征卫星信号中潜在多径效应的关键扰动特征，在设定的分析时间窗口内，对所提取的特征进行量化处理，并将处理后的特征指标作为特征向量，用于后续多径效应风险的智能识别与分类判断；
将特征向量输入至预先通过大量数据训练完成的深度学习模型中，通过深度学习模型对卫星信号的多径效应风险进行智能化预测；
当识别到卫星存在多径效应风险时，根据多径效应的风险程度实时降低该多径效应卫星的解算权重，增强高置信卫星的影响，实现对干扰卫星的有效抑制，提升整体定位与气象数据空间配准的精度，解决多径引起的数据错位与误判问题。
优选的，从GNSS接收机中提取目标卫星的载波相位观测值，具体的步骤如下：
首先，确定目标卫星的PRN号，并配置GNSS接收机以记录完整的观测数据（包括载波相位、伪距、信噪比等），确保开启对L1、L2等频点的相位观测输出；
其次，使用接收机配套的软件或通过标准通信协议（如NMEA、UBX、或厂商自定义协议）将观测数据实时导出或存储至本地；
然后，对导出的观测数据进行格式解析（如通过RINEX解析工具提取.L1或.L2字段），定位对应时间戳下目标PRN号所对应的相位观测值；
最后，将该相位值及其对应的UTC时间戳记录为时序数据对，并按采样频率依次整理，形成连续的载波相位时间序列，为后续的多径效应识别分析提供基础数据支持。
优选的，对获取的载波相位观测值进行原始周转模糊度消解，具体步骤如下：
首先，从GNSS接收机中提取目标卫星的连续载波相位观测值，并对采样间隔与时间戳进行完整性检查，确保数据连续且无严重缺失；
其次，对观测值序列进行滑动窗口差分分析，计算相邻历元之间的载波相位变化量，初步识别相位跳变点（Cycle Slip），若某一历元处的相位变化超过设定阈值（如0.05周以上），则标记为可能的周转异常点；
接着，通过双频相位组合（如广义线性组合方法）或采用模糊度固定算法（如LAMBDA方法），对跳变点进行模糊度整周估计，修复相应历元的整数周值；
然后，利用Kalman滤波或平滑处理方法对整个相位时间序列进行滤波拟合，消除小尺度波动并重建连续轨迹；
最后，对修正后的相位序列进行一致性验证，确保处理后的相位变化趋势物理合理，连续平滑，无明显跳变，即完成模糊度的初步消解过程。
优选的，应用特征工程技术提取出能够表征卫星信号中潜在多径效应的关键扰动特征，其中，提取的特征包括相位变化序列中正负扰动绝对值的偏态比值和相位变化序列在频谱中的能量分布广度，在设定的分析时间窗口内，对所提取的特征进行量化处理，分别生成扰动不对称指标和扰动谱离散度指标，将扰动不对称指标和扰动谱离散度指标作为特征向量，用于后续多径效应风险的智能识别与分类判断。
优选的，将扰动不对称指标和扰动谱离散度指标作为特征向量输入至预先通过大量数据训练完成的深度学习模型中，通过深度学习模型输出多径风险系数，通过多径风险系数对卫星信号的多径效应风险进行智能化预测。

训练完成的深度学习模型是指通过使用大量带标签的GNSS卫星信号载波相位观测数据，利用深度学习算法（如卷积神经网络CNN、长短期记忆网络LSTM、或时序Transformer模型）构建的一种可自主识别多径效应扰动特征与模式的智能识别模型。训练过程的核心目标，是使模型能够从输入的特征向量中学习到“扰动与风险之间”的映射关系，从而在面对新输入时具备较高的泛化能力，快速输出对应的多径风险系数。在该过程中，模型通过反复迭代训练、误差反向传播、参数调整，逐步拟合输入特征（例如扰动不对称指标与扰动谱离散度指标）与对应标签（是否存在多径、多径程度等级等）之间的复杂非线性关系。
以扰动不对称指标与扰动谱离散度指标作为输入的关键特征，这两个参数分别从时域与频域角度揭示卫星信号中存在的潜在异常扰动行为。扰动不对称指标主要衡量的是瞬时载波相位变化量中正负波动的不对称性。通常情况下，直达信号的相位变化应在统计意义上表现为近似对称的噪声扰动，而一旦受到多径反射影响，由于反射信号叠加方向和强度的不均匀性，会导致相位变化序列在一定时间段内出现明显的“偏向性”，如长时间持续正扰动或负扰动、上下波幅比例显著失衡等。通过扰动不对称指标的提取，可以敏感地捕捉这种“扰动偏斜趋势”，为模型提供反映多径路径方向性干扰的重要特征支撑。而扰动谱离散度指标则从频域角度分析信号的能量分布情况，其通过傅里叶变换或小波变换分析当前时间窗内的相位扰动频谱，并计算谱能量的分布广度。多径信号由于其由多个延迟路径叠加形成，往往会引入额外的高频分量或频率漂移，使得频谱能量出现分散、扩展或偏离现象。扰动谱离散度指标通过度量这种频谱发散程度，能够揭示多径扰动带来的频率域异常模式，为模型提供更深层次的扰动结构描述。
训练完成的深度学习模型，正是基于对这些扰动特征的大规模采样、分类与映射而构建的。在模型训练阶段，研发者通常会收集覆盖多种场景（如开阔地、城市峡谷、建筑密集区、高反射面环境等）下的卫星信号观测数据，并对这些数据进行标注（如“无多径”、“轻度多径”、“严重多径”），然后使用这些带标签的数据作为训练集，构建监督学习框架。模型的目标是最小化预测输出（即多径风险系数）与真实标签之间的差距，例如采用均方误差、交叉熵等损失函数进行优化。在训练过程中，模型通过对扰动不对称指标与谱离散度指数等输入特征的反复学习与结构权重更新，不断强化对多径效应扰动的识别能力。特别是在采用如LSTM或Transformer这类具备序列建模能力的模型结构时，可以进一步捕捉扰动特征随时间演化的模式，提高对“隐性多径”或“间歇型多径”现象的识别精度。
