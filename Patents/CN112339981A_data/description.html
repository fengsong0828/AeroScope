Description

一种用于智能低空载人飞行器的舱门锁机构
技术领域
本发明属于低空载人飞行器舱门结构设计领域，更具体地，涉及一种智能低空载人飞行器的舱门锁机构。
技术背景
在传统低空载人飞行器领域中，乘坐者具有较高的飞行操作控制权限，而且飞行器在飞行阶段对乘坐者操控依赖度较高，故飞行阶段对乘坐者操控水平要求较高，从而可以使用较为复杂和操控繁琐的舱门锁机构。
智能低空载人飞行器领域旨在借助地面站功能从而简化飞行器乘坐人员的操作流程和降低乘坐人员的操控水平要求，实现全程无人驾驶。这要求飞行器舱门锁除了保证结构强度、轻量化，又要考虑乘坐人员在任何情况下操作舱门锁的安全可靠和便利。本发明旨在提供一种结构简单可靠、无需外部能源且易于紧急逃生操作的舱门锁，在一定程度上解决该问题。
发明内容
针对现有技术中存在的问题，本发明提供了一种智能低空载人飞行器的舱门锁机构，该舱门锁机构结构简单，便于开、关舱门。
本发明的目的通过如下的技术方案来实现：一种用于智能低空载人飞行器的舱门锁机构，包括设置于舱体外侧的第一组件和设置于舱门内侧的第二组件；所述第一组件包括第一组件支座、锁销、拉环、第一复位弹簧，所述锁销设置于第一组件支座的导向孔中，所述锁销的端部与拉环固定连接，所述第一复位弹簧设置于锁销上；所述第二组件包括第二组件支座、锁钩、手板、转位轴和第二复位弹簧，所述锁钩通过转位轴与第二组件支座连接，所述第二复位弹簧设置于转位轴上，所述手板设置于锁钩上。
进一步地，所述第一组件支座嵌入式安装于舱体外侧，所述拉环通过舱体开孔伸入舱体内部。
进一步地，所述第二组件支座设置于舱门内侧边缘上，所述手板伸出舱门外侧。
进一步地，所述锁销的滑动行程大于或等于锁钩的宽度。
进一步地，所述第二组件支座还设有限位销，所述限位销位于锁钩的后方。
与现有技术相比，本发明的有益效果如下：第一组件装上有可以上下滑动且使用第一复位弹簧复位的锁销，操作者在机舱内部通过拉动拉环向上拉动锁销，从而实现在舱体内部解锁舱门；第二部件装有可以左右摆动且使用弹簧复位的锁钩，操作者在机舱外部可以通过按压手板使锁钩脱扣，从而实现在舱体外部解锁舱门。关舱门时，锁钩通过撞击锁销可以实现自动回弹上锁。本发明所设计的舱门锁机构不依赖电、气、液等外部动力源，使操作者在舱内和舱外均可手动操作开、关舱门，且结构简单可靠，操作方便。

具体实施方式
为了使本发明实施例的目的、技术方案更加清楚，下面将结合附图，对本发明中的技术方案进行清楚、完整地描述。显然，所描述的实施例是本发明一部分实施例，而不是全部的实施例。通常，在附图中所描述和示出的发明实施例的组件可以以其他不同细节结构和尺寸变化来实现。在本发明的具体实施例附图中，为了更清楚地描述舱门锁机构中各元件的工作原理，并不能构成对结构内部各零部件的大小、尺寸、形状的限定。
如图1，本发明提供了用于智能低空载人飞行器的舱门锁机构的结构示意图，包括设置于舱体外侧的第一组件1和设置于舱门内侧的第二组件2；所述第一组件1的细节图如图2a 所示，所述第一组件1包括第一组件支座101、锁销102、拉环103、第一复位弹簧104，所述锁销102设置于第一组件支座101的导向孔中，可以沿第一组件支座101的导向孔做直线滑动；所述锁销102的端部与拉环103固定连接，所述拉环103可以通过推动或拉动来驱使锁销102做直线往复滑动，所述第一复位弹簧104设置于锁销102上，用于锁销102的复位；所述第一组件支座101嵌入式安装于舱体外侧，所述拉环103通过舱体开孔伸入舱体内部；操作者在舱体内部可通过向上拉动拉环103使舱门解锁从而出离舱体，解锁后锁销102在第一复位弹簧104的作用下复位，待下次锁定操作，所述锁销102在最大行程处的复位压力不大于20N。
所述第二组件2的细节图如图2b所示,所述第二组件2包括第二组件支座201、锁钩202、手板203、转位轴204和第二复位弹簧206，所述锁钩202通过转位轴204与第二组件支座201连接，所述转位轴204用于固定锁钩202，所述第二组件支座201还设有限位销205，所述限位销205位于锁钩202的后方。限位销205限定锁钩202绕转位轴204做有限角度旋转并在锁定操作中钩住锁销102，所述第二复位弹簧206设置于转位轴204上，使锁钩202在做旋转运动时自动复位。所述手板203设置于锁钩202上，手板203用于手动按压使锁钩202 转动，所述第二复位弹簧206在手板203被按压至极限位置时的扭矩不大于1.8Nm。所述第二组件支座201设置于舱门内侧边缘上，所述手板203伸出舱门外侧，操作者在舱体外部可通过按压手板203使锁钩202沿转位轴204向远离锁销102的方向转动从而使舱门解锁，解锁后锁钩202在第二复位弹簧206作用下复位于限位销205的位置并待下次锁定操作。所述锁销102的滑动行程大于或等于锁钩202的宽度。
锁定状态结构示意图如图3a，在舱体内部或外部关舱门时，第二组件2跟随舱门移动至接近第一组件1，锁钩202撞击锁销102并沿转位轴204向远离锁销102的方向转动；第二组件继续向靠近舱体方向移动，锁钩202在第二复位弹簧206作用下回位并挂住锁销102，锁定操作结束。
舱内开锁结构示意图如3b，操作者在舱体内部可通过向上拉动拉环103使得锁销102沿第一组件支座101的导孔滑动，锁销102脱离锁钩202，舱门解锁打开，解锁后锁销102在第一复位弹簧104的作用下复位，待下次锁定操作。
舱外开锁结构示意图如3c，操作者在舱体外部可通过按压手板203使锁钩202沿转位轴204向远离锁销102的方向转动从而使舱门解锁，解锁后锁钩202在第二复位弹簧206作用下复位于限位销205的位置并待下次锁定操作。
综上所述，本发明提供的舱门锁机构，通过安装在舱体和舱门上的第一组件1和第二组件2，实现了在舱内和舱外均可手动锁定和解锁舱门，且不需要外部动力源，结构巧妙，操作便捷，且简单可靠，是智能低空载人飞行器在研制和试飞阶段舱门锁机构的较优选择。
