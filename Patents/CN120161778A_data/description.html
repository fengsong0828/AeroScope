Description

一种抗传感器故障的低空飞行器多模态推进控制方法
技术领域
本发明涉及飞行器智能控制领域，尤其具体涉及一种抗传感器故障的低空飞行器多模态推进控制方法。
背景技术
当前，低空经济作为国家重点支持的战略性新兴产业，正在加速催生新型航空平台，特别是短距起降与垂直起降类飞行器，广泛应用于城市空中交通、物流运输、应急响应等场景；与传统飞行器相比，短垂飞机具有多模式切换、多类型推进器混合工作的特点，对推进系统提出了更高的智能化、自适应与鲁棒性控制需求。
典型短垂飞行器在执行任务过程中需频繁经历垂直起飞、爬升、巡航、悬停、过渡、下降和垂直降落的多状态切换，推进系统结构复杂，包含电推进、涡扇推进、倾转机构及舵面系统，控制变量之间高度耦合，动态响应特性差异显著。
传统推进控制策略多为单模型控制器，难以兼顾多模态飞行需求，且在复杂扰动环境下响应迟缓、调参效率低，控制性能易退化；尤其在多传感器系统中，若存在测量异常或故障，易导致状态估计失真，进而影响控制策略决策，存在较大鲁棒性风险。
针对上述挑战，人工智能控制算法逐渐引入飞控领域；近年来，深度强化学习结合模型预测控制、模糊控制的多策略融合成为发展趋势，但实际应用中依然面临策略不稳定、切换不平滑及传感器异常缺乏鲁棒处理机制的问题。
为此，本发明提出一种抗传感器故障的低空飞行器多模态推进控制方法，融合控制策略输出、引入传感器残差感知机制并具备容错调参能力的推进控制方法。
发明内容
本发明的目的在于提供一种抗传感器故障的低空飞行器多模态推进控制方法，为解决上述现有技术问题，本发明通过以下技术方案实现：
本发明实施例提供的一种抗传感器故障的低空飞行器多模态推进控制方法，具体包括如下步骤：
S1：构建推进组件的非线性模型；
S2：实时采集飞行状态与传感器观测参数，输入至飞行状态识别模块与健康参数估算模块，分别识别飞行模式与每一推进组件健康状态；
S3：引入权重嵌入融合网络，输入状态残差与健康状态，生成所述控制策略群的加权系数，提取局部扰动特征，建模多通道状态时序依赖，输出残差加权向量；
S4：将识别状态及健康状态联合输入控制策略群推进控制优化，并分别计算控制动作输出；
S5：根据控制策略群的加权系数加权控制动作输出，生成最终推进指令，将最终推进指令分发至飞行器各推进子系统，各推进子系统反馈信号与传感器数据至经验池与健康参数估算模块，并进行策略更新与故障感知精度迭代。

本发明的有益效果：
1、本发明构建了一套融合控制策略机制，集成强化学习RE-DEORL、模型预测控制MPC与模糊控制方法，结合飞行状态与系统健康参数，实现控制策略输出的智能加权融合，打破了传统推进控制中单一策略难以适应多飞行状态的局限，有效提升了系统在多模式、多任务阶段下的泛化能力与响应灵活性；采用的RE-DEORL强化学习算法结合了Actor-Critic结构与DQN思想，融合经验回放机制与目标网络设计，提升了策略训练的稳定性与样本利用效率，避免策略陷入局部最优，同时通过动态扰动扩展机制提高了控制策略对复杂飞行环境变化的适应能力与收敛速度；
2、引入了基于CNN与Transformer构建的残差感知权重网络，结合GIRLS-EKF滤波结构进行多通道传感器信号处理与健康参数估算，能够在存在传感器异常、非高斯噪声或推进器性能退化情况下自动调整控制结构，实现系统的动态容错调节与鲁棒性能保障；基于飞行性能反馈与估算健康状态自适应调整控制器输出权重，实现了控制策略的“自进化”能力；同时，系统采用结构化神经网络对控制变量进行融合与解耦，结合推进器异构响应特性，实现多推进组件间协同驱动与功率最优分配，具备良好的工程可集成性与平台适应性，适用于电推进、涡扇混合、倾转旋翼等多类推进配置；实现了低空飞行器在复杂飞行状态与传感器异常工况下的推进控制鲁棒性与能效最优化，为智能控制架构提供了系统化解决方案，具有显著的应用价值与技术推广前景。
