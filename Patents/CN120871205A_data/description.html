Description

LEO/GNSS/INS紧组合低空飞行器定位测速方法及装置
技术领域
本文件涉及低空飞行器定位或多传感器组合定位领域，尤其涉及一种LEO/GNSS/INS紧组合低空飞行器定位测速方法、装置、电子设备及介质。
背景技术
全球导航卫星系统(Global Navigation Satellite System)与惯性导航系统INS(Inertial Navigation System)组合导航定位系统，通常采用松组合与紧组合模式实现组合，通过进行优势互补，实现载体连续可用定位。但是低空空域，尤其是城市高楼、郊区峡谷等场景下卫星信号容易受到遮挡，当可见卫星少于4颗时，GNSS无法解算载体位置与速度，GNSS/INS(Global Navigation Satellite System/Inertial Navigation System)松组合切换为单一的INS航位推算方式，定位误差随时间快速发散，定位精度受限。GNSS/INS紧组合，由于在距离域进行组合，仍然可以通过GNSS伪距、伪距率与INS解算的等效伪距、伪距率组合，短时间定位精度依然可以保持。但是当卫星信号遮挡严重，长时间可见卫星数量为1-2颗，由于观测信息不足，卫星空间几何分布较差，定位误差同样会发散，导致定位结果不可用。
与GNSS相比，低轨LEO(Low Earth Orbit)卫星轨道高度较低，轨道周期较短，单星覆盖范围小，需要数百颗甚至数万颗卫星组网，实现整体高密度覆盖。低轨卫星采用增强技术可以辅助GNSS与INS系统，提升GNSS/INS组合系统的性能。低轨卫星导航增强技术主要分为信号增强和信息增强两类，其中信息增强通过播发误差修正数据辅助用户提升定位精度。信号增强直接发射导航信号作为额外观测源，通过增加观测信号数量改善定位精度。
低轨卫星信号发射抵达地球表面的功率比中高轨卫星高，可穿透部分城市峡谷或森林环境，具有强遮挡力。此外，作为独立信号源，可在GNSS拒止环境下提供备份定位能力，提升系统韧性。LEO作为“伪卫星”，可以发射独立的测距信号，补充遮挡区域的卫星数量，改善卫星几何空间。另一方面，由于LEO卫星轨道动态变化快，多普勒效应显著，改善遮挡场景下低空飞行器的测速精度。因此，低轨LEO卫星可以用于对GNSS和INS进行补盲与增强，提升系统的综合定位性能。
现有技术中公开了一种低轨增强、GNSS与IMU融合的持续导航方法及导航装置，低轨LEO卫星采用信息增强，一方面转发GNSS卫星星历数据，另一方面通过低轨增强电文发送时刻的伪距信息、多普勒信息以及低轨卫星位置和速度信息进行单星定位，与INS导航结果进行融合，实现复杂电磁环境下GNSS定位的可靠性和持续性。该方法虽然采用低轨卫星信息增强方法，提高了开阔无遮挡环境下GNSS/INS组合定位系统的可靠性与持续性，但是信息增强需要依赖GNSS信号，在城市高楼、郊区峡谷等信号遮挡区域，无法发挥作用，因此不适用于卫星信号遮挡场景。
现有技术中还公开了一种低轨导航增强下进行GNSS/INS组合导航的方法和装置，采用低轨LEO卫星增强电文中的GNSS卫星精密轨道数据、GNSS卫星精密钟差数据、GNSS码偏差改正、地球自转参数、电离层改正和低轨卫星精密轨道、钟差，对GNSS和低轨LEO卫星的伪距载波相位观测值进行误差改正，得到高精度的GNSS导航定位结果，与INS在位置与速度域进行松组合，得到最终的高精度组合导航结果。该方法采用低轨LEO卫星信息增强方式，虽然缩短了高精度定位的收敛时间，但是同样适用于开阔无遮挡场景下的高精度实时定位，不适用于卫星信号遮挡场景下的高精度定位。
现有技术中还公开了一种实时自主组合导航定位方法及装置，利用低轨卫星播发的导航增强信息和中高轨卫星的星历，得到低轨卫星和中高轨卫星的精密轨道和钟差，修正低轨与中高轨卫星观测数据，联合解算得到载体的定位信息，与INS在位置与速度域进行松组合，采用线性卡尔曼滤波，得到最终的组合导航结果。该方法采用低轨LEO卫星信息增强方式，虽然提高了中高轨卫星的观测质量，实现连续精度的稳定导航定位服务，但是仍然不适用于卫星信号遮挡场景下的高精度定位。
现有技术还提出了一种低轨LEO卫星伪距观测信号增强的GNSS/IMU紧组合定位方法，基于INS误差参数模型和GNSS时钟误差模型构建组合系统状态模型，联合GNSS伪距观测模型与低轨LEO卫星伪距观测模型构建观测模型，观测方程为非线性伪距观测模型线性化后的线性模型，采用卡尔曼滤波进行滤波估计，获得组合后的定位结果。该方法采用低轨LEO卫星信号增强方式，理论上通过引入低轨LEO卫星的伪距增加观测量，提高系统的定位精度，但是伪距模型在线性化过程中由于线性截断也会引入较大的截断误差，影响定位精度，无法始终保证复杂场景下的定位精度。
当前低轨LEO卫星信息增强技术的主要缺点是需依赖GNSS卫星信号解析增强电文，在卫星信号被严重遮挡的场景，如城市高楼、峡谷，GNSS信号中断导致信息增强完全失效，无法解决遮挡场景下观测量不足(可见卫星<4颗)的连续定位问题，导致INS误差快速发散。
低轨LEO卫星信号增强技术的主要缺点是观测模型单一，多普勒观测值利用不足，。一方面仅依赖伪距观测值，未融合多普勒频移等多维信息，在遮挡场景下观测方程鲁棒性不足。另一方面未充分利用低轨卫星特有的高动态多普勒频移，导致测速精度未最大化。此外，紧组合观测模型中非线性化引入的截断误差，尤其在低仰角卫星或动态剧烈时，显著降低定位精度。
发明内容
本发明的目的在于提供一种LEO/GNSS/INS紧组合低空飞行器定位测速方法、装置、电子设备及介质，旨在解决现有技术中的上述问题。
本发明提供一种LEO/GNSS/INS紧组合低空飞行器定位测速方法，包括：
INS通过航位推算得到低空飞行器更新后的位置、速度与姿态状态信息；
根据接收到的GNSS和LEO信号，并结合低空飞行器更新后的位置、速度与姿态状态信息，构建LEO/GNSS/INS紧组合系统状态与量测模型；
采用超球体单形采样无迹卡尔曼滤波SS-UKF模型，对所述LEO/GNSS/INS紧组合系统状态与量测模型进行滤波估计，通过INS误差校正，得到最终组合后的低空飞行器位置、速度与姿态信息。
本发明提供一种LEO/GNSS/INS紧组合低空飞行器定位测速装置，包括：
推算模块，用于基于INS通过航位推算得到低空飞行器更新后的位置、速度与姿态状态信息；
构建模块，用于根据接收到的GNSS和LEO信号，并结合低空飞行器更新后的位置、速度与姿态状态信息，构建LEO/GNSS/INS紧组合系统状态与量测模型；
计算模块，用于采用超球体单形采样无迹卡尔曼滤波SS-UKF模型，对所述LEO/GNSS/INS紧组合系统状态与量测模型进行滤波估计，通过INS误差校正，得到最终组合后的低空飞行器位置、速度与姿态信息。
本发明实施例还提供一种电子设备，包括：存储器、处理器及存储在所述存储器上并可在所述处理器上运行的计算机程序，所述计算机程序被所述处理器执行时实现上述低轨卫星双观测信号增强的GNSS/INS紧组合低空飞行器定位测速方法的步骤。
本发明实施例还提供一种计算机可读存储介质，所述计算机可读存储介质上存储有信息传递的实现程序，所述程序被处理器执行时实现上述低轨卫星双观测信号增强的GNSS/INS紧组合低空飞行器定位测速方法的步骤。
采用本发明实施例，在传统GNSS/INS紧组合单一伪距观测信号的基础上，引入低轨LEO卫星的伪距与多普勒双观测信号，采用超球体单形采样无迹卡尔曼滤波SS-UKF模型进行滤波估，提升低空飞行器的定位测速精度，满足城市高楼、桥梁、郊区峡谷等卫星信号遮挡场景下的低空飞行器自主精确实时定位测速需求，适用于低空飞行器巡检、低空物流等应用场景。

也就是说，本发明实施例将低轨LEO卫星的伪距与多普勒频移双信号均引入GNSS/INS紧组合系统，构建LEO/GNSS/INS紧组合定位测速系统，可以改善既有组合系统存在的系统观测量单一、测速精度有限的问题，实现卫星信号遮挡场景下低空飞行器的精确定位测速。将超球体单形采样策略融入UKF非线性滤波框架中，构建超球体单形采样UKF滤波SS-UKF算法。一方面，状态模型由于为线性方程，仍然采用卡尔曼滤波，只需要进行1次线性计算，减少线性模型采用非线性UKF模型的计算资源浪费。另一方面，量测模型采用UKF非线性模型，可以避免量测模型线性化导致的截断误差，保证定位测速精度；采用超球体单形采样策略，采样点由原来的2n+1减少为n+2个，每个量测需要进行n+2次非线性计算即可，提高滤波速率。其中，n为系统状态个数。因此，SS-UKF可以平衡高维非线性模型的滤波精度与滤波效率，满足卫星信号遮挡场景下LEO/GNSS/INS紧组合定位测速系统的定位测速精度与实时性。
本发明实施例的技术方案解决了卫星信号遮挡场景下(卫星观测信号少于4颗时)低轨卫星增强的低空飞行器组合导航定位测速问题。在既有GNSS/INS紧组合系统的基础上，不改变系统状态模型，采用低轨卫星信号增强技术，引入低轨卫星伪距与多普勒观测信号，联合低轨卫星伪距观测模型/多普勒频移观测模型与GNSS伪距观测模型/伪距率模型构建优化后的非线性观测模型。同时，采用超球体单形采样无迹卡尔曼滤波SS-UKF(Spherical Simplex Unscented Kalman Filter)模型进行滤波估计，状态更新采用线性卡尔曼滤波，量测更新采用无迹卡尔曼滤波框架，并采用超球体单形采样模型进行采样，减少采样点个数，提高滤波实时性，解决了高维非线性紧组合导航定位模型的精确实时定位测速问题。
