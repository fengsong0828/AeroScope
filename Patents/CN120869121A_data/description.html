Description

一种低空飞行器的最短航线检测方法及系统
技术领域
本申请涉及航线规划技术领域，具体涉及一种低空飞行器的最短航线检测方法及系统。
背景技术
随着低空空域逐步开放以及城市空中交通(UAM)、物流配送无人机、应急救援等领域的迅速发展，低空飞行器的应用日益广泛。为保障飞行任务的效率与安全，航线规划作为核心基础之一，其智能化与高精度需求日益凸显。现有低空飞行器的航线规划方法通常基于二维地图或静态三维建模，借助A*、Dijkstra、RRT等常规路径搜索算法，在已知起止点和障碍物分布条件下，规划一条从起点至终点的可行路径。然而，在实际应用中，低空飞行环境具有空间结构复杂、动态障碍频发、飞行器本体动力学特性限制等诸多挑战。现有技术存在如下问题：其一，三维建模过程通常忽略飞行器最小转弯半径和爬升限制，导致路径在理论上可行但实际飞行中不可执行；其二，障碍物安全评估仅基于固定缓冲距离，难以适应飞行速度或环境能见度等动态变化；其三，路径生成后通常缺乏节点简化与光滑优化，导致存在冗余节点、飞行路径不够平滑的问题，影响飞行效率与系统响应能力。因此，亟需一种能够在复杂三维环境中实时融合障碍物动态特征、飞行器物理约束与安全裕度自适应调整的航线检测方法，提升低空飞行器在复杂空域环境下的路径规划质量与执行效率。
发明内容
本申请通过提供了一种低空飞行器的最短航线检测方法及系统，旨在解决现有低空飞行器航线规划过程中无法兼顾三维障碍环境与飞行器运动约束，导致规划路径精确不足、存在安全隐患的技术问题，实现在满足飞行器动态特性限制的前提下，快速生成避障有效、路径最短、节点冗余最小的最优航线，提高低空飞行安全性与航线规划效率的技术效果。
本申请公开的第一个方面，提供了一种低空飞行器的最短航线检测方法，所述方法包括：建立目标区域的三维环境模型，识别障碍物；根据识别障碍物的障碍范围、安全范围，进行行驶空间标记，构建飞行空间地图；以飞行器的当前位置为起点，目标飞行位置作为终点，在所述飞行空间地图中基于飞行器最小转弯半径、最大爬升率的约束条件，进行最小路径搜索，获得最小规划路径；对所述最小规划路径进行中间节点冗余验证，将满足验证评价结果的最小规划路径作为最短确定航线。
本申请公开的另一个方面，提供了一种低空飞行器的最短航线检测系统，所述系统包括：障碍物识别模块：建立目标区域的三维环境模型，识别障碍物；行驶空间标记模块：根据识别障碍物的障碍范围、安全范围，进行行驶空间标记，构建飞行空间地图；最小路径搜索模块：以飞行器的当前位置为起点，目标飞行位置作为终点，在所述飞行空间地图中基于飞行器最小转弯半径、最大爬升率的约束条件，进行最小路径搜索，获得最小规划路径；冗余验证模块：对所述最小规划路径进行中间节点冗余验证，将满足验证评价结果的最小规划路径作为最短确定航线。
本申请中提供的一个或多个技术方案，至少具有如下技术效果或优点：
上述一种低空飞行器的最短航线检测方法，该方法首先通过构建目标区域的三维环境模型，对区域内的障碍物进行识别和定位。随后，依据障碍物的实际位置与设定的安全边界，对飞行空间进行分类标记，将区域划分为可飞行区、障碍区和风险区，从而生成完整的三维飞行地图。之后，在该飞行地图中，以飞行器当前的位置为起点、预定目标位置为终点，结合飞行器的最小转弯半径与最大爬升率等运动限制条件，执行路径搜索算法，获取一条满足飞行特性的最短路径。最后，对该路径中间节点进行冗余性分析和验证，剔除不必要的节点或进行位置微调，确保航线在满足飞行安全和机动性能的前提下，实现距离最短、路径最优的飞行目标。

实施例一，如图1所示，本申请提供了一种低空飞行器的最短航线检测方法，所述方法包括：
建立目标区域的三维环境模型，识别障碍物。
在本申请实施例中，首先通过低空飞行器搭载的激光雷达(LiDAR)、深度摄像头、结构光传感器，或者通过已有的高精度地理信息系统(GIS)数据，采集目标飞行区域的空间信息，这些数据通常包含地面高度、建筑物轮廓、植被密度、杆塔和电缆等静态物体的空间位置。采集完成后，将采集到的数据进行预处理，包括坐标对齐、噪声剔除和点云密度归一化处理，构建出具有空间一致性的三维环境模型，该模型常采用体素栅格(Voxel Grid)结构，即将空间划分为大小一致的立方体单元，每个单元记录该位置是否被实体占据，以及障碍物的属性信息。三维环境模型建立完成后，基于设定的高度阈值、反射率特征等对模型中的实体区域进行分析，判断其是否构成飞行障碍物，例如，若某体素单元连续分布在一定高度范围以上，则识别为高架建筑物；若其反射特性与金属或植被相符，则进一步分类为电塔或树木等类型。最终，所有被识别出的障碍物将被精确标注其边界轮廓、空间尺寸及类型信息，形成结构化障碍物数据集，用于后续安全边界计算和路径规避规划。通过上述过程，不仅获得了目标区域的完整三维地图，还实现了对影响飞行安全的障碍物进行准确定位与分类识别。
根据识别障碍物的障碍范围、安全范围，进行行驶空间标记，构建飞行空间地图。
在一个实施例中，在完成障碍物识别和三维环境建模后，根据每个障碍物在三维模型中的边界位置，结合飞行器的类型、安全飞行规范及飞行状态(如速度、姿态控制能力)，设定一个标准安全距离，该距离用于表示飞行器与障碍物之间所需保持的最小安全间隔。随后，以障碍物边缘为中心，在三维空间中按该安全距离进行外扩，形成一个膨胀安全边界，这个膨胀边界不仅考虑了障碍物本身的尺寸，还为飞行器的操作误差、环境干扰等预留了缓冲空间。之后，将飞行空间按照一定精度进行栅格化处理，通常划分为多个立方体单元(体素单元)，每个单元根据其所在位置与障碍物或膨胀边界之间的关系，被赋予不同的标签类型，例如，位于障碍物实际边界内的单元标记为障碍区，飞行器绝对禁止穿越。完成所有体素单元的类别标记后，形成一张结构化的飞行空间地图，该地图以三维栅格的方式全面表达了飞行空间中各区域的通行等级和空间属性，为后续路径搜索算法提供可供计算的空间参考依据，并显著提高路径规划的效率与安全性。
进一步，本申请提供了根据识别障碍物的障碍范围、安全范围，进行行驶空间标记，构建飞行空间地图，包括：
获取标准安全距离；基于所述标准安全距离对障碍物边缘进行安全范围膨胀，获得膨胀安全边界；根据所述膨胀安全边界进行障碍范围、安全范围分割，生成障碍标记、安全标记；利用所述障碍标记、安全标记在三维飞行空间中进行类别标记，构建所述飞行空间地图。
优选的，在建立好三维环境模型并识别出障碍物后，首先获取标准安全距离，该标准安全距离可由飞行器的运行规范、行业安全标准、飞行任务类型等因素设定，例如，可设置为3米、5米或随飞行速度动态调整，标准安全距离的作用是在飞行器与障碍物之间预留出足够的空间缓冲，以防止由误差、风扰或传感器延迟等引发的碰撞风险。随后，基于该标准安全距离，对已识别的障碍物边缘进行三维膨胀处理，该处理通常使用障碍物风险类型进行风险安全距离评定，并根据评定结果围绕障碍物边缘生成一个等厚度的保护边界，作为膨胀安全边界，此边界包围了原始障碍物本体。之后，利用该膨胀边界，对空间区域进行分割并打标签，在这个过程中，被障碍物本体占据的区域被标记为障碍物，膨胀边界之外的区域标记为安全区，而障碍物边界与膨胀边界之间的区域，即飞行器不宜穿越但必要时可以考虑绕行的区域，被视为风险区。在完成上述标记后，三维空间已被划分为多个立方体单元(如以体素尺寸1米为例，整个飞行空间被划分为若干1立方米的单元格)，且每个体素单元都具有障碍标记。最终，所有体素单元的类别信息汇总形成一张完整的三维飞行空间地图，该地图不仅反映了空间中障碍物的位置和形态，还明确划定了哪些空间区域可供飞行、哪些区域应规避或禁止进入，该地图将作为路径搜索算法运行的基础输入，使得后续生成的航线既能避障安全，又具备实际可飞行性。
进一步，本申请提供了利用所述障碍标记、安全标记在三维飞行空间中进行类别标记，构建所述飞行空间地图，包括：
将三维空间栅格化，划分为N个立方体单元，N为正整数；将障碍物所占用的立方体单元标记为障碍物本体，所述膨胀安全边界与障碍物本体之间的空间标记为风险区，在所述膨胀安全边界之外的安全范围标记为安全区域，建立所述飞行空间地图。
可选的，在完成三维环境建模和障碍物膨胀边界的构建后，对目标飞行区域进行空间离散化处理，即将连续的三维空间进行栅格化。具体地，将整个三维空间划分为N个体积一致的立方体单元(体素单元)，每个单元的边长为固定值(如0.5米、1米或其他根据飞行器精度设定的值)，其中，N为一个正整数，表示划分总数，每个立方体单元用其在三维坐标系中的索引编号(i,j,k)唯一表示，并可存储状态标签。完成栅格划分后，基于障碍物与膨胀边界，按照以下规则对每个体素单元进行空间分类和标记，对于与障碍物几何体相交或完全包含于障碍物本体内的立方体单元，将其标记为障碍物本体，表示该空间位置被障碍物实际占据，飞行器禁止进入；对于未被障碍物本体占据，但落入膨胀安全边界内的立方体单元，即位于障碍物边缘与膨胀边界之间的部分，将其标记为风险区，这些风险区代表飞行器进入后虽不直接碰撞障碍，但存在操作误差、扰动风险，建议优先避开，仅在无可替代路径下可作为过渡路径考虑；对于所有不与障碍物本体或膨胀边界相交的立方体单元，将其标记为安全区域，这些安全区域代表在当前安全策略和飞行约束下的推荐飞行区域，是路径搜索的主要候选空间。整个标记过程可采用基于八叉树(Octree)或三维哈希表的空间映射结构加速实施，提高栅格判定效率。最终，将上述所有已标记的体素集合整合，生成一张三维飞行空间地图，其中每一个立方体单元的空间位置及通行属性(障碍、风险等)均被清晰记录，这张地图将作为搜索约束空间，确保航线在空间上可行、在逻辑上避障，并在执行时满足飞行器物理特性与安全需求。
进一步，本申请提供了获得膨胀安全边界，包括：
获取障碍物风险类型；根据所述障碍物风险类型进行风险安全距离评定；根据安全评定结果对所述标准安全距离进行调整，利用调整的标准安全距离进行安全边界膨胀，获得所述膨胀安全边界。
可选的，首先在三维环境模型中对已识别的障碍物进行风险类型识别，风险类型是指障碍物对飞行安全造成潜在威胁的性质与程度，通常基于物理特性(如是否具有刚性、反光性、透明性)、高度与尺寸(如障碍物是否高于平均建筑高度，是否超出飞行器爬升能力)、动态属性(是否为可移动障碍，如升降平台、吊车臂、鸟)等，结合对应的阈值进行判定，从而每类障碍物赋予一个障碍物风险类型标签，例如，高风险类型(高压电缆、活动吊臂、透明玻璃幕墙)、中风险类型(固定建筑、桥梁、植被树木)、低风险类型(矮墙、临时搭建物、小型障碍物)。识别完障碍物的风险类型后，进入风险安全距离评定阶段，此时，根据预设的风险类型—安全距离映射表，对标准安全距离进行调整，例如，高风险类型对应的标准安全距离调整为6米、中风险类型对应的标准安全距离调整为4米。此外，还会根据飞行速度、环境能见度等对调整后的标准安全距离进行再次调整，使标准安全距离更加满足实际环境需求。获得调整后的标准安全距离后，根据这个调整的标准安全距离对障碍物在三维空间中的边界进行空间膨胀处理，即基于三维欧几里得距离，围绕障碍物几何边界向外扩展等距离边缘，形成膨胀安全边界，这个膨胀安全边界将与障碍物本体一起参与飞行空间的区域划分，使得对于高风险障碍物给予更宽的规避缓冲，而对低风险目标则可压缩飞行区域保留空间，从而实现飞行路径的灵活规划与飞行资源的高效利用。
进一步，本申请提供了获得膨胀安全边界，还包括：
分别建立飞行器飞行速度、环境能见度与标准安全距离的调整关系；根据当前的飞行速度或环境能见度与对应的调整关系进行调整量计算，确定标准安全距离的调整量；利用所述调整量对所述标准安全距离进行调整，利用调整的标准安全距离进行安全边界膨胀，获得所述膨胀安全边界。
可选的，为了使标准安全距离更加精准，分别建立飞行速度与环境能见度对标准安全距离的调整关系，包括飞行速度—安全距离关系和环境能见度—安全距离关系，每个关系对应一个调整函数，其中，飞行速度—安全距离关系对应的调整函数为：Dv＝D0+V·Tr，其中，Dv为速度修正后的标准安全距离，D0为修正前的标准安全距离，V为低空飞行器当前速度，Tr为低空飞行器响应时间；环境能见度—安全距离关系对应的调整函数为：De＝max(D0,Dminγ·(Dmin-E))，其中，De为能见度修正后的标准安全距离，Dmin为最小预警探测距离，γ为能见度调整系数，如0.02～0.1，根据经验设定，E为当前能见度。随后，周期性地采集飞行器当前速度和能见度，并分别代入上述两个调整函数中计算调整量，得到对应的标准安全距离修正值。之后，使用计算的标准安全距离修正值对标准安全距离进行调整，替换掉原先的标准安全距离，再利用这个调整后的标准安全距离进行上述的安全边界膨胀操作，得到膨胀安全边界，这个膨胀安全边界作为障碍物的保护范围将被用于飞行空间的风险区识别与路径规划避让，确保飞行器在不同速度和环境条件下均具备动态可调的航线安全保障能力。

可选的，在低空飞行环境中，除了固定障碍物外，飞鸟、施工设备、无人机等动态障碍物对飞行安全也构成潜在威胁，因此，通过飞行器搭载的激光雷达、毫米波雷达、深度视觉相机等实时采集飞行器周围空间内物体的位置数据，并采用目标检测与跟踪算法(如YOLO+Kalman滤波、光流分析+区域生长等)对连续帧数据进行分析，从中识别出在空间中存在运动趋势的物体，即动态障碍物，每个被识别的动态障碍物被赋予一个唯一标识，并记录其实时空间坐标、速度矢量、尺寸等，其中，仅会将满足运动阈值(如位移>0.5m/s)的目标识别为动态障碍物，以避免静态误识。一旦动态障碍物被识别，会立即基于其当前运动速度、方向、历史轨迹进行短时运动预测，常用方法包括基于匀速模型的线性预测、基于运动轨迹库与行为模式匹配的机器学习预测(如LSTM或RNN)，预测结果是未来一段时间内(如5秒)动态障碍物可能占据的空间区域，称为动态安全边界，范围根据预测误差设置置信区域，如将轨迹两侧扩展若干米作为缓冲。完成动态安全边界预测后，将动态障碍物和预测结果映射至飞行空间地图的体素栅格结构中，将被动态障碍物在预测时间窗口内可能占据的体素单元标记为动态安全边界，这个动态安全边界构成了动态障碍物在空间中的包络范围，视为该障碍物短时间内可能到达的高风险空间区域；将动态安全边界之外，预设缓冲距离(例如0.5～1米)内的体素单元标记为风险区域，这部分区域代表低空飞行器接近但未进入动态障碍物预测路径的空间，可根据风险等级选择是否允许路径穿越，从而确保飞行器路径在飞行全程中的安全性和可行性。
以飞行器的当前位置为起点，目标飞行位置作为终点，在所述飞行空间地图中基于飞行器最小转弯半径、最大爬升率的约束条件，进行最小路径搜索，获得最小规划路径。
在一个实施例中，在飞行空间地图构建完成后，首先确定低空飞行器的起点与终点坐标，这两个点在三维空间中均为具体的体素单元中心位置，起点代表低空飞行器当前实时位置，终点则由飞行任务设定，如目标飞行位置。随后，依据已构建完成的三维飞行空间地图，从中提取出所有被标记为安全区域的体素单元，作为可通行空间，同时，将风险区视为仅在必要时可穿越的区域，将障碍区视为不可通行的区域，以此设定路径搜索约束条件。在此基础上，引入飞行器的动力学特性作为路径规划约束条件，包括最小转弯半径和最大爬升率(或下降率)，其中，最小转弯半径就是低空飞行器在水平方向上进行转向飞行时，所能维持的最小曲率半径；最大爬升率就是飞行器在竖直方向上单位时间内允许的最大高度变化速率。在满足以上约束的前提下，调用路径搜索算法对三维栅格空间进行最小路径搜索，常用算法包括三维A*(A-star)算法、Theta*、Lazy Theta*等。以三维A*算法为例，为每个候选节点设定一个启发式评估函数，计算其总代价为当前路径的累计飞行距离与该节点至终点的预估距离之和，在路径扩展过程中，仅选择位于安全区域内且满足最小转弯半径与最大爬升率限制的相邻节点进行搜索，并对路径中的方向变化和高度变化进行约束判断，再通过优先遍历总代价最小的节点序列，逐步扩展搜索范围，直到形成一条从起点到终点的连续路径，从而输出满足飞行器动力学约束条件的最短飞行路径作为最小规划路径，这个最小规划路径由一系列三维坐标点组成，反映了低空飞行器从当前位置至目标位置所需经过的空间通道，每个节点之间的空间转移都满足飞行器物理可实现性，路径整体避免了障碍物及其膨胀边界，为后续的飞行任务执行提供了高安全性、高效率的飞行路线。
对所述最小规划路径进行中间节点冗余验证，将满足验证评价结果的最小规划路径作为最短确定航线。
在一个实施例中，在完成最小规划路径的初步生成后，进一步对该路径中的中间节点进行冗余验证，以提升航线的紧凑性与实际可飞行性，例如，依次分析这些节点在路径中是否为飞行转向、爬升或规避障碍所必需的关键节点，是否在不违反飞行器最小转弯半径与最大爬升率约束的前提下，可以直接连接且完全处于安全区域内。随后，根据验证结果将冗余节点从路径中删除，使路径更加流畅，减少飞行姿态调整次数。最终，将经过上述冗余验证后的路径作为最短确定航线，这个最短确定航线不仅在空间上具备最小距离特性，还具备更高的实际飞行可执行性与控制稳定性。

优选的，在完成最小规划路径的生成后，对路径中的中间节点进行冗余删除或位置调整，以优化路径的平滑性并缩短总飞行距离。具体的，首先提取最小规划路径中的所有中间节点(即起点和终点之间的所有节点)，形成节点序列P＝{p1,p2,…,pn},，其中每个节点代表飞行器在路径中的一个转折点。随后，对路径中的每对中间节点pi和pj(其中j>i+1)进行检查，评估从pi到pj的直线路径是否与障碍物的膨胀安全边界存在碰撞。如果两点之间的直线路径完全处于安全区域内，且符合飞行器的最小转弯半径和最大爬升率约束，则认为路径中的中间节点pi+1，...，pj-1为冗余节点，这些节点不再是必需的，可以从路径中移除，并直接将pi和pj连接起来。对于不能完全剔除的节点，将对其进行角度调整。当两个相邻节点之间的转角过大，可能导致飞行器的转向过程不符合最小转弯半径的约束时，会计算合适的角度进行缩小或调整，调整后的节点将继续满足飞行器的动力学约束，同时使得路径更加平滑，减少飞行器的操控压力。在冗余节点剔除与角度调整后，基于贝尔曼最优性原理进行动态规划优化，验证当前路径是否为满足所有约束条件(如动力学约束、飞行器转向能力、爬升速率等)的全局最短路径，如果当前路径在某些部分不能满足约束，将进行进一步优化。若在路径优化过程中发现某些节点的路径仍可能受到障碍物影响，或者存在潜在的更短路径，会触发冲突重搜索，在此过程中，将重新评估新的潜在路径，并检查该路径是否与障碍物的膨胀安全区发生冲突，如果新路径在安全范围内且具有更短的飞行距离，则更新并选择该路径作为最终航线。最终，通过冗余节点剔除、角度调整、动态规划优化和冲突重搜索的组合，能够生成一条最短确定航线，不仅能够避开所有障碍物，并且符合飞行器的动力学特性，最大程度地减少飞行器的操控负担和飞行时间。
