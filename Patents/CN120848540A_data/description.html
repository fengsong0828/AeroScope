Description

面向低空无人飞行器的载荷分配方法及系统
技术领域
本申请涉及无人飞行器技术领域，尤其涉及面向低空无人飞行器的载荷分配方法及系统。
背景技术
随着无人飞行器的迅猛发展，低空无人飞行器因其操作灵活、成本低廉等优势，广泛应用于城市巡检、应急救援等多种场景。低空飞行区域通常存在复杂地形、多样障碍物以及动态变化的气象条件，这些因素对飞行器的飞行姿态、动力消耗和飞行安全产生显著影响。现有静态载荷分配方法多为静态配置，难以实时感知和响应环境变化，导致载荷配置不匹配，影响飞行器的稳定性和任务完成度。此外，低空无人飞行器在实际应用中常面临任务需求的频繁调整与切换，如监控任务的区域扩展、应急任务的临时增加等。传统载荷分配方案缺乏针对任务变化的动态调整机制，载荷配置缺乏灵活性，无法满足多任务、多场景的灵活切换需求，导致载荷利用效率低下，且飞行安全隐患增大。
综上所述，现有技术中存在由于飞行环境的多变性和不确定性，导致预设的载荷分配方案不适用，从而影响飞行安全的技术问题。
发明内容
本申请的目的是提供面向低空无人飞行器的载荷分配方法及系统，用以解决现有技术中存在由于飞行环境的多变性和不确定性，导致预设的载荷分配方案不适用，从而影响飞行安全的技术问题。

可选地，本地提取飞行采样需求，并依据所述飞行采样需求本地调用样本轨迹复杂度集和样本载荷分配策略集；对所述样本轨迹复杂度集和样本载荷分配策略集进行载荷特征聚合，得到多个样本复杂度区间和多个样本载荷模块分配；构建所述多个样本复杂度区间和多个样本载荷模块分配的索引规则，得到所述载荷分配规则库；采用所述轨迹复杂度序列遍历所述载荷分配规则库，得到所述载荷分配序列。
可选地，在所述低空无人飞行器从所述轨迹复杂度序列的第N轨迹分段进入第N+1轨迹分段时，从所述载荷分配序列映射调用第N+1载荷模块分配；基于所述第N+1载荷模块分配，动态调整任务调度器中感知模块、决策模块和控制模块的算力占比；以此类推，执行所述轨迹复杂度序列中各轨迹分段的载荷切换，直至所述目标飞行轨迹的飞行任务终结。

实施例一，请参阅附图1，本申请提供了面向低空无人飞行器的载荷分配方法，其中，所述面向低空无人飞行器的载荷分配方法通过面向低空无人飞行器的载荷分配系统执行，所述面向低空无人飞行器的载荷分配方法具体包括如下步骤：
S100：将目标飞行区域的边界经纬坐标输入GIS平台，获取区域GIS数据。
具体而言，获取目标飞行区域的边界经纬坐标，即飞行区域的四个角(或多个点)的经度和纬度数据。经度是指地球表面东西方向的角度，纬度是地球表面南北方向的角度。通过经纬度数据，可以精确界定飞行区域的位置。边界经纬坐标通过明确的位置定义出一个多边形区域，标示出飞行区域的边界。
GIS平台是地理信息系统，是一种用于获取、管理、分析和显示空间数据的系统。GIS平台能够处理不同种类的地理数据，如遥感图像、地形数据、道路网络等。将边界经纬坐标输入到GIS平台中，利用GIS平台的空间分析功能，获取目标飞行区域的详细空间数据(区域GIS数据)，包括地形起伏、建筑物分布、道路网格、气象数据，以及可能存在的禁飞区等。
GIS平台通过与卫星图像、地理数据库等集成，自动生成目标区域的详细空间数据。区域GIS数据是指与目标飞行区域相关的地理空间数据，包括地形数据、建筑物分布、道路、水体、禁飞区等信息。例如，假设目标飞行区域的边界坐标为北纬34.0522度，西经118.2437度至北纬34.0525度，西经118.2432度。将这些坐标输入GIS平台后，获取到该区域的高程数据(100米至150米)，土地覆盖类型(如城市、植被、水体等)，以及交通网络(如道路、铁路等)。通过获取区域GIS数据，有助于识别飞行区域的地形特征、潜在障碍物和关键兴趣点，从而提高飞行任务的安全性和效率。
S200：依据所述区域GIS数据对所述目标飞行区域进行复杂度量化，得到复杂度热力图。
进一步，本申请S200包括：
依据所述目标飞行区域的顶点经纬度坐标集构建外包络矩形；提取所述外包络矩形物理尺寸后，将所述物理尺寸的1/M粒度作为栅格尺度；以所述栅格尺度为分割约束，将所述目标飞行区域划分为多个等尺寸网格；依据所述区域GIS数据对所述多个等尺寸网格进行复杂度量化，得到所述复杂度热力图。
从所述区域GIS数据提取所述多个等尺寸网格的多个网格GIS数据；基于预设复杂度关联指标从第一网格GIS数据提取第一建筑密度、第一地形起伏度和第一障碍物高度方差；将所述第一建筑密度、第一地形起伏度和第一障碍物高度方差加载至加权模型，计算输出第一网格复杂度；以此类推，计算所述多个等尺寸网格的多个网格复杂度；依据所述多个网格复杂度对所述多个等尺寸网格执行相邻合并，直至输出所述复杂度热力图。
预设网格间隔距离和复杂度偏差阈；以所述网格间隔距离为约束，在所述多个等尺寸网格随机选定W个种子网格；采用所述W个种子网格遍历4-邻域网格，筛选满足所述复杂度偏差阈的相邻网格进行网格扩充合并后，计算W个更新网格复杂度；以此类推，执行网格合并迭代直至网格数量小于预设分割区域值，输出多个区域边界；空间无缝拼接所述多个区域边界拼接，生成所述复杂度热力图。
具体而言，获取目标飞行区域的顶点经纬度坐标集，即描述目标飞行区域的多个点的经纬度坐标集合，通常这些点按照一定顺序构成一个多边形区域。根据顶点经纬度坐标集构建外包络矩形，即能完全包含目标飞行区域的最小矩形。根据输入的飞行区域顶点经纬度坐标集，使用几何计算方法构建一个外包络矩形，即将目标区域用最小矩形包裹起来。外包络矩形是二维空间中能够包裹整个目标区域的最小矩形，用于简化复杂形状区域的分析与栅格化处理。
提取外包络矩形物理尺寸，即外包络矩形在实际地理空间中的宽度与高度，而非像素或坐标单位。设置一个粒度控制参数M(如M＝100)，将物理尺寸的1/M粒度作为栅格尺度。也就是说，将目标飞行区域等分为每边M个单元格，如M＝100，则每个网格表示整个区域1/100的宽度和高度。按照栅格尺度将整个外包络矩形划分成多个等尺寸网格，多个等尺寸网格都具有相同的尺寸和形状。
从区域GIS数据中提取多个等尺寸网格的多个网格GIS数据，包括每个网格的建筑分布、地形高度和障碍物信息。网格GIS数据是每一个等尺寸网格单元内所覆盖的GIS数据子集，通常包括该网格内所有的建筑物、地形高程点、障碍物等。从多个网格GIS数据中随机选择一个作为第一网格GIS数据，从第一网格GIS数据中提取第一建筑密度、第一地形起伏度和第一障碍物高度方差。第一建筑密度是第一网格GIS数据单位面积内建筑物的占地比例，通过计算建筑物投影面积总和与网格总面积的比值得到；第一地形起伏度是第一网格区域内高程变化的程度，通常用最大高差或标准差衡量；第一障碍物高度方差是第一网格区域内障碍物高度的方差，衡量高度分布的不均匀性，方差越大，障碍分布越复杂。例如，第一网格的建筑密度为30％(覆盖率)，地形起伏度为50米(高度差)，障碍物高度方差为10米2(波动程度)。
将第一建筑密度、第一地形起伏度和第一障碍物高度方差进行归一化后，加载至加权模型中，计算第一网格的第一网格复杂度。加权模型根据每个指标的相对重要性赋予不同的权重，如Ci＝w1*di+w2*ri+w3*vi，其中，Ci是第i个网格复杂度，di是第i个网格建筑密度，ri是第i个网格起伏度，vi是第i个网格高度方差，w1、w2、w3是权重。
对于多个等尺寸网格的其他网格，重复上述步骤，得到多个等尺寸网格的多个网格复杂度。根据多个网格复杂度对多个等尺寸网格进行相邻合并，将复杂度数值相近且空间相邻的多个网格单元合并成较大区域，以减少热力图中颜色跳变，提升可视化连续性。具体来说，预设网格间隔距离，即选取种子网格(初始聚类中心)时，各个种子网格之间必须保持的最小距离，防止种子聚集在局部区域，影响合并效果。预设复杂度偏差阈是两个网格之间复杂度数值的最大允许差值，若相邻网格的复杂度与当前种子网格的复杂度差小于此阈值，则认为它们相似，可以合并。
在多个等尺寸网格中，将网格间隔距离作为约束，随机选定W个种子网格，W个种子网格中任意两个种子网格中心点之间的距离小于网格间隔距离。采用W个种子网格遍历4-邻域网格，即向其上、下、左、右四个方向相邻网格查找。筛选出满足复杂度偏差阈的相邻网格进行网格扩充合并。也就是说，如果仙林网格的复杂度与当前网格复杂度的差值小于复杂度偏差阈，则进行网格扩充合并，计算W个更新网格复杂度，更新网格复杂度是基于合并前网格复杂度重新计算的。
重复这个网格合并迭代过程，直到网格数量小于预设分割区域值，停止扩充，得到多个合并区域，输出合并后的多个区域边界。预设分割区域值是聚类目标数量的上限值，也就是最终能接受的输出的最大复杂度区域数量。若某些区域合并完毕，其边界冻结，避免重叠。多个区域边界是合并后的不同复杂度区域之间的分界线。对多个区域边界进行空间无缝拼接，生成最终的复杂度热力图，确保了热力图的连续性和完整性。每个区域内填充与其复杂度值对应的颜色，形成色彩梯度图层。空间无缝拼接是指在空间上将多个合并后的网格区域边界组合为一张完整、连续的区域图，避免出现重叠或空白的拼接缝隙。
