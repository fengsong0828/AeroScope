Description

一种低空飞行器金属机壳加工用切割设备
技术领域
本发明涉及飞行器加工技术领域，具体为一种低空飞行器金属机壳加工用切割设备。
背景技术
在低空飞行器金属机壳的加工过程中，切割是一道关键工序，直接影响机壳的尺寸精度、连接强度及后续装配质量，目前，行业内针对低空飞行器金属机壳加工所采用的切割设备，多以传统激光切割机为核心，辅以简单的工装夹具构成加工系统，这些现有设备在基本切割功能上能够满足初步加工需求，即通过激光束的高温作用实现对金属材料的分离切割，然而现有技术在实际应用中存在诸多缺陷：
现有设备对切割过程中产生的烟尘、废气及微小颗粒等污染物的收集处理能力不足，多数设备缺乏有效的负压收集装置，或仅配备简单的抽风结构，难以实现对污染物的实时高效收集，这不仅会导致工作环境质量恶化，危害操作人员的身体健康，还会使污染物附着在激光切割机的内部精密部件上，造成设备侵蚀损坏，缩短设备使用寿命，影响切割工作的稳定高效进行；
再者，切割完成后的打磨处理环节存在明显短板，现有设备往往需要将切割后的工件转移至专门的打磨设备进行后续处理，工序衔接繁琐，降低了整体加工效率，即使部分设备集成了简单的打磨功能，也存在打磨不全面、不均匀的问题，其打磨件的运动方式单一，无法对工件切割处的内外壁进行全面覆盖，且缺乏有效的辅助手段去除表面毛刺、氧化层等杂质，难以保证打磨后的平整度和光洁度，同时，打磨过程中产生的碎屑和磨屑容易附着在工件或打磨件上，进一步影响打磨效果和效率，无法满足低空飞行器金属机壳高精度加工场景的需求。
发明内容
针对现有技术的不足，本发明提供了一种低空飞行器金属机壳加工用切割设备，解决了背景技术中所提及的技术问题。
为实现以上目的，本发明通过以下技术方案予以实现：一种低空飞行器金属机壳加工用切割设备，包括机箱，机箱表面两侧分别设置有处理组件与控制组件，控制组件一侧设置有对工件进行切割处理的切割组件；
切割组件包括固定安装在机箱上的固定座，固定座上转动连接有活动架，活动架一端活动连接有转动安装在机箱上的第一气缸，活动架另一端转动连接有调节杆，调节杆上螺纹连接有滑动安装在活动架上的移动座，移动座上固定安装有激光切割机。
作为本技术方案的进一步优选，机箱表面一侧固定安装有收集箱，收集箱顶部连通设置有第一连接管，第一连接管另一端连通设置有伸缩软管，伸缩软管另一端连通设置有第二连接管，第二连接管另一端连通设置有弯管，弯管另一端连通设置有吸收管，且吸收管与激光切割机的位置相互对应，移动座底部固定安装有固定箱，固定箱上设置有电机组，电机组输出端固定连接有转动安装在弯管内的风扇。
作为本技术方案的进一步优选，控制组件包括固定安装在机箱上的固定架，固定架外壁固定安装有驱动电机，固定架一侧转动安装有套筒，且套筒通过同步带轮传动件与驱动电机输出端传动连接，套筒一端固定安装有同步带轮传动件。
作为本技术方案的进一步优选，处理组件包括安装在机箱上的安装座，安装座顶部固定安装有伺服电机，伺服电机输出端固定连接有转动盘，转动盘轴心处固定安装有中心轴，转动盘偏离圆心处位置固定安装有外侧轴，中心轴、外侧轴的端部均设置有打磨件，中心轴上的打磨件对工件内壁进行打磨，外侧轴上的打磨件对工件外壁进行打磨处理。
作为本技术方案的进一步优选，打磨件包括固定安装在中心轴端部上的固定杆，固定杆另一端滑动连接有伸缩杆，固定杆两侧固定安装有第二气缸，第二气缸输出端与伸缩杆外壁固定连接，伸缩杆两侧固定安装有横板，横板上转动连接有表面打磨件，两个表面打磨件之间设置有对工件端部进行打磨处理的端部打磨件。
作为本技术方案的进一步优选，横板上转动连接有转动轴，且表面打磨件固定安装在转动轴上，转动轴内侧固定安装有第一锥齿轮，且其中一个转动轴端部设置有打磨电机，伸缩杆内部固定安装有第三气缸，第三气缸输出端转动连接有花键杆，花键杆另一端与端部打磨件固定连接，花键杆外壁滑动连接有套杆，且套杆转动安装在伸缩杆上，套杆外壁固定连接有与第一锥齿轮啮合连接的第二锥齿轮。
作为本技术方案的进一步优选，横板在竖直方向上滑动连接有活动框，活动框内壁两侧固定安装有齿杆，转动轴外壁固定连接有与齿杆啮合的半齿轮，横板两侧滑动连接有敲击杆，两个敲击杆外壁固定连接有安装架，且安装架与活动框外壁固定连接，安装架一侧设置有套设在敲击杆上的阻尼弹簧。
作为本技术方案的进一步优选，中心轴外壁套设有定位架，且定位架固定安装在转动盘上，定位架外壁转动连接有从动齿轮，且从动齿轮表面呈圆周阵列开设有弧形槽，定位架一侧固定安装有调节电机，调节电机输出端固定连接有与从动齿轮啮合的主动齿轮，定位架呈圆周阵列滑动连接有滑行杆，滑行杆端部固定连接有引导杆，且引导杆内侧端滑动安装在弧形槽内，滑行杆外侧端固定连接有弧形块。
与现有技术相比具备以下有益效果：
通过启动第一气缸可带动活动架转动，使激光切割机转动以精确调整与工件的高度距离，满足不同切割工艺要求；手动转动调节杆能带动激光切割机垂直上下移动，进一步灵活调整其与工件水平方向的相对位置，确保切割准确；通过开启电机组带动风扇转动形成负压，能将激光切割机切割产生的烟尘、废气及微小颗粒等污染物，顺着气流经吸收管、弯管、连接管、伸缩软管输送至收集箱，收集箱的过滤或容纳结构可有效拦截收集污染物，实现实时高效收集；有效改善激光切割机工作环境质量，减少污染物对操作人员身体健康的危害；避免污染物侵蚀损坏激光切割机内部精密部件，延长设备使用寿命，保障激光切割工作稳定高效进行。
通过精准定位与打磨初始保障：切割完成后，将两部分工件分离并固定定位，开启第二气缸带动伸缩杆、横板及表面和端部打磨件移动，使表面打磨件准确接触工件端部，为后续打磨提供初始位置；开启第三气缸带动花键杆移动，让端部打磨件精准位于两工件切割处之间，满足切割面打磨需求，开启打磨电机，通过一系列齿轮和花键传动，使表面和端部打磨件同步转动，同时开启伺服电机带动转动盘转动，进而使打磨件进行圆周运动，这种复合运动方式能对工件切割处的内外壁进行全面、均匀打磨，提高打磨质量和效率，转动轴转动带动半齿轮转动，配合齿杆使活动框带动敲击杆往复移动，对工件进行敲击，敲击产生的微小振动有助于打破工件表面应力集中区域，使打磨剂均匀分布，更全面去除表面毛刺、氧化层等杂质，提升平整度和光洁度；对于工件端部，敲击能使端部材料松动位移，在弹性与敲击力共同作用下，更好去除不规则边缘和毛刺，提高端部打磨精度，满足高精度加工场景需求，打磨产生的碎屑和磨屑若附着在工件或打磨件上会影响打磨效果和效率，敲击杆的敲击动作产生振动波，能将附着物震落，保持打磨区域清洁，确保打磨件与工件良好接触，提高打磨效率和效果。

具体实施方式
下面将结合说明书附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
实施例一：结合图1-图8所示，本发明提供一种技术方案：一种低空飞行器金属机壳加工用切割设备，包括机箱1，机箱1表面两侧分别设置有处理组件4与控制组件2，控制组件2一侧设置有对工件进行切割处理的切割组件3；
切割组件3包括固定安装在机箱1上的固定座31，固定座31上转动连接有活动架32，活动架32一端活动连接有转动安装在机箱1上的第一气缸33，活动架32另一端转动连接有调节杆34，调节杆34上螺纹连接有滑动安装在活动架32上的移动座35，移动座35上固定安装有激光切割机36；
机箱1表面一侧固定安装有收集箱37，收集箱37顶部连通设置有第一连接管38，第一连接管38另一端连通设置有伸缩软管39，伸缩软管39另一端连通设置有第二连接管310，第二连接管310另一端连通设置有弯管311，弯管311另一端连通设置有吸收管312，且吸收管312与激光切割机36的位置相互对应，移动座35底部固定安装有固定箱313，固定箱313上设置有电机组314，电机组314输出端固定连接有转动安装在弯管311内的风扇315。
在本发明的实施例中，通过启动第一气缸33，第一气缸33的活塞杆做伸缩运动，进而带动活动架32围绕其在固定座31上的铰接点进行转动，由于激光切割机36固定安装于活动架32上，活动架32的转动会同步带动激光切割机36朝着工件所在的一侧发生转动，这种转动方式能够精确地调整激光切割机36与工件之间的高度距离，以满足不同切割工艺对于高度参数的要求，同时，操作人员可以手动转动调节杆34，调节杆34通过其与移动座35之间的螺纹传动，将旋转运动转化为移动座35的直线运动，从而带动移动座35以及固定在移动座35上的激光切割机36在垂直方向上进行上下移动，通过这种上下移动操作，能够进一步灵活地调整激光切割机36与工件在水平方向上的相对位置关系，确保激光切割机36能够准确地对工件进行切割作业；
通过开启电机组314能够带动风扇315转动，风扇315高速旋转时会在弯管311以及整个气流通道内形成负压环境，由于吸收管312与激光切割机36的位置相互对应，在负压的作用下，激光切割机36在切割材料过程中产生的烟尘、废气以及微小颗粒等污染物，会顺着气流方向被迅速吸入吸收管312内，这些污染物进入吸收管312后，会依次通过弯管311、第二连接管310、伸缩软管39和第一连接管38，最终被输送到收集箱37中，收集箱37内部通常设置有过滤装置或者特定的容纳结构，能够对进入的污染物进行有效的拦截和收集，防止其再次扩散到周围环境中，随着电机组314持续运转，风扇315不断产生负压，使得整个气流循环持续进行，从而能够实时、高效地将激光切割机36工作区域产生的污染物收集到收集箱37内，有效改善了激光切割机工作时的环境质量，减少了污染物对操作人员身体健康的危害，同时也避免了污染物对激光切割机内部精密部件的侵蚀和损坏，延长了设备的使用寿命，保障了激光切割工作的稳定、高效进行。
实施例二：结合图4所示，在实施例一的基础上，控制组件2包括固定安装在机箱1上的固定架21，固定架21外壁固定安装有驱动电机22，固定架21一侧转动安装有套筒23，且套筒23通过同步带轮传动件24与驱动电机22输出端传动连接，套筒23一端固定安装有同步带轮传动件24。
在本发明的实施例中，在工件加工流程中，首先将工件以精确的定位方式从套筒23与卡盘25的中心位置穿过，卡盘25具备可靠的夹持功能，通过其内部的夹紧机构，如卡爪的径向移动，对工件施加均匀且足够的夹紧力，从而将工件稳固地夹持固定在预定位置，确保在后续加工过程中工件不会发生位移或松动，为高精度加工提供基础保障，当开启驱动电机22后，驱动电机22作为整个旋转系统的动力源开始工作，其输出端按照设定的转速和转向进行旋转，输出的扭矩通过同步带轮传动件24进行传递，同步带轮传动件24由主动带轮、从动带轮和同步带组成，主动带轮安装在驱动电机22输出端，从动带轮安装在套筒23上，在传动过程中，同步带与带轮之间依靠齿形啮合，避免了打滑现象，能够精确地将驱动电机22的旋转运动和动力传递到套筒23上，套筒23在接收到旋转动力后，带动与之刚性连接的卡盘25一起转动，由于卡盘25已经牢固地夹持着工件，所以工件也会随着套筒23和卡盘25同步转动，形成一个稳定的旋转加工系统，此时，切割组件3根据预先设定的加工参数和程序，对处于旋转状态的工件进行切割处理。

在本发明的实施例中，在切割工作完成后，将两部分工件进行分离，即两个工件之间的距离与端部打磨件411的规格相互适配，并对两部分工件进行固定定位，通过开启第二气缸423来实现部分打磨组件的初步定位，第二气缸423作为直线运动执行元件，其活塞杆的伸缩运动带动伸缩杆48进行直线移动，由于横板49与伸缩杆48固定连接，表面打磨件410和端部打磨件411又安装在横板49上，因此伸缩杆48的移动会带动横板49、表面打磨件410和端部打磨件411同步移动，使表面打磨件410准确与工件端部接触，为后续的打磨操作提供初始位置保障，接着开启第三气缸424，第三气缸424同样作为直线运动执行元件，其活塞杆的伸缩带动花键杆425进行直线移动，端部打磨件411安装在花键杆425上，所以花键杆425的移动会使端部打磨件411精准地位于切割后的两个工件的切割处之间，以满足对切割面进行打磨的需求，然后开启打磨电机413，打磨电机413作为动力源，其输出轴带动转动轴412同步转动，转动轴412与表面打磨件410和第一锥齿轮414固定连接，因此会带动表面打磨件410和第一锥齿轮414同步转动，第一锥齿轮414与第二锥齿轮427相互啮合，第一锥齿轮414的转动会带动第二锥齿轮427转动，第二锥齿轮427又与花键杆425通过花键连接，从而将动力传递给花键杆425和端部打磨件411，使端部打磨件411也同步转动，此时，转动的表面打磨件410能够对工件的外壁进行打磨处理，端部打磨件411能够对工件的内壁和切割面进行打磨处理，此外，开启伺服电机42，伺服电机42作为旋转运动的动力源，其输出轴带动转动盘43转动，中心轴44和外侧轴45安装在转动盘43上，因此会随着转动盘43同步转动，中心轴44和外侧轴45又分别带动对应的打磨件46转动，使得表面打磨件410与端部打磨件411在进行自身转动的同时，还进行圆周运动，这种复合运动方式能够使打磨件对工件切割处的内外壁进行全面、均匀的打磨处理，提高打磨质量和效率；
并且转动轴412转动时，能够带动半齿轮417同步转动，半齿轮417在转动时，配合两侧啮合的齿杆416能够带动活动框415上下往复移动，从而配合阻尼弹簧420的弹性作用力能够使得活动框415带动安装架419、敲击杆418往复移动，使得敲击杆418对工件进行敲击出来，从而能够配合表面打磨件410与端部打磨件411在打磨时对工件进行敲击处理，在打磨过程中，表面打磨件410和端部打磨件411对工件进行常规打磨操作时，敲击杆418的往复敲击动作能够使工件产生微小振动，这种振动有助于打破工件表面因打磨产生的应力集中区域，让打磨剂更均匀地分布在工件表面，使得表面打磨件410能够更全面、均匀地去除工件表面的毛刺、氧化层等杂质，提升工件表面的平整度和光洁度，改善打磨质量，对于工件的端部，端部打磨件411在打磨时，敲击杆418的敲击可以使端部材料产生一定的松动和位移调整，在弹性作用力和敲击力的共同作用下，能够更好地去除端部的不规则边缘和毛刺，使端部形状更加规整，提高端部打磨的精度，满足对工件端部尺寸和形状精度要求较高的加工场景，打磨过程中会产生大量的碎屑和磨屑，这些残留物如果附着在工件表面或打磨件上，会影响打磨效果和效率，敲击杆418的敲击动作能够产生振动波，将附着在工件表面和打磨件上的碎屑震落，保持打磨区域的清洁，确保表面打磨件410和端部打磨件411始终以良好的状态与工件接触，提高打磨效率和效果。
