Description

低空飞行器空调系统
技术领域
本发明涉及一种低空飞行器空调系统，属于低空飞行器技术领域。
背景技术
低空飞行器，作为一种新兴的空中交通工具，正在逐步进入实际应用阶段。随着无人机、飞行出租车等技术的快速发展，低空飞行器的空调系统成为保障乘客舒适性和设备正常运行的关键组成部分。这些飞行器通常具有较小的机舱空间，高度集成的设计和有限的能源供应，使得空调系统需要具备高效、轻量和低能耗的特点。当前，低空飞行器的空调系统多采用紧凑型压缩机、电子膨胀阀以及空气循环技术，以确保在高温、低温及空气密度变化的复杂环境下，舱内环境保持适宜。
如图1、图2所示，传统汽车热管理系统由多个部件组成，主要包括空调压缩机4、冷凝器5、储液干燥器11、热力膨胀阀12(即第一热力膨胀阀3和第二热力膨胀阀6)、蒸发器2和鼓风机7等主要部件。传统汽车热管理系统通过空调压缩机4驱动制冷剂循环，在系统内实现吸热和放热的过程，从而调节车内温度。
系统从空调压缩机4开始工作。低温低压的气态制冷剂被吸入空调压缩机4，经过压缩变为高温高压的气体。空调压缩机4通过皮带与汽车发动机连接，利用发动机的动力驱动制冷剂循环。
高温高压的气态制冷剂从空调压缩机4输出后进入冷凝器5。冷凝器5通常安装在车辆前部，其表面积较大，通过与外界空气的热交换，制冷剂释放热量并降温。当热量充分释放后，制冷剂从气态转变为高压液态。车辆在行驶过程中，空气流动加速了冷凝器5的冷却效果。
液态制冷剂经过储液干燥器11，储液干燥器11的内部装有吸湿材料，可以吸收制冷剂中的水分，同时过滤杂质。这一过程确保制冷剂纯净，避免在低温环境下发生冰堵或管道腐蚀，保障系统的长期稳定运行。
经净化的高压液态制冷剂进入热力膨胀阀12，热力膨胀阀12使其压力迅速下降，温度随之降低，形成低温低压的液态制冷剂。热力膨胀阀12还可根据需求调节制冷剂的流量，以精准控制系统的冷却能力。低温低压的液态制冷剂进入发器2。蒸发器2通常安装在车内空调系统的通风管道中。此时，制冷剂吸收车内空气中的热量并蒸发为气态。
空气循环系统通过鼓风机7，将车内的热空气引导至蒸发器2表面，与低温制冷剂充分接触后被冷却，再将冷空气吹入驾驶室1，实现降温。吸收热量后的气态制冷剂返回空调压缩机4，完成一个循环过程。
传统汽车热管理系统的核心部件是空调压缩机，但其体积及重量大，对低空飞行器等对重量和空间敏感的设备存在局限性。
发明内容
本发明要解决的技术问题是：如何降低低空飞行器的空调系统体积及重量的问题。
为了解决上述技术问题，本发明的技术方案是提供了一种低空飞行器空调系统，包括蒸发器，蒸发器的一侧设有将经过蒸发器气体吹向驾驶室的鼓风机，其特征在于，所述的蒸发器一端通过第一管路接通储气罐，储气罐内存储有高压冷却液，高压冷却液为被压缩成液态的制冷剂气体，第一管路上设有第一热力膨胀阀，蒸发器另一端通过第二管路连接至大气，第二管路上设有第二热力膨胀阀；高压冷却液通过第一热力膨胀阀迅速降压降温，转变为低温低压液体后进入蒸发器，蒸发器将低温低压液体汽化，汽化后的制冷剂气体通过第二热力膨胀阀排放至大气中。
优选地，所述的第一热力膨胀阀的两端并联有带有电池模组的支路，第一管路上且位于第一热力膨胀阀两端的位置分别设有三通阀，第一热力膨胀阀两端并联的带有电池模组的支路通过三通阀与第一管路连接。
优选地，所述的三通阀为能分别调节通向电池模组所在支路以及通向蒸发器冷却液流量的调节阀。
优选地，低空飞行器启动制冷系统后，所述的储气罐中的液态二氧化碳通过减压装置释放，被输送至三通阀，三通阀根据低空飞行器的负载状态和温度需求动态调节冷却液流向：
在高负载或快速充放电时，通过调节三通阀，将冷却液优先被引导至电池模组所在的支路，以集中散热；
在低负载或待机状态下，通过调节三通阀，调整流量以适应系统需求；高压冷却液通过第一热力膨胀阀迅速降压降温，转变为低温低压液体后进入蒸发器，在蒸发器中，冷却液吸收热量并汽化。
优选地，所述的蒸发器位于鼓风机和驾驶室之间，蒸发器位于鼓风机向着驾驶室吹风的路径上。
优选地，所述的制冷剂气体为二氧化碳。
本发明的低空飞行器空调系统由传统汽车热管理系统演变，将储气罐+可压缩气体瓶代替传统汽车热管理系统中的空调压缩机+冷凝器，并增加了带有电池模组的冷却回路，大大减少了零部件重量，降低了成本。
本发明的低空飞行器空调系统通过比传统汽车热管理系统更精简的设计，在确保有效散热的同时，避免了占用过多空间和增加过多重量的问题，从而有助于飞行器性能的提升。

具体实施方式
为使本发明更明显易懂，兹以优选实施例，并配合附图作详细说明如下。
本发明提供了一种低空飞行器空调系统，如图3所示，其包括蒸发器2，蒸发器2的一侧设有向着驾驶室1吹风的鼓风机7，蒸发器2位于鼓风机7和驾驶室1之间，且蒸发器2位于鼓风机7向着驾驶室1吹风的路径上，蒸发器2一端通过第一管路接通储气罐9，储气罐9内存储有被压缩成液态的制冷剂气体(二氧化碳)，蒸发器2与储气罐9之间的第一管路上设有第一热力膨胀阀3，第一热力膨胀阀3的两端并联有带有电池模组10的支路，蒸发器2与储气罐9之间的第一管路上且位于第一热力膨胀阀3两端的位置分别设有三通阀8，第一热力膨胀阀3两端并联的带有电池模组10的支路通过三通阀8与蒸发器2与储气罐9之间的第一管路连接。蒸发器2另一端通过第二管路连接至大气，该第二管路上设有第二热力膨胀阀6。其中，电池模组10为低空飞行器的工作电源。三通阀8为能分别调节通向电池模组10所在支路以及通向蒸发器2冷却液流量的调节阀。
本发明的原理是将液态的可压缩气体通过蒸发器2汽化，用鼓风机7将冷风吹进驾驶室1。
低空飞行器的工作环境与传统汽车显著不同，其特点是置空时间有限、对轻量化要求高。基于此，本发明的系统从传统汽车热管理系统的闭环制冷模式改进为一种利用有限容积已压缩制冷剂(二氧化碳)的非闭环制冷系统。存储高压液态制冷剂(二氧化碳)将代替传统汽车系统中的空调压缩机作为核心动力源。
动力电池是低空飞行器的主要能源来源，在飞行过程中会产生大量热量，尤其在高负载或快速充放电期间。本发明的低空飞行器热管理系统利用热交换原理保证系统安全稳定运行。
低空飞行器启动制冷系统后，储气罐9中的液态二氧化碳通过减压装置释放，被输送至三通阀8。三通阀8根据飞行器的负载状态和温度需求动态调节冷却液流向：在高负载或快速充放电时，冷却液优先被引导至电池区域(即电池模组10所在的冷却回路)以集中散热；在低负载或待机状态下，则调整流量以适应系统需求。高压冷却液通过第一热力膨胀阀3迅速降压降温，转变为低温低压液体后进入蒸发器2。在蒸发器2中，冷却液吸收电池或其他热源的热量并汽化，同时有效将热量带走。为提升冷却效率，鼓风机7强制空气流经蒸发器2，加速热交换过程并将多余热量排出。
本发明的低空飞行器空调系统与传统汽车热管理系统不同，冷却液完成制冷任务后，不会回到储气罐9，而是通过第二热力膨胀阀6排放至大气中。这种开环设计消除了传统闭环系统的循环需求，有效降低了系统复杂性和重量，为低空飞行器提供了轻量化、高效的热管理解决方案。

1、系统架构的差异
传统汽车的热管理系统主要依靠空调制冷循环来调节车内温度，通常包括空调压缩机、冷凝器、储液干燥器、热力膨胀阀、蒸发器和鼓风机等关键部件。制冷系统通过压缩机压缩制冷剂，使其从低压气体转变为高压气体，然后通过冷凝器冷却并转化为液体，再通过热力膨胀阀降压并冷却，最后在蒸发器中吸收车内的热量。这一过程中，空调压缩机和冷凝器是关键的热交换和温度调节部件。
与传统汽车系统不同，本发明的低空飞行器的热管理系统更加注重飞行器内部设备和动力电池的温控，而飞行器对重量和空间的严格要求也决定了热管理系统的架构差异。低空飞行器通常不使用空调压缩机和冷凝器，而是采用罐装的可压缩气体(二氧化碳)代替这些组件。这些可压缩气体以较高的压力储存在专门设计的储气罐9中，在需要时通过流体动力学控制释放到飞行器的热交换系统中进行温度调节。这种方法不仅简化了系统的设计，还有效减少了重量，适应了飞行器对空间和能源的高效利用需求。
2、冷却介质的差异
在传统汽车中，冷却液通常是液态的，如水和乙二醇的混合物，流经发动机、空调系统及其他热源，吸收热量后通过散热器等部件释放。空调系统中的冷却剂通常为氟利昂类制冷剂，在压缩机的作用下进行气液转换，通过蒸发器和冷凝器交换热量。这些液态冷却剂和气态制冷剂在热交换过程中起到了关键的热管理作用。
本发明的低空飞行器则采用了不同的冷却介质，特别是在热管理系统中，通常会选择罐装的可压缩气体。这些气体储存于储气罐9中，在需要时通过系统调节释放。在飞行器的热管理系统中，这些气体通常在高压下被注入热交换系统，吸收飞行器设备产生的热量，并通过快速膨胀和蒸发过程带走热量。与传统汽车使用的液态冷却剂不同，这种压缩气体在膨胀过程中能够直接吸收热量，从而有效调节飞行器的温度。
3、关键部件的差异
在传统汽车热管理系统中，空调压缩机和冷凝器是两个不可或缺的组件。压缩机通过机械驱动将制冷剂压缩成高压气体，然后通过冷凝器将其冷却并转化为液体。然而，这种系统较为复杂，并且需要大量空间和能量，尤其是空调压缩机的驱动通常需要消耗发动机的动力。
本发明的低空飞行器则取消了空调压缩机和冷凝器，改为使用罐装的可压缩气体。可压缩气体在高压状态下储存在特殊设计的储气罐9中，而不需要像传统系统那样进行复杂的压缩和冷却过程。这种设计减少了系统的体积和重量，同时避免了高能耗和复杂机械驱动的问题。通过三通阀和热力膨胀阀的精确控制，飞行器可以根据实际需要调整可压缩气体的释放量，从而调节飞行器内部的温度。
4、热管理效率与系统集成
传统汽车的空调系统虽然成熟，但其效率受到多个因素的限制，例如制冷剂流量控制、压缩机效率和冷凝器的散热性能等。而且，空调系统的运行通常需要发动机提供动力，这导致了能量的消耗和额外负担，特别是在长时间使用空调时。
本发明的低空飞行器的热管理系统，通过采用罐装可压缩气体和高效的热交换方式，能够更加高效地控制飞行器内部的温度。由于不依赖发动机驱动压缩机，飞行器的热管理系统更加灵活，能够在不同飞行模式下迅速响应温度变化，减少能量消耗。此外，系统的紧凑设计使得热管理系统能够更好地与飞行器的其他系统集成，提高了整体效率和可靠性。
5、重量与空间优化
低空飞行器在设计时面临着严格的重量和空间限制，因此其热管理系统的设计更注重轻量化和紧凑性。本发明取消空调压缩机和冷凝器后，采用罐装可压缩气体的方案显著减少了系统的重量和体积。相比传统汽车需要容纳多个大型部件，低空飞行器的热管理系统通过更精简的设计，在确保有效散热的同时，避免了占用过多空间和增加过多重量的问题，从而有助于飞行器性能的提升。
传统汽车热管理系统与本发明的低空飞行器热管理系统之间存在诸多差异，尤其是在系统架构和关键部件的选择上。低空飞行器通过取消传统的空调压缩机和冷凝器，采用罐装的可压缩气体替代，极大地简化了热管理系统的设计，降低了系统的重量和体积，提升了能效和可靠性。这种创新设计使得低空飞行器能够更加高效地控制飞行过程中产生的热量，确保设备在高效能和安全性方面的表现。
