Description

一种低空垂直起降飞行器用混合升力与推力系统及其运行 方法
技术领域
本发明属于航空动力系统技术领域，涉及一种低空垂直起降飞行器用混合升力与推力系统，具体涉及一种低空垂直起降飞行器用混合升力与推力系统。
背景技术
随着低空经济的提出，低空有人/无人飞行器发展迅速，目前此类飞行器多采用动力电池供电的技术路线，通过电机带动升力螺旋桨而提供升力。受限于动力电池能量密度的限制，目前电池供电的低空飞行器难以长时间续航，制约了低空有人/无人飞行器的发展。本专利提出了一种PEM燃料电池与燃气轮机联合的混合升力与推力系统来替代目前的动力电池动力系统，燃气轮机的喷气推力可以提供飞行器飞行的推进，燃气轮机与燃料电池发出的电能通过电动机带动升力螺旋桨提供升力。本发明为长航时飞行器提供了新的技术解决方案。
发明内容
为了解决上述问题，本发明提供一种低空垂直起降飞行器用混合升力与推力系统及其运行方法，具有可同时提供升力和推力的功能，调节灵活，且具有长航时续航的特点，本发明可为低空经济飞行器提供多电长航时混合动力技术。
本发明的技术方案如下：
一种低空垂直起降飞行器用混合升力与推力系统，包括低压压气机、起/发电机、高压压气机、燃气涡轮、可调推力喷管、PEM燃料电池、电动机、升力螺旋桨，低压压气机、起/发电机、高压压气机和燃气涡轮为同轴结构，低压压气机压缩的空气分为两路，一路压缩空气进入高压压气机进行进一步增压后进入主燃烧室，主燃烧室后端连接燃气涡轮；低压压气机的另一路压缩空气分别进入PEM燃料电池的阴极入口和可调推力喷管；PEM燃料电池对电汇流条供电，PEM燃料电池的反应后端连接混合器；燃气涡轮的排气口连接混合器，混合器将燃气涡轮的尾气和PEM燃料电池后尾气进行混合后，提供给后端的可调推力喷管进而提供推力；起/发电机连接电汇流条，电汇流条连接电动机，电动机的转轴驱动升力螺旋桨。
进一步的，还包括次级燃烧室，次级燃烧室设在PEM燃料电池的反应后端，PEM燃料电池中未完全反应的可燃气体和氧气在次级燃烧室中进一步燃烧，进一步燃烧的燃气再进入混合器中。
进一步的，还包括H2燃料罐和燃料罐，燃料罐连接主燃烧室，H2燃料罐连接PEM燃料电池的阳极入口。
进一步的，还包括系统控制器和起动器，起动器由电汇流条供电，起动器连接起/发电机；系统控制器连接并控制起动器、起/发电机、PEM燃料电池和推力调节阀，推力调节阀设在低压压气机分流向可调推力喷管的支路上。
进一步的，还包括DC/DC电力变换单元和AC/DC电力变换单元，起/发电机通过AC/DC电力变换单元给电汇流条供电，PEM燃料电池通过DC/DC电力变换单元给电汇流条供电。
进一步的，还包括水冷却路、换热器和载冷剂冷却回路，水冷却路经过换热器的冷端后分为两支水路：一支水路和进入PEM燃料电池的阴极的压缩空气混合为湿空气；另一支水路对PEM燃料电池整体进行换热，然后再喷入主燃烧室；载冷剂冷却回路经过换热器的热端后，分别对系统控制器、起动器、DC/DC电力变换单元和AC/DC电力变换单元换热，再进入换热器的热端循环。
进一步的，还包括次级排放阀，次级排放阀由系统控制系统连接控制，次级排放阀设在次级燃烧室的后端，次级排放阀另一端连接外部。
一种低空垂直起降飞行器用混合升力与推力系统的运行方法，使用前述的一种低空垂直起降飞行器用混合升力与推力系统，包括以下方法：
根据飞机状态和系统工作需求进行工作模式切换，飞行器起飞过程中需提供大功率能量，通过控制起/发电机调整燃气涡轮发动机电力提取，降低可调推力喷管的推力至零，PEM燃料电池同时工作，电力分配系统分配电动机电力驱动升力螺旋桨提供起飞升力；飞行器进入巡航阶段后，控制减少起/发电机的能量提取，燃气涡轮尾气通过可调推力喷管提供推力。
进一步的，在飞行器不同飞行高度对起/发电机的控制实现对低压压气机的转速调节，从而调节进入到PEM燃料电池阴极的高压空气流量稳定。
进一步的，对起/发电机进行控制，在飞机处于地面状态时，控制起/发电机以低转速节能工作；在飞机处于高空状态时，系统控制器控制起/发电机以高转速工作，同时打开水冷却路，辅助以水的冷却加湿，维持低空和高空状态下PEM燃料电池的工作环境温度。
本发明的优点：
1、本发明提供了一种低空垂直起降飞行器所使用的混合升力与推力系统，能够用于采用旋翼起降与推力巡航的有人机和无人机。
2、本发明减少了转轴的使用数量，通过将低压压气机、高压压气机、燃气涡轮、起/发电机安装在同一根轴上，去掉了传统的齿轮减速传动装置，极大简化了系统结构。
3、本发明的主燃烧室可接受煤油等便携燃料，并且设计了水喷结构，采用水冷转化为蒸汽的方式，避免了冷却空气的使用带来的性能衰减，同时减少污染物排放。
4、本发明通过设计H2燃料罐、PEM燃料电池、次级燃烧室组成的燃料电池发电单元。PEM燃料电池接受来自低压压气机的高温压缩气，压气机的高温高压气可以对燃料电池冷启动进行热空气吹扫，帮助PEM燃料电池解决低温起动的问题，同时压缩机出口的高温高压空气通过液态水的冷却而进行加湿的同时降温，简化了系统附件的要求。
5、本发明通过采用双燃料的思路，PEM燃料电池使用由H2罐提供的高纯度氢气，燃机单元使用由燃料罐提供的便携燃料，一方面在一定程度的解决了H2对航空飞行器不好携带的问题，即减少了对H2的需求，另一方面燃料罐携带煤油等便携燃料增加了系统的工作时间。
