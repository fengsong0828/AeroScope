Description

一种低空飞行器零部件的自动加工脱模装置
技术领域
本发明涉及自动脱模技术领域，特别涉及一种低空飞行器零部件的自动加工脱模装置。
背景技术
低空飞行器零部件自动加工脱模装置不仅提高了生产效率和产品质量，还为工厂带来了更安全、环保的工作环境以及更高的经济效益；自动化脱模装置能够显著减少人工干预，加快脱模速度，从而提升整个生产工艺流程的效率；通过精确控制脱模力和速度，可以避免对制品表面造成损伤，确保产品尺寸精度和表面质量的一致性；减少了工人直接接触高温模具或复杂结构的可能性，降低了劳动强度和职业健康风险；但目前市面上流行的飞行器零部件自动加工脱模装置存在比较明显的缺陷：首先，零件在热胀冷缩之后内壁会形成内应力，不利于后续脱模；其次，脱模完成后需要人工运走，增加了生产成本；因此设计一种低空飞行器零部件的自动加工脱模装置是很有必要的。
发明内容
本发明的目的是提供一种低空飞行器零部件的自动加工脱模装置，用以解决现有的零件在热胀冷缩之后内壁会形成内应力，不利于后续脱模和脱模完成后需要人工运走，增加了生产成本的缺陷。
为了解决上述技术问题，本发明提供如下技术方案：一种低空飞行器零部件的自动加工脱模装置，包括脱模平台、支撑平台、滑动平台、支撑板、注塑孔、透气孔、螺纹套筒、螺纹杆和机械爪，所述脱模平台内部开设的凹槽中对称转动连接螺纹杆，螺纹杆螺纹配合连接螺纹套筒，螺纹套筒滑动连接在下模具开设的凹槽中，下模具上的定位孔固定连接有上模具定位柱，上模具上开设有注塑孔和透气孔，上模具顶部固定连接有第一固定法兰，第一固定法兰上套接有汽缸，汽缸上套接有第二固定法兰，第二固定法兰固定连接在顶板上，顶板上开设有第一燕尾槽和第二燕尾槽，第一燕尾槽上滑动连接有第二滑块，第二滑块上开设的凹槽中转动连接有第二滑动轮和第二转动轴，第二滑动轮和第二转动轴紧贴着第二皮带。
作为本发明的进一步技术方案，所述脱模平台固定连接有支撑平台，支撑平台底部对称固定连接有桌腿，桌腿上固定连接桌脚。
作为本发明的进一步技术方案，所述脱模平台固定连接有下模具和滑动平台，滑动平台上固定连接有第一电机，第一电机的输出端固定连接有第一圆柱齿轮，第一圆柱齿轮啮合连接支撑板上开设的齿槽，支撑板滑动连接在下模具开设的通槽中，支撑板滑动连接在滑动平台开设的凹槽中。
作为本发明的进一步技术方案，所述支撑平台固定连接有第四电机，第四电机的输出端固定连接有第三圆柱齿轮，第三圆柱齿轮啮合连接第二圆柱齿轮，第二圆柱齿轮固定连接在螺纹杆上。
作为本发明的进一步技术方案，所述支撑平台上对称固定连接支撑柱的一端，支撑柱的另一端固定连接有顶板，顶板上固定连接有第一固定块、第二固定块、第三固定块和第四固定块，第一固定块和第二固定块上固定连接第一皮带，第一皮带贯穿第一滑块紧贴第一转动轴和第一滑动轮，第一转动轴和第一滑动轮转动连接在第一滑块上，第一滑块固定连接有第二电机，第二电机的输出端固定连接在第一转动轴上，第一滑块滑动连接在第二燕尾槽中。
作为本发明的进一步技术方案，所述第三固定块和第四固定块固定连接有第二皮带，第二皮带贯穿第二滑块，第二滑块固定连接第三电机，第三电机的输出端固定连接在第二转动轴上，第二滑块滑动连接在第一燕尾槽中。
作为本发明的进一步技术方案，所述第二滑块固定连接有固定臂，固定臂上固定连接有第七电机，固定臂上转动连接有第二机械臂，第二机械臂上转动连接有第一机械臂，第一机械臂上转动连接有机械爪，第一机械臂上固定连接有第六电机，机械爪上固定连接有第五电机。
本发明提供的一种低空飞行器零部件的自动加工脱模装置，其优点在于：通过材料从注塑孔进入到下模具和上模具形成的密闭空间进行零件制作，零件制造完成后完成后，第一电机的输出端转动带动第一圆柱齿轮转动，进而带动支撑板在滑动平台滑动，最终将支撑板滑出下模具开设的凹槽，实现平衡模具内外的压强的目的，有利与后续脱模；汽缸带动上模具缓缓上升，第二电机的输出端转动带动第一转动轴转动，进而带动第一滑块在第二燕尾槽中滑动，最终将上模具带离；第四电机的输出端转动带动第三圆柱齿轮转动，进而带动第二圆柱齿轮和螺纹杆转动，最终带动螺纹套筒缓缓上升，将零件推出下模具；第五电机、第六电机和第七电机分别启动，分别驱动机械爪、第一机械臂和第二机械臂抓取零件，第三电机的输出端转动带动第二转动轴转动，进而带动第二滑块在第一燕尾槽中滑动，最终带动固定臂、机械爪、第一机械臂和第二机械臂运动，运动到指定区域，机械爪放下零件，完成整个脱模过程，不需要人工干预，能够大大降低生产成本。
