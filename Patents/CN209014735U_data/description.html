Description

用于低空飞行器定位的控制电路和定位数据输出系统
技术领域
本实用新型涉及航空技术领域，尤其涉及一种用于低空飞行器定位的控制电路和低空飞行器定位数据输出系统。
背景技术
在低空天网建设的过程中，需要由用于低空飞行监控系统来对低空飞行走廊内的低空飞行物(含有人驾驶和无人驾驶的飞行器)进行实时监控，以方面地面管理系统能够实时的掌握空中飞行走廊的交通状态，确保飞行安全和空中飞行走廊的充分利用。同时，也需要空中飞行物以一种用于低空飞行通信网关实时向地面自动报告相关飞行参数，如北斗坐标、飞行状态信息等。
在实际使用过程中，雷达系统获得的坐标参数和通信网关报告的北斗系统获得的坐标参数会产生差异，因此，如何获得准确的坐标参数是目前亟待解决的问题。
实用新型内容
本实用新型旨在至少从一定程度上解决相关技术中的技术问题之一。
为此，本实用新型的第一个目的在于提出一种用于低空飞行器定位的控制电路，能够通过该电路来对比北斗系统和雷达系统获得的坐标数据所标示的电压值，校正误差，以获得准确的唯一的坐标所标示的电压值，生成唯一精准坐标。
本实用新型的第二个目的在于提出一种低空飞行器定位数据输出系统。
为达上述目的，本实用新型第一方面提出了一种方法，包括：输入电路，比较电路和输出电路，其中，所述输入电路与所述比较电路相连，所述输出电路与所述比较电路的相连，所述输出电路包括第一输入端和第二输入端，所述输入电路，用于接收北斗系统对低空飞行器实时采集的第一坐标电压值和雷达系统对所述低空飞行器实时采集的第二坐标电压值；所述比较电路，用于根据所述第一坐标电压值和所述第二坐标电压值以及预设的电压值，输出第一控制信号或者第二控制信号；所述输出电路，用于根据所述第一控制信号导通所述第一输入端，输出所述第一坐标电压值，或者，根据所述第二控制信号导通所述第二输入端，输出所述第二坐标电压值。
本实用新型提出的用于低空飞行器定位的控制电路，通过将输入电路与比较电路相连，输出电路与比较电路的相连，以便输入电路接收北斗系统对低空飞行器实时采集的第一坐标电压值和雷达系统对低空飞行器实时采集的第二坐标电压值，然后，比较电路根据第一坐标电压值和第二坐标电压值以及预设的电压值，输出第一控制信号或者第二控制信号，最后，输出电路根据第一控制信号导通第一输入端输出第一坐标电压值，或者，根据第二控制信号导通第二输入端输出第二坐标电压值。由此，能够通过该电路来对比北斗系统和雷达系统获得的坐标电压值，校正误差，以获得准确的唯一的坐标电压值。
