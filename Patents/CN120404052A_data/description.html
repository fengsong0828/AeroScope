Description

一种低空飞行器风洞内矩阵式风墙及其控制系统
技术领域
本发明涉及低空飞行器实验领域，特别涉及一种低空飞行器风洞内矩阵式风墙及其控制系统。
背景技术
对于飞行器(飞机、弹体等)特种风洞试验，比如自由飞、尾旋等实验，模型的质量特性(包括模型的重量、重心和转动惯量)要求严格。因为材料、制造及装配环节不可避免的精度损失，使完成模型的质量特性存在一定误差。当此误差大于设计要求时，需要对模型的质量特性进行微量调节。
现有技术是当初步实验完成后，会根据实验需求在飞行器上增加部件，但是后续增加部件的实验，还需要重新生产新的模型或者现场安装部件才能进行实验，操作时间较高，影响实验效率。
因此，提出一种低空飞行器风洞内矩阵式风墙及其控制系统来解决上述问题很有必要。
发明内容
本发明的目的在于提供一种低空飞行器风洞内矩阵式风墙及其控制系统，以解决现有技术是当初步实验完成后，会根据实验需求在飞行器上增加部件，但是后续增加部件的实验，还需要重新生产新的模型或者现场安装部件才能进行实验，操作时间较高，影响实验效率的问题。
为实现上述目的，本发明提供如下技术方案：一种低空飞行器，包括：
风洞飞行模型，所述风洞飞行模型设置于风洞内，且风洞飞行模型分为固定姿态和飞行姿态；
微调机构；所述微调机构用于对固定姿态的风洞飞行模型进行支撑和注入增重胶；
承接机构，所述承接机构至少设置有四个，四个承接机构环绕风洞飞行模型设置；
所述风洞飞行模型和对应的承接机构之间连接有轻质连接绳；
所述轻质连接绳包括绳体，所述绳体外侧包裹设置有气囊，所述轻质连接绳外侧设置有磁片。
其中，承接机构用于对轻质连接绳进行放出或收卷。
优选的，所述风洞飞行模型底端连接有两个支撑腿，所述支撑腿内部开设有空腔；
所述支撑腿两侧均开设有供对应的轻质连接绳穿过连接的连接孔。
优选的，所述微调机构包括底盘，所述底盘顶端固定有支撑柱，所述支撑柱的顶端固定连接有固定盘；
所述固定盘顶端固定有两个磁吸插块，两个支撑腿底端开设有供对应磁吸插块插入的插槽，所述插槽和对应的空腔连通。
优选的，所述固定盘的底端固定连接有存储盒和物料泵，所述存储盒和物料泵连接，所述物料泵通过管道和空腔连接，且管道上设置有计量阀；
其中，磁吸插块顶端开设有排料口，磁吸插块底端开设有进料口，所述进料口和排料口连通，进料口和物料泵连接。
优选的，所述底盘顶端开设有环形滑槽，四个承接机构底端滑动配合于环形滑槽内；
其中，承接机构包括移动座，所述移动座顶端连接电动推杆，所述电动推杆顶端连接有外壳，所述外壳内部转动连接有卷绕轴，所述轻质连接绳缠绕于卷绕轴外侧。
优选的，所述外壳内部固定有卷绕电机，所述卷绕电机的驱动轴和卷绕轴连接；
所述外壳上开设有供轻质连接绳进入的通口，所述通口进入端连接有锥形套，所述外壳内部固定有支腿，所述支腿顶端固定有支撑套，所述支撑套内部固定有清理海绵套，所述轻质连接绳穿过清理海绵套内侧。
优选的，所述底盘顶端开设有环形风槽，且环形风槽设置于环形滑槽外侧。
本发明还公开了一种低空飞行器风洞内矩阵式风墙，包括风洞，所述风洞由进口段、收缩段、实验段、扩散段、动力段和过渡段组成；
且进口段、收缩段、实验段、扩散段、动力段和过渡段相互贯通形成风力通道；
其中，动力段内部设置有多个动力框架，动力框架内固定设置有风机，所述过渡段内部固定有多个单元格，多个单元格照矩阵式排列，多个单元格内部开设有圆槽，所述圆槽和风机连通，用于引流风机的气流；
所述实验段顶端设置有可拆卸天窗，所述实验段两侧均设置有观察窗，且观察窗一侧设置有密封门。
优选的，所述进口段、收缩段、实验段、扩散段、动力段和过渡段底端均固定连接有固定腿。
本发明公开了一种低空飞行器风洞内矩阵式风墙控制系统，包括：
数据采集模块，用于采集风洞内的相关参数；
上位机；
若干下位机，所述下位机为工业可编程逻辑控制器plc，所述下位机通过tcp/ip标准通信协议与所述上位机进行数据传输；
通信模块，用于对数据采集模块采集到的数据进行通讯，并分别将采集到的数据传输至若干不同的下位机；
变频器，用于驱动所述风机，通过改变电源频率和电压调节风机中的风场；
i/o模块，分别连接所述上位机和所述下位机，用于实现数据的输入和输出，以及控制信号的传输和处理。
本发明的技术效果和优点：
1、在本发明的实际操作中，当风洞飞行模型处于固定姿态在风洞内进行气流模拟实验时，人员可以操作风洞飞行模型下降到固定盘上时，风洞飞行模型的插槽会和磁吸插块进行对接，通过磁吸的连接可以对风洞飞行模型进行固定，使风洞飞行模型呈固定姿态进行实验，提高实验的效率。
2、通过微调机构往风洞飞行模型内部注入快速固化型环氧树脂，达到增加风洞飞行模型的重量的目的，从而代替增加部件的重量，以此满足快速实验对比的需要，计量阀可以监测树脂的输出量，通过精确控制增重胶的注入量，能够实现对风洞飞行模型重量的微调或大量调节，满足高精度实验要求。
3、风洞飞行模型底端四角会通过轻质连接绳和对应的承接机构进行连接，且当风洞飞行模型处于飞行姿态时，轻质连接绳和承接机构之间呈放松状态连接，使风洞飞行模型在一定范围内飞行时不会受到轻质连接绳的影响，若风洞飞行模型在气流实验中产生故障掉落时，四个轻质连接绳可以对风洞飞行模型进行拉拽，避免直接和地面撞击，起到保护作用。
4、另外当风洞飞行模型处于固定姿态时，轻质连接绳和承接机构之间成拉紧状态连接，可以提高风洞飞行模型在固定姿态时的连接强度，避免气流影响，使风洞飞行模型产生倾倒和移动。
5、四个承接机构可以在环形滑槽进行滑动，带动承接机构进行移动，四个承接机构可以根据需要呈不同角度在风洞飞行模型外侧进行分布，达到不同的实验目的。
6、气流从环形风槽排出，并且向上吹出，形成一个环形的保护风墙，从而和风洞内的气流进行抵消，使保护风墙内部可以形成一个无气流环境，风洞飞行模型可以在保护风墙内部进行上升和下降，便于人员进行注胶或者对风洞飞行模型进行固定，避免重复开启和关闭风洞内风机产生的能量损耗。
7、通过在绳体外侧设置有气囊，当轻质连接绳随着产生的气流飘浮时，设置的气囊可以提高轻质连接绳整体的轻盈度，受气流影响更容易进行飘起，减少自身重量对风洞飞行模型的实验影响，同时通过增加气囊，可以对绳体外侧进行保护，减少气流的腐蚀和影响，并且可以提高对风洞飞行模型的保护，避免绳体直接和风洞飞行模型产生碰撞和摩擦，影响风洞飞行模型的实验数据，或减少对风洞飞行模型产生损伤。
8、两个相邻的轻质连接绳上的磁片相斥设置，可以避免气流使轻质连接绳之间靠近，造成轻质连接绳相互缠绕的现象，提高风洞飞行模型使用的安全性。

具体实施方式
本发明提供了如图1～图8所示的一种低空飞行器，包括：风洞飞行模型7，风洞飞行模型7底端连接有两个支撑腿13，支撑腿13内部开设有空腔。
如图1所示，风洞飞行模型7设置于风洞内，且风洞飞行模型7分为固定姿态和飞行姿态；
如图6所示，本发明公开了一种低空飞行器风洞内矩阵式风墙，包括风洞，风洞由进口段24、收缩段25、实验段26、扩散段27、动力段28和过渡段29组成，进口段24、收缩段25、实验段26、扩散段27、动力段28和过渡段29底端均固定连接有固定腿，且进口段24、收缩段25、实验段26、扩散段27、动力段28和过渡段29相互贯通形成风力通道。
风洞安装场地要求：风洞主体及配套设施的总长度限制不超过60m，宽度限制不超过24m，高度限制不超过16m。地面承重要求为不超过2吨/m2。根据安放场地地面的要求。对风洞不同部件采用了不同的支撑方式，尤其是对动力段28重量分布较大的段，进行了多固定腿加地面整体钢板布局，分散洞体的重量。
进口段24作为风洞的起始部分，其主要作用是将外界的气流引入风洞系统。进口段24的形状通常为喇叭口状，能够使气流平稳地进入风洞，减少气流的湍流和涡流，从而提高风洞的气流质量。
进口段24前端设置有进气唇口，采用半径为1.2m的圆弧结构进口尺寸为14.4m*12.4m，出口为11.5m*10m，长度为1.2m。
进口段24截面为矩形结构，板材为6mm厚干板拼接焊接而成。
外部设置3圈尺寸工字钢的强钢结构，交接处焊有加强板。辅助钢结构采用100mm*6mm的结构扁钢制成，间距约纵向600mm，横向1000mm。
收缩段25位于进口段24之后，其主要作用是将气流加速并使其更加稳定。收缩段25的截面积逐渐减小，根据伯努利方程，气流在通过收缩段25时速度会增加，压力会降低。
而风洞飞行模型7主要集中于实验段26进行飞行实验，试验段26两侧洞壁扩散弥补边界层增厚产生水平浮力问题，单边扩散角度为0.25度，试验段26截面尺寸为4.5m*3.5m，由10mm厚钢板铺设而成，外表设有主结构钢筋，副结构部件采用100mmm*8mm扁钢组成。
扩散段27入口为4.68m*3.5m，出口为8m*6m，总长度17m采用主梁结构，辅助加强肋结构。
其中，动力段28内部设置有多个动力框架，动力框架内固定设置有风机，过渡段29内部固定有多个单元格30，多个单元格30照矩阵式排列，多个单元格30内部开设有圆槽，圆槽和风机连通，用于引流风机的气流；
实验段26顶端设置有可拆卸天窗23，实验段26两侧均设置有观察窗，且观察窗一侧设置有密封门，且实验段26是风洞的核心部分，也是低空飞行器风洞飞行模型7进行飞行实验的主要场所。实验段26的截面积相对较小且保持恒定，以保证气流在该段内的速度和方向相对稳定。
如图1-4所示，设置有微调机构；微调机构用于对固定姿态的风洞飞行模型7进行支撑和注入增重胶，便于风洞飞行模型7直接在实验段26内进行重量的微调，提高实验的效率。
具体的，微调机构包括底盘1，底盘1顶端固定有支撑柱8，支撑柱8的顶端固定连接有固定盘11，底盘1的尺寸为1.2米×1.2米，采用高强度铝合金材料，厚度为10毫米，能够承受最大载荷1000千克，确保整个微调机构的稳定性和可靠性。
固定盘11顶端固定有两个磁吸插块12，两个支撑腿13底端开设有供对应磁吸插块12插入的插槽，插槽和对应的空腔连通。
固定盘11的底端固定连接有存储盒10和物料泵9，存储盒10和物料泵9连接，物料泵9通过管道和空腔连接，且管道上设置有计量阀，其中，磁吸插块12顶端开设有排料口15，磁吸插块12底端开设有进料口，所述进料口和排料口15连通，进料口和物料泵9连接。
在本发明的实际操作中，当风洞飞行模型7处于固定姿态在风洞内进行气流模拟实验时，人员可以操作风洞飞行模型7下降到固定盘11上时，风洞飞行模型7的插槽会和磁吸插块12进行对接，通过磁吸的连接可以对风洞飞行模型7进行固定，使风洞飞行模型7呈固定姿态进行实验，提高实验的效率。
进一步的，若风洞飞行模型7完成初始实验后，还需要考虑到后续风洞飞行模型7上增加部件的实验时，可以直接通过微调机构往风洞飞行模型7内部注入快速固化型环氧树脂，达到增加风洞飞行模型7的重量的目的，从而代替增加部件的重量，以此满足快速实验对比的需要，计量阀可以监测树脂的输出量，通过精确控制增重胶的注入量，能够实现对风洞飞行模型7重量的微调或大量调节，满足高精度实验要求。
快速固化型环氧树脂使用惠柏AM-8931A/B快速固化成型环氧树脂。
具体的，快速固化型环氧树脂可以通过物料泵9的抽吸，从排料口15输入到空腔内部。
如图1所示，设置承接机构5，承接机构5至少设置有四个，四个承接机构5环绕风洞飞行模型7设置，风洞飞行模型7和承接机构5之间连接有轻质连接绳6，其中，承接机构5用于对轻质连接绳6进行放出或收卷。
风洞飞行模型7和承接机构5的具体连接方式：
如图2所示，风洞飞行模型7上预开设有连接孔，轻质连接绳6的一端穿过连接孔，且在穿过处打结设置。
如图8所示，轻质连接绳6的一端连接有连接套33，所述连接套33套设于支撑腿13上，并且连接套33有两个半圆套相对连接而成，两个半圆套可以使用卡接或者螺栓固定。
风洞飞行模型7底端四角会通过轻质连接绳6和对应的承接机构5进行连接，且当风洞飞行模型7处于飞行姿态时，轻质连接绳6和承接机构5之间呈放松状态连接，使风洞飞行模型7在一定范围内飞行时不会受到轻质连接绳6的影响，若风洞飞行模型7在气流实验中产生故障掉落时，四个轻质连接绳6可以对风洞飞行模型7进行拉拽，避免直接和地面撞击，起到保护作用。
且由于风洞内的风机启动，产生的气流会使轻质连接绳6漂浮，并且风洞飞行模型7开孔时会损失一定的重量，通过气流的漂浮作用和损失的重量可以抵消风洞飞行模型7承受轻质连接绳6的重量，避免影响实验数据。
另外当风洞飞行模型7处于固定姿态时，轻质连接绳6和承接机构5之间成拉紧状态连接，可以提高风洞飞行模型7在固定姿态时的连接强度，避免气流影响，使风洞飞行模型7产生倾倒和移动。
轻质连接绳6包括绳体31，绳体31外侧包裹设置有气囊31，在本发明的实际操作中，通过在绳体31外侧设置有气囊31，当轻质连接绳6随着产生的气流飘浮时，设置的气囊31可以提高轻质连接绳6整体的轻盈度，受气流影响更容易进行飘起，减少自身重量对风洞飞行模型7的实验影响，同时通过增加气囊31，可以对绳体31外侧进行保护，减少气流的腐蚀和影响，并且可以提高对风洞飞行模型7的保护，避免绳体31直接和风洞飞行模型7产生碰撞和摩擦，影响风洞飞行模型7的数据，或减少对风洞飞行模型7产生损伤。
轻质连接绳6外侧设置有磁片34，磁片34设置多个，多个磁片34沿着轻质连接绳6长度方向等距离分布，两个相邻的轻质连接绳6上的磁片34相斥设置，可以避免气流使轻质连接绳6之间靠近，造成轻质连接绳6相互缠绕的现象，提高使用的安全性。
外壳19内部固定有卷绕电机16，卷绕电机16的驱动轴和卷绕轴18连接，通过卷绕电机16的启动，可以实现轻质连接绳6的放出和收卷，从而使轻质连接绳6和风洞飞行模型7之间形成放松和收紧状态，并且根据需要，可以放出较长的轻质连接绳6，避免影响风洞飞行模型7飞行较高空间。
同时，需要注意的是，承接机构5高度低于风洞飞行模型7飞行高度，避免承接机构5遮挡风洞飞行模型7前方的气流。
支撑腿13两侧均开设有供对应的轻质连接绳6穿过连接的连接孔，底盘1顶端开设有环形滑槽3，四个承接机构5底端滑动配合于环形滑槽3内；其中，承接机构5包括移动座4，移动座4顶端连接电动推杆14，电动推杆14顶端连接有外壳19，外壳19内部转动连接有卷绕轴18，轻质连接绳6缠绕于卷绕轴18外侧。
在本发明的实际操作中，四个承接机构5可以在环形滑槽3进行滑动，自动带动承接机构5进行移动，四个承接机构5可以根据需要呈不同角度在风洞飞行模型7外侧进行分布，并且可以自动调节高度，且四个承接机构5之间的间隔也可以进行调节，或四个承接机构5集中在一起进行对风洞飞行模型7进行连接，(例如风筝式连接)达到不同的实验目的。
轻质连接绳6可以采用超高分子量聚乙烯纤维材质。
外壳19上开设有供轻质连接绳6进入的通口，通口进入端连接有锥形套17，外壳19内部固定有支腿20，支腿20顶端固定有支撑套21，支撑套21内部固定有清理海绵套22，轻质连接绳6穿过清理海绵套22内侧。
在本发明的实际操作中，当轻质连接绳6被卷绕轴18收卷到外壳19内部后，轻质连接绳6会先通过锥形套17，锥形套17会将轻质连接绳6外侧的杂质刮除，并且随着轻质连接绳6的收卷，支撑套21可以对收卷到外壳19内部的轻质连接绳6进行支撑，且由于轻质连接绳6收卷移动会和清理海绵套22内侧进行摩擦，从而起到清理轻质连接绳6的作用。
底盘1顶端开设有环形风槽2，且环形风槽2设置于环形滑槽3外侧。
底盘1底端设置有气泵或者向上吹风的风机，通过启动风机或气泵，气流从环形风槽2排出，并且向上吹出，形成一个环形的保护风墙，保护风墙内部可以形成一个无气流环境，从而和风洞内的气流进行抵消，使风洞飞行模型7可以在保护风墙内部进行上升和下降，便于人员进行注胶或者对风洞飞行模型7进行固定。
重复关闭和开启风洞内风机会使设备的机械损耗增加，降低设备的使用寿命，同时也增加了维护成本。而一直开启风机虽然也会产生一定的能耗，但设备的运行相对稳定，使用寿命和维护成本相对较低。
本发明公开了一种低空飞行器风洞内矩阵式风墙控制系统，包括：
数据采集模块，用于采集风洞内的相关参数；
数据采集模块包括风向传感器、风速传感器、温度传感器和湿度传感器以及气压传感器，采集风洞内部数据。
上位机，对数据采集模块采集到的数据进行分析和处理，生成实时的风场模型，根据实验需求，编辑和设置不同的实验任务，包括风速、风向、温度、湿度等参数的设定，对实验数据进行分析，生成实验报告，为低空飞行器的设计和优化提供参考。
若干下位机，所述下位机为工业可编程逻辑控制器plc，所述下位机通过tcp/ip标准通信协议与所述上位机进行数据传输；根据上位机的指令，控制风机的启停、转速调节，以及环境调节系统的运行，将采集到的数据实时传输至上位机，同时接收上位机的控制指令，实时监测设备的运行状态，及时发现并处理故障，确保系统的稳定运行。
通信模块，用于对数据采集模块采集到的数据进行通讯，并分别将采集到的数据传输至若干不同的下位机，将数据采集模块采集到的数据实时传输至下位机，同时将下位机的数据传输至上位机，上位机的控制指令传输至下位机，确保系统的实时控制，通信模块的传输速率通常在100Mbps以上，确保数据传输的实时性和可靠性。
变频器，用于驱动所述风机，通过改变电源频率和电压调节风机中的风场；
i/o模块，分别连接所述上位机和所述下位机，用于实现数据的输入和输出，以及控制信号的传输和处理，接收来自传感器的模拟信号和数字信号，将其转换为适合上位机处理的数据格式，将上位机的控制指令转换为适合下位机执行的信号，实现对设备的精确控制，对输入和输出的信号进行处理，如滤波、放大等，确保信号的稳定性和可靠性。
