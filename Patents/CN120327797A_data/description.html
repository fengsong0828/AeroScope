Description

低空垂直起降飞行器控制方法、系统、设备以及存储介质
技术领域
本发明涉及飞行器技术领域，尤其涉及一种低空垂直起降飞行器控制方法、系统、设备以及存储介质。
背景技术
在全球科技飞速发展的浪潮下，低空经济作为新兴的经济形态进入快速培育阶段。低空垂直起降飞行器作为低空经济的核心载体之一，正凭借其具有垂直起降、运行成本低和噪音小等优点，广泛应用于城际交通、旅游观光和医疗运输等领域。
低空垂直起降飞行器的动力系统是其核心组成部分，目前主要有以下几种类型：纯电动动力系统、混合动力系统、燃油动力系统和氢能动力系统。为了推进低空垂直起降飞行器的低碳化发展，如何提高低空垂直起降飞行器的运行经济性和运行寿命成为重要的研究方向。
发明内容
本发明提供了一种低空垂直起降飞行器控制方法、系统、设备以及存储介质，以提高低空垂直起降飞行器的运行经济性和运行寿命。
根据本发明的一方面，提供了一种低空垂直起降飞行器控制方法，该方法借助低空垂直起降飞行器控制系统来实现，其中，低空垂直起降飞行器控制系统包括飞行器控制器、锂电池系统控制器、燃料电池系统控制器、燃料电池系统、锂电池系统、储氢系统、DCDC转换器、驱动电机和飞行器推进系统。该方法包括：
飞行器控制器根据获取到的飞行器控制指令，确定目标电池系统控制器，并向目标电池系统控制器下发功率需求指令；
目标电池系统控制器根据功率需求指令，控制目标电池系统工作，得到电池输出电能；
通过DCDC转换器对电池输出电能进行调节，得到稳定电能，并将稳定电能发送给驱动电机；
驱动电机根据稳定电能，控制飞行器推进系统运行，以实现对低空垂直起降飞行器的控制。
根据本发明的另一方面，提供了一种低空垂直起降飞行器控制系统，该系统包括：飞行器控制器、锂电池系统控制器、燃料电池系统控制器、燃料电池系统、锂电池系统、储氢系统、DCDC转换器、驱动电机和飞行器推进系统；其中，
飞行器控制器，用于获取飞行器控制指令，并根据获取到的飞行器控制指令，确定目标电池系统控制器；向目标电池系统控制器下发功率需求指令；
锂电池系统控制器，用于控制锂电池系统工作，得到锂电池输出电能；
燃料电池系统控制器，用于控制燃料电池系统工作，得到燃料电池输出电能；
储氢系统，用于为燃料电池系统提供氢气燃料；
DCDC转换器，用于对锂电池系统或燃料电池系统输出的电能进行调节，得到稳定电能，并将稳定电能发送给驱动电机；
驱动电机，用于接收稳定电能，并根据稳定电能，控制飞行器推进系统运行，以实现对低空垂直起降飞行器的控制。

根据本发明的另一方面，提供了计算机程序产品，包括计算机程序，计算机程序在被处理器执行时实现本发明任一实施例的低空垂直起降飞行器控制方法。
本发明实施例的技术方案，借助低空垂直起降飞行器控制系统来实现，即，飞行器控制器根据获取到的飞行器控制指令，确定目标电池系统控制器，并向目标电池系统控制器下发功率需求指令；目标电池系统控制器根据功率需求指令，控制目标电池系统工作，得到电池输出电能；通过DCDC转换器对电池输出电能进行调节，得到稳定电能，并将稳定电能发送给驱动电机；驱动电机根据稳定电能，控制飞行器推进系统运行，以实现对低空垂直起降飞行器的控制。相比于借助传统氢电混合驱动系统实现的低空垂直起降飞行器控制方法，即，在低空垂直起降飞行器起飞、巡航和降落的整个过程中，均是由锂电池系统和燃料电池系统同时作为动力源，锂电池系统按照低空垂直起降飞行器起飞、巡航或降落时所需的功率充放电，燃料电池系统按照锂电池系统中锂电池的充电功率需求，输出以一系列变载功率，给锂电池系统补充电能，上述技术方案，使得低空垂直起降飞行器在起降阶段，虽然仍然将锂电池系统和燃料电池系统同时作为动力源，为驱动电机提供电能，但是燃料电池系统无需给锂电池系统中的锂电池充电；使得低空垂直起降飞行器在巡航阶段，仅由燃料电池系统以固定功率为驱动电机单独供电，锂电池系统不参与工作，且燃料电池系统不给锂电池系统补充电能，即无需根据锂电池系统中锂电池的充电功率需求输出一系列变载功率，给锂电池系统中的锂电池充电。且在低空垂直起降飞行器起飞、巡航和降落的整个过程中，使得低空垂直起降飞行器控制系统中的燃料电池系统始终以固定功率为驱动电机提供电力，降低了变载功率对燃料电池膜电极的损害，提高了燃料电池系统中燃料电池的使用寿命；且燃料电池系统无需给锂电池系统中的锂电池充电，从而减少了燃料电池系统给锂电池系统中的锂电池充电的两次转化损耗，降低了锂电池系统中锂电池频繁充放电对锂电池运行寿命的影响，降低了燃料电池系统频繁更换工作电压对燃料电池运行寿命的影响，进而提高了低空垂直起降飞行器的运行寿命和运行经济性。

S101、飞行器控制器根据获取到的飞行器控制指令，确定目标电池系统控制器，并向目标电池系统控制器下发功率需求指令。
其中，飞行器控制指令是指用于控制低空垂直起降飞行器运动的指令；可选的，飞行器控制指令可以是起飞指令、巡航指令或降落指令。目标电池系统控制器是指低空垂直起降飞行器控制系统中，执行飞行器控制指令所需的电池系统控制器；可选的，目标电池系统控制器可以包括锂电池系统控制器和燃料电池系统控制器，也可以仅仅是燃料电池系统控制器。功率需求指令用于指示执行飞行器控制指令所需的功率。
具体的，飞行器控制器可以从低空垂直起降飞行器的控制面板中获取操作员输入的飞行器控制指令；若检测到获取到的飞行器控制指令为起飞指令或降落指令，则飞行器控制器确定目标电池系统控制器包括锂电池系统控制器和燃料电池系统控制器，并向锂电池系统控制器下发第一功率需求指令，向燃料电池系统控制器下发第二功率需求指令；若检测到获取到的飞行器控制指令为巡航指令，则飞行器控制器确定目标电池系统控制器为燃料电池系统控制器，并向燃料电池系统控制器下发燃料电池功率需求指令。
其中，第一功率需求指令用于指示执行起飞指令或降落指令时，锂电池系统所需的功率；可选的，第一功率需求指令中的第一需求功率根据低空垂直起降飞行器的飞行器设计总重、第一机型设计参数和第二机型设计参数确定。
其中，第一需求功率是指执行起飞指令或降落指令时，锂电池系统所需的功率。飞行器设计总重是指设计阶段确定的低空垂直起降飞行器的总重量；第一机型设计参数和第二机型设计参数均是设计低空垂直起降飞行器的机型时所需的固定参数。需要说明的是，第一机型设计参数与第二机型设计参数的取值不同；第一机型设计参数的取值一般为15kg/kW，即每千瓦功率对应的飞行器重量为15千克；第二机型设计参数的取值一般为5kg/kW，即每千瓦功率对应的飞行器重量为5千克。

S102、目标电池系统控制器根据功率需求指令，控制目标电池系统工作，得到电池输出电能。
其中，目标电池系统是指低空垂直起降飞行器控制系统中由目标电池系统控制器控制的电池系统。电池输出电能是指由低空垂直起降飞行器控制系统中的电池产生的能量。需要说明的是，若目标电池系统控制器包括锂电池系统控制器和燃料电池系统控制器，则电池输出电能包括第一输出电能和第二输出电能；若目标电池系统控制器为燃料电池系统控制器，则电池输出电能为燃料电池系统输出的电能。其中，第一输出电能是指低空垂直起降飞行器起飞阶段或降落阶段，锂电池系统输出的电能；第二输出电能是指低空垂直起降飞行器起飞阶段或降落阶段，燃料电池系统输出的电能。
需要说明的是，若目标电池系统控制器包括锂电池系统控制器和燃料电池系统控制器，则说明飞行器控制指令为起飞指令或降落指令，此时的功率需求指令包括第一功率需求指令和第二功率需求指令；若目标电池系统控制器为燃料电池系统控制器，则说明飞行器控制指令为巡航指令，此时的功率需求指令为燃料电池功率需求指令。
具体的，若目标电池系统控制器包括锂电池系统控制器和燃料电池系统控制器，则锂电池系统控制器根据第一功率需求指令，控制锂电池系统工作，得到第一输出电能；燃料电池系统控制器根据第二功率需求指令，控制燃料电池系统工作，得到第二输出电能；若目标电池系统控制器为燃料电池系统控制器，则燃料电池系统控制器根据燃料电池功率需求指令，控制燃料电池系统工作，得到电池输出电能。
还需要说明的是，在低空垂直起降飞行器起降阶段，锂电池系统参与放电，以补充燃料电池系统对飞行器起降功率供给的不足；且在低空垂直起降飞行器起降阶段，将锂电池系统和燃料电池系统同时作为动力源，为驱动电机提供电能，但燃料电池系统无需给锂电池系统中的锂电池充电，从而减少了燃料电池系统给锂电池系统中的锂电池充电的两次转化损耗，降低了锂电池系统中锂电池频繁充放电对锂电池运行寿命的影响，降低了燃料电池系统频繁更换工作电压对燃料电池运行寿命的影响，进而提高了低空垂直起降飞行器的运行寿命。
总而言之，当飞行器控制器获取到起飞指令时，即当低空垂直起降飞行器起飞时，飞行器控制器向锂电池系统控制器下发第一功率需求指令，向燃料电池系统控制器下发第二功率需求指令；锂电池系统控制器根据第一功率需求指令，控制锂电池系统以第一需求功率发电，得到第一输出电能；燃料电池系统根据第二功率需求指令，控制燃料电池系统以第二需求功率发电，得到第二输出电能。也就是说，当低空垂直起降飞行器起飞时，由锂电池系统和燃料电池系统共同为驱动电机提供电能。
当飞行器控制器获取到巡航指令时，即当低空垂直起降飞行器巡航时，飞行器控制器向燃料电池系统控制器下发燃料电池功率需求指令；燃料电池系统根据燃料电池功率需求指令，控制燃料电池系统以燃料电池需求功率发电，得到电池输出电能。也就是说，当低空垂直起降飞行器巡航时，由燃料电池系统单独供电，锂电池系统不参与工作，且燃料电池系统不给锂电池系统补充电能。
当飞行器控制器获取到降落指令时，即当低空垂直起降飞行器降落时，飞行器控制器向锂电池系统控制器下发第一功率需求指令，向燃料电池系统控制器下发第二功率需求指令；锂电池系统控制器根据第一功率需求指令，控制锂电池系统以第一需求功率发电，得到第一输出电能；燃料电池系统根据第二功率需求指令，控制燃料电池系统以第二需求功率发电，得到第二输出电能。由于燃料电池需求功率等于第二需求功率，因而当低空垂直起降飞行器降落时，虽然由锂电池系统和燃料电池系统共同为驱动电机提供电能，但是燃料电池系统以固定功率(即飞行器推进功率)持续运行，只是锂电池系统再次进入工作状态，以第一需求功率发电，直至低空垂直起降飞行器降落。在低空垂直起降飞行器降落后，对锂电池系统中的锂电池和储氢系统中的氢气进行检测，为锂电池系统中的锂电池充电，或更换锂电池系统中的锂电池，为储氢系统加氢或换氢。
S103、通过DCDC转换器对电池输出电能进行调节，得到稳定电能，并将稳定电能发送给驱动电机。
其中，DCDC转换器(直流-直流转换器，DC-to-DC converter)是电能转换的电路或是机电设备，可以将直流电源转换为不同电压的直流(或近似直流)电源。稳定电能是指使驱动电机运行所需的电能。
具体的，若飞行器控制指令为起飞指令或降落指令，则DCDC转换器对锂电池系统输出的第一输出电能和燃料电池系统输出的第二输出电能进行整合和调节，得到稳定电能；并将稳定电能发送给驱动电机。若飞行器控制指令为巡航指令，则DCDC转换器对燃料电池系统输出的电能进行调节，得到稳定电能；并将稳定电能发送给驱动电机。
S104、驱动电机根据稳定电能，控制飞行器推进系统运行，以实现对低空垂直起降飞行器的控制。
具体的，驱动电机根据接收到的稳定电能，控制飞行器推进系统运行，实现对低空垂直起降飞行器起飞、降落和巡航的控制。
本发明实施例的技术方案，借助低空垂直起降飞行器控制系统来实现，即，飞行器控制器根据获取到的飞行器控制指令，确定目标电池系统控制器，并向目标电池系统控制器下发功率需求指令；目标电池系统控制器根据功率需求指令，控制目标电池系统工作，得到电池输出电能；通过DCDC转换器对电池输出电能进行调节，得到稳定电能，并将稳定电能发送给驱动电机；驱动电机根据稳定电能，控制飞行器推进系统运行，实现对低空垂直起降飞行器的控制。相比于借助传统氢电混合驱动系统实现的低空垂直起降飞行器控制方法，即，在低空垂直起降飞行器起飞、巡航和降落的整个过程中，均是由锂电池系统和燃料电池系统同时作为动力源，锂电池系统按照低空垂直起降飞行器起飞、巡航或降落时所需的功率充放电，燃料电池系统按照锂电池系统中锂电池的充电功率需求，输出以一系列变载功率，给锂电池系统补充电能，上述技术方案，使得低空垂直起降飞行器在起降阶段，虽然仍然将锂电池系统和燃料电池系统同时作为动力源，为驱动电机提供电能，但是燃料电池系统无需给锂电池系统中的锂电池充电；使得低空垂直起降飞行器在巡航阶段，仅由燃料电池系统以固定功率为驱动电机单独供电，锂电池系统不参与工作，且燃料电池系统不给锂电池系统补充电能，即无需根据锂电池系统中锂电池的充电功率需求输出一系列变载功率，给锂电池系统中的锂电池充电。且在低空垂直起降飞行器起飞、巡航和降落的整个过程中，使得低空垂直起降飞行器控制系统中的燃料电池系统始终以固定功率为驱动电机提供电力，降低了变载功率对燃料电池膜电极的损害，提高了燃料电池系统中燃料电池的使用寿命；且燃料电池系统无需给锂电池系统中的锂电池充电，从而减少了燃料电池系统给锂电池系统中的锂电池充电的两次转化损耗，降低了锂电池系统中锂电池频繁充放电对锂电池运行寿命的影响，降低了燃料电池系统频繁更换工作电压对燃料电池运行寿命的影响，进而提高了低空垂直起降飞行器的运行寿命和运行经济性。

对于纯锂电池驱动系统，其锂电池容量为0.2Pq＝0.2×4＝0.8kWh，锂电池续航能耗为tPt＝1×1.3＝1.3kWh，锂电池总重量GL＝(0.2Pq+tPt)×1000/GL0＝(0.2×4+1×1.3)/500＝4.2kg，则基于上述锂电池的单位容量成本CL0＝500元/kWh，单位用电量成本CL1＝1元/kWh，得到纯锂电池驱动系统中锂电池的初装成本CLc＝(0.2Pq+tPt)×CL0＝(0.2×4+1×1.3)×500＝1050元，使用成本CLu＝(0.2Pq+tPt)×CL1＝(0.2×4+1×1.3)×1＝2.1元。由此可知，纯锂电池驱动系统的单位TCO成本的有效载重：
λL＝(Gz-GL)/(CLc+CLu)×10000＝(9-4.2)/(1050+2.1)×10000＝45.6kg/万元
对于纯燃料电池驱动系统，其燃料电池的功率需满足低空垂直起降飞行器的飞行器推进功率和飞行器起降功率需求，即纯燃料电池驱动系统中燃料电池的整机功率按照飞行器推进功率和飞行器起降功率中较大的飞行器起降功率配置，基于上述燃料电池标准质量功率密度得到纯燃料电池驱动系统中燃料电池的整机重量Gf＝Pq/Gr0＝4/1＝4kg；低空垂直起降飞行器的氢气需求量GHu＝0.064tPt+0.2×0.064×Pq＝0.064×1×1.3+0.2×0.064×4≈0.13kg。纯燃料电池驱动系统中的氢气采用储氢装置携带，上述标准液态储氢质量密度GH0＝10％wt，得到储氢装置中氢气的总重量GHc＝GHu/GH0＝0.13/10％＝1.3kg；基于上述燃料电池的整机单位功率成本液态储氢系统单位质量成本单位质量用氢成本得到纯燃料电池驱动系统中燃料电池的整机初装成本为Cf＝Pq×Cr0＝4×370＝1480元，纯燃料电池驱动系统的储氢初装成本为：
CHc＝(t×0.064×Pt+0.2×0.064×Pq)×CH0
＝(1×0.064×1.3+0.2×0.064×4)×800＝107.52元；
纯燃料电池驱动系统的氢气使用成本为:
CHu＝(t×0.064×Pt+0.2×0.064×Pq)×CH1
＝(1×0.064×1.3+0.2×0.064×4)×25＝3.36元；
由此可知，纯燃料电池驱动系统的单位TCO成本的有效载重为：
λf＝(Gz-Gf-GHc-GHu)/(Cf+CHc+CHu)×10000
＝(9-4-1.3-0.13)/(1480+107.52+3.36)×10000＝22.44kg/万元
上述实验证明，λd>λL>λf，即在低空垂直起降飞行器的飞行器设计总重、载重、飞行器续航时间、第一机型设计参数和第二机型设计参数一致的情况下，本发明实施例的低空垂直起降飞行器控制系统200的单位TCO成本的有效载重高于纯锂电池驱动系统的单位TCO成本的有效载重，高于纯燃料电池驱动系统的单位TCO成本的有效载重，这说明借助本发明实施例的低空垂直起降飞行器控制系统200实现的低空垂直起降飞行器控制方法，在相同的运行成本下，低空垂直起降飞行器的运行效率更高，运行更加经济，从而提高了低空垂直起降飞行器的运行经济性。其中，TCO(总体拥有成本，Total Cost of Ownership)是一项帮助组织来考核、管理和削减在一定时间范围内组织获得某项资产的相关联的所有成本的技术。
