Description

一种低空小型无人飞行器的拦截装置
技术领域
本实用新型涉及机械领域，特别涉及一种低空小型无人飞行器的拦截装置。
背景技术
随着遥感技术、导航技术和电子技术的不断成熟，小型无人飞行器制作成本逐渐降低，其应用范围和规模迅速扩大。小型无人飞行器为工、农业生产或其他领域带来了便利，但也对公共安全带来一定的隐患。有关部门针对小型无人飞行器设置了禁飞区，并逐步出台了相关的管理办法，但总有飞行器失控或不明身份的飞行器侵入等事件发生，需要强制将小型无人飞行器捕捉或拦截。由于低空小型无人飞行器体积小、成本低、飞行路线灵活，不适合采用一般军用地对空拦截装置，特别是在人口密集地区更加不适合使用。
发明内容
本实用新型的目的在于克服现有技术中存在的缺点，提供一种成本低，使用安全的低空小型无人飞行器的拦截装置，主要用于拦截小型固定翼飞机和其他飞行高度低、速度较快的小型无人飞行器。
本实用新型的目的通过下述技术方案实现：
一种低空小型无人飞行器的拦截方法，包括下述步骤：
(1)通过高压气体推动，由地面装置向空中发射一种与空气接触可快速固化的液体；
(2)该液体在脱离地面装置初期由缓时分解的外层约束，在达到目标区域后外层分解，液体与空气高速摩擦分解成细小絮状，并瞬时固化成絮状云团；
(3)固化后的絮状云团悬浮空中，出现在飞行器螺旋桨附近或飞行器行进路线前方，并被飞行器螺旋桨气流吸进，从而有效的干扰飞行器螺旋桨正常工作，达到拦截目的。
一种低空小型无人飞行器的拦截装置，包括发射器和拦截器；发射器包括压气活塞1、压气气缸2、单向阀3、手柄4、高压气缸5、启动阀6、填弹阀门7、触发扳机8、发射管9、瞄准器10、连接筋11、拦截器12和发射口挡板17；拦截器包括外壳13、拦截物质14、密封罩15、外壳定位筋16和氧化脱落层18；压气活塞1的后端置于压气气缸2的外部，压气活塞1的前端与压气气缸2的内壁密封滑动结合；压气气缸2位于手柄4的后方内部，高压气缸5位于手柄4的前方内部，压气气缸2与高压气缸5之间通过单向阀3联通，空气由压气气缸2向高压气缸5流动时单向阀3导通，空气由高压气缸5向压气气缸2流动时单向阀3关闭；高压气缸5与发射管9之间通过启动阀6联通，启动阀6一般处于关闭状态；触发扳机8位于手柄4和发射管9连接处的下方，触发扳机8与启动阀6机械联动，用于控制启动阀6的导通；填弹阀门7位于手柄4和发射管9连接处的上方，填弹阀门7打开时用于在发射管9填装拦截器，填弹阀门7关闭时与发射管9保持气密；发射管9的左端与手柄4连接，发射管9的右端出口为发射口，发射管9的出口外部上方安装有瞄准器10；靠近发射管9的右端出口附近设置有连接筋11，将发射管9的右端出口分隔为左右两个部分；连接筋11的侧面开口，用于退出拦截器12的外壳13；在连接筋11前方的发射管9内部固定有一个环形的发射口挡板17，用于挡住外壳13，使其不飞出发射管9。
拦截器12的外壳13呈U型，外壳13包括圆形的底座和壳身，底座直径大于壳身直径且略小于发射管9内部圆形横截面直径，可以起到活塞套作用，外壳13采用铝制或其它低密度硬质材料。
外壳13的前端设置有4个位置对称的外壳定位筋16，当拦截器12在发射管9内向右端出口方向运动时，外壳定位筋16起到固定拦截器12姿态的作用；拦截物质14由氧化脱落层18包裹，置于外壳13和硬质密封罩15构成的空间内，硬质密封罩15与外壳13之间密封连接，用于在拦截器12发射前阻止空气中的氧气接触氧化脱落层18。
上述低空小型无人飞行器的拦截装置的使用方法，包括下述步骤：
(1)通过外力对压气活塞1往复做功，压气活塞1的内部有单向回气通道，压气活塞1进行往复运动时，压气气缸2内部的空气通过单向阀3向高压气缸5累积压缩空气，此时启动阀6闭合；
(2)当高压气缸5内部的压力足够大时，将拦截器12通过填弹阀门7装载到发射管9的内部，并关闭填弹阀门7；
(3)手持手柄4，通用瞄准器10确定发射管9的右端出口方向；
(4)压动触发扳机8，通过联动打开启动阀6，高压气缸5内部的高压气体压入发射管9的内部，推动拦截器12高速向发射管9的右端出口方向运动；
(5)当拦截器12高速运动到发射管9的右端出口时，首先，外壳定位筋16与发射口挡板17接触，并被压断，起到一级缓冲作用；然后，外壳13的底座与发射口挡板17接触，外壳13被挡在发射管9的内部并从连接筋11的侧面开口处滑出发射管9，拦截器12的内部物质(包括拦截物质14、氧化脱落层18和密封罩15)由于惯性继续高速运动，飞向目标飞行器并进行拦截。
拦截物质14是一种高分子树脂材料，溶解在乙醇或乙醚等有机挥发性溶解质中，呈液态；发射前，液态的拦截物质14由氧化脱落层18、密封罩15和外壳13约束；发射后的短时间内，拦截物质14由氧化脱落层18和密封罩15继续约束，并与外壳13脱离；发射一定时间后，拦截物质14外层的氧化脱落层18与空气中氧气快速反应并脱落，此时高速运动的液态拦截物质14与空气发生高速摩擦并分解成细小的絮状；絮状拦截物质14在空中迅速挥发掉溶解质，呈固态絮状，由于空气浮力，大片的固态絮状拦截物质14可悬浮在空中一定时间；由于小型无人飞行器主要依靠电机和螺旋桨驱动，如果固态絮状拦截物质14出现在飞行器螺旋桨附近或出现在飞行器行进路线前方，会被螺旋桨气流吸进，并有效的干扰飞行器螺旋桨正常工作，使其迫降。
本实用新型与现有技术相比具有以下优点和效果：
1、本实用新型的方法适合拦截低空快速飞行的小型无人飞行器。
2、本实用新型的装置制作成本低、使用安全。
