Description

一种基于AI和物联网的低空飞行器运行监测管理系统
技术领域
本发明涉及低空飞行器运行监测技术领域，具体涉及一种基于AI和物联网的低空飞行器运行监测管理系统。
背景技术
无人机是一种无人驾驶的低空飞行器，可以在城市或者山区等复杂环境中灵活飞行，目前无人机已经广泛应用于农业和物流配送等各个领域。
其中，在农业领域中，无人机可以通过搭载的传感器和摄像头，采集目标区域内的农作物图片，用于分析农作物生长情况和病虫害信息；通常无人机在执行任务时，确定飞行线路，然后根据飞行线路的距离和无人机的实时剩余电量选择出能够执行本次飞行线路的无人机。
但是，在低空域中，也会有鸟类飞行，无人机在飞行过程中，需要躲避鸟类，若鸟类较多，无人机风险较大，容易导致无人机坠毁，且在无人机躲避鸟类时，需要耗费更多的电量，因此当无人机进入鸟类较多的空域时，因躲避鸟类造成电量耗费较多，无法完成本次飞行线路。
发明内容
本发明的目的在于提供一种基于AI和物联网的低空飞行器运行监测管理系统，解决以上技术问题：
本发明的目的可以通过以下技术方案实现：
一种基于AI和物联网的低空飞行器运行监测管理系统，所述管理系统包括：
鸟类分布采集模块，用于采集目标区域低空域内鸟类活动信息数据；
划分模块，用于将目标区域划分为若干个监测区域单元；
飞行器监测模块，用于监测飞行器获取飞行器的实时数据；
飞行信息模块，用于获取飞行信息数据；
分析模块，用于根据各监测区域单元的鸟类活动轨迹确定各监测区域单元的安全评级，根据各监测区域单元的安全评级结果和飞行信息数据确定飞行路线和飞行线路危险指数；再根据飞行线路危险指数和飞行器的实时数据进行分析，判断能够执行本次飞行线路的飞行器。
作为本发明进一步的方案：所述飞行器的实时数据包括飞行器的任务状态、实时电量和电池状态系数；所述飞行信息数据包括出发地位置信息数据和目的地位置信息数据；所述鸟类活动信息数据包括鸟类数量；所述飞行线路包括从出发地到目的地的飞行线路和从目的地返回出发地的飞行线路。
