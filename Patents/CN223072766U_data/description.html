Description

一种方便拆装脚架的低空飞行器
技术领域
本实用新型涉及低空飞行器，具体地，涉及一种方便拆装脚架的低空飞行器。
背景技术
低空飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。随着飞行器普及，个人飞行器必然是未来发展方向，轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向，无人机属于低速飞行器的一种。
现有技术中，现有的低速飞行器在使用时，由于其底部的脚架主体经常与地面接触，长时间使用容易发生磨损，需要进行更换，脚架主体与机架主体一般存在整体的不可拆性，从而不便于对脚架进行拆装，同时也会给后期的维修带来困难，使用效果不好，现在急需一种方便拆装脚架的低空飞行器来解决上述出现的问题。
实用新型内容
针对现有技术存在的不足，本实用新型目的是提供一种方便拆装脚架的低空飞行器，以解决上述背景技术中提出的问题，本实用新型结构合理，拆装便捷，使用效果好。
为了实现上述目的，本实用新型是通过如下的技术方案来实现：一种方便拆装脚架的低空飞行器，包括：
机架主体，机架主体底部对称设置有一对脚架连接杆，一对脚架连接杆底部均设置有脚架主体；
脚架主体包括设置在脚架连接杆底部的衔接柱，衔接柱底部设置有支撑柱，衔接柱顶部设置有插入脚架连接杆内部的连接柱，连接柱侧壁贯通开设有插孔；
连接柱与脚架连接杆之间设置有锁紧组件，锁紧组件包括贯穿脚架连接杆且插入插孔内部的螺纹锁紧杆，螺纹锁紧杆上设置有锁紧螺母。
进一步地，锁紧螺母侧壁设置有施力柱。
进一步地，螺纹锁紧杆包括设置在脚架连接杆侧壁的限位盘，限位盘端部设置有贯穿脚架连接杆侧壁且插入插孔内部的插柱，插柱末端侧壁设置有外螺纹，锁紧螺母与外螺纹相匹配。
进一步地，脚架连接杆底部开设有与连接柱相匹配的连接孔，衔接柱顶部对称设置有置于连接柱两侧且置于插孔下方的定位柱，脚架连接杆底部与定位柱相对应的位置处开设有与定位柱相匹配的定位孔。
进一步地，脚架连接杆侧壁开设有与连接孔相连通的通孔，脚架连接杆下端面与衔接柱上端面贴合时通孔的轴心线与插孔的轴心线重合，插柱的直径分别与插孔的内径、通孔的内径相匹配，限位盘的直径大于插柱的直径。
进一步地，脚架连接杆侧壁对称设置有一对用于分别对螺纹锁紧杆的端部、锁紧螺母的端部进行支撑的支撑管，螺纹锁紧杆贯穿支撑管，支撑管的内径与插柱的直径相匹配，限位盘的直径大于支撑管的内径。
根据本实用新型的一种方便拆装脚架的低空飞行器，通过使用螺纹锁紧杆贯穿脚架连接杆且插入连接柱的插孔内部，然后使用锁紧螺母与螺纹锁紧杆配合，从而将连接柱固定在脚架连接杆内部，从而将脚架主体安装在脚架连接杆上，当需要拆卸时，只需取下锁紧螺母将螺纹锁紧杆抽出即可使连接柱脱离脚架连接杆，从而对脚架主体与脚架连接杆分离，起到便于对脚架主体进行拆装的作用，使用效果好。

具体实施方式
为使本实用新型实现的技术手段、创作特征、达成目的与功效易于明白了解，下面结合具体实施方式，进一步阐述本实用新型。
如图1所示，本实用新型提供一种技术方案：一种方便拆装脚架的低空飞行器，包括：
机架主体1，机架主体1底部对称设置有一对脚架连接杆2，一对脚架连接杆2底部均设置有脚架主体4；
脚架主体4包括设置在脚架连接杆2底部的衔接柱42，衔接柱42底部设置有支撑柱41，衔接柱42顶部设置有插入脚架连接杆2内部的连接柱421，连接柱421侧壁贯通开设有插孔4211；
连接柱421与脚架连接杆2之间设置有锁紧组件3，锁紧组件3包括贯穿脚架连接杆2且插入插孔4211内部的螺纹锁紧杆31，螺纹锁紧杆31上设置有锁紧螺母32，该设计通过使用螺纹锁紧杆31贯穿脚架连接杆2且插入连接柱421的插孔4211内部，然后使用锁紧螺母32与螺纹锁紧杆31配合，从而将连接柱421固定在脚架连接杆2内部，从而将脚架主体4安装在脚架连接杆2上，当需要拆卸时，只需取下锁紧螺母32将螺纹锁紧杆31抽出即可使连接柱421脱离脚架连接杆2，从而对脚架主体4与脚架连接杆2分离，起到便于对脚架主体4进行拆装的作用，使用效果好。
参照图2和图3，锁紧螺母32侧壁设置有施力柱33，该设计通过使用施力柱33便于对锁紧螺母32施力，从而便于对锁紧螺母32进行拆装，从而进一步方便对脚架主体4进行拆装。
参照图3和图4，螺纹锁紧杆31包括设置在脚架连接杆2侧壁的限位盘311，限位盘311端部设置有贯穿脚架连接杆2侧壁且插入插孔4211内部的插柱312，插柱312末端侧壁设置有外螺纹313，锁紧螺母32与外螺纹313相匹配，该设计通过使用螺纹锁紧杆31的限位盘311便于对插柱312进行限位，插柱312插入连接柱421的插孔4211内部，同时与外螺纹313上的锁紧螺母32配合，使限位盘311和锁紧螺母32对脚架连接杆2进行夹紧，从而将连接柱421固定在脚架连接杆2内部，提高固定处的稳定性。
参照图1和图3，脚架连接杆2底部开设有与连接柱421相匹配的连接孔5，衔接柱42顶部对称设置有置于连接柱421两侧且置于插孔4211下方的定位柱43，脚架连接杆2底部与定位柱43相对应的位置处开设有与定位柱43相匹配的定位孔6，该设计通过使用定位柱43与定位孔6连接，避免脚架连接杆2与连接柱421之间发生侧滑，提高连接处的稳定性。
参照图3和图4，脚架连接杆2侧壁开设有与连接孔5相连通的通孔，脚架连接杆2下端面与衔接柱42上端面贴合时通孔的轴心线与插孔4211的轴心线重合，插柱312的直径分别与插孔4211的内径、通孔的内径相匹配，限位盘311的直径大于插柱312的直径，该设计通过使脚架连接杆2下端面与衔接柱42上端面贴合时通孔的轴心线与插孔4211的轴心线重合，从而便于将螺纹锁紧杆31贯穿脚架连接杆2的通孔插入连接柱421的插孔4211内部，加快螺纹锁紧杆31的插入速度，加快连接柱421固定在脚架连接杆2内部的速度，从而将脚架主体4快速的安装在脚架连接杆2上。
参照图3，脚架连接杆2侧壁对称设置有一对用于分别对螺纹锁紧杆31的端部、锁紧螺母32的端部进行支撑的支撑管21，螺纹锁紧杆31贯穿支撑管21，支撑管21的内径与插柱312的直径相匹配，限位盘311的直径大于支撑管21的内径，该设计增大了接触面积，提高了螺纹锁紧杆31上的限位盘311以及锁紧螺母32连接的稳定性。
参照图1-图4，作为本实用新型的一个实施例：当需要对脚架主体4进行安装时，工作人员首先将连接柱421插入连接孔5内部，同时使定位柱43插入定位孔6内部，当脚架连接杆2下端面与衔接柱42上端面贴合时通孔的轴心线与插孔4211的轴心线重合，从而加快螺纹锁紧杆31插入插孔4211内部的速度，然后将螺纹锁紧杆31上的插柱312贯穿支撑管21、脚架连接杆2上的通孔且插入连接柱421的插孔4211内部，使螺纹锁紧杆31上的限位盘311与其中一个支撑管21端部贴合，然后使锁紧螺母32与插柱312上的外螺纹313连接，使锁紧螺母32与另外一个支撑管21的端部贴合，从而使限位盘311和锁紧螺母32对脚架连接杆2进行夹紧，从而将连接柱421固定在脚架连接杆2内部，从而便于将脚架主体4安装在脚架连接杆2上，提高固定处的稳定性；
当需要拆卸时，只需取下锁紧螺母32将螺纹锁紧杆31抽出即可使连接柱421脱离脚架连接杆2，从而便于使脚架主体4与脚架连接杆2进行分离，从而起到便于对脚架主体4进行拆装的作用，使用效果好，提高了本实用新型的实用性。
