Description

低空太阳能飞行器
所属技术领域
本发明涉及能载人自由飞翔的低空太阳能飞行器，属低空飞行器类领域。
背景技术
目前，公知的低空飞行器种类很多，我60多年来经常做飞行梦，平时也没去多想这梦，但总是在睡着后就要做飞的梦，飞得却如此自然轻松快捷，漂在空中手一展脚一踮，想高就高想低就低想飞哪里就飞到哪里，真是我的不解之迷！还好；终于我有了与梦相似的飞行方案和措施，从理论上推演已成立，我头脑非常清醒肯定不是说梦话，这十多年来我国是世界上无人机的生产大国企业达400多家，这些无人机航拍侦察和撒农药很不错，但要无人机做其它事时，还是没那么顺心顺意，世界上和我国不少人都曾研究过低空飞行器，都因动力问题而没能成功，我也遇到同样难题，但我首先从太阳能电池板进行研究创新，并找到了电能转换率低的主要原因，将其变成电池毯就能使电量翻倍，就可满足低空飞行器的动力需求，所以才研究设计出了低空太阳能飞行器，理论时速300公里、飞行高度数百米，可在低空任意悬停，能像鸟儿一样在低空自由翱翔俯瞰美好河山，负重达150公斤，能左右转向和慢速向前，在百米高空就算掉于平地也是软着陆驾驶员无损，其技术难度比地面燃油轿车简单80％、节省材料80％，设计重量约70公斤，成本比轿车低一倍，飞行操作简单，该低空太阳能飞行器若投放市场10至20年，我国和世界将有70％的私家轿车和50％的摩托车被淘汰，很多轿车摩托车工业和轿车汽配摩配产业将转向，当今我国和世界出现的交通拥堵、空气污染、停车难、违规占用应急车道现象和世界能源争夺将不复存在，产油国不再吃香，马六甲海峡若被敌人封锁也不怕，到时我国自产油都用不完，60％的能源船运和车运工具将转行，我国将有9亿人的市场需求，世界将有30亿人的市场需求，旅游交通工具将消失80％，定会彻底减缓地面交通拥堵，人们出行将更方便快捷，与此相关产业将会雨后春笋般出现，如低空飞行驾校，低空交警，低空巡警，低空飞行交通规则制定和管理，低空飞行多层立体航线制定和管理，军事禁区规则制定和管理，地面山头指向航标灯，低空科技服务公司，低空救援和维修，当今科技发展，不怕做不到，就怕想不到，这次我又想到了，真是天耀中华！可成为我国战略性的新兴产业，我国和全人类社会的发展进步又将向前迈进一大步。
发明内容
本发明解决其技术问题所采用的技术方案是；设计个座椅式的低空太阳能飞行器，座椅人肩处有左翼阵列电风机群和右翼阵列电风机群，座椅下面也有底部阵列电风机群，座椅背后有慢速推进电风机、左右转向电风机和交通安全闪烁红灯及无线电自动声光救援信号，驾驶员头顶上有光伏太阳能电池毯(是我发明专利申号是201610547945.4)为配套项目，只有解决了足够的光伏能源该项目才能成立，座椅边下有蓄电池，光电进入蓄电池后再向电机提供电源，用不完的电储存待夜间用，有电量表指南针高度表和时速表，座椅脚踏处有脚操作杆和手操作柄，分别有升、降、前进、左右转向、悬停和慢速向前功能，并设有垃圾袋食品袋饮水装置，有防风罩护腿防寒罩，有手机托架耳塞望远镜托架和收音交通广播，座椅四个脚有缓冲撑簧和万向轮，光伏太阳能电池毯能当降落伞，若下降时或在高空掉于平地能软着陆保护驾驶员，有安全带，所有楼房屋顶可做起飞降落停机坪或折叠放于家中阳台或在阳台上起飞降落，但当今楼房阳台要重新改造或新楼要重新设计。
本发明的有益效果是；利用光电能源使人类交通向低空发展，方便快捷环保安全。
