Description

一种用于低空飞行器的压力模拟装置
技术领域
本发明属于低空飞行器的压力模拟装置领域，尤其涉及一种用于低空飞行器的压力模拟装置。
背景技术
半物理仿真技术是一种在飞行器研制过程中得到广泛应用的仿真技术，通过半物理仿真可以降低研制周期和成本，及时地判断设计方案的可行性，飞行器半物理仿真技术用于地面条件下验证飞行器系统功能和性能，特别是地面条件下模拟真实飞行状态时的静压和总压对于飞行器测试尤为重要，避免了全物理仿真中高风险、高成本问题。
发明内容
为了解决上述技术问题，种一种用于低空飞行器的压力模拟装置，其特征在于，包括人机接口/仿真接口、微控制器、压力容器、气泵、气泵驱动、数字式气压传感器、软管、空压输出接口，所述机接口/仿真接口，用于设置仿真条件、接收仿真数据；所述微控制器，用于装置的核心控制部件，实现气泵的控制算法、仿真数据解算、压力传感器数据计算；所述压力容器包括总压模拟容器、静压模拟容器，用于储存气体、稳定气体压力的压力容器；所述气泵，主要包括抽气泵和打气泵，用于在微控制器的作用下，对密闭的压力容器按照一定的控制算法，实现对应仿真数值建立真实的气体压力，准确跟随仿真设定的压力数值；所述数字式气压传感器测量压力容器内气体压力数值，物理量至电信号的转换。
2.根据权利要求1所述的一种用于低空飞行器的压力模拟装置，其特征在于，所述软管用于压力容器至压力输出口的软连接，中间接有过滤器和干燥装置，防止较大颗粒和水分进入压力容器。
3.根据权利要求1所述的一种用于低空飞行器的压力模拟装置，其特征在于，所述空压输出接口包括静压输出接口和总压输出接口用于与半物理仿真中飞控的空速管对接的接口。
本发明的有益效果为：利用地面条件下模拟真实飞行状态时的静压和总压对于飞行器测试，避免了全物理仿真中高风险、高成本问题。
