Description

一种适用于低空经济的飞行器飞行时间控制方法
技术领域
本发明属于飞行器控制技术领域，特别是涉及一种适用于低空经济的飞行器飞行时间控制方法。
背景技术
随着国家推动低空经济发展的不断深入，各种无人飞行器开始被大量的应用于生产和生活中。随着飞行器数量的不断提升，如何保证飞行器安全飞行的同时，提高飞行的效率是当前面临的新课题。其中，多架飞行器共用航路的场景时有发生。比如两架飞行器要降落在同一个机场，它们要使用相同的路径汇入降落航线。为了避免飞行器在空中相遇，必然要让它们一先一后的驶入降落航线。也即两架飞行器要有顺序的抵达降落航线的起点。这时就可能要其中的一架或两架飞行器调整飞行时间来彼此配合。
在目前的技术手段下，要完成这一操作需要飞行器驾驶员根据经验手动调整飞行航线和飞行速度，并在飞行过程中不断的检查和修正，来满足飞行时间上的要求，由于是手动调整飞行时间，其调整的精度、合理性和安全性都无法得到保证，存在着很多的局限性，不能满足安全、高效飞行的需求，故针对以上问题，提出下列方案。
发明内容
本发明的目的在于提供一种适用于低空经济的飞行器飞行时间控制方法，通过计算机可自动计算出最佳的调整方案，使飞行航路满足调整的要求，解决了现有飞行器驾驶员手动调整飞行时间，其调整的精度、合理性和安全性都无法得到保证的问题。
为解决上述技术问题，本发明是通过以下技术方案实现的：
本发明为一种适用于低空经济的飞行器飞行时间控制方法，所述控制方法包括以下步骤；
步骤S1、获取数据：获取当前飞行器飞行航路、位置、速度、气象以及要调整飞行时间的数据；
步骤S2、大气环境预测与地速计算：预测大气环境参数，结合校正空速计算飞行器地速；
步骤S3、飞行时间与方案判断：根据获取数据计算原航路飞行时间，对比实际需求判断是否调整飞行时间；
步骤S4、飞行时间的调整方案计算：根据方案判断的结果按照减少或增加飞行时间的算法计算新的调整方案；
步骤S5、飞行过程监测与调整：实时监控飞行速度，根据实际地速与理论地速偏差决定是否重新计算转弯点，在末段飞行时监测汇合点坐标和剩余飞行时间，真实地速与理论地速的偏差超阈值则报警提醒调整速度。
优选的，所述步骤S4、飞行时间的调整方案计算分为减少飞行时间的调整方案计算和增加飞行时间的调整方案计算两种情况；
所述减少飞行时间的调整方案计算方法为：
飞行速度计算：以当前飞行速度为默认速度，考虑大气环境影响转换为地速后计算从当前点到目标点直线距离的最小飞行时间，当大于需要的时间，增加飞行速度，重新计算，直到小于等于，或速度达到飞行器安全飞行允许的最大速度，当达到最大速度仍不满足，算法退出；
飞行距离计算：确定从当前点到目标点的最小距离和最大距离，其中，为当前位置到目标点的直线距离，为当前航路剩余部分的距离之和；
计算和的算术平均值作为新飞行距离，根据具体飞行要求计算对应的飞行时间；
计算偏差度，当小于阈值时，则确定新的飞行距离，算法结束；否则，根据与的大小关系更新和，返回继续计算，通过迭代使达到符合值；

优选的，所述增加飞行时间的调整方案计算具体包括：

本发明具有以下有益效果：
1、本发明通过计算机自动规划调整飞行器航路，取代了传统靠驾驶员手动调整的方式，计算机通过本方法能够在要求的精度内自动计算出最佳的调整方案，使飞行航路满足调整的要求，减少了人为因素造成的不准确和不稳定等问题，同时，在多架飞行器共用航路的场景下，能够更精准地控制每架飞行器的飞行时间，避免在空中相遇等安全隐患，提高了飞行的整体效率。
2、本发明在飞行过程中可通过计算机实时监测相关数据，通过本方法对大气环境进行预测，使飞行器能更好地适应大气环境变化，确保飞行轨迹符合预期，并实时比较实际地速与理论地速，当偏差超过阈值时可重新计算转弯点，在末段飞行路线监测中，可计算汇合点坐标和剩余飞行时间，对偏差进行判断和处理，进而能够及时发现并纠正飞行过程中的异常情况，保障飞行器安全抵达目标点。
当然，实施本发明的任一产品并不一定需要同时达到以上所述的所有优点。

计算汇合点到目标点的距离，当小于阈值时，则飞行器航向在可接受范围内，再计算剩余航程，按当前实际地速计算剩余飞行时间,当与要求的到达时间偏差小于阈值，飞行器当前飞行速度在可接受范围内，当偏差大于阈值，则算法报警提醒飞行员将飞行器速度调整为算法计算出的理论值。

飞行器在沿预定航路飞行的过程中，会遇到需要按时间要求到达航路中某个点的情况，这时需要对航路进行调整，本发明提供了一种使用计算机自动规划调整方案并实时监测方案执行情况，使飞行器按时抵达目标点的方法；
参考图3，图3是某飞行器的飞行航路示意图，飞行器目前处于航路的A点，并计划沿B、C、D点飞行到目标点E，如果在这时需要调整飞到E点的时间，则本方法会在航路上计算出一个转弯点T，当飞行器飞行到T点后终止原飞行计划，接着直接飞往E点，就可以按照要求的时间到达目标点；
并且当计算出最佳方案后，计算机会通过飞行器获取的实时飞行信息自动监测飞行过程，当由于大气环境的变化（如风力、风向的改变）导致飞机的真实飞行参数偏离计算值过多时，算法会重新按真实情况计算合适的转弯点，另外，在飞行器转弯后直飞目标点的过程中算法会实时计算飞行速度、方位等数据是否满足要求，如果不满足要求会报警提醒；
本方法将调整飞行器飞行时间的操作分为减少飞行时间和增加飞行时间两种情况，当需要减少飞行时间时，会直接计算出调整方案，当需要增加飞行时间时，本方法会先通过降低飞行器飞行速度的方法将增加飞行时间的需求转化为减少飞行时间的需求，然后再计算出调整方案，具体步骤如下：

减少飞行时间算法由三部分组成，分别是：飞行速度计算、飞行距离计算和转弯点计算；
飞行速度计算负责计算出飞行器合理的飞行速度，飞行距离计算负责计算出在合理速度下使飞行时间符合要求的飞行距离，转弯点计算负责在航路上计算出一个点，确保飞行器在该点飞向目标点后到达目标点的时间满足要求，三部分顺序执行，完成工作，如图4；
飞行速度计算：
默认的飞行速度采用当前飞行速度，但飞行器从当前点飞到目标点是有最短距离的，即两个点之间的直线距离，如果飞行器使用默认速度飞行距离所用时间仍然大于需要的时间，则说明按照默认速度飞行是无法满足时间要求的，这时需要将飞行速度增加一个单位（默认1米/秒），然后再计算是否符合要求，如果不符合就继续计算，直到找出合适的速度，如果速度增加到飞行器安全飞行允许的最大速度仍然未找到合适速度，则说明需求的时间无法达到，算法退出；
这里的当前速度是飞行器的校正空速，而算法使用地速计算飞行时间，从校正空速到地速的转换需要考虑大气环境条件的影响，本方法在上述步骤中给出了大气环境的预测算法；

步骤S5、飞行过程监测与调整：
因为飞行器飞行时的地面速度受环境影响而真实环境经常发生变化，所以它的实际飞行速度可能和估算的速度有偏差，需要不停的检测，当偏差较大时，需要按照实际的速度重新计算飞行器的转弯点；
当飞行器使用算法计算出转弯点后，在飞行的过程中，算法会不断的用飞行器的实时速度值与计算中使用的理论值进行比较，当发现实际地速 与理论地速 的差大于阈值（这里设为），即时，认为地速误差过大，需要重新计算转弯点，其具体流程如图10所示；
