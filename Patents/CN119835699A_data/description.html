Description

一种面向飞行器的低空任务卸载方法、装置及系统
技术领域
本发明涉及飞行器技术领域，尤其涉及一种面向飞行器的低空任务卸载方法及装置。
背景技术
为进一步开拓城市交通资源，减缓迅速激增的地面交通工具与有限的城市道路资源之间的矛盾，电动垂直起降飞行器已成为城市交通的研究热点。为确保电动垂直起降飞行器的正常控制和运行，保障电动垂直起降飞行器通信过程中数据传输的低时延和超高可靠性需求是现有研究关注的焦点，通过合理规划通信链路上的时隙和频率等资源，信令数据的传输效率得以提升。
然而，为承载新兴行业应用，电动垂直起降飞行器往往会产生计算密集型任务，任务数据需要消耗大量的算力和存储资源，需要实时借助边缘算力设备来进行任务卸载和计算，现有针对链路传输资源规划的方法不能满足对计算、存储性能要求严苛的电动垂直起降飞行器的需求。因此，面向电动垂直起降飞行器进行任务卸载规划需要联合考虑链路传输、计算和存储等多维资源的协同分配。
现有针对电动垂直起降飞行器任务卸载研究聚焦于时隙、频率等无线网络链路传输资源的分配来降低数据传输时延，然而在实际应用中，低空网络内的计算、存储等资源分配也会影响电动垂直起降飞行器低空任务卸载过程中整体的时延和可靠性。因此为了达到更加全面地面向传输、计算和存储等多个资源维度协同优化电动垂直起降飞行器低空任务卸载效率的目的，本发明构建一种面向电动垂直起降飞行器的低空任务卸载方法及装置。
发明内容
本发明提供了一种面向飞行器的低空任务卸载方法，包括：
采集飞行器当前飞行状态信息；
根据飞行器当前飞行状态信息建立任务卸载约束优化问题；
构建深度强化学习和任务卸载场景所需的深度强化学习网络模型；
利用深度强化学习网络模型求解任务卸载约束优化问题，得到低空任务卸载资源分配规划结果。
如上所述的一种面向飞行器的低空任务卸载方法，其中，建立任务卸载约束优化问题具体为建立电动垂直起降飞行器任务卸载的目标优化函数，即最大化所有时隙内成功卸载任务的总数，包括根据当前飞行状态信息确定的成功卸载到基站的任务个数和成功卸载到边缘算力设备的任务个数。
如上所述的一种面向飞行器的低空任务卸载方法，其中，为任务卸载约束优化问题设置任务卸载约束条件，包括通信连接约束条件和资源分配比例约束条件。
如上所述的一种面向飞行器的低空任务卸载方法，其中，在建立任务卸载约束优化问题后，还包括将任务卸载约束优化问题转化为马尔可夫决策过程，设计马尔可夫决策过程的状态空间、动作空间以及奖励；将飞行器当前飞行状态信息作为马尔可夫决策过程的状态空间；将任务卸载约束条件作为马尔可夫决策过程的动作空间；奖励用于利用回报塑型方式优化所述目标优化函数。
如上所述的一种面向飞行器的低空任务卸载方法，其中，构建深度强化学习网络模型包括构建策略估计神经网络和创建状态评判神经网络，策略估计神经网络根据飞行器当前飞行状态信息计算任务卸载动作方向，状态评判神经网络用于评判飞行器当前飞行状态信息的价值。
如上所述的一种面向飞行器的低空任务卸载方法，其中，利用深度强化学习网络模型求解任务卸载约束优化问题，具体包括：通过循环迭代方式更新策略估计神经网络和状态评判神经网络的参数；
更新策略估计神经网络，具体包括：在每次迭代时间步内，获取当前时隙上的状态空间特征，通过策略估计神经网络生成对应的动作决策，与环境交互执行动作；将状态转移到下一时隙的状态，并获取当前时间步的奖励，根据下一时隙的状态和任务卸载需求确定动作执行结果；迭代结束产生每次迭代的轨迹样本集合，轨迹样本集合包含当前状态空间特征、动作决策、下一状态空间特征、当前状态对应的奖励和动作执行结果；
更新状态评判神经网络，具体包括：采集轨迹样本集合；使用轨迹样本集合中的每条轨迹序列更新状态评判神经网络的参数。
本发明还提供一种面向飞行器的低空任务卸载装置，包括：所述装置执行上述任一项所述的一种面向飞行器的低空任务卸载方法。
本发明还提供一种面向飞行器的低空任务卸载系统，包括基站、电动垂直起降飞行器和低空飞行器；在基站侧安装所述低空任务卸载装置，在电动垂直起降飞行器和低空飞行器侧均配有边缘算力设备，电动垂直起降飞行器和每架低空飞行器在任务卸载时间段内都能够被基站的信号覆盖到。
本发明还提供一种计算机存储介质，其特征在于，包括：至少一个存储器和至少一个处理器；
存储器用于存储一个或多个程序指令；
处理器，用于运行一个或多个程序指令，用以执行上述任一项所述的一种面向飞行器的低空任务卸载方法。
本发明实现的有益效果如下：不同于传统的基于动态规划、启发式等算法的任务卸载优化算法，本发明设计了一种基于深度强化学习的智能资源规划算法，通过主动学习不同调度时隙内各任务与多维资源之间的映射关系，能够更加快速、高效的实现任务数据到低空算力设备的协同适配，提升每个调度时隙电动垂直起降飞行器任务的卸载成功率。

实施例一
图1描述了本发明实施例的一个应用场景。在该场景下，电动垂直起降飞行器沿着预先设定的空中走廊匀速飞行，飞行高度距离地面约300-500米，距离地面约150米处设有低空飞行器，如无人机群，每架无人机上配有边缘算力设备，可以提供计算能力，地面设有基站。基站配有高性能边缘服务器，提供高性能计算服务，所述高性能边缘服务器用于执行下述任务卸载方法；电动垂直起降飞行器和每架无人机在任务卸载时间段内都能够被地面基站的信号覆盖到。本实施例对电动垂直起降飞行器任务卸载场景中所包含的低空飞行器类型、飞行器数量、飞行高度以及地形地貌不作具体限定。
电动垂直起降飞行器在每个传输时隙内有不同的计算密集型任务需要卸载到无人机侧或基站侧，无人机侧的CPU算力相比基站侧的算力较小，计算时间开销大，但是无人机距离电动垂直起降飞行器距离更近，链路传输时延短，因此整体的回传时延需要权衡。
基站侧可提供的最大频率带宽、计算资源和存储资源量分别为{2MHz,30GHz,6kb}，无人机侧可提供的最大频率带宽、计算资源和存储资源量分别为{10MHz,250GHz,50kb}，电动垂直起降飞行器任务数据大小为0.5～1kb，CPU计算频率需求为50～100MHz，回传时延需求为20～60ms，数据传输功率为20dBm。
电动垂直起降飞行器到基站侧的路径损耗为L1＝-40-35lg(Di,s)，其中Di,s是电动垂直起降飞行器与基站之间的距离；电动垂直起降飞行器到无人机侧的路径损耗为L2＝-30-35lg(Di,u)，其中Di,u是电动垂直起降飞行器与无人机之间的距离，上述测量过程中设定背景噪声功率为-110dBm。

步骤230：构建深度强化学习和任务卸载场景所需的深度强化学习网络模型；
具体地，构建深度强化学习网络模型具体包括构建策略估计神经网络和状态评判神经网络。
策略估计神经网络用于根据当前的状态s(t)计算生成对应的动作a(t)，在不同状态下生成动作的概率为πθ(at|st)，其中θ是策略估计神经网络的参数矩阵。策略估计神经网络包含输入层、中间层和输出层。深度强化学习算法使用Adam优化器，学习率为0.0001，中间层包含两层全连接神经网络和一层ReLU，经验池大小设置为2048，采样的批量大小设为64，折扣因子均为0.95。其中，输入层维度等于s(t)的特征维度，输出层的维度等于a(t)的特征维度。具体地，输出层分为4个动作组，分别是卸载决策动作组、频率资源决策动作组、计算资源决策动作组和存储资源决策动作组。卸载决策动作组用于决策电动垂直起降飞行器的任务向哪个边缘算力设备卸载，此动作组内嵌实域扩展层，将Θi,s(t)和Θi,u(t)扩展到实数域范围，分别记作R[Θi,s(t)]和R[Θi,u(t)]，根据通信连接约束条件，动作组需要根据该实数值的大小决策当前任务卸载的方向，如果R[Θi,s(t)]＞R[Θi,u(t)]，则任务卸载到基站侧，否则任务卸载到低空飞行器侧。各资源决策动作组用于决策相应资源分配到当前任务的比例，各动作组均内嵌Softmax层以满足资源分配比例约束条件。策略估计神经网络包含M≤3个介于输入层和输出层之间的中间层，前M-1个中间层是全连接层，第M个中间层是ReLU层，用于非线性化特征。
状态评判神经网络用于评判当前状态的价值，包含输入层、中间层和输出层。输入层维度等于s(t)的特征维度，输出层维度等于1，输出值为Vμ(st)，用于评价输入状态s(t)具有的价值。其中μ是状态评判神经网络的参数矩阵。状态评判神经网络的中间层结构与策略估计神经网络中间层的结构相同。
