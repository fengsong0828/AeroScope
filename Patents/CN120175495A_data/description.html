Description

一种适用于低空掠海飞行器的液体水防护封严装置
技术领域
本发明属于航空宇航发动机密封领域，尤其涉及一种适用于低空掠海飞行器的液体水防护封严装置。
背景技术
随着国家大力推动低空经济的发展，无人机的应用领域日益广泛，运行环境也变得愈加复杂。对于飞行高度在1000米以下的飞行器，封严技术面临的主要挑战已从空气环境转向水环境。低空掠海飞行环境，水蒸气中含有大量盐类离子如氯化物(Cl-)、硫酸盐(SO4 2-)等，其中氯离子具有强渗透性，可破坏金属氧化保护膜(如铝合金的氧化铝层)，引发点蚀、应力腐蚀开裂和晶间腐蚀，由于海洋生物活动及大气污染产生的硫酸盐(SO4 2-)与水分结合生成硫酸，从而导致加速金属腐蚀；海水中的二氧化碳溶解后酸化水膜，促进金属的腐蚀；高速气流中，悬浮颗粒进入飞行器内部后进行碰撞，造成微切削(类似喷砂效应)，破坏防护涂层，并对其表面材料造成显著的水蚀。因此在低空掠海飞行中，含有大量杂质，酸性物质的液态水及水蒸气对低空飞行器的冲击、水蚀成为导致影响其飞行安全及工作寿命的主要原因。为了满足低空经济对航空器长航时和低能耗的需求，必须对现有封严结构进行创新设计，以应对液态水和水蒸气的挑战。
在众多封严结构中，篦齿封严因其低成本、高结构稳定性和易于维护等优点，更适合大规模生产，能够满足低空经济发展的需求。然而，篦齿封严结构在运行过程中，由于转子件与静子件之间存在磨损，以及掠海环境中水蚀现象的存在，导致节流间隙宽度变大，篦齿封严结构的泄漏量随之增大。因此，如何设计合理的篦齿封严结构，以实现密封两侧的较大压差，进而实现封严的效果，成为迫切需要解决的问题。从国内外研究可以看出，目前篦齿封严泄漏量降低主要集中在篦齿自身主要齿形参数的优化设计，对于低空掠海飞行的飞行器，其工作环境更加严峻，故衬套外形参数的设计优化也不应忽视。
处于低空掠海飞行的飞行器，液体水及水蒸气通过封严间隙进入封严装置内部并在冲击动能及离心力作用下产生往复反弹及飞溅现象，其原理如图1所示，在动能加成下使得水蚀现象进一步加剧，对于如何消除水滴反弹及飞溅的研究迫在眉睫；在传统篦齿封严装置防护下，进入封严装置内部的液体水积聚在封严腔体内，其仅起到减缓液体水进入下一级装置的作用，故缺少了进一步处理措施。因此封严技术的改善和泄漏量的降低成为提高低空掠海飞行器性能及工作寿命的重要措施之一。
发明内容
本发明所要解决的技术问题是针对上述现有的不足，提供一种适用于低空掠海飞行器的液体水防护封严装置。本发明的液体水防护封严装置能够有效减缓外部水进入低空飞行器的封严结构，同时还能将进入封严结构的水及时排出。
为实现上述技术目的，本发明采取的技术方案为：
一种适用于低空掠海飞行器的液体水防护封严装置，包括旋转轴和同轴套设于旋转轴外部的封严衬套，旋转轴上沿轴向间隔分布有多个封严篦齿，封严篦齿的外周面与封严衬套的内环面之间形成封严间隙，相邻封严篦齿之间形成封严腔体，封严篦齿的迎风面设置有内凹的篦齿弧面，封严衬套内表面在封严篦齿前方设置有内凹的衬套弧面，篦齿弧面与衬套弧面同圆心，进入封严腔体的水流部分沿着篦齿弧面流动，部分沿着衬套弧面流动，两部分水流在封严间隙前部交汇形成水幕，阻止水流进入下一级封严腔体，旋转轴内设置有排水通道，排水通道设置有若干个进水端，进水端与封严腔体连通，排水通道的出口端设置于液体水防护封严装置的外部，水流能经进水端进入排水通道并经出口端排出。
为优化上述技术方案，采取的具体措施还包括：
上述的衬套弧面表面设有石墨涂层。
上述的封严衬套在每个衬套弧面的前部均设置有一个引水通道，一个封严腔体对应一个衬套弧面和一个引水通道，引水通道前端位于对应的封严腔体前部的封严间隙处，引水通道后端与衬套弧面前端相接。
上述的引水通道后半部分水平且与衬套弧面前端相切。
每个封严腔体的底部设置有一个排水通道的进水端，排水通道的进水端设置有单向阀，单向阀允许封严腔体内的水分单向进入排水通道。
上述的单向阀为弹性膜片，单向阀一端固定在排水通道进水端的一侧，另一端为自由端，单向阀不受外力时，单向阀盖合在排水通道进水端上，受到外力时，单向阀的自身预设弹性使得单向阀自由端能向下旋转而不能向上旋转，当单向阀自由端向下旋转时，排水通道进水端开通。
上述的封严篦齿的数量为四个，相应的，封严腔体的数量为三个，排水通道有三个进水端。
上述的引水通道为中部向上凹的平滑弯曲通道，排水通道的进水端、出水端及弯管处均采用平滑过渡。
与现有技术相比，本发明的有益效果如下：
1、本发明改变封严篦齿的传统几何外形，将封严篦齿迎风面及封严衬套对应处设计为圆弧形槽结构，且为同心圆设计，流体流经弧形衬套及弧形篦齿并于篦齿齿尖处形成水幕，极大程度阻止后方来流进入下一级封严腔体。
2、本发明在每个衬套弧面的前部均设置有一个细小引水通道，分流一部分原流经封严间隙处的流体，从而减小原流经封严间隙处流体流速，减少流向下一级封严腔体总流量。
3、细小引水通道内流速高于外部流体流速，且引水通道后半部分水平且与衬套弧面前端相切，从而引水通道出流使得流经衬套弧面的流体加速，以促进篦齿齿尖处水幕形成，细小引水通道内沿程阻力相比封严篦齿与封严衬套处沿程阻力更大，从而进一步消耗流体动能，阻止流体进入下一级封严腔体，并使流体更容易落入封严腔体底部，进而进入排水通道。
4、衬套弧面表面设有石墨涂层，可以应对低空掠海飞行中进入封严装置后造成的水蚀现象。
5、本发明于旋转轴内设置排水通道，用于排出进入封严腔体内的流体，填补了常规篦齿封严装置在应对低空掠海飞行时无法排水的空白；排水通道入口处设有单向阀，此单向阀仅可由封严腔体向排水通道内部开启，当水滴撞击在单向阀部分时，单向阀打开，促使水滴快速排出，减少了因水滴在篦齿间来回撞击及反复飞溅对封严装置造成的破坏。
综上，本发明强化了封严篦齿与封严衬套间相互配合的节流、耗散效果；通过排水通道的设置，及时有效排出进入水分，减少飞行器内水分进入，延长飞行器使用寿命。
