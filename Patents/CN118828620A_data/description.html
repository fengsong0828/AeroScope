Description

一种低空无人飞行器的通信故障监测方法及装置
技术领域
本发明涉及故障监测技术领域，具体涉及一种低空无人飞行器的通信故障监测方法及装置。
背景技术
随着无人机技术的快速发展，低空无人飞行器在各个领域的应用日益广泛。在低空无人飞行器应用场景中，无人机的高效通信是其执行任务的关键。然而，由于低空无人飞行器飞行环境的复杂性和动态性，通信故障时有发生通信故障不仅可能导致无人机失去控制，还可能造成任务失败甚至造成安全隐患。传统的低空无人飞行器通信故障监测方法主要依赖于人工巡检和事后分析，这种方法存在实时性差、准确性不高等问题。
现有技术中存在低空无人飞行器通信故障监测实时性差、准确性不高的技术问题。
发明内容
本申请提供了一种低空无人飞行器的通信故障监测方法及装置，用于解决现有技术中存在低空无人飞行器通信故障监测效率低下、实时性差、准确性不高的技术问题。
鉴于上述问题，本申请提供了一种低空无人飞行器的通信故障监测方法及装置。

实施例一
如图1所示，本申请提供了一种低空无人飞行器的通信故障监测方法，该方法包括：
基于低空无人飞行器的通信流程信息，按照多个通信塔台结合低空无人飞行器对多个通信传感器进行布设，生成通信传感网络。
具体而言，低空无人飞行器是一种不搭载操作人员的飞行器，通过遥控或自主程序控制飞行。低空无人飞行器的通信流程信息是指无人机在执行任务过程中，与地面控制站或其他设备之间进行的通信活动的详细记录和描述，包括低空无人飞行器的通信发送端、传输通道和接收端的多个方面。分析无人飞行器的通信发送端、传输通道和接收端的特性，按照无人飞行器的飞行轨迹，覆盖无人飞行器的整个飞行区域合理布局的多个通信塔台，结合低空无人飞行器布设多个通信传感器。通过布设多个通信传感器，能够实时收集无人飞行器的通信数据，并将其传输至对应的通信塔台。最终，结合通信塔台和传感器生成通信传感网络，通信传感网络为无人飞行器的安全飞行和实时数据传输提供了强有力的保障。
根据所述通信传感网络对低空无人飞行器进行实时传感，获得实时通信信号数据。
对所述实时通信信号数据进行特征提取，生成多个信号特征。
具体而言，利用通信传感网络，实时获得低空无人飞行器在飞行过程中产生的通信信号数据。并根据无人飞行器的通信需求在实时通信信号数据中识别提取与通信信号相关的特征，形成多个信号特征。通过提取获得多个信号特征，可以更好地监测无人飞行器的通信性能和可能存在的问题。
构建故障检测模型，将所述实时通信信号数据结合所述多个信号特征同步至所述故障检测模型，获得故障检测结果。
具体而言，根据低空无人飞行器历史通信信号数据及其对应的故障标签作为训练数据集，使用训练集数据对机器学习模型进行训练，其中，机器学习模型可以是神经网络模型，在模型训练完成后，将训练好的模型作为故障检测模型。将实时通信信号数据结合之前提取的多个信号特征同步输入到故障检测模型中，故障检测模型对输入数据进行分析，并输出一个故障检测结果。通过将实时通信信号数据结合多个信号特征同步至故障检测模型，能够实现对低空无人飞行器通信系统的实时、准确故障检测，进而提高无人飞行器的安全性和可靠性。
根据所述故障检测结果制定故障监测方案，执行所述故障监测方案对低空无人飞行器进行智能通信故障监测。
具体而言，在获得故障检测结果后，分析故障检测结果，确定故障的具体类型，如信号丢失、传输延迟、数据错误等，并评估故障的严重性和对无人飞行器通信的影响。进而根据故障分析结果，确定哪些通信参数是需要重点监测的关键指标，制定故障监测方案。最后，执行故障监测方案对低空无人飞行器进行智能通信故障监测，实时收集和分析无人机的通信数据，实现对低空无人飞行器的智能通信故障监测，提高监测及时性、准确性，进而提高了无人机在执行任务时通信系统的稳定性和可靠性。
进一步的，基于低空无人飞行器的通信流程信息，按照多个通信塔台，结合低空无人飞行器对多个通信传感器进行布设，生成通信传感网络，方法包括：根据所述低空无人飞行器的通信发送端、传输通道、接收端，确定所述通信流程信息；调取低空无人飞行器的飞行轨迹，设置所述多个通信塔台；按照所述通信流程信息，结合所述多个通信塔台对低空无人飞行器布设多个通信传感器；基于低空无人飞行器的通信需求，结合所述飞行轨迹将所述多个通信传感器与所述多个通信塔台进行连接，构建所述通信传感网络。
具体而言，首先，识别无人飞行器的通信发送端、如无线电发射机、天线等，分析传输通道的特性，包括无线信号在空中的传播特性，以及识别通信接收端，即地面控制站或其他接收无人飞行器数据的设备，确定接收端的技术参数和接收范围，通过分析通信发送端、传输通道和接收端的工作原理和相互关系，获得低空无人飞行器通信流程信息，通信流程信息表示信号的传输路径和各个环节的功能。
然后，从低空无人飞行器历史数据库中获取低空无人飞行器的飞行计划和历史飞行数据，包括起飞点、航线、高度、速度和降落点等信息。根据飞行数据，生成空无人飞行器的飞行轨迹图。根据飞行轨迹和通信需求，选择合适的位置设置多个通信塔台，通信塔台是指在低空无人飞行器的通信中，用于提供信号中继和数据处理的地面设施。设置的通信塔台能够覆盖低空无人飞行器的整个飞行路径，确保飞行区域内信号覆盖均匀且无死角。
接下来，根据通信流程信息和通信需求，选择合适的通信传感器，如信号强度传感器、频谱分析仪、数据包监测器等，在低空无人飞行器的关键部位和/或需要监测的区域安装通信传感器，确保能够实时监测通信状态，并将数据传输到相应的通信塔台。
最后，基于低空无人飞行器的通信需求和飞行轨迹，将多个通信传感器与多个通信塔台进行物理连接，确保数据能够顺畅地在传感器和塔台之间传输，获得通信传感网络，该网络覆盖了低空无人飞行器的整个飞行轨迹。通过通信传感网络可以实时监测通信信号的传输情况，能够及时发现并处理通信故障，提高了通信故障监测及时性和准确性，进而保障无人机在执行任务时的通信质量和安全性。
进一步的，根据所述通信传感网络对低空无人飞行器进行实时传感，获得实时通信信号数据，方法包括：获取低空无人飞行器的多个实时飞行传感数据，所述多个实时飞行传感数据包含实时飞行高度数据、实时飞行速度数据、实时飞行角度数据、实时飞行位置数据；将所述实时飞行高度数据、所述实时飞行速度数据、所述实时飞行角度数据、所述实时飞行位置数据共享至所述通信传感网络进行通信分析，获得实时通信状态信息；基于所述实时通信状态信息调整所述通信传感网络，并对低空无人飞行器进行传感采集，获得所述实时通信信号数据。
具体而言，使用无人机上的多种传感器获取低空无人飞行器的多个实时飞行传感数据，如通过高度传感器、速度传感器、姿态传感器和定位传感器，分别获取实时飞行高度数据、实时飞行速度数据、实时飞行角度数据和实时飞行位置数据，其中，实时飞行角度数据包括飞行航向角、滚转角、俯仰角和偏航角。然后，将获取的实时飞行高度数据、实时飞行速度数据、实时飞行角度数据和实时飞行位置数据共享到通信传感网络中，通信传感网络在接收到低空无人飞行器的数据后，对实时飞行传感数据进行实时分析，评估当前的通信状态。并根据分析结果生成实时通信状态信息，包括信号强度、数据传输速率、延迟时间等指标。最后，基于实时通信状态信息，对通信传感网络进行动态调整，如信号频率、传输功率、路由策略等，以优化通信性能。在通信传感网络调整优化的同时，低空无人飞行器上的传感器也会持续进行数据采集，获取实时通信信号数据。通过对低空无人飞行器的多个实时飞行传感数据的获取、传输、分析和应用，为无人机的安全飞行和通信网络的优化提供有力支持，确保飞行过程中的通信稳定性和可靠性。
进一步的，对所述实时通信信号数据进行特征提取，生成多个信号特征，方法包括：对所述实时通信信号数据进行预处理，获得通信信号处理结果；根据所述通信信号处理结果，结合所述实时飞行高度数据进行特征提取，获得第一信号特征；根据所述通信信号处理结果，结合所述实时飞行速度数据进行特征提取，获得第二信号特征；根据所述通信信号处理结果，结合所述实时飞行角度数据进行特征提取，获得第三信号特征；根据所述通信信号处理结果，结合所述实时飞行位置数据进行特征提取，获得第四信号特征；将所述第一信号特征、所述第二信号特征、所述第三信号特征、所述第四信号特征添加至所述多个信号特征。
具体而言，对采集的实时通信信号进行预处理，得到经过处理的通信信号数据，即通信信号处理结果。预处理包括降噪、异常值去除和数据标准化，通过对信号进行降噪处理，如滤波、平滑等，提高信号的质量，通过检测并去除信号中的异常值，能够防止对后续通信分析造成干扰，通过对信号数据进行标准化处理，便于后续分析。
确定低空无人飞行器飞行高度的关键参数，包括飞行最大限高、正常飞行高度以及通信传输最大距离对应的飞行高度等。根据通信信号处理结果结合飞行高度的关键参数进行特征提取，获得第一信号特征，第一信号特征包括实时高度变化率、高度与通信稳定性的关系以及高度与通信传输质量关系。同理，确定实时飞行速度数据关键参数，包括飞行匀速、飞行加速时的加速度等，根据通信信号处理结果结合飞行速度数据关键参数进行特征提取，获得第二信号特征，第二信号特征包括实时飞行速度数据的变化率、飞行速度的稳定性以及速度变化与通信信号质量之间的关系；确定实时飞行角度数据的关键参数，包括航向角、滚转角、俯仰角和偏航角，根据通信信号处理结果结合飞行角度数据的关键参数进行特征提取，获得第三信号特征，第三信号特征包括姿态稳定性、角度变化与通信信号质量之间的关系，角度变化如航向角、滚转角、俯仰角和偏航角等；确定实时飞行位置数据关键参数，包括经纬度坐标、高度变化率，根据通信信号处理结果结合实时飞行位置数据关键参数进行特征提取，获得第四信号特征，第四信号特征包括经纬度坐标变化、高度变化率以及位置变化与通信信号质量之间的关系。最后，将上述提取的第一信号特征、第二信号特征、第三信号特征和第四信号特征添加到多个信号特征中，形成一个多维度的特征集合，能够全面反映无人机的飞行状态和通信质量，为后续的数据分析和决策提供支持。
进一步的，对所述实时通信信号数据进行预处理，获得通信信号处理结果，方法包括：对所述实时通信信号数据进行小波分解，获得信号小波系数；根据所述信号小波系数进行阈值量化，确定信号小波选取阈值；按照所述信号小波选取阈值对所述信号小波系数进行截取，对小于所述信号小波选取阈值的噪声信号置零，获得大于所述信号小波选取阈值的有效信号信息；对所述有效信号信息进行滤波重构，获得实时通信去噪信号数据；将所述实时通信去噪信号数据添加至所述通信信号处理结果。
具体而言，根据信号的特点选择合适的小波基函数，如Haar、Daubechies、Symlets等，将实时通信信号数据分解成多个尺度的小波系数，每个尺度的系数代表了信号在不同频率范围内的信息。进而根据信号的特性和噪声水平选择一个阈值，对小波系数进行阈值量化处理，即对小波系数的绝对值与阈值进行比较。根据阈值量化的结果，将小波系数分为两类：大于阈值的系数，即信号部分，和小于或等于阈值的系数，即噪声部分，并对小于等于信号小波选取阈值的噪声信号的小波系数进行置零操作，消除或减弱噪声对信号的影响，并保留大于信号小波选取阈值的有效信号信息的小波系数。最后，使用保留的有效小波系数进行小波逆变换，重构出去噪后的信号，相当于对信号进行了滤波，去除了噪声成分，获得实时通信去噪信号数据，并将实时通信去噪信号数据整合到原始的通信信号处理结果中。整合后的通信信号处理结果将包含更为准确、清晰的信号信息，有助于提高后续通信监测的准确性和可靠性。
