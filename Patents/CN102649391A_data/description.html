Description

地效式超低空两用飞行汽车或两用飞行器
(一)、技术领域
本发明涉及一种飞行器设计技术领域，特别是一种可作为代步车的地效式超低空两用飞行汽车或两用飞行器(亦称旋翼式超低空地效平地两用飞行器)。
(二)、背景技术
随着经济的发展，无论是城市和农村的交通道路建设、还是山区或平原公路网建设都有了极大的发展改善，但是，有更多的城镇居民因生活水平的提高而购置了小汽车以方便自己的出行，目前，城市道路的拥堵问题已经成为绝大多数城市几乎不可克服的难题(或成为绝大多数城市难以治愈的“癌症”)，即使在农村和山区，因为道路宽度窄小，交通条件差，又经常受到雨水冲刷破损，或冰雪冻雨影响，亦使原本畅通的交通条件受到极大影响。另，跟所有的机动车一样，小汽车(轿车)因其结构的原因，只能在平坦的公路道路上行驶，即使是越野车亦只能作陆地行驶，皆不具有空中飞行的能力，也不具有气垫船或其他地效飞行器的飞行功能。反之也一样，无论是飞机或直升飞机，还是气垫船或其他地效飞行器皆无法在陆地行驶(指着地行驶)，例如，中国申请号为201010263416.4的发明专利公开了一种“垂直起降小型飞行器”，该发明是利用螺旋桨产生的动能，再利用螺旋桨下面的叶片改变气流方向，使飞行器不仅能垂直起降和向前飞行，而且速度快、安全，驾驶操作简单，是未来向空中发展的理想交通工具，该小型飞行器的缺点是无法在陆地行驶(指着地行驶)。
为了克服上述缺陷，中国申请号为200810045544.4的发明专利公开了一种“飞行汽车”，该发明是在现有普通汽车的构造基础上再加装航空喷气式发动机或旋桨式航空发动机而构成，它具有两套各自独立的发动机和操纵机构系统，集普通汽车的功能、直升飞机功能于一体。汽车和直升飞机两种工况方式根据需要，可以随心所欲地进行转换，因为是在较高的高空飞行，在车厢的顶部装设有特制降落伞，在紧急情况下可以打开以确保飞行安全。飞行汽车具有安全、可靠、便捷，工作方式机动灵活的特性。又如，中国申请号为201110151353.8的发明专利亦公开了另外一种“飞行汽车”，该发明是在现有汽车的车厢顶上加装航空发动机和伸缩式活动机翼，航空发动机在车顶部固定安装或装置在机翼上随液压操作随同伸缩式机翼移动，在车顶液压伸缩式机翼两端装置的螺旋桨式发动机可0至90度往复式偏转角度，改变旋浆式发动机的偏转角度即能垂直起降。该发明同样具有汽车和飞机两套独立工作的发动机和操纵系统，在车厢顶部设置有燃油箱和多层组合式降落伞，在车顶尾部设有垂直翼，飞行安全可靠、用途广泛，应用前景很好。但上述飞行汽车存在的缺点是结构复杂，安装维修不方便，成本造价高，又因为是在较高的高空飞行，虽在车厢顶部皆装设有特制降落伞，仍缺乏安全感。
(三)、发明内容
本发明的目的是要克服现有技术的不足之处，从而提供一种结构合理，制造加工装配容易，能在公路上正常行驶，又能贴近地面或水面飞行的垂直起降的地效式超低空两用飞行汽车或两用飞行器(亦称旋翼式超低空地效平地两用飞行器)。使用该飞行汽车有利缓解城市道路拥堵问题，及有利解决农村和山区道路交通畅通条件差的问题，具有安全、可靠、便捷，出行方式机动灵活的特点。
本发明是以如下技术方案构成的：它包括组成地面行驶系统的车体部分、动力组部分及综合控制系统，组成地面行驶系统的车体部分包括车身机架、底盘、方向盘、驱动车轮，动力组部分包括电源及动力装置，其特征是：
1)、所述动力装置包括：安装在车身机架中部的装有共轴反转的上旋翼和下旋翼，通过上旋翼和下旋翼共轴反转以提供和产生向上主升力，安装在车身机架后部两侧的一对兼具纵向推进和转向用的水平涵道螺旋桨和驱动其运转的电机，安装在车身机架下方底部的起提供辅助升力作用的复合空气柜及其上、下压气机所组成的复合气垫装置；
2)、在所述车身机架的中部设有一个竖向涵道，所述用于安装上旋翼和下旋翼的共轴是由作为内主轴的第一主轴和活动安装套装在第一主轴外面的作为外主轴的第二主轴组成，它们同轴安装在车身机架中部的一个竖向涵道中；所述的上旋翼是安装在内主轴，即第一主轴上，在第一主轴下方固定安装有一伞形齿轮组件，并通过一联轴节与一为上旋翼提供运转动力的电机输出轴相连接；所述的下旋翼是安装在外主轴，即第二主轴上，在第二主轴下方亦固定安装有一伞形齿轮组件，并通过一联轴节与一为下旋翼提供运转动力的电机输出轴相连接；
3)、在所述的共轴外面，即作为外主轴的第二主轴的外面活动安装有一外轴套(即第三主轴)，在外轴套即第三主轴上方固定安装有一上压气机(位于竖向涵道进气口处)，在外轴套即第三主轴下方固定安装有一伞形齿轮组件，并通过一联轴节与一为上压气机提供运转动力的电机输出轴相连接；
4)、在所述作为内主轴的第一主轴的底部设有一下压气机主轴(又称为第四主轴)，所述的作为内主轴的第一主轴的底部通过一内装有限位轴承的联轴节与所述下压气机主轴(又称第四主轴)相连接，所述的下压气机主轴(即第四主轴)安装在限位轴承座中，在下压气机主轴(即第四主轴)底部伸入复合空气柜C中的部位固定安装有一下压气机(又称主压气机)，在所述下压气机主轴(即第四主轴)上方固定安装有一伞形齿轮组件，并通过一联轴节与一为下压气机提供运转动力的电机输出轴相连接；
5)、所述用于形成气垫的复合空气柜是由A、B、C三部分组成，其中A部分空气柜是由竖向涵道所构成，其中B部分空气柜是由位于汽车底盘下方的空间所构成，所述B部分空气柜的进气口是与A部分空气柜，即竖向涵道的出气口相连通，B部分空气柜的出气口分布在汽车底盘下方的边缘部(即汽车四周下方的侧沿部)，在B部分空气柜的下方具有一隔层板，其中C部分空气柜是由位于隔层板下方(即B部分空气柜下方)的空间所构成，在车身机架的两侧设有用于为C部分空气柜供气的侧向风道(涵道)，所述C部分空气柜的进气口是与所述侧向风道(涵道)的出气口相连通。
本发明在具体设计制造时，其技术特征还可进一步具体为：
一、所述的共轴反转的上旋翼和下旋翼是制成短翼型(翼形)结构，其长度小于或等于汽车车身的宽度，在上旋翼和下旋翼的外沿设有旋翼护圈，所述旋翼护圈是固定在车身机架上方的前、后锥形护圈架上。
二、在所述的上压气机是位于竖向涵道的进气口处。
三、所述的共轴底部即作为内主轴的第一主轴的底部是通过一内装有限位轴承的联轴节与下压气机主轴(又称第四主轴)相连接。
四、所述的机架上方的车身外壳是制成弧形曲线的造型结构，在车身机架下方四周设有用于形成气垫的围裙。
五、所述的用于地面行驶的驱动车轮装置是由前、后(纵向推进用的)电动驱动车轮所组成(相当于电动车的驱动车轮)，在每个驱动车轮装置中皆设有一纵向推进用的平路驱动电动机。
本发明结构合理，制造加工装配容易，能在公路上正常行驶，又能贴近地面或水面作超低空飞行，有利缓解城市道路拥堵问题，及解决农村和山区道路交通畅通条件差的问题，具有安全、可靠、便捷，出行方式机动灵活的特点。

本发明在具体设计制造时，其技术特征还可进一步具体为：
一、所述的共轴反转的上旋翼2和下旋翼3是制成短翼型(翼形)结构，其长度小于或等于汽车车身的宽度，在上旋翼和下旋翼的外沿设有旋翼护圈(保护壳)5，所述旋翼护圈(保护壳)5是固定在车身机架上方的前、后锥形护圈架39、40上。
二、在所述的上压气机8是位于竖向涵道(风道)41的进气口处。
三、所述的共轴底部即作为内主轴的第一主轴的底部是通过一内装有限位轴承的联轴节18与下压气机主轴23(又称第四主轴23)相连接。
四、所述的机架上方的车身外壳是制成弧形曲线(流水线)的造型结构，在车身机架下方四周设有用于形成气垫的围裙33。
五、所述的用于地面行驶的驱动车轮装置是由前、后电动驱动车轮26、27所组成(相当于电动车的驱动车轮)，在每个驱动车轮装置中皆设有一纵向推进用的平路驱动电动机。
本发明的工作原理过程如下：
以两用飞行汽车为例，作地面行驶时，驾驶员启动位于车头方向盘31前方的电控制及驱动控制箱柜(即电控柜，组成综合控制系统)30面板上的平路行驶按钮开关，启动地面行驶的平路驱动电动机(驱动车轮电机)28，带动作纵向推进用的前后电动驱动车轮26、27运转，即可作平路地面行驶(操控方法与现有电动汽车的相同)。
当遇到道路拥堵或道路受雨水冲刷破损，或受冰雪冻雨影响，宜作地效式超低空飞行时，驾驶员启动位于车头方向盘31前方的电控制及驱动控制箱柜(即电控柜，组成综合控制系统)30面板上的为上、下旋翼提供运转动力的电机17、14的按钮开关，通过上旋翼和下旋翼共轴反转以提供和产生向上主升力；启动用于复合气垫装置的按钮开关，使为上、下压气机8、25提供运转动力的电机11、21运转，从而在汽车底盘下方的复合空气柜形成气垫以提供辅助升力；启动纵向推进和转向用的按钮开关，使安装在车身车架后部两侧的一对兼具纵向推进和转向用的水平涵道螺旋桨22和驱动其运转的电机运转工作，即可作超低空飞行时的纵向推进和转向用途。
