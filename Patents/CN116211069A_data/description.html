CN116211069A

Description

一种低空飞行器的智能导航控制装置
技术领域
本发明属于控制装置领域，具体为一种低空飞行器的智能导航控制装置。
背景技术
导航控制所需的中控平台一般集成于一个平台上，称之为总控平台，其类似于桌体和显示控制器的结合体。
为方便整体布局放置，这种总控平台上的显示控制装置都是以某一个角度固定在桌体上，其安装以后无法移动，也无法进行调节。总控平台的控制操作是由控制人员来控制的，而控制人员的身高体型各不相同，这就导致不同的控制人员在坐在总控平台前，显示控制装置与其眼睛的角度是不同的，这就可能会出现显示控制平台的角度与控制人员的可视化角度不匹配的情况。
发明内容
要解决的技术问题：显示控制平台集成安装在固定角度，与控制人员可视化角度不匹配。
技术方案：本发明提供了一种低空飞行器的智能导航控制装置，包括本体结构、调节结构和外护结构：本体结构包括支架，支架的顶端设置台面，台面的顶端集成有显示控制装置，台面上固定有一水平的轴杆；调节结构包括两个对称安装在支架上的轴承，轴杆的两端分别与两个轴承配合连接，支架上还设置有用于驱动轴杆进行转动的驱动机构，集成有显示控制装置的台面通过轴杆和轴承与支架铰接，在驱动机构的作用下，可调节轴杆的转动角度，从而使得台面和显示控制装置可在一定范围内进行角度的调节，如此，控制人员即可根据自己的实际需要来将显示控制装置调整至最适合自己观看的角度；外护结构包括既可连接在显示控制装置正面也可连接在显示控制装置背面的U形的防护架，其既可以在不使用显示控制装置时连接在显示控制装置的正面来对显示控制装置进行保护，也可以在使用时将其连接在显示控制装置的背面，不妨碍使用。
进一步地，驱动机构包括安装在支架外侧的且输出轴与轴杆平行的电机，电机的输出轴和轴杆的外周侧均固定套接有相互啮合的且位于支架外的齿轮。
进一步地，支架的外侧还安装有一个外壳，外壳与支架的连接处开口，电机与两个齿轮均位于外壳内。
进一步地，外壳的顶端贯穿开设有用于散热的通孔，通孔设置有多个，多个通孔均匀分布。
进一步地，外壳的两侧均安装有耳板，每个耳板上均贯穿开设有耳孔，支架上开设有与耳孔相对应的连接孔，使得耳板可通过耳孔和连接孔配合连接螺柱进行固定，即外壳是可取下的，这样设置的目的是方便对其内部的齿轮进行定期保养和维护。
进一步地，每个轴承的外圈均与支架固定，每个轴承的内圈均与轴杆过盈配合，以使得轴杆可以被连接在原地进行转动。
进一步地，显示控制装置的两侧均固定有竖直的连接条，两个连接条上相互远离的一侧均开设有沿其高度方向进行分布的滑槽，每个滑槽内均卡合滑动连接有一滑块，防护架的两侧内壁分别与对应的滑块固定连接。
进一步地，滑槽与滑块既能滑动又能卡合的实现结构有多种，这里优选为每个滑槽上其槽底的宽度均大于其槽顶的宽度，每个滑块均与此滑槽相适配。
进一步地，上述中的每个滑槽均是T形槽，每个滑块均是与此T形槽相适配的T形块。
进一步地，上述中的每个滑槽均是梯形槽，每个滑块均是与此梯形槽相适配的梯形块。
技术效果：
本发明中，集成有显示控制装置的台面通过轴杆和轴承与支架铰接，在驱动机构的作用下，可调节轴杆的转动角度，从而使得台面和显示控制装置可在一定范围内进行角度的调节，如此，控制人员即可根据自己的实际需要来将显示控制装置调整至最适合自己观看的角度。
本发明中，防护架为一个U形的架体结构，其既可以在不使用显示控制装置时连接在显示控制装置的正面来对显示控制装置进行保护，也可以在使用时将其连接在显示控制装置的背面，不妨碍使用，转换方便。

在本装置空闲处，安置所有电器件与其相匹配的驱动器，并且通过本领域人员，将下述中所有驱动件，其指代动力元件、电器件以及适配的电源通过导线进行连接，具体连接手段，应参考下述表述中，各电器件之间先后工作顺序完成电性连接，其详细连接手段，为本领域公知技术。
实施例一：
本具体实施方式提供的低空飞行器的智能导航控制装置，如图1、图2和图3所示，包括本体结构、调节结构和外护结构，其中：
本体结构包括支架1，在这个支架1的顶端设置一个水平的台面2，台面2的顶端集成有显示控制装置3，台面2上还固定有一水平的轴杆5；
调节结构包括两个对称安装在支架1上的轴承4，上述中的轴杆5的两端分别与两个轴承4配合连接，以使得轴杆5、台面2及显示控制装置3都是可转动的连接在支架1上的，另外，在支架1上还设置有用于驱动轴杆5进行转动的驱动机构6，如图3所示，驱动机构6包括安装在支架1外侧的且输出轴与轴杆5平行的电机601，这个电机601的输出轴和轴杆5的外周侧均固定套接有相互啮合的且位于支架1外的齿轮602，通过电机601的驱动，在两个齿轮602的作用下，即可同步带动轴杆5进行转动，从而使与轴杆5一体的台面2和显示控制装置3可以在一定范围内进行角度的调整，这样，控制人员即可根据自己的视角来对显示控制装置3的显示角度来进行调节。
外护结构包括既可连接在显示控制装置3正面也可连接在显示控制装置3背面的U形的防护架10，如图5所示，在显示控制装置3的两侧均固定有竖直的连接条7，两个连接条7上相互远离的一侧均开设有沿其高度方向进行分布的滑槽8，每个滑槽8内均卡合滑动连接有一滑块9，而上述中的防护架10的两侧内壁分别与对应的滑块9固定连接，如此，使得防护架10是从上方连接在显示控制装置3上的，其既可以连接在正面进行保护，也可连接在背面不影响正常使用。
除上述之外：
一，如图4所示，在支架1的外侧还安装有一个外壳11，外壳11与支架1的连接处开口，上述中的电机601与两个齿轮602均位于外壳11内被进行保护，除此之外，在外壳11的顶端还贯穿开设有用于散热的通孔12，关于通孔12，其设置有多个，多个通孔12均匀分布，以得到更好的散热效果；另外，关于外壳11，其两侧均安装有耳板13，每个耳板13上均贯穿开设有耳孔，同时，在支架1上开设有与耳孔相对应的连接孔，如此，使得耳板13可通过耳孔和连接孔配合连接螺柱14进行固定，即外壳11是可取下的，这样设置的目的是方便对其内部的齿轮进行定期保养和维护。
二，上述中的每个轴承4的外圈均与支架1固定，每个轴承4的内圈均与轴杆5过盈配合，以使得轴杆5可以被连接在原地进行转动。
三，上述中的滑槽8与滑块9既能滑动又能卡合的实现结构有多种，本申请中优选每个滑槽8上其槽底的宽度均大于其槽顶的宽度，每个滑块9均与此滑槽8相适配，这样的实现结构也有多种，例如每个滑槽8均是T形槽，每个滑块9均是与此T形槽相适配的T形块；亦或是每个滑槽8均是梯形槽，每个滑块9均是与此梯形槽相适配的梯形块。
实施例二：
本具体实施方式提供的低空飞行器的智能导航控制装置，包括本体结构、调节结构和外护结构，其中，本体结构和外护结构的实现方式均与实施例一中的相同，这里便不加赘述，不同的是调节结构中的驱动机构，实施例一种的驱动机构可以理解是电动自动转动的方式，这里可以替换成手动转动的方式，具体结构如下：轴杆5的一端延伸至支架1外并安装一个圆盘，同时，支架1上靠近这个圆盘的外侧也固定一个圆盘，这个圆盘与轴杆5活动贯穿连接，两个圆盘上均贯穿开设多个呈环形阵列分布的安装孔，如此，在两个安装孔之间插销一插销即可实现角度的调节，当需要调节角度时，拔下插销进行转动，转动完毕以后，插上插销即可实现固定。
