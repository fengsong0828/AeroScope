Description

一种低空飞行器管制和防御装置
技术领域
本实用新型涉及管制和防御装置领域，特别涉及一种低空飞行器管制和防御装置。
背景技术
管制和防御装置即用于对飞行器进行拦截并进行管制的一种装置，日常用于安保人员。随着对管制和防御装置的不断重视与投入，管制和防御装置在一定程度上得到了很好地改进，能够便于使用人员进行操作，但现有的管制和防御装置还存在一定的不足，相信未来将会得到进一步完善，以满足使用时的不同需求；现有的管制和防御装置在使用时存在一定的弊端，不能很好的进行背负携带，当光线较暗的情况瞄准镜失效从而不能很好的进行瞄准，不能很方便的对其瞄准镜进行快速拆卸或安装，导致错过拦截时机，在实际的使用过程中带来了一定的影响，为此，我们提出一种低空飞行器管制和防御装置。
实用新型内容
本实用新型的主要目的在于提供一种低空飞行器管制和防御装置，可以有效解决背景技术中的问题。
为实现上述目的，本实用新型采取的技术方案为：
一种低空飞行器管制和防御装置，包括装置主体，所述装置主体的一侧外表面活动安装有电池盖，且装置主体的前端外表面一侧设有散热口，所述装置主体的上端外表面固定安装有衔接槽，且衔接槽的上端外表面设有衔接块，所述衔接块的上端外表面设有瞄准镜，且瞄准镜的上端外表面一侧固定安装有激光器，所述激光器的一侧外表面活动安装有盖帽，且激光器的前端外表面一侧靠近盖帽活动安装有开关按钮，所述激光器的一侧外表面固定安装有灯头，且瞄准镜的一侧外表面固定安装有防护镜片，所述装置主体的另一侧外表面固定安装有信号天线，且装置主体的下端外表面一侧固定安装有连接块，所述连接块的一侧外表面活动安装有开关按键，且连接块的下端外表面固定安装有把手，所述把手的一侧外表面固定安装有GPS发射天线，且装置主体的内部一侧活动安装有蓄电池，所述装置主体的内部另一侧固定安装有电路板，且装置主体的前端设有肩带。
优选的，所述装置主体的前端外表面一侧靠近散热口的一侧设有卡扣，且肩带通过卡扣与装置主体活动连接。
优选的，所述蓄电池的输出端与电路板的输入端电性连接，且电路板的内部设有干扰芯片。
优选的，所述把手的外表面设有防滑纹，且把手的形状为长方形。
优选的，所述激光器的内部设有小型电板和纽扣电池，小型电板和纽扣电池均与激光器活动连接。
优选的，所述瞄准镜通过衔接块与装置主体活动连接，且瞄准镜的形状为圆柱形。
与现有技术相比，本实用新型具有如下有益效果：该低空飞行器管制和防御装置，通过设置有肩带，从而能很好的进行背负携带，通过设置有激光器，从而当光线较暗的情况瞄准镜失效可打开激光器，从而发射出激光进行瞄准，通过设置有衔接槽，从而衔接槽和衔接块的配合能很方便的对其瞄准镜进行快速拆卸或安装，不会因为在安装瞄准镜时错过拦截时机，整个低空飞行器管制和防御装置结构简单，操作方便，使用的效果相对于传统方式更好。

为使本实用新型实现的技术手段、创作特征、达成目的与功效易于明白了解，下面结合具体实施方式，进一步阐述本实用新型。
如图1-4所示，一种低空飞行器管制和防御装置，包括装置主体1，装置主体1的一侧外表面活动安装有电池盖2，且装置主体1的前端外表面一侧设有散热口3，装置主体1的上端外表面固定安装有衔接槽4，且衔接槽4的上端外表面设有衔接块5，衔接块5的上端外表面设有瞄准镜6，且瞄准镜6的上端外表面一侧固定安装有激光器7，激光器7的一侧外表面活动安装有盖帽8，且激光器7的前端外表面一侧靠近盖帽8活动安装有开关按钮9，激光器7的一侧外表面固定安装有灯头10，且瞄准镜6的一侧外表面固定安装有防护镜片11，装置主体1的另一侧外表面固定安装有信号天线12，且装置主体1的下端外表面一侧固定安装有连接块13，连接块13的一侧外表面活动安装有开关按键14，且连接块13的下端外表面固定安装有把手15，把手15的一侧外表面固定安装有GPS发射天线16，且装置主体1的内部一侧活动安装有蓄电池17，装置主体1的内部另一侧固定安装有电路板18，且装置主体1的前端设有肩带19；
装置主体1的前端外表面一侧靠近散热口3的一侧设有卡扣，且肩带19通过卡扣与装置主体1活动连接；蓄电池17的输出端与电路板18的输入端电性连接，且电路板18的内部设有干扰芯片；把手15的外表面设有防滑纹，且把手15的形状为长方形；激光器7的内部设有小型电板和纽扣电池，小型电板和纽扣电池均与激光器7活动连接；瞄准镜6通过衔接块5与装置主体1活动连接，且瞄准镜6的形状为圆柱形。
需要说明的是，本实用新型为一种低空飞行器管制和防御装置，在使用时，工作人员需先顺时针将电池盖2打开，将蓄电池17塞入装置主体1内部，再将其逆时针旋转使其闭合，从而装置可正常使用，工作人员可将其瞄准镜6下端的衔接块5对准装置主体1上端的衔接槽4塞入，从而衔接槽4和衔接块5产生摩擦，顺利并快速的安装完毕，准备工作完成后工作人员可单手握住把手15，将其装置主体1扛在肩上，从而信号天线12和GPS发射天线16正对前方，工作人员发现目标后，可通过瞄准镜6对其飞行器进行瞄准，从而再通过手指按下开关按键14，使其装置主体1内部的电路板18启动其干扰芯片，从而GPS发射天线16和信号天线12同时向飞行器发出二点四GHz到五点八GHz之间电磁干扰波，使其飞行器内部操作板受到干扰失灵不受控制，使其掉落，从而对飞行器进行管制，肩带19可将装置主体1斜挎在肩上，可随时方便装置投入使用，当瞄准镜6失灵或夜晚光线不佳无法进行瞄准时，可按下开关按钮9，将其激光器7打开，从而激光器7通电从灯头10出发射一条射线，工作人员可使用其射线进行瞄准，激光器7使用的是纽扣电池供电，可顺时针旋转盖帽8进行电池更换，较为实用。
