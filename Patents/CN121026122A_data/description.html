Description

面向城市低空物流的低空飞行器路径规划与动态调整方法
技术领域
本发明涉及城市低空飞行控制技术领域，特别涉及面向城市低空物流的低空飞行器路径规划与动态调整方法。
背景技术
随着城市低空物流迅速发展，低空飞行器在配送末端展现出巨大应用潜力。然而，城市低空环境具有结构复杂、动态障碍物频出、可用空域资源有限等特点，飞行路径规划面临空域密度过高、路径冲突多等挑战。现有技术多依赖静态三维地图或固定禁飞边界进行路径搜索，缺乏对环境动态变化与多飞行器交互行为的实时响应，难以兼顾安全性、鲁棒性和飞行平滑性等关键指标。其次，常用路径调整方法普遍采用单层控制，难以处理复杂状态空间下的高维动作规划。此外，多飞行器调度规则局限于优先级排序与时间窗协调简单，缺乏对路径冲突、协同编队效益等因素的量化分析与全局优化。
以静态城市地图为基础，利用传统启发式搜索或群体智能优化算法求解飞行路径，是当前城市低空物流飞行器路径规划的常见方法。当前，为满足飞行器自主完成配送任务的需求，一般采用单目标优化算法基于静态障碍建图生成路径点序列，用于最短距离或最短时间路径搜索。但在面对城市空域中复杂动态障碍物、突发风险场景以及多飞行器冲突时，该类方法缺乏实时性与协同性，路径鲁棒性差，飞行效率与安全性难以保障。
发明内容
本发明提出一种面向城市低空物流的飞行器路径规划与动态调整方法，基于多目标路径优化、环境状态感知与预测、强化学习驱动的路径调整及多飞行器编队协同决策，有助于飞行器在复杂低空域中自主进行任务路径生成、避障调整、编队协同与交通流调度等全流程控制。
本发明采用如下技术方案：
本发明提供一种面向城市低空物流的飞行器路径规划与动态调整方法，包括以下步骤
S1、基于城市三维环境等参数构建飞行场景模型；
S2、采用多目标优化算法进行路径规划；
S3、利用环境预测模型与强化学习算法动态调整路径；
S4、基于多飞行器协同决策机制进行低空交通流协同。

S1.3、通过UAS Service Platform提供的空域划分规则数据，获取静态障碍物所处空域的权限类型与飞行限制等级。由其空间位置(xi,yi,zi)，确定合法飞行空域与静态禁飞边界；

S2.3、提出一种三阶段融合路径搜索算法(AGACO-TPS，A*-GA-ACO with Three-Phase Search)，基于S2.2定义的目标函数及约束，生成满足要求的最优路径点序列P*。利用A*算法在S1构建的三维环境模型中，从飞行器起点Ps到终点Pt，快速生成一条可行路径P0，作为初始解。基于初始路径P0，采用GA算法构建初始种群，通过路径个体的选择、交叉与变异，在多轮进化迭代中产生多个可选路径{P1,P2,...,Pm}。针对可选路径集中存在高风险、抖动以及局部不平滑段，采用ACO算法进行局部路径修正，优化路径段质量，生成路径{P′1,P′2,...,P′m}。对局部优化后的路径集{Pi}，计算每条路径的函数值fobj(P′i)。最终，按式(4)确定出目标函数值最小的路径P*，作为最优路径解。

本发明提供面向城市低空物流的飞行器路径规划与动态调整方法，基于动态环境感知与风险趋势预测机制，融合飞行器机载传感器输入与图卷积网络建模结果，提出一种多目标优化与三阶段路径搜索相结合的路径规划方法，并结合双层强化学习模型对路径进行动态调整，进一步在多飞行器冲突图基础上，提出利用MALNS-DRC融合算法进行路径冲突规避与灵活编队重构。
