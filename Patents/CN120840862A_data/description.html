Description

一种倾转多旋翼低空飞行器
技术领域
本发明涉及航空技术领域，具体是一种兼具垂直起降、空中悬停、低速/高速巡航能力的倾转多旋翼低空飞行器，适用于低空经济场景。
背景技术
现有多旋翼飞行器无法理想地兼顾垂直起降、空中悬停、低速巡航、高速远程巡航多种飞行模式：纯多旋翼飞行器巡航效率低，而传统固定翼飞行器需跑道起降。现有倾转旋翼方案存在结构复杂，操控困难，旋翼倾转时机身各处应力不平衡，影响结构稳定性等问题。此外，低空飞行器为适应狭窄场地起降，有必要缩小宽度尺寸，这又带来展弦比过小，导致气动干扰大、诱导阻力增大、飞行稳定性降低的问题。
因此，有必要提出创新的技术方案解决上述技术问题。
发明内容
本发明针对以上技术问题，提供一种倾转多旋翼低空飞行器,通过旋翼组整体倾转结构消除了翼身两侧产生的不平衡应力，并在倾转过程中保持旋翼姿态稳定，可以实现垂直起降、空中悬停、低速巡航及高速巡航等多种飞行模式；所述飞行器的翼身融合设计显著提高了燃油经济性和飞行速度，增强了高速远程巡航能力。
为实现上述目的，本发明的技术方案是：
一种倾转多旋翼低空飞行器，包括翼身、至少两组并列设置在所述翼身底部的旋翼组及飞控系统；
所述旋翼组设有板架，其两端对称设置至少一个旋翼；所述旋翼组通过所述板架活动铰接于所述翼身的底部，两端的旋翼对称设置于所述翼身的两侧；两侧旋翼转向相反，抵消旋翼旋转产生的反向力矩；
所述翼身上设有顶推装置连接所述板架，所述飞控系统控制所述顶推装置推动所述板架旋转使两端的旋翼在水平、倾斜以及竖直状态之间转换。
该倾转多旋翼低空飞行器的翼身两侧对称设置旋翼，且两侧的旋翼通过板架连接成一个整体的旋翼组，顶推装置整体推动旋翼组转动，确保旋翼倾转时的姿态稳定，并消除旋翼倾转时对翼身两侧产生的不平衡应力。
顶推装置推动板架旋转使两端的旋翼在水平、倾斜以及竖直状态之间切换，旋翼处于水平状态适用于垂直起降和空中悬停，旋翼倾斜状态适用于低速巡航，旋翼竖直状态适用于高速巡航，实现了所述飞行器的多种飞行模式。
进一步优化方案，所述翼身的尾部表面对称设有两个垂直安定面，其后缘分别设有垂直舵；两个所述垂直安定面的上部之间设有水平安定面，其后缘设有水平舵。
垂直安定面及水平安定面有利于确保水平飞行时的稳定性，垂直舵和水平舵可以调整飞行姿态和飞行方向。
进一步优化方案，所述翼身的下部设有支撑于降落面的起落架。起落架的作用之一是起降时支撑所述飞行器于降落面。
进一步优化方案，所述起落架上设有运载舱，其上表面与所述翼身的底面之间设有间隙形成气流通道。
起落架同时用于固定运载舱，有利于简化飞行器的结构及增加强度。运载舱的上表面和翼身的下表面留有一定空隙距离，利于气流通过，以确保所述飞行器整体的气动性能。运载舱与翼身分离设计，动力舱内置于翼身，即运载舱与动力舱隔离，从而运载舱内可以隔离动力舱内的噪声、震动和高低温，为运载舱的乘客提供舒适的环境，也有利于方便装卸货物和提升安全性。
进一步优化方案，所述顶推装置为伺服电动推杆，其位于所述翼身的两侧；所述伺服电动推杆一端旋转连接于所述翼身的侧面，另一端活动铰接于所述板架上。
旋翼组通过翼身两侧的顶推装置的推杆伸缩推动板架旋转，实现翼身两侧的旋翼在水平、倾斜以及竖直状态之间转换，两侧的顶推装置同步动作。
进一步优化方案，所述板架通过若干个铰接器与所述翼身的底部铰接；
所述铰接器包括铰接座、铰接头及铰接轴，所述铰接座为底部平直的U型，所述铰接头通过所述铰接轴旋转连接于所述铰接座U型结构之间；所述铰接座连接在所述翼身的底部，所述铰接头连接在所述板架上。
进一步优化方案，所述板架包括两根平行设置的直杆，中间通过若干连接板连接固定；
两根所述直杆沿长度方向的内侧其中一侧设有液压高压油管，另一侧设有液压低压油管；所述直杆两端外侧分别设有散热器。
液压高压油管及液压低压油管的作用是将动力从翼身内的动力舱传送到旋翼，散热器及时就近的将管内液压油的热量散发掉，保证液压传动系统的稳定。
进一步优化方案，所述翼身的两侧设有侧翼，其为薄板结构，外轮廓为所述翼身侧截面的等距外扩。翼身两侧的侧翼能有效隔离翼身下表面的高压气流和上表面的低压气流，避免上下表面气流混合形成涡流产生诱导阻力。同时侧翼还起到垂直安定面的作用，有利于飞行的稳定性。
进一步优化方案，所述翼身的两侧设有小翼，其外形为固定翼气动外形。翼身两侧增加小翼可以产生更多升力，提升飞行升阻比，利于节省燃油，同时有利于提升飞行稳定性。
进一步优化方案，所述小翼安装时设有迎角，优选地，迎角角度为5°～10°。小翼安装时保持一个适当的迎角有利于进一步提升升力。
进一步优化方案，所述小翼的腹部为内凹曲面结构。小翼腹部的内凹曲面可以进一步提升气动性能。
进一步优化方案，所述顶推装置的推杆长度最短时所述旋翼处于水平状态，推杆长度最长时所述旋翼处于竖直状态，推杆长度位于最短和最长之间时所述旋翼处于不同角度的倾斜状态。通过控制顶推装置的推杆长度实现旋翼的姿态调整，结构简单，操控可靠。
进一步优化方案，所述板架或/和顶推装置容纳于所述翼身内部，仅旋翼外露。板架或/和顶推装置内置于翼身内有益于减少飞行时的空气阻力，但需要做好翼身的气密性设计。
进一步优化方案，所述翼身外形为固定翼气动外形。翼身采用固定翼的气动外形，飞行时翼身产生向上的升力，相较于旋翼产生升力，能极大地提高燃油经济性和飞行速度。
进一步优化方案，所述翼身内部集成动力舱。翼身内部集成动力舱，结构紧凑，有效缩短传动距离，利于提高传动效率，提高飞行稳定性。
进一步优化方案，所述小翼其翼型与所述翼身的翼型相同，侧截面的尺寸为所述翼身的侧截面等比例缩小而得。
本发明的倾转多旋翼低空飞行器与现有技术相比具有如下技术优点：
1、旋翼组整体倾转机构：两侧旋翼通过板架连接为刚性整体，由顶推装置同步驱动倾转，确保旋翼倾转时的姿态稳定，同时消除单侧不平衡应力，防止翼身变形，利于简化翼身结构，提升飞行安全；
2、三种飞行状态有序平稳转换：旋翼水平状态，旋翼提供全升力，适用于垂直起降和空中悬停；旋翼倾斜状态，旋翼提供前飞推力和部分升力，适用于低速巡航；旋翼竖直状态，旋翼完全提供推力，翼身提供全部升力，飞行速度快，燃油效率高，适用于高速巡航；
3、模块化气动增效组件：侧翼隔离翼身上下表面气流，减少诱导阻力；小翼可以产生更多升力，提升飞行升阻比，利于节省燃油；
4、分离式舱体设计：翼身内置动力舱，运载舱独立悬挂；翼身底部留气流通道，降噪减震，提升舒适性、安全性，便于快速装卸货物。

具体实施方式
下面将结合附图中的实施例对本发明作进一步地详细说明。
如图1至图23所示，本发明的倾转多旋翼低空飞行器具体一个实施例。
如图1及图2所示，该实施例的倾转多旋翼低空飞行器，包括翼身2、至少两组并列设置在翼身2底部的旋翼组11及飞控系统（图中未示）；图中所示为优选方案，翼身2长度方向的前后分别设置一组共两组旋翼组11。
翼身2为固定翼气动外形，内部集成动力舱（图中未示），动力舱内安装航空发动机、变速器、液压泵、燃油箱、飞控系统等。
翼身2为翼身融合设计，即翼身2的外形按固定翼的气动外形设计。该飞行器属于低空低速飞行器，适合采用圆头尖尾特征的翼型。优选地，该固定翼的翼型选择Gottingen398 1919、Clark Y 1922、NACA2412、NACA4412中的一种。
如图2及图5所示，旋翼组11设有板架10，其两端对称设置一个旋翼1，当然在其他实施例中也可以是两个或者多个小尺寸的旋翼并列。旋翼组11通过板架10活动铰接于翼身2的底部，两端的旋翼1对称设置于翼身2的两侧。
如图3、图7、图11、图14所示，翼身2上设有顶推装置9连接板架10，飞控系统控制顶推装置9推动板架10旋转使两端的旋翼1在水平、倾斜以及竖直姿态之间转换。
该倾转多旋翼低空飞行器的翼身2采用固定翼的气动外形，飞行时翼身2产生向上的升力，相较于旋翼1产生升力，能极大地提高燃油经济性和飞行速度；翼身2内部集成动力舱，结构紧凑，飞行稳定性好。
翼身2两侧对称设置旋翼1，且两侧的旋翼1通过板架10连接成一个整体，即旋翼组11。旋翼组11两端的旋翼1的转向相反，相互抵消旋翼旋转产生的反扭矩。两侧的顶推装置9同步整体推动旋翼组11转动，确保旋翼倾转时姿态稳定，同时消除旋翼倾转时对翼身2两侧产生不平衡应力而导致翼身2变形的隐患，保证了飞行的安全性。
顶推装置9推动板架10旋转使两端的旋翼1在水平、倾斜以及竖直姿态之间转换，旋翼1处于水平状态适用于垂直起降和空中悬停，旋翼1倾斜状态适用于低速巡航，旋翼1竖直状态适用于高速巡航，从而实现该飞行器的多种飞行模式。
图3中所示的X方向，为飞行器的水平飞行方向。翼身2靠近X方向的部分称为机头/头部，远离X方向的部分称为机尾/尾部。机身长度方向中轴线从机尾到机头的方向称为纵向。约定人与飞行方向X同向观察时，左手边为所述飞行器左侧，右手边为所述飞行器右侧。
如图1及图2所示，翼身2的尾部表面沿纵向对称设有两个垂直安定面5，其后缘分别设有垂直舵8；两个垂直安定面5的上部之间设有水平安定面6，其后缘设有水平舵7；水平安定面6离翼身2上表面有合适的距离。
垂直安定面5、水平安定面6、垂直舵8、水平舵7组成垂直尾翼，其位于机尾。垂直安定面5及水平安定面6有利于确保飞行的稳定性，垂直舵8和水平舵7可以调整飞行姿态和飞行方向。
如图1所示，翼身2的下部设有支撑于降落面的起落架4。起落架4有利于飞行器垂直起降时支撑于降落面上。起落架4上设有运载舱3，其上表面与翼身2的底面之间设有间隙形成气流通道。
起落架4同时用于固定运载舱3，有利于简化飞行器的结构及增加强度。运载舱3的上表面和翼身2的下表面留有一定空隙距离，利于气流通过，以确保飞行器整体的气动性能。运载舱3与翼身2分离设计，动力舱内置于翼身2，即运载舱3与动力舱隔离，从而运载舱3内可以隔离动力舱内的噪声、震动和高低温，为运载舱3的乘客提供舒适的环境，也有利于方便装卸货物和提升安全性。
如图3、图7及图8所示，优选地，顶推装置9采用伺服电动推杆，其位于翼身2的左右两侧；顶推装置9一端旋转连接于翼身2的侧面，另一端活动铰接于板架10上。具体地，顶推装置9上部通过推杆固定轴12连接在翼身2侧部，下部通过推杆底座104连接在板架10上。
翼身2左右两侧的顶推装置9的推杆91伸缩推动板架10旋转，实现翼身2两侧的旋翼1在水平、倾斜以及竖直姿态之间转换，两侧的顶推装置9同步动作。
如图5及图11所示，板架10通过若干个铰接器101与翼身2的底部铰接。优选地，所述铰接器101的数量为3个。
如图5、图6及图7所示，铰接器101包括铰接座1011、铰接头1012及铰接轴1013，铰接座1011为底部平直的U型，铰接头1012通过铰接轴1013旋转连接于铰接座1011的U型结构之间；铰接座1011连接在翼身2的底部，铰接头1012连接在板架10上。
如图4所示，板架10包括两根平行设置的直杆102，中间通过若干连接板103连接固定；两根直杆102沿长度方向的内侧其中一侧设有液压高压油管105，另一侧设有液压低压油管106；直杆102两端外侧分别设有散热器107；铰接器101的铰接头1012具有贯穿孔，一根直杆102穿过其中并固定连接；板架10两端还连接有液压马达安装板108，用于安装旋翼1中的液压马达；还设有涵道支杆109安装旋翼1外围的涵道。
液压高压油管105及液压低压油管106的作用是将动力从翼身2内的动力舱传送到旋翼1上，散热器107及时就近的将管内液压油的热量散发掉，保证传动系统的稳定。
如图17及图20所示，翼身2的两侧设有侧翼13，其为薄板结构，外轮廓大于翼身2的外轮廓；具体的，侧翼13的外轮廓为翼身2侧截面轮廓线的等距外扩。翼身2两侧的侧翼13能有效隔离翼身2下表面的高压气流和上表面的低压气流，避免上下表面气流混合形成涡流产生诱导阻力。同时侧翼13还起到垂直安定面5同样的作用，有利于飞行的稳定性。
如图18及图20所示，翼身2的两侧设有小翼14，其外形为固定翼气动外形。优选地，其翼型与翼身2的翼型相同，侧截面的尺寸为翼身2的侧截面等比例缩小而得，比如为50%左右。翼身2两侧增加小翼14可以产生更多升力，提升飞行升阻比，利于节省燃油，同时也利于进一步提升飞行稳定性。
如图23所示，小翼14安装时设有迎角α。小翼14安装时保持一个适当的迎角有利于进一步提升升力。优选地，迎角α=5°～10°。
如图19所示，小翼14的腹部为内凹曲面结构。小翼14腹部的内凹曲面可以进一步提升气动性能。
如图20、图21及图22所示，小翼14安装于机身两侧，与侧翼13连接。
如图7、图9及图10所示，顶推装置9的推杆91长度最短时旋翼1处于水平姿态，旋翼1提供全升力，适用于垂直起降和空中悬停。
如图14、图15及图16所示，推杆91长度最长时旋翼1处于竖直姿态，旋翼1提供全部推力，翼身2提供全部升力，燃油效率显著提升，适用于高速巡航。
如图11、图12及图13所示，推杆91长度位于最短和最长之间时旋翼1处于不同角度的倾斜状态，旋翼1提供前飞推力和部分升力，翼身2提供部分升力，适用于低速巡航。
该倾转多旋翼低空飞行器通过控制顶推装置的推杆长度实现旋翼的状态调整，简单可靠，易于操控。
本发明的倾转多旋翼低空飞行器的另外一个实施例，板架10和顶推装置9可以容纳于翼身2内部，仅旋翼1外露。板架10和顶推装置9内置于翼身2内有益于减少飞行时的空气阻力，但需要做好翼身2的气密性设计。
在其他实施例中，侧翼13及小翼14的安装有所区别，具体是可以只装侧翼13不装小翼14；也可以只装小翼14不装侧翼13，按具体机型的性能和成本要求来定。
该发明的倾转多旋翼低空飞行器,通过旋翼组整体倾转结构消除了翼身两侧产生的不平衡应力，倾转过程中旋翼的姿态稳定，可以实现垂直起降、空中悬停、低速巡航及高速巡航等多种飞行模式；翼身融合设计显著提高了燃油经济性和飞行速度，增强了高速巡航能力。
