CN120327838A

Description

一种面向低空特性的飞行器及可伸展手臂协同抓取装置
技术领域
本发明涉及飞行设备技术领域，尤其是一种面向低空特性的飞行器及可伸展手臂协同抓取装置。
背景技术
无人机作为低空飞行载具，凭借其高机动性与智能化作业能力，在物流领域逐渐成为革新传统供应链体系的关键技术载体。其核心应用价值体现在以下两方面：
其一，地理适应性突破。依托垂直起降(VTOL)与多旋翼/固定翼混合构型设计，无人机可克服复杂地形限制，通过GNSS导航与路径规划算法实现山区、海岛及灾区的应急物资精准投送，显著提升偏远区域物流可达性。
其二，末端配送效率重构。基于自动避障系统与视觉识别技术，无人机可执行城市“最后一公里”配送任务，通过预设坐标或动态定位将包裹投递至用户指定接收点(如智能快递柜、建筑露台等)，结合自动化分拣系统形成端到端闭环，降低人力成本的同时优化城市交通资源配置。
中国专利文献CN119284173A公开了一种适用于无人机抓取物流货物的模块化抓取装置及方法，无人机本体的下方水平安装有呈“X”字形的折叠架，夹持套筒的外侧固定安装有夹爪；支撑架呈“V”字形，支撑架的旋转中心处设置有弧形夹板；悬空挡板下表面设置有吸盘，悬空挡板的上方设置有驱动组件，驱动组件驱动折叠架折叠、并同时对吸盘内腔抽真空；有益效果为：通过设置有呈“X”字形的折叠架，夹持杆下端安装有夹爪，弧形夹板共计两个可从两侧对小尺寸的货物进行抓取，对于表面粗糙的货物，利用摩擦力直接夹持抓取，对于表面光滑的货物，利用吸盘真空吸附进行辅助抓取，从而自适应不同类型的货物，增大了本装置的适用范围。
现有的飞行器及可伸展手臂协同抓取结构主要不足体现在以下几个方面：第一是动态稳定性不足，飞行器在搭载可伸展机械臂时，由于负载分布随臂部运动不断变化，导致重心持续偏移，严重影响飞行姿态稳定性，且机械臂在伸展状态下，飞行器抗风扰能力下降，易受外界气流干扰产生剧烈晃动，然而传统控制算法难以实时补偿机械臂运动带来的力矩扰动，导致飞行器出现振荡甚至失控现象，在抓取瞬间，机械臂与目标物的接触冲击会进一步破坏飞行平衡，增加系统失稳风险。第二是抓取机构适应性局限，现有抓取装置难以兼顾负载能力与操作灵活性，刚性机械爪对不规则物体的适应性差，而软体抓手又存在承载能力不足的问题，并且抓取机构缺乏有效的力觉反馈机制，无法实时感知抓取力度，容易造成目标物滑脱或损伤。面对不同材质、形状的物体时，需要人工更换末端执行器，严重影响作业效率。在动态抓取场景下，传统抓取机构响应速度不足，难以准确捕捉运动目标。第三是当前机械臂的结构设计普遍存在折叠能力不足的问题，大多数机械臂采用固定式或半折叠式设计，在非工作状态下仍保持较大体积，不仅占用运输空间，特别是在空间受限的作业环境中，还增加了设备搬运的难度，严重制约了移动作业平台的机动性，使得飞行器难以实现快速响应和灵活部署。
发明内容
本发明克服了现有技术中的缺点，提供一种面向低空特性的飞行器及可伸展手臂协同抓取装置，稳定性好、强度高，收缩后体积大幅减小，适应狭窄空间运输存储；协同抓取机结构中，机械手指仿生曲面与凹凸织构增加摩擦力，镂空减重减少负载，力反馈传感器实现自适应抓取。
为了解决上述技术问题，本发明是通过以下技术方案实现的：
一种面向低空特性的飞行器及可伸展手臂协同抓取装置，包括：
飞行器部分，包括躯干构件及连接的若干个旋翼结构；
可伸展手臂部分，包括若干个基于折纸思想设计的模块化折展单元，所述模块化折展单元包括上下设置的第一机架构件以及第二机架构件，第一机架构件与第二机架构件之间连接有作为第一驱动机构的随动支链结构；相邻模块化折展单元通过对应的第一机架构件与第二机架构件的刚性连接，形成可折叠与展开的桁架式结构；
协同抓取机结构，包括手掌机构和四个对称分布的手指机构，所述手掌机构上的蜗轮蜗杆传动驱动所述手指机构实现单自由度同步开合；
其中，所述飞行器部分设置有舱体，所述可伸展手臂部分在收缩状态下可完全收纳于封闭舱体内；所述协同抓取装置固定于可伸展手臂部分的末端，形成低空抓取作业的协同系统。
更进一步地，所述随动支链结构包括与第一机架构件转动连接的第一板体，以及与第二机架构件转动连接的第二板体，第一板体的两侧转动连接有第三板体，第三板体转动连接有第四板体，第四板体与第二板体转动连接；从而令随动支链结构实现多自由度折叠与展开。
更进一步地，所述随动支链结构是对称连接在第一机架构件以及第二机架构件两侧；所述舱体可转动设置有闭合板，并在非工作状态下通过闭合板封闭舱体，降低空气阻力。
更进一步地，所述手掌机构包含架体，架体与位于最下端的模块化折展单元连接，架体上安装有第二驱动机构，第二驱动机构的输出轴与蜗杆实现传动连接；所述手指机构由若干组机械手指构成，若干组机械手指沿蜗轮轴线呈等距圆周分布设置，每个所述机械手指均与一个蜗轮固定连接，蜗轮与蜗杆相互啮合，共同构成单自由度平面耦合运动副；
当所述第二驱动机构驱动蜗杆旋转时，蜗杆带动与之啮合的蜗轮同步转动，进而控制若干组机械手指绕所述蜗轮轴线同步展开或收拢，形成环状包络面，以此实现对物体的抓取动作。
更进一步地，所述机械手指的夹持面为仿生曲面构型，表面经超声振动加工形成微米级凹凸织构；所述机械手指本体设有镂空减重结构，以降低整体质量。
更进一步地，所述机械手指与对应蜗轮通过径向键槽配合实现周向定位，配合界面设有限位销进行轴向约束。
更进一步地，所述机械手指设置有力反馈传感器，力反馈传感器与第二驱动机构电性连接，力反馈传感器实时感知抓取力度并调整夹持状态，实现自适应抓取。
更进一步地，所述模块化折展单元的数量是4至12个。
更进一步地，所述可伸展手臂部分在展开状态下的扩展比为1:10，收缩后体积减少至展开状态的10％，以适应狭窄空间运输与存储需求。
更进一步地，所述飞行器部分与可伸展手臂部分的连接处设置缓冲机构，用于吸收抓取瞬间的冲击力，维持飞行姿态稳定性。
与现有技术相比，本发明的有益效果是：
本发明所述飞行器部分及可伸展手臂部分协同抓取装置采用了轻量化桁架式结构，基于三角形稳定原理优化受力分布，在保证结构刚度的同时显著降低整体重量，能适应于低空飞行抓取和冗余操作的应用要求。
本发明所述飞行器部分及可伸展手臂部分协同抓取装置采用了折纸思想设计，可实现展开和收缩两种状态，且两面均有相同的随动支链结构，可实现向左向右方向的运动，同时具有良好的折展比，有利于运输与存储。
本发明所述飞行器部分及可伸展手臂部分协同抓取装置采用蜗轮蜗杆传动的单自由度多指机械抓手，其单自由度设计简化了控制系统架构，每个机械抓手的手指仅需单个驱动电机即可实现同步开合运动。
本发明所述飞行器部分及可伸展手臂部分协同抓取装置的机械臂采用模块化设计，使系统更易于扩展和维护，通过拆除和增加模块单元的数目来控制装置的总长度使得该机械臂机构在展开的过程中，适用于不同的工况，满足抓取未知目标的实际需求，并实现一机多用。

具体实施方式
以下结合附图对本发明的优选实施例进行说明，应当理解，此处所描述的优选实施例仅用于说明和解释本发明，并不用于限定本发明。
如图1至图3所示，本发明公开一种面向低空特性的飞行器及可伸展手臂部分协同抓取装置，包括飞行器部分1、可伸展手臂部分部分2以及协同抓取机结构3，其中飞行器部分1包括躯干构件101及连接的若干个旋翼结构102；与无人机工作原理类似，旋翼结构102启动带动整个躯干构件101运动；躯干构件101作为飞行器部分1的主体，为其他部件提供安装基础。若干个旋翼结构102对称分布在躯干构件101周围，旋翼结构102是由电机以及旋翼组成，通过电机驱动旋翼旋转，为飞行器提供升力和飞行所需的动力。另外，飞行器部分1设置有舱体103，该舱体103用于收纳可伸展手臂部分部分2。舱体103可转动设置有闭合板104，在非工作状态下，闭合板104转动将舱体103封闭，这样可以降低飞行器在飞行过程中的空气阻力，提高飞行效率。
如图4至图8所示，可伸展手臂部分部分2包括若干个基于折纸思想设计的模块化折展单元201，模块化折展单元201的数量为4至12个，优选是6个。每个模块化折展单元201包括上下设置的第一机架构件2011以及第二机架构件2012，第一机架构件2011与第二机架构件2012之间连接有作为第一驱动机构的随动支链结构2013。其中，随动支链结构2013包括与第一机架构件2011转动连接的第一板体20131，以及与第二机架构件2012转动连接的第二板体20132，第一板体20131的两侧转动连接有第三板体20133，第三板体20133转动连接有第四板体20134，第四板体20134与第二板体20132转动连接；随动支链结构2013对称连接在第一机架构件2011以及第二机架构件2012两侧，从而令稳定性好，强度高；从而令随动支链结构2013实现多自由度折叠与展开，当第一驱动机构工作时，通过改变各板体之间的相对转动角度，实现模块化折展单元201的折叠与展开动作。
相邻模块化折展单元201通过对应的第一机架构件2011与第二机架构件2012的刚性连接，形成可折叠与展开的桁架式结构。可伸展手臂部分部分2在收缩状态下可完全收纳于封闭舱体103内，在展开状态下的扩展比为1:10，收缩后体积减少至展开状态的10％，这种特性使得该可伸展手臂部分部分2能够适应狭窄空间运输与存储需求。
如图9至图12所示，协同抓取机结构3包括手掌机构301和四个对称分布的手指机构302，协同抓取装置固定于可伸展手臂部分部分2的末端。其中，手掌机构301包含架体3011，架体3011与位于最下端的模块化折展单元201连接，架体3011上安装有第二驱动机构3012，第二驱动机构3012是伺服电机，第二驱动机构3012的输出轴与蜗杆3013实现传动连接。手指机构302由若干组机械手指3021构成，若干组机械手指3021沿蜗轮3014轴线呈等距圆周分布设置，每个机械手指3021均与一个蜗轮3014固定连接，蜗轮3014与蜗杆3013相互啮合，共同构成单自由度平面耦合运动副。从图11看出，机械手指3021与对应蜗轮3014通过径向键槽3022配合实现周向定位，配合界面设有限位销进行轴向约束，确保机械手指3021与蜗轮3014之间的连接稳定可靠。
当第二驱动机构3012驱动蜗杆3013旋转时，蜗杆3013带动与之啮合的蜗轮3014同步转动，进而控制若干组机械手指3021绕蜗轮3014轴线同步展开或收拢，形成环状包络面，以此实现对物体的抓取动作。
此外，机械手指3021的夹持面为仿生曲面构型，表面经超声振动加工形成微米级凹凸织构，这种设计可以增加机械手指3021与被抓取物体之间的摩擦力，提高抓取的稳定性。同时，机械手指3021本体设有镂空减重结构，以降低整体质量，减少可伸展手臂部分部分2的负载。
机械手指3021设置有力反馈传感器，力反馈传感器与第二驱动机构3012电性连接。在抓取过程中，力反馈传感器实时感知抓取力度，并将信号反馈给第二驱动机构3012，第二驱动机构3012根据反馈信号调整夹持状态，实现自适应抓取，避免因抓取力过大损坏物体或抓取力过小导致物体掉落。
另外，在飞行器部分1与可伸展手臂部分部分2的连接处设置缓冲机构，当协同抓取机结构3进行抓取动作时，会产生瞬间的冲击力，缓冲机构可以吸收这些冲击力，维持飞行器的飞行姿态稳定性，确保飞行器在抓取作业过程中的安全和稳定；缓冲机构可以是采用弹簧缓冲器。
本发明的工作原理是：在非工作状态下，可伸展手臂部分部分2收缩并收纳于飞行器部分1的舱体103内，舱体103的闭合板104关闭，降低空气阻力。当需要进行低空抓取作业时，飞行器部分1飞行至目标位置，打开舱体103的闭合板104，可伸展手臂部分部分2的随动支链结构2013展开，使可伸展手臂部分部分2从舱体103中伸出并展开至工作状态。然后，飞行器调整姿态，使协同抓取机结构3对准目标物体，第二驱动机构3012驱动蜗杆3013旋转，带动蜗轮3014转动，使机械手指3021同步展开或收拢，实现对目标物体的抓取。在抓取过程中，力反馈传感器实时监测抓取力度，确保抓取的稳定性和安全性。抓取完成后，可伸展手臂部分部分2收缩，重新收纳于舱体103内，飞行器返回。
本发明中可伸展手臂部分部分2基于折纸思想的模块化折展单元201，稳定性好、强度高，收缩后体积大幅减小，适应狭窄空间运输存储；协同抓取机结构3中，机械手指3021仿生曲面与凹凸织构增加摩擦力，镂空减重减少负载，力反馈传感器实现自适应抓取。
飞行器部分1的舱体103的闭合板104可降低飞行时空气阻力，提高效率，使可伸展手臂部分部分2在收缩状态下完全收纳到舱体103内，减小外挂式设计带来的空气阻力增大的问题，这种设计可降低风阻系数，显著提高飞行速度与续航能力，在高速机动或复杂气流环境下，内置可伸展手臂部分部分2不会产生额外的气动干扰，使飞行器部分1能保持更好的姿态稳定性。同时，全封闭式装配也为可伸展手臂部分部分2提供了全方位的物理防护，有效避免雨水、沙尘等环境因素对可伸展手臂部分部分2的侵蚀，也可避免可伸展手臂部分部分2与障碍物发生意外碰撞。除此之外，可伸展手臂部分部分2具有良好的折展比，有利于运输与存储；同时，可伸展手臂部分部分2采用模块化的设计方式，有利于在复杂的环境中根据不同工况来改变其模块数量以达到适合的工作状态，特别是进入狭窄或障碍物较多的区域进行操作。最后，可伸展手臂部分部分2机构使用三角形桁架式结构，在确保高刚度的同时，其开放式设计提供了充裕的内部空间用于走线和传感器布置，并通过铰接节点实现多自由度折叠与展开。配合末端执行器，集成自适应抓取和力反馈功能，使系统兼具结构稳定性和操作灵活性，尤其适用于空间非合作目标的精准抓取任务。
