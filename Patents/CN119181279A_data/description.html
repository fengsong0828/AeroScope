Description

一种低空飞行器的管理系统
技术领域
本发明涉及低空经济深度发展下的精密轨道交通管理，尤其涉及一种低空飞行器的管理系统。
背景技术
低空飞行器，尤其是无人机的普及与应用的广泛化，给低空飞行器管理系统带来了前所未有的挑战与技术不足。现有的低空飞行器管理系统在实时性与精度上存在局限，难以实现对大量无人机的高密度、高速度飞行的有效监控，尤其是在复杂的城市环境中，系统的反应速度与数据处理能力往往跟不上无人机的飞行速度与变化频率，导致监控盲区与安全隐患。
其次，低空飞行器管理系统在数据融合与信息共享上存在瓶颈，不同系统、不同平台间的信息交互与协同作业能力较弱，难以形成全面、统一的低空飞行态势感知；当前的系统在自主决策、动态路径规划、应急避障等方面的能力有限，难以适应复杂多变的低空飞行环境。
低空飞行器管理系统的现有技术不足主要体现在实时性与精度、数据融合与数据快速处理等方面，这些问题的存在严重制约了低空飞行器管理系统的效能与安全性，亟需通过技术创新突破加以解决。
发明内容
为实现上述目的，本发明提供了一种低空飞行器的管理系统，系统包括空间图像采集单元、多源融合管控单元、多模态信号再生单元、控制平台，其中：
空间图像采集单元，用于获取飞行器管控相关数据，飞行器管控相关数据包括飞行器周边预设范围内的环境信息、空间图像信息、空间电磁图谱信息以及飞行器信息；
多源融合管控单元与空间图像采集单元通讯连接，用于接收飞行器管控相关数据，并对飞行器管控相关数据进行协议转换，并将经过协议转换的飞行器管控相关数据发送至控制平台；
多模态信号再生单元，基于控制平台发送的模拟信号配置参数生成对应模式的模拟信号，模拟信号用以控制飞行器执行控制平台生成的飞行器管理策略；
控制平台，用于接收经过协议转换的飞行器管控相关数据，并对飞行器管控相关数据进行数据分类，至少获得第一类型数据和第二类型数据。
其中，控制平台基于第一类型数据，通过人工神经网络完成多基站的临近覆盖区域电磁特性估计和临近空间电磁覆盖模型构建，以建立飞行器周边预设范围内各个探测基站的电磁信号特性与多模态信号再生单元的不同信号再生模式的对应关系，训练获得飞行器当前所处环境下多模态信号再生单元各个模式的模拟信号配置参数；第一类型数据包括空间电磁图谱信息和探测基站的位置数据。
控制平台基于第二类型数据生成飞行器管理策略，并基于飞行器管理策略向多模态信号再生单元发送对应模式的模拟信号配置参数。
优选的，多模态信号再生单元包括第一再生模式、第二再生模式和第三再生模式，其中：
第一再生模式下，多模态信号再生单元请求信号收发单元根据对应模式的模拟信号配置参数生成分时异步信号；
第二再生模式下，多模态信号再生单元请求信号收发单元根据对应模式的模拟信号配置参数生成共频增强信号；
第三再生模式下，多模态信号再生单元请求信号收发单元根据对应模式的模拟信号配置参数生成宽频瞬时脉冲信号。
信号收发单元包括以下装置一种或多种：地基增强伪卫星，信号基站，GNSS卫星等。
优选的，控制平台基于空间电磁图谱信息向多模态信号再生单元发送对应再生模式的模拟信号配置参数，包括：
响应于飞行器周边预设范围内电磁频段固定且与定位频段共存，控制平台向多模态信号再生单元发送对应第一再生模式的模拟信号配置参数，以触发多模态信号再生单元请求信号收发单元生成分时异步信号；
响应于飞行器周边预设范围内电磁信号完全遮蔽，控制平台向多模态信号再生单元发送对应第二再生模式的模拟信号配置参数，以触发多模态信号再生单元请求信号收发单元生成共频增强信号；
响应于飞行器周边预设范围内电磁环境覆盖的频谱范围大于预设阈值，控制平台向多模态信号再生单元发送对应第三再生模式的模拟信号配置参数，以触发多模态信号再生单元请求信号收发单元生成宽频瞬时脉冲信号。
优选的，控制平台还对第二类型数据进行归一化处理以转换成人工神经网络可识别的参量，第二类型数据包括空间图像信息和飞行器信息，空间图像信息包括多源数据量测的目标飞行器位置信息、目标飞行器姿态信息以及异物信息，飞行器信息包括飞行器预设的飞行器位置信息和姿态信息；
控制平台基于人工神经网络，构建基于多源数据量测的目标飞行器位置信号与位置置信度之间的映射关系，由此确定目标飞行器位置信息的置信水平；
控制平台响应于置信水平高于预设置信阈值，控制平台基于第二类型数据中目标飞行器位置信息生成飞行器管理策略。
优选的，控制平台还用于根据目标飞行器位置信息判断飞行器是否进入预设警戒区域，响应于飞行器进入预设警戒区域，控制平台制定飞行器安全驱离管理策略。
在飞行器安全驱离管理策略下，多模态信号再生单元向飞行器发送干扰信号，以屏蔽飞行器的GNSS信号接收，并且，多模态信号再生单元向飞行器发送诱导信号，诱导飞行器转向指定区域。
优选的，控制平台响应于飞行器未进入预设警戒区域，且，飞行器偏离预设飞行轨迹，控制平台制定飞行器路径纠偏管理策略；
在飞行器路径纠偏管理策略下，多模态信号再生单元首先向飞行器发送路径提醒信号，以触发飞行器主动纠偏；
响应于超出预设提醒信号发送次数，飞行器仍偏离预设飞行轨迹，多模态信号再生单元发送纠偏信号以定向导航纠偏，迫使飞行器修正回到原有航线。
优选的，在飞行器纠偏管理策略下，响应于定向导航纠偏迫使飞行器修正回到原有航线后，多模态信号再生单元控制地基增强伪卫星和GNSS组合对飞行器进行导航。
优选的，第二类型数据还包括环境信息，控制平台基于第二类型数据中目标飞行器所处环境信息生成对应的飞行器管理策略；环境信息包括以下类型信息中的一种或多种：温度信息、湿度信息、气流信息、空气质量。
优选的，控制平台还用于制定飞行器应急避险管理策略；在应急避险管理策略下，控制平台建立特情任务，控制平台根据特情设定避险区域和新规划路径；
当飞行器按新规划路径飞行时，控制平台实时监听飞行器路线状态，直至飞行器进入预定位置，特情结束；
若飞行器未进入新的规划路径，则控制平台控制地基增强伪卫星对飞行器进行导航，强制飞行器进入新规划路径直至特情结束，或者直至飞行器抵达避险区域。
优选的，空间图像采集单元基于以下监控数据源获取飞行器管控相关数据，监控数据源包括：区域探测雷达、信号探测雷达和高清摄像头；
区域探测雷达通过多站协同，实时探测雷达覆盖区域内的飞行器及其它异物的位置；
信号探测雷达用于获取飞行器周边预设范围内的空间电磁图谱信息，包括电磁信号频率、带宽、强度；
高清摄像头，用于获取空间图像信息，包括飞行器实体信息、姿态信息、以及飞行器周边预设范围内其它异物的信息。
本发明提供的一种低空飞行器的管理系统，所带来的的有益效果为：通过多源融合对飞行器通导感控进行管理，为当前低空经济发展提供了一种可行性好、成熟度高、通用性强的集成化飞行管理机制，为低空的安全保障提供了一种有效的解决方案。

一种示例中，如图2所示，控制平台13对飞行器管控相关数据进行数据分类和分析处理，至少获得第一类型数据和第二类型数据。其中，第二类型数据可以包括飞行器位置信息、空间环境信息。如图3所示，控制平台13能够基于第二类型数据生成飞行管理策略。第一类型数据至少包括空间电磁图谱信息，控制平台13可以基于飞行器管控相关数据中的空间电磁图谱信息，通过人工神经网络完成多基站的临近覆盖区域电磁特性估计和临近空间电磁覆盖模型构建，以建立目标飞行器周边预设范围内探测基站的电磁信号特性与多模态信号再生单元12多种信号再生模式之间的对应关系。
多模态信号再生单元12具有多种信号再生模式，从而多模态信号再生单元12可以对模拟信号的信号再生模式进行配置。探测基站的电磁信号特性与多模态信号再生单元12多种信号再生模式之间的对应关系是指：控制平台13根据目标飞行器当前所处区域的探测基站的电磁信号特性，控制多模态信号再生单元12选用对应的信号再生模式(具体对应关系在后阐述)。
控制平台13还可以利用多模态信号再生单元12对模拟信号的配置参数进行配置，例如，模拟信号的配置参数可以包括：包括模拟信号的频段配置参数、带宽配置参数、强度配置参数等。具体的，针对目标飞行器临近区域的空间电磁图谱信息，控制平台13向多模态信号再生单元12发送对应的模拟信号配置参数，多模态信号再生单元12根据该模拟信号配置参数向目标飞行器发送模拟信号。其中，多模态信号再生单元12根据该模拟信号配置参数向目标飞行器发送模拟信号是指：由多模态信号再生单元12发送信号，并最终由目标飞行器接收。例如，可以是多模态信号再生单元12直接向目标飞行器发送符合模拟信号配置参数的模拟信号，或者，还可以是多模态信号再生单元12通过请求信号收发单元向目标飞行器发送符合模拟信号配置参数的模拟信号，或者，通过上述两种方式的组合实现。在本申请实施例中，选择以多模态信号再生单元12向请求信号收发单元的方式，间接向目标飞行器发送符合模拟信号配置参数的模拟信号。
根据以上说明，管理系统通过空间图像采集单元10，获得包括目标飞行器周边的电磁图谱信息、位置信息和空间环境信息的多种实时信息，为后续的分析提供了全面的数据支撑；通过对上述多种信息的分析处理，可以获得第一类型数据和第二类型数据，控制平台13可以基于第二类型数据生成对目标飞行器的飞行管理策略，为飞行器的管理提供了有力的策略支持，并且，控制平台13还可以基于第一类型数据对模拟信号的配置参数进行配置，生成适应目标飞行器周边预设范围电磁特性的模拟信号，优化了飞行器与管理系统的通信质量，确保飞行器高效、安全地执行飞行任务，避免了复杂空间对飞行器的干扰，实现了对飞行器多源融合的管控。
作为一种可选的实现方式，空间图像采集单元10基于以下监控数据源中的一种或多种获取飞行器管控相关数据，监控数据源包括：区域探测雷达、信号探测雷达和高清摄像头；

高清摄像头，用于获取空间图像信息，空间图像信息可以包括飞行器实体信息、姿态信息、以及飞行器周边预设范围内其它异物的信息。
具体地，空间图像采集单元10可以从多种监控数据源获取飞行器管控相关数据，监控数据源可以包括区域探测雷达、信号探测雷达和高清摄像头，以全面地采集飞行器管控所需要的相关数据。区域探测雷达通过多站协同工作，能够实时监测雷达覆盖范围内的目标飞行器及其他各类异物的精确位置，提供目标飞行器周边预设范围内连续的空间态势感知；信号探测雷达通过捕捉目标飞行器周边预设范围内的空间电磁图谱，详细地记录了电磁信号的频率、带宽和强度，为目标飞行器的通信状态和潜在干扰源分析提供数据支持；高清摄像头负责捕捉高分辨率的空间图像信息，空间图像信息可以包括目标飞行器自身的实体信息和姿态信息，还包括目标飞行器周边预设范围内其他异物的信息，从而为目标飞行器的视觉识别和环境感知提供直观依据。
空间图像采集单元10通过上述多监控数据源融合的方式，增强飞行器管控相关数据准确性的同时，为目标飞行器飞行安全、飞行路径规划和紧急情况响应等关键操作提供了全面、细致的数据支撑，提升了目标飞行器飞行任务的执行效率和安全性。

第二再生模式下，多模态信号再生单元12请求信号收发单元根据对应模式的模拟信号配置参数生成共频增强信号；

如图7所示，若目标飞行器未按照新的规划路径飞行，控制平台13通过地基增强伪卫星向目标飞行器发送精确的导航信号，强制目标飞行器进入新规划路径。地基增强伪卫星的参与确保了控制平台13强制控制飞行器驶向新规划路径，管理系统实时监测飞行器路径调整情况，直至目标飞行器成功进入新规划路径，从而保障特情任务的圆满结束。
上述飞行器应急避险管理策略融合了多源数据的采集与处理、控制平台13对飞行器的实施监控与动态路径规划以及精确导航纠偏，构建了高效、安全的的飞行器应急避险管理策略，提升了飞行器在执行飞行任务的灵活性与安全性，为飞行器在紧急情况下的安全飞行提供了有力保障。
