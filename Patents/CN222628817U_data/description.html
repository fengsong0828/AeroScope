Description

一种低空飞行器透明件
技术领域
本实用新型属于飞行汽车部件技术领域，涉及一种低空飞行器透明件。
背景技术
目前航空透明件及低空飞行器透明件基本采用板材热弯、模压或者吸塑成型，在通过CNC进行周圈加工，最后进行表面抛光处理，这种工艺复杂，要求严格，耗时长，效率低。
实用新型内容
本实用新型所要解决的技术问题，是针对现有技术的现状，而提供一种耐刮擦、耐候性好，缩减优化了生产制程，大幅的降低生产成本，提升产品的生产效率及合格率的低空飞行器透明件。
本实用新型解决上述技术问题所采用的技术方案为：一种低空飞行器透明件，其特征在于，包括有基材本体、UR层以及复合膜片层，基材本体、PUR层以及复合膜片层在模内一体成型，所述的复合膜片层在模内形成在基材本体的内侧面上，PUR层在模内形成在基材本体的外表面上。
作为其中一种方式，在上述的一种低空飞行器透明件中，所述的基材本体为采用PC材料制成，所述的复合膜片层为PC层和PMMA层。
作为另一种方式，在上述的一种低空飞行器透明件中，所述的基材本体为采用PC材料制成，所述的复合膜片层依次为PC层、硬化层、防反射层以及保护层，所述的硬化层形成在PC层的内侧面上，防反射层位于硬化层和保护层之间。
作为第三种方式，在上述的一种低空飞行器透明件中，所述的基材本体为采用PC材料制成，所述的复合膜片层依次包括保护涂层、导电涂层、PC层、硬化层、防反射层以及保护层，所述的硬化层形成在PC层的内侧面上，防反射层位于硬化层和保护层之间，保护涂层与PC层之间还设置有导电涂层。
作为第四种方式，在上述的一种低空飞行器透明件中，所述的复合膜片层依次包括有PC层、第一粘接层、调光层、第二粘接层、PMMA层、硬化层、防反射层以及保护层，所述的调光层位于两个粘接层之间，PMMA层通过第一粘接层粘接在调光层的内侧面，在PMMA层的内侧面上依次形成有硬化层、防反射层以及保护层。
与现有技术相比，本实用新型的优点在于使得基材本体的韧性及耐热性大大的提升，能够有效的满足鸟撞实验；通过膜内的反应成型、复合膜片或者硬化涂层，能够有效的提升耐刮擦及耐候性；在工艺上复合膜片及膜内成型能够在模具上实现一步成型，缩减优化了生产制程，大幅的降低生产成本，也能提升产品的生产效率及合格。

如图1和图2所示，本低空飞行器透明件，包括有基材本体100、PUR层200以及复合膜片层1003，基材本体100、PUR层200以及复合膜片层1003在模内一体成型，复合膜片层1003在模内形成在基材本体100的内侧面上，PUR层200在模内形成在基材本体100的外表面上，这里复合膜片层1003成型能够在模具上实现一步成型，缩减优化了生产制程，大幅的降低生产成本，也能提升产品的生产效率及合格率，来提高透明件的韧性及耐热性，能够有效的满足鸟撞实验，这里基材本体100为采用PC材料制成，复合膜片层1003为PC层1001和PMMA层300，通过PUR层200可以省去做各种油漆层，基材本体100采用PC材料，第一射通过复合膜片层1003进行IML工艺膜内注塑，第二射通过膜内反应在外表面形成PUR保护介质，通过模具一步成型，实现耐刮擦及耐候性要求。
如图2所示，作为第2种方式，基材本体100为采用PC材料制成，所述的复合膜片层1003依次为PC层1001、硬化层400、防反射层500以及保护层600，硬化层400形成在PC层1001的内侧面上，防反射层500位于硬化层400和保护层600之间，通过硬化层400增强整个低空飞行器透明件的硬度，同时防反射层500可以提升产品透光性能。
如图3所示，作为第3种方式，基材本体100为采用PC材料制成，复合膜片层1003依次包括保护涂层1002、导电涂层700、PC层1001、硬化层400、防反射层500以及保护层600，硬化层400形成在PC层1001的内侧面上，防反射层500位于硬化层400和保护层600之间，保护涂层1002与基材本体100之间还设置有导电涂层700，这里通过硬化层400增强整个低空飞行器透明件的硬度，同时防反射层500可以提升产品透光性能，导电涂层700可以实现风挡加热需求，满足除冰除雾性能。
如图4所示，作为第4种方式，复合膜片层1003依次包括有PC层1001、第一粘接层900、调光层800、第二粘接层1000、PMMA层300、硬化层400、防反射层500以及保护层600，调光层800位于两个粘接层之间，PMMA层300通过第一粘接层900粘接在调光层800的内侧面，在PMMA层300的内侧面上依次形成有硬化层400、防反射层500以及保护层600，这里调光层800可以提升产品透光性能，也能实现透明件明暗效果调节功能，同时防反射层500可以提升产品透光性能，实现表面耐刮擦及耐候要求。
