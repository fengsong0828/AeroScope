Description

一种低空飞行器认证方法、装置、终端及存储介质
技术领域
本发明涉及航空通信与无线通信领域，尤其涉及的是一种低空飞行器认证方法、装置、终端及存储介质。
背景技术
在当前的航空领域，用于低空飞行器的定位和身份验证的常见方法包括使用GPS（全球定位系统）或类似的卫星定位系统来获取飞行器的位置信息。飞行器通常会配备GPS接收器，以接收来自卫星的位置信号，并据此确定其当前位置、速度和航向等信息。另外，飞行器通常也会使用通信设备，如ADS-B（Automatic Dependent Surveillance-Broadcast，广播式自动相关监视）系统，来向地面控制台发送位置信息。ADS-B系统能够实现飞行器之间的自动广播和接收，使得飞行器能够相互检测和跟踪，并向地面控制台发送位置和身份信息。
然而，现有的上述方法缺乏对低空飞行器的位置和身份的有效验证，存在低空飞行器身份伪造或位置误报的风险，导致通信安全受到影响。
因此，现有技术还有待改进和发展。
发明内容
本发明要解决的技术问题在于，针对现有技术的上述缺陷，提供一种低空飞行器认证方法、装置、终端及存储介质，旨在解决现有技术缺乏对低空飞行器的位置和身份的有效验证，存在低空飞行器身份伪造或位置误报的风险，导致通信安全受到影响的问题。

当第二验证结果为通过时，本实施例通过二级信息验证模块接收一级信息验证模块发送过来的目标飞行器的飞行器位置信息和通感一体基站发送的通感一体基站数据，对飞行器位置信息和通感一体基站数据进行数据清洗，保证数据形式的一致性；采用信息比较算法对飞行器位置信息和通感一体基站数据进行比较和分析，检查位置信息、速度等参数是否一致，以确保对目标飞行器身份验证和位置验证的准确性和有效性。
在一种实现方式中，基于所述通感一体基站数据对所述飞行器位置信息进行验证，确定第三验证结果，包括：
步骤S301、根据所述通感一体基站数据和所述飞行器位置信息确定所述目标飞行器对应的偏离误差；
步骤S302、获取偏离误差阈值，根据所述偏离误差阈值和所述偏离误差确定第三验证结果。
简单来说，当目标飞行器回报的飞行器位置信息和通感一体基站数据中的位置信息之间的差值较小时，则判断目标飞行器回报的飞行器位置信息是准确地、未被篡改过的。本实施例中根据通感一体基站数据中的位置信息和飞行器位置信息来计算目标飞行器对应的偏离误差，并将偏离误差和偏离误差阈值对比，得到第三验证结果。如果偏离误差小于或等于偏离误差阈值，则第三验证结果为通过；如果偏离误差大于偏离误差阈值，则第三验证结果为不通过。本实施例通过根据通感一体基站数据和飞行器位置信息计算偏离误差，从而根据偏离误差确定第三验证结果，能够有效的检验出如信息被窃取或篡改，从而导致身份伪造或位置误报的问题或者飞行器基于GPS定位时，受到信号干扰或遮挡，导致回报的飞行器位置信息的准确性和完整性受到影响的问题。
在一种实现方式中，所述偏离误差阈值的确定方法包括：
获取若干历史偏离误差，根据所述若干历史偏离误差确定偏离误差均值和偏离误差标准差；
获取实时环境数据，根据所述实时环境数据、所述偏离误差均值以及所述偏离误差标准差确定所述偏离误差阈值。
具体地，本实施例通过获取历史偏离误差，根据历史偏离误差计算偏离误差均值和偏离误差标准差，通过偏离误差均值和偏离误差标准差来衡量目标飞行器的偏离误差是否在正常范围内。同时，为了能够更准确的评估目标飞行器的位置信息，通过在偏离误差阈值的计算过程中目标飞行器当前时刻所在位置的实时环境数据（大风、暴雨、阴天、晴天等），计算得到不同的偏离误差阈值。
在一种实现方式中，根据所述实时环境数据、所述偏离误差均值以及所述偏离误差标准差确定所述偏离误差阈值，包括：，为所述偏离误差阈值，为所述偏离误差均值，为调节系数，为所述偏离误差标准差，为基于所述实时环境数据确定的修正项，还可基于实时偏离误差与历史偏离误差的对比结果动态调整。
在一种实现方式中，引入目标飞行器对应的机型（如无人机、民航飞机、直升机等）参数来设置调节系数。目标飞行器的机型可以根据目标飞行器对应的飞行器位置信息或者标准飞行器参数确定。
在一种实现方式中，所述方法还包括：
当所述第二验证结果或所述第三验证结果为不通过时，对所述飞行器位置信息进行分析，确定所述目标飞行器对应的异常行为模式；
根据所述异常行为模式对所述目标飞行器进行异常处理。
具体地，当第二验证结果不通过时，一级处理验证模块将飞行器位置信息发送给异常处理模块或者当第三验证结果不通过时，二级信息验证模块将飞行器位置信息发送给异常处理模块。异常处理模块接收到飞行器位置信息后，利用机器学习和数据挖掘技术对飞行器位置信息进行分析，检测出可能存在异常行为模式，如目标飞行器偏航、目标飞行器标识信息不正确、位置信息不完整等，根据检测到的异常行为模式，进行相应的异常处理，包括发送警报信息、调整目标飞行器的检测策略等，从而进一步提高飞行器定位的准确性和安全性。
