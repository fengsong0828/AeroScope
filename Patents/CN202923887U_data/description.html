Description

一种低空飞行器
技术领域
本实用新型涉及一种飞行工具，特别是一种结构简单、经济实用、可以缓解地面交通压力的低空飞行器，属于飞行交通工具技术领域。
背景技术
随着社会发展，地面的车辆越来越多，交通压力也越来越大，堵车时间大大增加，车祸也随之多了起来；这样的现状给人们的生活带来了不便，生命安全受到严重影响；为了缓解地面交通压力，交通工具应该逐渐进入飞行领域；而现在的飞行交通工具大多结构复杂，成本过高，操作过于繁琐，多数都应用于军事领域，不能适合于普通民用。
实用新型内容
针对上述存在的技术问题，本发明的目的是：提出了一种结构简单、经济实用、可以缓解地面交通压力的低空飞行器。
本实用新型的技术解决方案是这样实现的：一种低空飞行器，包含驾驶舱；所述驾驶舱中设置有驾控装置和驾驶座椅；所述驾驶舱的上侧设置有主梁，驾驶舱的下侧设置有起落架；所述主梁的前端和后端分别设有前横梁和后横梁；所述前横梁与后横梁的左右两端上均设置有一个飞行螺旋桨，四个飞行螺旋桨均可前后倾斜；所述后横梁的正中间设置有后舵螺旋桨；所述飞行螺旋桨与后舵螺旋桨通过内燃机或电动机驱动；所述驾驶舱处设置有油箱或电池柜。
优选的，所述四个飞行螺旋桨均可前后倾斜。
优选的，所述起落架与驾驶舱之间设置有减震器。
由于上述技术方案的运用，本实用新型与现有技术相比具有下列优点：
本实用新型的低空飞行器，通过4个飞行螺旋桨产生推进力，飞行器的升降和行进均通过4个飞行螺旋桨的倾斜实现，飞行器的左右行进方向由后舵螺旋桨产生的左向或右向推力实现，起落架与驾驶舱之间设有减震器，可以降低飞行器落地时的震颤；因此，本实用新型的低空飞行器，可在1000米以下的低空飞行，甚至“掠地”飞行，可以缓解地面交通压力；结构简单，制造容易，因而成本低廉，仅相当于一辆轿车的成本，最高时速限定200公里以下，一般正常运行时速设定为80公里，低速飞行安全可靠，螺旋桨的动力可采用内燃机或者电动机，从而适应能源供应状况。

具体实施方式
下面结合附图及具体实施例对本实用新型作进一步的详细说明。
附图1-4为本发明所述的一种低空飞行器，包含驾驶舱1；所述驾驶舱1中设置有驾控装置和驾驶座椅；所述驾驶舱1的上侧设置有主梁2，驾驶舱1的下侧设置有起落架3；所述起落架3与驾驶舱1之间设置有减震器8；所述主梁2的前端和后端分别设有前横梁4和后横梁5；所述前横梁4与后横梁5的左右两端上均设置有一个飞行螺旋桨6，四个飞行螺旋桨6均可前后倾斜；所述后横梁的正中间设置有后舵螺旋桨7；所述飞行螺旋桨6与后舵螺旋桨7通过内燃机(未示出)驱动；所述驾驶舱1的下部隔层设置有油箱。
启动时，四个飞行螺旋桨6由内燃机驱动旋转，出风面垂直向下，调控风力，使飞行器缓慢升起离地；操纵四个飞行螺旋桨6向后方倾斜，使出风面朝向后下方，产生向前的反作用力，使飞行器向前飞行；飞行速度通过调整风力实现；飞行高度通过调整出风面方向的角度实现；行进左右方向通过调整后舵螺旋桨7的风力实现；着陆时，操纵四个飞行螺旋桨6垂直向下，缓慢降低风力，使得飞行器平稳着陆。
本实用新型的低空飞行器，可在1000米以下的低空飞行，甚至“掠地”飞行，可以缓解地面交通压力；结构简单，制造容易，因而成本低廉，仅相当于一辆轿车的成本，最高时速限定200公里以下，一般正常运行时速可设定为80公里，低速飞行安全可靠，螺旋桨的动力可采用内燃机或者电动机，从而适应能源供应状况。
