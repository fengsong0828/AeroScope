Description

一种用于低空飞行器监测的实时显示装置
技术领域
本实用新型涉及实时监测显示技术领域，具体是涉及一种用于低空飞行器监测的实时显示装置。
背景技术
低空飞行器，又被简称为“无人机”，是利用无线电遥控设备和自备的程序控制装置操纵的不载人飞机，或者由车载计算机完全地或间歇地自主地操作。随着社会的发展，大多使用者常通过实时显示装置对无人机进行监测。
目前大多数的实时显示装置多为固定式的一体化设计，需要外出工作时携带不便，且无人机携带时容易受到碰撞进而损坏，基于此，本方案提供一种用于低空飞行器监测的实时显示装置解决上述提出的问题。
实用新型内容
为解决上述技术问题，提供一种用于低空飞行器监测的实时显示装置，本技术方案解决了上述背景技术中提出的问题。
为达到以上目的，本实用新型采用的技术方案为：
一种用于低空飞行器监测的实时显示装置，其特征在于，包括便携箱，所述便携箱的下端安装有万向自锁轮组，所述便携箱的后端固定连接有推拉杆，所述便携箱的上端左右两侧均转动安装有箱盖，所述便携箱的内壁固定连接有隔板，所述隔板的上端固定安装有双向电机，所述双向电机的两个输出端均固定连接有锥齿轮一，所述锥齿轮一的上端与锥齿轮二相啮合，所述锥齿轮二固定连接在螺纹杆的表面，所述螺纹杆左端安装在固定框架的内部，所述固定框架的外侧端与便携箱的内壁固定连接，所述螺纹杆的上端表面螺纹连接有升降板，所述升降板的上端中部设置有显示组件，所述隔板的下侧设置有无人机收纳箱。
优选的，所述升降板的上端左右两侧均固定连接有固定板，所述固定板的上端固定连接有连接杆，所述连接杆的上端与驱动块相铰接，所述驱动块滑动安装在滑槽的内部，所述滑槽开设在箱盖的内侧端。
优选的，所述便携箱的内壁下侧固定连接有滑轨，所述无人机收纳箱的外侧端固定连接有滑块，所述滑块的另一端与滑轨滑动连接。
优选的，所述无人机收纳箱的内部设置有放置板，所述放置板的下端固定安装有若干组弹簧阻尼器。
优选的，所述固定框架的内壁固定连接有限位板，所述限位板设置在双向电机的上侧。
优选的，所述显示组件包括显示屏和操作台，所述操作台固定安装在升降板的上端，所述显示屏转动安装在操作台的后端。
与现有技术相比，本实用新型提出一种用于低空飞行器监测的实时显示装置，具备以下有益效果：
1.本实用新型中设置有万向自锁轮组和推拉杆，允许使用者能够轻松地拖拉箱子，推拉杆的作用是提供一个舒适的手柄，方便移动箱子，万向轮能够360度自由旋转，这样箱子可以方便地改变方向。同时，自锁功能可以确保箱子停靠时轮子不滑动，增加了箱子在位置稳定性上的优势。
2.本实用新型中显示组件安装在升降板上，当不使用时，显示组件收纳在便携箱中，从而可以保护显示组件不会受到外界冲击，需要使用时，通过双向电机驱动两侧的锥齿轮一，通过锥齿轮二传动，从而带动螺纹杆转动，螺纹杆转动进而驱动升降板移动，升降板移动过程中，通过固定板带动连接杆一起移动，连接杆的另一端与驱动块相铰接，进而使得驱动块在滑槽的内部滑动，进而使得箱盖被驱动块顶起，使得箱盖随升降板上升而打开，下落而关闭。
3.本实用新型中设置有无人机收纳箱，箱体的内部设置有放置板，用于放置无人机，并通过固定机构对无人机进行固定约束，放置板的下端连接有若干组弹簧阻尼器，无人机收纳箱用于保护无人机不会受到外界的碰撞，弹簧阻尼器则通过吸收和消散冲击能量，有效地控制和减少无人机受到振动和冲击，进而有效保护无人机的安全。

具体实施方式
以下描述用于揭露本实用新型以使本领域技术人员能够实现本实用新型。以下描述中的优选实施例只作为举例，本领域技术人员可以想到其他显而易见的变型。
参照图1-4所示，一种用于低空飞行器监测的实时显示装置，其特征在于，包括便携箱1，便携箱1的下端安装有万向自锁轮组14，便携箱1的后端固定连接有推拉杆13，允许使用者能够轻松地拖拉箱子，推拉杆13的作用是提供一个舒适的手柄，方便移动箱子，万向轮能够360度自由旋转，这样箱子可以方便地改变方向。同时，自锁功能可以确保箱子停靠时轮子不滑动，增加了箱子在位置稳定性上的优势，便携箱1的上端左右两侧均转动安装有箱盖2，便携箱1的内壁固定连接有隔板101，隔板101的上端固定安装有双向电机3，双向电机3的两个输出端均固定连接有锥齿轮一4，锥齿轮一4的上端与锥齿轮二5相啮合，锥齿轮二5固定连接在螺纹杆7的表面，螺纹杆7左端安装在固定框架6的内部，固定框架6的外侧端与便携箱1的内壁固定连接，螺纹杆7的上端表面螺纹连接有升降板8，升降板8的上端中部设置有显示组件9，升降板8的上端左右两侧均固定连接有固定板801，固定板801的上端固定连接有连接杆802，连接杆802的上端与驱动块202相铰接，驱动块202滑动安装在滑槽201的内部，滑槽201开设在箱盖2的内侧端。当不使用时，显示组件9收纳在便携箱1中，从而可以保护显示组件9不会受到外界冲击，需要使用时，通过双向电机3驱动两侧的锥齿轮一4，通过锥齿轮二5的传动，从而带动螺纹杆7转动，螺纹杆7转动进而驱动升降板8移动，升降板8移动过程中，通过固定板801带动连接杆802一起移动，连接杆802的另一端与驱动块202相铰接，进而使得驱动块202在滑槽201的内部滑动，进而使得箱盖2被驱动块202顶起，使得箱盖202随升降板8上升而打开，下落而关闭。
具体的，本实施例中，便携箱1的内壁下侧固定连接有滑轨102，无人机收纳箱10的外侧端固定连接有滑块1001，滑块1001的另一端与滑轨102滑动连接。

本实用新型的工作原理是：使用者通过推拉杆13配合万向自锁轮组14将便携箱1移至预定的位置，随后通过自锁功能可以确保箱子停靠时轮子不滑动，增加了箱子在位置稳定性上的优势，随后通过双向电机3驱动两侧的锥齿轮一4，通过锥齿轮二5的传动，从而带动螺纹杆7转动，螺纹杆7转动进而驱动升降板8移动，升降板8移动过程中，通过固定板801带动连接杆802一起移动，连接杆802的另一端与驱动块202相铰接，进而使得驱动块202在滑槽201的内部滑动，进而使得箱盖2被驱动块202顶起，并拉出无人机收纳箱10，通过操作台控制无人机启动。
当不使用时，显示组件9收纳在便携箱1中，从而可以保护显示组件9不会受到外界冲击，无人机放置在放置板11上，并通过固定机构(图中未画出)对无人机进行固定约束，放置板11的下端连接有若干组弹簧阻尼器12，无人机收纳箱10用于保护无人机不会受到外界的碰撞，弹簧阻尼器12则通过吸收和消散冲击能量，有效地控制和减少无人机受到振动和冲击，进而有效保护无人机的安全。
