Description

一种避开雷达的量子低空飞行器
技术领域
本发明涉及无人机技术领域，具体为一种避开雷达的量子低空飞行器。
背景技术
在人类对天空探索的漫漫征程中，飞行器技术的每一次革新都如同一座熠熠生辉的里程碑，照亮了前行的道路。
长期以来，雷达系统一直是空域监测的支柱，尤其在跟踪高空和超音速飞行器方面发挥着重要作用，然而，低空飞行的日益普及暴露了传统雷达系统的弱点，诸如量子雷达技术等近期的进展，据称增强了对低空飞行航空器的探测能力，然而由于低空环境中环境噪声增加、地形杂波干扰以及大气干扰等因素，这些技术仍然相对低效，难以满足复杂环境下的特殊任务需求，如在低空环境中，自然和人为噪声干扰严重，导致高噪声信号比，破坏信号准确性，因此提出一种避开雷达的量子低空飞行器。
发明内容
本发明的目的在于提供一种避开雷达的量子低空飞行器，以解决上述背景技术中提出的由于低空环境中环境噪声增加、地形杂波干扰以及大气干扰等因素，这些技术仍然相对低效，难以满足复杂环境下特殊任务需求的问题。
为实现上述目的，本发明提供如下技术方案：
一种避开雷达的量子低空飞行器，包括：无人机，所述无人机的四角均固定安装有连接臂，四个所述连接臂的上表面均固定安装有无刷电机，所述无刷电机的外表面套装有冷却机构，所述冷却机构的散热端位于顶盖与无人机的上表面之间，所述顶盖固定安装在无人机上表面，以此使所述无人机在飞行的过程中，能够让风流从顶盖与无人机之间的间距内流过，进而实现对冷却机构散热端的吹扫；
其中，所述无人机的上表面开设有内嵌槽，所述内嵌槽内涂覆有导热硅脂，让导热硅脂与所述冷却机构的散热端下表面充分接触，能够让所述无人机内所产生的热量通过上表面涂抹的导热硅脂一同传导给冷却机构的散热端一同散热，以此能够将所述无人机内部所产生的热量辐射到飞机外表面，从而降低所述无人机在红外频谱中的可见度；
其中，所述无人机外表面固定安装有信号干扰机构，所述信号干扰机构受控于控制器，而控制器集成于无人机内，所述信号干扰机构可通过精确控制量子态的变化，使干扰信号与探测雷达信号在空间中发生干涉，从而抵消雷达信号的探测能力，不易被敌方的信号检测系统察觉。
优选的，所述无人机和连接臂的外表面涂覆有雷达吸波材料(RAM)，并通过纳米涂层技术集成到所述无人机和连接臂外表面，此雷达吸波材料(RAM)可最大限度减少雷达信号反射，上，这大幅降低了雷达散射截面(RCS)，显著增强了所述无人机的隐形能力。
优选的，所述信号干扰机构包括微波发生器，所述微波发生器呈内嵌式固定安装在顶盖内，所述微波发生器通过同轴电缆与天线相连接，所述天线呈阻尼式滑缩于滑管内，所述滑管转动安装在收纳槽内，所述收纳槽开设于连接臂下表面，所述微波发生器受控于控制器，而控制器的信号接收端与所述电磁噪声传感器和麦克风的信号发射端相连接，所述电磁噪声传感器和麦克风固定安装在无人机外表面，以此能够让所述电磁噪声传感器将检测到的电磁噪声和声学噪声信号传输给控制器进行分析和处理。
优选的，控制器能够对检测到的电磁噪声和声学噪声信号进行分析、滤波、放大等处理，以提取其频率、相位、编码等特征信息，然后根据这些信息来调整干扰信号，使其在频率和相位上与敌方信号保持同步，进而利用这些处理结果以及系统的预设目标，对所述微波发生器进行精确控制，调整其输出的微波信号的参数，能够让所述微波发生器在控制器的指令下，产生特定的干扰信号，这些干扰信号通过所述天线发射出去，用于干扰敌方的通信、雷达等系统，同时，能够确保干扰信号与敌方信号在时间、频率等方面实现同步干扰，以达到最佳的干扰效果。
优选的，所述天线的下表面固定安装有万向球，所述万向球外表面转动安装有支撑盘；
其中，所述连接臂的上表面螺纹安装有蝶形螺栓，所述蝶形螺栓能够转动穿进收纳槽内并顶触在滑管的外表面处，以此来实现对所述滑管的旋转施加锁止目的。
优选的，四个所述连接臂的一端均固定安装有激光雷达，所述激光雷达的信号发射端与控制器的信号接收端相连接，且四个所述激光雷达之间通过以太网电缆连接，以此使四个所述激光雷达在硬件层面实现同步触发，从而保证它们在同一时刻开始进行数据采集，在这种方式下，所述激光雷达之间通过网络相互通信，交换时间戳或其他同步信息，然后根据这些信息进行时间校准和同步地形的绘制。
优选的，所述微波发生器、电磁噪声传感器、激光雷达、天线和控制器的型号分别为AgilentE8257DPSG、TEConnectivity1-1527747-1、VelodyneHDL-32E、PasternackPE94920和NVIDIAJetsonTX2。
优选的，所述冷却机构包括四个导热环，四个所述导热环分别套装在四个无刷电机的外表面处，且四个所述导热环的外表面均连通安装有两组连接管，且每组所述连接管的另一端均从U形套中穿出并与其他连接管形成环路相互连通连接，所述U形套固定安装在连接臂两端。
优选的，八组所述连接管相连通形成的环路之间分别与微型水泵的进水口和出水口相连通安装，所述微型水泵固定安装在无人机一端；
其中，而八组所述连接管相连通形成的环路另一端之间连通安装有呈曲线形管，所述曲线形管处于顶盖与无人机之间。
优选的，所述曲线形管外表面套装有导热板，所述导热板的下表面通过导热硅脂与无人机的上表面充分接触，所述导热板的上表面固定安装有散热鳍片。
与现有技术相比，本发明的有益效果是：
1、通过无人机、连接臂、顶盖、无刷电机、冷却机构和信号干扰机构的设计，在无人机和连接臂的表面涂覆有先进的雷达吸波材料(RAM)，可最大限度减少雷达信号反射，这些材料通过纳米涂层技术集成到无人机和连接臂上，当雷达发射的电磁波照射到无人机上时，雷达吸波材料(RAM)能够吸收这些电磁波的能量，将其转化为热能或其他形式的能量消耗掉，从而最大限度减少雷达信号反射，这大幅降低了雷达散射截面(RCS)，使雷达接收到的反射信号极其微弱，显著增强了无人机的隐形能力，不易被雷达探测到，并且在飞行的过程中同时启动信号干扰机构将检测到的外界电磁噪声和声学噪声信号传输给控制器进行分析和处理，然后让控制器能够对检测到的电磁噪声和声学噪声信号进行分析、滤波、放大等处理，以提取其频率、相位、编码等特征信息，然后根据这些信息来调整干扰信号，使其在频率和相位上与敌方信号保持同步，进而利用这些处理结果以及系统的预设目标，来对产生的量子态变化精确控制，让这些干扰信号与探测雷达信号在空间中发生干涉，相互抵消或削弱，使得雷达无法准确接收到目标反射的真实信号，从而降低了雷达对无人机的探测能力，使无人机不易被敌方的信号检测系统察觉，而检测到的声学噪声可以对无人机的飞行姿态和轨迹进行动态优化，例如，在靠近强电磁干扰源或高噪声区域时，适当调整飞行高度、速度和方向，既维持隐形又确保飞行稳定，并且无人机在飞行的过程中，能够让风流从顶盖与无人机之间的间距内流过，能够使风流吹扫在之间的冷却机构的散热端外表面，而冷却机构是一同吸附无人机和无刷电机所产生的热量，让其在飞行的过程中随风流散发，以此能够将无人机内部和无刷电机外表面所产生的热量辐射到外表面，从而降低无人机在红外频谱中的可见度，使其更难被敌方的红外探测设备发现，在军事应用中，这有助于无人机执行侦察、监视等任务时，避免被敌方红外预警系统或热成像仪探测到，提高任务的成功率和自身的生存能力。
2、通过微波发生器、电磁噪声传感器、激光雷达、天线、滑管和蝶形螺栓的设计，操控无人机进行侦查作业时，可供工作人员将蝶形螺栓扭出，并将滑管从连接臂下表面开设的收纳槽内翻转出，并将滑管内的天线从中拉出，而天线另一端的支撑盘则能够为无人机的起落进行支撑，直至对天线翻转出的角度调整好后可将蝶形螺栓旋紧，以此使蝶形螺栓的螺纹部能够顶触在滑管的外表面处，来实现对滑管和天线的旋转施加锁止目的，随后可一同启动电磁噪声传感器和麦克风，让电磁噪声传感器开始持续监测无人机周围环境中的电磁噪声信号，由于其具备高灵敏度与宽频响应特性，能够捕捉到各种频率范围的电磁干扰，无论是来自自然环境中的电磁波动，还是敌方电子设备、通信系统以及雷达设施所产生的电磁信号，都能被精确感知，并将其转化为电信号，而同步工作的麦克风，即可收集无人机周边的声学噪声，让其在执行侦查作业的环境中，声学噪声包含丰富信息，例如可能存在的人员活动声音、机械设备运转声、交通工具行驶声等，让麦克风将这些声学振动转化为电信号，为系统提供额外的环境信息维度，而采集到的电磁噪声和声学噪声电信号，通过预先设定的线路传输至集成在无人机内的控制器，让控制器运用内置的先进算法，对输入的电磁噪声和声学噪声信号进行深度分析，针对电磁噪声信号，算法着重提取其频率、相位、调制方式以及编码特征等关键信息，通过对这些特征的精确解析，能够识别出不同类型的电磁信号源，区分敌方雷达信号与其他背景电磁干扰，对于声学噪声信号，算法则聚焦于声音的频率范围、强度变化以及特定声音模式，以此判断周围是否存在潜在的威胁源或目标对象，且在特征提取过程中，控制器还会对信号进行滤波处理，去除噪声信号中的冗余成分和干扰杂质，提高信号的纯净度和信噪比，同时，对有用信号进行放大操作，增强信号的强度，以便后续更精准地分析和处理，随后基于对电磁噪声和声学噪声信号的分析结果，结合系统预设的干扰策略和目标，让控制器操控微波发生器开始生成针对敌方通信和雷达系统的干扰信号，并且由于无人机在飞行侦察过程中，所处环境的电磁和声学信号是动态变化的，控制器会持续监测信号变化情况，并实时调整干扰信号的参数，能够让微波发生器根据控制器的指令，产生特定参数的干扰信号，并通过同轴电缆将其传输至天线，天线将电信号转换为电磁波，向周围空间辐射出去，由于天线在之前已根据侦查需求调整好角度并锁定，此时能够将干扰信号准确地发射至目标区域，干扰敌方通信和雷达系统的正常工作，使得雷达无法准确接收到目标反射的真实信号，从而降低了雷达对无人机的探测能力，使无人机不易被敌方的信号检测系统察觉，且在干扰信号发射的同时，四个连接臂一端固定安装的激光雷达也在持续工作，激光雷达通过发射激光束并接收反射光，获取周围环境的三维空间信息，生成高精度的点云数据，这些点云数据一方面用于辅助无人机的自主导航和避障，确保无人机在复杂环境中安全飞行，另一方面，与电磁噪声、声学噪声分析结果以及干扰信号发射状态相结合，为操作人员提供全面的环境态势感知，帮助其更好地评估侦查作业效果，并根据实际情况及时调整无人机的飞行路线、干扰策略等。
3、通过微型水泵、连接管、导热环、散热鳍片和曲线形管的设计，无人机在飞行的过程中，可一同启动微型水泵不断地抽入并排出连接管内的冷却液，让冷却液依次流入连接管、导热环和曲线形管形成回路不断流动，而冷却液流入导热环内时，能够使导热环吸附无刷电机外表面所产生的热量并供送进曲线形管内，而曲线形管即可将吸附冷却液中的热量传导给导热板，而导热板即可通过上表面固定连接的散热鳍片散发，并且由于无人机在飞行过程中，空气会从顶盖与无人机之间的间距内流过，这股流动的空气会与散热鳍片表面充分接触，根据热传递原理，热量会从温度较高的散热鳍片传递到温度较低的空气中，将散热鳍片上的热量带走，并且通过无人机的持续飞行，能够不断补充新的冷空气，持续维持热交换的进行，保证散热的高效性，并且导热板不仅将曲线形管传来的热量传递给散热鳍片，还会通过下表面的导热硅脂与无人机上表面充分接触，进而一同吸附了无人机所产生的热量，进一步帮助降低无人机的表面温度，随后经过散热鳍片散热后的冷却液，温度会有所降低，此时，微型水泵继续工作，将温度降低后的冷却液抽回连接管中，使其再次流入导热环，由于冷却液温度低于导热环的温度，根据热传递原理，冷却液又能从导热环吸收无刷电机产生的热量，开始新一轮的热量传输与散热循环，进而能够降低无人机在运行过程中的温度，减少热量辐射到飞机外表面，降低其在红外频谱中的可见度。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。

综上：该量子低空飞行器，能够对检测到的电磁噪声和声学噪声信号进行分析、滤波、放大等处理，以提取其频率、相位、编码等特征信息，然后根据这些信息来调整干扰信号，使其在频率和相位上与敌方信号保持同步，进而利用这些处理结果以及系统的预设目标，来对产生的量子态变化精确控制，让这些干扰信号与探测雷达信号在空间中发生干涉，相互抵消或削弱，使得雷达无法准确接收到目标反射的真实信号，从而降低了雷达对无人机1的探测能力，使无人机1不易被敌方的信号检测系统察觉，而检测到的声学噪声可以对无人机1的飞行姿态和轨迹进行动态优化，让其既维持隐形又确保飞行稳定，解决了由于低空特有的信号衰减和环境干扰问题。
