Description

一种低空长航时飞行器
技术领域
本发明涉及飞行器技术领域，特别涉及一种低空长航时飞行器。
背景技术
飞行器包括无人机驾驶飞行器和有人驾驶飞行器，无人机驾驶飞行器简称“无人机”，是利用无线电遥控设备或嵌入式程序操控的不载人飞机。目前应用比较广泛的是旋翼式无人机，其动力来自旋翼的高速旋转所提供的升力，通过控制每个旋翼的不同旋转速度来实现无人机的升降悬停等不同的动作。
大多数固定翼飞行器需要在长而平整的跑道上起降，对地形复杂区域场景中使用受限。固定翼飞行器无法悬停必须保持一定的前进速度才能产生升力，这限制了其在需要定点作业的场景中的应用。
旋翼飞行器（如直升机、多旋翼无人机等）依靠旋翼旋转产生升力和推力，具备垂直起降、空中悬停等独特优势，但也存在以下缺点：
飞行速度与航程受限
速度上限较低：因为旋翼高速旋转时，桨叶前缘易产生激波阻力，后缘可能出现失速，限制了速度提升，旋翼飞行器的飞行速度通常低于固定翼飞行器。
航程较短：旋翼飞行器的燃油效率较低，且旋翼系统消耗大量动力，导致其航程有限。多旋翼无人机的航程更短，一般仅能飞行数十公里，依赖电池供电时续航问题更突出。
发明内容
本发明的目的在于提供一种低空长航时飞行器，具有兼具固定翼长续航与旋翼垂直起降特性，解决传统固定翼依赖跑道的局限性。调整件驱动后桨叶多角度旋转，避免能量浪费于无效姿态调整，前、后桨叶角度协同调节，实现垂直起降、悬停、转向操控。实现续航能力的显著提升，设计在保持尾翼件稳定性的同时，减少了飞行姿态调整的能量消耗。的优点。
为实现上述目的，本发明提供如下技术方案：一种低空长航时飞行器，包括：
机身，其两侧固定有对称的固定翼，每个固定翼上设置有可转动的前桨叶机构；
尾杆，尾杆由前杆和后杆组成，前杆的一端固定在机身的尾部，前杆和后杆之间与调整件相接，所述调整件包括限位件一和限位件二，限位件一两端与前杆和后杆连接，限位件二卡在限位件一内旋转，后桨叶机构装配在限位件二上；
尾翼件，其装配在后杆上；
传动件，其设置在机身和固定翼内，用于驱动前桨叶机构角度调整，传动件包括传动轴和齿轮组，传动轴通过锥齿轮由伺服电机驱动，传动轴两端啮合的齿轮组与前桨叶机构相接；
作为一种可选的实施方式，所述前桨叶机构和后桨叶机构结构相同，包括无刷电机、电机座以及桨叶，无刷电机装配在电机座内，无刷电机的轴与桨叶相接。
作为一种可选的实施方式，所述固定翼的迎风端上设置支架，电机座两端连接的轴插入支架内，传动轴上连接的齿轮组与前桨叶机构的电机座轴上的齿轮啮合。
作为一种可选的实施方式，所述前桨叶机构的电机座的底部固定有支撑脚，固定翼的底部开设有容纳支撑脚的条形槽。
作为一种可选的实施方式，所述尾翼件包括尾翼一、尾翼二以及套环，套环装配在后杆上，尾翼一和尾翼二对称设置在套环上。
作为一种可选的实施方式，所述前杆内设置有驱动限位件一旋转的驱动轴，驱动轴接入机身内的电机，限位件一的两端与后杆固定，电机驱动限位件一和后杆同步旋转。
作为一种可选的实施方式，所述限位件一为内部中空的结构，限位件二两端连接的转轴插入限位件一内，并与限位件一内的电机连接，后桨叶机构的电机座装配在限位件二上，机身内引出的导线经过驱动轴、限位件一以及转轴为无刷电机供电。
作为一种可选的实施方式，所述限位件一的两端分别与前杆和后杆固定，限位件一的内周面开设有环槽，环槽内卡入旋转的转环以及固定有导电环，转环由前杆内的电机驱动旋转，转环上固定的电机与限位件二相接，后桨叶机构的电机座装配在限位件二上。
作为一种可选的实施方式，所述限位件二上装配有封住环槽开口的环片，限位件二上连接有与导电环相接的导电杆。
作为一种可选的实施方式，所述机身内引出的导线经过导电环、导电杆和限位件二为无刷电机供电。
本发明的技术效果和优点：
1、垂直起降与巡航模式分离：起飞和降落时，桨叶机构处于竖直方向，依靠电机推力升空；巡航时切换至水平方向，固定翼利用气动升力维持飞行，电机仅提供前进动力，减少能量消耗，兼具固定翼长续航与旋翼垂直起降特性，解决传统固定翼依赖跑道的局限性。
2、调整件驱动后桨叶多角度旋转，在转向时精准控制推力方向，避免能量浪费于无效姿态调整，前、后桨叶角度协同调节，实现垂直起降、悬停、转向等全场景操控。
3、通过桨叶机构的模式切换、最终实现续航能力的显著提升，设计在保持尾翼件稳定性的同时，减少了飞行姿态调整的能量消耗。

限位件一51经过的电机驱动可绕后杆42旋转，从而改变后桨叶机构8的角度，可以将后桨叶机构8的桨叶33进行角度调整，同理限位件一51内的电机驱动限位件二52旋转，即后桨叶机构8的桨叶33在进行角度调整，通过上述两个调整结构让桨叶33能够调整至任意角度。
飞行器起飞、下降以及悬停：
限位件一51旋转至水平平面同时限位件二52也旋转至水平平面，这时候后桨叶机构8处于竖直方向，同时前桨叶机构3也旋转至竖直方向，前桨叶机构3和后桨叶机构8产生的气流推动飞行器上升、下降以及悬停；
飞行器飞行：
飞行器在起飞后，将前桨叶机构3和后桨叶机构8旋转至水平方向，则前桨叶机构3和后桨叶机构8旋转产生的气流推动飞行器飞行，而飞行器在飞行的过程中固定翼2的上下压强差提供升力，本结构设置的飞行器可以实现垂直升降以及悬停。
同时本飞行器利用固定翼2的结构在长时间的飞行过程中，通过固定翼2的结构设置，利用压强升力减少了动力功率需求，延长了飞行器的长时间续航。
飞行器在飞行的过程转向：前桨叶机构3处于水平方向，提供飞行器飞行的动力，将后桨叶机构8的桨叶33旋转至竖直平面，则后桨叶机构8的桨叶33旋转产生的气流则推动飞行器转向。
前桨叶机构3的收放策略：当飞行器进入巡航模式时，将前桨叶机构3旋转至水平方向，并将支撑脚收入固定翼2的条形槽内，消除地面支撑结构带来的额外风阻。
后桨叶机构8的角度优化：在巡航时，通过调整件5将后桨叶机构8调整至与飞行方向一致，减少桨叶迎风面积，降低诱导阻力。
本结构解决了传统固定翼2飞行器需要跑道进行滑跑，导致会有场地条件的需求问题，同时也解决了固定翼2飞行器无法垂直升降以及悬停的问题。
