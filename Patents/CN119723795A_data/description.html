Description

一种基于气象观测的低空飞行器结冰预警方法及系统
技术领域
本发明属于低空飞行器结冰预警技术领域，特别涉及一种基于气象观测的低空飞行器结冰预警方法及系统。
背景技术
低空飞行器是指在距地面较低高度范围内飞行的各类飞行器。低空飞行器一般在1000米以下空域飞行，在0-3000米的云层范围内都有结冰风险，尤其是在1500米到3000米高度，是直升机等低空飞行器常见的结冰高发区域。
为了降低有人驾驶的低空飞行器在执行飞行任务时遇到飞行器结冰的风险，通常会获取预测的气象数据，并根据预测的气象数据制定飞行路线，但这种方式较为依赖气象预测的精准度，可以理解的，预测的气象数据越准确，制定的飞行路线越合理，飞行器结冰的概率也就越低，换句话说，若气象突变，而飞行员仍按照既定路线飞行，或者较晚采取应对措施，会有较大的安全隐患。
发明内容
基于此，本发明实施例当中提供了一种基于气象观测的低空飞行器结冰预警方法及系统，旨在实时根据气象数据，评估飞行器的结冰情况，以降低飞行风险。
本发明实施例的第一方面提供了一种基于气象观测的低空飞行器结冰预警方法，所述方法包括：
获取低空飞行器同一时期的历史气象数据、历史飞行数据、飞行器自身数据以及积冰情况，所述历史气象数据包括温度、湿度、风速、风向、气压、降水类型以及云层相对位置，历史飞行数据包括飞行高度、飞行速度以及飞行时间，所述飞行器自身数据包括材质和表面特性，所述积冰情况包括积冰部件、积冰厚度和积冰类型；
根据积冰部件的划分，分别建立对应的神经网络模型，并分别进行训练，得到对应的目标神经网络模型，具体的，将历史气象数据、历史飞行数据以及飞行器自身数据作为所述神经网络模型的输入，将间隔预设时间的积冰厚度和积冰类型作为所述神经网络模型的输出；
实时获取低空飞行器采集的气象数据、飞行数据以及飞行器自身数据，分别输入至各积冰部件对应的目标神经网络模型中，输出对应的积冰厚度和积冰类型；
根据目标神经网络模型的输出结果，判断预设积冰部件是否达到预警标准；
若判断预设积冰部件达到预警标准，则进行第一预警；
若判断预设积冰部件未达到预警标准，则根据各积冰部件的积冰厚度和积冰类型，进行加权处理，得到评价值；
判断所述评价值是否大于阈值；
若判断所述评价值大于阈值，则进行第二预警。
进一步的，所述方法还包括：
获取初始规划路径，根据预警情况，调整所述初始规划路径。
进一步的，所述获取初始规划路径，根据预警情况，调整所述初始规划路径的步骤包括：
获取当前低空飞行器飞行位置的三维卫星云图，确定三维卫星云图中有利于缓解结冰的目标区域；
当预警情况为所述第一预警时，则根据距离当前低空飞行器飞行位置最近的第一目标区域，规划低空飞行器驶入所述第一目标区域的第一飞行路径，并根据所述第一飞行路径，调整所述初始规划路径；
当预警情况为所述第二预警时，则根据所述三维卫星云图，修正气象预测数据，并根据修正后的气象预测数据生成目标三维卫星云图，规划低空飞行器偏离所述初始规划路径程度最小的第二飞行路径，并根据所述第二飞行路径，调整所述初始规划路径。
进一步的，所述根据修正后的气象预测数据生成目标三维卫星云图，规划低空飞行器偏离所述初始规划路径程度最小的第二飞行路径的步骤包括：
将所述初始规划路径映射至所述目标三维卫星云图中，并将所述间隔预设时间飞行路径对应的线段从所述初始规划路径中移除，形成缺口；
获取所述间隔预设时间飞行路径对应的预设空间范围中的目标气象预测数据，将所述目标气象预测数据分别输入至各积冰部件对应的目标神经网络模型中，输出对应的预测积冰厚度和预测积冰类型；
根据预测积冰厚度和预测积冰类型，确定目标规划路线；

进一步的，所述根据预测积冰厚度和预测积冰类型，确定目标规划路线的步骤包括：

在所述第二目标区域中，确定低空飞行器的最短飞行路线为所述目标规划路线。
进一步的，所述神经网络模型由输入层、隐藏层和输出层构成，所述输入层包含若干输入节点，输入节点数量与输入数据种类数相同，用于接收不同类型的输入数据；

实施例一
根据本发明实施例，提供了一种基于气象观测的低空飞行器结冰预警方法实施例，需要说明的是，在附图的流程图示出的步骤可以在诸如一组计算机可执行指令的计算机系统中执行，并且，虽然在流程图中示出了逻辑顺序，但是在某些情况下，可以以不同于此处的顺序执行所示出或描述的步骤。
在本实施例一中提供了一种基于气象观测的低空飞行器结冰预警方法，可用于电子设备中，如计算机中。请参阅图1，图1示出了本发明实施例一提供的一种基于气象观测的低空飞行器结冰预警方法的实现流程图，具体包括步骤S01至步骤S08。
步骤S01，获取低空飞行器同一时期的历史气象数据、历史飞行数据、飞行器自身数据以及积冰情况。
具体的，历史气象数据包括温度、湿度、风速、风向、气压、降水类型以及云层相对位置，历史飞行数据包括飞行高度、飞行速度以及飞行时间，飞行器自身数据包括材质和表面特性，积冰情况包括积冰部件、积冰厚度和积冰类型，需要说明的是，上述的数据对低空飞行器的结冰均有不同程度的影响。
其中，对于温度来说，机体表面结冰通常发生于0℃至-20℃温度范围内，而强结冰多发生于-2℃至-10℃；对于湿度、气压来说，当云中的空气湿度达到温度露点差在0℃至5℃之间时，更容易发生结冰；对于风速、风向来说，可以结合飞机的飞行速度，当相对速度较大时，飞行器与过冷水滴的碰撞频率和能量增加，过冷水滴更容易在飞行器表面附着并迅速冻结；对于降水类型来说，在冻雨、毛毛雨天气中，外界温度在0℃以下时，过冷状态的雨滴一旦与飞行器接触便容易结冰。在雨夹雪的环境中，雪花与过冷水滴同时存在，也会增加飞行器结冰的可能性；对于云层相对位置来说，当飞行穿越积雨云或浓积云的上方时，机体表面通常会出现强结冰；而穿越层云或层积云时，通常发生弱积冰或中积冰；对于飞行高度来说，在1500米到3000米高度，是常见的结冰高发区域；对于材质和表面特性来说，飞行器表面如果是导热性好的金属材质，在低温环境下更容易使表面温度快速降低，促使水汽凝结结冰。若表面粗糙不平，会增加过冷水滴的附着概率，也更容易结冰。
另外，根据积冰部件的功能和位置的不同，积冰效果也存在差异，例如，直升机的旋翼等转动部件，由于转动过程中与空气的热交换以及复杂的气流情况，比固定部件更容易出现结冰情况。发动机进气口、空速管、风挡玻璃、天线等突出部位，也因更容易与过冷水滴接触而容易结冰。
上述的数据均由低空飞行器上搭载的各类传感器收集得到，示例性的，积冰厚度以及云层相对位置等可以通过在飞行器上安装视觉传感器，如摄像头等，拍摄飞行器表面的图像。通过图像识别技术，对图像中的积冰区域进行分析和处理。
在本发明其他一些实施例当中，还可以借鉴高山气象站积冰观测资料以及风、湿、温、压等气象数据。

其中，将历史气象数据、历史飞行数据以及飞行器自身数据作为所述神经网络模型的输入，将间隔预设时间的积冰厚度和积冰类型作为所述神经网络模型的输出。在数据输入神经网络模型之前，可以进行数据预处理，包括缺失值处理（插值法）、异常值剔除（基于物理约束）、特征工程（Z-score归一化）以及标准化（Z-score归一化）等。

当预警情况为第二预警时，说明存在潜在风险，适应性调整路径即可，则根据三维卫星云图，修正气象预测数据，并根据修正后的气象预测数据生成目标三维卫星云图，规划低空飞行器偏离初始规划路径程度最小的第二飞行路径，并根据第二飞行路径，调整初始规划路径，可以理解的，获取的是实时的三维卫星云图，也即气象数据的准确度较高，通过准确度较高的气象数据修正之前预测的气象数据，以此获得修正后的较为准确的目标三维卫星云图。
具体的，将初始规划路径映射至目标三维卫星云图中，并将间隔预设时间飞行路径对应的线段从初始规划路径中移除，形成缺口；
获取间隔预设时间飞行路径对应的预设空间范围中的目标气象预测数据，将目标气象预测数据分别输入至各积冰部件对应的目标神经网络模型中，输出对应的预测积冰厚度和预测积冰类型，可以理解的，预设空间范围指的是以间隔预设时间飞行路径对应的线段为基准，在高度上进行扩展，得到的矩形、球形等空间范围，目标气象预测数据则是根据目标三维卫星云图获取的；
根据预测积冰厚度和预测积冰类型，确定目标规划路线，其中，根据积冰部件的重要程度，以及积冰部件对应的预测积冰厚度和预测积冰类型，确定预测积冰厚度和预测积冰类型满足要求的，且在预设空间范围中的第二目标区域，示例性的，若存在a、b、c、d、e、f、g这7个积冰部件，其中，a、b、c为关键积冰部件，d、e、f、g为普通积冰部件，根据a、b、c的重要等级，依次在预设空间范围内剔除不满足要求的位置点，不满足要求可以理解为积冰类型不满足、积冰厚度超标等，在剔除不满足要求的位置点后的预设空间范围内，确定d、e、f、g这四个普通积冰部件均满足要求的位置点，并将该位置点合并，形成第二目标区域；
在第二目标区域中，确定低空飞行器的最短飞行路线为目标规划路线；
将目标规划路线对应的线段合并至缺口处，得到第二飞行路径。
综上，本发明上述实施例当中的基于气象观测的低空飞行器结冰预警方法，该方法通过获取低空飞行器同一时期的历史气象数据、历史飞行数据、飞行器自身数据以及积冰情况；根据积冰部件的划分，分别建立对应的神经网络模型，并分别进行训练，得到对应的目标神经网络模型；实时获取低空飞行器采集的气象数据、飞行数据以及飞行器自身数据，分别输入至各积冰部件对应的目标神经网络模型中，输出对应的积冰厚度和积冰类型；根据目标神经网络模型的输出结果，判断预设积冰部件是否达到预警标准；若是，则进行第一预警；若否，则根据各积冰部件的积冰厚度和积冰类型，进行加权处理，得到评价值；判断评价值是否大于阈值；若判断评价值大于阈值，则进行第二预警，具体的，通过评估飞行器的结冰情况，可以有效降低飞行风险。
