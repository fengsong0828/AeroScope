Description

一种检测用具有障碍物探测的低空飞行器
技术领域
本发明涉及飞行器技术领域，具体为一种检测用具有障碍物探测的低空飞行器。
背景技术
飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。随着飞行器普及，个人飞行器必然是未来发展方向。轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向。
低空飞行器的种类多种多样，一般为多旋翼的飞行器，旋翼较多从而体积增大，不利于收纳和存放，占用较大的空间，并且飞行器在降落的过程中极易出现炸机的事故，降落速度较快，容易出现支撑腿断裂的情况，从而造成飞行器倾倒，甚至翻倒的情况，从而造成损坏，同时飞行器在飞行的过程中会碰到各类障碍物，靠摄像头和人眼一般无法及时发现和规避障碍物，从而造成飞行器撞毁的现象发生。
发明内容
本发明的目的在于提供一种检测用具有障碍物探测的低空飞行器，以解决上述背景技术中提出的问题。
为了解决上述技术问题，本发明提供如下技术方案：一种检测用具有障碍物探测的低空飞行器，包括飞行器主体、旋翼、挡块、驱动电机、微型激光雷达本体、电机槽、安装臂、折叠槽、安装座以及转轴，所述飞行器主体底部中心一侧与安装座顶部固定连接，且安装座底部与微型激光雷达本体顶部固定连接，所述飞行器主体两侧分别开设有折叠槽，且折叠槽两端分别通过转轴与安装臂中心一端转动连接，所述飞行器主体两侧中心两端分别开设有电机槽，所述安装臂另一端分别与驱动电机一侧中心固定连接，所述驱动电机输出端中心分别套接固定有旋翼，且驱动电机输出端顶部与挡块底部中心固定连接。
检测用具有障碍物探测的低空飞行器还包括活动杆、支撑套筒、底板、橡胶拉绳、第一弹簧、第一固定块以及第二半球头,所述飞行器主体底部四角位置分别与活动杆顶部焊接，所述活动杆底部贯穿支撑套筒顶部中心，且活动杆与支撑套筒滑动连接，所述活动杆中下部外围套接有第一固定块，且第一固定块底部与第一弹簧顶端固定连接，且第一弹簧底端与支撑套筒内部底面固定连接，所述第一固定块顶部四角位置分别与橡胶拉绳底端固定连接，且橡胶拉绳顶端分别与支撑套筒内部顶面四角位置固定连接。
检测用具有障碍物探测的低空飞行器，还包括安装槽、固定杆、第一半球头、卡槽、锁紧杆、第二导向杆、第一安装块、第二安装块、固定凹槽以及第二弹簧，所述安装臂一端与第二半球头一侧固定连接，所述安装臂位于第二半球头相邻一侧一角位置开设有卡槽，且卡槽内卡接有第一半球头，所述第一半球头固定安装在锁紧杆一端，且锁紧杆另一端与第一安装块一侧中心固定连接，所述第一安装块另一侧与固定杆一侧中下部固定连接，所述固定杆另一侧中下部与第二安装块一侧固定连接，且第二安装块另一侧与第二弹簧一端固定连接，所述第二弹簧另一端固定安装在固定凹槽内部，所述固定凹槽开设在安装槽一侧中下部，所述安装槽开设在飞行器主体两侧中下部边缘处。
进一步，所述支撑套筒底部与底板顶部中心固定连接。
进一步，检测用具有障碍物探测的低空飞行器，还包括第一导向杆以及导向槽，所述支撑套筒内部底面中心与第一导向杆底端固定连接，所述第一导向杆顶部位于导向槽内部，且第一导向杆与导向槽滑动连接，所述导向槽开设在活动杆底面中心。
进一步，所述微型激光雷达本体与飞行控制器电性连接，且微型激光雷达本体与飞行控制器均与蓄电池电性连接。
进一步，检测用具有障碍物探测的低空飞行器还包括第二固定块，所述第二安装块顶部与底部中心分别与第二固定块相向一侧固定连接，所述第二固定块(26)中心贯穿有第二导向杆，且第二导向杆与第二固定块滑动连接，所述第二导向杆一端分别与安装槽一侧顶部与底部固定连接。
与现有技术相比，本发明所达到的有益效果是：
1.该发明通过微型激光雷达本体进行障碍物探测，随后传递到飞行控制器来对低空飞行器的飞行姿态进行调整，便于规避障碍物，有利于提升飞行的安全性。
2.该发明通过橡胶拉绳以及第一弹簧的拉力作用来降低飞行器主体降落时受到的冲击，避免了飞行器底部支撑架断裂的情况发生，有利于提升飞行器降落时的稳定性。
3.该发明通过拉动固定杆，从而松开第一半球头与卡槽，随后将安装臂和驱动电机转动折叠到折叠槽内部，便于安装臂的折叠收纳，减少了低空飞行器的体积，便于携带。
