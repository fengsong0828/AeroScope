Description

低空飞行器防撞控制方法、装置、设备及存储介质
技术领域
本发明涉及低空飞行器控制技术领域，尤其涉及到一种低空飞行器防撞控制方法、装置、设备及存储介质。
背景技术
面对未来低空飞行器的大量增加，低空飞行安全将是一个非常重要的问题，城市上空存在大量的通讯网络，可以通过5G来解决，对于大量缺少基站的地方将会是一个飞行器的盲区，这个时候，一方面需要管理手段，一方面也需要防撞的设备，辅助飞行器进行安全飞行。
目前，针对低空飞行器的防撞控制，通常面临着以下问题：（1）现有技术严重依赖布设大量基站的通讯网络环境，在缺少基站的区域，由于飞行器之间无法进行位置共享与信息同步，使得飞行器之间的防撞避让控制变得异常困难；（2）不同飞行器在执行飞行任务时，为了保证相互之间的飞行安全，需要留出安全间隔距离，而由于飞行器的飞行轨迹存在不确定性（航管飞机通常仅公布起始位置和已飞行的轨迹，并不会公开未飞行的轨迹；非航管飞机通常用于执行临时任务，也不会公开整个飞行轨迹），使得飞行器在规划飞行路径时存在着较大的限制，影响目标飞行器的飞行轨迹规划合理性，无法平衡目标飞行器的飞行安全与飞行效率之间的矛盾。
因此，如何提高低空飞行器的防撞避让有效性、飞行安全性以及飞行轨迹规划合理性，是一个亟需解决的技术问题。
发明内容
本发明提供一种低空飞行器防撞控制方法、装置、设备及存储介质，旨在解决上述至少一个技术问题。
为实现上述目的，本发明提供一种低空飞行器防撞控制方法，包括以下步骤：
获取目标飞行器的目标飞行任务，提取所述目标飞行任务中的计划飞行轨迹；其中，所述计划飞行轨迹包括目标飞行器按照标准采样时间周期进行采样获得的若干个计划飞行轨迹点；
在检测到目标飞行器执行所述目标飞行任务的飞行动作时，获取目标飞行器在每个计划飞行轨迹点的第一位置信息；
接收航管广播信号，获取所述航管广播信号中的若干条航管信息，提取每条航管信息中每个第一避让飞行器的飞行状态信息，基于所述飞行状态信息，确定每个第一避让飞行器的第二位置信息；
驱使无线扩频设备发送第一扩频信号，根据每个第二避让飞行器反馈的第二扩频信号，确定每个第二避让飞行器的第三位置信息；
根据目标飞行器的第一位置信息、每个第一避让飞行器的第二位置信息和每个第二避让飞行器的第三位置信息，执行目标飞行任务的防撞检测，获得防撞检测结果，根据所述防撞检测结果调整目标飞行器的计划飞行轨迹。
可选的，获取目标飞行器的目标飞行任务，提取所述目标飞行任务中的计划飞行轨迹步骤，具体包括：
访问飞行任务数据库；其中，所述飞行任务数据库中存储有若干个飞行任务，每个飞行任务包括执行飞行任务的飞行器标识信息和计划飞行轨迹，每个计划飞行轨迹包括若干个计划飞行轨迹点；
根据目标飞行器的飞行器标识信息，在所述飞行任务数据库中匹配目标飞行任务，提取所述目标飞行任务中的计划飞行轨迹。
可选的，在检测到目标飞行器执行所述目标飞行任务的飞行动作时，获取目标飞行器在每个计划飞行轨迹点的第一位置信息步骤，具体包括：
在检测到目标飞行器执行目标飞行任务的飞行动作时，执行北斗解析线程，对接收到的北斗NMEA标准报文进行解析；
根据NMEA的字符串格式，提取出北斗NMEA标准报文中目标飞行器的第一位置信息；其中，所述第一位置信息包括目标飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
可选的，接收航管广播信号，获取所述航管广播信号中的若干条航管信息，提取每条航管信息中每个第一避让飞行器的飞行状态信息，基于所述飞行状态信息，确定每个第一避让飞行器的第二位置信息步骤，具体包括：
接收航管广播信号，对所述航管广播信号中的I/Q数据进行解码，利用CRC校验确定数据正确性；
按照DF17的格式对航管广播信号进行解码，获得第一避让飞行器的飞行状态信息，提取出第二位置信息；其中，所述第二位置信息包括第一避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
可选的，驱使无线扩频设备发送第一扩频信号，根据每个第二避让飞行器反馈的第二扩频信号，确定每个第二避让飞行器的第三位置信息步骤，具体包括：
驱使无线扩频设备发送第一扩频信号；其中，所述第一扩频信号存储有位置信息共享请求指令；
接收第二避让飞行器在接收到第一扩频信号中的位置信息共享请求指令后，生成并反馈的包含有第二避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据的第二扩频信号；
提取所述第二扩频信号中的第三位置信息；其中，所述第三位置信息包括第二避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
可选的，根据目标飞行器的第一位置信息、每个第一避让飞行器的第二位置信息和每个第二避让飞行器的第三位置信息，执行目标飞行任务的防撞检测，获得防撞检测结果步骤，具体包括：
提取第一位置信息、第二位置信息和第三位置信息中的精度数据、纬度数据和高度数据，将所述精度数据、所述纬度数据和所述高度数据作为一组位置坐标在三维坐标系中进行绘制；
计算目标飞行器与第一避让飞行器和第二避让飞行器中每个避让飞行器之间的间隔距离，将所述间隔距离小于第一预设距离值且不小于第二预设距离值的第二避让飞行器作为候选避让飞行器，将所述间隔距离小于第二预设距离值的第二避让飞行器和间隔距离小于第一预设距离值的第一避让飞行器作为预警避让飞行器；
其中，所述第一预设距离值大于所述第二预设距离值。
可选的，根据所述防撞检测结果调整目标飞行器的计划飞行轨迹步骤，具体包括：
根据每个候选避让飞行器和预警避让飞行器的飞行速度数据、标准采样时间周期和对应的精度数据、纬度数据、高度数据，以保持当前标准采样时间周期的采样点的飞行姿态为基准，估测候选避让飞行器和预警避让飞行器在下一标准采样时间周期的采样点的预测位置信息；
以目标飞行器在下一标准采样时间周期的采样点的计划位置与每个预警避让飞行器的预测位置的间隔距离皆不小于第一预设距离值为第一约束条件，以目标飞行器在下一标准采样时间周期的采样点的计划位置与每个候选避让飞行器的预测位置的间隔距离不小于第二预设距离值的比例大于预设安全飞行占比比例为第二约束条件，以目标飞行器在下一标准采样时间周期的采样点的计划位置与计划飞行轨迹点的距离最小为目标，优化求解获得目标飞行器在下一标准采样时间周期的采样点的计划位置；
将所述计划位置替换到目标飞行器的计划飞行轨迹中下一标准采样时间周期的采样点的计划飞行轨迹点，以调整目标飞行器的计划飞行轨迹；
在每个位置估测优化周期，统计候选避让飞行器和预警避让飞行器在下一标准采样时间周期的采样点的预测位置信息与实际位置信息的误差距离超过目标误差距离阈值的比例；
若该比例超过标准比例区间的最大值，则按照标准调整比例调整所述标准采样时间周期，并重新生成计划飞行轨迹，若该比例低于标准比例区间的最小值，则按照标准调整比例调整所述标准采样时间周期，并重新生成计划飞行轨迹。

需要说明的是，目前，针对低空飞行器的防撞控制，通常面临着以下问题：（1）现有技术严重依赖布设大量基站的通讯网络环境，在缺少基站的区域，由于飞行器之间无法进行位置共享与信息同步，使得飞行器之间的防撞避让控制变得异常困难；（2）不同飞行器在执行飞行任务时，为了保证相互之间的飞行安全，需要留出安全间隔距离，而由于飞行器的飞行轨迹存在不确定性（航管飞机通常仅公布起始位置和已飞行的轨迹，并不会公开未飞行的轨迹；非航管飞机通常用于执行临时任务，也不会公开整个飞行轨迹），使得飞行器在规划飞行路径时存在着较大的限制，影响目标飞行器的飞行轨迹规划合理性，无法平衡目标飞行器的飞行安全与飞行效率之间的矛盾。
为了解决上述问题，本实施例通过提取目标飞行任务的计划飞行轨迹，检测目标飞行器在每个计划飞行轨迹点的第一位置信息，利用接收的航管广播信号，提取航管类飞行器的第二位置信息，根据发射并接收的反馈扩频信号，确定非航管类飞行器的第三位置信息，基于目标飞行器的第一位置信息、避让飞行器的第二位置信息和第三位置信息，执行计划飞行轨迹中每个计划飞行轨迹点的防撞检测与预警，同时，通过估测每个避让飞行器在下一标准采样时间周期的采样点的预测位置，考虑目标飞行器与不同防撞预警等级的避让飞行器在下一标准采样时间周期的采样点的不同安全间隔要求，以目标飞行器在下一计划飞行轨迹点最小偏移为目标，调整目标飞行器的计划飞行轨迹，提高了低空飞行器的防撞避让有效性、飞行安全性以及飞行轨迹规划合理性。
在优选的实施例中，获取目标飞行器的目标飞行任务，提取所述目标飞行任务中的计划飞行轨迹步骤，具体包括：
S110：访问飞行任务数据库；其中，所述飞行任务数据库中存储有若干个飞行任务，每个飞行任务包括执行飞行任务的飞行器标识信息和计划飞行轨迹，每个计划飞行轨迹包括若干个计划飞行轨迹点；
S120：根据目标飞行器的飞行器标识信息，在所述飞行任务数据库中匹配目标飞行任务，提取所述目标飞行任务中的计划飞行轨迹。
本实施例中，通过预先在飞行任务数据库中存储若干个飞行任务，然后利用接收到的目标飞行器的飞行器标识信息，在飞行任务数据库中匹配目标飞行任务，进而获取到目标飞行器当前需要执行飞行任务的计划飞行轨迹。
在优选的实施例中，在检测到目标飞行器执行所述目标飞行任务的飞行动作时，获取目标飞行器在每个计划飞行轨迹点的第一位置信息步骤，具体包括：
S210：在检测到目标飞行器执行目标飞行任务的飞行动作时，执行北斗解析线程，对接收到的北斗NMEA标准报文进行解析；
S220：根据NMEA的字符串格式，提取出北斗NMEA标准报文中目标飞行器的第一位置信息；其中，所述第一位置信息包括目标飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
更进一步的，接收航管广播信号，获取所述航管广播信号中的若干条航管信息，提取每条航管信息中每个第一避让飞行器的飞行状态信息，基于所述飞行状态信息，确定每个第一避让飞行器的第二位置信息步骤，具体包括：
S310：接收航管广播信号，对所述航管广播信号中的I/Q数据进行解码，利用CRC校验确定数据正确性；
S320：按照DF17的格式对航管广播信号进行解码，获得第一避让飞行器的飞行状态信息，提取出第二位置信息；其中，所述第二位置信息包括第一避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
更进一步的，驱使无线扩频设备发送第一扩频信号，根据每个第二避让飞行器反馈的第二扩频信号，确定每个第二避让飞行器的第三位置信息步骤，具体包括：
S410：驱使无线扩频设备发送第一扩频信号；其中，所述第一扩频信号存储有位置信息共享请求指令；
S420：接收第二避让飞行器在接收到第一扩频信号中的位置信息共享请求指令后，生成并反馈的包含有第二避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据的第二扩频信号；
S430：提取所述第二扩频信号中的第三位置信息；其中，所述第三位置信息包括第二避让飞行器的精度数据、纬度数据、高度数据和飞行速度数据。
本实施例中，通过提取目标飞行任务的计划飞行轨迹，检测目标飞行器在每个计划飞行轨迹点的第一位置信息，利用接收的航管广播信号，提取航管类飞行器的第二位置信息，根据发射并接收的反馈扩频信号，确定非航管类飞行器的第三位置信息，以此，为执行计划飞行轨迹中每个计划飞行轨迹点的防撞检测与预警提供数据支撑。
在优选的实施例中，根据目标飞行器的第一位置信息、每个第一避让飞行器的第二位置信息和每个第二避让飞行器的第三位置信息，执行目标飞行任务的防撞检测，获得防撞检测结果步骤，具体包括：
S510：提取第一位置信息、第二位置信息和第三位置信息中的精度数据、纬度数据和高度数据，将所述精度数据、所述纬度数据和所述高度数据作为一组位置坐标在三维坐标系中进行绘制；
S520：计算目标飞行器与第一避让飞行器和第二避让飞行器中每个避让飞行器之间的间隔距离，将所述间隔距离小于第一预设距离值且不小于第二预设距离值的第二避让飞行器作为候选避让飞行器，将所述间隔距离小于第二预设距离值的第二避让飞行器和间隔距离小于第一预设距离值的第一避让飞行器作为预警避让飞行器；
其中，所述第一预设距离值大于所述第二预设距离值。
更进一步的，根据所述防撞检测结果调整目标飞行器的计划飞行轨迹步骤，具体包括：
S530：根据每个候选避让飞行器和预警避让飞行器的飞行速度数据、标准采样时间周期和对应的精度数据、纬度数据、高度数据，以保持当前标准采样时间周期的采样点的飞行姿态为基准，估测候选避让飞行器和预警避让飞行器在下一标准采样时间周期的采样点的预测位置信息；
S540：以目标飞行器在下一标准采样时间周期的采样点的计划位置与每个预警避让飞行器的预测位置的间隔距离皆不小于第一预设距离值为第一约束条件，以目标飞行器在下一标准采样时间周期的采样点的计划位置与每个候选避让飞行器的预测位置的间隔距离不小于第二预设距离值的比例大于预设安全飞行占比比例为第二约束条件，以目标飞行器在下一标准采样时间周期的采样点的计划位置与计划飞行轨迹点的距离最小为目标，优化求解获得目标飞行器在下一标准采样时间周期的采样点的计划位置；
S550：将所述计划位置替换到目标飞行器的计划飞行轨迹中下一标准采样时间周期的采样点的计划飞行轨迹点，以调整目标飞行器的计划飞行轨迹；
S560：在每个位置估测优化周期，统计候选避让飞行器和预警避让飞行器在下一标准采样时间周期的采样点的预测位置信息与实际位置信息的误差距离超过目标误差距离阈值的比例；
S570：若该比例超过标准比例区间的最大值，则按照标准调整比例调整所述标准采样时间周期，并重新生成计划飞行轨迹，若该比例低于标准比例区间的最小值，则按照标准调整比例调整所述标准采样时间周期，并重新生成计划飞行轨迹。
