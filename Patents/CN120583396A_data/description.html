Description

低空通信协同网装机方法、装置、飞行器以及存储介质
技术领域
本申请涉及低空通信网络技术领域，尤其涉及低空通信协同网装机方法、装置、飞行器以及存储介质。
背景技术
随着无人机(UAV)、直升机和其他低空飞行器的广泛应用，构建一个稳定、高效的600米以下低空通信协同网络变得尤为重要。此类通信网络不仅支持对各类飞行器进行实时监控和管理，同时也为应急响应、物流配送、农业监测以及城市空中交通等多个领域提供了关键的技术支撑。
然而，在600米以下高度层内，由于地形复杂性、建筑物遮挡、电磁干扰等因素的影响，实现稳定可靠的通信覆盖成为一大挑战。现有的地面移动通信网络设计主要针对地表用户，并未充分考虑低空环境下的特殊需求，如快速移动目标的追踪、高密度飞行器的同时接入等。因此，需要一种专门针对低空区域优化设计的通信解决方案来满足这些特定要求。
发明内容
本申请的主要目的在于提供一种低空通信协同网装机方法、装置、飞行器以及存储介质，旨在解决如何实现飞行器在低空环境中的通信能力和安全性的技术问题。
为实现上述目的，本申请提出一种低空通信协同网装机方法，所述低空通信协同网装机方法应用于具有低空飞行能力的飞行器，所述低空通信协同网装机方法包括：
在所述飞行器上安装接地保护组件，所述接地保护组件通过导电材料与飞行器机体连接；

当检测到所述飞行器的燃油或液压系统中的第一弹性密封件不符合预设密封性能要求时，拆除所述第一弹性密封件，并安装第二弹性密封件；
基于所述接地保护组件的安装、所述电池组件的更换以及密封件的更换，完成所述飞行器的低空通信协同网络的装机。

本申请实施例的主要解决方案是：在飞行器上安装接地保护组件，该接地保护组件通过导电材料与飞行器机体连接；对飞行器的ELT主机进行电池组件更换操作，该电池组件用于在紧急情况下持续发送信号；当检测到所述飞行器的燃油或液压系统中的第一弹性密封件不符合预设密封性能要求时，拆除第一高弹性密封件，并安装第二高弹性密封件；基于接地保护组件的安装、电池组件的更换以及高弹性密封件的更换，完成飞行器的低空通信协同网络的装机。
在本实施例中，为便于表述，以下以具有低空飞行能力的飞行器为执行主体进行阐述。
在600米以下高度层内，由于地形复杂性、建筑物遮挡、电磁干扰等因素的影响，实现稳定可靠的通信覆盖成为一大挑战。现有的地面移动通信网络设计主要针对地表用户，并未充分考虑低空环境下的特殊需求，如快速移动目标的追踪、高密度飞行器的同时接入等。因此，需要一种专门针对低空区域优化设计的通信解决方案来满足这些特定要求。
本申请提供一种解决方案，首先在飞行器上安装接地保护组件，所述接地保护组件通过导电材料与飞行器机体连接，以防止静电积累和雷击对航电系统的损害；然后对飞行器的应急定位发射器(ELT)主机进行电池组件更换操作，确保在紧急情况下能够持续发送定位信号，提高搜救效率；随后当检测到所述飞行器的燃油或液压系统中的第一弹性密封件不符合预设密封性能要求时，拆除第一高弹性密封件，并安装第二高弹性密封件，以保障系统密封性能，提升飞行安全性；最后，基于上述接地保护组件的安装、电池组件的更换以及密封件的更换，完成飞行器的低空通信协同网络的装机。通过上述方法，各类低空飞行器能够在600米以下各高度层实施飞行回传的数据和性能，从而更好地支持构建低空通信协同网络。
需要说明的是，本实施例的执行主体可以是一种具有数据处理、网络通信以及程序运行功能的计算服务设备，例如无人机、直升机等，或者是一种能够实现上述功能的飞行器等。以下以空客直升机为例，对本实施例及下述各实施例进行说明。
基于此，本申请实施例提供了一种低空通信协同网装机方法，参照图1，图1为本申请低空通信协同网装机方法第一实施例的流程示意图。
本实施例中，所述低空通信协同网装机方法包括步骤S10～S40：
步骤S10，在所述飞行器上安装接地保护组件，所述接地保护组件通过导电材料与飞行器机体连接；
需要说明的是，接地保护组件是指飞信器上的接地保护组件，其通过导电材料与飞行器机身进行可靠连接，具备释放静电和抵御雷击的功能，用于保障航电系统的安全运行。
可以理解的是，由于于低空飞行环境中存在较强的电磁干扰，且飞行器在飞行过程中易积累静电、甚至遭遇雷击，可能对航电系统造成严重损害，影响通信稳定性与飞行安全性，因此执行步骤S10，可以在飞行器上构建有效的接地路径，防止静电积累和雷击电流对关键电子设备的破坏，从而提升飞行器在复杂电磁环境中的适应能力与通信可靠性。
在一种可行的实施方式中，步骤S10可以包括步骤S11～S13：
步骤S11，标记并存档接地保护旧组件的线缆连接顺序，并根据所述线缆连接顺序设置用于释放螺栓的交叉顺序；
具体地，在拆除接地保护旧组件之前，使用标签或记录笔标记每根线缆的连接位置，并通过拍照的方式记录线缆连接顺序，确保后续重新连接时不会出现错误。
然后，根据线缆连接顺序确定螺栓的释放顺序，通常采用交叉顺序(如对角线顺序)来均匀释放螺栓，以避免因单侧受力导致组件变形或损坏。
步骤S12，按照所述交叉顺序释放螺栓至预设扭矩，并使用铜刷清理所述接地保护旧组件的蒙皮接触面直至满足预设表面粗糙度衡量标准；
在确定螺栓的释放顺序为交叉顺序后，使用扭矩扳手按照预先设置的交叉顺序松开螺栓。由于初始释放扭矩设置为5N·m，因此逐步减小螺栓的紧固力，直至完全松开。此过程应缓慢且均匀进行，以防止组件突然移动或损坏。
在拆除接地保护旧组件后，使用铜刷(NASM20427)清理蒙皮接触面，去除氧化层和杂质。其中，蒙皮接触面是指飞行器机体表面与接地保护组件直接接触的部分。
在清理过程中需确保该蒙皮接触面达到规定的表面粗糙度标准(Ra≤3.2μm)，以保证后续导电层的附着力和电气性能。其中，可以使用表面粗糙度测试仪(如Fluke 1507)进行测量，确保清理后的接触面符合要求。
步骤S13，对满足所述预设表面粗糙度衡量标准的接地保护组件进行接地处理、定位安装以及线缆重构，完成接地保护新组件的安装处理。
在清理后的接触面上喷涂厚度为0.05±0.01mm的导电层。并将喷涂后的接地保护组件置于温度25℃、相对湿度50％的环境中静置30分钟，以确保导电层充分固化。
然后，使用定位销对齐接地保护新组件的位置，确保其与飞行器机体的精确配合。接着使用螺栓将接地保护新组件预紧到指定位置(预紧程度为10N·m)，并逐步增加扭矩至最终值。其中，最终值包括：
主承力螺栓(M10)：最终扭矩应为28±2N·m；
辅助螺栓(M8)：最终扭矩应为15±1N·m。
并且，扭矩分三次递增(30％、60％和100％)，以确保螺栓均匀受力，避免因一次性拧紧导致的应力集中问题。
随后，对接地线端子进行压线，确保压接力达到或超过3000psi，并将屏蔽层360°环接到接地线端子上(符合MIL-DTL-38999标准)，以增强电磁兼容性(EMC)，减少外界电磁干扰对信号传输的影响。
最后，根据预设的扭矩值，将所有螺栓扭紧至最终值，确保接地保护新组件能够牢固地固定在飞行器机体上。
通过上述步骤，可以确保接地保护新组件在飞行器上的正确安装，从而有效防止静电累积和雷击对航电系统的损害，提升飞行器的安全性和通信可靠性。
步骤S20，对所述飞行器的应激定位发射器ELT主机进行电池组件更换操作，所述电池组件用于在紧急情况下持续发送信号；
需要说明的是，ELT主机是指安装于飞行器上的应急定位发射器，其主要功能是在飞行器发生事故或其他紧急状况时自动启动，并通过国际卫星搜救组织(COSPAS-SARSAT)系统发送定位信号，以便搜救人员快速确定失事飞行器的位置。
所述电池组件包括主电池(Lithium SOCl□12V)以及备用电池(Ni-Cd10.5V)。其中，主电池用于为ELT主机提供正常工作电压及足够的能量以支持至少48小时的连续信号发射，备用电池则作为冗余电源，在主电池失效或电量不足时自动接入，确保ELT主机持续运行。
由于直升机在长期运行过程中，ELT主机的电池存在自然老化、容量衰减甚至漏液等风险，因此执行步骤S20，可以有效保障ELT主机在飞行器遭遇突发状况时具备可靠的电力供应，从而提升飞行器的应急响应能力和遇险后搜救效率。
在一种可行的实施例方式中，步骤S20可以包括步骤S21～S23：
步骤S21，断开所述ELT主机的直流供电电源，并测量ELT主机内部电路的残余电压是否小于预设电压；
为保障更换电池组件过程中的电气安全，在开始更换电池组件操作之前，首先将ELT主机与飞行器主电源系统完全隔离。具体而言，首先关闭飞行器中与ELT主机相关的直流供电开关，并使用绝缘工具断开ELT主机与飞行器配电系统的连接线路。
然后，采用高阻抗数字万用表或专用测试设备，对ELT主机内部电路进行残余电压检测；判断所测得的残余电压是否低于预设安全电压阈值(例如小于1V)，以确保后续拆卸和安装操作不会因残留电能造成短路、电击或损坏电子元件。
步骤S22，若所述ELT主机的残余电压小于所述预设电压，则移除所述主电池的绝缘片，将所述主电池的插头插入对应接口，并等待所述主电池的LED指示灯转为绿色常亮，以确认所述主电池激活完成；
需要说明的是，主电池作为ELT主机的主要能源供应单元，其激活状态直接决定了ELT主机能否在紧急情况下持续发送定位信号至少48小时。
若检测到残余电压低于预设安全电压阈值，则开始主电池的安装操作。
具体而言，首先拆卸电池航盖，其中该电池舱盖被4颗螺钉扭紧，其扭矩为0.8N·m。同时，拔出旧电池插头。
然后，移除主电池出厂时用于防止意外激活的绝缘片，并将主电池的插头准确插入ELT主机上的指定电源接口，确保插接牢固且无反接。在插入主电池的插头到ELT主机上后，根据主电池的状态指示灯进行激活状态的判断，例如，若状态指示灯显示红色，表示正在初始化或电压未稳定；当LED转为绿色常亮状态，表明主电池已成功激活并与ELT主机建立正常通信。
步骤S23，在所述ELT主机的端口接入备用电池线束，以连接所述备用电池。
需要说明的是，备用电池作为冗余电源，用于在主电池失效或电量不足时自动投入运行，确保ELT主机始终具备应急通信能力。
具体地，当主电池安装完成后，在ELT主机的J2端口接入备用电池线束(线束固定间距≤150mm)，并确保线束插接牢固，并使用卡扣或绑扎带固定，防止因振动导致松脱。最后使用该电池线束连接ELT主机与备用电池。
通过上述步骤，可以有效保障ELT主机在飞行器发生突发状况时具备可靠的电力供应，从而提升飞行器的应急响应能力和遇险后搜救效率，是实现飞行器低空通信协同网络中关键安全功能的重要支撑环节。
步骤S30，当检测到所述飞行器的燃油或液压系统中的第一密封件不符合预设密封性能要求时，拆除所述第一密封件，并安装第二密封件；
需要说明的是，密封件是指具有优异弹性和耐压性能的非金属密封元件，应用于飞行器燃油系统与液压系统的管路连接、作动筒、阀门、泵等关键部位，用于防止液体泄漏并维持系统压力稳定。
需要理解的是，由于飞行器燃油及液压系统在高空、高速、振动频繁的复杂环境下运行，对密封性能要求极高，任何微小的泄漏都可能引发严重的安全事故，因此执行步骤S30，可以有效恢复燃油或液压系统的密封完整性，提升其在极端工况下的可靠性与安全性，保障飞行器在低空通信协同网络中各项功能的稳定运行。
在一种可行的实施例方式中，步骤S30可以包括步骤S31～S33：
步骤S31，当检测到所述飞行器的燃油或液压系统中的第一密封件不符合预设密封性能要求时，对所述第一密封件的待拆除区域进行清洁处理，并将所述第一密封件进行拆除；
具体地，在更换飞行器燃油或液压系统中的高弹性密封件之前，检测第一密封件是否符合预设密封性能要求，即检测该第一密封件是否存在存在老化、变形、泄漏等情况，若存在，首先对第一密封件所在的安装区域无腐蚀性清洁剂(如异丙醇或航空级溶剂)对密封槽及其周边区域进行擦拭，去除油污、灰尘、老化残渣及其他杂质。
在完成清洁后，使用专用拔具(SST－PLUG－02)沿轴向均匀施力，将第一密封件从密封槽中取出。
步骤S32，检查密封槽是否有损坏，若否，则将经过润滑处理后的第二密封件降温至预设温度；
完成第一密封件拆除后，需对密封槽状态进行检测，检查密封槽是否有划伤(深度＞0.1mm需按SRM修复)。
在密封槽状态良好的前提下，开始准备新密封件。首先，在第二密封件外表面涂覆厚度为5-10μm的全氟聚醚润滑脂(MIL-G-27617B)，以减少装配过程中的摩擦阻力。
然后，将润滑后的密封件置于低温环境中，使该密封件降温至-10℃，使其体积收缩后便于顺利装入到密封槽。该降温处理利用了橡胶材料的热胀冷缩特性，有助于实现过盈配合下的顺畅安装，同时避免因强行压装造成的密封件扭曲或撕裂。
步骤S33，使用导向套管将降温后的第二密封件垂直压入所述密封槽中，完成第二密封件的更换。
首先准备与密封件尺寸匹配的导向套管，该套管内壁应光滑且无锐角，防止损伤密封件。然后将已完成降温处理的第二密封件放置在密封槽上方，调整位置使其与槽口对齐。使用导向套管对准密封件中心，并通过手动或气动压装工具缓慢、匀速地将密封件垂直压入密封槽。在压装过程中应保持施力方向与密封槽轴线一致，防止偏斜导致密封件皱褶或错位。
当密封件完全嵌入密封槽后，撤除导向套管，并目视检查密封件是否平整贴合、有无翻边或扭曲；
安装完成后，按照十字交叉顺序分3级拧紧盖板螺栓，包括：将螺栓预紧2N·m，随后递增至2N·m，最终达到扭矩8±0.5N·m。
最后，检查迷瞪见的层外翻量是否大于或等于预设阈值(例如0.5mm)。
通过上述步骤，可以有效保障飞行器燃油或液压系统的密封完整性，提升其在极端环境下的运行可靠性，从而为低空通信协同网络中飞行器的安全稳定运行提供坚实支撑。此外，为了确保飞行器燃油系统中更换后的高弹性密封件具备良好的密封性能，在完成第二密封件安装后需执行系统的泄漏测试。该测试分为两个阶段：静态测试和动态测试，分别用于检测微观渗漏与结构承压能力。
其中，静态测试包括：在燃油系统中注入红色荧光染料，确保染料在系统中循环分布均匀，并完全覆盖所有密封区域。随后使用紫外灯对密封圈及其周边区域进行照射检查。若存在微渗漏，染料将在紫外光下呈现明显荧光反应，指示泄漏点位置；若密封圈周围无任何荧光渗出痕迹，表明密封界面完整，未发生液体微泄漏。
动态测试包括：将燃油系统封闭，并使用气源或液压泵将系统压力提升至35kPa(表压)。然后关闭截止阀，停止加压后开始计时，保持压力稳定续15分钟，期间观察并记录系统压力变化情况，计算压降值。若保压期间系统压降不超过0.5kPa，则表明密封结构具备足够的耐压能力和系统完整性。
通过上述两个阶段的泄漏测试流程，可以全面评估飞行器燃油系统中高弹性密封件的密封性能。静态测试利用荧光染料与紫外灯检测微观渗漏，动态测试则通过加压保压方式验证系统在模拟工况下的稳定性。两项测试均满足预设合格标准后，方可确认密封件更换作业完成且系统恢复至适航状态。
