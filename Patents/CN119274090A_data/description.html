Description

低空飞行器基于海上建筑飞行环境识别监测方法及系统
技术领域
本申请涉及低空飞行器的技术领域，尤其涉及低空飞行器基于海上建筑飞行环境识别监测方法及系统。
背景技术
在低空飞行器的飞行过程中，需要对飞行环境进行识别，以便于更好的进行导航、避障、定位等，更有利于低空飞行器的执行相关任务。目前的低空飞行器对于飞行环境的识别一般依托于雷达、计算机视觉等技术，仅依赖于雷达和计算机视觉等技术在简单、静态的飞行环境中足以支撑飞行任务的执行实现，然而在复杂、动态的飞行环境中例如海上建筑中，基于雷达、计算机视觉等技术识别飞行环境往往难以良好支撑低空飞行器的飞行任务的执行。
发明内容
本申请提供了低空飞行器基于海上建筑飞行环境识别监测方法及系统，其有利于在海上建筑的飞行环境中支撑低空飞行器的飞行任务更好的执行。
第一方面，本申请提供了低空飞行器基于海上建筑飞行环境识别监测方法。所述方法包括：
获取飞行环境监测信息；
基于预构建的建筑物特征库，识别所述飞行环境监测信息中的特别建筑物特征，并获取特别建筑物特征的建筑物状态信息；
根据所述特别建筑物特征和建筑物状态信息确定特别建筑物特征的建筑物影响空间；
结合所述飞行环境监测信息和建筑物影响信息确定环境监测结果信息，所述环境监测结果信息包括飞行环境空间的空间状态信息，所述空间状态信息为允许飞行或禁止飞行。
通过采用上述技术方案，能够对包含海上建筑的飞行环境中的特别建筑物特征进行识别，进而智能确定特别建筑物特征的建筑物影响空间，有利于更为智能合理的确定飞行环境空间的空间状态，进而有利于支撑低空飞行器飞行过程中更好的路线规划、避障等，避免海上建筑物和低空飞行器相互影响，即有利于支撑低空飞行器飞行任务更好的执行。
进一步地，所述基于预构建的建筑物特征库，识别所述飞行环境监测信息中的特别建筑物特征，并获取特别建筑物特征的建筑物状态信息包括：
基于所述飞行环境监测信息，构建环境障碍物模型，并确定环境障碍物模型的运动状态信息，所述运动状态信息为运动中状态或静止中状态；
基于所述建筑物特征库，在所述环境障碍物模型中识别特别建筑物特征，所述建筑物特征库中包含建筑物特征模型，若所述环境障碍物为运动中状态且所述环境障碍物模型与所述建筑物特征模型的模型相似度高于第一相似度阈值，则确定所述环境障碍物为特别建筑物特征，若所述环境障碍物为静止中状态且所述环境障碍物模型与所述建筑物特征模型的模型相似度高于第二相似度阈值，则确定所述环境障碍物为特别建筑物特征，所述第一相似度阈值低于第二相似度阈值；
根据环境障碍物模型的运动状态信息确定特别建筑物特征的建筑物状态信息。
进一步地，确定环境障碍物模型与建筑物特征模型的模型相似度的方法包括：
在环境障碍物模型和建筑物特征模型中识别相同轮廓特征点；
使相同轮廓特征点对齐，确定环境障碍物模型与建筑物特征模型的模型重合体积和模型未重合体积；
计算模型重合体积的平方减去模型未重合体积的结果再除以模型重合体积的平方为所述模型相似度。
进一步地，所述根据环境障碍物的运动状态信息确定特别建筑物特征的建筑物状态信息包括：

进一步地，所述根据所述特别建筑物特征和建筑物状态信息确定特别建筑物特征的建筑物影响空间包括：
根据所述建筑物状态信息确定预设时长内的建筑物动作空间；
确定所述建筑物动作空间为建筑物影响空间。

第一方面，本申请提供了低空飞行器基于海上建筑飞行环境识别监测方法。该方法可以由低空飞行器执行。
低空飞行器一般为无人机，其搭载有雷达以及摄像头，以便于对飞行环境进行监测，得到飞行环境监测信息，再对飞行环境监测信息进行处理，即可得到可感知的飞行环境空间的空间状态信息，区分飞行环境空间的空间状态信息有利于更好的进行飞行路线规划，避免低空飞行器和海上建筑之间相互影响。
图1示出了本申请实施例中的低空飞行器基于海上建筑飞行环境识别监测方法的流程图。
参照图1，该方法具体包括以下步骤：
S110：获取飞行环境监测信息。
在本申请实施例中，飞行环境监测信息包括雷达监测信息和监测视频信息，结合雷达监测信息和监测视频信息可以综合体现低空飞行器周围飞行空间的情况。
S120：基于预构建的建筑物特征库，识别所述飞行环境监测信息中的特别建筑物特征，并获取特别建筑物特征的建筑物状态信息。
本步骤的方法具体包括：基于所述飞行环境监测信息，构建环境障碍物模型，并确定环境障碍物模型的运动状态信息，所述运动状态信息为运动中状态或静止中状态；基于所述建筑物特征库，在所述环境障碍物模型中识别特别建筑物特征，所述建筑物特征库中包含建筑物特征模型，若所述环境障碍物为运动中状态且所述环境障碍物模型与所述建筑物特征模型的模型相似度高于第一相似度阈值，则确定所述环境障碍物为特别建筑物特征，若所述环境障碍物为静止中状态且所述环境障碍物模型与所述建筑物特征模型的模型相似度高于第二相似度阈值，则确定所述环境障碍物为特别建筑物特征，所述第一相似度阈值低于第二相似度阈值；根据环境障碍物模型的运动状态信息确定特别建筑物特征的建筑物状态信息。
环境障碍物模型为三维模型，利用飞行环境监测信息进行三维重建，可以得到环境障碍物模型，并且基于时序能够确定环境障碍物模型的运动状态信息。建筑物特征库中预存储有建筑物特征模型，并且存储有建筑物特征模型的动作模式，如此比较环境障碍物模型和建筑物特征模型的模型相似度，即可确定环境障碍物模型符合哪一建筑物特征模型，由于海上建筑物在工作时可能会搭载货物或人员，搭载货物或人员也会被识别为环境障碍物特征的一部分，故若环境障碍物模型处于静止中状态，则需要较高的第二相似度阈值确定环境障碍物模型符合建筑物特征模型，若环境障碍物模型处于运动中状态，则仅需要较低的第一相似度阈值即可确定环境障碍物模型符合建筑物特征模型，如此即可确定环境障碍物模型中的特别建筑物特征。特别建筑物特征表征能够动作的海上建筑。
在本步骤的方法中，确定环境障碍物模型与建筑物特征模型的模型相似度的方法包括：在环境障碍物模型和建筑物特征模型中识别相同轮廓特征点；使相同轮廓特征点对齐，确定环境障碍物模型与建筑物特征模型的模型重合体积和模型未重合体积；计算模型重合体积的平方减去模型未重合体积的结果再除以模型重合体积的平方为所述模型相似度。
在本步骤的方法中，所述建筑物状态信息包括建筑物形态和建筑物动作；根据环境障碍物模型确定特别建筑物特征的建筑物形态，根据建筑物形态和环境障碍物模型的运动状态确定建筑物动作。由于海上建筑物的动作模式预存储于建筑物特征库中，在确定建筑物状态信息即确定建筑物形态(当前姿态)和建筑物动作后，可以结合动作模式预判后续的建筑物动作，即确定完整的建筑物状态信息。
S130：根据所述特别建筑物特征和建筑物状态信息确定特别建筑物特征的建筑物影响空间。
本步骤的方法具体包括：根据所述建筑物状态信息确定预设时长内的建筑物动作空间；确定所述建筑物动作空间为建筑物影响空间。由于建筑物状态信息包含建筑物形态和建筑物动作，故能够结合未来一段时间内的建筑物动作来确定建筑物动作空间，进而确定建筑物影响空间。
S140：结合所述飞行环境监测信息和建筑物影响信息确定环境监测结果信息，所述环境监测结果信息包括飞行环境空间的空间状态信息，所述空间状态信息为允许飞行或禁止飞行。
在本步骤的方法中，一般将固定的环境障碍物模型影响的空间和特别建筑物特征的建筑物影响空间均标记为禁止飞行状态，其他空间可以标记为允许飞行状态，如此即可实现海上建筑物飞行环境的环境监测结果信息。利用环境监测结果信息可以指导低空飞行器的避障以及飞行路径规划等操作，有利于低空飞行器更好的飞行，进而有利于飞行任务的实现。
