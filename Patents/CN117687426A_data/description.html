Description

一种低空环境下的无人飞行器飞行控制方法和系统
技术领域
本发明涉及计量技术领域，具体的，涉及一种低空环境下的无人飞行器飞行控制方法和系统。
背景技术
目前监控无人飞行器行业发展迅速，引入了大量智能移动终端，用于代替人工实现自动检查监控无人飞行器部件状况。监控无人飞行器部件巡检距离远，位置高，在高电压下危险系数大，而人工巡检耗时耗力，效率较低。由于巡检的结果需及时存储，工作人员常携带照相机和GPS导航仪等设备，上传至监控无人飞行器维修中心存档，这在交通不发达的地方无疑增加巡检难度，同时监控无人飞行器部件故障不明，人工故障检测检查十分危险，也易造成人员受伤。目前图像可视化行业发展迅猛，它在监控无人飞行器故障检测上面也有着很多的应用，但普遍方案所使用的双目视觉系统和机载多线雷达过于昂贵，增加了巡检成本。
然而由于自然、人为等多种因素，监控无人飞行器系统的故障是不可避免的，造成对用户施工的中断或施工质量的下降，甚至损坏监控无人飞行器设备。另一方面，在故障检测规模和复杂程度不断增加的同时，监控无人飞行器系统自动化水平也在不断加强，越来越多的自动装置应用到监控无人飞行器系统，当故障检测发生故障时无法第一时间有效的判断出故障地点造成了很大的损失。
