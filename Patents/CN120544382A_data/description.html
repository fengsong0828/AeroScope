Description

低空飞行器与智慧高速路侧单元信息交互的方法与系统
技术领域
本发明涉及信息交互领域，具体来说，特别涉及低空飞行器与智慧高速路侧单元信息交互的方法与系统。
背景技术
随着经济社会的快速发展和低空经济政策的持续发布，低空经济作为战略性新兴产业，将迎来蓬勃发展。低空经济涵盖了无人机配送、航空旅游、空中出租车、农业航空作业等多个领域，未来这一产业将成为全球经济的重要组成部分。随着技术的不断进步，低空飞行器也逐渐成为支撑低空经济发展的重要支柱，预计将在交通、物流、农业等行业发挥越来越关键的作用。
尽管低空飞行器在促进低空经济发展方面具有巨大潜力，但在安全监管方面仍然面临巨大挑战。传统的航空器通常配备完整的通信系统、导航系统以及数据处理平台，能够实时监控飞行状态，确保飞行安全。而低空飞行器作为新兴的飞行工具，通常缺乏完善的通信系统和数据分析能力。许多低空飞行器依赖于驾驶人员或操控人员的主观判断，缺乏对飞行器实时状态、周围环境以及可能出现的飞行风险进行全面的数据监控和预警。这种依赖人工判断的飞行模式使得低空飞行器的安全性难以保障，特别是在多飞行器密集运营或复杂天气条件下，容易发生碰撞或误操作，导致安全隐患。
针对相关技术中的问题，目前尚未提出有效的解决方案。
发明内容
有鉴于此，本发明提供低空飞行器与智慧高速路侧单元信息交互的方法与系统，以解决上述提及的问题。

需要说明的是，数据清洗是确保数据质量和一致性的关键步骤。对于飞行信息来说，清洗处理包括以下几个方面：
检查缺失值：检查飞行信息中的每一项是否有缺失(例如，飞行器的某个位置、速度、气象数据等)。

归一化：对不同量纲的数据进行归一化处理，尤其是在进行距离计算时，确保所有数据维度对计算的贡献是平等的。

S13、对于每个信息类别集合中的数据，利用多线程优化加密算法对每个分块数据进行前向加密处理，得到加密信息；
具体来说，优化加密算法为采用前向保密，即每次会话使用新密钥，提高数据安全性。具体流程包括：
1.初始化并计算公钥：端到端加密通信设为A端到B端，A和B两端共同确定公开参数P和G，对于A端，确定私钥a，对于B端，确定私钥b，a应满足1<a<P-1，b也应满足1<b<P-1，则A、B端公钥可按照下式计算：
A＝Ga\bmodP；
B＝Gb\bmodP；
其中，P为素数，G为整数，G是P的原根，P和G可公开。a和b为一个随机的非公开整数。
2.公钥交换与共享：A端将计算得到的公钥A发送到B端，B端将计算得到的公钥B发送到A端，完成公钥交换。A端基于公钥B和私钥a计算共享密钥K-{A}。B端基于公钥A和私钥b计算共享密钥K-{B}。
K-{A}＝Ba\bmodP＝(Gb)a\bmodP＝Gab\bmodP；
K-{B}＝Ab\bmodP＝(Ga)b\bmodP＝Gab\bmodP；
K-{A}＝K-{B}＝Gab\bmodP；
K-{A}等于K-{B}，A端与B端获得共享密钥K，利用共享密钥K可对端到端的通信数据进行加密和解密操作。由于每次会话A端和B端都会重新生成私钥，所以，即使之前的会话密钥被泄露，也不会影响到其他会话的安全性，实现了前向保密。

神经网络的训练过程是通过误差函数(通常是均方误差，MSE)来衡量预测值与实际值之间的差距，然后通过损失函数来优化网络的权重参数。

损失函数定义了神经网络的训练目标，通常选择MSE作为损失函数，目的是最小化位置估计的误差。
反向传播算法是神经网络训练的核心，通过梯度下降法来优化网络权重。通过计算误差对网络权重的梯度，并更新权重，网络逐步提高对位置坐标的预测精度。

信息传输模块1，用于通过密度划分算法对预先获取低空飞行器的飞行信息进行数据分块处理，利用端到端加密方法对每个数据分块进行加密处理，并将加密后的信息传输至智慧高速路侧单元；
飞行轨迹确定模块2，用于智慧高速路侧单元将加密后的信息进行解密处理，并结合多边测量法确定低空飞行器的实时飞行轨迹，将实时飞行轨迹传送至中控平台；
飞行调整模块3，用于将实时飞行轨迹与预设的飞行轨迹进行对比研判，并根据研判结果，利用改进的人工势场法对低空飞行器飞行轨迹进行优化，生成飞行优化指令，低空飞行器根据飞行优化指令执行飞行调整。
综上所述，借助于本发明的上述技术方案，本发明以智慧高速路侧单元这种具有强大边缘计算能力的硬件设施和软件系统相统一的设备，作为低空飞行器信息交互的媒介，在此基础上提出的一种低空飞行器与智慧高速路侧单元信息交互的方法和系统具有很强的前瞻性、创新性和市场性。本发明采用智慧高速路侧单元和中控平台对低空飞行器飞行轨迹和状态进行独立计算并对比分析，保证结果的可靠性，智慧高速路侧单元采用一套低空飞行器飞行轨迹和状态的分析方法，中控平台采用另外一种低空飞行器飞行轨迹和状态的分析方法，将二者的结果对比分析，结果一致，低空飞行器正常飞行，则继续保持现有飞行轨迹和状态，若结果不一致，或结果均表明低空飞行器偏航，则需要发布纠偏指令。本发明实现了低空飞行器与智慧高速路侧单元的信息交互，充分利用了智慧高速路侧单元设备，节省了需对低空飞行器建立信息连接的基础设施建设费用，降低了低空飞行器大规模应用的投资，从经济性角度大大助推了低空飞行器规模化应用进程，采用独立监测分析低空飞行器运行轨迹与状态的方法，避免了低空飞行器飞行异常情况的误判，减少了错误报警信息的发布，提高了系统运行效率。
