Description

一种适用于中低空飞行器航拍测量的图像矫正装置
技术领域
本发明属于飞行器航拍测量的技术领域，更具体的，本发明涉及一种适用于中低空飞行器航拍是图像矫正的技术领域。
背景技术
航拍测量，又称航空摄影测量，指的是在飞行器上用航摄仪器对地面连续摄取像片，结合地面控制点测量、调绘和立体测绘等步骤，绘制出地形图的作业。航空摄影测量可用于野外微地貌研究、林地考察、考古、文物修缮等领域。航空摄影测量的作业涉及到像片控制点联测，像片控制点一般是航摄前在地面上布设的标志点，也可选用如道路交叉点等像片上明显地物点，用测角交会、测距导线、等外水准、高程导线等普通测量方法测定其平面坐标和高程。
对航拍图像进行复原加工是航拍的一项重要工作，但是因为湿度、颗粒物等影响，航怕到的图像和实际实物颜色往往存在差别，这样导致人们对图像处理时很难还原实物的实际颜色。
发明内容
针对现有技术中存在的由于湿度、颗粒物等影响，航怕到的图像和实际实物颜色往往存在差别，这样导致人们对图像处理时很难还原实物的实际颜色问题，本发明提供了一种适用于中低空飞行器航拍测量的图像矫正装置，与现有技术相比，本发明提供的一种适用于中低空飞行器航拍测量的图像矫正装置结构简单，使用方便，容易识别，利用航拍飞行器先行对图像矫正装置进行抓取，根据抓取颜色和实际颜色的色差，对拍到的所有实物进行颜色调整，从而获得实物真实的颜色，在航拍领域具有广泛的实用性。
为了解决上述问题，本发明采用的技术方案如下：
本发明提供的一种适用于中低空飞行器航拍测量的图像矫正装置，包括承载板和三原色板，承载板上设置三原色板，三原色板包括黄色板、蓝色板和红色板，相邻三原色板之间设置分离槽，黄色板、蓝色板和红色板上分别设置对准点。
本发明中，对准点为三种不同形状的孔。
本发明中，分离槽将三原色板平均分成三份。
本发明中，承载板包括第一承载板、第二承载板和第三承载板，第一承载板和第二承载板之间设有支撑柱，第一承载板上设置三原色板，第二承载板上设置驱虫盒，第三承载板上设置水平仪，第一承载板中部设置通孔，通孔和驱虫盒对应设置。
本发明中，第二承载板下部设有四根连接杆，连接杆上设有两个活动关节，连接杆分成两组，每组连接杆上活动关节通过连接板连接，两个连接板中部设有螺杆，螺杆尾部设有手把。
本发明中，连接杆底部设有支腿，连接杆中部设有第三承载板。
本发明中，支腿与连接杆螺纹连接。
本发明中，支腿底部设有尖刺。
本发明中，第二承载板上设置折叠杆，折叠杆上部设有日光灯。
本发明采用的支撑板、三原色板、驱虫盒、支腿、折叠杆、日光灯、螺杆和水平仪均可通过公共渠道购买。
使用本发明提供的技术方案可以达到如下有益效果：
本发明提供的一种适用于中低空飞行器航拍测量的图像矫正装置，简单可靠，能够根据使用位置进行高度调整，根据飞行器状态和所拍摄物体的角度调整高度与角度，使用方便，而且能够准确抓取三种不同的三原色，方便航拍后对图像进行矫正，在航拍领域具有广泛的实用性。
