Description

基于低空飞行器舱体内智能空气净化器
技术领域
本实用新型属于空气净化器的设计及制造技术领域，具体涉及一种基于低空飞行器舱体内智能空气净化器。
背景技术
作品从低空飞行器内环境空气净化入手，通过滤芯装置对舱体内部的有害气体和大颗粒污染物进行净化。外形设计使其融入到低空飞行器内部构造中；电子模块实现了智能化自动控制；同时增加了电子屏蔽技术。在不干扰飞行器正常工作的同时，实现了空气净化的目的。一方面可以保障电子元器件的正常工作，另一方面可以保障飞行员的健康。
实用新型内容
本实用新型针对现有技术中的不足，提出一种基于低空飞行器舱体内智能空气净化器，具有结构简单，性能可靠，成本低廉，自动控制，净化效率高等优点。
本实用新型解决其技术问题所采用的方案是：
一种基于低空飞行器舱体内智能空气净化器，由支撑装置、滤芯装置和电子模块装置组成。
所述支撑装置包括外壳、百叶窗、防震架。外壳中安装有百叶窗、防震架、滤芯装置及电子模块装置。
所述滤芯装置包括箱体、活性炭、中效过滤棉、高效过滤棉、风机。风机安装在箱体上端。活性炭、中效过滤棉、高效过滤棉安装在箱体中部。
所述电子模块装置包括电源I、电源II、继电器、传感器、防电磁干扰罩。电源I安装在外壳底部，与继电器、传感器相连接，为继电器、传感器供电。电源II在外壳的外部，与风机相连接，为其供电。继电器安装在防电磁干扰罩内，一端与电源I相连接，另一端和传感器相连接。防电磁干扰罩安装在箱体外侧。传感器安装在外壳两侧，与继电器相连接。

具体实施方式
下面结合附图对本实用新型做进一步阐述：
本实用新型所述的是一种基于低空飞行器舱体内智能空气净化器，由支撑装置、滤芯装置和电子模块装置组成。
所述支撑装置包括外壳1、百叶窗9、防震架6。外壳1中安装有百叶窗9、防震架6、滤芯装置及电子模块装置。
所述滤芯装置包括箱体2、活性炭5、中效过滤棉3、高效过滤棉4、风机8。风机8安装在箱体2上端；活性炭5、中效过滤棉3、高效过滤棉4安装在箱体2中部。
所述电子模块装置包括电源I12、电源II13、继电器10、传感器7、防电磁干扰罩11。电源I12安装在外壳1底部，与继电器10、传感器7相连接，为继电器10、传感器7供电；电源II13在外壳1的外部，与风机8相连接，为其供电；继电器10安装在防电磁干扰罩11内，一端与电源I12相连接，另一端和传感器7相连接；防电磁干扰罩11安装在箱体2外侧；传感器7安装在外壳1两侧，与继电器10相连接。
具体实施方式为：当传感器7检测到空气中污染物浓度超过设定值时，传感器7向继电器10发出低电平信号，继电器10接收信号后，导通电源II13和风机8的回路。风机8开始工作，把污染空气由箱体2下部抽进滤芯装置中。经过中效过滤棉3、高效过滤棉4和活性炭5的过滤，污染空气被净化，由百叶窗9排出，达到空气净化的效果。当污染物浓度降低到设定值以下时，传感器7向继电器10发出高电平信号，继电器10接收信号后，断开电源II13和风机8的回路，风机8停止工作，空气净化完成。通过调整传感器7启动参数，从而满足不同环境中的要求。
防震架6具有减震功能，并且限制了滤芯装置的晃动，从而在运行中保持平衡；防电磁干扰罩11可以防止强磁的干扰和破坏，避免电子模块装置受到强磁破坏无法正常工作；中效过滤棉3和高效过滤棉4主要吸收大颗粒污染物，活性炭5主要吸收有害气体。
