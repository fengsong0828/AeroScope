Description

一种具备折叠机翼结构的便携式单人低空矢量喷射飞行器
技术领域
本发明属于飞行器领域，具体涉及一种具备折叠机翼结构的便携式单人低空矢量喷射飞行器。
背景技术
单人飞行器是指由人类制造，能够飞离地面并在空间中飞行，同时由个人控制的飞行设备。这类技术的起源可以追溯到早期的飞行梦想家和发明家，例如莱昂纳多·达·芬奇，他们对人类飞翔的可能性进行了设想。然而，直到20世纪中叶，随着航空技术的显著进步，单人飞行器的构想才逐渐变为现实。
随着飞行器普及，个人飞行器必然是未来发展方向。个人飞行器不仅能够提供快速便捷的空中出行方式，还能在特定场合如紧急救援、军事侦察等方面发挥重要作用。传统的个人飞行器主要包括火箭背包、喷气背包、动力滑翔翼等。
现有技术中，单人飞行器普遍存在体积庞大的特点，大多数个人飞行器由于其结构设计，体积较大，难以方便地携带和存放，在携带和存放是需要占用巨大的空间，限制了其在日常生活中的应用范围，较大的体积使得这些飞行器在运输、使用过程中非常不便。因此，如何克服上述存在的技术问题和缺陷成为重点需要解决的问题。
发明内容
本发明的发明目的在于克服背景技术中所描述的缺陷，从而实现一种具备折叠机翼结构的便携式单人低空矢量喷射飞行器，以解决现有技术中单人飞行器结构体积较大，难以方便的携带和存放，占用空间较大，影响适用性，使用过程中较为不便等问题。
为实现上述发明目的，本发明的技术方案是：
一种具备折叠机翼结构的便携式单人低空矢量喷射飞行器，包括机架部、动力部和负载部，所述动力部设置于所述机架部下方，所述负载部设置于所述动力部下方。
具体的，所述机架部包括机翼中架、伸缩骨架和机翼蒙皮，所述动力部设置于机翼中架下方，所述负载部设置于动力部下方。所述伸缩骨架对称设置于所述机翼中架上部两侧，所述机翼蒙皮固定包裹铺设于所述伸缩骨架外部。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述机翼中架俯视时为T字形，其前端横梁处顶部设置有定位梁，所述T字形机翼中架底部中间位置侧部和底部一体设置有吊架，所述动力部吊装于所述吊架底部。
同时，所述伸缩骨架对称设置于所述机翼中架两侧，所述伸缩骨架为多连杆结构，包括摆杆A、摆杆B、摆杆C、连杆A、连杆B和主动摇杆。
具体的，所述摆杆A前端铰接连接定位梁，后端铰接连接连杆A的中部；所述摆杆A中部右侧铰接连接主动摇杆的前端，主动摇杆的后端可位移、可固定设置于T字形机翼中架的尾部，所述主动摇杆的中部通过摆杆B铰接连接所述连杆A的端部，所述连杆A的另一端铰接连接摆杆C的中部；所述摆杆A中部左侧铰接连接连杆B的端部，所述连杆B的另一端铰接连接摆杆C的前端；所述主动摇杆与摆杆A的铰接点靠近摆杆A前端，连杆B与摆杆A的铰接点靠近摆杆A的后端。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述机翼中架上设置有伸缩驱动单元，所述伸缩驱动单元包括伸缩电机、螺杆、螺纹铰接套和螺杆支架，所述螺杆的前端、尾端和中部转动连接有螺杆支架，所述螺杆支架一体或分体固定设置于所述机翼中架上。
具体的，所述伸缩电机所述螺杆前端的螺杆支架侧部，且驱动连接所述螺杆前端，所述螺纹铰接套螺纹设置于所述螺杆中部与尾端的螺杆支架之间的螺杆上，所述螺纹铰接套两侧均铰接连接所述主动摇杆的后端。
另外，所述伸缩驱动单元上部设置有保护盖，所述保护盖顶部形状为由前至后一体设置的上升、缓降的过渡状曲面，所述保护盖对应所述主动摇杆后端处开设有凹口。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，当所述伸缩骨架完全展开时，所述螺纹铰接套位于所述螺杆中部的螺杆支架处，当所述伸缩骨架完全收拢时，所述螺纹铰接套位于所述螺杆尾端的螺杆支架处。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述摆杆C的前端通过拉绳连接所述定位梁端部，摆杆C的后端通过拉绳连接所述机翼中架的尾部。
优选的，所述摆杆C与所述拉绳为铰接连接，所述拉绳材质为钢丝绳。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述动力部包括油箱、发动机、传动单元和动力输出单元，所述油箱设置于所述机翼中架前端下部的中间位置，所述发动机设置于所述吊架底部，所述吊架位于所述油箱后侧的机翼中架侧部和底部，所述动力输出单元对称设置于所述发动机两侧，所述油箱通过油路连接发动机，所述发动机通过传动单元驱动并固定连接所述动力输出单元。
同时，所述发动机通过控制线路连接有右控制手柄。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述油箱前部设置有收窄部，其最前端做圆滑处理，所述油箱的注油口设置于所述油箱的上端，其注油口处低于所述机翼中架最高处。
优选的，所述油箱顶部注油口处设置有可拆卸式整流罩。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述传动单元设置于所述发动机前端，所述传动单元包括衍架、主动轮、被动轮和传动带，所述衍架固定横向设置于所述发动机前端，所述主动轮、被动轮和传动带均设置于所述衍架内部，所述发动机的动力输出轴贯穿衍架后设置有主动轮，所述被动轮设置于所述动力输出单元的转轴上，所述主动轮和被动轮之间通过所述传动带连接。
优选的，所述衍架上排列开设有多个风洞，所述风洞前后侧边缘处均经过倒角圆滑处理。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述动力输出单元包括涵道筒体、连接架、前螺旋桨叶、后螺旋桨叶和同轴反转齿轮箱，所述衍架贯穿所述涵道筒体后，其端部一体设置有同轴反转齿轮箱，所述衍架贯穿所述涵道筒体处设置有支架，所述同轴反转齿轮箱的前端和后端的动力输出轴上分别设有前螺旋桨叶和后螺旋桨叶，所述被动轮设置于所述同轴反转齿轮箱的动力输出轴上；所述同轴反转齿轮箱位于所述涵道筒体圆心位置，其顶部、底部、侧部均设置有连接架，同轴反转齿轮箱通过其顶部、底部、侧部的连接架固定支撑连接所述涵道筒体的内壁。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述涵道筒体前端还设置有防护罩，其后端设置有锥形罩，所述锥形罩外侧设置有拉绳，所述拉绳的另一端连接所述摆杆C后端和/或机翼蒙皮最外部的尖端处，所述锥形罩尾部收口处为敞口设计，其尾部收口处设置有矢量调节单元。
具体的，所述矢量调节单元包括尾喷转向罩、导流板、侧调节电机和上调节电机，所述尾喷转向罩套接于所述锥形罩的尾部敞口处，所述尾喷转向罩顶部和底部铰接连接所述锥形罩的尾部敞口处，所述尾喷转向罩左侧和右侧开有弧形凹槽，以防止向左右转动时尾喷转向罩与锥形罩敞口处相互干涉。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述导流板设置有多个，多个导流板排列设置于所述尾喷转向罩后端内部，所述导流板中部通过铰轴铰接连接所述尾喷转向罩的侧壁，所述导流板前端设置有联动立板，所述联动立板与多个成排设置的导流板前端铰接连接，所述联动立板上端和下端的尾喷转向罩内壁上均设置有限位滑槽，所述联动立板顶端和底端滑动设置于所述限位滑槽内。
具体的，所述上调节电机下嵌式固定设置于联动立板上方的尾喷转向罩顶部，所述上调节电机的输出轴上设置有螺纹杆，所述螺纹杆螺纹连接所述联动立板顶部。
同时，所述侧调节电机对称设置于所述尾喷转向罩左侧和右侧的弧形凹槽处，所述侧调节电机通过电机支架设置于所述锥形罩的尾部敞口处，所述侧调节电机的输出轴上也设置有螺纹杆，所述尾喷转向罩的弧形凹槽处也设置有电机支架，所述螺纹杆螺纹连接并贯穿所述尾喷转向罩的弧形凹槽处的电机支架。
优选的，所述联动立板与上调节电机之间的螺纹杆外部、侧调节电机的电机支架与尾喷转向罩弧形凹槽处的电机支架之间的螺纹杆外部均套设有复位弹簧，所述螺纹杆端部设置有止位块。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述上调节电机和侧调节电机均为可正反转电机，所述上调节电机和侧调节电机均通过控制线路连接左控制手柄。
在上述的具备折叠机翼结构的便携式单人低空矢量喷射飞行器中，所述负载部包括下载架、背带，所述下载架固定设置于所述发动机下部，所述背带固定设置于所述下载架底部。
优选的，所述背带为五点式安全背带，所述背带为具有安全卡扣的可调节式背带。
与现有技术相比，本发明的具备折叠机翼结构的便携式单人低空矢量喷射飞行器至少具有以下有益效果：
1.本发明的具备折叠机翼结构的便携式单人低空矢量喷射飞行器，通过多连杆结构的伸缩骨架设计，飞行器在不使用时可以轻松收拢，占用的空间较小，方便携带和存放。解决了传统单人飞行器体积大、难以携带的问题，提高了飞行器的适用性和便利性。
2.同时，通过伸缩骨架的多点支撑设计确保了机翼在展开状态下的结构强度，提高了飞行器的稳定性和安全性，机翼蒙皮、防护罩、锥形罩、风洞和整流罩的设计减少了飞行器的空气阻力，提高了空气动力学性能。
使得飞行器在飞行过程中更加高效，延长了续航时间和提高了飞行速度，油箱的流线型设计和风洞的冷却功能优化了动力系统的运行环境，提高了发动机的工作效率。
3.通过五点式安全背带和逃生背包的设计提供了全面的安全保障，确保用户在紧急情况下能够迅速脱离飞行器。电磁铁和紧急脱离手柄的设计使得脱离过程快速可靠，提高了用户的安全性；同时，电磁铁和插接立柱的设计确保了脱离架在正常飞行过程中的稳固连接，减少机械故障的风险。
4.伸缩驱动单元和矢量调节单元提高了飞行器的自动化程度，使得用户可以更方便地调整飞行姿态。通过控制手柄进行操作，确保了用户的舒适性和操作便利性。

为实现对模具部进行支撑，参见图1、图3，所述机架部1包括机翼中架101、伸缩骨架102和机翼蒙皮103，所述动力部2设置于机翼中架101下方，所述负载部3设置于动力部2下方。通过动力部2运行与机架部的机翼中架101、伸缩骨架102及机翼蒙皮103配合，从而完成升空。
