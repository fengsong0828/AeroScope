Description

一种人形低空飞行器
技术领域
本实用新型属于航空低空飞行器应用技术领域，涉及一种人形低空飞行器。
背景技术
随着国家低空领域的开放，随之而来的是航空飞行器如雨后春笋般的势头蓬勃发展，低空飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。随着飞行器普及，个人飞行器必然是未来发展方向。轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向。
但现有的航空低空飞行器其设计的外形缺乏新颖性，使人们可能会产生某种程度的视疲劳，且现有的航空低空飞行器好多很难实现直升直降及零距离起飞零距离降落，且使用场合也比较局限，结构复杂、价格昂贵，操作比较难。
实用新型内容
本实用新型解决的技术问题在于提供了一种人形低空飞行器，改变了原有飞行器的外形设计方式，增加了外形设计的新颖性，且实现了直升直降及零距离起飞零距离降落，应用范围较广，结构简单，价格低廉、操作简单。
本实用新型是通过以下技术方案来实现：
一种人形低空飞行器，包括人形机头和机身，机身包括胸部支持段和两个分开的腿部支持段，机身的两侧设有对称的人形机体；胸部支持段上设有两个对称的主翼，主翼上设置有副翼舵机安装座，副翼与主翼相连接，在腿部支持段的尾部上设有升降舵舵机安装座，升降舵舵机安装座上设有升降舵，人形机体的尾部设有方向舵安装座，方向舵舵机安装座上设有方向舵；机身胸部支持段上还设置有安装电机的电机座；在方向舵舵机安装座、升降舵舵机安装座和副翼舵机安装座上安装有并联电路组成的为低空飞行器提供动力的舵机；人形机头上设置有驱动电机及舵机转动的电池。
副翼通过铰链固定连接于主翼上。
主翼形成了人形低空飞行器的手臂部位，且最外端的形状为手形，展弦比8.28，且主翼前缘后掠38~39度。
同一侧的方向舵与升降舵错位布局。
副翼的面积为主翼面积的1/3。
所述机身的下缘与主翼的下沿位于同一水平面上
人形低空飞行器重心到机头的距离占人形低空飞行器总距离的18%，，人形低空飞行器重心到脚底的距离占人形低空飞行器总距离的82%。
方向舵通过铰链固定连接于方向舵舵机安装座上；升降舵通过铰链固定连接于升降舵舵机安装座上。
人形机头上还连接有脖子；人形机头和脖子上还设置有吸能区。
机身的胸部支持段内置有螺旋桨、接收器和电子调速器，所述螺旋桨背推式安装在机身的胸部支持段内；所述接收器、电子调速器安装于螺旋桨的后方。

本实用新型提供了一种人形低空飞行器，针对现有技术中出现的不足，本实用新型首先从外形上打破了常规飞行器的外形设计风格，将该低空飞行器设计成人体形状的低空飞行器，在设计上具有很强的新颖性。

具体实施方式
下面结合附图对本实用新型做进一步详细描述，所述是对本实用新型的解释而不是限定。
参见图1、图2，一种人形低空飞行器，包括人形机头11和机身10，机身10包括胸部支持段和两个分开的腿部支持段，机身10的两侧设有对称的人形机体8；胸部支持段上设有两个对称的主翼9，主翼9上设置有副翼舵机安装座1，副翼6与主翼9上相连接，在腿部支持段的尾部上设有升降舵舵机安装座2，升降舵舵机安装座2上设有升降舵5，人形机体8的尾部设有方向舵安装座4，方向舵舵机安装座4上设有方向舵3；机身10胸部支持段上还设置有安装电机的电机座7；在方向舵舵机安装座4、升降舵舵机安装座2和副翼舵机安装座1上安装有并联电路组成的为低空飞行器提供动力的舵机；人形机头上设置有驱动电机及舵机转动的电池。
具体的，在方向舵舵机安装座4、升降舵舵机安装座2和副翼舵机安装座1上安装有并联电路组成的为低空飞行器提供动力的舵机，并在机身10胸部支持段上的电机座7上安装电机，且在人形低空飞行器的人形机头11安装有电池，来驱动电极及舵机转动，其中，人形机头11安装的电池可以通过先在人形机头11中设置电池容纳槽来实现，并利用电路将电池与电机及舵机连通，使用时，通过电池来驱动电机及舵机的转动，来实现飞行器的飞行。
其中，副翼6通过铰链固定连接于主翼9上，这样，当人形低空飞行器在空中飞行时，可以根据不同的风向，通过调整副翼6来改变人形低空飞行器方向。
进一步地，主翼9形成了人形低空飞行器的手臂部位，且最外端的形状为手形，展弦比8.28，且主翼9前缘后掠38~39度，其中，主翼9前沿是指主翼前端的边缘线。
优选的，主翼9前缘后掠约39度，最好是39度附近范围的角度。
具体的，主翼9总面积约为150000mm＾2，主翼9最大厚度在整个机翼翼弦靠近前缘的29%处，将该处设置为主梁位置，且该处的厚度为13mm，后缘60%处设樯，该处厚度为5mm，其中，后缘是指主翼9的后边缘线，即就是与副翼6连接处；在主翼9的后边缘线上设置一连接副翼的樯，该樯具体为铰链。
进一步地，所述方向舵3的个数为两个，所述升降舵5的个数为两个，同一侧的方向舵3与升降舵5错位布局，且对称设置。
所述机身10的下缘与主翼9的下沿位于同一水平面上，副翼6的面积为主翼9面积的1/3，人形低空飞行器重心到人形机头11的距离占人形低空飞行器总距离的18%，人形低空飞行器重心到脚底的距离占人形低空飞行器总距离的82%。其中，方向舵3的形状为人脚形状。
进一步地，方向舵3通过铰链固定连接于方向舵舵机安装座4上；升降舵5通过铰链固定连接于升降舵舵机安装座2上。人形机头11上还连接有脖子12；人形机头11和脖子12上还设置有吸能区。机身10的胸部支持段内置有螺旋桨、接收器和电子调速器，所述螺旋桨背推式安装在机身10的胸部支持段内；所述接收器、电子调速器安装于螺旋桨的后方。
其中，方向舵3通过铰链固定连接于方向舵舵机安装座4上，升降舵5通过铰链固定连接于升降舵舵机安装座2上，这样通过改变方向舵3及升降舵5的位置，可以实现飞机的俯仰和偏转，保证飞机能平稳飞行。螺旋桨设置于机身10的胸部支持段的前方，人形低空飞行器脖子的后方；接收器、电子调速器安装于螺旋桨的后方，需要说明的是，螺旋桨、接收器和电子调速器同时安装于机身10的胸部支持段上。
此外，由于本实用新型在降落时受力大，因此在尺寸窄的机身部位设置有吸能区，经实际验证在数次的实际碰撞测试中，该吸能区发挥着有力的作用，使飞机更加的抗摔。
具体的，在设计时人体形状的飞行器，机身的胸部支持段装配的电机采用前拉式、放置于电机座上，以此来加强主梁、其配重的电池选在头部布置、接收器、电子调速器等设备安装在电机座后方位，主翼及升降舵、方向舵上的舵机采用并联电路来驱动，人形低空飞行器的重心到头顶距离占总长18%，机身8的下缘与副翼6的下沿平齐；发动机安装于机身中前部的电极座上，人形低空飞行器的重心到脚底距离占总长82%，采用双垂尾设计及双升降舵设计，且垂尾使用对称平板翼型。
此外，在人形飞行器气动性能方面，采取对主要部位重点设计的方法,所以在胳膊、脚部及脖子部位采用上述独特的设计。在胳膊部位的翼形设计，是飞行器飞行稳定，空中飞行升力及阻力协调的关键，若以人体轴线为x轴，在胳膊后部位的舵面，以差动的方式使得在空中飞行时绕x轴旋转，起到了副翼的作用，在脚部为了防止升降舵，方向舵的干涉，采用错位的布局方法。
其中，使用KT板材料为飞机主要材料，另外使用10*10mm桐木条加强，采用碳纤维材料加强板式机身。
KT板是一种由聚苯乙烯（polystyrene，简称PS）颗粒经过发泡生成板芯，经过表面覆膜压合而成的一种新型材料，板体挺括、轻盈、不易变质、易于加工，并可直接在板上丝网印刷（丝印板）、油漆（需要检测油漆适应性）、裱覆背胶画面及喷绘，广泛用于广告展示促销、建筑装饰、文化艺术及包装等方面。本实用新型采用KT板作为飞机的主要材料，这样可以使制作出来的飞行器轻盈，且耐摔，也不容易走形。
本实用新型的具体设计结构包括以下：
（1）副翼舵机安装座*2，副翼采用差动控制，后掠角为39度，将两个9g舵机安装于副翼舵机安装座上，副翼舵机安装座位于主翼上，利用四杆机构实现主翼的动作，主翼使得人形飞机绕x轴的旋转，使得动作控制更加的稳定。
（2）升降舵舵机安装座*2，用于安装向升降舵提供动力的舵机，可以使升降舵实现对方向的准确控制。
（3）方向舵*2，方向舵实现了人形飞行器绕z轴的旋转。面积定为单边16050mm＾2，实验证明其纵向安定性能良好。
（4）方向舵舵机安装座*2，用于安装向方向舵提供动力的舵机，可以使方向舵实现对方向的准确控制。
（5）升降舵*2，升降舵实现飞行器的绕Y轴的旋转，面积定为单边20309mm＾2，实验证明其有良好的性能。
（6）副翼。
（7）电机座*1。本飞行器推重比为1.2，采用国产某品牌电机配置1047桨动力性能良好。在四级风力情况下测试其飞行动力良好。
（8）机体。
（9）主翼，副翼舵机安装座1安装于主翼9的上方，其中副翼面积为1/3主翼面积。
（10）机身。
（11）人形机头。电池安装于人形机头中，分别向安装于副翼舵机安装座1、升降舵舵机安装座2和方向舵舵机安装座4上的舵机及安装于电极座上的电机提供动力。
（12）脖子。设置有吸能区，可以用来防止撞击变形，吸收能量，脖子部位细的连接部位采用韧性材料如，塑料粘和木头，该材料可以弯曲变形吸收能量，使飞行器更加轻巧耐用。
本实用新型的工作原理为：人形飞行器的机翼，即手臂，产生升力，以支持飞行器在空中飞行，同时也起到一定的稳定和操作作用，在人形飞行器的机翼上安装有副翼，操作副翼可使飞机产生滚转。
水平尾翼由固定的水平的安定面和由铰接连接的升降舵组成，垂直尾翼由其上固定的垂直安定面和由铰接连接的方向舵组成，尾翼的作用是操纵飞机的俯仰和偏转，保证飞机能平稳飞行。
动力装置主要用来产生拉力，使人形飞行器前进。本发明采用锂电池供电，由电子调速器供给无刷电机三相交流电流，从地面发射出调制信号由接收机接收并解调后控制电子调速器，进而控制电机转速。其余各路舵机由同样的方式控制。
需要说明的是，上述描述的机翼为主翼9，其在人形低空飞行器上，处于手臂位置，所述水平的安定面即就是机身10，机身10上的腿部支持段上设置有升降舵舵机安装座2、升降舵5通过铰链连接到水平的安定面上的升降舵舵机安装座2上，以此来形成水平尾翼。此外，所述的垂直安定面即就是机体8，人形机体8的尾部设有方向舵安装座4，方向舵舵机安装座4上通过铰链连接于方向舵3上，以此来形成垂直尾翼。所述的动力装置包括电机及舵机，以及用于向电机及舵机提供动力的电池。
本实用新型除了具有新颖的人体形状的人形飞行器之外，其稳定的飞行性能更具吸引性，飞行器内段主翼在螺旋桨的气流下产生的升力，平衡它自身的重量，具有了直升直降的特点，手抛式起飞，使其0距离起飞，高升力翼型，在降落时可0距离降落，更适合在多种地形飞行。
进一步，在结构方面做了合理的吸能区，具体在其头部、脖子部位；头部下方有一片仿人头形板，撞击变形，吸收能量，且具有更换方便的特点，脖子部位细的连接部位采用韧性材料，例如塑料粘和木头，弯曲变形吸收能量，使飞行器更加轻巧耐用，其螺旋桨内置在人形飞行器的胸部，为背推式安装，使得飞行器更加安全，对自身电路设备也起到了保护。
