Description

一种低空飞行器管理系统
技术领域
本发明涉及低空飞行器管理技术领域，具体涉及一种对无人机等低空飞行器的飞行进行管理和控制的系统。
背景技术
大量的无人机“黑飞”事件存在严重的安全隐患。它的直接危害是严重扰乱空中交通管制，危害空中交通安全。就像地面交通一样，如果没有一系列的交通法规加以防范，没有大量的交警指挥管控，就不会有顺畅安全的行车。“黑飞”事件还可能危及人民群众生命财产安全，扰乱社会治安。民用无人机重量较轻的大概两三公斤，重量较重的有几十公斤，在几十米甚至一百多米的高空中掉下来，可能造成人员伤亡。不法分子可能利用无人机携带摄像机、非法传单或危险爆炸物等严重侵害人民群众的隐私及安全，也有可能危害国家安全。
发明内容
本发明的目的在于提供一种低空飞行器管理系统，可以对所有的低空飞行器进行进行管理控制，避免“黑飞”及各种不安全的事件的发生。
本发明的技术方案如下：
一种低空飞行器管理系统，其特征在于：包括空域管理分系统和飞行器管理分系统，其中空域管理分系统主要对所有的低空区域进行管理控制，使所有低空区域可以安全有效的进行防护及管控；飞行器管理分系统负责对所有登记的受控低空飞行器进行统一监督管理、控制。
所述空域管理分系统包括空域分级管理系统及飞行器处置系统，所述空域分级管理系统将所有的低空区域分为禁止空域和受控空域，禁止空域禁止所有低空飞行器的飞行，在受控空域，低空飞行器可以通过申请、授权等手段在此空域进行飞行并受到相应的管理控制；所述飞行器处置系统对所有的低空飞行器进行监视、管理、控制、拦截处置；
所述飞行器管理分系统包括受控飞行器监管系统和飞行预案管控系统，其中所述受控飞行器监管系统实现对飞行器的身份认证、通信、定位及对飞行器的主动管理控制；所述飞行预案管控系统负责对所有受控飞行器的飞行计划报备、飞行轨迹管理及违规管理。
本发明通过低空飞行器管理系统可以将所有的合法低空飞行器进行统一管理，包含飞行计划报备管理、飞行器身份认证、飞行器实时轨迹管控、飞行空域划分与管理等。能对受控飞行器的身份、飞行预案、飞行轨迹进行监视与控制。对于不受控飞行器，能在受控区域内监视并通过飞行器反制系统阻止其进入禁止空域。实现对低空空域内无人飞行器的有效管理。

具体实施方式
本发明对低空飞行器的管理方案是在所有的低空飞控系统中加装统一的无线应答机，相关的安全及监管部门有一个问答密钥，监管部门甚至还有一个可接管控制的控制密钥，这样不仅可以有效的对无人机进行统一的管理控制，而且可以减少“黑飞”事件的出现。
如图1所示，为实现上述方案，本发明的低空飞行器管理系统主要分为空域管理分系统和飞行器管理分系统。其中空域管理分系统包含空域分级管理和飞行器处置系统；飞行器管理分系统包含受控飞行器管理系统和飞行预案管控系统。
1、空域管理分系统：此分系统主要对所有的低空区域进行管理控制，使所有低空区域可以安全有效的进行防护及管控。分为空域分级管理系统及飞行器处置系统。
(1)空域分级管理系统：
空域分级管理系统将所有的低空区域分为禁止空域(禁止所有低空飞行器的飞行空域)、受控空域(低空飞行器可以通过申请、授权等手段在此空域进行飞行并受到相应的管理控制等)、其他空域。
(2)飞行器处置系统：
飞行器处置系统是对所有的低空飞行器进行监视、管理、控制、拦截处置的系统。
2、飞行器管理分系统：此分系统是对所有登记的受控低空飞行器进行统一监督管理、控制的系统。
(1)受控飞行器监管系统：
受控飞行器监管系统实现对飞行器的身份认证、通信、定位及对飞行器的主动管理控制。
(2)飞行预案管控系统：
飞行预案管控包括对所有受控飞行器的飞行计划报备系统、飞行轨迹管理系统及违规管理系统。
