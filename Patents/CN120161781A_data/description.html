Description

一种面向地效基于自抗扰的旋翼低空飞行器控制方法
技术领域
本发明属于飞行器控制技术领域，特别是涉及一种面向地效基于自抗扰的旋翼低空飞行器控制方法。
背景技术
随着低空经济的发展和社会各行业的经济需求，飞行任务越来越多，例如无人机物流、空中出租车、农药喷洒、应急救援等。面对诸多飞行任务，飞行器的近地悬停能力尤为重要，需要飞行器应对低空复杂气流时的高稳定性、精准控制性以及高安全性。旋翼低空飞行器（旋翼低空飞行器）飞行器在近地飞行时虽有独特的安全性，但同时也会受到近地气流的影响。
旋翼低空飞行器结构和飞行特性独特，它包含多个控制通道，旋翼产生的升力以及舵面控制作用的相互耦合情况使得在建立空气动力学模型时难以精确复刻，存在严重的非线性性和不确定性。传统的PID控制方法在实际飞行中难以实现高精度高稳定控制，传统PID控制系统取误差值时一般直接取成：

发明内容
本发明目的在于解决旋翼低空飞行器控制过程中存在的非线性、不确定性以及耦合干扰等问题，提高飞行器的控制精度和飞行稳定，提出一种面向地效基于自抗扰的旋翼低空飞行器控制方法。
为了实现本发明目的，本发明公开了一种面向地效基于自抗扰的旋翼低空飞行器控制方法，针对带舵面三旋翼飞行器的控制系统，构建了多通道协同控制架构，包含高度保持、偏航稳定、俯仰姿态、滚转姿态四个功能通道；其中高度通道、俯仰通道、滚转通道采用基于扩张状态观测器的自抗扰控制模型，通过非线性状态误差反馈实现扰动抑制；偏航通道采用参数整定优化的PID控制器，利用其周期性扰动抑制特性提升控制精度。
进一步地，针对近地悬停工况下地效干扰显著的问题，建立包含气动耦合效应和地面效应模型的近地悬停控制场景，具体步骤包括：
步骤1、建立近地飞行的飞行器动力学模型，并基于机体模型添加适配的地效模型；
步骤2、根据步骤1所得飞行器动力学模型建立多通道的协同控制框架；
步骤3、基于步骤2所得多通道的协同控制框架构造通道之间耦合干扰观测器；
步骤4、在各通道的干扰观测器的基础上对控制器加入跟踪信号模块和非线性的状态误差反馈模块，最终形成自抗扰控制的控制量输出值；
步骤5、使用离散化的方式整定自抗扰控制参数值；
步骤6、基于协同控制中的偏航通道进行参数整定优化的PID控制设计。
