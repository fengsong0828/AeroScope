Description

低空飞行器用轴向磁通电机混合冷却系统及定子结构
技术领域
本发明涉及轴向磁通电机技术领域，尤其涉及低空飞行器用轴向磁通电机混合冷却系统及定子结构。
背景技术
常规燃油低空飞行器振动噪声大、飞行成本高、环境污染严重，且驱动电机在持续运行时会产生大量的热，如果电机温度过高，会影响电机的可靠性及动力性能，为了维持电机的正常工作，需要设计合理的冷却结构对电机进行冷却。
现阶段低空飞行器最常用的冷却方式为水冷，但是随着电机功率升高，水冷的效果已经满足不了电机的散热要求，而油液不导电，所以冷却油可以直接接触到电机部件，直接带走电机各部件的热量，因此油冷系统的冷却效果远远好于水冷，成为了电机冷却系统的发展趋势。
现有油冷电机一般用油管进行喷油冷却，将冷却油输送到油管中，然后再从油管上面的喷油孔进行喷油，将油喷到电机绕组等部件上进行冷却。但是这种冷却方式需要设计安装额外的油管，所以结构比较复杂、安装工艺较难、可靠性较低；而且使用油管喷油一般只能将冷却油喷到电机各部件的表面，冷却效果不佳，为此我们提出低空飞行器用轴向磁通电机混合冷却系统及定子结构，本发明更多借用空气进行风冷，更加节能环保，且针对风冷会使电机出现两侧温度不均匀的问题，即两侧端盖结构不同，也能够通过端盖的设计进行解决。
发明内容
本发明的目的在于提供低空飞行器用轴向磁通电机混合冷却系统及定子结构，以解决上述背景技术中所提出的问题。
为了解决上述的技术问题，本发明采用了如下技术方案：
低空飞行器用轴向磁通电机混合冷却系统，包括上端盖(1)，以及设置在上端盖(1)一侧的下端盖(7)，所述上端盖(1)与下端盖(7)之间设置有转子总成(6)，所述上端盖(1)与下端盖(7)之间装配有混合冷却机构，所述混合冷却机构包括被动风冷组件和油冷组件，所述上端盖(1)和下端盖(7)之间固定有外环(2)，所述外环(2)的内侧装配有内环(4)，所述外环(2)和内环(4)之间装配有被动风冷组件和油冷组件，所述被动风冷组件和油冷组件均用于轴向磁通电机散热。

优选的，所述螺旋油道(15)内侧的一端靠近下端盖(7)侧面的距离小于螺旋油道(15)内侧的另一端靠近下端盖(7)侧面的距离。

低空飞行器用轴向磁通电机的定子结构，适用于低空飞行器用轴向磁通电机混合冷却系统，包括绕组(3)和定子(5)，所述外环(2)和内环(4)之间均布固定有多个绕组(3)，所述绕组(3)的内侧均设置有定子(5)。

转子：通过在转子背铁上的旋转热管19将永磁体上的热量导出到热管8上端，再通过上端盖1喷油将旋转热管19的热量用冷却油带出，冷却油经由下端盖7回收，以达到降低永磁体温度的目的。
定子：通过插入的两种热管，即热管8和环路热管9，同时热管8和环路热管9起到支架的作用，将绕组3的温度导出到热管8和环路热管9上端，热管8和环路热管9上端设有散热翅片10，并伸出到整个电机壳体外，通过散热翅片10和空气接触带走热量，以达到降温的目的。
