Description

一种用于低空飞行器的定位装置
技术领域
本实用新型涉及GPS定位装置技术领域，具体为一种用于低空飞行器的定位装置。
背景技术
GPS即全球定位系统，是美国研制的卫星导航定位系统，今采用wgs84坐标系统，因地球在天球空间中的位置是不稳定的，故协议用wgs84某一刻的北极点指向位置，如一般的飞行器上多安装有GPS定位器，进而以便于了解飞行器的位置。
市场上的飞行器定位装置在进行使用时由于飞行器的体积不够大，从而使得定位装置在与飞行器进行连接时存在难以匹配的问题，同时也存在定位器的安装调节不够便捷的问题，为此，我们提出一种便于与飞行器安装连接、调节便利的用于低空飞行器的定位装置。
实用新型内容
本实用新型的目的在于提供一种用于低空飞行器的定位装置，以解决上述背景技术中提出的飞行器定位装置在进行使用时由于飞行器的体积不够大，从而使得定位装置在与飞行器进行连接时存在难以匹配的问题，同时也存在定位器的安装调节不够便捷的问题。
为实现上述目的，本实用新型提供如下技术方案：一种用于低空飞行器的定位装置，包括定位体和内室，所述定位体的上下两端均固定有握条，且定位体四角的内部均贯穿设置有气孔，所述定位体的中段衔接有转轴，且转轴的外侧连接有衔接柱，所述衔接柱的外端固定有大强力磁铁，所述内室的底端安置有安装固定座，且内室位于定位体的内部，所述安装固定座的内部设置有螺孔，且安装固定座的顶端连接有硅胶垫，所述大强力磁铁的外侧安置有固定座，且固定座的内部设置有内槽，所述内槽的内端衔接有中心衔接柱，且中心衔接柱的外壁固定有卡条，所述固定座顶端的外侧设置有横穿槽，且横穿槽的下端连接有竖穿槽，所述中心衔接柱的底端固定有小强力磁铁。
优选的，所述定位体与握条为固定连接，且握条关于定位体的中心对称，而且握条为扁平状结构。
优选的，所述衔接柱通过转轴与定位体构成旋转结构，且衔接柱与大强力磁铁为固定连接。
优选的，所述安装固定座包括螺孔和硅胶垫，且安装固定座与硅胶垫为粘接连接。
优选的，所述固定座通过内槽与中心衔接柱构成滑动结构，且中心衔接柱与卡条和小强力磁铁均为固定连接。
优选的，所述横穿槽与竖穿槽为连通结构，且横穿槽设置有两个，而且横穿槽的长度为固定座横截面周长的四分之一。
与现有技术相比，本实用新型的有益效果是：该用于低空飞行器的定位装置设置有定位体与握条，通过为扁平状结构的握条可以来方便手来进行握持，从而方便手提定位体，便于移动定位体，且其设置有衔接柱，通过转轴的旋转，使得定位体与衔接柱之间发生旋转，进而方便来调节定位体的方位，而通过大强力磁铁则可以来将定位体吸附到固定面上，极为便捷，且其设置有安装固定座，通过螺孔可以来将电器元件如电路板螺接到安装固定座上，而经过硅胶垫则可以让安装固定座与电器件之间具有个缓冲性，进而来对电器件起到一个保护的效果，且其设置有固定座，通过内槽与中心衔接柱之间的滑动，使得中心衔接柱得以进行上下调节，而经过小强力磁铁吸附在固定面上，使得固定座与固定面之间得以固定，而经过四个小强力磁铁与固定面之间的吸附固定则可以来对定位体进行限位，防止定位体发生旋转，且其设置有横穿槽与竖穿槽，经过卡条卡在横穿槽的内端，可以对中心衔接柱进行固定，而经过卡条在横穿槽内端的滑动，并滑动至横穿槽与竖穿槽连接处时，卡条可以顺着竖穿槽滑下，进而解除掉中心衔接柱与固定座之间的固定，方便选择使用中心衔接柱底端小强力磁铁，经过内室可以来安装一些GPS定位模块实现定位。

具体实施方式
下面将结合本实用新型实施例中的附图，对本实用新型实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本实用新型一部分实施例，而不是全部的实施例。基于本实用新型中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本实用新型保护的范围。
请参阅图1-4，本实用新型提供一种技术方案：一种用于低空飞行器的定位装置，包括定位体1、握条2、气孔3、转轴4、衔接柱5、大强力磁铁6、内室7、安装固定座8、螺孔9、硅胶垫10、固定座11、内槽12、中心衔接柱13、卡条14、横穿槽15、竖穿槽16和小强力磁铁17，定位体1的上下两端均固定有握条2，且定位体1四角的内部均贯穿设置有气孔3，定位体1与握条2为固定连接，且握条2关于定位体1的中心对称，而且握条2为扁平状结构，通过为扁平状结构的握条2可以来方便手来进行握持，从而方便手提定位体1，便于移动定位体1，定位体1的中段衔接有转轴4，且转轴4的外侧连接有衔接柱5，衔接柱5通过转轴4与定位体1构成旋转结构，且衔接柱5与大强力磁铁6为固定连接，通过转轴4的旋转，使得定位体1与衔接柱5之间发生旋转，进而方便来调节定位体1的方位，而通过大强力磁铁6则可以来将定位体1吸附到固定面上，极为便捷，衔接柱5的外端固定有大强力磁铁6，内室7的底端安置有安装固定座8，且内室7位于定位体1的内部，安装固定座8包括螺孔9和硅胶垫10，且安装固定座8与硅胶垫10为粘接连接，通过螺孔9可以来将电器元件如电路板螺接到安装固定座8上，而经过硅胶垫10则可以让安装固定座8与电器件之间具有个缓冲性，进而来对电器件起到一个保护的效果，安装固定座8的内部设置有螺孔9，且安装固定座8的顶端连接有硅胶垫10，大强力磁铁6的外侧安置有固定座11，且固定座11的内部设置有内槽12，固定座11通过内槽12与中心衔接柱13构成滑动结构，且中心衔接柱13与卡条14和小强力磁铁17均为固定连接，通过内槽12与中心衔接柱13之间的滑动，使得中心衔接柱13得以进行上下调节，而经过小强力磁铁17吸附在固定面上，使得固定座11与固定面之间得以固定，而经过四个小强力磁铁17与固定面之间的吸附固定则可以来对定位体1进行限位，防止定位体1发生旋转，内槽12的内端衔接有中心衔接柱13，且中心衔接柱13的外壁固定有卡条14，固定座11顶端的外侧设置有横穿槽15，且横穿槽15的下端连接有竖穿槽16，横穿槽15与竖穿槽16为连通结构，且横穿槽15设置有两个，而且横穿槽15的长度为固定座11横截面周长的四分之一，经过卡条14卡在横穿槽15的内端，可以对中心衔接柱13进行固定，而经过卡条14在横穿槽15内端的滑动，并滑动至横穿槽15与竖穿槽16连接处时，卡条14可以顺着竖穿槽16滑下，进而解除掉中心衔接柱13与固定座11之间的固定，方便选择使用中心衔接柱13底端小强力磁铁17，中心衔接柱13的底端固定有小强力磁铁17。
工作原理：对于这类的用于低空飞行器的定位装置，首先通过螺孔9可以来将电器元件如电路板螺接到安装固定座8上，而经过硅胶垫10则可以让安装固定座8与电器件之间具有个缓冲性，进而来对电器件起到一个保护的效果，而经过内室7可以来安装一些GPS定位模块实现定位，而通过为扁平状结构的握条2可以来方便手来进行握持，从而方便手提定位体1，便于移动定位体1，而通过转轴4的旋转，使得定位体1与衔接柱5之间发生旋转，进而方便来调节定位体1的方位，而通过大强力磁铁6则可以来将定位体1吸附到固定面上，极为便捷，当通过内槽12与中心衔接柱13之间的滑动，使得中心衔接柱13得以进行上下调节，而经过小强力磁铁17吸附在固定面上，使得固定座11与固定面之间得以固定，而经过四个小强力磁铁17与固定面之间的吸附固定则可以来对定位体1进行限位，防止定位体1发生旋转，同时可经过卡条14卡在横穿槽15的内端，可以对中心衔接柱13进行固定，而经过卡条14在横穿槽15内端的滑动，并滑动至横穿槽15与竖穿槽16连接处时，卡条14可以顺着竖穿槽16滑下，进而解除掉中心衔接柱13与固定座11之间的固定，方便选择使用中心衔接柱13底端小强力磁铁17，就这样完成整个用于低空飞行器的定位装置的使用过程。
