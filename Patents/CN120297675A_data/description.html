Description

一种低空载人飞行器的站点选址方法
技术领域
本发明涉及城市低空交通技术领域，尤其是涉及一种低空载人飞行器的站点选址方法。
背景技术
城市空中交通作为深化通用航空装备示范应用的重要领域，是支撑“全国123出行交通圈”(都市区1小时通勤、城市群2小时通达、全国主要城市3小时覆盖)的关键基础设施。其中，低空载人飞行器(以电动垂直起降机eVTOL为主)的规模化应用，亟需构建高效的城市空中交通网络，而站点选址的合理性直接决定了网络的服务质量与运营效率。
然而，现有技术中针对低空载人飞行器站点选址的研究仍面临多重挑战。首先，选择城市空中交通出行的需求未知。城市空中交通是前所未有的交通方式，所以低空载人飞行器选址必须具有前瞻性，以满足未来可能的用户需求。其次，低空空域管制政策不明确。现行法规如《无人驾驶航空器飞行管理暂行条例》等，没有明确关于低空载人飞行器飞行空域的限制。再者，成本与效益难以平衡。低空载人飞行器建造成本未知，需要根据站点所在地的地价来估计，以便在有限的预算里做出最具性价比的选择。最后，优化算法的应用滞后。优化算法更新迭代迅速，新型高效的算法尚未充分应用于低空载人飞行器站点选址问题中。
经过检索，中国发明专利申请公开号CN118365026A公开了一种载人电动垂直起降飞行汽车选址及选线方法、装置及介质，属于城市低空交通运输领域，方法包括：获取所有需求点的信息，并使用聚类算法确定电动垂直起降平台站点的候选集合；根据需求点的交通信息生成基于交通大数据驱动下的用户出行路线；根据候选集合和用户出行路线，构建数学模型，并以运营商和用户的总成本为优化目标优化模型，最终获得最优的飞行汽车垂直起降平台选址及系统线网路线运营的决策。该现有专利申请存在未考虑候选站点的需求量，从而选址不科学的问题。
如何实现更科学合理的低空载人飞行器的站点选址，成为需要解决的技术问题。
发明内容
本发明的目的就是为了克服上述现有技术存在的缺陷而提供一种低空载人飞行器的站点选址方法。
本发明的目的可以通过以下技术方案来实现：
根据本发明的一个方面，提供了一种低空载人飞行器的站点选址方法，该方法包括以下步骤：
S1：基于潜在需求点和管制因素获得初选低空载人飞行器站点；
S2：通过聚类算法获取候选低空载人飞行器站点；

S4：采用牛顿-拉夫逊优化算法求解站点选址模型，其中站点选址模型的目标函数为：用户需求量最大且成本最小。

与现有技术相比，本发明具有以下有益效果：
1)本发明提出三段式的低空载人飞行器站点选址方法，根据管制因素决定性地筛选出初选低空载人飞行器站点；考虑所有初选低空载人飞行器站点的需求量，根据初选低空载人飞行器站点聚类出候选低空载人飞行器站点；用牛顿-拉夫逊优化算法求解站点选址模型确定低空载人飞行器站点地址，其中模型考虑了低空载人飞行器的分级服务范围、有效载荷、空域管制、用户需求量以及成本，最终综合考虑多因素实现科学合理且快速收敛的站点规划。
2)本发明基于哈达玛积的管制矩阵，精准排除禁飞区域，提升选址合规性。
3)本发明考虑打车用户为潜在的低空载人飞行器用户，将每个初选低空载人飞行器站点附近的打车数据作为其用户需求量，基于三种等级的初选低空载人飞行器服务范围，确定出不同低空载人飞行器站点的用户需求量，有利于提升低空资源利用率，也为需求量预估和后续建模提供有力的数据支撑。
4)本发明对牛顿-拉夫逊优化算法进行了改进，相比粒子群和PSO算法在收敛速度和求解最优解上有更大提升，使得求解速度更快、站点选址更准确。
