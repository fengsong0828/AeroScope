Description

低空载人连衣飞行器
所属技术领域
本发明涉及低空飞行的低空载人连衣飞行器，属低空飞行器类领域。
背景技术
目前，公知所有低空飞行器的结构造型各异，但还没有像衣服裤子一样穿起就能起飞降落的飞行器，本发明是这样创新的。
发明内容
本发明解决其技术问题所采用的技术方案是；将衣服裤子设为一体，衣服是短袖，裤子是短裤脚大腿的一半长，短裤脚是靠粘粘相互粘贴固定以满足人腿的大小舒适为宜，在衣服背上固定一块轻质材料安装多个涵道壁电风机为上升和前飞动力，涵道壁电风机若垂直状可升高降落，若倾斜就向前飞，都由掌中柄控制，在腰间两侧设有蓄电池，开关和调节钮随电线至到手掌固定，可在平地山坡树林楼顶阳台任意起飞降落，不飞时就脱下挂在阳台墙壁上便宜收拾停放，这只适宜在不冷天气飞行代步。
本发明的有益效果是；结构造型简单实用，能圆人的低空飞行梦。

具体实施方案
首先做好轻质材料板(7)轻质材料板与上衣固定孔(4)助推器托脚(3)左右转向机托脚(1)根据上衣短袖(2)可调大小短裤脚(8)的上衣背面在轻质材料板与上衣固定孔(4)上固定，再安装涵道壁电风机助推器(5)安全防撞防护网罩(15)助推器垂直限位柱(11)助推器调节件(12)控制连杆线(6)至掌中控制柄(9)，装上左右转向风机(13)经检查合格后就穿上接通蓄电池电源就可试飞，若向前飞就操作掌中控制柄(9)升降慢进风机(10)控制连杆线(6)助推器调节件(12)就将涵道壁电风机助推器(5)顶上助推器倾斜状向前飞(14)相互连接组成。
