Description

一种低空涡喷飞行器专用轴承
技术领域
本实用新型涉及轴承装置领域，尤其涉及一种低空涡喷飞行器专用轴承。
背景技术
低空涡喷飞行器是一种在低空飞行环境中使用的喷气式飞行器，其专用轴承在飞行器的性能和安全中起着至关重要的作用，轴承支撑着飞行器的旋转部件，如发动机转子、螺旋桨等，确保其在高速旋转时的稳定性和精度。
现有技术虽然通过在旋转部件上使用专用轴承来提高转动时的稳定性，但是轴承在使用时自身容易产生振动的问题，从而导致在使用时容易造成设备发生损坏，因此，本领域技术人员提供了一种低空涡喷飞行器专用轴承，以解决上述背景技术中提出的问题。
实用新型内容
本实用新型的目的是为了解决现有技术中存在的缺点，而提出的一种低空涡喷飞行器专用轴承，提供一种通过在外环的内部设置有两排并列的滚柱，具有更高的旋转精度和更低的振动水平，在外环上设置有储油槽，防止因过热而导致的材料性能下降或轴承故障的低空涡喷飞行器专用轴承。
为实现上述目的，本实用新型提供了如下技术方案：一种低空涡喷飞行器专用轴承，包括外环，所述外环侧壁开设有加注口，所述外环内部设置有储油槽，所述外环内侧壁两侧分别开设有第二凹槽，两个所述第二凹槽内侧壁分别呈圆周阵列有若干润滑口，两个所述第二凹槽内侧壁分别滑动连接有若干滚柱，所述加注口内侧壁滑动连接有密封塞，所述外环内侧壁设置有内环，所述内环侧壁两侧分别开设有第一凹槽；
通过上述技术方案，通过在外环的内部设置有两排并列的滚柱，可以承受更大的径向和轴向载荷，由于有两排滚柱，载荷可以分布在更多的接触点上，从而提高了轴承的承载能力，由于两排滚柱可以更好地平衡载荷，双列轴承通常具有更高的旋转精度和更低的振动水平，这对于精密机械和高速运转的设备尤为重要。
进一步地，若干所述润滑口分别贯穿第二凹槽内侧壁与储油槽内部连通；
通过上述技术方案，有效的瑞滚柱进行润滑。
进一步地，所述加注口贯穿外环与储油槽内部连通，所述密封塞与加注口内侧壁相互适配；
通过上述技术方案，有效的对储油槽进行密封。
进一步地，若干所述滚柱与两个第一凹槽侧壁滑动连接；
通过上述技术方案，有效的提高滚柱转动时的稳定性。
进一步地，所述外环与内环同轴心设置；
通过上述技术方案，提高装置整体的稳定性。
进一步地，两个所述第二凹槽与两个第一凹槽分别倾斜设置；
通过上述技术方案，具有更高的旋转精度和更低的振动水平。
进一步地，所述外环和内环的材质均为M50钢材；
通过上述技术方案，提高装置整体的结构强度。
本实用新型具有如下有益效果：
1、本实用新型中，通过在外环的内部设置有两排并列的滚柱，可以承受更大的径向和轴向载荷，由于有两排滚柱，载荷可以分布在更多的接触点上，从而提高了轴承的承载能力，由于两排滚柱可以更好地平衡载荷，双列轴承通常具有更高的旋转精度和更低的振动水平，这对于精密机械和高速运转的设备尤为重要。
2、本实用新型中，在外环上设置有储油槽，利用储油槽可以储存一定量的润滑油，确保在轴承运转过程中，润滑油能够持续、均匀地分布在滚柱和外环之间，减少摩擦和磨损，并且润滑油在循环过程中可以带走轴承运转时产生的热量，帮助轴承保持适宜的工作温度，防止因过热而导致的材料性能下降或轴承故障。
