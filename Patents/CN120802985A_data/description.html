Description

低空飞行器的智能避障系统及方法
技术领域
本发明涉及低空飞行器避障领域，尤其涉及低空飞行器的智能避障系统及方法。
背景技术
随着无人机、低空载人飞行器等低空飞行器在物流运输、环境监测、应急救援等领域的广泛应用，其在复杂低空环境下的安全避障需求愈发迫切。低空环境具有障碍物类型多样、分布复杂且动态变化的特点，例如城市中的建筑、电线，山区的树木、岩石，以及运动中的鸟类、其他飞行器等，这些都对低空飞行器的避障能力提出了极高要求。
然而，现有的低空飞行器避障技术存在诸多不足。一方面，在障碍物识别能力上存在局限。传统避障系统多依赖单一传感器或简单的多传感器融合，难以全面准确地感知环境信息。例如，仅依靠激光雷达虽能快速获取距离信息，但在复杂光照、恶劣天气条件下，对小型、透明或纹理相似障碍物的识别能力较弱；而单纯的视觉传感器易受光照变化、遮挡等因素影响，导致障碍物漏检或误检，无法为飞行器提供可靠的环境感知数据。
另一方面，在避障决策与路径规划方面，现有技术难以适应复杂多变的低空场景。传统的路径规划算法，如A*算法、Dijkstra算法等，计算复杂度高，实时性差，难以满足飞行器快速决策的需求。同时，这些算法在处理动态障碍物和狭窄复杂空间时，往往生成的路径不够优化，可能出现路径过长、转折过多，甚至陷入局部最优解的情况，使得飞行器在避障过程中存在碰撞风险，无法高效、安全地完成避障任务。
发明内容
为了克服现有技术存在的缺点与不足，本发明提供低空飞行器的智能避障系统及方法。
本发明所采用的技术方案是，低空飞行器的智能避障系统，包括：
基于时空域特征增强的双目视觉感知单元，用于以交错式像素采样模式同步采集低空环境的双路图像信息，并通过光流估计建立图像帧间像素运动关联，形成多通道图像张量数据；
自适应权重分配的特征增强注意力单元，用于通过多层级注意力计算模块，依据图像区域的纹理复杂度、边缘梯度强度以及目标轮廓特征，对多通道图像张量数据各通道及空间位置的特征进行动态加权，生成增强后的特征张量；
多尺度特征融合的多维解析处理单元，用于对增强后的特征张量进行多尺度卷积运算，提取不同分辨率下的语义和几何特征，通过跨尺度特征拼接与融合操作，构建包含全局语义和局部细节的三维特征空间；
基于动态阈值判定的障碍物识别单元，用于在三维特征空间内利用预设的障碍物特征模板库，采用滑动窗口搜索策略，通过计算特征相似度和空间距离阈值，识别出环境中的障碍物，并获取位置坐标、尺寸信息和置信度得分；
避障路径规划决策单元，用于根据障碍物识别单元输出的信息，结合飞行器当前位置、速度矢量以及预设安全距离阈值，采用启发式搜索算法在飞行允许区域生成多条候选避障路径；
运动控制指令生成单元，用于将避障路径规划决策单元筛选出的最优避障路径转换为飞行器的姿态调整参数和速度控制指令，通过姿态解算算法和动力学模型计算电机转速调节量和舵机偏转角度。

智能避障系统的避障方法，包括以下步骤：
步骤S1、利用基于时空域特征增强的双目视觉感知单元以交错式像素采样模式同步采集低空环境的双路图像信息，并通过光流估计建立图像帧间像素运动关联，形成多通道图像张量数据；
步骤S2、将多通道图像张量数据输入自适应权重分配的特征增强注意力单元，通过多层级注意力计算模块，依据图像区域的纹理复杂度、边缘梯度强度以及目标轮廓特征，对各通道及空间位置的特征进行动态加权，生成增强后的特征张量；
步骤S3、把增强后的特征张量传输至多尺度特征融合的多维解析处理单元，进行多尺度卷积运算，提取不同分辨率下的语义和几何特征，通过跨尺度特征拼接与融合操作，构建包含全局语义和局部细节的三维特征空间；
步骤S4、将三维特征空间数据送入基于动态阈值判定的障碍物识别单元，在该空间内利用预设的障碍物特征模板库，采用滑动窗口搜索策略，通过计算特征相似度和空间距离阈值，识别出环境中的障碍物，并获取其位置坐标、尺寸信息和置信度得分；
步骤S5、避障路径规划决策单元根据障碍物识别单元输出的信息，结合飞行器当前位置、速度矢量以及预设安全距离阈值，采用启发式搜索算法在飞行允许区域生成多条候选避障路径，通过路径代价评估函数筛选出最优避障路径；
步骤S6、运动控制指令生成单元将最优避障路径转换为飞行器的姿态调整参数和速度控制指令，通过姿态解算算法和动力学模型计算电机转速调节量和舵机偏转角度，驱动飞行器执行避障动作。
有益效果：本发明提出低空飞行器的智能避障系统及方法，在障碍物识别层面，基于时空域特征增强的双目视觉感知单元采用交错式像素采样同步采集双路图像，并利用光流估计建立帧间运动关联，结合自适应权重分配的特征增强注意力单元，依据图像纹理、边缘及轮廓特征对各通道特征动态加权，增强关键区域特征表达，弥补了单一传感器在复杂光照、恶劣天气下对特殊障碍物识别不足的问题；多尺度特征融合的多维解析处理单元进一步提取不同分辨率语义和几何特征，构建三维特征空间，结合动态阈值判定机制，显著提升障碍物识别的准确性与完整性。在避障决策与路径规划方面，避障路径规划决策单元依据障碍物识别结果，结合飞行器实时状态与预设安全阈值，通过启发式搜索生成候选路径，并运用综合考虑避障安全、路径长度和方向变化的代价评估策略，筛选出最优路径；同时，分层搜索策略依据飞行器与障碍物距离动态调整搜索范围，提高路径规划效率。运动控制指令生成单元将最优路径转化为精准的姿态和速度控制指令，驱动飞行器安全避障。整套系统实现了对复杂低空环境中障碍物的精准识别与高效避障，极大提升了低空飞行器在复杂场景下运行的安全性与可靠性。

具体实施方式
需要说明的是，在不冲突的情况下，本申请中的实施例及实施例中的特征可以相互结合，下面结合附图和具体实施例对本申请作进一步详细说明。
如图1所示，低空飞行器的智能避障系统，包括：
基于时空域特征增强的双目视觉感知单元，用于以交错式像素采样模式同步采集低空环境的双路图像信息，并通过光流估计建立图像帧间像素运动关联，形成多通道图像张量数据；
具体的，该单元作为系统环境信息采集的基础模块，采用两台高分辨率工业级CMOS相机进行双路图像采集。相机具备全局快门技术，帧率可达60帧/秒，分辨率为2560×1920像素，像元尺寸3.2μm，能够清晰捕捉50米范围内的环境细节，对直径不小于15厘米的障碍物可有效识别。两台相机光轴平行度控制在0.02°以内，基线距离固定为350毫米，确保视差计算的精准性。通过交错式像素采样模式，配合硬件同步电路实现亚毫秒级同步采集，获取包含RGB色彩信息的双路图像数据。同时，内置的光流估计算法基于多层金字塔结构，可计算相邻帧间像素点的运动矢量，其运动检测精度达0.05像素/帧，能识别速度低至0.3米/秒的动态目标，最终生成包含色彩、深度及运动信息的多通道图像张量数据，为后续处理提供多维原始信息。
该单元的实施过程严格遵循参数校准与数据同步流程。在安装阶段，需对相机的内外参数进行标定，通过张氏标定法获取精确的相机内参矩阵与畸变系数，外参标定则确保双相机的旋转和平移矩阵误差在可接受范围。工作时，图像采集模块以固定频率同步触发两台相机，采集的数据经千兆以太网传输至处理单元，在数据预处理环节，通过去除噪声、校正白平衡等操作，保障数据质量。光流估计模块对连续帧图像进行分层处理，从低分辨率图像开始计算初步运动矢量，再逐级细化至原始分辨率，有效提升计算效率与准确性，最终将处理后的多通道图像张量数据输出至后续单元进行特征增强与分析。
自适应权重分配的特征增强注意力单元，与所述基于时空域特征增强的双目视觉感知单元连接，用于通过多层级注意力计算模块，依据图像区域的纹理复杂度、边缘梯度强度以及目标轮廓特征，对多通道图像张量数据各通道及空间位置的特征进行动态加权，生成增强后的特征张量；
具体的，该单元专注于对多通道图像张量数据进行特征强化，其核心架构包含空间注意力与通道注意力两个子模块。空间注意力模块基于图像区域的纹理复杂度与边缘梯度强度分配权重，通过局部二值模式(LBP)算法计算纹理特征，采用8邻域模式量化局部纹理信息，同时利用Sobel算子检测边缘，梯度强度计算采用3×3模板，能够精准识别图像中纹理丰富、边缘显著的区域，并赋予更高权重，突出障碍物轮廓等关键信息。通道注意力模块则依据各特征通道间的相关性评估重要性，通过皮尔逊相关系数衡量通道间语义关联，对与障碍物识别紧密相关的通道增强权重，抑制无关信息，实现对数据特征的针对性强化。
单元的实施采用分层迭代优化策略。在初始化阶段，通过预训练模型为各注意力模块赋予初始权重参数，其中空间注意力的权重矩阵维度与图像特征图尺寸匹配，通道注意力的权重向量长度对应特征通道数。运行过程中，系统根据实时输入的图像张量数据，采用随机梯度下降算法进行权重更新，每次更新步长设为0.01，学习率为0.001，通过反向传播算法最小化特征损失函数，持续优化权重分配。经过该单元处理后，原始图像张量数据中与障碍物相关的特征得到显著增强，无关背景信息被有效抑制，生成的增强特征张量为后续障碍物识别提供了更具区分度的输入数据。
多尺度特征融合的多维解析处理单元，与所述自适应权重分配的特征增强注意力单元连接，用于对增强后的特征张量进行多尺度卷积运算，提取不同分辨率下的语义和几何特征，通过跨尺度特征拼接与融合操作，构建包含全局语义和局部细节的三维特征空间；
具体的，该单元基于多分支卷积神经网络架构，实现对增强特征张量的多尺度特征提取与融合。单元包含三个并行卷积分支，分别负责不同尺度特征的处理。第一分支采用3×3卷积核，步长为1，填充为1，专注提取图像局部细节特征，能够捕捉障碍物表面纹理、细小结构等信息；第二分支使用5×5卷积核，步长为2，填充为2，用于提取中等尺度语义特征，可识别障碍物的部分结构形态；第三分支配备7×7卷积核，步长为4，填充为3，主要提取全局语义特征，用于判断障碍物的整体类别与轮廓。各分支输出的特征图经批归一化层稳定数据分布，并通过ReLU激活函数引入非线性，增强网络表达能力。最后，采用通道拼接与加权求和的方式，将不同尺度特征融合，构建包含全局语义与局部细节的三维特征空间。
在实施过程中，各卷积分支的参数配置经过严格优化。卷积核数量分别设定为64、128、256，以平衡计算量与特征提取能力；批归一化层的动量参数设为0.9，确保数据归一化的稳定性；加权求和的融合权重通过训练学习获得，初始值采用均匀分布随机初始化。为避免过拟合现象，网络训练时采用Dropout策略，随机丢弃20％的神经元连接。最终生成的三维特征空间数据，既保留了障碍物的局部细节特征，又整合了全局语义信息，为后续障碍物识别提供了丰富的特征维度，提升了系统对不同类型、不同尺度障碍物的识别能力。
基于动态阈值判定的障碍物识别单元，与所述多尺度特征融合的多维解析处理单元连接，用于在三维特征空间内利用预设的障碍物特征模板库，采用滑动窗口搜索策略，通过计算特征相似度和空间距离阈值，识别出环境中的障碍物，并获取其位置坐标、尺寸信息和置信度得分；
具体的，该单元依托预设的障碍物特征模板库，在三维特征空间内实现对环境中障碍物的精准识别。模板库包含不同类型障碍物的特征描述，如球形、柱状、平面结构等，每个模板由多组特征向量构成，涵盖形状、纹理、空间位置等信息。识别过程采用滑动窗口搜索策略，窗口尺寸根据三维特征空间分辨率设定，最小窗口覆盖区域对应现实场景中0.2×0.2×0.2立方米的空间范围。单元通过计算待识别区域特征与模板库中特征的相似度，并结合空间距离阈值进行判定。相似度计算采用余弦距离与曼哈顿距离相结合的方式，综合评估特征向量的方向与数值差异；空间距离阈值根据飞行器当前速度动态调整，速度每增加1米/秒，阈值增加0.5米，确保对近距离快速移动障碍物的及时响应。
单元实施时，首先对三维特征空间数据进行网格化划分，网格精度为0.1米×0.1米×0.1米，以提高搜索效率。滑动窗口在各网格内依次遍历，每次移动步长为窗口尺寸的50％，确保无遗漏检测。当待识别区域与模板的相似度超过动态设定的阈值，且其空间距离满足安全要求时，判定为障碍物，并记录其位置坐标(精度达0.1米)、尺寸信息(误差不超过5％)及置信度得分(取值范围0-1)。为适应环境变化，系统每5秒对模板库进行一次更新，通过聚类算法合并相似模板，剔除冗余信息，保证模板库的时效性与准确性，从而实现对复杂环境中各类障碍物的可靠识别。

具体的，该单元以障碍物识别结果为基础，结合飞行器实时状态参数，生成安全高效的避障路径。单元接收飞行器当前位置坐标(精度0.1米)、速度矢量(精度0.01米/秒)、航向角(精度0.1°)等信息，同时参考预设的安全距离阈值(默认值为3米，可根据环境复杂度调整)。采用启发式搜索算法在可行区域内生成多条候选避障路径，搜索过程基于八叉树空间划分，树节点分辨率为0.5米×0.5米×0.5米，确保路径规划的精细度。候选路径评估综合考虑避障安全性、路径长度、方向变化等因素，安全性通过计算路径与障碍物的最小距离衡量，路径长度采用欧几里得距离计算，方向变化通过相邻路径段的夹角累加评估。通过预设的代价评估策略，对每条候选路径进行打分，得分最低者被确定为最优避障路径。
在实施过程中，单元采用分层搜索策略提升计算效率。初始阶段，以较大搜索步长(1米)进行粗粒度搜索，快速筛选出大致可行方向；随后在关键区域采用较小步长(0.2米)进行精细搜索，优化路径细节。搜索范围根据飞行器与最近障碍物的距离动态调整，距离每减少1米，搜索范围扩大20％，确保对突发障碍物的及时响应。为避免陷入局部最优解，算法引入随机扰动机制，每次迭代以5％的概率对当前最优路径进行局部调整。最终生成的最优避障路径包含一系列离散路径点，相邻点间距不超过0.5米，为飞行器的运动控制提供精确指引。

具体的，该单元负责将最优避障路径转换为飞行器可执行的控制指令，实现精准避障动作。单元接收最优避障路径的离散路径点坐标信息，结合飞行器当前姿态参数(俯仰角、滚转角、偏航角，精度0.1°)和动力学模型，通过姿态解算算法计算飞行器姿态调整参数。姿态调整采用比例-积分-微分(PID)控制策略，比例系数、积分系数、微分系数分别设为0.8、0.2、0.1，确保姿态调整的稳定性与快速性。速度控制指令根据路径点间的距离与预设的最大速度(默认值为10米/秒)进行规划，采用梯形加减速算法控制速度变化，加速度与减速度均设为2米/秒2，避免速度突变对飞行器造成冲击。
在实施过程中，单元将姿态调整参数和速度控制指令转换为电机转速调节量和舵机偏转角度。对于四旋翼飞行器，通过改变四个电机的转速差实现姿态调整，转速调节精度为1转/秒；舵机偏转角度控制精度为0.5°，确保飞行器能够按照规划路径平稳飞行。指令输出采用脉宽调制(PWM)信号，信号频率为50Hz，占空比调节范围为5％-15％，以适配飞行器的驱动系统。同时，单元内置安全保护机制，当检测到控制指令超出飞行器性能极限时，自动触发紧急制动程序，保障飞行器运行安全。

如图2所示，智能避障系统的避障方法，包括以下步骤：
步骤S1、利用基于时空域特征增强的双目视觉感知单元以交错式像素采样模式同步采集低空环境的双路图像信息，并通过光流估计建立图像帧间像素运动关联，形成多通道图像张量数据；
步骤S2、将多通道图像张量数据输入自适应权重分配的特征增强注意力单元，通过多层级注意力计算模块，依据图像区域的纹理复杂度、边缘梯度强度以及目标轮廓特征，对各通道及空间位置的特征进行动态加权，生成增强后的特征张量；
步骤S3、把增强后的特征张量传输至多尺度特征融合的多维解析处理单元，进行多尺度卷积运算，提取不同分辨率下的语义和几何特征，通过跨尺度特征拼接与融合操作，构建包含全局语义和局部细节的三维特征空间；
步骤S4、将三维特征空间数据送入基于动态阈值判定的障碍物识别单元，在该空间内利用预设的障碍物特征模板库，采用滑动窗口搜索策略，通过计算特征相似度和空间距离阈值，识别出环境中的障碍物，并获取其位置坐标、尺寸信息和置信度得分；
步骤S5、避障路径规划决策单元根据障碍物识别单元输出的信息，结合飞行器当前位置、速度矢量以及预设安全距离阈值，采用启发式搜索算法在飞行允许区域生成多条候选避障路径，通过路径代价评估函数筛选出最优避障路径；
步骤S6、运动控制指令生成单元将最优避障路径转换为飞行器的姿态调整参数和速度控制指令，通过姿态解算算法和动力学模型计算电机转速调节量和舵机偏转角度，驱动飞行器执行避障动作。
低空飞行器智能避障系统及方法深度融合改进的注意力机制与双目视觉传感多维解析模型，针对传统系统在障碍物识别领域的不足，本方案构建了多层级协同感知体系。基于时空域特征增强的双目视觉感知单元采用交错式像素采样与光流估计技术，同步采集双路图像并建立帧间运动关联，相较于单一传感器，能更全面捕捉环境动态变化与细节信息。自适应权重分配的特征增强注意力单元，依据图像纹理复杂度、边缘梯度强度及目标轮廓特征，对多通道图像张量数据进行动态加权，强化关键区域特征表达；结合多尺度特征融合的多维解析处理单元，通过提取不同分辨率语义和几何特征，构建三维特征空间，配合动态阈值判定机制，即使面对小型、透明障碍物，或处于复杂光照、恶劣天气环境，也能精准识别，有效避免漏检误检问题。
在避障决策与路径规划方面，传统算法存在的实时性差、路径优化不足等缺陷被有效解决。避障路径规划决策单元基于障碍物识别结果，结合飞行器实时状态与预设安全阈值，利用启发式搜索快速生成候选路径，并通过综合考量避障安全性、路径长度及方向变化的代价评估策略，筛选出最优路径。同时，分层搜索策略依据飞行器与障碍物距离动态调整搜索范围，显著提升计算效率。运动控制指令生成单元将最优路径精准转化为姿态调整参数与速度控制指令，通过姿态解算算法和动力学模型计算电机转速调节量和舵机偏转角度，确保飞行器在复杂狭窄空间中灵活、高效地完成避障动作，避免陷入局部最优解，极大增强了飞行器在动态复杂环境下的适应性与安全性。
