Description

一种便携式低空遥感微型无人飞行器
技术领域
本实用新型涉及无人驾驶飞机领域，特别涉及一种便携式低空遥感微型无人飞行器。
背景技术
微型无人飞行器具有独特的飞行能力和使用价值，其无人员伤亡、体积小、造价低、战场生存能力强。在军用方面，微型无人飞行器既能执行各自非杀伤性任务，又能执行各种软硬杀伤性任务，包括侦查、监视、目标截获、诱饵、攻击、通信中继等；在民用方面，微型无人飞行器在航拍、大气监测、交通监控、资源勘探、电力巡检、森林防火、农业等方面具有更广泛的前景。微型无人飞行器在外出作业时，一般需要将其存放在携带箱内。为了实现外出作业的便携性，这势必也就要求整个无人机的携带箱必须体积小、重量轻等特点。微型无人飞行器的载体重要组成部分就是机身，目前主流的微型无人飞行器机身多由复合材料采用一次模压成型。由于该结构的微型无人飞行器的机身为一整体、无法分开存放，因而机身的总长便决定了其携带箱的体积大小、便携性能差。在同样的机长情况下，若能让微型无人飞行器的机身分成几段放入携带箱的话，那么整个携带箱的尺寸就能得到大幅的缩减。而微型无人飞行器的机身被分成几段，又如何把几段机身连接起来，同时又能保证机身的强度呢？这是急需解决的问题。
实用新型内容
本实用新型所要解决的技术问题是提供一种便携式低空遥感微型无人飞行器，机身既能够方便地进行拆分携带，也能够保证组合后机身的使用强度。
为解决上述问题，本实用新型所设计的一种便携式低空遥感微型无人飞行器，包括机身，其不同之处是所述机身由2块或2块以上的机身组件组合而成，即2块相连的机身组件上分别设有相互配合的母型接头和公型接头；其中母型接头为内凹的多边形的凹腔，且该凹腔的截面面积由外向内逐渐减小；公型接头则为外凸的多边形的凸柱，且该凸柱的截面面积由外向内逐渐增大；组合时，公型接头嵌入母型接头内，其凹腔和凸柱的凸凹截面相对接。
上述方案中，所述母型接头的凹腔底部和公型接头的凸柱前端分别嵌有永磁体，且位于两相连母型接头和公型接头上的永磁体的位置相对、并能够相互吸合。
上述方案中，所述永磁体嵌于薄木片的中部，薄木片携带永磁体粘接在母型接头的凹腔底部中央和公型接头的凸柱前端中央。
上述方案中，所述母型接头的凹腔的深度长于公型接头的凸柱的高度。
上述方案中，位于机身的相对前方的机身组件上设有的是母型接头，而位于机身的相对后方的机身组件上设有的是公型接头。
本实用新型与现有技术相比，其机身采用2块或2块以上的机身组件通过相互嵌合的母型接头和公型接头对接而成，这种连接方式构思巧妙、结构简单、制作简单、组装快速，使得微型无人飞行器更便于携带，因而更适于野外作业的需要；此外机身组件在模具设计时就按照多边形嵌入型公母型接头设计，对接后其本身就具备一定的连接强度，而本实用新型还在母型接头和公型接头内各设一永磁体，对接后永磁体的两极相互作用后会产生强大的吸引力，为机身的连接强度提供了更为可靠的保证。

具体实施方式
图1为本实用新型一种便携式低空遥感微型无人飞行器，包括机身。所述机身由2块或2块以上的机身组件1组合而成，即2块相连的机身组件1上分别设有相互配合的母型接头2和公型接头3；其中母型接头2为内凹的多边形的凹腔2-1，且该凹腔2-1的截面面积由外向内逐渐减小；公型接头3则为外凸的多边形的凸柱3-1，且该凸柱3-1的截面面积由外向内逐渐增大；组合时，公型接头3嵌入母型接头2内，其凹腔2-1和凸柱3-1的凸凹截面相对接。参见图2。
为了进一步提高两机身组件1间连接的可靠性，我们还可以在母型接头2的凹腔2-1内和/或公型接头3的凸柱3-1上增加磁性材料。如可以在母型接头2的凹腔2-1内设置一永磁体4，而在公型接头3的凸柱3-1上设置一铁片；或在公型接头3的凸柱3-1上设置一永磁体4，而在母型接头2的凹腔2-1内设置一铁片；也可以同时在母型接头2的凹腔2-1内和公型接头3的凸柱3-1上各设置一永磁体4；通过磁力来增加两相连机身组件1间连接强度。在本实用新型优选实施例中，母型接头2的凹腔2-1底部和公型接头3的凸柱3-1前端分别嵌有薄型的永磁体4，且位于两相连母型接头2和公型接头3上的永磁体4的位置相对、并能够相互吸合。为了确保两相连机身组件1对接所产生的吸引力最大，同时也为了避免因永磁体4的加入而导致机身重量的增加，所述永磁体4最好嵌于薄木片5的中部，薄木片5携带永磁体4粘接在母型接头2的凹腔2-1底部中央和公型接头3的凸柱3-1前端中央。生产过程中，可以将薄片状的永磁体4胶粘于薄木片5的上表面的中部，然后将薄木片5的下表面通过胶水粘接在母型接头2的凹腔2-1的底部中央和公型接头3的凸柱3-1的前端中央。此外，粘接过程中还需注意两对接永磁体4间的磁力方向，确保两者之间能够相互吸合在一起。参见图3和图4。
上述母型接头2的凹腔2-1的深度应等于或长于公型接头3的凸柱3-1的高度，以便让两相连机身组件1之间的嵌合更加紧密。而在本实用新型优选实施例中，母型接头2的凹腔2-1的深度应长于公型接头3的凸柱3-1的高度，其差值可以介于3-4mm之间，以供永磁体4等连接加强配件的置入。为了避免微型无人飞行器在飞行过程中因风力使两相连机身组件1之间的分离，提高连接的可靠性，位于机身的相对前方的机身组件1上设有的是母型接头2，而位于机身的相对后方的机身组件1上设有的是公型接头3，这样流经两机身组件1连接处的气流不仅不会灌入母型接头2的凹腔2-1内、使公型接头3的凸柱3-1从公型接头3的凹腔2-1中脱离，相反还能够让母型接头2的凹腔2-1在气流的推动下，向公型接头3的凸柱3-1方向运动、使母型接头2和公型接头3对接地更加紧密。
