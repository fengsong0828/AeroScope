Description

多低空飞行器自动编队控制系统
技术领域
本发明涉及程序控制系统技术领域，具体涉及多低空飞行器自动编队控制系统。
背景技术
多低空飞行器在飞行的时候，一般都会呈现列队的形式进行飞行，这是由于无人飞行器始终与地面的地面操控端保持着实时的通信，由地面的操控终端进行列队的指挥和调度。
但是对于一些信号不强的地方，在低空飞行器在飞行的时候，不能持续地接收到地面操控端发生的指令，导致无法进行飞行的调整，该情况下，一般都会使得无人飞行器原地等候待命，从而使得在飞行的时候无人飞行器处于停滞不前的状态，此外在飞行器进行编队飞行时，一般会设置长机，当长机在飞行中出现故障时，也会影响其他飞行器继续运行，进而会影响飞行器执行飞行任务。
为此提出多低空飞行器自动编队控制系统。
发明内容
本发明的目的在于：为解决上述背景技术中提到的问题，本发明提供了多低空飞行器自动编队控制系统。
本发明为了实现上述目的具体采用以下技术方案：
多低空飞行器自动编队控制系统，包括地面控制中心和控制编队，所述地面控制中心包括地面控制主机，所述地面控制主机连接有显示屏，所述地面控制主机内置有对空通信模块，所述空中编队包括数量不定的无人机，所述无人机表面设置有地面通信模块、环境探测模块、队内通信模块和飞行控制模块，且地面通信模块和对空通信模块远程无线连接。
进一步地，所述无人机包括长机和僚机，所述长机的实时数量为一个，且剩余无人机均为僚机。
进一步地，所述环境探测模块包括激光雷达、北斗导航模块、全景摄像头和距离传感器。
进一步地，所述队内通信模块包括发送机和接收机，所述对内通信模块还连接有随机选择器。
进一步地，所述飞行控制模块包括横侧向控制器、纵向控制器和速度控制器。
进一步地，所述无人机内置有运动计算模块。
本发明的有益效果如下：
地面控制主机通过对空通信模块与地面通信模块进行通信，并对空中飞行器编队进行控制，无人机通过环境探测模块对环境进行感知并定位，并可将飞行信息传送回显示屏进行显示，当地面控制中心与空中编队失去联络时，通过飞行控制模块继续接管运行，继续执行飞行任务，当长机出现故障损毁后，通过对内通信模块进行自动继承，使用其他僚机成为新长机，继续保持编队飞行，在使用中实现了便于对多低空飞行器进行自动编队飞行的效果，在地面与飞行器失去联络的情况下，飞行编队仍能继续执行并完成飞行任务，并且能够在编队中自动继承更新长机，在初代长机损坏的情况下，仍然能够执行任务，从而保障了飞行编队的稳定性。

如图1至图6所示，多低空飞行器自动编队控制系统，包括地面控制中心和控制编队，所述地面控制中心包括地面控制主机，所述地面控制主机连接有显示屏，所述地面控制主机内置有对空通信模块，所述空中编队包括数量不定的无人机，所述无人机表面设置有地面通信模块、环境探测模块、队内通信模块和飞行控制模块，且地面通信模块和对空通信模块远程无线连接，更具体的为，地面控制主机通过对空通信模块与地面通信模块进行通信，并对空中飞行器编队进行控制，无人机通过环境探测模块对环境进行感知并定位，并可将飞行信息传送回显示屏进行显示，当地面控制中心与空中编队失去联络时，通过飞行控制模块继续接管运行，继续执行飞行任务，当长机出现故障损毁后，通过对内通信模块进行自动继承，使用其他僚机成为新长机，继续保持编队飞行。
如图1所示，无人机包括长机和僚机，长机的实时数量为一个，且剩余无人机均为僚机，需要说明的是，长机通过地面通信模块与地面控制中心的对空通信模块进行通信，长机与僚机之间、僚机与僚机均通过队内通信模块进行通信联络。
如图2所示，环境探测模块包括激光雷达、北斗导航模块、全景摄像头和距离传感器，更具体的为，通过激光雷达持续发送探测信号，对飞行环境进行实时检测，通过北斗导航模块为飞行路径进行导航，通过全景摄像头可实时获取飞行环境视频信息，并传递给地面控制中心显示屏，通过距离传感器来检测各无人机的距离，在无人机编队防碰撞控制策略中，一个重要的目标就是保证相邻僚机间的距离保持一定的安全范围，即给定一个安全距离常量，编队保持与防碰撞两种任务可以合理地兼容，在防碰撞时，僚机的横侧控制器关闭，纵向控制器和速度控制器继续跟踪长机。当距离传感器检测到相邻僚机在威胁范围内时，即开启防碰撞控制器，当相邻僚机间距离恢复至理想值时，重新打开僚机的横侧向控制器，从而将编队队形迅速恢复至理想队形。
如图2所示，队内通信模块包括发送机和接收机，对内通信模块还连接有随机选择器，需要说明的是，通过在每架无人机上设置一对发射机、接收机单播从而利用发射机和单播接收机形成无人机之间点对点通信，每架无人机每隔N秒向长机上传自己的状态信息，确认自身仍正常运行，同时长机也向其他每一架无人机发送状态信息，如果某个僚机机在N秒之后没有向长机发送状态信息，并且长机也无法向其发送信息，则判断该僚机发生损毁，若是其他僚机均无法向长机发送状态信息，并且也无法收到长机信息，则判断该长机发损毁，此时需要在编队中自动继承产生新长机，在飞行前，所有飞行器均进行编号，随机选择器中存储有所有飞行器编号，当一架飞行器损毁后，随机选择器中的飞行器编号被去除，系统自动选择距离长机最近的无人机进行“选举”，通过该无人机内的随机选择器进行随机编号选择，“选举”出新的长机。
如图2、图3、图5所示，飞行控制模块包括横侧向控制器、纵向控制器和速度控制器，更具体的为，滚转角控制是长机横侧向控制的关键，长机通过航迹跟踪控制回路来控制自身按照预定航线飞行，主要为了消除长机飞行过程中实时位置与航线的侧偏距，侧偏距的偏差信号作为横侧向控制器的输入量，减少侧偏距误差，除了设计合理的控制参数外，还要对侧偏距偏差信号进行限幅，即可实现航迹角控制指令在合理范围内，长机在沿航线自动飞行时，通过控制尾舵上的升降舵摆动方向来控制长机的爬升、平飞和下滑三个飞行动作，控制速度控制器发动机是无人机最主要的动力来源，发动机推力的大小与无人机飞行的快慢紧密相关，同时速度的大小又是无人机飞行安全的重要指标，因此将发动机通道作为速度控制通道，长机的速度控制器主要完成长机发动机通道的油门的控制。长机在沿预定航线自动飞行时，通过油门输出量大小改变发动机推力来实现长机的加速、匀速和减速。
如图2、图4所示，无人机内置有运动计算模块，需要说明的是，由于僚机与长机的飞行过程一致，当爬升至安全高度后，僚机跟随长机进入集结区域，在集结区域各僚机通过调整自身飞行状态，使得编队内各无人机的间距达到理想数值，然后以固定队形进行编队飞行，当长机将自身的位置信息发送给僚机后，僚机在得到长机的坐标后和飞行数据，通过运动计算模块来计算出自身的理想坐标位置，并且分析得到与自身实时位置的横侧向和纵向偏差，从而方便通过飞行控制模块中的横侧向控制器、纵向控制器和速度控制器对自身飞行状态进行调整，以便达到编队飞行状态。
综上所述：地面控制主机通过对空通信模块与地面通信模块进行通信，并对空中飞行器编队进行控制，无人机通过环境探测模块对环境进行感知并定位，并可将飞行信息传送回显示屏进行显示，当地面控制中心与空中编队失去联络时，通过飞行控制模块继续接管运行，继续执行飞行任务，当长机出现故障损毁后，通过对内通信模块进行自动继承，使用其他僚机成为新长机，继续保持编队飞行，在使用中实现了便于对多低空飞行器进行自动编队飞行的效果，在地面与飞行器失去联络的情况下，飞行编队仍能继续执行并完成飞行任务，并且能够在编队中自动继承更新长机，在初代长机损坏的情况下，仍然能够执行任务，从而保障了飞行编队的稳定性。
