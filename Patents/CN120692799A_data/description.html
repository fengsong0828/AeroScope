CN120692799A

Description

一种低空飞行器综合智能航电显控装置
技术领域
本发明属于航空电子设备技术领域，具体为一种低空飞行器综合智能航电显控装置。
背景技术
综合显控器是航电的核心，采用加固型高亮、高分辨率液晶显示器，主要显示飞机速度、高度、姿态、航向、发动机参数、导航信息、视频信息等，并具有数据、视频和音频记录功能，数据记录仪可记录飞行参数、语音和视频数据。综合显控器与数据记录仪采用一体化结构设计。
中国发明专利CN111232232A公开了一种用于航电设备综合显示控制的装置和方，根据应用场景、飞行阶段的不同，通过自由选择、切换各子系统的组合显示方式，能够结合实体按键旋钮和交互界面来对多种航电设备进行集中显示和控制，提高了提高人机交互效率及综合显示操作的可靠性和准确性；通过灵活地配置实体按键旋钮与交互界面的对应关系，不仅降低了座舱仪表和控制面板硬件连接复杂度，还能够提高控制输入的冗余度，确保输入功能的可靠性。
现有技术中的航电显控装置设置的显示屏均是固定安装，在进行人员观看时，在室外手光影影响会导致屏幕反光，不能进行手动调节角度灵活，无法满足不同视角需求。

为解决上述背景技术中提出的问题，本发明采用如下的技术方案。
一种低空飞行器综合智能航电显控装置，包括端盒、机盒以及前封板，端盒、机盒以及前封板自后向前依次装配组成航电显控装置壳体，端盒和机盒内置电子元件，机盒的前表面内侧边设有一圈衬边，前封板搭接在衬边上与机盒形成连接，前封板上开设容纳槽，容纳槽居中固定设有一对连接片，连接片上转动连接有大臂，大臂的前端铰接有小臂，小臂的前端上表面转动连接有屏板，屏板内嵌显示屏。
优选地，机盒的前表面顶角处且位于衬边的上表面设有半圆形凸起，前封板的顶角处开设定位缺口与半圆形凸起适配，半圆形凸起上螺纹连接有固定帽，固定帽顺时针拧动时挤压前封板边缘对前封板进行固定。
优选地，大臂上开设有供小臂嵌入的通槽，且连接片、大臂和小臂的厚度均相等，大臂与屏板的厚度之和与前封板上开设的容纳槽深度相等。
优选地，大臂与连接片的连接处、小臂与大臂的连接处以及屏板与小臂的连接处均通过设置的轴套阻尼结构进行转动连接，其中轴套阻尼结构主要由转轴、摩擦片、压缩弹簧以及锁紧螺母组成，使得大臂、小臂和屏板通过手动调节角度后得以固定。
优选地，前封板的一侧开设有腰形通槽，且腰形通槽上通过短轴转动连接有槽挡板，机盒的其中一个内侧面设置有芯片安装盒，且腰形通槽与芯片安装盒的接口正对。
优选地，芯片安装盒包括固定在机盒内的底卡以及装配在其前端的顶卡，底卡与顶卡的两端均设有连接耳，并利用螺丝穿过连接耳使的底卡与顶卡固定相连，底卡与顶卡之间夹持有插接板，插接板上开设有插槽，插槽与腰形通槽相对，插槽内插接有图形模块。
优选地，机盒内安装有背光驱动板以及视频叠加模块，其中背光驱动板与显示屏信号连接，图形模块与视频叠加模块信号连接，机盒内靠近后表面处设有主控底板，且主控底板与背光驱动板、视频叠加模块以及芯片安装盒内的图形模块信号连接。
优选地，机盒后表面设有翻边，翻边两侧等距开设有多个腰孔，且主控底板的两侧开设有螺孔，主控底板通过螺母穿过螺孔与腰孔固定在机盒后表面。
优选地，端盒内置记录仪模块以及CPU，且记录仪模块和CPU之间信号连接，且记录仪模块内置存储卡，CPU上至少连接视频转换板和视频编码板。
优选地，主控底板通过以太网外接数据源，数据源包括飞行参数类和任务载荷类。
相比于现有技术，本发明的有益效果为：
(1)本发明中大臂、小臂、屏板通过轴套阻尼结构转动连接，且厚度与容纳槽深度适配，从而实现手动调节角度灵活，满足不同视角需求，装置外观紧凑，减少飞行时的气流干扰，也便于设备在飞行器驾驶舱内布局，进一步，转轴、摩擦片、压缩弹簧、锁紧螺母组成的轴套阻尼，在飞行器复杂工况下，能通过摩擦力抵消屏板、臂杆的晃动趋势，确保显示角度稳定，避免画面抖动影响飞行员判读，相比无阻尼结构的显控屏，抗干扰能力显著提升。
(2)本发明中机盒衬边与半圆形凸起加固定帽的连接方式，既通过“衬边搭接”实现初步定位，又利用“半圆形凸起加固定帽”完成锁紧，相比传统螺丝全周固定，装配更高效，且顶角定位缺口能避免前封板安装错位，提升结构稳定性，且端盒、机盒、前封板自后向前依次装配，形成清晰的分层结构便于电子元件的分区安装，后期维修时可快速拆卸对应模块，降低维护难度。
(3)本发明中图形模块通过芯片安装盒独立装配，且与视频叠加模块、背光驱动板、主控底板信号连通，让图形渲染、视频叠加和背光调节分工明确，提升显控画面的清晰度与信息密度，帮助飞行员快速获取关键数据，芯片安装盒的插槽设计与腰形通槽，为系统功能扩展预留空间，如需升级图形渲染能力，可直接更换高性能图形模块；后期新增传感器，也可通过主控底板以太网接口快速接入，提升装置的适配性与生命周期。

实施例一
本实施例着重阐述低空飞行器综合智能航电显控装置的整体架构与各部分之间的连接方式，该航电显控装置由端盒300、机盒200以及前封板100共同组成，们自后向前依次装配，构建起航电显控装置壳体，端盒300和机盒200承担着容纳各类电子元件的重要职责，是实现航电显控功能的核心载体，机盒200的前表面内侧边精心设置了一圈衬边200a，前封板100的安装提供了稳定的支撑基础，前封板100恰好搭接在衬边200a之上，从而与机盒200顺利形成连接。
在图3和图5中，本实施例中的在机盒200的前表面顶角部位且位于衬边200a的上表面，特别设置有半圆形凸起，与之相对应，前封板100的顶角处则精确开设了定位缺口100a，该缺口与半圆形凸起完美适配，精准的定位设计，极大地提高了前封板100安装的准确性与便捷性，此外，半圆形凸起上还采用螺纹连接的方式安装有固定帽205，当操作人员顺时针拧动固定帽205时，会对前封板100的边缘施加均匀的挤压力，进而实现对前封板100的牢固固定，该固定方式不仅操作简便，而且能够在飞行器飞行过程中，有效抵御震动等外力影响，确保前封板100始终保持稳定连接。
在图4中，本实施例中的前封板100上专门开设了容纳槽，在容纳槽的居中位置，牢固地固定着一对连接片101，连接片101作为关键的连接枢纽与大臂102实现转动连接，大臂102的前端则通过铰接的方式与小臂103相连，小臂103的前端上表面进一步与屏板104转动连接，屏板104内部镶嵌有显示屏105，为观看人员提供直观的航电信息展示。
值得一提的是，本实施例中的大臂102上特意开设有供小臂103嵌入的通槽，该设计在不使用时可有效减少空间占用，使装置结构更加紧凑，而且，连接片101、大臂102和小臂103的厚度均保持相等，同时大臂102与屏板104的厚度之和恰好与前封板100上开设的容纳槽深度相等，系列精确的尺寸设计，不仅保证了整体结构的紧凑性与协调性，还为显示屏的灵活调节提供了坚实基础，确保在收纳时大臂102和小臂103能够置于容纳槽中，且通过屏板104进行覆盖遮挡，不仅可以减小体积，还能够起到保护美观作用，进一步，在大臂102与连接片101的连接处、小臂103与大臂102的连接处以及屏板104与小臂103的连接处，均精心设置了轴套阻尼结构，该结构主要由转轴、摩擦片、压缩弹簧以及锁紧螺母等关键部件组成，当观看人员需要调节显示屏角度时，只需手动操作，克服轴套阻尼结构的阻力，即可灵活调整大臂102、小臂103和屏板104的角度，而在调节完成后，由于轴套阻尼结构的作用，各部件能够牢固固定在设定角度，确保显示屏在飞行过程中不会因震动等因素而改变角度，为观看人员提供稳定的视觉视角。
在图4和图5中，本实施例中的前封板100的一侧还开设有腰形通槽100b，该通槽上通过短轴转动连接有槽挡板106。槽挡板106的设置既能够保护腰形通槽100b内部的结构，又便于在需要时打开，为后续操作提供便利，机盒200的其中一个内侧面设置有芯片安装盒204，腰形通槽100b与芯片安装盒204的接口恰好正对，这种巧妙的布局设计，使得芯片的安装与更换操作更加便捷高效。
实施例二
本实施例主要聚焦于航电显控装置内部元件的具体结构以及们之间的信号连接关系。
在图3和图5中，本实施例中的芯片安装盒204在整个装置中起着关键的数据处理与图形显示支持作用，包括固定在机盒200内的底卡2041以及装配在其前端的顶卡2044，底卡2041与顶卡2044的两端均设有连接耳2045，通过螺丝穿过连接耳2045，能够将底卡2041与顶卡2044紧密固定相连，在底卡2041与顶卡2044之间，夹持有插接板2042，插接板2042上开设有插槽2043，插槽2043与腰形通槽100b相对，插槽2043内可便捷地插接图形模块，该图形模块负责处理和生成飞行器相关的图形信息，为显示屏提供准确的图像数据，进一步，机盒200内部还安装有背光驱动板202以及视频叠加模块203，背光驱动板202与显示屏105建立信号连接，其主要功能是根据飞行环境的光线变化，智能调节显示屏105的背光亮度，确保观看人员在各种光照条件下都能清晰地读取显示屏上的信息，图形模块与视频叠加模块203之间也存在信号连接，视频叠加模块203负责将图形模块生成的图形信息与其他视频数据进行叠加处理，从而在显示屏105上呈现出丰富、全面的航电信息画面。
在图6中，本实施例中在机盒200内靠近后表面处设有主控底板201，主控底板201犹如整个装置的“大脑”，与背光驱动板202、视频叠加模块203以及芯片安装盒204内的图形模块均建立了信号连接。通过这些连接，主控底板201能够协调各模块之间的数据传输与处理工作，确保整个航电显控系统的稳定运行，进一步，本实施例中的机盒200的后表面设有翻边，翻边两侧等距开设有多个腰孔200b，主控底板201的两侧对应开设有螺孔，通过螺母穿过螺孔与腰孔200b，可将主控底板201牢固地固定在机盒200的后表面。这种安装方式不仅保证了主控底板201的稳定性，还便于其与其他外部设备进行连接。主控底板201通过以太网外接数据源，这些数据源涵盖了飞行参数类和任务载荷类等重要信息，飞行参数类数据包括飞行器的速度、高度、姿态等实时信息，任务载荷类数据则与飞行器所执行的具体任务相关，如侦察数据、测绘数据等。这些丰富的数据为航电显控装置提供了全面的信息支持，使其能够为观看人员提供准确、详细的飞行状态和任务信息。
在图8中，本实施例中的端盒300内置记录仪模块301以及CPU302，记录仪模块301和CPU302之间建立了信号连接，CPU302负责对整个航电显控装置所涉及的各类数据进行综合处理与分析，协调各个模块之间的工作流程，因此内部至少连接视频转换板和视频编码板，记录仪模块301则承担着记录飞行过程中关键数据的重要任务，包括飞行参数、设备运行状态等信息。这些记录的数据不仅有助于飞行后的数据分析与总结，还能在出现故障时为故障排查提供有力依据，确保飞行器的安全可靠运行，因此记录仪模块301内置存储卡。
如图9所示为本实施例中显控器内部元件的原理图，本实施例中基于流程图的航电显控器原理及数据指令传输过程详解如下：
一、系统核心功能与原理
航电显控器是机载综合信息处理与显示终端，通过数据采集处理显示记录流程，整合飞行导航、环境感知、任务载荷(光电吊舱)等信息，为观看人员提供可视化交互界面，同时支持数据记录与传输。
二、核心模块与部件分工
1.输入数据源
飞行参数类：飞行导航设备、大气数据处理器、姿态航向器→提供飞行姿态(高度、速度、航向等)、环境数据(气压、气温等)；
任务载荷类：光电吊舱→采集侦察/观测视频图像；机载语音通联和卫星转换→提供语音、卫星通信数据。
2.主控处理单元(主控板底板201为核心)
图形模块：负责飞行参数、任务数据的图形化渲染，将原始数据转换为可视化界面元素(如仪表、地图、标识等)；
背光驱动板202：接收主控指令，控制液晶显示屏背光亮度，适配不同环境光需求；
视屏叠加模块203：将图形模块生成的航电界面(如飞行参数)与光电吊舱视频流(经视频转换板、编码板处理后)进行叠加融合，输出复合显示画面。
3.显示与交互单元
显示屏105：呈现最终叠加后的综合画面(飞行参数+光电吊舱视频+其他任务信息)；
按键/警示灯：提供人机交互接口(参数设置、模式切换等)，并通过警示灯反馈系统状态(故障、告警等)。
4.视频处理与记录单元
光电吊舱链路：光电吊舱→视频转换板(格式转换)→视频编码板(压缩编码)→一方面通过以太网回传至主控板视屏叠加模块，另一方面写入存储卡记录；
存储卡：存储编码后的视频数据(含光电吊舱影像、叠加信息)，支持事后复盘分析。
三、数据指令传输全流程
1.飞行参数与任务数据流向
飞行导航/大气数据/姿态航向器→主控底板201(图形模块处理)→生成航电图形界面；
光电吊舱→视频转换板(原始视频格式适配)→视频编码板(H.264/H.265等编码压缩)→以太网传输→主控板视屏叠加模块(与航电图形叠加)，同时，编码后数据写入存储卡记录；
机载语音通联/卫星数据→主控板底板(图形模块或视屏叠加模块203按需处理，如语音波形显示、卫星链路状态标识)。
2.显示输出流程
视屏叠加模块203融合后的数据→经HDMI传输至图形模块(或直接驱动显示)→背光驱动板控制亮度→液晶显示屏呈现综合画面(飞行参数+光电视频+任务信息)。
3.交互与控制流向
按键操作→主控板底板接收指令→触发图形模块更新界面、背光驱动板调节亮度、视屏叠加模块切换显示模式等；警示灯直接由主控板根据系统状态(故障、数据异常等)驱动点亮。
四、关键数据接口与协议
以太网：光电吊舱视频回传、视频编码板→主控板数据交互的主要通道，基于TCP/IP协议传输编码视频流、任务指令；
HDMI：视屏叠加模块与图形模块(或直接与显示屏)的高清视频传输接口，保障画面高画质输出；
存储接口：视频编码板→存储卡通常为SATA、PCIe或专用视频存储协议，实现高速写入。
五、典型应用场景与逻辑
正常飞行模式：飞行参数(导航、姿态等)图形化显示+光电吊舱实时视频叠加→飞行员通过显示屏掌握飞行状态与任务环境，按键可切换显示布局(如全屏光电影像、分屏参数+视频)；
告警触发：大气数据异常/姿态失稳→主控板触发警示灯(声光告警)，同时图形模块在显示屏弹出告警标识(如红色闪烁框、文字提示)；
数据记录：飞行全程，视频编码板持续将叠加画面写入存储卡；地面维护时，可通过接口导出存储卡数据，分析飞行过程、任务执行细节。
综上所述，本实施例中航电显控器通过“多源数据采集→图形化处理+视频融合→可视化显示+交互控制→数据记录回溯”的闭环流程，实现飞行状态、任务载荷的一体化呈现，是机载信息化系统的“神经中枢界面”。
