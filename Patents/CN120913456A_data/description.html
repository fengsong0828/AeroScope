Description

面向低空多飞行器混合起降场的交通全景态势可视化系统
技术领域
本发明涉及低空空域管理，具体涉及面向低空多飞行器混合起降场的交通全景态势可视化系统。
背景技术
随着低空空域的逐步开放及低空飞行活动的日益频繁，低空多飞行器混合起降场面临的交通管理问题愈发复杂。当前，低空交通领域涉及多种类型的飞行器，其飞行状态、定位信息等数据来源广泛且格式多样，包括雷达定位数据、飞行器状态数据、电子围栏数据和机位占用数据在内的多源异构数据，传统数据处理方式难以对这些数据进行统一有效的处理与分析，无法构建准确、全面的低空立体交通模型。
同时，在低空交通态势信息展示方面，现有技术难以满足全面、直观地呈现飞行器的飞行状态和位置的需求。不同机型具有独特特性，对于飞行器图标与航迹的显示缺乏差异化处理，导致用户难以快速、直观地了解各飞行器的具体情况。而且，低空交通态势信息展示不够精细，无法实时、准确地为用户提供空域各区域的状态。
此外，在低空飞行活动中，“黑飞”现象时有发生，对正常低空交通的秩序和安全构成严重威胁。现有技术对于黑飞目标的识别、预警、反制措施的联动执行及可视化展示存在不足，不能及时有效地应对“黑飞”问题。并且，缺乏将多源异构数据、网格状态数据和反制联动信息进行融合并直观呈现的技术手段，难以让用户全面、直观地掌握低空交通全景态势，不利于低空交通的高效管理和安全运行。
发明内容
（一）解决的技术问题
针对现有技术所存在的上述缺点，本发明提供了面向低空多飞行器混合起降场的交通全景态势可视化系统，能够有效克服现有技术所存在的无法全面、直观、精准地展示低空交通态势信息，难以让用户全面、直观地掌握低空交通全景态势的缺陷。
（二）技术方案
为实现以上目的，本发明通过以下技术方案予以实现：
面向低空多飞行器混合起降场的交通全景态势可视化系统，包括以下组成部分：
多模态数据集成引擎，对雷达定位、飞行器状态、电子围栏和机位占用在内的多源异构数据进行统一数据处理，并构建低空立体交通数字孪生体；
三维动态渲染模块，根据不同机型的特性和需求，差异化显示图标与航迹，实现对低空飞行器的三维动态可视化，便于用户直观地了解飞行器的飞行状态和位置；
红绿灯状态实时叠加模块，将空域划分为多个三维网格单元，形成北斗三维网格空间，实时计算每个三维网格单元的状态，并叠加至地理信息底图的相应位置处，为用户提供空域的交通态势信息；
应急干预可视化模块，对黑飞目标进行自动识别和标红预警，联动反制设备对黑飞目标进行跟踪和反制，并生成处置日志；
多源数据融合可视化模块，将多源异构数据、网格状态数据和反制联动信息进行融合，实现“飞行实体+红绿灯网格+反制联动”三位一体的可视化呈现，为用户提供全面、直观的低空交通全景态势。
优选地，所述多模态数据集成引擎的功能包括：
数据接入：采用多种通信协议与不同数据源进行连接，实现数据的实时采集，对于雷达定位数据，通过雷达接口协议获取飞行器的位置、速度信息；飞行器状态数据通过飞行器自身的通信链路获取，包括飞行高度、电池电量和飞行模式；电子围栏数据和机位占用数据通过与相关管理系统进行数据交互获取；
数据清洗：对采集到的多源异构数据进行预处理，去除噪声数据、重复数据和异常数据；
数据转换：将清洗后的数据转换为统一格式，便于后续的存储和分析；
数据存储：采用分布式数据库存储处理后的数据，以提高数据的存储和查询效率，同时构建低空立体交通数字孪生体，将飞行器、电子围栏和机位在内的实体以对象的形式进行存储和管理，每个对象包含其属性信息和空间信息。

可视化呈现：在地理信息底图上同时显示多源异构数据、网格状态数据和反制联动信息，其中多源异构数据根据三维动态渲染模块的规则进行差异化显示，黑飞目标标红预警；网格状态数据以不同颜色的方块叠加显示；反制联动信息通过动态线条或箭头表示反制设备的轨迹和反制方向，实现“飞行实体+红绿灯网格+反制联动”三位一体的可视化呈现，为用户提供全面、直观的低空交通全景态势。
（三）有益效果
与现有技术相比，本发明所提供的面向低空多飞行器混合起降场的交通全景态势可视化系统，具有以下有益效果：
1）全面直观呈现，优化态势感知
多模态数据集成引擎对多源异构数据进行统一数据处理，并构建低空立体交通数字孪生体，为系统提供了准确且全面的数据基础；三维动态渲染模块根据不同机型特性差异化显示图标与航迹，实现对低空飞行器的三维动态可视化；红绿灯状态实时叠加模块将空域划分为三维网格单元，并实时计算网格状态叠加至地理信息底图，提供低空交通态势信息；多源数据融合可视化模块实现“飞行实体+红绿灯网格+反制联动”三位一体的可视化呈现，这些功能模块共同作用，使用户能够全面、直观地了解低空交通全景态势，极大地优化了用户对于低空交通态势的感知能力，有助于及时发现潜在问题和异常情况；
2）及时高效应急，保障飞行安全
应急干预可视化模块具备对黑飞目标的自动识别和标红预警功能，能够迅速发现非法飞行行为，并联动反制设备对黑飞目标进行跟踪和反制，同时生成处置日志，这一全流程的自动化应急处理机制，大大缩短了从发现“黑飞”现象到采取反制措施的时间间隔，提高了应急响应速度和处理效率，在面对黑飞等紧急情况时，能够及时有效地保障低空飞行活动的安全，减少黑飞对正常低空交通的秩序和安全的威胁；
3）深度数据融合，助力科学决策
多源数据融合可视化模块将多源异构数据、网格状态数据和反制联动信息进行深度融合，为用户提供了综合、全面的信息展示，这种融合呈现方式打破了数据之间的壁垒，使用户能够从多个维度、多个层面了解低空交通的实时运行状况，基于这些丰富且准确的信息，用户能够做出更加科学、合理的决策，例如合理规划飞行路线、优化空域资源分配、制定应急预案等，从而提升低空多飞行器混合起降场的整体运营管理水平。

五、多源数据融合可视化模块（如图1所示）的功能包括：
数据融合：将多模态数据集成引擎提供的多源异构数据、红绿灯状态实时叠加模块提供的网格状态数据和应急干预可视化模块提供的反制联动信息进行融合，采用基于时间同步的数据融合方法，确保不同数据源的数据在时间上的一致性；
可视化呈现：在地理信息底图上同时显示多源异构数据、网格状态数据和反制联动信息，其中多源异构数据根据三维动态渲染模块的规则进行差异化显示，黑飞目标标红预警；网格状态数据以不同颜色的方块叠加显示；反制联动信息通过动态线条或箭头表示反制设备的轨迹和反制方向，实现“飞行实体+红绿灯网格+反制联动”三位一体的可视化呈现，为用户提供全面、直观的低空交通全景态势。
本申请技术方案中，多模态数据集成引擎对多源异构数据进行统一数据处理，并构建低空立体交通数字孪生体，为系统提供了准确且全面的数据基础；三维动态渲染模块根据不同机型特性差异化显示图标与航迹，实现对低空飞行器的三维动态可视化；红绿灯状态实时叠加模块将空域划分为三维网格单元，并实时计算网格状态叠加至地理信息底图，提供低空交通态势信息；多源数据融合可视化模块实现“飞行实体+红绿灯网格+反制联动”三位一体的可视化呈现，这些功能模块共同作用，使用户能够全面、直观地了解低空交通全景态势，极大地优化了用户对于低空交通态势的感知能力，有助于及时发现潜在问题和异常情况。
多源数据融合可视化模块将多源异构数据、网格状态数据和反制联动信息进行深度融合，为用户提供了综合、全面的信息展示，这种融合呈现方式打破了数据之间的壁垒，使用户能够从多个维度、多个层面了解低空交通的实时运行状况，基于这些丰富且准确的信息，用户能够做出更加科学、合理的决策，例如合理规划飞行路线、优化空域资源分配、制定应急预案等，从而提升低空多飞行器混合起降场的整体运营管理水平。
