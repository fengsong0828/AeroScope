Description

一种基于CLKAN-RF框架的低空飞行器传感器异常检测方法
技术领域
本发明涉及低空飞行器传感器异常检测技术领域，具体为一种基于CLKAN-RF框架的低空飞行器传感器异常检测方法。
背景技术
低空经济产业是指利用低空空域（通常指距地面1000米以下的高度范围）进行的各种商业活动和服务的集合。随着航空技术的发展和相关政策法规的完善，低空经济正在成为新的经济增长点，涵盖了多个领域，并展现出巨大的发展潜力，其中各类低空飞行器是实现低空经济高质量发展的核心载体。
在低空飞行中，准确及时地感知外部环境和内部状态对低空飞行器的安全飞行非常重要，很大程度上依赖于各种机载传感器。然而，这些传感器作为精密仪器，对运行环境高度敏感，经常面临极端温度、高湿度和电磁干扰等恶劣条件，很容易出现异常或故障，导致测量信号出现偏差，严重影响指令计算和行动操作，有可能造成坠机事故。因此，开发有效的传感器异常检测方法对于提高低空飞行器可靠性、安全性和飞行效率至关重要。
现有的低空飞行器传感器异常检测技术主要分为基于模型的方法和数据驱动的方法前者需要精确的物理模型，而数据驱动方法无需精确的动态建模，目前已成为主流方法。
但是数据驱动的异常检测方法仍面临一些挑战：首先，获取带标签的真实传感器异常数据困难且成本高昂；其次，低空飞行器传感器数据本身具有高维性，直接利用此类数据往往会加剧过拟合风险和计算复杂性；此外，大多数研究强调数据中的时间相关性，却忽略了空间模式，限制了捕捉复杂特征的能力，降低了检测性能；最后，真实低空飞行器传感器数据中的随机噪声和干扰会增加漏检率和误检率，尤其是在使用静态阈值时，现有异常检测模型准确性低及鲁棒性差。
发明内容
本发明意在提供一种基于CLKAN-RF框架的低空飞行器传感器异常检测方法，用于执行低空飞行器传感器异常检测任务，能解决带标签的真实传感器异常数据稀缺难以获取、依赖专家经验和不相关变量对模型训练的负面影响、以及现有异常检测模型准确性低及鲁棒性差的问题。
本发明提供如下基础方案：一种基于CLKAN-RF框架的低空飞行器传感器异常检测方法，包括如下内容：
S1、获取低空飞行器正常飞行状态下的多变量传感器的数据，对多变量传感器数据进行预处理和特征选择，并划分为训练集和测试集；
S2、采用1D CNN、LSTM和KAN，构建 CLKAN多元回归模型；
S3、采用训练集训练CLKAN多元回归模型，并根据预设的目标函数，获取最优的CLKAN多元回归模型；

S5、采用双向自适应指数加权移动平均法平滑残差序列；
S6、根据平滑后的残差计算动态阈值，并通过比较测试残差和动态阈值，进行传感器异常检测。

本方案的有益效果：本方案中CLKAN-RF异常检测框架不依赖大量带标签异常数据，只使用易获取的低空飞行器正常飞行状态下的多变量传感器数据；首先，对原始数据进行重采样、归一化等一系列处理，并利用MIC相关性分析方法选择传感器参数；然后，通过所设计的CLKAN回归模型全面捕捉了低空飞行器传感器数据中复杂的时空特征和相互依存关系，基于最小损失保存最佳模型；再然后，将含异常样本的测试集输入经过训练的模型，并根据预测值和实际值计算训练残差和测试残差，另外，对残差序列使用双向自适应指数加权移动平均法平滑处理；最后，根据平滑后的残差计算动态阈值，并通过比较测试残差和动态阈值实现传感器高精度异常检测。
本方案能够实现低空飞行器传感器多种异常检测，在数据方面，由于获取大量带标签的真实传感器异常数据困难且成本高昂，故在训练期间，只使用易获取的低空飞行器正常飞行状态下的多变量传感器数据，并通过数据注入仿真生成各种传感器异常样本用于检测方法性能；在回归预测方面，充分利用低空飞行器传感器数据中时间、空间关联关系，相比当前其他先进回归算法，拥有更优越的预测性能；在预测残差检测方面，不采用固定的统计检测阈值对预测残差直接进行判读，而是对残差序列使用双向自适应指数加权移动平均法平滑后计算自适应的动态阈值，克服随机噪声和静态阈值对检测性能的不利影响。
具体地，1、本方案中不依赖大量标签异常数据，只使用低空飞行器正常飞行状态下的多变量传感器的数据，显著降低低空飞行器传感器异常检测的成本；
2、使用MIC相关性分析方法选择传感器参数，以避免不相关变量对模型训练的负面影响，同时降低计算复杂性和对专家知识的依赖；
3、结合一维CNN在局部特征、LSTM在时间特征提取和KAN在非线性数据表征方面的优势，全面捕捉了低空飞行器传感器数据中复杂的时空特征和相互依存关系，显著增强了模型的特征提取能力和鲁棒性；
4、通过双向自适应指数加权移动平均法来平滑残差序列，消除随机噪声和数据波动对模型性能的不利影响，能够显著降低误报率；
5、采用一种自适应动态阈值机制，克服了传统静态阈值的局限性，进一步降低漏报和误报，有效提升检测的精确度。
综上所述，本方案用于执行低空飞行器传感器异常检测任务，能解决带标签的真实传感器异常数据稀缺难以获取、依赖专家经验和不相关变量对模型训练的负面影响、以及现有异常检测模型准确性低及鲁棒性差的问题。

本实施例中，数据采集通过定制版的Carbon Z T-28模型飞机收集的，这款飞机翼展达2米，装备有电动发动机、副翼、襟副翼、升降舵和方向舵等关键飞行控制面。此外，飞机还集成了Holybro PX4 2.4.6自动驾驶仪、皮托管、GPS模块以及Nvidia Jetson TX2机载计算机，以支持高精度的飞行控制和数据处理。将来自该飞机多个机载传感器采集的数据经过图2所示的数据处理和特征选择后，将正常飞行状态下的数据输入到CLKAN多元回归模型训练；CLKAN模型具体结构如图3所示，包括一维卷积层、ReLU激活层、池化层、两层LSTM和两层KAN；随后，基于最小训练损失保存基于时空联合学习的最佳CLKAN多元回归模型。
为了评估CLKAN方法的有效性将其与当前先进回归模型LSTM-RF和TCN进行了比较，同时为进一步验证模型各组件的有效性做了针对性的消融实验，对不同的模型采用相同的数据，根据模型的输出计算各模型的均方误差（MSE）、均方根误差（RMSE）、平均绝对误差（MAE）、拟合优度（）；实验结果如图4所示，展示了CLKAN多元回归模型的MSE、MAE、RMSE均为最小，为0.9263，与其他方法相比最为接近1。
综合来看，本方案具有最为优秀的回归效果；同时，分别去掉1D CNN、LSTM和KAN后模型预测性能均变差，有力证实了各设计组件的有效性。
同时，得到如图5所示各方法在低空飞行器传感器异常任务上性能指标对比图。从图5中可以看出，相比其他方法CLKAN-RF性能指标最优。另外，为了验证CLKAN-RF不同组成部分的有效性，进行了有针对性的消融实验。值得注意的是，去掉RF（残差滤波）后，FPR急剧增大，ACC大幅下降，造成大量异常误检。反过来，这说明这种双向自适应指数加权移动平均法残差滤波能大幅消除随机噪声造成的负面影响。另外，从图6可以看出，三种类型的异常检测中，动态阈值相较于静态阈值均展现出了其优势，说明采用自适应的动态阈值有利于进一步提高异常检测性能。
本方案中CLKAN-RF异常检测框架不依赖大量带标签异常数据，只使用易获取的低空飞行器正常飞行状态下的多变量传感器数据。首先，对原始数据进行重采样、归一化等一系列处理，并利用MIC相关性分析方法选择传感器参数；然后，通过所设计的CLKAN回归模型全面捕捉了低空飞行器传感器数据中复杂的时空特征和相互依存关系，基于最小损失保存最佳模型；再然后，将含异常样本的测试集输入经过训练的模型，并根据预测值和实际值计算训练残差和测试残差，另外，对残差序列使用双向自适应指数加权移动平均法平滑处理；最后，根据平滑后的残差计算动态阈值，并通过比较测试残差和动态阈值实现传感器高精度异常检测。
本方案能够实现低空飞行器传感器多种异常检测，在数据方面，由于获取大量带标签的真实传感器异常数据困难且成本高昂，故在训练期间，只使用易获取的低空飞行器正常飞行状态下的多变量传感器数据，并通过数据注入仿真生成各种传感器异常样本用于检测方法性能；
在回归预测方面，充分利用低空飞行器传感器数据中时间、空间关联关系，相比当前其他先进回归算法，拥有更优越的预测性能；
在预测残差检测方面，不采用固定的统计检测阈值对预测残差直接进行判读，而是对残差序列使用双向自适应指数加权移动平均法平滑后计算自适应的动态阈值，克服随机噪声和静态阈值对检测性能的不利影响。
本方案直接使用易获取的低空飞行器正常飞行状态下的多变量传感器数据显著降低了数据获取的成本；利用MIC相关性分析方法选择传感器参数有效避免对专家知识的依赖和无关参数的负面影响；通过先进的CLKAN回归模型全面捕捉了低空飞行器传感器数据中复杂的时空特征和相互依存关系，显著提高了异常检测方法的准确率和鲁棒性；同时，采用一种双向自适应指数加权移动平均法与动态阈值有效结合的残差检测策略，进一步提高了异常检测方法的性能、适应性和可靠性。
