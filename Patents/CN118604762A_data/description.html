Description

基于波束赋型天线对低空飞行器感知识别方法
技术领域
本发明涉及雷达追踪技术领域，尤其涉及基于波束赋型天线对低空飞行器感知识别方法。
背景技术
低空飞行器数量快速增长，低空飞行器在多个领域都展现了重要价值，但同时也给低空空域管理带来了挑战，传统的雷达识别技术在低空飞行器存在识别精度低的缺陷。
发明内容
本发明提供基于波束赋型天线对低空飞行器感知识别方法，用以解决现有技术中对低空飞行器识别精度低的缺陷，实现提高低空飞行器识别精度。
本发明提供一种基于波束赋型天线对低空飞行器感知识别方法，包括：
获取多个天线阵列的回波信号，基于所述回波信号确定飞行器参数，所述飞行器参数反映飞行器的状态，所述多个天线阵列分别位于预设范围内的不同方位；
基于所述飞行器参数预测所述飞行器的运动轨迹；
基于所述回波信号以及所述飞行器参数确定初始波束成形信息，所述初始波束成形信息包括所述天线阵列的相位和权重；
将所述运动轨迹、所述飞行器参数、所述回波信号以及所述初始波束成形信息输入至已训练的赋型模型中，获取所述赋型模型输出的天线阵列控制参数，所述天线阵列控制参数包括所述天线阵列的相位、权重和振幅；
基于所述天线阵列控制参数控制所述天线阵列以实现对所述飞行器的实时追踪。

下面结合图1-2描述本发明提供的基于波束赋型天线对低空飞行器感知识别方法，如图1所示，该方法包括步骤：

本发明提供的方法，通过基于天线阵列的回波信号确定反映飞行器状态的飞行器参数，基于飞行器参数预测飞行器的运动轨迹，基于回波信号以及飞行器参数确定初始波束成形信息，初始波束成形信息包括天线阵列的相位和权重，将运动轨迹、飞行器参数、回波信号以及初始波束成形信息输入至已训练的赋型模型中，对初始波束成形信息进行进一步修正，获取赋型模型输出的天线阵列控制参数，天线阵列控制参数包括天线阵列、相位、权重和振幅，基于天线阵列控制参数控制天线阵列实现对飞行器的实时追踪，本发明基于飞行器的状态实时改变天线阵列控制参数，可以实现基于飞行器的实际状态对天线阵列发出的雷达波进行实时调整，提高对低空飞行器的识别精度。
本发明提供的方法，可以是每隔一段时间执行一次，即每隔一段时间基于当前的回波信号、波束赋型状态确定新的天线阵列控制参数，并基于新的天线阵列控制参数控制天线阵列，这样可以实现对飞行器的实时追踪。
本发明提供的方法中，天线阵列为大规模天线阵列，由稀疏圆阵构成，可以在特定方向上形成三维立体多波束，对特定空域范围多个低空飞行器进行扫描和识别，阵列间距为或以获得良好的波束形状和抑制旁瓣。
天线阵列发送雷达波，传输至飞行器和其他的物体，会被反射，被天线阵列接收，天线阵列接收到的雷达波信号为回波信号。对于天线阵列接收到的雷达波信号可以进一步通过匹配滤波算法进行处理，得到用于后续处理的回波信号。
本发明提供的方法中，利用回波信号得到反映飞行器状态的飞行器参数，具体地说，飞行器参数包括飞行器的高度、速度、与天线阵列的角度以及对天线阵列发射的雷达波的散射能力(RCS)。基于回波信号确定飞行器参数，包括：
基于回波信号的强度，通过脉冲压缩计算得到飞行器对天线阵列发射的雷达波的散射能力；
基于回波信号，利用多普勒频移得到飞行器的速度；
基于当前的天线阵列的波束赋型信息以及回波信号的到达方向确定飞行器与天线阵列的角度。
天线阵列的波束赋型信息包括天线阵列的相位、权重和振幅。
基于飞行器参数预测飞行器的运动轨迹，可以是通过粒子滤波算法实现。
基于回波信号以及飞行器参数确定初始波束成形信息，可以是通过波束成形算法实现，波束成形算法包括最大似然估计算法和最小均方误差算法。
进一步地，飞行器参数还可以包括ADS-B信息，即航空交通管制(ATC)的监视系统的数据。
基于回波信号以及飞行器参数确定的初始波束成形信息的精确度不够高，本发明提供的方法中，对于初始波束成形信息，还通过赋型模型进行修正。
赋型模型包括分类层、融合层以及决策层，将运动轨迹、飞行器参数、回波信号以及初始波束成形信息输入至已训练的赋型模型中，获取赋型模型输出的天线阵列控制参数，包括：
将飞行器参数输入至分类层，获取分类层输出的各种飞行器参数对应的类别概率，每个飞行器参数对应的类别概率对应飞行器参数与运动轨迹、回波信号以及初始波束成形信息之间的关联程度；
将飞行器参数、运动轨迹、回波信号、初始波束成形信息以及类别概率输入至融合层，基于类别概率将飞行器参数与运动轨迹、回波信号以及初始波束成形信息进行融合，得到融合结果；
将融合结果输入至决策层，获取决策层输出的天线阵列控制参数。
本发明提供的方法中，通过赋型模型，对飞行器以及天线阵列的多源信息进行融合来预测可以发出更适合追踪飞行器的天线阵列的赋型信息作为天线阵列控制参数来控制天线阵列，实现提高飞行器识别精确性的效果。
具体地说，本发明提供的赋型模型中，对多源信息进行融合，为了提高融合的准确性，本发明提供的方法中，并不是直接对所有的多源信息一同进行融合，而是先将飞行器参数进行分类，确定飞行器参数与其他信息之间的关联程度，基于这些关联程度进行融合。飞行器参数有很多种，将各种飞行器参数输入至分类层后，分类层输出每一种飞行器参数与其他的信息(运动轨迹、回波信号以及初始波束成形信息)之间的关联程度，分类层的输出以一个特征矩阵表示，特征矩阵的每一行对应一种飞行器参数，每一行中的每个元素对应飞行器参数分别与其他的信息之间的关联程度。
融合层包括第一融合层和第二融合层，分类层输出的各种飞行器参数对应的类别概率可以反映飞行器参数受其他种类信息的影响程度，在第一融合层中，基于类别概率，可以将飞行器参数加入至运动轨迹、回波信号和初始波束成形信息中关联程度最大的信息中，得到各组待融合信息，也就是说，待融合信息总共有三组，分别为运动轨迹和对应的飞行器参数、回波信号和对应的飞行器参数、初始波束成形信息和对应的飞行器参数。在第一融合层中，利用贝叶斯滤波数据融合算法将每组待融合信息中的信息进行融合，得到各个第一融合结果，在第二融合层中，根据多源信息(回波信号、初始波束成形信息和运动轨迹)的可靠性进行权重分配，基于权重对各个第一融合结果进行融合，得到第二融合结果，第二融合结果为分类层输出的融合结果。
在决策层中，利用最大似然估计算法对各个第二融合结果进行决策，得到天线阵列控制参数并输出。
得到赋型模型输出的天线阵列控制参数后，基于天线阵列控制参数控制天线阵列，使得天线阵列发出理想的波束，可以适应飞行器的状态变化，使得接收到的回波信号更加有利于对飞行器的识别。
