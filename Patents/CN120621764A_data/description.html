Description

低空飞行器的折叠式起落架装置
技术领域
本发明涉及起落架技术领域，尤其涉及低空飞行器的折叠式起落架装置。
背景技术
无人机作为低空飞行器的一种，因其高效、灵活的特点，广泛应用于农业植保领域。
然而，现有技术中的农业植保无人机在低空喷洒作业时，通常采用固定式起落架结构，虽然结构简单，但是飞行过程中无法折叠，增加了空气阻力，导致无人机能耗上升，续航能力降低，而且，低空作业时，固定式起落架容易与高秆作物(如玉米、甘蔗等)发生缠绕或碰撞，从而影响飞行安全。
发明内容
针对现有技术的不足，本发明提供了低空飞行器的折叠式起落架装置，解决了现有技术中农业植保无人机多采用固定式起落架，空气阻力大、能耗高的技术问题，具备能够有效降低风阻、减少能耗的优点。
为解决上述技术问题，本发明提供了如下技术方案：低空飞行器的折叠式起落架装置，包括飞行器本体，飞行器本体上设有用于农作物施肥的药液喷洒机构，所述飞行器本体的下端设置有高度可调节的起落架机构，起落架机构上设置有降落缓冲机构，降落缓冲机构用于降低着地瞬间对起落架机构造成的冲击，飞行器本体上设置有联动辅助机构，低空飞行过程中，药液喷洒机构会自动向农作物喷洒药物，降落时，降落缓冲机构会自动充气实现软着陆，能有效防止对起落架机构造成较大冲击，所述起落架机构包括固定板体，固定板体可拆卸安装在飞行器本体上，固定板体上对称设置有安装底座，两个安装底座上分别活动安装有一级架体，一级架体上滑动连接有二级架体，安装底座的内部设有驱动一级架体旋转的折叠驱动件，二级架体的下端设置有支撑横板，支撑横板与二级架体之间设置有伸缩接头，二级架体斜向下伸出时，伸缩接头会同步伸长，二级架体朝着一级架体的内部收缩时，伸缩接头会同步缩短，从而保证起落架机构能够收缩折叠。
优选的，所述折叠驱动件包括套设在一级架体上的从动齿轮，安装底座的内部设有与从动齿轮啮合的驱动齿轮，驱动齿轮转动时，会通过与从动齿轮的配合，使一级架体旋转，从而使一级架体从竖直状态转动至水平状态。
优选的，所述一级架体的内部固定安装有用于推动二级架体的电伸缩杆，电伸缩杆伸长时，会使二级架体向外伸出，进而使支撑横板高度降低。
优选的，所述降落缓冲机构包括开设在支撑横板底部的安装凹槽，安装凹槽的内部设置有缓冲气囊，支撑横板上固定安装有充气组件，充气组件用于向缓冲气囊输入气体，初始时，缓冲气囊处于收缩状态，充气组件通电运转后，会自动向缓冲气囊的内部充入气体，从而使缓冲气囊膨胀变大。
优选的，所述支撑横板上嵌入设置有安装圆管，安装圆管上滑动连接有金属滑块，金属滑块上固定安装有橡胶垫片，金属滑块与安装圆管之间固定连接有蓄力弹簧，着陆前，蓄力弹簧处于松弛状态，着陆后，金属滑块会对蓄力弹簧造成挤压，从而对着地产生的冲击进行缓冲。
优选的，所述安装圆管的内壁上嵌入设置有电磁铁组件，电磁铁组件通电时，会对金属滑块进行磁吸固定，电磁铁组件断电后，金属滑块会在蓄力弹簧的作用下快速伸出。
优选的，所述药液喷洒机构包括可拆卸安装在飞行器本体上的药液储箱，药液储箱上设置有加药通道，药液储箱的两侧对称设置有喷洒组件，飞行器本体飞行过程中，喷洒组件会将药液储箱内部的药液均匀喷出。
优选的，所述固定板体的两侧对称设置有传感器组件，传感器组件能够对飞行器本体的飞行高度进行实时监测，从而与控制器配合对起落架机构的状态进行调整。
优选的，所述联动辅助机构包括固定安装在一级架体上的加固横杆，加固横杆上固定安装有弧形齿条，飞行器本体的内部活动安装有活动转轴，活动转轴的外部套设有固定齿轮，药液储箱的内部活动安装有搅拌竖轴，搅拌竖轴与活动转轴之间设置有传动组件，搅拌竖轴上固定安装有搅拌叶片，搅拌竖轴转动时，会通过与搅拌叶片的配合对药液进行搅拌混合，从而保证药液的均匀性。
优选的，所述固定齿轮与弧形齿条啮合，传动组件为锥齿轮组，弧形齿条移动时，会使活动转轴旋转，活动转轴旋转时会利用传动组件使搅拌竖轴转动。
借由上述技术方案，本发明提供了低空飞行器的折叠式起落架装置，至少具备以下有益效果：
1、本发明通过设置起落架机构，采用动态折叠设计，起飞后，二级架体回缩带动伸缩接头缩短，从而减少整体高度，能有效降低飞行时的空气湍流，同时，一级架体旋转至与机身平行，使起落架机构的横截面面积大幅减小，能显著降低高速飞行时的风阻损耗。
2、本发明通过设置起落架机构，实现了不同工况下的高效切换，地面支撑模式下，一级架体竖直锁定且二级架体完全伸出，形成稳定构型，确保支撑力均匀分布，飞行收纳模式下，电伸缩杆回拉二级架体，结合齿轮组旋转折叠一级架体，实现紧凑收纳，能有效避免触碰作物，显著提升了作业效率与复杂地形的适应能力，同时保障了起落架机构的稳定性和可靠性。
3、本发明通过设置降落缓冲机构，降落过程中，充气组件首先使缓冲气囊快速充气膨胀，形成第一道缓冲屏障，从而有效吸收初始冲击能量，气囊接触地面后，内部气体通过阀门缓慢排出，使橡胶垫片平稳接触地面完成第二段缓冲，实现了飞行器本体的智能软着陆，有效避免了传统硬着陆对起落架造成的损伤。
4、本发明通过设置降落缓冲机构，着陆完成后，金属滑块压缩蓄力弹簧储存能量，并通过电磁铁组件实现稳定锁定，下次起飞时，电磁铁组件断电瞬间释放蓄力弹簧储存的能量，为飞行器本体提供向上的辅助推力，成功的将缓冲系统与起飞辅助功能一体化集成，优化了能源利用效率。
5、本发明通过设置联动辅助机构，创造性地将起落架折叠动作与药液搅拌功能联动结合，无需额外动力源，在起落架常规收放过程中自动完成药液搅拌，既确保了药剂均匀混合，避免沉淀导致的喷洒不均，又实现了能源的智能复用。
6、本发明通过设置联动辅助机构，通过弧形齿条与锥齿轮组的精密配合，将起落架的旋转运动高效转化为搅拌竖轴的垂直转动，具有传动效率高、维护简单的特点，且完全避免了电子控制系统在农药腐蚀环境下的故障风险。
7、本发明通过设置药液喷洒机构，实现了喷洒作业与起落架状态的协同优化，传感器组件实时监测飞行高度并反馈至控制器，使起落架机构能够根据作业场景自动调节折叠状态，这种自适应调节机制不仅能使飞行风阻降低，还避免了起落架与作物的意外碰撞，实现了植保作业安全性和经济性的双重提升。

实施例一
现有技术中的农业植保无人机在低空喷洒作业时，通常采用固定式起落架结构，虽然结构简单，但是飞行过程中无法折叠，增加了空气阻力，导致无人机能耗上升，续航能力降低，而且，低空作业时，固定式起落架容易与高秆作物(如玉米、甘蔗等)发生缠绕或碰撞，从而影响飞行安全。为了解决现有技术中存在的这一技术缺陷，如图1-图7所示，本实施例提出了低空飞行器的折叠式起落架装置，能有效降低飞行时的空气湍流，显著降低高速飞行时的风阻损耗，该装置飞行器本体1上设有用于农作物施肥的药液喷洒机构2，飞行器本体1的下端设置有高度可调节的起落架机构3，起落架机构3上设置有降落缓冲机构4，降落缓冲机构4用于降低着地瞬间对起落架机构3造成的冲击，飞行器本体1上设置有联动辅助机构5，低空飞行过程中，药液喷洒机构2会自动向农作物喷洒药物，降落时，降落缓冲机构4会自动充气实现软着陆，能有效防止对起落架机构3造成较大冲击。
具体的，起落架机构3包括固定板体301，固定板体301可拆卸安装在飞行器本体1上，固定板体301上对称设置有安装底座302，两个安装底座302上分别活动安装有一级架体303，一级架体303上滑动连接有二级架体304，一级架体303的内部固定安装有用于推动二级架体304的电伸缩杆，电伸缩杆伸长时，会使二级架体304向外伸出，进而使支撑横板306高度降低，安装底座302的内部设有驱动一级架体303旋转的折叠驱动件305，折叠驱动件305包括套设在一级架体303上的从动齿轮，安装底座302的内部设有与从动齿轮啮合的驱动齿轮，驱动齿轮转动时，会通过与从动齿轮的配合，使一级架体303旋转，从而使一级架体303从竖直状态转动至水平状态，二级架体304的下端设置有支撑横板306，支撑横板306与二级架体304之间设置有伸缩接头307，二级架体304斜向下伸出时，伸缩接头307会同步伸长，二级架体304朝着一级架体303的内部收缩时，伸缩接头307会同步缩短，从而保证起落架机构3能够收缩折叠。
根据上述内容可知，起飞前，如图1所示，一级架体303处于竖直状态，二级架体304处于伸出状态，伸缩接头307处于伸长状态，从而对飞行器本体1进行稳定支撑。
飞行器本体1起飞后，首先，二级架体304会在电伸缩杆的作用下朝着一级架体303的内部收缩，从而使支撑横板306的高度上升，与此同时，伸缩接头307会同步缩短，从而使起落架机构3的体积缩小，能有效降低起落架机构3在飞行过程中的风阻。
当飞行器需要低空盘旋进行药液喷洒作业时，一级架体303会在驱动齿轮的作用下，从竖直状态转动成与固定板体301平行的水平状态，既能在一定程度上防止药液飞溅到起落架机构3上，又能防止起落架机构3与高秸农作物发生碰撞，兼顾了气动性能优化与田间作业的适应性，大幅提升了飞行安全性和作业效率。
本实施例通过设置起落架机构3，采用动态折叠设计，起飞后，二级架体304回缩带动伸缩接头307缩短，从而减少整体高度，能有效降低飞行时的空气湍流，同时，一级架体303旋转至与机身平行，使起落架机构3的横截面面积大幅减小，能显著降低高速飞行时的风阻损耗；而且，本实施例通过设置起落架机构3，实现了不同工况下的高效切换，地面支撑模式下，一级架体303竖直锁定且二级架体304完全伸出，形成稳定构型，确保支撑力均匀分布，飞行收纳模式下，电伸缩杆回拉二级架体304，结合齿轮组旋转折叠一级架体303，实现紧凑收纳，能有效避免触碰作物，显著提升了作业效率与复杂地形的适应能力，同时保障了起落架机构3的稳定性和可靠性。
实施例二
为了实现了飞行器本体1的智能软着陆，避免传统硬着陆对起落架造成的损伤，在实施例一的基础上，如图1、图2、图4以及图6-图8所示，本实施例设置了降落缓冲机构4，具体的，降落缓冲机构4包括开设在支撑横板306底部的安装凹槽401，安装凹槽401的内部设置有缓冲气囊402，支撑横板306上固定安装有充气组件403，充气组件403用于向缓冲气囊402输入气体，初始时，缓冲气囊402处于收缩状态，充气组件403通电运转后，会自动向缓冲气囊402的内部充入气体，从而使缓冲气囊402膨胀变大，支撑横板306上嵌入设置有安装圆管404，安装圆管404上滑动连接有金属滑块405，金属滑块405上固定安装有橡胶垫片406，金属滑块405与安装圆管404之间固定连接有蓄力弹簧407，着陆前，蓄力弹簧407处于松弛状态，着陆后，金属滑块405会对蓄力弹簧407造成挤压，从而对着地产生的冲击进行缓冲，安装圆管404的内壁上嵌入设置有电磁铁组件408，电磁铁组件408通电时，会对金属滑块405进行磁吸固定，电磁铁组件408断电后，金属滑块405会在蓄力弹簧407的作用下快速伸出。
根据上述内容可知，飞行器着陆前，充气组件403会自动向缓冲气囊402的内部充气，接下来，如图7所示，缓冲气囊402会向下膨胀变大，从而在起落架机构3着地前进行缓冲，防止起落架机构3降落时与地面硬接触造成损伤。
缓冲气囊402与地面接触后，充气组件403会停止工作，随后，缓冲气囊402内部的气体会通过阀门缓慢向外排出，从而使橡胶垫片406能够缓慢平稳的与地面接触，实现软着陆。
缓冲气囊402内部的气体全部排出后，整个飞行器本体1的重量都会压在橡胶垫片406上，接下来，金属滑块405会朝着安装圆管404的内部滑动，从而对蓄力弹簧407造成挤压。
随后，电磁铁组件408会通电产生磁性，对金属滑块405进行磁吸固定，从而使蓄力弹簧407始终处于压缩蓄能状态，下一次起飞时，电磁铁组件408会自动断电，接下来，金属滑块405会在蓄力弹簧407的作用下快速向外伸出，从而给飞行器本体1一个向上的推力。
本实施例通过设置降落缓冲机构4，降落过程中，充气组件403首先使缓冲气囊402快速充气膨胀，形成第一道缓冲屏障，从而有效吸收初始冲击能量，气囊接触地面后，内部气体通过阀门缓慢排出，使橡胶垫片406平稳接触地面完成第二段缓冲，实现了飞行器本体1的智能软着陆，有效避免了传统硬着陆对起落架造成的损伤；而且，本实施例通过设置降落缓冲机构4，着陆完成后，金属滑块405压缩蓄力弹簧407储存能量，并通过电磁铁组件408实现稳定锁定，下次起飞时，电磁铁组件408断电瞬间释放蓄力弹簧407储存的能量，为飞行器本体1提供向上的辅助推力，成功的将缓冲系统与起飞辅助功能一体化集成，优化了能源利用效率。
实施例三
为了实时监测飞行高度并反馈至控制器，使起落架机构3能够根据作业场景自动调节折叠状态，在上述实施例的基础上，如图1-图3所示，本实施例设置了药液喷洒机构2，具体的，药液喷洒机构2包括可拆卸安装在飞行器本体1上的药液储箱201，药液储箱201上设置有加药通道202，药液储箱201的两侧对称设置有喷洒组件203，飞行器本体1飞行过程中，喷洒组件203会将药液储箱201内部的药液均匀喷出，固定板体301的两侧对称设置有传感器组件204，传感器组件204能够对飞行器本体1的飞行高度进行实时监测，从而与控制器配合对起落架机构3的状态进行调整。
根据上述内容可知，起飞前，工作人员会将药液通过加药通道202添加到药液储箱201的内部，起飞后，药液会通过喷洒组件203自动向外喷出。
与此同时，固定板体301两侧的传感器组件204会实时监测飞行高度，并将数据反馈至控制器，从而动态调节起落架机构3的折叠状态。
本实施例通过设置药液喷洒机构2，实现了喷洒作业与起落架状态的协同优化，传感器组件204实时监测飞行高度并反馈至控制器，使起落架机构3能够根据作业场景自动调节折叠状态，这种自适应调节机制不仅能使飞行风阻降低，还避免了起落架与作物的意外碰撞，实现了植保作业安全性和经济性的双重提升。
实施例四
为了将起落架折叠动作与药液搅拌功能联动结合，实现能源的智能复用，如图2、图9以及图10所示，本实施例设置了联动辅助机构5，具体的，联动辅助机构5包括固定安装在一级架体303上的加固横杆501，加固横杆501上固定安装有弧形齿条502，飞行器本体1的内部活动安装有活动转轴503，活动转轴503的外部套设有固定齿轮504，药液储箱201的内部活动安装有搅拌竖轴506，搅拌竖轴506与活动转轴503之间设置有传动组件505，固定齿轮504与弧形齿条502啮合，传动组件505为锥齿轮组，弧形齿条502移动时，会使活动转轴503旋转，活动转轴503旋转时会利用传动组件505使搅拌竖轴506转动，搅拌竖轴506上固定安装有搅拌叶片507，搅拌竖轴506转动时，会通过与搅拌叶片507的配合对药液进行搅拌混合，从而保证药液的均匀性。
根据上述内容可知，起飞完成后，一级架体303会在折叠驱动件305的作用下转动至水平贴合状态，一级架体303转动时会使弧形齿条502同步转动，弧形齿条502转动时会通过与固定齿轮504的配合，使活动转轴503转动。
活动转轴503转动时，会通过与传动组件505的配合使搅拌竖轴506转动，从而对药液储箱201内部的药液进行搅拌混合，避免发生沉淀，保证药液的均匀性。
本实施例通过设置联动辅助机构5，创造性地将起落架折叠动作与药液搅拌功能联动结合，无需额外动力源，在起落架常规收放过程中自动完成药液搅拌，既确保了药剂均匀混合，避免沉淀导致的喷洒不均，又实现了能源的智能复用；而且，本实施例通过设置联动辅助机构5，通过弧形齿条502与锥齿轮组的精密配合，将起落架的旋转运动高效转化为搅拌竖轴506的垂直转动，具有传动效率高、维护简单的特点，且完全避免了电子控制系统在农药腐蚀环境下的故障风险。
