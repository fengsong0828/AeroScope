Description

离心式低空飞行器
本实用新型涉及一种飞行器，特别是涉及一种离心式低空飞行器。
飞行器种类各式各样，中国专利CN2074662介绍了一种扑翼式飞行器，由提供动力的发动机，空气压缩机，推拉机翼扑动的冲击发动机，支承机翼的翼轴，翼根铰接在翼轴上，可绕翼轴上下扑动的机翼，导向翼板及操纵转盘，操纵杆及行走轮等构成。可在空中作进退升降和垂直升降等形式飞行，适用于通常代步或娱乐及民事和军事的低空飞行。
本实用新型的目的在于提供一种无机翼，设置有两个螺旋桨的离心式低空飞行器。
本实用新型离心式低空飞行器是这样实现的，它包括仓体，设置在仓体内的操纵台，安装在仓体底面的可收支撑架，仓体顶部开有排气通道，并安装中部有凹槽且与排气通道连为一体的隔离盘，设置在仓体上由离心发动机带动的离心螺旋桨和由逆向发动机带动并安装在隔离盘凹槽内的逆向螺旋桨，仓体底部活动连接有带车轮的下底盘。其中隔离盘与仓体顶部可为一整体结构，排气通道一般为6个，末端连接有可转动的活动咀，用于控制飞行方向。带车轮的下底盘用于在地面上作短距离行驶。而飞行时又能脱落，以减轻重量，为保证离心螺旋桨正常运转，防止起动和停转时损坏离心螺旋桨，应设在离心螺旋桨轴上设置缓冲及惯性释放器，它包括装套在离心螺旋桨轴上的带缺口的缓冲器轴套，装套在缓冲器轴套上的离心螺旋桨固定盘，安装在离心螺旋浆固定盘上且与缺口相配合的制动销，其一端固定在离心螺桨轴上，另一端固定在缓冲器轴套上的缓冲弹簧座，安装在缓冲弹簧座上的缓冲弹簧。工作原理是，当1号发动机运转时，将动力传递给离心螺旋桨轴，此时，缓冲弹簧座又将压缩弹簧产生的压缩缓冲动能传递给缓冲器轴套，使缓冲器轴套转动，其上的缺口与制动销相配合，使离心螺旋浆固定盘与缓冲器轴套成为一整体，从而离心螺旋浆得到缓冲起步。当离心螺旋浆发动机停转时，离心螺旋浆由于惯性作用而继续转动，此时，缓冲器轴套上的缺口与制动销由相互配合变为相互滑让，虽然离心螺旋桨轴已不动，但离心螺旋桨可继续转动，直至惯性释放完。为防止发动机因事故忽然停转，保证飞行安全，最好为离心发动机和逆向发动机配置安装保险牙箱，它包括箱体，设置在箱体内并安装在离心螺旋桨轴上的离心螺旋桨轴伞齿，安装在逆向螺旋桨轴上的逆向螺旋桨轴伞齿，安装在离心螺旋桨发动机轴上且与离心螺旋桨轴伞齿配合的离心螺旋发动机轴伞齿，安装在逆向发动机轴上且与逆向螺旋桨轴伞齿配合的逆向发动机轴伞齿，以及可与离心螺旋桨轴伞齿及逆向螺旋桨轴伞齿配合的游动齿轮。工作原理是，如果其中一个发动机停止运转，只要通过操纵台将游动齿轮与离心螺旋桨轴伞齿和逆向螺旋桨轴伞齿啮合，可使两螺旋桨轴作低功能运转，使其安全降落。
本实用新型的工作原理是，离心螺旋桨转动对，将隔离盘上的空气向盘沿外排离，使隔离盘上的压力大大降低；同时逆向螺旋桨转动时又将隔离盘凹槽中的空气通过排气通道排出，使隔离盘上为负压区，隔离盘下为常压区，从而产生强大的压力差，以获得上升力。此外，利用排气通道排出空气的压力，转动排气通道末端的活动咀，可控制飞行器的飞行方向。由于两螺旋桨运转方向相反，也通过可改变两螺旋桨的转速，使其产生扭力差来改变飞行方向。
本实用新型的优点在于结构简单，不需跑道，可垂直升降，进行低空飞行，可缓解地面交通拥挤，主要用于客货运输及游乐等。
