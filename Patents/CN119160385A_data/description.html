Description

一种防旋翼断裂的低空飞行器
技术领域
本发明涉及低空飞行器技术领域，尤其涉及一种防旋翼断裂的低空飞行器。
背景技术
低空飞行器是指主要在1000米以下范围内运行的飞行器，现有低空飞行器主要包括固定翼飞行器和旋翼飞行器两种，其中旋翼飞行器因其具有竖直升降的功能，对起降环境要求较低，在野外亦可进行升降，因此在测绘和勘测领域被广泛使用，现有旋翼式飞行器（简称飞行器）大都采用多旋翼的结构，且为保证飞行器在飞行过程中的稳定性和方便对旋翼进行检修，旋翼大多处于飞行器偏下的位置，由于现有飞行器支撑架大多为固定式的U形结构，飞行器在飞行过程中出现突发情况而需要紧急降落时，若飞行器降落的地面不平整，飞行器的支撑架在与地面接触过程中，飞行器的旋翼会存在地面接触的情况，导致旋翼损毁无法继续使用。
发明内容
本发明提供了一种防旋翼断裂的低空飞行器，为了克服飞行器在不平整的地面降落时，飞行器的旋翼容易与地面接触并损坏的问题。
本发明的技术实施方案为：一种防旋翼断裂的低空飞行器，包括有：底座，所述底座的上侧设置有机壳；连接架，具有四个，均固接于所述底座；升降机构，数量与所述连接架的数量相同，分别设置于相邻所述连接架上，所述升降机构用于带动所述底座移动，所述升降机构包括有：动力电机，所述动力电机的输出轴固接有旋翼；液压伸缩杆，数量与所述连接架的数量相同，均固接于所述底座，所述液压伸缩杆的伸缩端铰接有支撑架，且二者之间设置有第一扭簧；连通壳，固接于所述底座，所述液压伸缩杆通过第一输液管与所述连通壳连通；密封组件，设置于所述连通壳上，用于调节四个所述液压伸缩杆之间的连通关系。
进一步的，所述密封组件包括有：密封环，转动连接于所述连通壳内，所述连通壳内滑动连接有密封壳，所述密封壳与所述连通壳之间设置有第一弹簧，所述密封壳与所述密封环滑动连接；密封板，数量具有两个，均滑动连接于所述密封壳，所述密封板与所述密封壳之间设置有第一拉簧，所述连通壳和所述密封环均与所述密封板滑动连接；动力组件，设置于所述底座上，用于带动所述密封板移动；转动组件，设置于所述底座上，用于带动所述密封壳转动。
进一步的，所述动力组件包括有：滑动杆，滑动连接于所述密封壳，所述密封壳与所述滑动杆之间设置有第二弹簧，所述滑动杆与所述连通壳滑动连接；挤压块，固接于所述滑动杆，所述密封板与所述挤压块挤压配合；第一电动推杆，固接于所述底座，所述第一电动推杆的伸缩端固接有调节架，所述调节架与所述滑动杆转动连接。
进一步的，所述转动组件包括有：传动电机，固接于所述底座上；花键壳，固接于所述传动电机的输出轴，所述花键壳与所述底座转动连接，所述花键壳与所述密封壳花键连接；检测组件，设置于所述支撑架上，用于检测所述支撑架是否与地面接触。
进一步的，所述检测组件包括有：检测板，数量与所述支撑架的数量相同，分别滑动连接于相邻所述支撑架，所述支撑架设置有检测腔，所述检测板位于该检测腔内，所述检测板与所述支撑架之间设置有第三弹簧，所述支撑架的检测腔通过连通管与所述连通壳连通；限位环，滑动连接于所述连通壳内，所述限位环与所述密封壳滑动连接，所述密封壳与所述限位环之间设置有第四弹簧；限位组件，设置于相邻所述液压伸缩杆上，用于限制所述液压伸缩杆伸缩部移动的距离。
进一步的，所述限位组件包括有：限位壳，数量与所述液压伸缩杆的数量相同，分别固接并连通于相邻所述液压伸缩杆的固定部，所述限位壳通过输气管与所述连通壳连通；限位杆，滑动连接于所述限位壳内，所述限位杆与所述限位壳之间设置有第二拉簧，所述限位杆与所述液压伸缩杆的伸缩部限位配合。
进一步的，还包括有：电动滑轨，固接于所述连接架，所述电动滑轨内设置有滑块，所述动力电机与该滑块固接；固定组件，数量与所述动力电机的数量相同，分别设置于相邻所述连接架上，用于加强对所述动力电机的固定；调节机构，数量与所述动力电机的数量相同，分别设置于相邻所述连接架上，用于检测所述液压伸缩杆伸缩端的移动距离。
进一步的，所述固定组件包括有：第二电动推杆，固接于所述连接架内，所述第二电动推杆的伸缩端固接有调节伸缩杆，所述调节伸缩杆与所述连接架滑动连接；固定板，固接于所述调节伸缩杆的伸缩端，所述固定板与所述连接架滑动连接；固定架，数量为所述连接架的二倍，铰接于所述连接架，且所述固定架与所述连接架之间设置有第二扭簧，所述固定架与所述动力电机限位配合，所述固定架与所述固定板挤压配合。
进一步的，所述调节机构包括有：滑动管，固接于所述连接架内，所述滑动管滑动连接有升降架，所述升降架与所述动力电机固接，所述滑动管内滑动连接有调节板，所述调节板与所述升降架挤压配合；挤压杆，固接于所述液压伸缩杆的伸缩端；储液管，固接于所述底座，所述调节伸缩杆和所述滑动管均通过第二输液管与所述储液管连通，所述储液管滑动连接有限位架，所述储液管与所述限位架之间设置有第三拉簧，所述限位架与所述挤压杆挤压配合；调节块，滑动连接于所述储液管内，所述调节块与所述储液管之间设置有第四拉簧，且所述限位架与所述调节块限位配合。
进一步的，所述升降架位于所述滑动管内的部分设置有通孔。
本发明的有益效果为：1、本发明在飞行器下降的过程中，通过四个液压伸缩杆与连通壳连通，使飞行器在下降过程中调整液压伸缩杆伸缩端的伸出量，使飞行器在落地后保持水平状态，从而适应地面的不平整，减小飞行器在落下的过程中出现旋翼受损的概率。
2、本发明在飞行器出现偏斜时，通过滑动杆分隔连通壳，调节四个液压伸缩杆之间的连通关系，仅使飞行器偏斜一侧的两个液压伸缩杆连通，从而对飞行器偏斜的一侧进行单独调平，保证旋翼不会与地面接触。
3、本发明在飞行器出现偏斜时，通过陀螺仪检测飞行器偏斜的方向并启动相应位置的电动滑轨，使电动滑轨带动该位置的旋翼向上移动远离地面，从而减小旋翼与地面接触的概率。

具体实施方式
下面结合具体实施例对本发明作进一步描述，在此发明的示意性实施例以及说明用来解释本发明，但并不作为对本发明的限定。
一种防旋翼断裂的低空飞行器，如图1-图6所示，包括有：底座1，底座1的上侧设置有机壳2；连接架3，具有四个，均固接于底座1；升降机构4，数量与连接架3的数量相同，分别设置于相邻连接架3上，升降机构4用于带动底座1移动，升降机构4包括有：动力电机402，动力电机402的输出轴固接有旋翼403；液压伸缩杆5，数量与连接架3的数量相同，均固接于底座1，液压伸缩杆5的伸缩端铰接有支撑架6，且二者之间设置有第一扭簧；连通壳7，固接于底座1，液压伸缩杆5通过第一输液管与连通壳7连通；密封组件，设置于连通壳7上，用于调节四个液压伸缩杆5之间的连通关系。
在上述方案中，提出了一种能使飞行器在不平整的地面上保持水平的方式，旨在解决现有飞行器在不平整的地面上降落时，机身在降落的过程中无法保证水平，导致旋翼403容易与地面接触并损毁的问题；底座1的下部左右两侧均设置有倒角，且底座1下部倒角的边缘设置有圆角，用于减小底座1移动时气流对其产生的阻力；机壳2的上部为流线型，用以减小其移动时受到气流的阻力，机壳2的前后两侧均设置有门，用于使工作人员进入机壳2内，连接架3为空心结构，用于减小本装置的总重量；升降机构4用于带动底座1上升，动力电机402为双轴电机，动力电机402的两个输出轴均固接有旋翼403，用于保证本装置上升时具有足够的动力；液压伸缩杆5用于调整支撑架6与底座之间的距离，保证在降落过程中底座1处于水平状态，且在此过程中，处于高位置的液压伸缩杆5的伸缩端收回，其内的液压油通过连通壳7向低处的液压伸缩杆5内流动，使低处的液压伸缩杆5的伸缩端伸出，从而使此处的支撑架6能提前与地面接触并进行支撑，从而使底座1水平，支撑架6用于适应地面的倾斜角度，增加支撑架6与地面的接触面积。
进一步的，如图4-图6所示，密封组件包括有：密封环8，转动连接于连通壳7内，连通壳7内滑动连接有密封壳9，密封壳9与连通壳7之间设置有第一弹簧，密封壳9与密封环8滑动连接；密封板10，数量具有两个，均滑动连接于密封壳9，密封板10与密封壳9之间设置有第一拉簧，连通壳7和密封环8均与密封板10滑动连接；动力组件，设置于底座1上，用于带动密封板10移动；转动组件，设置于底座1上，用于带动密封壳9转动。
上述方案中，在飞行器受大风影响出现偏斜时，控制四个液压伸缩杆5之间的连通关系，使飞行器偏斜时先落地的一侧水平，连通壳7、密封环8和密封壳9的轴线同轴，密封壳9用于带动两个密封板10移动；密封环8将连通壳7分为上下两个腔室，且液压伸缩杆5与连通壳7的上侧腔室连通，密封壳9与两个密封板10配合将连通壳7的上侧腔室进一步分为前后两个腔室（以图6的方向为参考），从而使四个液压伸缩杆5之间两两连通，两个密封板10的外侧面均为弧面，且弧面的曲率与连通壳7的内侧面的曲率相同，使密封板10与连通壳7之间能够紧密贴合，从而保证密封板10与连通壳7之间的密封性，密封环8设置有两个滑动槽，且密封板10能遮挡密封环8相邻的滑动槽，在密封板10将连通壳7的上侧腔室分隔为两个后，这两个腔室内的液压油无法通过密封环8的滑动槽互相流通，且密封板10的下部始终位于密封环8的滑动槽内，使密封板10与密封环8之间不会出现相对转动，动力组件用于带动密封板10移动，使密封板10分隔连通壳7的上侧腔室；转动组件用于带动密封壳9转动，使密封板10的角度改变，从而调正四个液压伸缩杆5之间的连通关系。
工作流程：在飞行器遇到紧急情况需要降落时，若无大风影响飞行器降落时，飞行器保持水平竖直向下移动，若地面水平，此时四个支撑架6同时与地面接触，液压伸缩杆5伸缩部伸出的距离不变，飞行器正常降落，若地面不水平，以地面的左侧低于右侧为例，随着飞行器向下移动，右侧的支撑架6先与地面接触且接触后停止移动，随着飞行器继续向下移动，右侧的液压伸缩杆5的伸缩部被压缩，右侧液压伸缩杆5内的液压油通过连通壳7进入左侧的液压伸缩杆5内，使左侧液压伸缩杆5的伸缩端伸出并带动其上的支撑架6向下移动，直至左侧的支撑架6与地面接触后其停止移动，此时四个支撑架6均停止移动，支撑架6完成降落。
当飞行器在下降过程中受风力影响向一侧偏斜时，以飞行器右侧向下偏斜，且地面前侧高于其后侧为例，在检测到机身倾斜之后，动力组件和转动组件同时启动，动力组件带动两个密封板10移动，使密封板10移动并拉伸第一拉簧，直至密封板10的外侧面与连通壳7内侧面贴合后停止移动，两个密封板10和密封壳9将连通壳7上部腔室分为两个，在此过程中转动组件带动密封壳9转动，密封壳9带动密封板10转动调整连通壳7上部两个腔室的位置，直至密封板10为前后朝向后，连通壳7的上侧腔室分为左右两个腔室，使得右侧的两个液压伸缩杆5与连通壳7上部右侧的腔室连通，左侧的两个液压伸缩杆5与连通壳7上部左侧的腔室连通，随着飞行器向下移动，当右前侧的支撑架6与地面接触后，右前侧液压伸缩杆5的伸缩端被压缩，其内的液压油进入右后侧的液压伸缩杆5内，右后侧的液压伸缩杆5的伸缩端向下伸出并带动其上的支撑架6向下移动，直至右后侧的支撑架6与地面接触后，右侧的两个液压伸缩杆5的伸缩端均停止移动，此时驾驶员驾驶飞行器继续向下移动，使飞行器左部的两个支撑架6与地面接触，左侧的两个液压伸缩杆5重复上述过程进行调平，直至左侧两个液压伸缩杆5的伸缩端均停止移动后，飞行器的左右两侧分别完成调平，动力组件带动密封板10移动复位并解除对连通壳7上侧腔室的分隔，若此时飞行器水平，左右两侧液压伸缩杆5内的液压油不流动，若此时飞行器不水平，左右两侧液压伸缩杆5内的液压油通过连通壳7上侧的腔室互相流动完成对飞行器的最终调平，直至飞行器水平后，四个液压伸缩杆5内的液压油均停止流动，之后转动组件带动密封壳9及其上零件反向移动复位。
进一步的，如图3-图6所示，动力组件包括有：滑动杆11，滑动连接于密封壳9，密封壳9与滑动杆11之间设置有第二弹簧，滑动杆11与连通壳7滑动连接；挤压块12，固接于滑动杆11，密封板10与挤压块12挤压配合；第一电动推杆13，固接于底座1，第一电动推杆13的伸缩端固接有调节架14，调节架14与滑动杆11转动连接。

工作流程：在上述飞行器下降的过程中，以飞行器右侧向下偏斜，且地面后侧向上偏斜为例，在飞行器在下降的过程中，陀螺仪检测飞行器机身偏斜的方向，之后陀螺仪将检测的信号传递到右侧的两个电动滑轨401，使右侧的两个旋翼403向上移动，之后飞行器逐渐向下移动，并在四个检测板17与地面接触后，由于飞行器后侧的两个液压伸缩杆5处于高位，后侧的两个液压伸缩杆5的伸缩端逐渐收回，前侧两个液压伸缩杆5的伸缩端逐渐伸出，使得后侧的两个旋翼403均向下移动，后侧的两个储液管412带动其上的零件向下移动，当挤压杆411与限位架413接触后，挤压杆411向左挤压限位架413，使限位架413拉伸第三拉簧并解除对调节块414的限位，调节块414在第四拉簧的带动下向上移动，并将储液管412内的液压油送入调节伸缩杆405内，使调节伸缩杆405的伸缩端伸出并带动固定板406移动，固定板406对固定架407的挤压力减小，使得固定架407在第二扭簧作用下转动并解除对动力电机402的固定，之后液压油进入滑动管408内，由于右后侧的旋翼403已向上移动，该处的调节板410与升降架409分离，此时右后侧的调节板410仅向上移动，左后侧的调节板410向上移动并推动升降架409同步移动，使升降架409带动动力电机402相对于液压伸缩杆5向上移动，使左后侧的旋翼403上升并远离地面。
