Description

一种用于低空载人电动垂直起降飞行器的电机磁悬浮定位 装置
技术领域
本发明涉及一种悬浮定位装置，具体涉及一种用于低空载人电动垂直起降飞行器的电机磁悬浮定位装置。
背景技术
低空载人飞行器作为一种新兴的交通运输工具，其驱动系统的性能直接影响飞行器的可靠性、能效和安全性。目前，大多数桨叶驱动电机仍采用传统的机械轴承连接定子和转子，依靠物理接触传递动力。尽管该技术成熟度高，但存在诸多固有缺陷，难以满足现代飞行器对长航时、高机动性和低维护成本的要求。首先，机械轴承在运行过程中会产生显著的摩擦阻力，导致能量损耗，降低电机效率，进而影响飞行器的续航能力。同时，在频繁启停、快速变倾角等复杂工况下，轴承磨损加剧，使用寿命大幅缩短，需要定期维护甚至更换，增加了运营成本。更严重的是，轴承的突发性失效可能导致动力中断，威胁飞行安全，这对载人飞行器而言是不可忽视的风险。
为解决轴承技术的局限性，磁悬浮方案被视为替代方向。然而，现有磁悬浮技术仍存在明显不足。例如，部分方案仅实现径向悬浮，而缺乏轴向悬浮能力，导致飞行器在爬升或俯冲时转子因轴向推力偏移而与定子发生接触碰撞。另一些方案采用被动磁悬浮设计，磁场分布固定，无法根据飞行载荷或外部扰动实时调整悬浮力，在载重变化或遭遇气流冲击时容易引发转子失稳。此外，部分磁悬浮系统仍依赖辅助轴承作为保护措施，未能彻底消除机械接触，摩擦问题依然存在。更关键的是，现有磁悬浮系统大多不具备动态调节能力，当飞行器在垂直起降和水平巡航模式之间切换时，无法自适应调整悬浮参数，导致振动加剧或飞行姿态不稳定。
除了悬浮技术本身的缺陷，现有驱动电机的密封与防护设计也存在短板。传统密封方式（如橡胶密封圈或机械密封）虽然能有效阻隔外部灰尘和水分，但由于与转轴直接接触，会产生额外的摩擦损耗，降低系统效率。而非接触式密封（如迷宫密封）虽然减少了摩擦，但若间隙设计不合理，可能导致密封失效，在潮湿或多尘环境中无法提供可靠防护，进而引发电机内部短路或元件腐蚀，影响长期可靠性。
综上所述，当前低空载人飞行器驱动系统的核心问题可归纳为：1）机械接触导致的能耗高、寿命短及安全隐患；2）磁悬浮系统动态调节能力不足，难以适应复杂飞行工况；3）防护与密封设计无法兼顾低摩擦和高可靠性。
发明内容
本发明的目的在于提供一种用于低空载人电动垂直起降飞行器的电机磁悬浮定位装置，该装置采用轴向-径向双自由度主动磁悬浮结构实现无摩擦动力传递，集成智能动态调控系统实时优化悬浮力以适应复杂飞行工况，并配备高性能非接触式密封装置，从而显著提升低空载人飞行器驱动系统的能效比、可靠性和安全性。
本发明的目的在于提供一种电机磁悬浮定位装置，包括：
壳体，所述壳体起到保护支撑的作用；
双悬浮机构，包括定子轴、转子轴、定子、转子、电磁组件和磁性组件，所述定子轴固定于壳体内的，所述转子轴可旋转且同轴套设于定子轴外围，所述定子固定连接定子轴，转子固定连接转子轴并环绕定子设置，所述转子轴驱动转子绕定子轴轴线旋转，电磁组件与磁性组件，分别设置于所述电机悬浮定位装置的相对静止部与旋转部，通过电磁作用维持轴向悬浮稳定，以适应低空载人电动垂直起降飞行器垂直起降、水平巡航及变倾角飞行工况；
密封防水机构，所述密封防水机构固定在壳体上侧起到密封作用；
控制系统，所述控制系统用于控制电机磁悬浮定位装置的运行，包括：
传感器组，用于实时采集转子的姿态数据、加速度数据及转子与定子之间的间隙数据；
控制器，通过CAN总线与电机驱动模块建立通信连接，用于接收并处理所述传感器组采集的数据，通过控制算法生成控制指令，动态适配低空载人电动垂直起降飞行器飞行姿态变化；
电机驱动模块，与电流调节模块电性连接，用于驱动所述双悬浮机构产生悬浮力；
电流调节模块，与所述控制器通过信号连接，用于根据控制指令调节输出电流；
电源模块，由低空载人电动垂直起降飞行器主电池独立供电，为所述控制系统提供工作电能。
在一个实施例中，所述定子轴固定于所述壳体内部，其轴线与壳体中心轴线重合，定子固定套装于定子轴的外周面，且与壳体相对静止，所述转子呈圆筒状，嵌套于定子外侧，所述转子轴贯穿转子并与转子固定连接，转子轴可绕定子轴的轴线转动，当转子轴旋转时，带动转子同步转动，实现外转子结构的旋转功能，所述电磁组件安装在壳体上，所述磁性组件安装在双悬浮机构上，二者相互作用以维持低空载人电动垂直起降飞行器飞行时的悬浮稳定性。
在一个实施例中，所述双悬浮机构由径向悬浮机构和轴向悬浮机构构成，所述径向悬浮机构由转子轴周向均匀分布的磁性组件与壳体内侧周向分布的电磁组件构成，适应低空载人电动垂直起降飞行器水平巡航时的离心力载荷；所述轴向悬浮机构由安装在定子上下两端处的磁性组件和安装在壳体上下两端对应位置的电磁组件组成，动态抵消低空载人电动垂直起降飞行器垂直起降时的轴向推力。
在一个实施例中，所述转子轴周向嵌入永磁体环，壳体内侧周向分布电磁铁，通过闭环控制电磁铁电流调节径向磁力，维持转子在低空载人电动垂直起降飞行器变倾角飞行时的居中状态。
在一个实施例中，所述定子上下两端固定饼状永磁体，壳体上下两端对应安装电磁铁，通过永磁体与电磁铁的磁力耦合，动态抵消低空载人电动垂直起降飞行器垂直起降及姿态调整时的转子轴向位移。
在一个实施例中，所述电磁铁表面安装一层氮化硅陶瓷涂层，以适应低空载人电动垂直起降飞行器户外飞行的高温及摩擦防护需求。
在一个实施例中，所述密封防水机构为一种空气迷宫密封装置，所述空气迷宫密封装置通过多级环形凹槽与凸脊交替排列形成迂回气流通道，利用气体粘滞阻力阻挡低空载人电动垂直起降飞行器户外飞行时的外部介质侵入，同时避免摩擦损耗。
在一个实施例中，所述密封装置通过螺栓固定于壳体端盖，与转子间保持间隙，实现无接触密封。
在一个实施例中，所述控制器被配置为，根据所述传感器组采集的低空载人电动垂直起降飞行器飞行姿态数据，通过比例-积分-微分闭环控制算法生成控制指令，通过所述电流调节模块实时调节输出至电机驱动模块的电流，以动态维持转子在垂直起降、水平巡航以及变倾角工况下的悬浮稳定状态。
在一个实施例中，所述传感器组还包括：
霍尔传感器，设置于定子与转子的气隙区域内，用于实时检测转子永磁体的径向磁通密度分布，补偿低空载人电动垂直起降飞行器高速飞行时的气隙磁场畸变；
加速度计，固定于转子端部，用于采集转子在X/Y/Z三个轴向的加速度数据以监测飞行姿态，参与低空载人电动垂直起降飞行器变倾角飞行时的动态平衡校正；
温度传感器，嵌入定子内，用于实时监测电机工作温度，为电流调节模块提供低空载人电动垂直起降飞行器高负荷飞行时的热补偿控制依据。

实施例
在飞行器工作过程中，磁悬浮定位装置根据飞行姿态动态调节磁悬浮力以实现无接触稳定运行。起飞阶段桨叶处于竖直状态时，控制系统4通过传感器实时监测转子204姿态，向轴向悬浮机构22输出增强电流，使定子两端永磁体与壳体电磁铁产生高强度轴向磁耦合，完全抵消转子204重力与起飞加速度带来的轴向载荷。进入巡航阶段后，桨叶切换至水平姿态，径向悬浮机构21开始主导控制，壳体1内周向分布的电磁铁组根据离心力与气动载荷变化，通过闭环算法动态调整径向磁力分布，确保转子204在高速旋转中保持径向居中悬浮。当飞行器执行变倾角机动时，传感器组41同步采集多维加速度与气隙磁场数据，控制系统4融合飞行姿态参数与转子204动态平衡状态，对轴向与径向磁力进行动态权重分配，通过电流调节模块44实现两轴悬浮力的平滑过渡。降落阶段初期维持低功率悬浮状态，随着高度下降逐步减小电磁铁激励电流，利用定子永磁体与转子204间的预紧磁力实现软着陆，最终通过永磁体固有磁吸力使转子204稳定着陆于安全位置，全程避免机械接触与冲击损伤。
本发明具有如下有益效果：
1）采用磁悬浮设计消除机械摩擦，提升能效和可靠性；
2）智能调节系统实时适应飞行姿态变化，确保运行稳定；
3）创新密封结构在零摩擦前提下实现优异防护性能。
