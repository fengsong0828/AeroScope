Description

一种基于超大规模MIMO的低空飞行器轨迹检测与信道跟踪 方法
技术领域
本发明涉及一种基于超大规模MIMO的低空飞行器轨迹检测与信道跟踪方法，属于无线通信技术领域。
背景技术
第六代移动通信技术(6G)已在全球范围内展开广泛研究，作为下一代通信技术，它将提供更高的网络速度、连接密度和智能化能力，极大地促进数字经济的发展潜力。随着技术不断进步和应用场景的拓展，低空经济的开发逐渐成为推动下一代数字经济发展的重要方向之一。市场需求的增长、技术创新的突破以及产业链的完善，推动了低空经济的快速发展。低空飞行器(如无人机、电动垂直起降飞行器等)在物流配送、农业监测、城市空中出行等多个领域的应用，正在改变传统产业格局。与此同时，相关政策和法规的完善为低空经济的健康、可持续发展奠定了坚实基础。通过开发低空空域，低空经济正在成为推动经济增长、提升产业效率和促进技术创新的重要引擎。
低空经济的发展不仅推动了低空空域的开放和管理，也催生了对低空飞行通信需求的增长。特别是在6G网络的支撑下，低空飞行器的通信需求正在呈现爆发式增长。低空飞行器对高速、低延迟的通信需求极为迫切，作为6G的关键技术之一，超大规模MIMO技术的应用提供了丰富的场景。超大规模MIMO技术结合低空经济的应用，能够为低空飞行器提供更加高效的通信支持和定位应用。通过空分复用技术，超大规模天线阵列能够有效提升低空飞行器的通信和定位能力。然而，随着阵列规模的迅速增加，传统的全数字硬件架构面临着巨大的硬件成本压力，尤其是在超大规模MIMO系统中，收发机的硬件开销急剧上升，成为其技术普及和实际应用的重大障碍。同时，随着阵列规模的增大和近场效应的产生，使得用户定位和信道估计的复杂度随之增加，这进一步增大了超大规模MIMO在实际应用中的挑战。为了在保证用户定位和信道估计精度的前提下降低硬件成本和计算复杂度，混合波束成形架构成为一种潜在的解决方案。混合波束成形技术通过在射频和基带之间进行信号处理的分层设计，减少了射频链路的数量，降低了系统的硬件成本。同时，低空飞行器的动态特性使得超大规模MIMO系统面临时变信道的挑战，需结合深度学习、图像处理等先进技术，以降低时变信道的计算复杂度，并提高频谱效率。
6G超大规模MIMO技术和低空经济的结合，将为未来通信系统带来巨大的创新潜力。通过技术创新和合理的架构设计，有望克服现有的硬件成本和计算复杂度瓶颈，推动超大规模MIMO技术在实际应用中的广泛部署，为6G时代的低空经济提供强大的通信保障。因此，针对超大规模MIMO在低空经济应用中的需求，研究低成本硬件架构、混合波束成形算法以及低复杂度的飞行器轨迹及信道跟踪方案显得尤为重要。这些技术将有助于提升低空飞行器的通信和定位质量，推动低空经济的快速发展，并进一步推动6G技术在低空经济中的应用，为未来的通信需求提供支持。
发明内容
本发明所要解决的技术问题是：提供一种基于超大规模MIMO的低空飞行器轨迹检测与信道跟踪方法，设计能够恢复特征明显的信道图像的波束成形矩阵，并从基站接收信号中恢复信道图像，设计基于图像的关键点检测网络，获取飞行器的位置，设计信道外推方案，并基于所有时刻位置信息设计飞行器的轨迹跟踪，实现低成本、低复杂度的飞行器识别与轨迹跟踪。
本发明为解决上述技术问题采用以下技术方案：
一种基于超大规模MIMO的低空飞行器轨迹检测与信道跟踪方法，处于预设观测区域内的超大规模MIMO系统包括一个基站、一个已认证用户飞行器以及S个散射体飞行器，S个散射体飞行器包括已认证飞行器和未认证飞行器；基站配置M根均匀线阵天线，并采用混合波束成形架构；基站天线分为Q个子阵列，每个子阵列天线数目为M/Q，导频个数为M/2Q；
所述轨迹检测与信道跟踪方法包括如下步骤：
步骤1，用户飞行器向基站发送导频信号，基站在混合波束成形架构下，设计波束成形矩阵，从基站接收信号中恢复信道图像；
步骤2，利用关键点检测网络对从基站接收信号中恢复的信道图像进行网络推理，得到用户飞行器和所有散射体飞行器的粗估计位置；
步骤3，对于第一时刻的信道估计，利用正交匹配追踪优化对步骤2得到的粗估计位置进行优化，得到估计位置；对基站接收信号使用最小二乘算法得到路径增益，实现信道估计；
步骤4，对于后续时刻的信道跟踪，根据接收信号判断当前时刻是否有路径漏检，若有则根据步骤3第一时刻的信道估计方法对当前时刻进行信道估计，否则根据上一时刻估计出的飞行器位置及飞行器的可移动范围使用正交匹配追踪优化进行信道外推，实现信道跟踪；
步骤5，基于所有时刻估计出的各飞行器位置，使用匈牙利算法对不同时刻的位置进行匹配和关联，得到各飞行器的移动轨迹；
步骤6，基站基于各飞行器的移动轨迹，与已认证用户飞行器和已认证飞行器的路径进行比较，实现对各飞行器的轨迹纠正。
本发明采用以上技术方案与现有技术相比，具有以下技术效果：
1、本发明通过设计混合波束成形矩阵，能够以较低硬件成本和导频开销适配低复杂度的基于图像关键点检测的飞行器位置估计及信道估计。
2、本发明通过使用高精度的图像关键点目标检测，能够一次性估计出所有飞行器的位置，在实现超越同尺寸码本下的正交匹配方案达到的信道估计精度的前提下，有效降低方法的复杂度。
3、本发明通过使用匈牙利算法对不同时刻的估计位置进行匹配，有效完成对飞行器的轨迹追踪。
4、本发明通过设计基于估计轨迹的飞行器监管算法，能够有效、精准实现对低空飞行器的轨迹检错、信号中断检测以及入侵检测。

步骤二，飞行器位置估计阶段：使用如图2所示关键点检测网络，在预先标注好的训练集上进行训练，测试时输入从基站接收信号中恢复的信道图像，得到飞行器的位置粗估计。
网络首先通过骨架网络ResNet-50提取图像通用特征，然后经由三个相同的由一个3*3大小、64通道的卷积核，一个归一化层，一个ReLU激活层，和一个1*1大小、2通道的卷积核组成的检测头后，分别输出三个同样长宽的特征图，代表热力图、坐标预测偏移特征图、和目标大小特征图。热力图上的每个点的值代表目标中心点落在该点的概率，坐标预测偏移特征图代表对应热力图上的每个点的坐标偏移预测值，目标大小特征图代表对应热力图上的每个点的目标大小预测值。在做推理时，网络输出还会经过一个分数阈值过滤器，即删除热力图预测概率较小的关键点，从而给出最后的关键点预测值，即经过一次网络推理得到所有用户飞行器和散射体飞行器的位置粗估计。
步骤三，第一时刻信道估计阶段：基于步骤二得到的飞行器位置信息设计每条路径附近小范围、细粒度码本的正交匹配追踪优化，得到精确的位置信息使用最小二乘算法得到路径增益实现信道估计。
步骤四，后续时刻信道跟踪阶段：基于前一时刻估计出的位置信息根据接收信号判断是否有路径漏检，如果残差信号满足下式，则重复步骤三进行这一时刻的信道估计：
∥(a(x,z))Hyres(t)∥2>＝σ2(ln(M)-ln(-ln(1-Pfa)))
其中，Pfa为虚警概率。否则，只需要在上一时刻检测出的飞行器位置以及飞行器的可移动范围内使用正交匹配追踪进行信道外推即可完成信道跟踪。
步骤五，飞行器检测位置关联与轨迹匹配阶段：基于所有时刻估计出的飞行器位置信息，使用匈牙利算法对不同时刻的检测出的位置进行匹配和关联，将前后两时刻估计出的所有位置视为两个二分图，使用匈牙利算法实现代价为1-IOU的二分图最大匹配，IOU即检测框的交并比。由此得到飞行器位置估计在不同时刻的最优匹配，并形成移动轨迹。
步骤六，飞行器监管阶段：基站端基于估计出的用户飞行器和散射体飞行器的位置，与基站端已认证飞行器路径集合进行比较，对于已认证飞行器，若某一时刻未检测到其位置，则判断该飞行器在该时刻信号中断，并通过估计出的轨迹信息判断该飞行器的真实位置，与预设轨迹对比，实现轨迹检错；对于未认证飞行器，若连续某些时刻检测到位置信息均与基站已认证飞行器所有预设轨迹偏差较大，则判断该飞行器为入侵飞行器。
