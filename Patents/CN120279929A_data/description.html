Description

一种应用于低空飞行器的轻量化语音增强方法及其模型
技术领域
本发明属于语音增强技术领域，具体涉及一种应用于低空飞行器的轻量化语音增强方法及其模型。
背景技术
现有的语音增强技术主要基于深度神经网络，多尺度卷积降噪方法混合神经网络结构，这些方法在PC端或服务器环境下虽能实现较高的语音质量，但在低空飞行器应用场景中存在局限性。具体而言，现有模型的参数量和计算量过大，远超机载嵌入式处理器的实时处理能力，导致部署困难；同时，现有方案，如基于WFST的实时语音增强算法，在嵌入式平台上的处理延迟通常超过80ms，无法满足航空通信对低延迟(<20ms)的严格要求；此外，传统语音增强方法虽对稳态噪声有效，但对低空飞行器特有的非平稳噪声，如旋翼谐波、气流脉冲噪声的适应性不足，在信噪比剧烈波动的环境下性能显著下降；
因此，现有技术难以在低空飞行器的高噪声、低算力及严苛实时性要求条件下实现高效可靠的语音增强，亟需一种轻量化、低延迟且噪声鲁棒性强的优化方案。
发明内容
本发明的目的在于提供一种应用于低空飞行器的轻量化语音增强方法及其模型，旨在解决现有语音增强技术在机载嵌入式环境下的高计算复杂度、高延迟及噪声鲁棒性不足等问题。
本发明采用的技术方案如下：一种应用于低空飞行器的轻量化语音增强方法，包括如下步骤：
S1，通过麦克风采集低空飞行器环境下的带噪语音信号，带噪语音信号来源于低空飞行器飞行时与地面语音声源距离在100米范围内所产生的语音,或在低空飞行器飞行过程中座舱内进行语音交流时所产生的语音；
S2，将采集到的带噪语音信号输入至轻量化语音增强模型，利用预处理模块对带噪语音信号进行预处理，得到时频特征；
S3，将时频特征输入坐标注意力模块，提取空间分布特征,并进行自适应加权，得到加权语音信号；
S4，将加权语音信号输入时间卷积模块，提取瞬态特征与稳态特征，输出优化特征信号；
S5，将优化时间特征信号通过小规模可分离卷积模块处理，得到增强语音特征；
S6，将增强语音特征输入至掩码计算模块，对增强语音特征进行掩码计算，生成增强语音频谱，并通过逆短时傅里叶变换还原为时域语音信号。
