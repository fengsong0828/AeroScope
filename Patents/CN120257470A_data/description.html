Description

面向低空典型场景的飞行器驾驶舱模拟方法及设备
技术领域
本发明涉及飞行模拟技术领域，具体地说，涉及一种面向低空典型场景的飞行器驾驶舱模拟方法、系统、设备及存储介质。
背景技术
随着低空经济快速发展(无人机物流、城市空中交通、应急救援等)，飞行员在复杂低空场景(建筑密集区、山地峡谷、恶劣气象)下的态势感知与应急处置能力面临更高要求。传统飞行模拟系统多基于固定场景与预设脚本，难以动态重构低空障碍物分布、实时气象扰动等要素，导致训练环境与真实工况存在显著差异。现有技术中，数字孪生技术虽已应用于飞行仿真，但在虚实交互实时性(延迟＞50ms)、多模态人机协同(手势识别率＜80％)及自适应控制(静态参数无法适配个体差异)方面存在瓶颈。此外，低空飞行特有的突发风险(如飞鸟群撞击、微型无人机避让)对模拟系统的环境感知精度(现有激光雷达误差≥5cm)与决策响应速度(避障延迟≥1s)提出更高挑战。
发明内容
本发明的目的在于提供一种面向低空典型场景的飞行器驾驶舱模拟方法、系统、设备及存储介质，旨在解决目前面临的低空飞行技能训练的技术壁垒，融合数字孪生、混合现实(MR)、多模态传感与自适应控制技术，进行飞行员低空飞行技能训练、人机工效评估及智能航空器控制系统验证，能够扩展至城市空中交通(UAM)操作员培训与无人驾驶航空器仿真测试。
本发明公开了一种面向低空典型场景的飞行器驾驶舱模拟方法，包括：
收集低空场景多源数据，构建数字孪生混合现实的飞行器仿真模型；
建立飞行器仿真模型和飞行员之间的人机在环多模交互；
获取人机在环多模交互过程中产生的人机交互数据，输入飞行器智能优化控制模型，通过模型输出结果进行飞行器的智能优化控制。
其中，收集低空场景多源数据，构建数字孪生混合现实的飞行器仿真模型，包括：
收集低空场景多源数据，建立飞行器动力学模拟器和基于不同飞行器机型的定制化座舱；
为飞行器动力学模拟器安装多模态人机交互传感器阵列；
为飞行器动力学模拟器提供飞行辅助和飞行参数优化；
为飞行器动力学模拟器构建虚实融合可视化显示界面。
其中，收集低空场景多源数据，建立飞行器动力学模拟器和基于不同飞行器机型的定制化座舱，包括：

定义飞行员操作指令与飞行控制逻辑的映射关系；
采用Unreal Engine 5Nanite虚拟化微多边形几何技术，对飞行器仿真模型进行动态全局光照(Lumen系统)和实时流体模拟(Chaos物理引擎)，完成虚拟仿真引擎的渲染；
构建时空多模态数据库，用于存储历史飞行轨迹、传感器标定参数及环境扰动模式。
其中，为飞行器动力学模拟器安装多模态人机交互传感器阵列，包括：

通过CAN总线，采集飞行器动力学模拟器的液压执行器压力值；
通过内嵌眼动追踪单元的增强现实头显设备，记录飞行员注意力分布热点。
其中，为飞行器动力学模拟器提供飞行辅助和飞行参数优化，包括：
通过机器视觉耦合UWB超宽带定位技术(精度±10cm)实现飞行员头部与手部实时追踪；
结合AI视觉行为特征提取算法动态修正MR场景空间坐标偏移；
基于数字孪生模型的仿真结果，采用强化学习算法生成飞行参数优化方案。
其中，建立飞行器仿真模型和飞行员之间的人机在环多模交互，包括：
多模态输入，包括手势识别、眼动控制和语音交互；
触觉反馈，包括振动反馈和力反馈；
多源数据融合，采用时间戳对齐算法同步传感器数据，通过注意力机制加权融合手势、眼动和语音指令优先级。
其中，获取人机在环多模交互过程中产生的人机交互数据，输入飞行器智能优化控制模型，通过模型输出结果进行飞行器的智能优化控制，包括：
构建基于LSTM+卡尔曼滤波的新型网络架构预测飞行员操作意图，当检测到持续航向偏离时，自动补偿舵面偏转量；
开发数字孪生对比引擎，计算实际与虚拟飞行轨迹的Hausdorff距离，超限时触发触觉告警；
预设故障模式，并设计故障恢复策略；
基于Q-learning算法动态调整训练难度，采用模型剪枝技术压缩神经网络参数量。
本发明公开了一种面向低空典型场景的飞行器驾驶舱模拟系统，包括：
数字孪生混合现实模块，用于收集低空场景多源数据，构建数字孪生混合现实的飞行器仿真模型；
人机在环多模交互模块，用于建立飞行器仿真模型和飞行员之间的人机在环多模交互；
飞行智能优化控制模块，用于获取人机在环多模交互过程中产生的人机交互数据，输入飞行器智能优化控制模型，通过模型输出结果进行飞行器的智能优化控制。
本发明公开了一种计算机设备，包括输入输出单元、存储器和处理器，存储器中存储有计算机可读指令，计算机可读指令被处理器执行时，使得处理器执行如前述实施方式方法中的步骤。
本发明公开了一种存储有计算机可读指令的存储介质，计算机可读指令被一个或多个处理器执行时，使得一个或多个处理器执行如前述实施方式方法中的步骤。
区别于现有技术，本发明的一种面向低空典型场景的飞行器驾驶舱模拟方法，通过融合混合现实(MR)与高精度数字孪生模型，构建动态可扩展的低空场景库；结合人机在环多模交互技术(眼动-手势-语音协同)，提升操作自然度；引入强化学习驱动的智能优化算法，实现飞行控制参数的自适应调整，最终形成“感知-决策-评估”闭环训练体系。本发明融合数字孪生、混合现实(MR)、多模态传感与自适应控制技术，进行飞行员低空飞行技能训练、人机工效评估及智能航空器控制系统验证，能够扩展至城市空中交通(UAM)操作员培训与无人驾驶航空器仿真测试，解决目前面临的低空飞行技能训练的技术壁垒，填补低空动态场景模拟的技术空白。
