Description

一种低空无人飞行器的补能路径规划方法及系统
技术领域
本申请涉及飞行路径规划技术领域，具体涉及一种低空无人飞行器的补能路径规划方法及系统。
背景技术
无人飞行器续航能力有限，补能点的合理布局与高效路径规划成为保障其长时稳定运行的关键，然而，低空飞行环境具有高度动态性，气象条件变化、地理障碍物分布以及空域管制政策的实时调整，对无人飞行器补能路径规划提出了更高要求。
当前低空无人飞行器补能路径规划方法多依赖静态地图与预设规则，难以适应气象突变(如突发强风、暴雨)、空域临时管制等动态场景，导致规划路径与实际环境脱节，飞行器被迫中断任务或面临续航不足风险；此外，多数路径规划仅以最短路径或最低能耗为单一目标，忽略空域拥堵、飞行安全等多维度约束，在高密度飞行区域易引发路径冲突与安全隐患，难以满足低空无人飞行器高效、安全运行的需求。
发明内容
本申请通过提供了一种低空无人飞行器的补能路径规划方法及系统，旨在解决现有补能路径规划使用静态地图与预设规则，难以实时响应气象突变、障碍物动态变化及空域临时管制，存在飞行器被迫中断任务或面临续航不足风险的技术问题。

优选的，预测各个补能点的补能排队时间；删除所述补能排队时间超出预设等待时长的补能点，所述补能点候选集合的中的每一条补能路径包含至少两个满足预设等待时长的补能点。

优选的，获取历史飞行数据，结合所述空域管制信息，获取所述适配补能路径的预测拥堵概率；根据所述预测拥堵概率，判断是否触发备选路径生成机制。

实施例一，如图1所示，本申请提供了一种低空无人飞行器的补能路径规划方法，所述方法包括：
获取低空飞行区域的实时监测数据，所述实时监测数据包括气象信息、地理障碍物分布信息及空域管制信息；根据所述实时监测数据，确定补能点候选集合。
具体而言，低空飞行区域是指距离地面一定高度(一般在1000米以下)的空域范围，这个区域的飞行环境复杂多变。实时监测数据是通过各种传感器、监测设备以及通信系统等获取的能够反映当前低空飞行区域状态的信息集合，其中，气象信息包括风速、风向、气温、气压、降雨量、能见度等气象参数，这些参数会实时影响无人飞行器的飞行安全性和能耗；地理障碍物分布信息涉及低空区域内的建筑物、山脉、通信塔、桥梁等固定或临时障碍物的位置、高度和形状等数据，对于规划飞行路径以避免碰撞至关重要；空域管制信息涵盖空域的开放状态、临时飞行限制、禁飞区划定、空中交通流量控制指令等内容，能指导无人飞行器合法、有序地飞行；补能点候选集合则是从众多可能的补能点中，根据实时监测数据筛选出的符合一定初步条件的补能点的集合，这些补能点有可能为无人飞行器提供能量补充服务。
获取低空飞行区域的实时监测数据是整个补能路径规划的基础环节。要想准确规划补能路径，就必须全面、精准地掌握飞行环境的状态；通过在低空飞行区域内布置气象监测站、利用卫星遥感技术获取地理障碍物信息，以及与空中交通管理部门的管制信息系统进行实时数据交互，能够在每分钟甚至更短的时间间隔内获取海量的实时监测数据，例如，气象监测站可以在每1分钟-5分钟更新一次气象信息，包括风速精确到0.1米/秒、气温精确到0.1摄氏度等详细数据；地理障碍物分布信息可通过定期的地形测绘和实时的建筑施工申报系统等渠道更新，确保飞行器知晓最新的障碍物位置和高度变化；空域管制信息则实时更新空域的开放和限制状态，为后续确定补能点候选集合提供了准确的环境依据。
根据这些实时监测数据确定补能点候选集合，是为初步筛选出在当前环境条件下有适合无人飞行器补能的地点，只有这些在环境适配度上初步符合要求的补能点才能被纳入最终的补能路径规划中，从而保证补能路径规划的可行性和安全性，例如，在一个10平方公里的低空飞行区域内，原本有20个补能点，但通过考虑气象条件(如风速过大导致某些补能点周边乱流严重)、地理障碍物(某些补能点附近新出现了超高建筑)以及空域管制(部分补能点所在空域因临时活动被管制)等因素后，筛选出8个补能点进入候选集合，为下一阶段的路径生成和择优提供关键的基础数据支持。
引入低空无人飞行器的位置点云、剩余电量，结合所述补能点候选集合，确定多条补能路径；对所述多条补能路径进行适配择优，确定适配补能路径，以所述适配补能路径对所述低空无人飞行器进行补能导航规划。
具体而言，低空无人飞行器的位置点云是指通过多个传感器(如激光雷达、GPS等)获取的飞行器在低空环境中的三维位置数据点集合，这些数据点能够精确描述飞行器在空间中的位置状态；剩余电量是飞行器当前电池所剩余的电量数值，直接关系到飞行器能否安全抵达补能点；补能路径是指从飞行器当前位置到补能点的一条完整航线，包括飞行的起点、终点以及中间可能涉及的航路点等信息，每条补能路径都有其独特的飞行距离、时间、能耗等特征。
适配择优是一个决策过程，旨在根据飞行器的状态(如剩余电量、飞行速度等)和补能路径的特性(如路径长度、预计飞行时间、能耗估算等)，结合补能点的服务能力(如充电功率、可用接口数量等)，挑选出最适合飞行器当前状况的补能路径。补能导航规划则是基于确定的适配补能路径，为飞行器生成详细的导航指令，引导其沿着规划路径安全、高效地抵达补能点，这涉及到路径指引、飞行姿态调整、速度控制等多个方面的指令生成。
在引入低空无人飞行器的位置点云和剩余电量后，结合补能点候选集合确定多条补能路径的过程，具体来说，位置点云提供了飞行器精确的三维位置信息，剩余电量则明确了飞行器紧迫的续航需求。以位置点云为起点，以补能点候选集合中的各个补能点为终点，利用路径规划算法(如A*算法、Dijkstra算法等)，在考虑低空飞行环境中的各种约束(如障碍物、空域限制等)的情况下，生成多条可行的补能路径。
对多条补能路径进行适配择优，确定适配补能路径并进行补能导航规划的过程，是整个补能路径规划方法的关键决策环节，在此过程中，需要综合考虑多种因素，例如，从剩余电量角度看，若飞行器剩余电量较低，那么距离更短、预计能耗更低的补能路径可能会被优先选择；从飞行安全角度考虑，避开空域管制区域、障碍物密集区域的补能路径更具优势；从补能效率角度出发，优选补能点充电功率高、可用接口数量多的路径；通过构建适配择优模型，对每条补能路径进行量化评估，如赋予不同的权重系数(如距离权重0.4、安全权重0.3、补能效率权重0.3)，评估每条路径的综合适配度分数，按照分数由高到低排序，挑选出适配补能路径；根据确定的适配补能路径，生成补能导航规划，包括飞行方向、速度、高度等指令，引导飞行器精准抵达补能点，确保其能够及时补能，继续完成飞行任务，有效提升飞行器在复杂动态低空环境下的运行效率和安全性。
进一步，本申请提供了根据所述实时监测数据，确定补能点候选集合，所述方法包括：
评估各个补能点的环境适配度；删除所述环境适配度低于预设适配度阈值的补能点，确定所述补能点候选集合。
具体而言，环境适配度用于衡量补能点在当前低空飞行环境下的适用程度，考虑了多种环境因素，如气象条件(风速、降雨等)对补能操作安全性的影响，地理障碍物(如建筑物、山丘等)与补能点的距离和相对位置关系，以及空域管制对该补能点所在区域的限制程度等；预设适配度阈值是根据无人飞行器的运行安全标准、补能效率要求等预先设定的一个基准值，筛选出在当前环境下足够适合进行补能操作的补能点，排除那些环境条件过于恶劣或存在较大安全隐患的补能点。
评估各个补能点的环境适配度是补能路径规划中关键的筛选环节，在低空飞行环境中，气象条件的变化可能对飞行器的飞行安全和补能操作产生重大影响，例如，当风速超过一定阈值时，飞行器在靠近补能点的过程中出现飞行不稳定的情况，增加碰撞风险；降雨可能会干扰通信设备或影响补能设备的正常运行；地理障碍物的存在也会影响飞行器能否安全抵达补能点，如果补能点附近有较高的建筑物或山脉，飞行器需要绕行或调整飞行高度，会增加能耗和飞行时间；空域管制则直接决定了飞行器能否合法进入补能点所在区域；通过综合考虑这些因素，对每个补能点进行环境适配度评估，可以量化每个补能点的适用性。
删除环境适配度低于预设适配度阈值的补能点，确定补能点候选集合的过程，能够有效缩小补能点的选择范围，提高后续路径规划的效率和安全性；将环境适配度低于预设适配度阈值的补能点排除后，剩下的补能点组成补能点候选集合。这些补能点在环境条件上相对更适合进行补能操作，能够为后续结合飞行器位置和剩余电量确定多条补能路径提供更优质的基础数据，从而提高整个补能路径规划方案的可行性和有效性，保障无人飞行器在低空复杂环境下的补能安全和任务连续性。
进一步，本申请所述方法包括：
预测各个补能点的补能排队时间；删除所述补能排队时间超出预设等待时长的补能点，所述补能点候选集合的中的每一条补能路径包含至少两个满足预设等待时长的补能点。
具体而言，补能排队时间是指补能点在同一时间可服务的飞行器数量有限，其他飞行器需要等待补能完成的时间，例如一个补能点有3个充电接口，当有5架飞行器需要补能时，后面2架就需要排队等待前面的飞行器补能完成，这个时间受补能点的服务效率(如充电功率、接口数量等)、当前排队飞行器数量以及每架飞行器的补能需求量等因素影响；预设等待时长是根据无人飞行器的任务紧急程度、续航能力等预先设定的一个最大可接受等待时间，其目的是筛选出能够满足飞行器任务时间要求的补能点，避免飞行器在补能点长时间等待，耽误任务执行。
预测各个补能点的补能排队时间是通过分析补能点的服务能力和当前的使用状况来估算的，补能点的服务能力包括其充电功率、可用充电接口数量等，若补能点的充电功率为10kW，补能点有5个接口，每个接口可同时为1架飞行器充电，且当前有6架飞行器在排队，每架飞行器平均需要补能20kWh，那么每架飞行器的补能时间约为2小时(20kWh/10kW)，补能排队时间则根据排队顺序依次累加，例如，第1到5个飞行器无需排队等待，第6个飞行器需要等待约2小时，以此类推，同时，结合历史数据和实时飞行计划，也能更准确地预测补能排队时间，比如通过分析该补能点在相同时间段的历史排队情况，以及当前已知的飞行器补能计划。
删除补能排队时间超出预设等待时长的补能点后，补能点候选集合中的每一条补能路径包含至少两个满足预设等待时长的补能点，这个过程能够确保飞行器在补能过程中有多个可靠的选择，避免因排队时间过长而耽误任务，优选的，在将这些补能点排除后，剩下的补能点组成的候选集合中的每条补能路径都包含至少两个满足等待时长要求的补能点，例如，一条补能路径包含补能点A(排队时间0.5小时)和补能点B(排队时间0.3小时)，另一条补能路径包含补能点C(排队时间0.4小时)和补能点D(排队时间0.2小时)，这样在后续的路径规划中，飞行器可以根据实际情况选择合适的补能点进行补能，既能满足续航需求，又能保证任务的高效执行，提高整个补能路径规划的实用性和可靠性。
进一步，本申请所述方法还包括：
根据所述低空无人飞行器的能耗参数，获取所述多条补能路径对应的路径能耗信息；通过所述多条补能路径对应的路径能耗信息，配置第一适配择优约束条件。
具体而言，能耗参数是指低空无人飞行器在飞行过程中与能量消耗相关的参数，包括飞行器的重量、飞行速度、动力系统的能效、不同飞行姿态(如爬升、平飞、盘旋等)下的能耗系数等，反映飞行器在不同飞行状态下的能耗特性；路径能耗信息是指针对每条补能路径，结合能耗参数和路径特征(如距离、飞行高度变化、预计飞行时间等)计算出的预计能量消耗数据，是评估补能路径是否适用于当前飞行器续航状态的重要依据；适配择优约束条件是指为了筛选出最适合飞行器的补能路径而设定的限制准则，基于路径能耗信息制定的第一适配择优约束条件，旨在确保飞行器在选择补能路径时，所选路径的能耗能够满足飞行器的剩余电量和任务续航要求。
获取多条补能路径对应的路径能耗信息的过程，需要综合考虑飞行器的能耗参数和补能路径的特征。以飞行器的重量为例，假设飞行器重量为5千克，其在平飞状态下每公里能耗为0.1度电，爬升状态下每百米高度增加能耗额外增加0.05度电，对于一条包含10公里平飞段和爬升200米的补能路径，其路径能耗可以通过得出：平飞能耗为10公里×0.1度电/公里＝1度电，爬升能耗为200米/100米×0.05度电＝0.1度电，总路径能耗为1.1度电，通过这种方式，可以评估出每条补能路径的能耗信息。
将路径能耗信息用于配置第一适配择优约束条件，能够确保飞行器选择的补能路径在能耗上是可行的，那么第一适配择优约束条件可以设定为：路径能耗≤剩余电量-备用电量，在对多条补能路径进行筛选时，只有路径能耗满足这个条件的补能路径才会被保留下来，进入后续的择优环节，提前过滤掉那些能耗超出飞行器承受范围的补能路径，避免飞行器因能量不足而无法完成补能过程或返航，提高补能路径规划的合理性和安全性。
进一步，本申请所述方法还包括：
上传各个补能点包括补能设备的充电功率、可用充电接口数量的服务能力信息；通过所述各个补能点的服务能力信息，配置第二适配择优约束条件。
具体而言，上传指的是将各个补能点的服务能力信息通过通信网络(如无线通信网络等)发送至路径规划系统，以便系统能够获取并利用这些信息进行后续的分析和决策；补能点的服务能力信息包括补能设备的充电功率和可用充电接口数量等，充电功率代表了补能设备为飞行器充电的速度；可用充电接口数量则反映了补能点能够同时为多少架飞行器提供补能服务，如一个补能点有5个可用充电接口，就可同时服务5架飞行器；第二适配择优约束条件是基于补能点的服务能力信息制定的，其目的是确保补能点能够高效地为飞行器提供补能服务。
上传各个补能点的服务能力信息是补能路径规划中的关键环节。通过通信网络，将各补能点的充电功率和可用充电接口数量等信息实时上传至路径规划系统，例如，补能点A上报其充电功率为10kW，可用充电接口数量为3个；补能点B上报的充电功率是8kW，可用充电接口数量为5个，系统收集这些信息后，可以了解到每个补能点的补能能力和服务容量。
基于这些服务能力信息配置第二适配择优约束条件，可以确保飞行器选择的补能路径中的补能点能够满足其补能需求，例如，假设飞行器的剩余电量需要在1小时内补充8kWh的能量，那么第二适配择优约束条件可以设定为补能点的充电功率≥8kW(8kWh/1h)，并且可用充电接口数量≥1(确保有可用接口)，这样，在对多条补能路径进行筛选时，只有那些满足第二适配择优约束条件的补能路径才会被保留，由此，避免飞行器抵达补能点后因充电接口不足或充电速度过慢而耽误任务，有效提升补能路径规划的可靠性和效率，确保飞行器能够及时、高效地完成补能，继续执行任务。
进一步，本申请提供了对所述多条补能路径进行适配择优，确定适配补能路径，所述方法包括：
基于所述第一适配择优约束条件、第二适配择优约束条件，设置能耗权重系数和服务能力权重系数；根据所述能耗权重系数和服务能力权重系数组合加权，得到多条补能路径的适配度；根据所述多条补能路径的适配度由高到低进行排序，确定适配补能路径。
具体而言，能耗权重系数是用来衡量能耗因素在补能路径适配度评估中所占重要程度的比例系数，体现了飞行器任务对能耗的关注度；服务能力权重系数则衡量补能点服务能力因素在适配度评估中的重要程度，反映任务对补能效率和可靠性的要求；适配度融合了能耗和服务能力两方面因素，用来量化衡量每条补能路径的适用程度，适配度越高说明该路径越符合飞行器的补能需求；组合加权是指通过将各个评估因素(这里指能耗和服务能力)分别乘以对应的权重系数后再相加，得到一个综合的评估结果，即补能路径的适配度。
基于第一适配择优约束条件(能耗约束)和第二适配择优约束条件(服务能力约束)，设置能耗权重系数和服务能力权重系数的过程，类似于为两个关键因素分配重要性比重，例如，在紧急任务中，快速补能至关重要，此时可将服务能力权重系数设为0.6，能耗权重系数设为0.4，因为快速补能能更及时地为飞行器提供能量支持，保障任务连续性，如某补能路径在能耗评估中得分为80分(满分100)，在服务能力评估中得分为90分，那么其适配度计算为：适配度＝(0.4×80)+(0.6×90)＝86分；通过这种方式，综合考虑能耗和服务能力两个方面，为每条补能路径计算出一个反映其整体适用性的分数。
根据适配度对多条补能路径进行排序并确定适配补能路径的过程，能为飞行器选择出最合适的补能路径；按照适配度由高到低排序后，分数最高的路径很可能具有较低的能耗且补能点的服务能力强，这样的路径能使飞行器高效、快速地完成补能，更好地保障任务执行效率和飞行安全。
进一步，本申请所述方法包括：
获取历史飞行数据，结合所述空域管制信息，获取所述适配补能路径的预测拥堵概率；根据所述预测拥堵概率，判断是否触发备选路径生成机制。
具体而言，历史飞行数据是指过去飞行器在低空区域飞行时记录的各类信息，包括飞行时间、飞行轨迹、途经补能点、遇到的空域管制情况、飞行速度、能耗等数据。这些数据能够反映飞行器在不同空域和环境下的飞行状态和规律；预测拥堵概率是指根据历史飞行数据和空域管制信息，对适配补能路径上可能出现的拥堵情况进行概率估算，拥堵一般由多种因素引发，如空域管制导致的飞行器聚集、特定时段补能需求激增等；备选路径生成机制是一种在主路径可能出现问题(如拥堵概率较高)时，及时生成备用补能路径的策略或方法，旨在保障飞行器能够顺利、高效地完成补能任务。
获取适配补能路径的预测拥堵概率的过程，需要综合分析历史飞行数据和空域管制信息。通过对历史飞行数据的挖掘，可以了解在不同时间段、不同空域区域以及不同空域管制政策下，补能路径上发生拥堵的频率和严重程度，例如，历史数据显示在工作日上午9点至11点，某个低空区域的补能路径因周边物流飞行器集中补能，拥堵发生概率上升；空域管制信息则能实时反映当前空域的限制情况，如临时禁飞、限流等，这些管制措施可能改变飞行器的飞行路线，导致某些补能路径的拥堵概率上升。结合这些信息，利用统计模型或机器学习算法，可以较为准确地预测出适配补能路径在当前情况下的拥堵概率。
根据预测拥堵概率判断是否触发备选路径生成机制，能够有效提高补能路径规划的可靠性和灵活性，例如，设定拥堵概率阈值为30％，当预测拥堵概率超过这个阈值时，自动触发备选路径生成机制；在有多条补能路径的情况下，系统会基于实时数据和算法，快速生成新的备选路径，这些备选路径可能绕开了拥堵区域，或者利用了其他空域资源，这样，飞行器就能及时调整补能计划，选择更畅通的路径前往补能点，避免因拥堵导致的补能延误，确保飞行任务的连续性和高效性。
