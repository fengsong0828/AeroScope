CN107097965A

Description

一种低空飞行拍摄微型飞行器
技术领域
本发明涉及自动拍摄领域，具体说，属于一种航拍设备。
背景技术
现有的空中飞行拍摄设置过于复杂，且成本高，不容易随时实施。
发明内容
本发明的目的是提供一种较高清晰度的低空飞行拍摄微型飞行器，为了实现上述
目的，本发明采用如下技术方案：
1、本发明属于一种低空飞行拍摄微型飞行器，其特征为，还包括低空飞行拍摄微型飞行器悬索，低空自拍
传感器悬索的长度在1000厘米到50000厘米之间，其上端被固定在低空自拍设备上，下端悬挂
低空飞行拍摄微型飞行器，其可被卷绕在母机内，也可被下垂1000厘米到50000厘米的空中。
2、按照权利要求1的低空飞行拍摄微型飞行器，其特征，低空飞行拍摄微型飞行器悬索属于单根绳索。
3、按照权利要求1或2的低空飞行拍摄微型飞行器，其特征，低空飞行拍摄微型飞行器悬索属于细索。
4、按照权利要求1的低空飞行拍摄微型飞行器，其特征，低空飞行拍摄微型飞行器悬索属于两根电缆。
5、按照权利要求1的低空飞行拍摄微型飞行器，其特征，低空飞行拍摄微型飞行器悬索属于光缆。
从上可知，本发明具有拍摄清晰度较高的优点。
具体实施方式
本发明公开一种实施例，本实施例主要由一根细索和信号处理发射设备构成。
本实施例中的单根细索的长度为50000厘米，其上端被卷曲缠绕在低空飞行拍摄微型飞行器内部，下端悬挂着多种低空飞行拍摄微型飞行器，还悬挂着多种信号处理发射设备。
