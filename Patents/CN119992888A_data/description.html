Description

一种低空飞行器的航线路径计算与防碰撞方法
技术领域
本发明属于低空空域飞行器飞行技术领域，尤其涉及一种低空飞行器的航线路径计算与防碰撞方法。
背景技术
随着飞行器数量在低空空域的持续增加，如何有效预防飞行器之间的碰撞，已经成为低空产业发展中迫切需要解决的关键技术难题。这一挑战不仅关系到飞行安全，也直接影响到低空经济的可持续发展和整个空域管理的效率。在这一背景下，各种先进的飞行监控技术、智能避碰系统以及无人机自主飞行技术正在被积极研发和应用，以期构建一个更加安全、高效和智能的低空飞行环境。
对于低空空域飞行器的防碰撞研究，首先是需要获取飞行器的航线路径，现有技术中，一般是直接记录经纬度坐标，但是坐标数据不能直观体现飞行器位置，特别是只能记录采样时刻点的位置，两个采样点之间的路径信息无法体现出。另外，在飞行器之间的防碰撞计算时，一般也只是计算坐标距离，不能根据不同的位置情况进行分别预警。
发明内容
鉴于上述问题，本发明的目的在于提供一种低空飞行器的航线路径计算与防碰撞方法，旨在解决现有技术中，路径计算显示不直观、不完全以及防碰撞预警不够具体的技术问题。
本发明采用如下技术方案：
一方面，所述低空飞行器的航线路径计算方法，包括下述步骤：
步骤S1、获取飞行器的三维尺寸数据，根据地球网格层级的地球网格单元大小，以尺寸最大值确定当前飞行器对应的地球网格层级；
步骤S2、获取飞行器的飞行路径数据建立路径数据集合，其中所述飞行路径数据为各时刻点的地球空间坐标数据；
步骤S3、根据所述路径数据集合，计算飞行器飞行路径所经过当前地球网格层级的网格集合。
另外还包括：
步骤S4、在三维空间中，显示出飞行器所经过的网格集合作为飞行器的航线路径。
另一方面，所述低空飞行器的防碰撞方法，包括所述低空飞行器的航线路径计算方法，进一步还包括：
步骤S5、对于两架飞行器F1和F2，从对应航线路径的网格集合中获取t时刻点位置点和所在的网格单元和，并得到对应的编码和；
步骤S6、根据和的关系，以及网格单元和的中心点距离，判断网格单元和之间的空间关系，包括重合、包含、相邻、相离；
步骤S7、如果网格单元和之间的空间关系是重合或包含关系，则飞行器F1和F2会发生碰撞；若网格单元和之间的关系是相邻关系，则飞行器F1和F2存在发生碰撞的风险；若网格单元和之间的关系是相离关系，则飞行器F1和F2不会发生碰撞。
本发明的有益效果是：本发明技术方案中，按照地球网格将空间进行划分的方式，记录飞行器航线路径所经过的地球网格单元，生成网格集合，不以坐标点形式显示路径，以三维空间的地球网格单元形式显示航线路径，使得路径显示更直观、位置更现实更全面，而且也有利于后续的碰撞分析。另外，本发明通过判断飞行器所在网格单元之间的空间关系，分析低空飞行器之间是否会或者可能会发生碰撞，解决了飞行器在低空空域飞行过程中的防碰撞检测的问题。
