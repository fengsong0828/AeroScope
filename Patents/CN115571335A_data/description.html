Description

一种低空观光载人飞行器、系统及其信息传输方法
技术领域
本发明属于飞行器技术领域，尤其涉及一种低空观光载人飞行器、系统及其信息传输方法。
背景技术
根据欧洲安全小组(EHEST)统计，全球直升机的年均致命事故率高达30％以上。最根本原因来源于直升机本身的特性，直升机主要由旋翼和尾桨协调配合进行工作，而由于旋转中的旋翼在水平于地表时只能提供升力、无法提供向前飞行的拉力，因此直升机在平飞时必须以俯冲姿态来确保旋翼旋转产生的力中能分离出拉力，这种特性导致直升机相比于固定翼飞机更容易产生升力不足或失稳等问题。
根据市场调查，直升机螺旋桨高速旋转时声音高达110分贝，是人体所能适应分贝的1.5倍，长期处于大量噪音环境中会对人身心健康造成严重影响。直升机噪音的主要来源是空气交换作用在桨叶周围气流上的力在桨叶表面产生的气动冲击和湍流边界层以及直升机发动机内置电机转动。
根据市场调查，由于电动式旋翼机能耗比相对于固定翼较差，目前大部分2座电动载人飞行器的续航时间在30mins内。电池的频繁更换、充放电都会产生不同程度的安全隐患。
根据市场调查，目前大部分的观光直升机主要采用钢化玻璃材料，对于强穿透力的UVA波段紫外线无法起到很好的阻挡作用，同时由于厚玻璃的折射作用，常常出现黑的地方是阴影，没黑的地方太阳直射，极度影响了乘客的观赏体验。
根据市场调查，目前直升机领域主要通过飞行员驾驶完成观光飞行任务。由于飞行员的人力成本极高，同时在高密度飞行过程中飞行员容易产生疲劳导致飞行安全性无法得到保障。
发明内容
基于背景技术存在的技术问题，本发明提出了一种低空观光载人飞行器、系统及其信息传输方法。
本发明提出的一种低空观光载人飞行系统，包括：
载人飞行器，所述载人飞行器用在载人进行低空观光，且在飞行过程中按照自动规划路线行驶；
客户端，客户通过所述客户端登录相应APP或PC端登录官网系统进行线上预约、地点选择、支付服务费、了解载人飞行器实时位置信息等，客户端配套网络集群管理模块；
地面调度平台，所述地面调度平台用于将所有信息整合及处理，接收来自载人飞行器的动态飞行信息，对载人飞行器进行路线规划并发送相关指令，通过实时同步载人飞行器飞行状态、位置信息及站点状态信息，并将相关信息发送至相应APP及PC端官网系统，同时通过APP或PC端官网系统对客户订单信息进行收集及处理；
起落站点，所述起落站点用于载人飞行器在完成服务后，在地面调度平台预设好的地点进行自动停降，起落站点上的电感飞机感应器将实时更新站点上载人飞行器的停放情况，并将信息同步到地面调度平台。
进一步，所述地面调度平台内设置有地面调度平台模块，所述地面调度平台模块可为载人飞行器提供及时的信息决策，所述地面调度模块使用了多个通讯设备。
本发明还提出的一种低空观光载人飞行系统的信息传输方法，包括以下步骤：
S1：载人飞行器起飞前，地面调度平台的计算机终端系统收集的外部空间信息，向载人飞行器提供当前的飞行交通和飞行线路状况以及服务信息；
S2：载人飞行器在飞行过程中，地面调度平台通过机载信息单元或飞行路线动态信息，向载人飞行器提供飞行条件、飞行路线状况、飞行器运行情况、交通服务等实时信息，通过路径诱导系统对载人飞行器进行定位和导航；
S3：载人飞行器在飞行结束的时候，地面调度平台向载人飞行器传输预设好的地点位置，载人飞行器在最佳路线上，以最佳的飞行状态抵达目的地，实现整体系统的高效有序运作。
本发明还提出的一种低空观光载人飞行器，包括机体和内核，所述机体的尾部设置有多动力混合模块，所述机体的上表面设置有曲面液晶控光模块，所述机体的头部设置有射频雷达模块，所述射频雷达模块为5.8GFMCW射频雷达，所述机体上设置有前翼和后翼，所述前翼和后翼均为折叠的展翼与旋翼结构，所述前翼和后翼上均设置有涵道声学减噪模块，所述机体的动能使用新型固态锂电池。
进一步，其中，所述内核为ARM内核的处理器芯片，使用ARM内核的处理器芯片进行嵌入式系统开发，可实现载人飞行器姿态信息、雷达避险信息、平台调度信息等的实时收集。
进一步，其中，所述曲面液晶控光模块利用超白玻璃复合液晶调光膜技术，使透光率高达91.5％的同时减少90％以上的紫外光线透入，可实现智能透光与光线调控。
进一步，其中，所述内核中设置有自动驾驶模块，所述自动驾驶模块包括视觉辅助模块、全球卫星导航模块、舱内救援模块；
进一步，其中，所述涵道声学减噪模块使用了涵道内壁添加声衬与机舱内层隔音材料技术。
与现有技术相比，本发明的有益效果是：
1、本发明使用了多动力混动模块、舱内救援系统的新型倾转旋翼机，以解决目前观光直升机安全性较低问题，利用地面调度平台实时监测飞行环境和飞行器配备的射频雷达系统配合，提前预知空中风险，采用自动驾驶技术，配备一键切换紧急迫降系统、舱内救援系统，保障机舱内与景区内人员的人身安全；
2、本发明的载人飞行器将采用电动系统为主，同时利用涵道内壁添加声衬与机舱内层隔音材料技术结合降低飞行器产生噪音，同时利用飞行器空气动力学设计改进与缓冲材料结合提高飞行稳定性、改善乘坐舒适度；
3、本发明的载人飞行器采用可折叠式展翼与旋翼结合设计，空间利用率高，兼顾了飞行器体积小及垂直起降的功能，可满足短距起降及空间占有面积小等需求；
5、本发明采用了新型倾转旋翼设计，结合了电动式旋翼机自动起降及固定翼飞行器低能耗的优势，同时采用自主研发的新型固态锂电池将续航时间提高，实现强续航能力；
6、本发明的载人飞行器将采用曲面液晶控光模块实现智能光控，利用超白玻璃复合液晶调光膜技术，减少外界紫外光线透入，同时可调节机舱内光线条件；
7、本发明中配套网络集群管理模块将通过自动驾驶技术及地面调度平台大数据处理进行景区内所有飞行器的飞行路线规划等，高密度飞行管理，将事故率降到最低。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。
实施例一：
一种低空观光载人飞行系统，包括：
载人飞行器，载人飞行器用在载人进行低空观光，且在飞行过程中按照自动规划路线行驶；
客户端，客户通过客户端登录相应APP或PC端登录官网系统进行线上预约、地点选择、支付服务费、了解载人飞行器实时位置信息等，客户端配套网络集群管理模块；
地面调度平台，地面调度平台用于将所有信息整合及处理，接收来自载人飞行器的动态飞行信息，对载人飞行器进行路线规划并发送相关指令，通过实时同步载人飞行器飞行状态、位置信息及站点状态信息，并将相关信息发送至相应APP及PC端官网系统，同时通过APP或PC端官网系统对客户订单信息进行收集及处理；
起落站点，起落站点用于载人飞行器在完成服务后，在地面调度平台预设好的地点进行自动停降，起落站点上的电感飞机感应器将实时更新站点上载人飞行器的停放情况，并将信息同步到地面调度平台。
其中，地面调度平台内设置有地面调度平台模块，地面调度平台模块可为载人飞行器提供及时的信息决策，地面调度模块使用了多个通讯设备。
地面调度平台是有信息整合及处理的中心枢纽，通过实时同步载人飞行器飞行状态、位置信息及站点状态信息，并将相关信息发送至本团队研发APP及PC端官网系统，来实现智能化平台管理。客户通过手机端登录本团队研发APP或PC端登录企业官网进行线上预约、地点选择、支付服务费、了解飞行器实时位置信息等。客户的订单信息会第一时刻通过后台发送到地面调度平台，经过平台智能算法处理后给载人飞行器发送调度指令，在特定站点进行载客飞行服务。载人飞行器在飞行过程中会按照自动规划路线行驶，同时将动态飞行信息同步给地面调度平台，以确保飞行过程的安全有序。载人飞行器在完成服务后会自动停降在平台预设好的起落站点上，站点上的电感飞机感应器将实时更新站点上载人飞行器的停放情况，并将信息同步到地面调度平台，至此完成一次完整的工作流程。
实施例二；
一种低空观光载人飞行系统的信息传输方法，包括以下步骤：
S1：地面调度平台可为载人飞行器提供及时的信息决策，地面调度平台使用了大量的通讯设备，对有载人飞行器的飞行情况，地理位置信息，电量剩余信息等进行收集统筹，经智能算法处理后给予起飞申请的载人飞行器进行路线规划以及给予飞行调度指令，载人飞行器起飞前，地面调度平台的计算机终端系统收集的外部空间信息，向载人飞行器提供当前的飞行交通和飞行线路状况以及服务信息，帮助载人飞行器规划起飞时间与飞行路线；
S2：载人飞行器在飞行过程中，地面调度平台通过机载信息单元或飞行路线动态信息，向载人飞行器提供飞行条件、飞行路线状况、飞行器运行情况、交通服务等实时信息，通过路径诱导系统对载人飞行器进行定位和导航；
S3：载人飞行器在飞行结束的时候，地面调度平台向载人飞行器传输预设好的地点位置，载人飞行器在最佳路线上，以最佳的飞行状态抵达目的地，实现整体系统的高效有序运作。
实施例三：
一种低空观光载人飞行器，包括机体和内核，机体的尾部设置有多动力混合模块1，机体的上表面设置有曲面液晶控光模块2，机体的头部设置有射频雷达模块3，射频雷达模块3为5.8GFMCW射频雷达，机体上设置有前翼和后翼，前翼和后翼均为折叠的展翼与旋翼结构，采用可折叠式倾转旋翼，利用特殊的结构设计结合旋翼飞机的垂直起降功能与固定翼飞机的高效节能优势，前翼和后翼上均设置有涵道声学减噪模块4，机体的动能使用新型固态锂电池。
其中，内核为ARM内核的处理器芯片，使用ARM内核的处理器芯片进行嵌入式系统开发，可实现载人飞行器姿态信息、雷达避险信息、平台调度信息等的实时收集。载人飞行器采用了ARM内核的处理器芯片进行嵌入式系统开发，可实现飞行器姿态信息、雷达避险信息、平台调度信息等的实时收集。
其中，曲面液晶控光模块2利用超白玻璃复合液晶调光膜技术，使透光率高达91.5％的同时减少90％以上的紫外光线透入，可实现智能透光与光线调控。
其中，内核中设置有自动驾驶模块，自动驾驶模块包括视觉辅助模块、全球卫星导航模块、舱内救援模块，结合5G网络实现智能网联、算法开发等实现自动驾驶、路线规划、自动飞行，位置报告，自动避险等功能。
其中，涵道声学减噪模块4使用了涵道内壁添加声衬与机舱内层隔音材料技术，实现了减震降噪功能；同时利用飞行器空气动力学设计改进与缓冲材料结合提高飞行稳定性、改善乘坐舒适度。
以上所述仅是本发明的优选实施方式，应当指出，对于本技术领域的普通技术人员来说，在不脱离本发明原理的前提下，还可以做出若干改进和润饰，这些改进和润饰也应视为本发明的保护范围。
