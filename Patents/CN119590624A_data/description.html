Description

一种低空载人飞行器
技术领域
本发明涉及载人飞行器技术领域，具体为一种低空载人飞行器。
背景技术
载人飞行器作为现代航空领域的璀璨成果，正重塑着人类的出行与探索方式。从常见的民用直升机到前沿的电动垂直起降飞行器，它们凭借独特设计为人们插上翱翔天际的翅膀。在结构设计上，为保障飞行安全与舒适，采用高强度轻质材料，如碳纤维复合材料，既减轻机体重量，又能耐受飞行中的各类应力。驾驶舱配备先进航电系统，集成高清显示屏、智能导航与自动驾驶辅助功能，让飞行员精准掌控飞行姿态、方位与航线，即便是复杂气象条件下也能从容应对。
当前市面上的载人飞行器虽取得诸多突破，但仍存在一定局限性，其中较为突出的是难以灵活改变停止于水面和地面的状态。从设计构造来看，多数传统民用直升机与电动垂直起降飞行器，起落架多为固定式或仅适配常规陆地降落，若遇水面迫降需求，其结构无法有效缓冲冲击，更缺乏水上漂浮与稳定停泊的设计，极易导致飞行器侧翻、下沉，危及驾乘人员安全。
发明内容
为实现以上目的，本发明通过以下技术方案予以实现：一种低空载人飞行器，包括腔室，所述腔室的下表面固定连接有固定板，所述腔室的上表面设置有飞行机构，所述飞行机构包括固定架和固定箱，所述固定箱固定连接在腔室的外表面，所述固定架固定连接在固定箱的上表面，通过设置飞行机构，可以在控制下，使腔室能够完成起飞工作，并控制下在空中完成不同方向飞行的效果；所述腔室的上表面贯穿有空气流通机构，该空气流通机构用于对腔室中的空气进行净化，所述空气流通机构包括排气管，所述排气管贯穿腔室的上表面，所述排气管的顶端固定连接有第一支撑杆，所述第一支撑杆的端部固定连接有步进电机，所述步进电机的输出端通过联轴器安装有转动杆，所述转动杆的端部固定连接有扇叶；所述固定箱的下表面设置有降落机构，所述降落机构包括连接架，所述连接架固定连接在固定箱的外侧面，所述连接架的底端固定连接有限位圈，所述限位圈的内腔处转动连接有转动块，所述转动块的端部固定连接有转动架，通过设置空气流通机构，可以使腔室内腔中的空气能够排出腔室，进而使腔室内腔中的空气在空中飞行的时候能够与外界交换，在飞行器飞行的时候，腔室需要保持密封，以防止外界噪音和气流快速进入腔室内腔，但是高密封性会导致腔室内腔中的废气得不到排出，因此需要设置空气流通机构，并该空气流通机构能够控制降落机构的工作状态，通过设置步进电机，可以在接入电源并打开开关之后，使转动杆带动着扇叶产生旋转的效果，并能够控制转动杆正转和反转，通过设置降落机构，可以在飞行器需要降落在水中或滑行降落的时候，能够对腔室的底部进行保护，使飞行器能够漂浮在水面之上或在陆面产生移动的效果。
优选的，所述腔室的外表面固定连接有弧形玻璃罩，所述腔室的外表面固定连接有限位架，所述限位架的外表面转动连接有转动板，所述转动板的外表面固定连接有舱盖，所述舱盖与腔室的外侧面相挤压适配，所述腔室内腔的底面设置有座椅，所述腔室的内壁处设置有操作台，通过设置弧形玻璃罩，可以使飞行员在腔室内腔中的时候，能够通过弧形玻璃罩观察外面的状况，并且采用弧形的形状能够减少空气的阻力，通过设置限位架，可以对转动板进行限位，进而使舱盖能够产生旋转，进而使舱盖能够对腔室进行封闭，通过设置操作台，可以使飞行员对飞行器进行操控。
优选的，所述固定板的外侧面固定连接有弯杆，所述弯杆的端部固定连接有弹性底杆，所述固定板的外侧面对称固定连接有挤压杆，所述挤压杆的底端固定连接有挤压块，所述挤压块与弹性底杆的上表面相挤压适配，通过设置弹性底杆，可以在飞行器降落的时候，使弹性底杆能够与地面得到支撑，通过设置挤压杆和挤压块，可以在飞行器降落的时候，对弹性底杆形变的趋势进行限位，防止弹性底杆形变过大而使腔室与地面接触，从而防止腔室产生损坏而威胁人员的安全。
优选的，所述固定箱的外侧面贯穿有第一透气板，所述固定箱的内腔处设置有蓄电池，所述固定架的上表面固定连接有中央控制器，所述中央控制器通过线排与蓄电池相连接，通过设置蓄电池，可以对装置飞行需要的电力进行储存，并通过线排与中央控制器接触，进而使中央控制器能够接入电路。
优选的，所述固定架的边角处均固定连接有连接管，所述中央控制器的外表面连接有导线，所述导线贯穿连接管，所述连接管远离中央控制器的一端固定连接有固定圈，所述固定圈的内壁处固定连接有伺服电机，所述伺服电机通过导线与中央控制器相连接，所述伺服电机的输出端通过联轴器安装有螺旋桨，通过设置导线，可以将中央控制器与伺服电机连接在一起，进而在中央控制器的控制下，使伺服电机产生旋转，进而带动输出端的螺旋桨产生旋转，从而使飞行器能够产生向下的气流，最终使飞行器能够起飞和空中飞行的效果。
优选的，所述排气管内壁的底部固定连接有筛板，所述排气管内壁的顶部固定连接有限位管，所述限位管的内腔处滑动连接有第二透气板，所述第二透气板的上表面固定连接有第一弹簧，所述第一弹簧的顶端固定连接在限位管内壁的顶部，所述第二透气板的下表面固定连接有连接柱，所述连接柱的底端固定连接有堵塞板，所述排气管内壁的底部固定连接有堵塞圈，所述堵塞板与堵塞圈的上表面相挤压适配，通过设置筛板，可以防止腔室内腔中的物品进入排气管的内腔中，通过设置限位管，可以对第二透气板进行限位，使第二透气板能够在限位管的内腔中产生垂直上下移动的效果，通过设置第一弹簧，可以在第二透气板上下移动之后，使第二透气板能够产生回弹并恢复原位，通过设置堵塞板和堵塞圈，可以在接触在一起的时候，使腔室内腔中的空气不会进入排气管的内腔中，从而在步进电机反转并且扇叶向排气管内腔中吹气流的时候，使堵塞板与堵塞圈接触，进而使空气不会进入腔室的内腔中，而在步进电机正转并扇叶将排气管内腔中的气流排出的时候，使堵塞板远离堵塞圈，进而使腔室内腔中的空气与外界的空气产生对流。
优选的，所述排气管的外侧面贯穿有排气机构，所述排气机构包括连接箱，所述连接箱贯穿排气管的外侧面，所述连接箱的内壁处固定连接有第三透气板，所述第三透气板的内腔处滑动连接有滑动杆，所述滑动杆靠近排气管的一端固定连接有密封板，所述密封板的外圈处固定连接有密封圈，所述密封圈与连接箱的内壁相挤压适配，所述滑动杆的外表面套设有第二弹簧。
优选的，所述连接箱远离排气管的一端固定连接有弯管，所述弯管的端部固定连接有触发机构，所述触发机构包括连接框，所述连接框固定连接在固定箱的外侧面，所述连接框的内壁处固定连接有密封管，所述弯管远离排气管的一端固定连接在密封管的顶端，所述密封管的内腔处滑动连接有活塞，所述活塞的下表面固定连接有连接杆，所述连接杆的底端固定连接有齿板，所述活塞的下表面固定连接有第三弹簧，所述第三弹簧的底端固定连接在密封管内腔的底面。
优选的，所述转动块远离转动架的一侧固定连接有齿轮，所述齿轮与齿板的齿条处相啮合，该齿轮外表面的齿块只分布有四分之一圆周，通过设置齿轮，并只在齿轮外表面四分之一圆周设置齿块，使齿板在向下移动的时候，使齿轮只会产生九十度范围的转动，所述转动架的外表面固定连接有减速机构，所述减速机构包括限位箱，所述限位箱固定连接在转动架的外表面，所述限位箱的内腔处滑动连接有移动板，所述移动板的外表面滑动连接有滑动块，所述滑动块的端部固定连接有倾斜板，所述限位箱的内壁处固定连接有隔杆，所述滑动块与隔杆的相对面之间固定连接有第四弹簧，通过设置减速机构，可以在飞行器降落的时候，自动增加飞行器的阻力，从而使飞行器能够快速减速，通过设置限位箱，可以对移动板进行限位，使移动板能够产生伸缩工作，通过设置倾斜板，可以在气流吹动下，使倾斜板受到风力而使移动板展开，进而增加与空气中的阻力，通过设置第四弹簧，可以在移动板移动之后，使移动板能够回弹至原位。
优选的，所述限位箱的外表面固定连接有第二支撑杆，所述第二支撑杆的端部固定连接有漂浮机构，所述漂浮机构包括第一气箱，所述第一气箱固定连接在第二支撑杆的端部，所述第一气箱的外表面固定连接有第三支撑杆，所述第三支撑杆的端部固定连接有第二气箱，所述第一气箱的外表面固定连接有限位框，所述限位框的内腔处转动连接有转动柱，所述转动柱的端部固定连接有滚轮，通过设置漂浮机构，可以使飞行器在需要降落在湖泊和海面上的时候，能够给飞行器底部提供一个向上的浮力，进而使飞行器能够漂浮在水面之上，同时能够在飞行器需要滑行降落的时候，使飞行器稳定地在地面上滑行，通过设置第一气箱和第二气箱，并且横截面为三角形，可以增加飞行器在水面上浮力的同时，能够减少空气的阻力，通过设置限位框，可以对转动柱进行限位，使转动柱能够带动着滚轮产生稳定的旋转，进而使飞行器在滑行的时候，能够在路面稳定的滑动。
本发明提供了一种低空载人飞行器。具备以下有益效果：
一、该低空载人飞行器，通过设置空气流通机构，可以使腔室内腔中的空气能够排出腔室，进而使腔室内腔中的空气在空中飞行的时候能够与外界交换，在飞行器飞行的时候，腔室需要保持密封，以防止外界噪音和气流快速进入腔室内腔，但是高密封性会导致腔室内腔中的废气得不到排出，因此需要设置空气流通机构，并该空气流通机构能够控制降落机构的工作状态。
二、该低空载人飞行器，通过设置排气机构，可以使步进电机反转并且扇叶向排气管内腔中吹气流的时候，使气流能够进入连接箱的内腔中，通过设置滑动杆，可以带动密封板产生横向移动的效果，通过设置密封板和密封圈，可以在气流吹动下，产生横向移动，进而使气流能够通过连接箱与密封圈之间的间隙排出。
三、该低空载人飞行器，通过设置减速机构，可以在飞行器降落的时候，自动增加飞行器的阻力，从而使飞行器能够快速减速，通过设置限位箱，可以对移动板进行限位，使移动板能够产生伸缩工作，通过设置倾斜板，可以在气流吹动下，使倾斜板受到风力而使移动板展开，进而增加与空气中的阻力，通过设置第四弹簧，可以在移动板移动之后，使移动板能够回弹至原位。
四、该低空载人飞行器，通过设置漂浮机构，可以使飞行器在需要降落在湖泊和海面上的时候，能够给飞行器底部提供一个向上的浮力，进而使飞行器能够漂浮在水面之上，同时能够在飞行器需要滑行降落的时候，使飞行器稳定地在地面上滑行。
五、该低空载人飞行器，通过设置弧形玻璃罩，可以使飞行员在腔室内腔中的时候，能够通过弧形玻璃罩观察外面的状况，并且采用弧形的形状能够减少空气的阻力，通过设置限位架，可以对转动板进行限位，进而使舱盖能够产生旋转，进而使舱盖能够对腔室进行封闭，通过设置操作台，可以使飞行员对飞行器进行操控。

第三实施例，如图10－图13所示，所述转动块73远离转动架75的一侧固定连接有齿轮74，所述齿轮74与齿板676的齿条处相啮合，该齿轮74外表面的齿块只分布有四分之一圆周，通过设置齿轮74，并只在齿轮74外表面四分之一圆周设置齿块，使齿板676在向下移动的时候，使齿轮74只会产生九十度范围的转动，所述转动架75的外表面固定连接有减速机构76，所述减速机构76包括限位箱761，所述限位箱761固定连接在转动架75的外表面，所述限位箱761的内腔处滑动连接有移动板762，所述移动板762的外表面滑动连接有滑动块763，所述滑动块763的端部固定连接有倾斜板764，所述限位箱761的内壁处固定连接有隔杆766，所述滑动块763与隔杆766的相对面之间固定连接有第四弹簧765，通过设置减速机构76，可以在飞行器降落的时候，自动增加飞行器的阻力，从而使飞行器能够快速减速，通过设置限位箱761，可以对移动板762进行限位，使移动板762能够产生伸缩工作，通过设置倾斜板764，可以在气流吹动下，使倾斜板764受到风力而使移动板762展开，进而增加与空气中的阻力，通过设置第四弹簧765，可以在移动板762移动之后，使移动板762能够回弹至原位。

工作原理：使用时，飞行员进入腔室1的内腔中，之后转动转动板12，使舱盖13对腔室1定位外侧面进行封闭，之后控制操作台14，并使蓄电池55产生电流并通过中央控制器52和导线57流向伺服电机510，伺服电机510旋转之后，会带动螺旋桨511产生旋转，进而产生向下的气流，在气流逐渐增大的时候，腔室1会逐渐上升，从而使飞行器完成起飞工作；当在飞行的过程中，需要对腔室1内腔中的气流进行交换的时候，操作人员将步进电机63接入电源并控制步进电机63的输出端正转，从而使扇叶64产生气流，并将排气管61内腔中的空气排出，在吸力的影响下，第二透气板610在限位管69的内腔中向上移动，从而使堵塞板613不再对堵塞圈614堵塞，最终使腔室1内腔中的空气能够进入排气管61的内腔，并最终排出，达到对腔室1内腔中的空气更换的效果；在需要将飞行器垂直降落在地面上的时候，控制操作台14，使飞行器平稳的向下降落，在降落的过程中，弹性底杆4与地面接触并产生形变，从而对力进行缓冲，同时，挤压杆9和挤压块10会对弹性底杆4的上表面进行挤压，从而防止腔室1过度降低而导致的与地面接触；在需要将飞行器降落在水面之上或者滑行降落的时候，操作人员控制步进电机63反转，从而使扇叶64反方向旋转，并将外界的气流吹进排气管61的内腔中，在持续吹气的过程中，密封板664和密封圈666被吹动，并产生横向移动，进而使气体进入连接箱661的内腔中，并通过弯管667进入密封管672的内腔中，在气压的影响下，活塞673带动着连接杆675和齿板676向下移动，在移动的过程中，齿轮74与齿板676相啮合，进而使转动块73和转动架75产生旋转，并使限位箱761处于垂直风向的方向，而第一气箱781和第二气箱783处于平行水面的效果，当气流吹向倾斜板764的时候，倾斜板764受到挤压力，进而使移动板762伸展出，增大与空气的接触面积，达到对飞行器减速的效果，飞行器在进入水面的时候，第一气箱781和第二气箱783会产生向上的浮力，从而使飞行器能够停靠在水面之上。
