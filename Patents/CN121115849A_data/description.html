Description

电动垂直起降飞行器的低空避障系统及方法
技术领域
本发明涉及电动垂直起降飞行器技术领域，尤其是涉及电动垂直起降飞行器的低空避障系统及方法。
背景技术
在电动垂直起降飞行器（eVTOL）低空飞行阶段，特别是进场降落和离场起飞过程中，三维环境感知与避障能力直接关系到飞行安全。
当前主流避障系统主要依赖单一类型的传感器，如激光雷达或毫米波雷达，这些方案存在明显的局限性：激光雷达虽然精度高但视场角有限，难以实现全方位覆盖；毫米波雷达在复杂低空环境中易受多径效应干扰，对小型障碍物的检测能力不足。
光学相机系统虽然具有高分辨率和成本优势，但在实际应用中面临三大挑战：动态范围受限导致高对比度场景下细节丢失，低光环境下信噪比急剧下降，以及多相机协同工作时视场覆盖难以优化。
特别是在城市峡谷等复杂低空环境中，飞行器需要同时应对前向飞行路径和下方降落区域的多维度障碍物，现有单一传感器方案无法满足全方位、高精度的三维避障需求。
这些技术瓶颈严重制约了电动垂直起降飞行器在复杂环境下的安全运行能力。
发明内容
本发明的目的在于提供电动垂直起降飞行器的低空避障系统及方法，以解决现有技术中存在的至少一项技术问题。
为解决上述技术问题，本发明提供电动垂直起降飞行器的低空避障系统，包括电动垂直起降飞行器的低空避障系统，所述电动垂直起降飞行器包括机身、动力系统以及机翼/升力系统，所述低空避障系统包括传感器组件和光学组件；所述传感器组件用于进行图像信息的采集，包括：
前向相机，设置于所述电动垂直起降飞行器的机身的机头前部中心；
下向相机，设置于所述电动垂直起降飞行器的机身的底部中心；
复用相机，所述复用相机成对设置，且成对设置的两个所述复用相机分别设置在所述电动垂直起降飞行器的机身的两侧；
其中，所述复用相机倾斜设置于所述机身两侧的交界线上，使得所述前向相机和成对设置的所述复用相机的视场至少部分重叠；所述下向相机和成对设置的所述复用相机的视场至少部分重叠。
进一步地，所述复用相机通过机械支架可调安装在所述交界线上；所述复用相机在俯仰方向上前倾25°±3°、下倾20°±3°的复合角度；在水平方向上保持5°±2°的内倾角。
进一步地，所述前向相机为第一人称视角相机，采用90°×60°广角镜头；所述下向相机为鸟瞰视角相机，采用80°×50°的镜头；述复用相机采用110°×70°超广角镜头。

为解决上述技术问题，本发明提供电动垂直起降飞行器的低空避障方法，所述电动垂直起降飞行器包括用于执行所述低空避障方法的低空避障系统；所述低空避障系统为前述的电动垂直起降飞行器的低空避障系统，所述低空避障系统包括传感器组件和光学组件；
所述低空避障方法包括：
曝光与发光时序同步控制：基于现场可编程门阵列同步控制所述传感器组件和所述光学组件，使得所述光学组件工作于脉冲模式，并设置所述传感器组件的全局快门曝光起始时间；
传感器数据融合处理：建立自适应加权融合机制，动态分配各相机置信度权重，输出实时三维避障地图；
系统工作流程控制：分时激活所述传感器组件中的各传感器进行障碍物检测和地形扫描，并在设定时间内完成数据融合处理；并根据环境光照条件自动调整所述光学组件的补光强度和所述传感器组件的曝光参数。
进一步地，所述曝光与发光时序同步控制包括：
时钟基准生成：采用低抖动的恒温晶振产生主时钟，通过所述现场可编程门阵列的锁相环进行频率倍增；
脉冲调制控制：采用三阶段脉冲驱动所述光学组件；
曝光时序编排：采用交错式触发策略对所述传感器组件进行触发；
抗干扰设计：采用自适应延迟补偿算法消除信号传输差异。
进一步地，所述脉冲调制控制包括：
预充电阶段：将所述光学组件充电至工作电压；
恒流阶段：恒流驱动，控制所述光学组件光强上升时间；
泄放阶段：快速放电电路确保脉冲后沿陡峭。
进一步地，所述传感器数据融合处理包括：
前向障碍物检测：对所述传感器组件采集的图像信息进行特征的提取与匹配；
下方地形重建：采用半全局立体匹配算法，结合Census变换与梯度信息进行代价计算；
自适应加权融合：构建置信度评估模型，动态分配所述传感器组件中各传感器的权重；
三维避障地图生成：采用截断符号距离函数融合获取所述三维避障地图。
采用上述技术方案，本发明具有如下有益效果：通过设置传感器组件和光学组件，通过传感器组件中的多传感器布局架构，使传感器之间形成视场重叠区，继而实现无盲区的环境感知；同时，通过控制传感器组件和光学组件的精确同步控制，解决了低光条件下成像质量下降的痛点；突破了现有技术中单视角感知的局限，以在在各种光照条件下实现厘米级精度的三维避障，为电动垂直起降飞行器安全起降提供可靠的技术保障。

低空避障系统工作流程采用精确的时序控制架构，实现多模态传感器的高效协同工作。具体执行过程如下：
周期触发机制：低空避障系统主控单元生成20Hz的同步脉冲信号（周期50ms），每个周期划分为三个时序阶段：
前向感知阶段（0-15ms）：激活前向相机111和位于机身左侧的复用相机113，同时开启前向补光模块121。前向相机111和位于机身左侧的复用相机113曝光时间严格控制在前向补光模块121脉冲结束后0.5ms（脉宽0.8ms），确保采集到稳定的反射光信号。
下方感知阶段（15-30ms）：切换至下向相机112和位于机身右侧的复用相机113，同步启动下向补光模块122。采用与前向感知阶段相同的曝光时序控制，但下向补光模块122的补光强度根据前向感知阶段检测的环境亮度自动调节（调节范围10%-100%）。
数据处理阶段（30-50ms）：并行执行前向障碍检测和下方地形重建算法，最终进行三维地图融合。
