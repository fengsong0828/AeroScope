Description

一种低空空域飞行器飞行路线规划系统
技术领域
本发明涉及路线规划技术领域，尤其涉及一种低空空域飞行器飞行路线规划系统。
背景技术
路线规划技术领域主要涉及在特定约束条件下，为移动目标生成最优或可行移动路径的理论与方法。该领域包括但不限于路径生成算法、路径平滑与优化、障碍规避、动态环境适应、能耗约束、时间窗口调度等内容。路线规划广泛应用于自动驾驶、机器人导航、交通管理、无人机调度、船舶航迹规划等场景中，重点关注目标从起点到终点过程中如何在空间中移动以达到效率、成本、安全性等指标最优。
其中，低空空域飞行器飞行路线规划系统是用于为在低空空域内飞行的飞行器生成飞行路径的系统。其用途在于根据飞行任务要求、飞行器性能参数、空域限制、地形信息、气象条件等要素，生成满足航线安全、效率和合规性的飞行路径。该系统能够提升飞行器在复杂低空环境下的飞行自主性与运行协调能力，广泛适用于城市空中交通、物流配送、巡查监控等应用场景。
传统规划系统仅依赖任务要求与常规空域、气象、地形等静态信息进行路径生成，难以识别调度任务中标签化表达所隐含的空域排斥特性，导致部分任务中对特定禁入区域或视距保持要求无法明确量化处理，在路径选取过程中缺乏对空域排斥等级的差异化判定机制，在空域段提取时亦未对低空障碍密集度与风剪角变化进行路径级别的联合约束，造成路径规划结果存在穿越局部扰动强区域的可能，若遇高密度建筑环境或多变气流条件，将导致飞行器路径不稳定、飞控指令频繁调整，降低路径执行一致性与飞行安全水平。
发明内容
本发明的目的是解决现有技术中存在的缺点，而提出的一种低空空域飞行器飞行路线规划系统。
为了实现上述目的，本发明采用了如下技术方案：一种低空空域飞行器飞行路线规划系统，所述系统包括：
任务解析模块获取低空飞行器调度任务数据，判断标签项是否为具有空域排斥特性的限制项，将空域划设等级表中的空域段进行编号比对，生成空域排斥标志信息；
空域建模模块基于所述空域排斥标志信息，判断排斥标志对应空域段是否为强制排斥标记，若为强制排斥则设为无效区域，提取剩余空域段中边界封闭完整、视距飞行规则等级大于基本VFR等级基准值的空域段，生成低空飞行路径候选空域集合；
路径生成模块基于所述低空飞行路径候选空域集合，采集每个路径段对应的低空层高度、风剪角分布与建筑物投影密度，对低空障碍投影数量与空域段宽度比值执行风险分布筛选，生成低空飞行风险引导初始路径信息；
可行性验证模块调用所述低空飞行风险引导初始路径信息，判断每段路径高度是否处于风压变化剧烈区段，剔除高风险标签路径段，将剩余路径段中连续稳定片段进行组合处理，生成低空飞行可行路径序列。
本发明改进有，所述空域排斥标志信息包括空域识别编码、任务限制映射表、排斥等级索引、空域适应标记和冲突判断值，所述低空飞行路径候选空域集合包括可通行空域段编号、边界坐标组、通视规则标签、最小飞行高度组和区域通行优先级，所述低空飞行风险引导初始路径信息包括路径节点编号图、风剪角偏移值图层、路径障碍密度评分集、方向差角参考集和路径段风险权重集，所述低空飞行可行路径序列具体为节点高度集合、路径段风压敏感等级、路径连续性映射序列和风压剔除路径标识集。
本发明改进有，所述任务解析模块包括：
任务信息提取子模块获取低空飞行器调度任务数据，任务数据包括任务类型、飞行起止点对、视距保持时长与禁入区域等级，提取任务标签中标记为路径规划强相关属性的字段项，对提取字段中的字段值进行编号归类，并以编号归类结果为基础建立任务标签索引列表，生成任务标签索引序列；
限制标签筛选子模块基于所述任务标签索引序列，根本标记为路径强制限制属性的标签项，调用标签项对应的属性等级与排斥等级标准值，判断属性等级是否大于排斥等级标准值，并对满足条件的标签项执行筛选处理，生成空域排斥限制编号集；
空域标志生成子模块根据所述空域排斥限制编号集，调用空域划设等级表中空域段的空域编号与划设等级值，比较空域段等级值与排斥限制编号集中每一编号对应的标签等级值大小关系，并将大于关系成立的空域段设置为空域排斥标志段，建立空域段编号与排斥标志的对应结构表，生成空域排斥标志信息。
本发明改进有，所述空域建模模块包括：
边界数据提取子模块基于所述空域排斥标志信息，根据每条空域段对应编号，调用低空空域层级下编号对应空域段的空间边界坐标、最低飞行高度限制与视距飞行规则等级三项参数，提取有效数据结构中的数值型参数项，生成空域段结构参数值集；
排斥标志判断子模块基于所述空域段结构参数值集，根据每个空域段的编号信息与对应排斥标志值，获取空域段标识等级与排斥等级阈值，执行空域标识等级与排斥等级阈值的差值计算，并将差值结果与设定可行区间范围进行比较判断，运算获取可判别段的排斥相对强度值，将排斥相对强度值大于排斥判断基准值的空域段标记为强制排斥区段，生成排斥等级筛分结果；
可行区域筛选子模块调用所述排斥等级筛分结果，根据标记为非强制排斥区段的空域编号，提取对应空域段的边界坐标点组并进行封闭性检验，同时比较视距飞行规则等级是否大于基本VFR等级基准值，若满足封闭性成立且等级大于基准值，则纳入路径规划候选区域列表，生成低空飞行路径候选空域集合。
本发明改进有，所述路径生成模块包括：
连接图构建子模块基于所述低空飞行路径候选空域集合，根据每个空域段的边界顶点坐标点集，对坐标点按照空间连接边界规则生成区域拓扑边连接关系，建立节点邻接序列和对应边界方向关系矩阵，生成空域段连接图结构信息；
节点映射识别子模块根据所述空域段连接图结构信息，采集飞行器任务调度表中的任务起点坐标与终点坐标，执行坐标点与图结构中节点的距离比较，判断其最短距离对应的节点编号，并记录为映射起点与映射终点，生成任务节点映射索引对；
路径风险筛选子模块调用所述任务节点映射索引对，遍历连接图结构中每个路径段节点序列，对路径段中每个单元段采集风剪角变化值、路径朝向夹角、障碍物投影数量与空域段有效宽度，计算障碍分布比值并与风剪扰动值进行联合判别，运算获取每个路径段的综合风险值，根据路径段的综合风险值小于路径风险筛选阈值的路径段集合，构建连续路径图段，生成低空飞行风险引导初始路径信息。
本发明改进有，所述可行性验证模块包括：
高度风压判断子模块调用所述低空飞行风险引导初始路径信息，根据路径段的节点高度值与对应路径段的风压梯度值，调用飞行器平台技术规范中定义的升限高度，对每个路径段节点的高度值与风压梯度变化区间范围进行交叉判断，筛除处于边界模糊区段的数据并标识剩余区段所属稳定或剧烈类别，生成风压区段判断分类信息；
风险强度计算子模块基于所述风压区段判断分类信息，根据被标记为风压扰动剧烈类别的路径段，采集每段路径的风压变化幅度、节点平均高度、路径坡度变化量、飞行器升限高度，执行风压与高度相对风险系数计算，运算获取剧烈路径段的风险强度值，将风险强度值大于风压风险基准值的路径段标注为高风险类别，生成高风险路径标记结果；
稳定路径组合子模块调用所述高风险路径标记结果，根据未被标识为高风险的路径段，按路径编号顺序对相邻路径段执行连续性判断，排除断裂段并组合为可连续路径段组，建立结构映射关系并重新生成任务对应路径索引集，得到低空飞行可行路径序列。
本发明改进有，所述系统还包括：
路径输出模块基于所述低空飞行可行路径序列，根据每段路径的节点坐标，获取飞行器航向控制数据结构模板，构建连续指令控制点组，将路径段首尾节点按飞行指令序列顺序排列，组装形成路径执行集，生成低空航线控制结构序列；
所述低空航线控制结构序列包括路径段控制指令集、航向控制点编号、速度控制配置表、飞行转向角度序列和路径段执行时间轴。
本发明改进有，所述路径输出模块包括：
航向模板生成子模块基于所述低空飞行可行路径序列，根据每段路径段的全部节点坐标，对节点坐标进行编号排序，并调用飞行器配置库中的航向控制模板结构，依据路径方向向量与节点密度区间匹配表进行模板结构索引定位，获取对应模板匹配关系表，生成控制模板结构索引信息；
节点指令构建子模块基于所述控制模板结构索引信息，根据对应模板结构的字段属性，提取每段路径段的首尾节点坐标点作为路径段执行起点与终点，对路径段方向向量与模板参考航向进行夹角判断，结合飞行器任务编号的速度参数设定范围，匹配每段路径段速度常量与航向调整区间边界值，建立路径段编号与指令字段的绑定关系，生成路径节点执行指令信息；
指令集组装子模块基于所述路径节点执行指令信息，根据路径段顺序，按节点编号构建连续路径指令集列表，将每段路径段的方向编码、速度值、转向控制角度阈值进行参数嵌套，实施路径段航行控制数据字段整合，生成低空航线控制结构序列。
与现有技术相比，本发明的优点和积极效果在于：
本发明中，通过对飞行任务中视距保持时长与禁入等级的结构化剥离，可针对具有空域排斥特性的标签信息执行精度化筛选，通过编号映射空域等级表，生成可判定强制性排斥关系的标志信息，在空域边界提取中引入视距飞行规则等级判断以剔除不满足最低视距基准要求的空域段，构建连接图并执行风剪角与建筑物投影密度联合计算，对路径段角度差与障碍密度比值进行动态加权，生成带有风剪差引导的初始路径信息，通过路径段高度与风压变化梯度之比的量化分析，对处于不稳定气流区的路径执行强度风险标注，实现路径连续性与稳定性判别，在路径组合阶段基于稳定片段序列构建连续指令点组并设置速度与转向调整阈值，增强路径执行序列对飞行控制的适应性与执行鲁棒性，提升在复杂低空环境下的航线可行性与规避风险能力。

请参阅图1，本发明提供一种技术方案：一种低空空域飞行器飞行路线规划系统，系统包括任务解析模块、空域建模模块、路径生成模块、可行性验证模块和路径输出模块；
任务解析模块获取低空飞行器调度任务数据，任务数据包括任务类型、飞行起止点对、视距保持时长与禁入区域等级，对任务标签中的三项内容执行优先剥离，判断标签项是否为具有空域排斥特性的限制项，并以限制项编号与等级为基准生成排斥标志，将空域划设等级表中的空域段进行编号比对，生成空域排斥标志信息；
空域建模模块基于空域排斥标志信息，调用每个空域段在低空空域层级下的空间边界坐标、最低飞行高度限制、视距飞行规则等级，判断排斥标志对应空域段是否为强制排斥标记，若为强制排斥则设为无效区域，若为非强制排斥则设为优先级较低区域，提取剩余空域段中边界封闭完整、视距飞行规则等级大于基本VFR等级基准值的空域段，生成低空飞行路径候选空域集合；
路径生成模块基于低空飞行路径候选空域集合，根据每个空域段的边界顶点集，生成空域段连接图，获取飞行器任务起止坐标点，标识起止点在空域连接图中的映射节点，采集每个路径段对应的低空层高度、风剪角分布与建筑物投影密度，对路径段风剪角变化值与路径朝向差值执行角度差判断，对低空障碍投影数量与空域段宽度比值执行风险分布筛选，对筛选后路径段中的节点依据风剪差与障碍风险得分执行加权偏移值运算，生成低空飞行风险引导初始路径信息；
可行性验证模块调用低空飞行风险引导初始路径信息，根据路径段节点的高度信息、风压梯度、飞行器升限高度，判断每段路径高度是否处于风压变化剧烈区段，计算风压变化幅度与升限高度间的比值，对路径段进行风险强度标签标注，剔除高风险标签路径段，将剩余路径段中连续稳定片段进行组合处理，生成低空飞行可行路径序列；
路径输出模块基于低空飞行可行路径序列，根据每段路径的节点坐标，获取飞行器航向控制数据结构模板，构建连续指令控制点组，将路径段首尾节点按飞行指令序列顺序排列，设定每段路径的飞行速度常量与转向角调整阈值，组装形成路径执行集，生成低空航线控制结构序列；
最低飞行高度限制指依据航空法规对飞行器在特定空域或地形条件下最低可安全飞行的高度限制，通常依据飞行规则和地形图设定；视距飞行规则等级指空域分类中对飞行器能否以目视规则飞行的限制；风剪角定义为飞行器在短距离内遇到的风速/风向变化所形成的角度差，是风剪效应的衡量指标；建筑物投影密度表示在特定网格单元内出现的高于最低通航高度的障碍物数量，用于评估空域中飞行障碍的复杂度；风压梯度代表单位距离内风压的变化率，是影响飞行路径扰动的关键气象参数，常通过高空气象数据或LIDAR回波获得；转向角调整阈值为飞行器在连续航向调整中能够承受或有效控制的最大方向变化速率，用于限定路径中转弯段控制精度；

任务信息提取子模块获取低空飞行器调度任务数据，任务数据包括任务类型、飞行起止点对、视距保持时长与禁入区域等级，提取任务标签中标记为路径规划强相关属性的字段项，对提取字段中的字段值进行编号归类，并以编号归类结果为基础建立任务标签索引列表，生成任务标签索引序列；
任务信息提取子模块基于调度任务中各飞行器的任务类型、起止点坐标、视距保持时长与禁入区域等级等字段展开处理，首先对每一条任务记录提取其原始数据字段，并进行结构化编号处理，例如任务类型“监测任务”设为T1、“物流配送”设为T2、“地理测绘”设为T3，飞行起止点对采用经纬度坐标形式记录后统一转换为二维索引对，在本实施例中，UAV001的起止点为(116.30,39.90)-(116.40,39.95)，编号为N1-N5，随后视距保持时长采用区间标准划分，设定视距保持时长被划分为三个区间：[0,90]、(90,150]、(150,180]，分别设定区间等级值为V1=1、V2=2、V3=3，该设置依据为飞行器地面控制系统通视保持能力经验值，例如大多数小型无人机在控制距离内通视维持能力约为90秒，因此以90秒为一级分界线，150秒为最大连续飞行时间设定上限，UAV002对应90秒划入V1，UAV001的120秒划入V2，禁入区域等级以国家空域划设标准等级（1–5级）为参考，依托任务数据库设定的路径规划标签体系，将等级3以上视为限制属性标签，任务中字段项依此进行编号标注，如禁入等级4标记为F4，对各项已编号字段设定编号映射函数，组合编号构成初始任务标签元组，对每一任务提取结果以元组形式归入标签序列中，例如UAV003标签组合为T3、N3-N6、V3、F4，最后对全任务表中的标签元组进行序列集合归并并去重，生成任务标签索引序列；

限制标签筛选子模块基于任务标签索引序列，根本标记为路径强制限制属性的标签项，调用标签项对应的属性等级与排斥等级标准值，判断属性等级是否大于排斥等级标准值，并对满足条件的标签项执行筛选处理，生成空域排斥限制编号集；
限制标签筛选子模块依据任务标签索引序列中标记为路径强制限制属性的标签项开展判断处理，首先调用排斥等级标准值参照表，其中将禁入等级大于等于3的标签视为具备强制性，本实施中设定排斥等级标准值R_base=3，该数值参考国家民用空域管理细则对飞行高度在150米以下区域所规定的空域分级，对任务中禁入区域等级与R_base进行大小关系判断，属性等级值大于R_base即视为应筛除限制项，例如F4所对应等级值为4，而当前排斥等级标准值为3，则满足条件保留F4，此外，对于视距保持时长的辅助限制基准值，设定最小通视约束等级V_min=2，依据多数机型的控制连续性时长定义，其来源于表1中样本平均保持时长大于105秒所划分等级为V2，因此设定V_min=2为筛选门槛，对任务中设定保持等级低于V_min的标签视为无稳定控制保障条件，如V1对应任务字段将被标记为排斥任务，最终形成的编号集示例为{F3,F4,V1}，以构建路径段筛选依据，生成空域排斥限制编号集。
空域标志生成子模块根据空域排斥限制编号集，调用空域划设等级表中空域段的空域编号与划设等级值，比较空域段等级值与排斥限制编号集中每一编号对应的标签等级值大小关系，并将大于关系成立的空域段设置为空域排斥标志段，建立空域段编号与排斥标志的对应结构表，生成空域排斥标志信息；
空域标志生成子模块根据空域排斥限制编号集，对空域划设等级表中所有空域段执行等级比对操作，在具体实施中，首先获取每个空域段的编号与其设定等级值，例如编号S12的空域段等级值为4，编号S7等级为2，对照排斥编号集中F4对应值为4，执行比较，将空域段等级值与排斥等级值进行判别时，设定大于等于判断基准为“是否满足空域等级值≥标签等级值”，若关系成立则该空域被标记排斥，此外对于复合限制判断，如F4与V1联合，需对空域段的航迹可通性指数进行辅助约束判断，若该段平均通视覆盖时长低于V_min=2，则联合作为排斥条件成立，设定排斥标签组权重判别权重W=1.0，当两个标签均满足时排斥权重乘积为W^2=1.0，则直接判定剔除，若仅满足一项则记为W=1，仍列入待比对序列，最后生成编号-排斥布尔对构成的结构化列表，例如{S12:True,S7:False,S19:True}，据此建立空域标识结构表，形成空域排斥标志信息。

获取低空飞行可行路径序列中每段路径段的所有节点坐标，首先对每段路径的所有节点坐标点进行编号处理，例如路径段S1包含节点A1、A2、A3，其三维坐标依次为(10,20,100)、(20,30,110)、(30,35,120)，按编号顺序依次记录为A1至A3，同时计算节点之间的密度区间，在本例中节点间平均间距为15m，根据路径段方向向量为(20,15,20)，求得路径方向向量模长为约32.4，对应方向角为45°，查表调用飞行器配置库中定义的航向控制模板结构，方向区间匹配条件为30°至60°，节点密度匹配区间为12m至18m，此时符合模板编号为T01的所有条件，从模板结构中读取其模板字段属性，对当前路径段建立节点编号与模板字段的映射索引，最终生成控制模板结构索引信息，其中方向区间匹配的上下限分别作为方向角偏差匹配的“阈值”上下限，设置依据为飞行器航向偏移误差在15°范围内时具备稳定控制能力，节点密度匹配区间的上下限则作为节点密度设定的“基准值”，该值取自任务类型为城市复杂空域环境下，路径分段控制要求中推荐的密度范围。
