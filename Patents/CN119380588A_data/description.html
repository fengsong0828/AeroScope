Description

一种低空飞行器实时监管及冲突预警系统
技术领域
本发明涉及监管预警技术领域，尤其涉及一种低空飞行器实时监管及冲突预警系统。
背景技术
随着低空飞行器在城市空域、农业作业、环境监测等领域的广泛应用，低空空域越发拥堵，低空飞行器在过程中不可避免出现飞行冲突的问题，严重影响飞行安全；此外，低空飞行器在飞行过程中存在飞行路径偏离以及非法飞行等安全隐患，而现有技术对低空飞行器的监管力度不足。
因此，本发明提供一种低空飞行器实时监管及冲突预警系统。
发明内容
本发明提供一种低空飞行器实时监管及冲突预警系统，通过对低空飞行器进行多源数据采集，获取低空飞行器的实时飞行数据；构建低空飞行器监管预警平台，将实时飞行数据传输至低空飞行器监管预警平台中进行数据融合处理，根据数据融合处理结果输出低空飞行器的实时监管数据；当判定低空飞行器存在冲突可能性时，根据低空飞行器的实时监管数据预测低空飞行器的当前飞行路线，结合预设飞行路线以及同一飞行区域的其他低空飞行器的冲突飞行路线，预测飞行冲突点，生成冲突预警数据；根据冲突预警数据动态控制低空飞行器的飞行路线，解除飞行冲突点，对低空飞行器进行冲突控制；既有效实现了对低空飞行器的飞行监管，又确保了低空飞行器的飞行安全。
本发明提供一种低空飞行器实时监管及冲突预警系统，包括：
数据采集模块，用于对低空飞行器进行多源数据采集，获取低空飞行器的实时飞行数据；
实时监管模块，用于构建低空飞行器监管预警平台，将实时飞行数据传输至低空飞行器监管预警平台中进行数据融合处理，根据数据融合处理结果输出低空飞行器的实时监管数据；
冲突预警模块，用于当判定低空飞行器存在冲突可能性时，根据低空飞行器的实时监管数据预测低空飞行器的当前飞行路线，结合预设飞行路线以及同一飞行区域的其他低空飞行器的冲突飞行路线，预测飞行冲突点，生成冲突预警数据；
冲突控制模块，用于根据冲突预警数据动态控制低空飞行器的飞行路线，解除飞行冲突点，对低空飞行器进行冲突控制。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，数据采集模块，包括：
多源数据单元，用于获取低空飞行器的多源数据监管源，对低空飞行器进行多源数据采集，其中，多源数据监管源包括：卫星监管源、地面监管源以及无人机自身监管源；
飞行数据单元，用于对多源数据采集结果进行数据处理，统一数据格式，合并相同数据，获取低空飞行器的实时飞行数据。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，实时监管模块，包括：
平台构建单元，用于获取低空飞行器实时监管需求以及冲突预警需求，设定平台架构，构建低空飞行器监管预警平台，其中，平台架构包括：数据传输架构、数据融合架构、实时监管架构以及冲突预警架构；
根据数据传输架构建立低空飞行器监管预警平台与多源数据监管源的数据传输通道；
数据传输单元，用于将实时飞行数据根据数据传输通道传输至低空飞行器监管预警平台中。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，实时监管模块，包括：
数据融合单元，用于根据预设数据融合策略将传输至低空飞行器监管预警平台中的实时飞行数据进行数据融合处理，其中，预设数据融合策略包括：深度学习融合策略以及卡尔曼滤波融合策略；
监管数据单元，用于根据低空飞行器实时监管需求从数据融合处理结果中输出低空飞行器的实时监管数据，其中，低空飞行器实时监管需求包括：飞行位置监管需求、飞行区域监管需求、飞行状态监管需求、飞行数据监管需求以及飞行环境监管需求。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，冲突预警模块，包括：
冲突判定单元，用于根据低空飞行器的实时监管数据获取低空飞行器的当前飞行位置数据以及当前飞行区域，若当前飞行区域的低空飞行器的数量不低于预设飞行数量时，判定低空飞行器存在冲突可能性，标记当前飞行区域为冲突飞行区域；
获取低空飞行器的预设飞行路线，获取低空飞行器的预设飞行路线的预设飞行区域，对预设飞行区域进行逐区域冲突判定，根据逐区域冲突判定结果确定预冲突飞行区域。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，冲突预警模块，包括：
预警启用单元，用于当低空飞行器飞行至预冲突飞行区域或飞行至冲突飞行区域时，启用低空飞行器监管预警平台的冲突预警架构，对低空飞行器进行冲突预警；
飞行预测单元，用于根据低空飞行器的实时监管数据预测低空飞行器的当前飞行路线，记为低空飞行器的预测飞行路线；
冲突范围单元，用于根据低空飞行器的设备参数数据设定低空飞行器的冲突范围并根据历史飞行数据划分冲突范围，其中，碰撞范围包括：低冲突范围、中冲突范围以及高冲突范围；
当其它低空飞行器处于低空飞行器的低冲突范围时，判定低空飞行器与其它低空飞行器无飞行碰撞；
当其它低空飞行器处于低空飞行器的中冲突范围时，判定低空飞行器与其它低空飞行器可能存在飞行碰撞；
当其它低空飞行器处于低空飞行器的高冲突范围时，判定低空飞行器与其它低空飞行器已发生飞行碰撞；
飞行区域分层单元，用于根据冲突飞行区域建立坐标系，根据低空飞行器的低冲突范围对冲突飞行区域坐标系进行高度分层，获取飞行区域高度层；
冲突飞行路线单元，用于获取低空飞行器的当前飞行区域高度层，根据当前飞行区域高度层获取同一飞行区域高度层的其它低空飞行器，并根据低冲突范围设定低空飞行器的冲突时间窗口以及冲突空间窗口，获取所述其它低空飞行器的飞行路线，若所述其它低空飞行器的飞行路线与低空飞行器的预设飞行路线在冲突时间窗口以及冲突空间窗口同时存在，则判定所述其它低空飞行器的飞行路线为冲突飞行路线；
飞行路线选择单元，用于设定路线误差阈值，根据历史预测飞行路线数据获取预测准确度，获取预设飞行路线与预测飞行路线的路线误差，根据预测准确度进行加权处理，若加权处理完成的路线误差低于路线误差阈值，则选择预测飞行路线进行冲突预测；
若加权处理完成的路线误差不低于路线误差阈值，则选择预设飞行路线进行冲突预测；
冲突预测单元，用于根据低空飞行器的实时监管数据确定低空飞行器的飞行速度规律，结合选择进行冲突预测的飞行路线预测低空飞行器与其它低空飞行器的飞行冲突点；
飞行碰撞单元，用于获取低空飞行器与其它飞行器的相对位置数据以及飞行速度数据，当其它飞行器处于低空飞行器的中冲突范围时，计算得到飞行冲突点的飞行碰撞概率；

当其它低空飞行器处于低空飞行器的中冲突范围时，飞行冲突点的飞行碰撞概率不变；

冲突预警单元，用于根据剔除后的飞行冲突点生成冲突预警数据。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，冲突控制模块，包括：
冲突解除单元，用于当监测到冲突预警数据时，根据预设冲突解除策略结合低空飞行器的实时监管数据确定低空飞行器的实时冲突解除策略，其中，预设冲突解除策略包括：调整飞行速度策略、调整飞行高度策略以及调整飞行方向策略；
冲突控制单元，用于手动或自动执行实时冲突解除策略，解除飞行冲突点，对低空飞行器进行冲突控制。
根据本发明提供的一种低空飞行器实时监管及冲突预警系统，还包括：
控制监测单元，用于对执行实时冲突解除策略的低空飞行器进行持续监测；
控制效果单元，用于根据持续监测结果获取执行实时冲突解除策略的低空飞行器的飞行冲突点以及飞行冲突点的飞行碰撞概率，当飞行冲突点数量减少或飞行碰撞概率降低时，判定实时冲突解除策略执行有效；
否则，重新确定实时冲突解除策略。

当其它低空飞行器处于低空飞行器的高冲突范围时，判定低空飞行器与其它低空飞行器已发生飞行碰撞；

若加权处理完成的路线误差不低于路线误差阈值，则选择预设飞行路线进行冲突预测；

该实施例中，根据低空飞行器的低冲突范围对冲突飞行区域坐标系进行高度分层，既确保冲突预警精度，又确保冲突预警效率。
该实施例中，当前飞行区域高度层是指低空飞行器在当前时段的飞行区域高度层。
该实施例中，根据当前飞行区域高度层获取同一飞行区域高度层的其它低空飞行器，即判定其它飞行区域高度层的其它低空飞行器与当前飞行区域高度层的低空飞行器不存在飞行碰撞，用于简化冲突预警分析。

该实施例中，冲突飞行路线是指根据冲突时间窗口以及冲突空间窗口所确定的对低空飞行器飞行存在冲突的其它低空飞行器的飞行路线。
该实施例中，路线误差阈值是指预设飞行路线与预测飞行路线可接受的路线误差，当预测飞行路线与预设飞行路线存在路线误差，则认为低空飞行器的飞行出现偏差，当预测飞行路线与预设飞行路线之间的路线误差不高于路线误差阈值，则预测飞行路线可接受，无需对低空飞行器进行飞行路线监管，选择预测飞行路线进行冲突预测。
该实施例中，若加权处理完成的路线误差不低于路线误差阈值，则说明低空飞行器的飞行路线出现不可接受偏差，需对低空飞行器根据预设飞行路线进行监管处理，将低空飞行器调整至预设飞行路线，因此，选择预设飞行路线进行冲突预测。

该实施例中，获取预设飞行路线与预测飞行路线的路线误差，根据预测准确度进行加权处理，用于确保路线误差的准确性。

该实施例中，飞行冲突点是指预测低空飞行器与其它低空飞行器发生冲突碰撞的飞行点。
该实施例中，飞行冲突点数量不唯一，同一飞行冲突点可能与多个其它低空飞行器存在飞行冲突。
该实施例中，飞行碰撞概率是指低空飞行器在飞行冲突点发生飞行碰撞的概率。
该实施例中，预测时间点是指预测低空飞行器与其它低空飞行器达到最小相对位置数据的时间点。
该实施例中，预测最小相对位置数据是指预测低空飞行器与其它低空飞行器之间的最小相对位置数据。
该实施例中，低冲突范围下限数据是指低空飞行器不发生飞行碰撞的最小距离范围，即最小安全飞行距离。
该实施例中，将飞行碰撞概率为0以及飞行碰撞概率为1的飞行冲突点进行剔除，当飞行碰撞概率为0，则飞行安全，无需进行冲突预警；当飞行碰撞概率为1，则应在之前时段对低空飞行器的飞行路线进行调整。
