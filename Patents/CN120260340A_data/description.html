Description

一种低空智联网中飞行器全生命周期安全风险评估方法和 系统
技术领域
本发明涉及低空飞行的技术领域，具体地涉及一种低空智联网中飞行器全生命周期安全风险评估方法和系统。
背景技术
随着低空飞行器(如无人机、eVTOL等)数量的激增和应用的多元化，传统的风险评估方法传统风险评估方法依赖静态模型或人工经验，无法实时响应低空飞行器突发状态变化(如极端天气、空域冲突、设备故障等)，导致风险评估滞后，威胁飞行安全。
发明内容
本发明实施例的目的是提供一种低空智联网中飞行器全生命周期安全风险评估方法和系统，该设备低空智联网中飞行器全生命周期安全风险评估方法和系统可以满足低空飞行器执行任务时高速机动的实时避障。
为了实现上述目的，本发明实施例提供一种低空智联网中飞行器全生命周期安全风险评估方法，所述低空智联网中飞行器全生命周期安全风险评估方法包括：
获取飞行器在全飞行周期中当前所处阶段；
根据当前飞行所处的阶段，确定当前动态权重分配模型；
获取并向当前动态权重分配模型输入所需的输入数据得到输出动态权重值，所述输入数据包括飞行器状态信息、低空基站信息和气象雷达信息；
根据所述动态权重分配模型的输出动态权重值确定所述飞行器的各安全风险概率，并根据所述各安全风险概率执行应急措施。
