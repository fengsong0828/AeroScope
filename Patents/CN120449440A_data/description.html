Description

一种高速公路低空飞行器优化布置方法
技术领域
本发明涉及飞行器优化技术领域，具体来说，涉及一种高速公路低空飞行器优化布置方法。
背景技术
低空经济作为一种战略性新兴经济产业，这两年得到快速发展，截至2024年，我国低空经济规模已突破5000亿元，无人机、eVTOL(electric Vertical Take-off andLanding，电动垂直起降飞行器)等低空飞行器在物流、应急、载人等场景加速应用，但基础设施配套滞后成为主要瓶颈。高速公路沿线具备天然的空域资源与地面基础设施(如服务区、监控系统)，但现有技术缺乏对飞行器起降、路径规划、能源补给的系统性整合方案。
在高速公路沿线规划低空飞行服务站点或起降枢纽，利用低空飞行器可以开展道路巡检、物流配送、旅游观光、应急救援、低空客运等应用。但是，低空经济作为一种新经济现象，目前对于其含义、如何建设等问题尚缺乏基本认识，特别是低空路网基础设施的建设，需要根据拟投入的低空飞行器数量开展规划，但是对于如何根据实际需求，综合考虑各种影响因素在起降枢纽配置合理数量的低空飞行器方面，尚没有理论依据。
当低空飞行器过量布置时使得组织调度复杂，装备得不到充分利用，当低空飞行器过少配置时则会影响任务效率和完成度。因此，目前尚没有成熟的技术标准用于低空飞行器的优化布置，这导致存在以下缺点：
1)影响因素考虑不足：低空飞行器的布置数量受人口规模、交通条件、空域资源、经济发展、消费能力、应用场景、社会情绪等多种因素影响，目前缺少基于影响因素确定低空飞行器布置数量的方法，导致低空飞行器布置数量存在盲目性；
2)对影响因素没有合理评估：对各影响因素所起作用大小缺少科学合理的评估，导致据此确定的低空飞行器数量不准确。
针对相关技术中的问题，目前尚未提出有效的解决方案。
发明内容
针对相关技术中的问题，本发明提出一种高速公路低空飞行器优化布置方法，以克服现有相关技术所存在的上述技术问题。
为此，本发明采用的具体技术方案如下：
一种高速公路低空飞行器优化布置方法，包括以下步骤：
S1、根据影响因素构建评价指标体系，并利用层次分析法及熵权法，确定各评价指标的综合权重，识别评价指标的重要性；
S2、基于评价指标的重要性，构建低空飞行器优化布置模型，并利用模拟退火算法求解得到服务站点低空飞行器的静态最优数量配置；
S3、利用社会情绪共振规避算法，动态调整服务站点低空飞行器的静态最优数量配置，得到服务站点低空飞行器的最终数量配置；
S4、根据服务站点低空飞行器的最终数量配置，结合社会情绪数据优化低空飞行器的飞行路径。

S12、利用层次分析法确定各评价指标的主观权重，并通过熵权法分析各评价指标的客观权重；
具体地，层次分析法(AHP)主要用于确定各个评价指标的相对重要性。具体操作步骤如下：
1)建立层次结构模型：
目标层：确定最终目标，即“优化低空飞行器布置”。
准则层：选择一级准则，通常是影响飞行器布置的几个关键因素，如：人口规模、交通条件、空域资源、经济发展、消费能力等。
指标层：在每个准则下，列出相关的具体评价指标。
例如：
目标层：低空飞行器优化布置；
准则层：人口规模、交通条件、空域资源、经济发展、消费能力、应用场景、辐射范围；
指标层：每个准则下的具体指标(如“人口密度”是“人口规模”下的一个指标)；
2)构建成对比较矩阵：
对于每一对准则(或指标)，进行成对比较。专家根据经验和知识判断每一对因素的相对重要性，打分通常是1至9的尺度：
1表示两个因素同等重要；3表示一个因素稍微重要于另一个；5表示一个因素明显重要于另一个；7表示一个因素非常重要于另一个；9表示一个因素极其重要于另一个；成对比较的结果可以生成一个对称矩阵，其中每个元素表示两者之间的相对重要性。
3)计算权重：
特征值法：通过矩阵的特征值方法，计算各个因素的权重(即每个因素对最终目标的贡献度)。通过计算最大特征值，得到归一化后的权重值。
具体地，熵权法是一种客观权重确定方法，通过评价指标的数据离散度来计算各个指标的权重，离散度越高，说明该指标对评估结果的贡献越大。具体步骤如下：
1)数据标准化：对各个指标的数据进行标准化处理，使其在相同的尺度下进行比较。常用的标准化方法有最小-最大标准化和Z-score标准化；
2)计算熵值：根据标准化后的数据，计算每个指标的信息熵；
3)计算权重：根据熵值计算每个指标的权重，权重值越大，表明该指标的信息量越多，对最终评价的影响也越大；

负面情绪：表示反对、不喜欢、担忧、消极的情感态度，例如“低空飞行器很吵，影响生活质量”。

机器学习方法：训练情感分类模型(如朴素贝叶斯、支持向量机等)来预测文本的情感极性。
深度学习方法：利用深度神经网络(如LSTM、BERT等)对文本进行情感分类，这些模型能够学习到文本中的复杂语境，从而识别出情感极性。

低强度的情感：表示情感较弱，如“低空飞行器还行，不算特别好”。
情感强度的计算方法：
词典方法：通过使用带有情感强度分值的情感词典(如VADER情感词典)，为每个情感词汇赋予一个强度评分。
机器学习方法：利用回归模型或分类模型，预测文本中的情感强度。
深度学习方法：基于神经网络模型(如BERT)对文本进行多层次的分析，输出文本的情感强度。

当计算出每一轮迭代的共振指数(RI)后，可以将其与一个预设的共振指数阈值进行比较。当RI超过阈值时，就表示存在潜在的情绪共振风险，需要采取相应的策略进行调整。
策略调整机制：
1)共振指数低于阈值：
如果RI小于设定的阈值，说明公众情绪相对平稳，情绪波动和情绪一致性较低，可以继续当前的部署策略，维持正常的低空飞行器配置。
2)共振指数超过阈值：
识别潜在的情绪共振区域：当RI超过阈值时，首先需要识别哪些地区的情感波动较大，并考虑是否需要针对这些地区采取特别的沟通或部署策略。例如，某些地区的公众情绪异常激烈，可能需要进行更加密集的宣传、教育和沟通，缓解公众情绪。
降低情感波动：在发现情绪共振的风险后，可能需要调整低空飞行器的配置或飞行路径，避免在情绪波动较大的区域进行过于激烈的服务部署，减少可能的冲突和不安。
增加公众参与度：如果多个地区的情感一致性较高(例如，多个地区都呈现负面情绪)，可以考虑增加公众参与的环节，如通过线上调查、意见反馈或互动平台，了解公众的具体担忧，并及时做出响应，提升公众的参与感和认同感。
强化沟通和反馈机制：加强和当地社区的沟通，解释低空飞行器的用途和安全性，降低不确定性带来的负面情绪。通过正面的媒体宣传、专题讲座等方式向公众传递信息，增强他们的接受度。
调整更新策略，当共振指数超过设定阈值时，可以选择以下几种调整策略：
调整低空飞行器的配置：减少或增加某些服务站点的低空飞行器配置，根据公众情绪的波动来灵活调整。
调整飞行路径和飞行频率：避免飞行器经过情绪波动较大的地区或调整飞行频率，减轻公众的情绪压力。
实施缓慢部署策略：如果情绪波动较大，可以考虑暂时放缓低空飞行器的部署进度，逐步扩展到各个地区，观察情绪反馈的变化。
