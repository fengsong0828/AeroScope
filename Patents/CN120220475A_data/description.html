Description

一种高速公路与空域协同的低空飞行器航线规划方法
技术领域
本发明涉及路径规划领域，具体来说，涉及一种高速公路与空域协同的低空飞行器航线规划方法。
背景技术
随着低空飞行器的不断发展，低空经济已经成为一种战略性新兴经济产业，而利用高速公路的路网资源布设低空飞行器，不仅能共享基础设施、降低成本，还能提升运输效率、促进区域经济发展。但是，目前低空飞行器的航线规划方法未考虑高速公路的拓扑结构，如车辆交通流量、路网密度和服务设施分布等。
传统人工势场法在路径规划中存在局部极小值陷阱、动态障碍物适应性差等问题，且未充分利用地面基础设施(如高速公路)与低空空域的协同关系。现有技术难以在复杂空域环境中高效生成兼顾安全性与效率的航线。
具体来说，现有低空飞行器航线规划方法存在以下问题：
(1)空域孤立规划：传统低空航线依赖空域障碍物信息，未利用地面高速公路网络的导航价值，导致路径冗余且资源浪费。
(2)动态适应性差：现有势场法对动态障碍物(如其他飞行器、突发气象区)建模不足，易陷入局部极小值或避障失效。
(3)未充分利用高速公路路网与低空空域的协同关系，导致规划结果效率低、冲突风险高。
针对相关技术中的问题，目前尚未提出有效的解决方案。
发明内容
针对相关技术中的问题，本发明提出一种高速公路与空域协同的低空飞行器航线规划方法，以克服现有相关技术所存在的上述技术问题。
为此，本发明采用的具体技术方案如下：
一种高速公路与空域协同的低空飞行器航线规划方法，包括：
基于决策树算法对空域层级进行初步分类和规则提取，并通过贝叶斯网络建立包含空域层级间概率依赖关系的贝叶斯网络模型，以优化各空域层级间的分类精度；
通过动态势场更新机制，更新斥力场参数；在每个空域层级内，计算高速公路引力场、静态斥力场及动态斥力场，并融合得到每个空域层级的综合势场值；
利用贪婪策略和优先级规则，并结合空域层级的综合势场值，获取飞行器航线的初始路径；
根据人工势场局部优化算法和动态冲突检测机制，对飞行器航线的初始路径进行调整，得到飞行器航线路径。
进一步的，基于决策树算法对空域层级进行初步分类和规则提取，并通过贝叶斯网络建立包含空域层级间概率依赖关系的贝叶斯网络模型，以优化各空域层级间的分类精度包括：
获取原始空域的数据特征集，包括空域物理特征、气象特征、飞行器操作特征及空域管理特征；
使用决策树算法，基于数据特征集中的特征，递归地划分数据，生成决策树模型，并利用交叉验证法评估决策树模型性能，以优化决策树模型；
通过决策树模型对原始空域的新数据特征集进行预测，以实现初步的空域层级分类；遍历决策树模型的每条路径，通过从根节点到叶节点，提取条件关系形式的分类规则；
基于条件关系形式的分类规则，并结合最大似然估计法计算的条件概率表，构建包含空域层级间概率依赖关系的贝叶斯网络模型；
将初步分类结果输入贝叶斯网络模型，根据贝叶斯网络中的概率依赖关系，调整初步的空域层级分类结果。
进一步的，遍历决策树模型的每条路径，通过从根节点到叶节点，提取条件关系形式的分类规则包括：
从决策树模型的根节点开始，通过递归函数来遍历决策树模型的每个节点，直到到达叶节点；每个非叶节点包含条件，每个叶节点包含类别标签；
在递归过程中，记录每条路径上的所有条件，并在到达叶节点时，记录叶节点对应的类别标签，形成条件关系形式的分类规则；
形成条件关系形式的分类规则之后还包括：
合并相似的分类规则，以减少规则集的复杂性；应用剪枝技术去除分类规则中的过拟合部分。
进一步的，基于条件关系形式的分类规则，并结合最大似然估计法计算的条件概率表，构建包含空域层级间概率依赖关系的贝叶斯网络模型包括：
获取条件关系形式的分类规则，并根据条件关系形式的分类规则中的条件和结论部分定义贝叶斯网络的变量节点，根据条件关系形式的分类规则中的因果关系定义变量节点之间的边；
使用最大似然估计来计算每个变量节点的条件概率表，并为每个变量节点填充条件概率表；
通过变量节点、变量节点之间的边及条件概率表，融合得到包含空域层级间概率依赖关系的贝叶斯网络模型。
进一步的，通过动态势场更新机制，更新斥力场参数包括：
根据飞行器任务的紧急度，调整高速公路引力场计算时的引力强度系数；根据飞行器任务续航时长，调整高速公路引力场计算时的衰减半径。
进一步的，计算高速公路引力场、静态斥力场及动态斥力场，并融合得到每个空域层级的综合势场包括：
通过高速公路引力场计算公式，计算高速公路引力场强度；
获取静态障碍物的几何中心坐标、高度及边界信息，并将空域划分为三维网格；计算每个三维网格中心到所有静态障碍物的最小距离；
当三维网格中心到所有静态障碍物的最小距离小于或等于静态障碍物安全距离时，则利用静态斥力场计算公式，计算静态障碍物的静态斥力值强度；当出现多个静态障碍物叠加区域时，取多个静态斥力值中的最大值作为最终的静态斥力值强度；
获取动态障碍物的位置、速度、航向及预计持续时间，并计算飞行器与动态障碍物预测轨迹之间的最小距离；
利用动态斥力场计算公式，计算动态障碍物的动态斥力值强度，并叠加至空域网格，同时，将过时的动态障碍物的动态斥力场衰减至零；
通过对高速公路引力场强度、静态斥力场强度及动态斥力场强度进行加权求和的方式，得到每个空域层级的综合势场值。

本发明的有益效果为：
(1)物理-信息空间耦合：通过高速公路与空域势场融合，将地面基础设施转化为空域导航资源，降低飞行耗能。
(2)多粒度障碍物建模：采用静态障碍物精确建模+动态障碍物实时预测的方法，可以兼顾安全性和计算效率。
(3)本发明通过改进人工势场模型，引入高速公路引力场与动态障碍物斥力场协同作用，实现低空飞行器航线的快速生成与动态优化。
(4)本发明充分发挥低空飞行器航线规划和地面交通基础设施的协同性，有效提高低空飞行器的航线的合理性和效率。
(5)本发明基于决策树和贝叶斯网络的组合方法，不仅优化低空飞行器航线规划中的空域层级分类，还简化了规则集，增强了模型的解释力和实用性，为低空飞行器航线规划提供了强有力的支持，使得飞行器在航线时，高度调整的更加精准和灵活，适用于飞行器当下的具体外部环境和飞行器任务要求。

需要说明的是，使用决策树算法基于上述特征集中的信息递归地划分数据。目标是生成决策树模型，该模型能够根据输入的特征预测出相应的空域层级类别。为了确保模型的有效性和避免过拟合，通常会采用交叉验证的方法来评估和优化模型性能。
利用生成的决策树模型对新的空域数据进行预测，从而实现初步的空域层级分类。同时，遍历决策树的每一条路径(从根节点到叶节点)，从中提取出条件关系形式(IF-THEN)的分类规则。这些规则本质上描述了在特定条件下如何确定空域的层级归属。
基于得到的分类规则以及最大似然估计法计算的条件概率表，构建一个贝叶斯网络模型。这个模型不仅包含了变量节点之间的因果关系(由分类规则定义)，还考虑了变量之间概率依赖性的量化表示(通过条件概率表)。
将通过决策树模型获得的初步分类结果作为输入，应用到构建好的贝叶斯网络模型中。依据贝叶斯网络中的概率依赖关系，进一步调整和优化最初的空域层级分类结果。这样做可以更准确地反映实际情况下的不确定性因素，提高分类的精确度。

递归函数是在其定义或实现中直接或间接调用自身的函数。在遍历决策树模型的过程中，递归函数通过从根节点开始逐步深入每个分支直到叶节点，来记录路径上的所有条件，并最终形成分类规则。
定义一个递归函数extract_rules，该函数接收两个参数：当前处理的节点和到目前为止累积的条件列表(conditions)。初始调用时，conditions为空列表。这个列表随着递归过程而动态增长。每进入一个新的分支，就将该分支的条件添加到列表中。当到达叶节点时，累积的条件列表包含了从根节点到该叶节点路径上的所有条件。
通过剪枝技术(如预剪枝或后剪枝)，在一定程度上去除那些对整体分类贡献不大但增加了模型复杂度的部分。在规则提取阶段，考虑移除那些过于特定或者基于少量样本生成的规则，从而提高规则集的稳定性和准确性。

最大似然估计是一种参数估计方法，用于估算给定数据集下模型参数的最佳值。即在给定其父节点的状态时，该节点取不同状态的概率。
首先收集数据：首先需要一个足够大的数据集，其中包含了所有相关变量的历史观测值。
计算概率：对于每个变量节点，基于其父节点的状态组合，计算该节点处于各个可能状态的概率。具体来说，就是统计在不同的父节点状态下，子节点出现各种状态的频率，然后将这些频率转化为概率。

轨迹预测：预测未来tpred秒(一般5-15S)内的轨迹。
最小接近距离：计算飞行器与障碍物预测轨迹之间的最小距离dproj。
例如，从气象部门获取最新的雷暴位置、速度和方向信息，同时从飞行控制系统获取飞行器的速度、高度和计划路线。使用数值天气预报模型预测雷暴在未来数小时内的移动路径。将飞行器和雷暴的预测路径输入到计算程序中，找出两者之间最小的距离。

4.协同势场融合与高度分层
(1)势场叠加规则
综合势场计算：
Utotal(x，y，z，t)＝watt·Uatt+wrep·(Urep-static+Urep-dynamic)；
其中，watt、wrep分别为引力与斥力权重系数，默认watt＝1.0，wrep＝1.2。
(2)空域高度分层策略
分层定义：
将低空空域按高度划分为多个层级(例如：0-50m为禁飞层，50-100m为低空物流层，100-200m为高速交通层)。
收集原始空域的数据特征集，并训练决策树模型，并利用交叉验证法评估模型性能。比如，使用10折交叉验证来确保模型的泛化能力，并通过调整参数(如最大深度、最小样本分裂数等)优化模型以避免过拟合。
从根节点开始递归遍历每条路径直至叶节点，记录每个非叶节点的条件及叶节点对应的类别标签。例如，″如果当前飞行高度＞50米且＜100米，则为空域层级B(低空物流层)″。
提取规则：根据遍历结果形成具体规则，如″如果能见度＞5公里且飞行高度在50-100米之间，则推荐为低空物流层″。
合并相似规则：识别出具有相似条件但不同类别标签的规则，并尝试合并它们简化规则集。
剪枝：去除那些过于具体或对整体分类贡献不大的分支，减少过拟合的风险。
根据提取出的分类规则，将各个条件(如能见度、飞行高度等)作为变量节点，并依据因果关系建立节点间的有向边。例如，″能见度″影响是否适合″低空物流层″的决定。
计算条件概率表：对于每个变量节点，使用最大似然估计法基于训练数据计算其条件概率表。例如，计算当″能见度＞5公里＝true″时，″飞行高度在50-100米＝true″的条件下，″属于低空物流层＝true″的概率。
构建网络：结合所有变量节点、它们之间的边以及相应的条件概率表，构建最终的贝叶斯网络模型。该模型不仅反映了各变量间的关系，还量化了这些关系的概率依赖性。
将初步分类结果输入到构建好的贝叶斯网络模型中，利用模型中的概率依赖关系进一步调整和细化各空域层级的分类结果。例如，对于一个新输入的飞行任务，如果初步分类建议为“低空物流层”，但贝叶斯网络分析显示，在当前气象条件下存在较高的不确定性，可能需要重新评估或选择更安全的飞行高度区间。
分层势场计算：
每层独立计算引力与斥力场，避免跨层干扰；飞行器切换高度层时，需满足垂直势场梯度连续(如从100m层降至80m层时，需确保下方无高斥力障碍物)。
5.动态势场更新机制
实时数据接入：
(1)通过5G/卫星链路接入实时气象数据(风速、降雨)、空域管制指令、其他飞行器状态。
(2)更新动态障碍物斥力场参数(位置、速度、预测轨迹)。
自适应参数调整：
(1)根据任务紧急度自动调节kroad，比如紧急任务，则可提高kroad的值，使引力强，飞行路径贴近高速公路；降低kroad的值，则引力弱，允许更大偏离高速公路，以规避障碍。
(2)根据能耗限制调节σ，比如长续航任务时提高σ值，从而扩大引力场作用范围，减少偏航能耗。
三：初始路径生成
1.路径骨架构建
沿高速公路引力场选取关键节点(如每5公里一个节点)，生成初始路径骨架；通过A*算法连接节点，避开静态斥力场高值区域。A*算法又称A星算法，是启发式搜索算法中的一种。
2.启发式路径扩展

优先级规则：紧急任务飞行器优先使用低空层(50-100米)，长续航任务飞行器优先沿高速公路引力场飞行。
