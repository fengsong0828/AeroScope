Description

一种用于低空飞行器的螺旋桨装置
技术领域
本实用新型涉及飞行器设备的技术领域，特别是涉及一种用于低空飞行器的螺旋桨装置。
背景技术
现有的低空飞行器中，通常采用固定设置的螺旋桨结构，其在低空飞行器上始终处于固定的张开状态，而其始终保持张开状态时，会使其桨叶受到的空气阻力大，还会导致整个飞行器的中心集中靠前，进而容易导致飞机在空中处于滑翔阶段时出现快速下坠状态。
实用新型内容
本实用新型要解决的技术问题在于克服现有技术中的缺陷，从而提供一种用于低空飞行器的螺旋桨装置。
为实现上述目的，本实用新型采用了如下技术方案：
一种用于低空飞行器的螺旋桨装置，包括制动桨座、桨轴座、转轴，以及两片桨叶；
所述制动桨座用于与低空飞行器相连；
所述桨轴座通过所述转轴与所述制动桨座转动连接；
所述桨叶一端通过安装轴转动安装于所述桨轴座上，所述安装轴的中心轴线平行于所述桨轴座和所述桨叶的旋转平面，以使所述桨叶停止转动并受到风阻时，能够向靠近所述低空飞行器两侧方向转动。
优选地，所述转轴贯穿所述制动桨座，并与所述制动桨座转动连接；
所述转轴一端与所述低空飞行器的蓄力源相连，另一端通过轴头组件与所述桨轴座相连。
优选地，所述轴头组件包括轴头和弹性元件；
所述转轴前端与所述轴头相连，以将所述低空飞行器的蓄力源的动力传输至所述轴头，并驱动所述轴头转动；
所述弹性元件一端与所述轴头相连，另一端与所述桨轴座相连，且所述弹性元件处于自由状态时，所述桨轴座一侧端面与所述制动桨座相抵，另一侧端面与所述轴头端面之间具有预设间隙。
优选地，所述桨轴座与所述轴头通过抵接驱动结构相连；
所述抵接驱动结构包括第一抵接件和第二抵接件；
所述第一抵接件和所述第二抵接件中，其中一个固定安装于所述轴头上，另一个安装于所述桨轴座上；
所述第一抵接件和所述第二抵接件沿所述转轴的轴向上滑移连接，沿所述转轴轴向上抵接限位。
优选地，所述桨轴座上设有空腔；所述弹性元件局部位于所述空腔内。
优选地，所述轴头上形成有定位槽；
所述转轴一端设有折弯部，所述折弯部嵌入并固定于所述定位槽内；
所述转轴的另一端设有钩环，所述钩环与所述低空飞行器的蓄力源相连。
优选地，还包括下中空转轴，所述下中空转轴固定安装于所述制动桨座上设置的第一通道内，并与所述转轴滑动连接；
和/或，
还包括上中空转轴，所述上中空转轴一端固定于所述轴头上设有第二通道内，并与所述桨轴座和所述转轴均滑动配合连接。
优选地，所述上中空转轴与所述制动桨座之间设有垫片，所述垫片设置为钢性材料；
和/或，
还包括整流罩，所述整流罩设置于所述轴头组件上。
优选地，所述制动桨座朝向所述桨轴座的一端设有后单向齿；
所述桨轴座朝向所述制动桨座的一端设有前单向齿；
所述后单向齿与所述前单向齿相抵时，所述桨轴座相对制动桨座在所述桨叶旋转方向的反方向上锁止。
优选地，所述后单向齿具有第一水平抵接面，所述前单向齿上具有第二水平抵接面；
所述第一水平抵接面与所述第二水平抵接面时，两片所述桨叶呈水平分布于所述低空飞行器两侧。
相比现有技术，本实用新型的有益效果在于：本实用新型所提供的一种用于低空飞行器的螺旋桨装置，通过安装轴将桨叶转动安装在桨轴座上，且安装轴的中心轴线平行于桨轴座和桨叶的旋转平面，使得低空飞行器处于滑翔阶段，如手掷或橡皮筋弹性势能释放完毕后，桨叶能够在风阻的作用下向靠近低空飞行器机身的两侧方向转动，以使桨叶能够折叠收拢至机身的两侧，既能够减少整个低空飞行器滑翔阶段受到的空气阻力，又能够改变整个低空飞行器的重量分布，避免低空飞行器因为现有方案中固定螺旋桨导致低空飞行器头部过重，进而引起低空飞行器在滑翔阶段快速下降的问题，能够有效增加低空飞行器在空中滑翔时间，保证低空飞行器平稳飞行。
此外，可以理解的是，可转动设置的桨叶，还可在低空飞行器起飞阶段时，通过展开的桨叶，调整整个低空飞行器重心相对靠近低空飞行器中心位置，避免机身重心靠后而引起的拉翻现象，进一步保证低空飞行器平稳飞行。

参见图1至图9，本实用新型实施例提供了一种用于低空飞行器的螺旋桨装置，包括制动桨座1、桨轴座2、转轴3和桨叶4。其中，制动桨座1用于与低空飞行器相连；桨轴座2通过转轴3与制动桨座1转动连接；桨叶4一端通过安装轴5转动安装于桨轴座2上，安装轴5的中心轴线平行于桨轴座2和桨叶4的旋转平面，以使桨叶4停止转动并受到风阻时，能够向桨叶4能够向靠近低空飞行器两侧方向转动。
应当理解的是，上述方案中，通过安装轴5将桨叶4转动安装在桨叶4座上，且安装轴5的中心轴线平行于桨轴座2和桨叶4的旋转平面，使得低空飞行器处于控制滑翔阶段，如手掷或橡皮筋弹性势能释放完毕后，桨叶4难以维持旋转的动能时，桨叶4能够在风阻的作用下向靠近低空飞行器两侧的方向转动，以使桨叶4能够折叠收拢至机身的两侧，既能够减少整个低空飞行器滑翔阶段受到的空气阻力，又能够改变整个低空飞行器的重量分布，避免低空飞行器因为现有方案中固定螺旋桨导致的头部过重、风阻过大，进而引起低空飞行器在滑翔阶段快速下降的问题，能够有效增加低空飞行器在空中滑翔时间。此外，可以理解的是，可转动设置的桨叶4，还可在低空飞行器起飞阶段时，通过展开的桨叶4，调整整个低空飞行器重心相对靠近低空飞行器中心位置，避免机身重心靠后而引起的拉翻现象，进一步保证低空飞行器平稳飞行。
参见图1至图3，为了实现桨叶4的转动安装，桨轴座2上设有槽口，桨轴座2和桨叶4上均设有圆形孔，安装时，先将桨叶4插装至槽口内，并使桨轴座2和桨叶4上的圆形孔对齐，再将安装轴5贯穿圆形孔，以将桨叶4的一端转动安装至槽口内，以使桨叶4能够绕安装轴5的中心轴线转动。
应当理解的是，为了保证桨轴座2、桨叶4和安装轴5的稳定安装，同时，能满足桨叶4的转动需求，安装轴5可与设置于桨轴座2上的圆形孔过盈配合(胀接)，与桨叶4上的圆形孔间隙配合，使得安装轴5和桨轴座2相对固定，桨叶4绕安装轴5转动。当然，也可使得安装轴5与设置于桨叶4上的圆形孔过盈配合(胀接)，与桨轴座2上的圆形孔间隙配合，使得安装轴5和桨叶4相对固定，桨叶4和安装轴5绕桨轴座2上设置的圆形孔转动。
参见图1至图3，制动桨座1一侧设有机身安装部12，用于将整个制动桨座1固定安装在低空飞行器上。值得说明的是，为了使得机身安装部12适配不同类型的低空飞行器，机身安装部12可设置为矩形、圆形等，低空飞行器可采用木杆类、塑胶类、碳纤类、金属类材料中的一种或几种组合材料进行制作。
参见图1至图6，转轴3贯穿制动桨座1，并与制动桨座1转动连接；转轴3一端与低空飞行器的蓄力源相连，另一端通过轴头组件6与桨轴座2相连。应当理解的是，低空飞行器的蓄力源可设置为橡筋，橡筋的弹性势能释放时能够驱动转轴3和轴头组件6同步转动，而轴头组件6由于桨轴座2相连，进而能够驱动桨轴座2和桨叶4同步转动。
具体地说，轴头组件6包括轴头61和弹性元件62；转轴3前端与轴头61相连，以将低空飞行器的蓄力源的动力传输至轴头61，并驱动轴头61转动；弹性元件62一端与轴头61相连，另一端与桨轴座2相连；弹性元件62处于自由状态时，桨轴座2一侧端面与制动桨座1相抵，另一侧端面与轴头61端面之间具有预设间隙。
为了通过轴头61驱动桨轴座2，同时使得桨轴座2能够向轴头61稳定滑移至其远离轴头61的一端脱离制动桨座1，桨轴座2与轴头61通过抵接驱动结构7相连。
具体地，抵接驱动结构7包括第一抵接件71和第二抵接件72；第一抵接件71和第二抵接件72中，其中一个固定安装于轴头61上，另一个安装于桨轴座2上；第一抵接件71和第二抵接件72沿转轴3的轴向上滑移连接，沿转轴3轴向上抵接限位。
为了方便说明，以第一抵接件71设置在轴头61上，第二抵接件72设置在桨轴座2上为例。
第一抵接件71设置为第一圆弧板，第一圆弧板一端与轴头61固定连接，另一端向桨轴座2延伸，第二抵接件72设置为第二圆弧板，第二圆弧板一端与桨轴座2固定连接，另一端向轴头61延伸，第一圆弧板的侧边缘与第二圆弧板的侧边缘相抵之间始终抵接，第一圆弧板和第二圆弧板的内圆弧面正对设置，以使第一圆弧板和第二圆弧板能够围合成容纳弹性元件62局部的腔。
进一步地说，第一圆弧板和第二圆弧板可设置为同轴、直径相等且对称设置的圆弧板。
当然，在其他实施例中，第一抵接件71也可设置为矩形块，第二抵接件72也可设置为矩形筒状结构，能够满足桨轴座2和轴头61能够轴向滑移，又能够周向限位即可。
进一步地说，为了便于弹性元件62的安装，同时在一定程度上减小螺旋桨装置的轴向长度，在本实施例中，桨轴座2上设有空腔22；弹性元件62的局部位于空腔22内。不难理解的是，弹性元件62可设置为弹性橡胶套、弹簧等柔性结构，其一端位于空腔22内，另一端位于第一圆弧板和第二圆弧板围城成的腔内。
进一步地说，轴头61上形成有定位槽611；转轴3一端设有折弯部31，折弯部31嵌入并固定于定位槽611内；转轴3的另一端设有钩环32，钩环32与低空飞行器的蓄力源相连，以使得低空飞行器的蓄力源(如橡筋)释能的过程中，能够驱动转轴3转动，以使转轴3能够驱动轴头61转动，进一步驱动桨轴座2和桨叶4转动。
参见图3以及图7至图9，还包括下中空转轴100，下中空转轴100固定安装于制动桨座1上设置的第一通道10内，并与转轴3滑动连接。具体地，可下中空转轴100的外径大于第一通道10的直径，以使得下中空转轴100相对于制动桨座1固定安装，而下中空转轴100的内径则略大于转轴3的直径，使得转轴3、下中空转轴100的旋转中心稳定，减少晃动风险，同时减少转轴3和下中空转轴100之间的摩擦力，提高飞行性能。不难理解的是，通过下中空转轴100的设置还可减少转轴3对制动桨座1的摩擦损耗。
同理，还包括上中空转轴200，上中空转轴200一端固定于轴头61上设有第二通道610内，并与桨轴座2和转轴3均滑动配合连接。可以理解的是，上中空转轴200的外径大于第二通道610的直径，以使得上中空转轴200相对于轴头61固定安装，而上中空转轴200外径还略小于桨轴座2上第三通道20的直径，上中空转轴200的内径则略大于转轴3的直径，进而能够保证转轴3、上中空转轴200、桨轴座2和轴头61的中心稳定，同时，减少桨轴座2和上中空转轴200、上中空转轴200和转轴3之间的摩擦力，提高飞行性能。不难理解的是，通过上中空转轴200的设置还可减少转轴3对桨轴座2的摩擦损耗。可以理解的是，轴头61、上中空转轴200和制动桨座1相互配合，能够实现对桨轴座2轴向移动距离进行限位。
进一步地说，上中空转轴200与制动桨座1之间设有垫片8，垫片8设置为钢性材料，进而能够减少上中空转轴200对制动桨座1的损耗。
可以理解的是，垫片8设置为环状，垫片8外径大于下中空转轴100和上中空转轴200的直径，而内径则小于或等于下中空转轴100和上中空转轴200的直径，且垫片8的内径还大于转轴3的直径。
进一步地说，还包括整流罩9，整流罩9设置于轴头组件6上。整流罩9可与轴头61过盈配合。整流罩9的内壁还可设置与折弯部31的端面相抵的限位凸起，进一步对转轴3进行轴向限位，避免转轴3轴向移动。
由于惯性作用，桨叶4和桨轴座2无法瞬间停转而继续转动，进而导致橡筋容易反向蓄劲，进而容易使得桨叶4和桨轴座2会反向转动，导致空气阻力增大，进而引起低空飞行器快速下坠。
参见图3至图5，制动桨座1朝向桨轴座2的一端设有后单向齿11；桨轴座2朝向制动桨座1的一端设有前单向齿21；后单向齿11与前单向齿21相抵时，桨轴座2相对制动桨座1在桨叶4旋转方向的反方向上锁止。
应当理解的是，通过前单向齿21和后单向齿11相互配合，一方面，能够增加桨轴座2的摩擦力，以增加桨轴座2的制动效果，有利于使得桨叶4和桨轴座2能够同时停止旋转，避免桨轴座2在桨叶4停止转动后由于惯性作用下继续旋转引起低空飞行器的蓄力源二次蓄力的情况；另一方面，由于前单向齿21和后单向齿11相抵时，能够使得桨轴座2相对制动桨座1在桨叶4旋转方向的反方向上锁止，使得即使低空飞行器的蓄力源二次蓄力后也不会影响桨叶4和桨轴座2停转后出现反向旋转现象，避免低空飞行器快速下坠的情况。
进一步地说，后单向齿11具有第一水平抵接面111，前单向齿21上具有第二水平抵接面211；第一水平抵接面111与第二水平抵接面211时，两片桨叶4呈水平分布于低空飞行器两侧，进而避免低空飞行器中心偏移导致其侧滑，进一步避免引起低空飞行器快速下坠的情况。
综上可知，低空飞行器飞行过程中，各结构运行原理如下：
初始状态(制动状态)，各结构处于静止状态，弹性元件62处于自由状态，桨轴座2一端的前单向齿21(第一水平抵接面111)与制动桨座1一端的后单向齿11(第二水平抵接面211)相抵，两片桨叶4呈水平分布于低空飞行器机身的两侧。轴头61和桨轴座2之间具有预设间隙，第一抵接件71和第二抵接件72始终相抵。
当低空飞行器运行时，先手动推动桨轴座2向轴头61方向滑移，前单向齿21脱离后单向齿11(即脱离制动状态)，同时能够增加第一抵接件71和第二抵接件72的抵接面积，随着飞行时间推移，低空飞行器经历释能阶段后又恢复至制动状态。
具体地，以低空飞行器的蓄力源为橡筋为例进行说明，橡筋释放弹性势能，带动转轴3转动，转轴3再带动轴头61转动，轴头61进一步驱动桨轴座2和桨叶4转动，此过程中，由于第一抵接件71和第二抵接件72之间的摩擦力较大(可理解为旋转作用力导致两个端面之间作用力增大，进而导致摩擦力增大)，两者能够始终保持抵接状态。随着橡筋弹性势能释放时间，桨轴座2和桨叶4的旋转转速按由快变慢的趋势变化，飞机从急速上升状态进入平稳爬升状态。在橡筋弹性势能释放将尽时，低空飞行器桨从稳定爬升状态进入下滑阶段，由于橡筋弹性势能减小，第一抵接件71和第二抵接件72之间的摩擦力也随之减小，在弹性元件62弹性回复力作用下，第二抵接件72随着桨轴座2向远离第一抵接件71方向移动，直至前单向齿21与后单向齿11相抵，在两者“顺行逆止”功能下(即桨轴座2能够沿橡筋释能过程中的运转方向继续运转，而沿运转反方向逆止)，桨轴座2产生制动停止旋转，桨轴座2上的桨叶4同步停止转动，并复位至初始状态。值得说明的是，在橡筋弹性势能减小后，桨叶4会在空气阻力的作用下绕安装轴5转动，以收拢至机身的两侧。
