CN120857442A

Description

一种用于低空飞行器电源的高效轻质换热器
技术领域
本发明涉及的一种高效轻质换热器，特别是涉及应用于飞行器热管理技术领域的一种用于低空飞行器电源的高效轻质换热器。
背景技术
低空飞行器电子元件在运行时产生大量热量，特别是飞行器的电源。独立式风道正成为低空经济下飞行器电子元件散热设计的革新方向，其通过定制化气流路径与智能控制技术的深度融合，为轻量化、高功率密度的飞行器电子系统提供高效热管理解决方案。该设计摒弃传统分散式散热结构，转而构建与电子元件热源高度耦合的定向风道系统，利用空气动力学优化与材料科学突破，实现气流精准导引与热能快速耗散。
中国专利CN202322462636.5说明书公开了《一种便于安装的无人机电池散热器》，通过水冷器的设置，水冷器是散热系统的核心组件，它负责将循环管内的循环液流动过程中产生的冷空气传递到隔板，以吸收电池释放的热量，通过换热器的设置，便于当无人机电池过热时，换热器可以通过冷却介质的流动，吸收电池的热量，冷却介质可以是空气、液体等。中国专利CN202010939600.X说明书公开了《一种带有电路板散热结构的无人机》通过将电路板布置在散热部件的两个热传导板之间，将电路板包围，并且通过热传导板的传递将电路板整体产生的热量快速的均匀地传递给电路板散热部件顶板和底板，在循环通道内持续性的流动冷却介质时，冷却介质与电路板产生的热量进行交换，进而将电路板产生的热量带走，相对于传统的散热风扇的散热结构，将电路板包围并且通过冷却介质的循环流动对电路板进行全面散热，散热均匀并且散热效果更好。
现有技术中，风冷散热器多采用单一形状肋片，如附图3所示，导致局部散热不均或流动阻力过高，现有的换热器大多采用板式肋片，虽然流阻非常小，但其换热性能有限，已逐渐无法满足散热需求，为了进一步减轻重量，发展具有高传热性能、低流动阻力的换热结构势在必行。
发明内容
针对上述现有技术，本发明要解决的技术问题是，现有的换热器大多采用板式肋片，虽然流阻非常小，但其换热性能有限，已逐渐无法满足散热需求。
为解决上述问题，本发明提供了一种用于低空飞行器电源的高效轻质换热器，包括风冷散热道，风冷散热道的底部设有热换凹区，且风冷散热道的两端分别开设为气流入口和气流出口，气流入口和气流出口的内部分别固定连接有入口导流肋片和出口导流肋片，入口导流肋片与出口导流肋片之间固定连接有肋片模块，风冷散热道内部固定连接有柱肋模块，肋片模块与柱肋模块交替对应；
肋片模块包含平直形肋片和波浪形肋片，柱肋模块包含圆柱形肋柱、半圆形肋柱和半圆三角形肋柱，平直形肋片和波浪形肋片与圆柱形肋柱、半圆形肋柱和半圆三角形肋柱单独组合以及自由组合；
波浪形肋片顶部和底部与风冷散热道的内壁滑动连接，且波浪形肋片由钛合金材料制成，风冷散热道与热换凹区之间矩阵固定分布有温控标体，温控标体的顶端与波浪形肋片的内凹部之间固定连接有连接筋，连接筋由形状记忆合金材料制成。
在上述用于低空飞行器电源的高效轻质换热器中，根据使用环境，选择适合的肋片模块与柱肋模块组合方式的风冷散热道，便于提供更大换热表面积，并通过扰动流体有效提高热交换效率，进而有效提高飞行器的散热效果。
作为本申请的进一步改进，温控标体的中部开设有与热换凹区连通的活动槽，温控标体由气凝胶材料制成，气凝胶材料制成的温控标体导热率低，实现温控标体的隔热效果。
作为本申请的再进一步改进，连接筋的顶部和底部均呈弧形弯曲设置，且连接筋的顶部和温控标体的顶部之间包覆有隔温套，活动槽与连接筋的底部弧形端套接，形状记忆合金材料制成的连接筋以弧形弯曲位置进行弯曲变化，从而利用连接筋的弯曲幅度的变化推动波浪形肋片形变，改变波浪形肋片的局域弯曲幅度，从而有效增强该区域的流体扰动能力，有效提高该区域的热置换能力，隔温套有效提高连接筋的隔温能力，从而延长连接筋的形变状态时间，即延长提高该区域的热置换能力作用时间。
作为本申请的更进一步改进，活动槽的下方口固定连接有封堵片，连接筋的底部弧形端与封堵片接触对应，封堵片由铝合金材料制成，利用封堵片对活动槽下方进行封闭，有效避免风冷散热道内部流体泄漏到热换凹区内，并且通过连接筋与封堵片的接触热传导，便于连接筋的受热形变，后续在连接筋形变后，连接筋与封堵片断开接触，从而有效降低对连接筋的散热，从而延长连接筋的形变时间。
作为本申请的又一种改进，波浪形肋片的下部边缘开设有内容槽，内容槽的内部固定镶嵌有导热硅胶带，导热硅胶带与风冷散热道的内壁滑动连接，利用导热硅胶带有效提高波浪形肋片与风冷散热道内壁的换热能力。
作为本申请的又一种改进的补充，导热硅胶带的中部等距固定连接有垫脚片，垫脚片的两端与风冷散热道的内壁固定连接，导热硅胶带通过垫脚片与风冷散热道进行缓冲固定，保留导热硅胶带一定的活动幅度空间。
作为本申请的又一种改进的补充，垫脚片的两端均设置有拱形缓冲部，且垫脚片由弹性塑料制成，垫脚片通过拱形缓冲部进行弯曲幅度变化，实现垫脚片适应导热硅胶带的移动缓冲。
作为本申请的再一种改进，拱形缓冲部靠近导热硅胶带的一端底部开设有锁定孔，锁定孔的内部插接有锁定螺钉，锁定螺钉的底部与风冷散热道的内壁螺纹连接，且锁定螺钉的顶部与波浪形肋片的侧面接触对应，通过锁定螺钉穿过锁定孔螺纹连接风冷散热道，从而固定该区域的波浪形肋片位移能力，从而为波浪形肋片的弯曲幅度调整提供受力支点。
综上所述，本发明提出的高效轻质换热器根据使用环境，选择适应的肋片模块与柱肋模块组合型号，为电源系统有效提供更大换热表面积，并通过扰动流体有效提高热交换效率，确保飞行器电源系统在长时间高功率输出情况下有效提供高效散热效果，通过优化结构设计，最大程度减轻了换热器的重量，确保飞行器具有更好的载荷和续航能力，在使用肋片模块采用波浪形肋片时，通过温控标体对热换凹区进行局域性温度监测，当该区域的温度触发连接筋的热形变能力时，连接筋热形变后推动波浪形肋片的内凹部达到进一步弯曲的目的，从而增强该区域波浪形肋片的不规则性，有效增强流体的紊流扰动能力，局部提高该区域的换热能力，后续在温度降低后，波浪形肋片自动形状恢复。

具体实施方式
下面结合附图对本申请的两种实施方式作详细说明。
第一种实施方式：
图1至图8示出，一种用于低空飞行器电源的高效轻质换热器，包括风冷散热道1，风冷散热道1的底部设有热换凹区101，且风冷散热道1的两端分别开设为气流入口102和气流出口103，气流入口102和气流出口103的内部分别固定连接有入口导流肋片104和出口导流肋片105，入口导流肋片104与出口导流肋片105之间固定连接有肋片模块，风冷散热道1内部固定连接有柱肋模块，肋片模块与柱肋模块交替对应；
肋片模块包含平直形肋片201和波浪形肋片202，柱肋模块包含圆柱形肋柱301、半圆形肋柱302和半圆三角形肋柱303，平直形肋片201和波浪形肋片202与圆柱形肋柱301、半圆形肋柱302和半圆三角形肋柱303单独组合以及自由组合；
本高效轻质换热器根据使用环境，选择适合的肋片模块与柱肋模块组合型号的风冷散热道1，有效提供更大换热表面积，特别是在柱肋模块之间，流体的速度会减慢并出现流动的不规则性，产生旋涡，这种涡流极大增强了流体的热交换能力，涡流通过增强流体的紊流，使得热量可以更快速地从圆柱表面传递到流体中吸走内部设备产生的热量，同时使气体在流动中带走更多的热量，并通过扰动流体有效提高热交换效率，确保飞行器电源系统在长时间高功率输出情况下有效提供高效散热效果，通过优化结构设计，最大程度减轻了换热器的重量，确保飞行器具有更好的载荷和续航能力。
本发明技术方案的关键在于通过优化风冷散热道1内肋片模块与柱肋模块的组合，以提高换热器的散热效率，减轻飞行器的重量，并确保飞行器电源系统在工作过程中的温控稳定性。
采用平直形肋片201构建风冷散热道1内部换热结构时，平直形肋片201呈直线排列，加工成本低，易于大规模生产，流动阻力小，气流路径平顺，压降较低，适合低流速或自然对流场景，换热效率中等，但湍流效果较弱，热边界层较厚，均匀性高，热量分布均匀，适合对温度一致性要求高的场景。
采用波浪形肋片202构建风冷散热道1内部换热结构时，波浪形肋片202的波浪形结构破坏边界层，有效提高换热系数，有效提高散热效率，相同体积下散热能力更强，适合高热流密度场景，流动阻力大，气流扰动导致压降较高，需更大驱动功率，紧凑设计，在有限空间内实现更高散热性能，如服务器或高功率电子设备，在强制对流环境下，搭配高速风扇或泵，可抵消高阻力劣势，发挥高效散热优势。
圆柱形肋柱301特点：对称结构，气流绕流时边界层分离均匀，产生周期性涡旋，增强湍流混合，换热系数中等，阻力压降较低，流动阻力居中，机械强度圆形截面抗压、抗变形能力较强，适合高振动环境，加工简单，但密集排列时成本较高。
半圆形肋柱302特点：非对称结构，气流在凸面加速、凹面形成涡流，局部换热系数提升，阻力压降略高于圆柱形，安装方向影响性能，空间上可贴合表面布置，适合受限空间内的紧凑设计。
半圆三角形肋柱303特点：尖锐前缘引发强烈边界层分离，形成高湍流度尾流区，换热系数最高，阻力压降显著增大，需更大驱动功率，单位体积换热面积大，适合高热流密度场景，锐角易导致应力集中，冲压或蚀刻工艺要求较高。
第二种实施方式：
相较于第一种实施方式，主要是具体了肋片模块中波浪形肋片202的形式，具体新增结构如下，其余结构与第一种实施方式一致。
图9至图11示出，波浪形肋片202顶部和底部与风冷散热道1的内壁滑动连接，且波浪形肋片202由钛合金材料制成，风冷散热道1与热换凹区101之间矩阵固定分布有温控标体4，温控标体4的顶端与波浪形肋片202的内凹部之间固定连接有连接筋401，连接筋401由形状记忆合金材料制成，温控标体4的中部开设有与热换凹区101连通的活动槽402，温控标体4由气凝胶材料制成，气凝胶材料制成的温控标体4导热率低，实现温控标体4的隔热效果；
在肋片模块采用波浪形肋片202组合时，钛合金材料制成的波浪形肋片202具有优秀的弹性形变能力，通过温控标体4对热换凹区101进行局域性温度监测，即该区域的温度触发连接筋401的热形变能力，连接筋401热形变后推动波浪形肋片202的内凹部达到进一步弯曲的目的，从而增强该区域波浪形肋片202的不规则性，即有效增强流体的紊流扰动能力，局部提高该区域的换热能力，后续在温度降低后，连接筋401温度降低后形状恢复，连带着波浪形肋片202也自动形状恢复，从而有效避免该区域长时间扰流，影响其他区域的流体流通能力。
图10至图11示出，连接筋401的顶部和底部均呈弧形弯曲设置，且连接筋401的顶部和温控标体4的顶部之间包覆有隔温套403，活动槽402与连接筋401的底部弧形端套接，形状记忆合金材料制成的连接筋401以弧形弯曲位置进行弯曲变化，从而利用连接筋401的弯曲幅度的变化推动波浪形肋片202形变，改变波浪形肋片202的局域弯曲幅度，从而有效增强该区域的流体扰动能力，有效提高该区域的热置换能力，隔温套403有效提高连接筋401的隔温能力，从而延长连接筋401的形变状态时间，即延长提高该区域的热置换能力作用时间，活动槽402的下方口固定连接有封堵片404，连接筋401的底部弧形端与封堵片404接触对应，封堵片404由铝合金材料制成，利用封堵片404对活动槽402下方进行封闭，有效避免风冷散热道1内部流体泄漏到热换凹区101内，并且通过连接筋401与封堵片404的接触热传导，便于连接筋401的受热形变，后续在连接筋401形变后，连接筋401与封堵片404断开接触，从而有效降低对连接筋401的散热，从而延长连接筋401的形变时间；
在连接筋401因温度变化而进行弯曲变形时，连接筋401顶部的弧形弯曲幅度缩小，从而实现连接筋401顶部对波浪形肋片202施加横向推力，从而推动波浪形肋片202的内凹部进一步弯曲，达到提高流体流动路径的进一步不规则调整，有效增强流体的紊流扰动能力，局部提高该区域的换热能力，连接筋401底部的弧形弯曲幅度增大，实现连接筋401与封堵片404断开接触，切断连接筋401与封堵片404之间的热传递，同时利用隔温套403对外露的连接筋401进行包裹隔温，实现在该区域进行高效扰流散热时，连接筋401的散热能力相对更低，从而有效提高连接筋401热形变状态的维持时间，有效避免波浪形肋片202弯曲形变时间过短，有效避免影响波浪形肋片202的局域扰流增强能力。
图12至图13示出，波浪形肋片202的下部边缘开设有内容槽5，内容槽5的内部固定镶嵌有导热硅胶带501，导热硅胶带501与风冷散热道1的内壁滑动连接，利用导热硅胶带501有效提高波浪形肋片202与风冷散热道1内壁的热换能力，导热硅胶带501的中部等距固定连接有垫脚片502，垫脚片502的两端与风冷散热道1的内壁固定连接，导热硅胶带501通过垫脚片502与风冷散热道1进行缓冲固定，保留导热硅胶带501一定的活动幅度空间，垫脚片502的两端均设置有拱形缓冲部503，且垫脚片502由弹性塑料制成，垫脚片502通过拱形缓冲部503进行弯曲幅度变化，实现垫脚片502适应导热硅胶带501的移动缓冲，拱形缓冲部503靠近导热硅胶带501的一端底部开设有锁定孔504，锁定孔504的内部插接有锁定螺钉505，锁定螺钉505的底部与风冷散热道1的内壁螺纹连接，且锁定螺钉505的顶部与波浪形肋片202的侧面接触对应，通过锁定螺钉505穿过锁定孔504螺纹连接风冷散热道1，从而固定该区域的波浪形肋片202位移能力，从而为波浪形肋片202的弯曲幅度调整提供受力支点；
