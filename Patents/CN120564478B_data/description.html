Description

面向低空多飞行器混合起降场的协同航迹规划方法及系统
技术领域
本发明涉及飞行器航迹规划，具体涉及面向低空多飞行器混合起降场的协同航迹规划方法及系统。
背景技术
随着低空空域的逐步开放以及无人机等低空飞行器应用的日益广泛，低空多飞行器混合起降场景愈发常见。在低空多飞行器混合起降场中，大量不同类型的飞行器同时起降、飞行，空域环境复杂多变，航迹规划面临诸多严峻挑战。
传统飞行器航迹规划方法多采用人工规划或基于简单规则的自动规划，难以适应低空多飞行器混合起降场景的高动态性和复杂性。一方面，传统方法对空域的划分不够精细，无法准确、实时地反映空域状态，难以满足多飞行器协同运行的需求；另一方面，在航迹规划过程中，对于航迹交叉风险、时空冲突等问题的处理能力有限，不能有效保障飞行器飞行的安全性和高效性。
此外，在面对时空冲突时，缺乏科学合理的冲突消解策略，尤其是对于不同优先级飞行器的避让路径规划不够优化，容易导致低优先级飞行器的避让路径不合理，增加飞行时间和能耗，甚至影响整个低空飞行系统的运行效率。因此，亟需一种面向低空多飞行器混合起降场的协同航迹规划方法及系统，以解决上述问题，保障低空飞行系统安全、有序、高效地运行。
发明内容
针对现有技术所存在的上述缺点，本发明提供了面向低空多飞行器混合起降场的协同航迹规划方法及系统，能够有效克服现有技术所存在的难以对低优先级飞行器的避让路径进行合理规划，以及无法有效保障飞行器在起降阶段的飞行安全的缺陷。
为实现以上目的，本发明通过以下技术方案予以实现：
面向低空多飞行器混合起降场的协同航迹规划方法，包括以下步骤：
S1、基于北斗网格码对空域进行三维立体剖分，并对三维网格单元进行空域状态实时映射；
S2、根据网格状态对三维网格单元进行划分；
S3、构建几何航迹生成规则，消除航迹交叉风险；

S5、构建双队列动态调度机制，通过队列分离和轮询分配引导飞行器安全降落；
其中，根据四维时空冲突检测结果进行冲突消解时，对于低优先级飞行器，采用改进量子启发式路径规划算法进行避让路径规划，引导低优先级飞行器绕行冲突区域。

5）低空复杂场景的强适应性
从空域建模到冲突消解再到落地引导，形成覆盖“规划-检测-调度-执行”的闭环系统，有效应对城市峡谷、GPS信号遮挡等低空场景特有挑战，在建筑物密集区通过多传感器融合导航（如北斗+视觉+UWB）保障定位精度，并且支持百架级飞行器同时起降，满足未来城市空中交通（UAM）的规模化运营需求。

对于执行载人任务的飞行器，为其设定第二优先级权重，允许飞行器穿越可飞行区和缓冲区，并在缓冲区内享有仅次于执行救援任务的飞行器的优先通行权；
对于执行物流任务的飞行器，为其设定第三优先级权重，仅允许飞行器穿越可飞行区，且需避让高优先级权重的飞行器；
对于执行巡检任务的飞行器，为其设定第四优先级权重，仅允许飞行器穿越可飞行区，且需避让高优先级权重的飞行器。
