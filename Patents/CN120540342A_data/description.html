Description

一种面向低空飞行器状态不确定性的风险评估及避障方法
技术领域
本申请涉及飞行器控制技术领域，尤其是一种面向低空飞行器状态不确定性的风险评估及避障方法。
背景技术
随着城市人口持续增长与城市功能不断提升，居民对高效出行方式的需求持续上升，而传统依赖地面道路的交通系统已面临容量瓶颈，难以有效应对高密度出行需求。城市低空空域作为尚未充分开发的新型空间资源，被视为未来城市出行体系的重要补充。然而，与传统高空航空运行环境相比，城市低空空域具有空间狭窄、障碍物密集、飞行密度高等特征，飞行器在运行过程中需频繁应对来自建筑物、其他飞行器、突发气象等多种风险源带来的潜在威胁。

具体实施方式
为了使本申请的目的、技术方案及优点更加清楚明白，以下结合附图及实施例，对本申请进行进一步详细说明。应当理解，此处所描述的具体实施例仅仅用以解释本申请，并不用于限定本申请。
在对本申请实施例进行详细说明之前，首先对本申请实施例中涉及的部分相关技术进行说明，如下：
鉴于现有技术存在的问题，如何实现对复杂空域中潜在风险的准确建模和实时感知，并将其有效融入飞行路径规划和避障策略中，是保障城市低空交通系统安全运行的关键问题。构建动态响应空域环境变化的风险评估机制，能够为飞行器提供更具安全性的路径决策支持，从源头上降低冲突和碰撞的概率，为城市低空交通的可持续运营打下基础。
目前已有多种风险评估与避障方法被提出，如A*、RRT、几何避障法、速度障碍法等，虽在结构化或静态环境中具有效果，但通常需要较高的计算资源，难以适应实时动态复杂的低空环境。此外，飞行器在实际运行中存在误差的影响，导致其观测状态（速度、位置）与真实状态出现偏差，进而偏离规划路径，进入潜在高风险区域。因此，面向动态、不确定的飞行环境，传统方法在安全性和鲁棒性方面存在不足，难以满足城市低空空域风险评估对实时性与可靠性的双重要求。
为提升低空飞行器在复杂环境中的运行安全性，现有的城市低空空域避障与风险评估研究已经提出了多种路径规划与风险建模方法。常见的路径规划方法包括A*、Dijkstra、D*、快速随机树（RRT）、RRT*等图搜索类算法及几何避障方法等改进版本。这些方法在结构化或静态环境中具有效果显著、易于实现等优点，尤其是在已知地图或可预见静态障碍物环境下，能够规划出有效避障路径。然而，这些传统方法通常面临两个主要问题：一方面，路径规划过程对计算资源要求较高，在面对动态、非结构化的城市低空空域时，难以满足实时性的需求；另一方面，这些方法多数是基于飞行器状态确定性假设进行建模和规划，忽略了飞行器在实际运行过程中由于传感误差、风扰、环境干扰等带来的状态不确定性，进而影响风险评估的可靠性与避障的合理性。
针对低空运行中的实时飞行器交互风险建模问题，近年来一些研究引入了“风险场”或“势能场”模型，从场论视角刻画空域中潜在威胁源（如障碍物与其他飞行器等）对飞行器运行安全的影响。这一类方法通常受到人工势场理论的启发，将风险源抽象为排斥源，目标点视为吸引源，在空间中形成分布式的势场或风险势场，飞行器根据在场中的受力调整自身路径，实现避障或趋向目标。例如，有学者构建了结合动态障碍物的三维风险场模型，考虑飞行器与障碍物间距离信息；也有研究尝试使用多源融合策略，将静态障碍、动态障碍与空域规则共同纳入势场建模框架中，以反映更真实的风险分布态势。这些模型在一定程度上克服了传统避障算法难以处理连续空间复杂分布问题的缺陷，为飞行器避障提供了直观且连续的风险信息。
然而，现有风险场模型仍存在一定缺陷，影响飞行器在复杂动态空域中的风险评估精度。首先，现有风险场模型在构建过程中假设飞行器的状态为完全可观测且确定的，即飞行器的实际位置与观测位置高度一致。这一假设在理想环境下较为合理，但在实际城市低空空域运行中，飞行器状态往往受到多种因素影响而产生不确定性偏移。如GNSS等导航系统存在精度限制，易受城市高楼、天气等因素干扰，导致位置估计误差；另一方面，飞行器在飞行过程中容易受到气流扰动、执行系统误差等外部扰动因素的影响，使其位置或速度可能发生轻微但不可忽略的偏移。若风险评估模型未考虑这种不确定性，则可能导致风险评估结果与实际情况存在显著偏差，从而产生两种风险：一是飞行器在被判定为安全区域内运行，但由于实际偏移而进入高风险区域；二是系统对部分区域风险高估，限制了飞行器的路径规划灵活性，降低了空域利用效率。此外，一些风险场模型在场强设计过程中考虑因素较为有限，普遍基于飞行器与障碍物之间的距离计算排斥力场，忽视了诸如飞行器速度、类型、质量、运动方向等动态特征的影响，难以准确反映空域中飞行器交互风险。
相关术语解释：
（一）风险场模型。
风险场是本申请中用于表示空间风险分布的核心模型。其本质是一种基于势场思想的空间映射函数，用于描述障碍物或其他飞行器在其周围区域所产生的风险影响。风险场的强度通常与飞行器与障碍物之间的空间距离有关，距离越近风险越高。此外，本技术中的风险场还引入了障碍物的速度和质量等动态属性，使得风险场具有方向性和动态感知能力，能够更真实地反映障碍物对飞行器的潜在威胁。
（二）飞行器状态偏移概率建模。
在实际运行过程中，飞行器受到多种干扰因素影响，如卫星定位误差、传感器噪声、风扰动等，导致其真实状态与观测状态之间存在差异。为了刻画这种偏移特性，本申请采用概率建模方式，分别对飞行器的位置和速度建立独立的概率分布模型。通过这种方式，飞行器的状态不再被视为确定值，而是服从一定的概率分布。此外，这种“包围式建模”思想，不追求对实际误差分布的精确拟合，而是采用相对更保守的建模方式，用于覆盖飞行器可能出现的所有区域，从而提升系统对误差的容忍性和建模的安全冗余。
（三）风险概率场的构建思路。
传统的风险势场仅考虑飞行器当前位置与障碍物的空间关系，难以反映其在不同可能状态下所面临的整体风险。本申请基于飞行器状态的不确定性分布，提出构建风险概率场的方法，即通过对飞行器可能出现的各个位置及速度所感知到的风险场强进行加权整合，形成一个更具合理性的风险空间图谱，从而实现更为安全、主动的避障控制策略。
（四）排斥力建模与导航引导机制。
风险概率场构建完成后，本申请通过风险场强与飞行器当前状态之间的关系，计算飞行器受到的排斥力，用于引导其避开高风险区域。与此同时，飞行器也受到来自目标点的吸引力。两种力场共同作用于飞行器动力学模型，驱动其完成避障与导航控制。相比于基于硬约束或启发式规划的传统方法，本技术通过势场力引导实现了更加自然、连续的路径调整，且具备良好的响应性。
（五）整体建模与系统实现流程。
本申请流程自下而上依次包括：飞行器状态不确定性建模、风险场建模、风险概率场生成、排斥与吸引力计算，构成一个可实时迭代的闭环控制体系。在运行过程中，系统根据飞行器当前观测状态生成其概率分布，并据此评估其在空域中的风险程度，从而驱动避障路径的动态调整。该方法适用于城市低空、物流航线、密集交通等多种应用场景，尤其在存在大量不确定性与干扰因素的环境下，具备较强的鲁棒性与实用性。

为了克服现有技术中未能充分考虑飞行器状态不确定性及风险场建模不完善的问题，本实施例提出了一种面向低空飞行器状态不确定性的风险评估及避障方案。该方案构建了融合飞行器状态不确定性的风险概率场建模框架，首先，基于包围高斯分布的偏移模型描述飞行器在观测误差影响下的实际运行偏差，作为风险场构建的重要基础。其次，进一步考虑飞行器的速度、航向、质量等动态属性对风险场强的影响，提出基于飞行器运行状态的动态场强设计方法，提升模型在动态环境下的适应性。最后，融合所构建的飞行器状态概率分布与动态风险场模型，设计了一种适用于低空飞行器的动态飞行器风险概率场，为并飞行器避障提供决策支持，有效提升低空飞行的安全性，基于飞行器状态不确定性的风险概率场模型避障流程如图2所示。
具体地，本实施例可以包括以下技术方案：
1、飞行器状态不确定性分布模型构建。
在实际低空飞行器的运行过程中，其观测位置不可避免地受到多种误差源的干扰，例如GNSS定位误差、传感器噪声、环境遮挡、信号多径效应等。这些误差将导致飞行器的实际运行状态（位置、速度）与其观测状态之间产生偏差。为了准确评估飞行器在空域中所处状态的安全性，本实施例建立飞行器运行状态的不确定性模型，以此为基础构建风险概率场。
该模型主要考虑飞行器的三维位置与速度的动态状态特征，结合飞行器运动的不确定性，基于飞行器状态偏差分布建模推测其在空域中可能出现的状态分布。但构建高精度的飞行器偏移概率分布模型通常需要大量历史观测数据，高昂的计算成本高难以满足实际系统中对实时性和泛化能力的要求。因此，本实施例采用包围高斯分布的思想对飞行器状态偏差概率进行建模，构建一个方差更大、覆盖范围更广的高斯分布，用于“包围”飞行器在一定时段内可能出现的状态区域，如图3所示。这种方法不仅有效简化建模过程，而且显著降低对历史数据规模和质量的依赖。在包围策略下，即使真实偏差分布呈现非高斯形态，仍能通过一个保守的高斯模型对其进行近似包容，确保风险评估的安全性与鲁棒性。

综上，本实施例包括以下技术手段：
（1）融合飞行器状态不确定性的偏移分布建模机制：通过构建基于扰动影响与定位误差的概率偏移模型，量化飞行器在实际运行中与其规划路径之间的偏离程度，为风险评估提供基础支持。
（2）面向飞行器运行风险的动态风险场建模方法：基于人工势场理论，考虑飞行器速度、航向、质量等动态属性的风险场强函数，提高对飞行器状态变化的响应能力。
（3）融合不确定性建模的风险概率场建构方法：在风险场的基础上融合飞行器偏移概率分布，将各类风险源的观测偏差转化为概率表达，形成风险概率场模型，实现对潜在区域内飞行器运行状态不确定性的概率化刻画，提升对飞行器在空域运行风险的评估能力。
（4）基于风险场引导的低空飞行器避障路径规划策略：基于风险概率场模型，设计一套具备实时计算能力的飞行器避障方法，在动态空域环境基于风险实时引导飞行器执行避障操作。
本实施例的有益效果包括：
（一）综合考虑飞行器状态不确定性，建模更加精确。
本实施例通过引入飞行器状态不确定性与风险场建模，弥补了现有技术中未充分考虑飞行器观测误差的问题。传统的风险场模型通常假设飞行器观测状态精确无误差，采用静态或简化的风险评估方法。然而，在复杂动态低空环境下，飞行器不仅受到各种障碍物和其他飞行器的影响，还会因定位误差、外部气象干扰以及动态路径规划中的不可预见因素而发生状态偏移，这些因素对基于飞行器实时状态及其周围空域的风险评估产生显著影响。
针对这一问题，本实施例在一般风险场的基础上，创新性地提出了基于飞行器状态偏移的风险概率场，能够动态地对飞行器的定位误差进行建模，通过计算误差分布，建立飞行器状态的不确定性模型。这一创新使得风险评估不仅考虑了飞行器的当前状态，还考虑了其误差的分布，从而可以更为准确地分析飞行器在复杂空域环境中与其他飞行器的交互风险，避免了传统方法中因忽略飞行器不确定性带来的风险评估偏差。
（二）实时动态风险场设计，提升避障安全性。
本实施例的风险场设计进一步考虑了飞行器的动态属性，如速度与航向，使得飞行器与空域中其他因素的相互作用关系得以更精确建模。相比于传统静态模型，动态风险场能够根据飞行器状态的实时变化动态更新风险分布，更贴合实际飞行环境，有效提升了风险评估的准确性和及时性。能帮助飞行器提前识别潜在风险，实现更具前瞻性的避障策略。
此外，本实施例还设计了飞行器在场中的受力模型，考虑飞行器在复杂环境中所受的外力，并利用这些受力信息引导飞行器进行避障。通过模拟飞行器所受外部风险场的影响，引导飞行器依据受力变化做出反应，从而实现避障，减少潜在的碰撞风险。这一设计不仅提升了飞行器在复杂动态环境中的避障能力，使其能够在各种外部影响下安全飞行。与传统静态模型相比，本实施例能够更有效地适应飞行环境中的突发变化，提供更加安全、可靠的避障方案。
