Description

低空电缆的遥控巡检飞行器
技术领域
本实用新型属于电缆巡检设备技术领域，更具体地说，特别涉及低空电缆的遥控巡检飞行器。
背景技术
低空电缆的安全运维关系到供电系统是否平稳安全，在低空电缆的使用中，需要对低空电缆的运行情况进行监视，现有的低空电缆巡检方式一般为巡检人员使用遥控无人机对低空电缆进行巡检，现有的低空电缆遥控巡检飞行器一般包括：无人机体、遥控器、摄像头等。
现有申请号为：CN201521037111.6本实用新型公开了一种带电流互感器的配电线路巡检装置，包括飞行器，飞行器的机身内设有方向舵伺服机、升降伺服机、螺旋桨马达及电池。机身内设有电流互感器和控制器。电流互感器连有抓握钳，抓握钳作为电流互感器的一次侧用于抓握被巡检配电线路的电缆，抓握钳设置在机身的端部。控制器设有无线通讯模块，无线通讯模块用于与遥控器通讯，控制器分别连接方向舵伺服机、升降伺服机、螺旋桨马达和电流互感器。通过遥控器的操作手柄控制飞行器启动、升降、转向、悬停在配电线路的电缆处，并控制抓握钳抓握电缆，测得电流值，并通过无线通讯模块传递到遥控器的显示屏显示。方便巡检人员检测配电线路电流。
基于上述，现有的低空电缆的遥控巡检飞行器在使用中，当电缆外表存在破损时，一般直接向控制中心发出信号，控制中心派维修人员到目标位置进行维修，但是一般巡检飞行器只能对目标电缆的大致位置进行标记，导致维修人员需要再次寻找破损点，检修效率较低，而且在等待维修期间，破损点容易被电缆线的电流击穿，导致经济损失。
实用新型内容
为了解决上述技术问题，本实用新型提供低空电缆的遥控巡检飞行器，以解决现有的低空电缆的遥控巡检飞行器在使用中，当电缆外表存在破损时，一般直接向控制中心发出信号，控制中心派维修人员到目标位置进行维修，但是一般巡检飞行器只能对目标电缆的大致位置进行标记，导致维修人员需要再次寻找破损点，检修效率较低，而且在等待维修期间，破损点容易被电缆线的电流击穿，导致经济损失的问题。
本实用新型低空电缆的遥控巡检飞行器的目的与功效，由以下具体技术手段所达成：
低空电缆的遥控巡检飞行器，包括无人机体、支撑底板、摄像头、支撑固定柱、清洁固定箱和巡检标记机构；
所述支撑底板共设置两组，两组支撑底板均固定连接在无人机体的下方；
所述摄像头设置在无人机体的下方；
所述支撑固定柱共设置四组，四组支撑固定柱分别为圆柱状结构，支撑固定柱均固定连接在支撑底板的下方；
所述清洁固定箱固定连接在支撑底板的下方，并且清洁固定箱下端设置有长方形槽结构；
所述巡检标记机构固定连接在无人机体的下方。
进一步的，所述支撑固定柱包括有：支撑滑动件、吸盘底座和缓冲弹簧；
所述支撑滑动件共设置四组，四组支撑滑动件分别为圆筒状结构，支撑滑动件均滑动连接在支撑固定柱的下方；
所述吸盘底座共设置四组，四组吸盘底座分别固定连接在支撑滑动件的下方，并且吸盘底座分别为吸盘结构；
所述缓冲弹簧共设置四组，四组缓冲弹簧的上端均与支撑固定柱的内部固定连接，四组缓冲弹簧的下端均与支撑滑动件的内部固定连接。
进一步的，所述清洁固定箱包括有：清洁驱动电机、清洁驱动丝杠、清洁滑块和清洁板；
所述清洁驱动电机固定连接在清洁固定箱的左侧；
所述清洁驱动丝杠转动连接在清洁固定箱的内部，并且清洁驱动丝杠的左端与清洁驱动电机的转轴同轴固定连接，清洁驱动丝杠的左右两端分别设置有不同旋向的螺纹；
所述清洁滑块共设置两组，两组清洁滑块分别螺纹连接在清洁驱动丝杠的外周；
所述清洁板共设置两组，两组清洁板分别为半圆环状结构，清洁板分别固定连接在清洁滑块的下方。
进一步的，所述清洁固定箱还包括有：柔性清洁件；
所述柔性清洁件共设置两组，两组柔性清洁件分别固定连接在清洁板的内周，并且柔性清洁件分别为半圆环橡胶结构。
进一步的，所述巡检标记机构包括有：标记驱动箱、标记驱动电机、标记限位板、标记丝杠和标记滑板；
所述标记驱动箱固定连接在无人机体的下方；
所述标记驱动电机固定连接在标记驱动箱的内部上端；
所述标记限位板共设置两组，两组标记限位板分别固定连接在标记驱动箱的内部前后两侧，并且两组标记限位板分别设置有长方形槽结构；
所述标记丝杠转动连接在标记驱动箱的内部，并且标记丝杠的上端与标记驱动电机的转轴同轴固定连接；
所述标记滑板螺纹连接在标记丝杠的外周，并且标记滑板的前后两端分别滑动连接在标记限位板的长方形槽内部。
进一步的，所述巡检标记机构还包括有：标记环和固定球；
所述标记环设置有多组，多组标记环均滑动连接在标记驱动箱的内部，并且多组标记环的作用两侧下端分别设置有圆形凹槽结构，标记环均设置在标记滑板的下端，并且标记环均为半圆环状塑料结构；
所述固定球共设置四组，四组固定球分别固定连接在标记驱动箱的内部。
与现有技术相比，本实用新型具有如下有益效果：
本实用新型通过支撑滑动件的设置，当整个飞行器降落时，此时吸盘底座可以使无人机体与地面的接触更加稳定，同时缓冲弹簧实现了对无人机体的缓冲功能，实现了对无人机体的缓冲保护。
本实用新型通过清洁板的设置，当进行低空电缆的巡检时，此时将无人机体遥控到电缆的上方，然后打开清洁驱动电机，清洁驱动电机转动带动清洁驱动丝杠转动，清洁驱动丝杠转动带动清洁滑块对向移动，清洁滑块对向移动带动清洁板对向移动，此时清洁板对向移动将电缆线包绕，待无人机体飞行巡检时，两组清洁板实现了对电缆线表面的清洁维护，延长了电缆线的使用寿命。
本实用新型通过巡检标记机构的设置，当无人机体的操作者发现电缆线有破损的位置时，此时操作者可以通过打开标记驱动电机，标记驱动电机转动带动标记丝杠转动，标记丝杠转动带动标记滑板向下移动，标记滑板向下移动带动标记环向下移动，标记环向下移动卡接到破损的电缆线的位置，实现了对电缆破损处的临时补漏并标记了电缆破损点，方便了后续维修人员寻找电缆线破损点，加快了维修的效率。
本实用新型实现了对无人机体的缓冲保护，实现了对电缆线表面的清洁维护，延长了电缆线的使用寿命，实现了对电缆破损处的临时补漏并标记了电缆破损点，方便了维修人员寻找电缆线破损点，提高了电缆维修的效率。
