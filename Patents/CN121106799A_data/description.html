Description

一种具备高飞行稳定性的低空飞行器
技术领域
本发明涉及低空飞行技术领域，具体为一种具备高飞行稳定性的低空飞行器。
背景技术
当前低空飞行器技术虽日趋成熟且应用广泛，但民用低空飞行器在复杂流场环境下的飞行适应性存在显著不足，其存在以下技术问题：
升力稳定性、均匀性差：以悬停为核心特点的旋翼式低空飞行器，在复杂流场中，其飞行速度与流场速度大小量级相近，然而流场速度矢量的大小和方向在时间和空间上分布紊乱，导致旋翼难以维持设计状态的稳定、均匀的来流流场，极易出现旋翼前沿攻角，从而导致旋翼表面的流动分离。而且这种流动分离随时间和空间变化复杂，这不仅造成旋翼升力合力大幅下降、升力沿展向分布复杂，更关键的是升力合力方向多变，使得铅锤方向分力无法克服低空飞行器自身及负载重量，直接威胁飞行高度维持。
飞行控制难度大：升力的不稳定和不均匀会使低空飞行器处于固有振动状态和不利应力状态，大幅增加飞行控制难度，严重恶化稳定飞行能力，极端情况下会导致低空飞行器失控，引发翻覆坠落事故。
部件寿命缩短：复杂流场引发的振动和不利应力，会加速旋翼等关键部件的损耗，显著降低其使用寿命。
应用场景受限：复杂流场在实际应用中频繁出现，如强对流天气、高速公路隧道、有风流过的高层建筑附近、山岭地带强日光照射形成的风场等，而现有技术缺乏有效应对手段，只能被动避开这些场景，极大限制了低空飞行器的服务范围和应用能力。
发明内容
针对现有技术中存在的问题，本发明的目的是：提供一种具备高飞行稳定性的低空飞行器，升力稳定性好，飞行控制简单，扩展了应用场景。
为了达到上述目的，本发明采用如下技术方案：
一种具备高飞行稳定性的低空飞行器，包括轴流涵道风扇阵列、负载置物托盘、连接件和分段圆弧连接环；
轴流涵道风扇阵列包括多个可单独调节转速的轴流涵道风扇，多个轴流涵道风扇均匀布置在同一圆周上，且该圆周的直径远大于轴流涵道风扇的直径；
所有轴流涵道风扇下游均配置有相同的收缩型导流风道，收缩型导流风道的出风口朝下，相邻的轴流涵道风扇通过位于圆周上的分段圆弧连接环连接；
负载置物托盘安装在轴流涵道风扇阵列所在圆周的中心位置，负载置物托盘与轴流涵道风扇之间通过周向均布的连接件连接。
进一步地，轴流涵道风扇阵列所在的圆周直径至少为轴流涵道风扇直径的五倍。
进一步地，轴流涵道风扇阵列所在的圆周被等分为四个象限，每个象限的圆周上布置有多个同等数量的轴流涵道风扇，且轴流涵道风扇的总数量为四的倍数。
进一步地，每个象限内的轴流涵道风扇旋向一致，相邻象限内的轴流涵道风扇旋向相反。
进一步地，连接件向圆周中心方向弯折突起，负载置物托盘的位置高于分段圆弧连接环的位置。
进一步地，连接件的长度可调，以适配不同直径的负载置物托盘。
进一步地，负载置物托盘用于放置低空飞行器的常用设备。
进一步地，还包括电源和控制系统，电源用于为轴流涵道风扇及控制系统提供电能，控制系统用于调节轴流涵道风扇的运行状态。
总的说来，本发明具有如下优点：
针对升力稳定性差的问题，本发明通过在大直径的圆周上均匀布置轴流涵道风扇并在轴流涵道风扇下游配置收缩型导流风道，一方面，收缩型导流风道可提升轴流涵道风扇的托举力；另一方面对轴流涵道风扇内部轴向贯通气流流场进行规整，使得轴流涵道风扇轴向气流流速可以维持在三十米每秒以上的水平（收缩型导流风道内部的速度比起轴流涵道风扇轴向气流流速更高一些），远高于轴流涵道风扇外部的复杂流场的气流速度大小，这样形成的气流相对于轴流涵道风扇叶片的流动有利于抑制轴流涵道风扇外部复杂气流紊乱导致的对风扇叶片前沿的直接不稳定、不均匀有显著攻角的冲击（类似于整流的作用），降低风扇叶片表面流动分离的概率和范围；最后，多个均匀分布的轴流涵道风扇形成阵列式升力源，即使部分风扇因流场紊乱出现明显升力波动，其余风扇仍能提供相对稳定升力补充，避免整体升力合力大幅下降或方向剧烈变化，保障铅锤方向升力分力足以克服机身及负载重量；
针对飞行控制难度大的问题，本发明通过将轴流涵道风扇均匀布置在大直径圆周上，使低空飞行器的质量尽可能分布在圆周外侧，质量分布半径增大可显著提升系统转动惯量；同时，大直径圆周使各轴流涵道风扇在高度紊乱流场中处于不同流场区域，流场尺度差异导致各风扇所受作用力矩关联度降低，力矩相互抵消效应增强；转动惯量提升与力矩抵消增强的共同作用，可大幅降低低空飞行器的转角加速度，避免因转角过大引发倾覆坠落；
本飞行器在水平面内的正常运动则是利用航空技术中熟知的适用于多升力源飞行器的速差推力原理完成的。
本发明的结构设计使低空飞行器在强对流天气、高速公路隧道、高层建筑附近及山岭强日光风场等高度紊乱流场中，仍能维持稳定升力与姿态，突破现有低空飞行器被动避开紊乱流场的限制，可在更多场景中执行飞行任务，拓展应用范围。

具体实施方式
下面来对本发明做进一步详细的说明。
如图1所示，一种具备高飞行稳定性的低空飞行器，包括轴流涵道风扇阵列、负载置物托盘1、连接件4和分段圆弧连接环2；
轴流涵道风扇阵列包括多个可单独调节的轴流涵道风扇3，多个轴流涵道风扇3均匀布置在同一圆周上，且该圆周的直径远大于轴流涵道风扇3的直径；优选地，轴流涵道风扇阵列所在的圆周直径至少为轴流涵道风扇3直径的五倍。
所有轴流涵道风扇3均配置有相同的出风口向下的收缩型导流风道，相邻的轴流涵道风扇3通过位于圆周上的分段圆弧连接环2连接；
负载置物托盘1安装在轴流涵道风扇阵列所在圆周的中心位置，负载置物托盘1与轴流涵道风扇3之间通过周向均布的连接件4连接。
进一步地，轴流涵道风扇阵列所在的圆周被等分为四个象限，每个象限的圆周上布置有多个同等数量的轴流涵道风扇3，且轴流涵道风扇3的总数量为四的倍数。
进一步地，每个象限内的轴流涵道风扇3旋向一致，相邻象限内的轴流涵道风扇3旋向相反。
进一步地，连接件4向圆周中心方向弯折突起，负载置物托盘1的高度高于分段圆弧连接环2的高度。通过弯折突起连接件4将负载置物托盘1置于分段圆弧连接环2上方一定高度，避免负载直接接触紊乱气流或被风扇气流干扰，延缓关键部件的损耗，延长使用寿命，同时负载集中在中央位置，不破坏机身质量圆周分布的均衡性，保障转动惯量设计效果，进一步维持飞行稳定性。
进一步地，连接件4的长度可调，以适配不同直径的负载置物托盘1。
进一步地，负载置物托盘1用于放置低空飞行器的常用设备。
进一步地，还包括电源和控制系统，电源用于为轴流涵道风扇3及控制系统提供电能，控制系统用于调节轴流涵道风扇3的运行状态。

对于复杂流场中的物体而言，它各部分受到的力无论大小还是方向都有一个复杂的空间分布，这种复杂的空间分布随着空间距离的增加相互之间的对于物体的力矩的抵消越强烈，因此低空飞行器的轴流涵道风扇阵列所在圆周的直径越大，飞行器整体所受到的力矩的矢量和相对越小。同时当圆周的直径越大的时候，相应的转动质量m到转轴的距离r也越大，而且转动惯量大小和距离r的平方成正比。这两个效果共同作用就导致，当圆周的直径增加的时候，本发明的低空飞行器的转动角加速度随着圆周直径的平方显著降低，由此实现了本发明的低空飞行器的高飞行稳定性，减轻了低空飞行器发生倾覆坠落的风险。
当本发明的低空飞行器起飞时，在初始状态下，每个轴流涵道风扇3以相同转速旋转，这样可以适应低速空气流场的情况。这时候，飞行器以几乎水平的姿态向上（轴流涵道风扇3的总托举力大于飞行器的自重和负载）/向下（轴流涵道风扇3的总托举力小于飞行器的自重和负载）飞行。当进入复杂流场中的时候，就会有倾斜和升降运动出现。由于本发明飞行器具有保持状态稳定的能力，它能够自我消化流场中一定强度的剪切流动。对于本发明最不利的情形是强烈的上下剪切流动，而当流场中出现了这样持续的强烈剪切流动时，该飞行器会有一定的一侧倾斜的现象发生，此时由于轴流涵道风扇3的推力方向从重力的铅锤方向的偏离，该飞行器会自动出现远离强烈剪切流动区域的趋势，也就是说，飞行器会从强风区被吹走，进入到弱风区，从而恢复稳定的飞行姿态。水平面上的剪切流动对于该飞行器的影响相对弱一些，因为轴流涵道风扇3本身内部的沿着轴向的贯穿速度都有三十多米每秒以上，远远高于水平方向的外部气流流场中的速度，这样有利于抑制轴流涵道风扇3内部的分离流动的产生，从而使得轴流涵道风扇3维持较高的运行效能，即产生足够的托举力，来保证该飞行器的飞行状态。
作为一种被动控制理念的设计，本发明的低空飞行器不需要额外的控制机构及相应的能量消耗，故而它实现的高稳定性飞行方法原理清晰、结构简单而且效果显著可靠，并且生态友好、节能降噪、广谱适用、简单易行因而经济高效。
应用实例
应用例一：一定强度的强对流天气中。本发明的飞行器因为具备高稳定飞行能力，而可以协助展开区域救灾、要点照明、线路巡察等任务，使得人们的正常生活不被恶劣气象条件破坏。
应用例二：高速公路汽车隧道里。来往络绎不绝的高速车流在隧道的狭窄空间中形成强剪切流流场，同时伴有通风系统的分布式气流造成局部复杂流动区域，对飞行器的高稳定性飞行能力提出了挑战，这里正有本发明的低空飞行器的用武之地。
应用例三：有一定风速下高层建筑附近。在高层建筑附近，当有一定风速时，无论是大楼顶部，还是正面、背面和两个侧面，由于气流通道的截面变化，有的地方会有加速流动，有的地方会有减速流动，有的地方会有漩涡流动，飞行器在这种区域飞行，就是不断地穿越各种不同的流场状态，本发明的飞行器也可以在此大显身手。
应用例四：日光强烈照射下的山区风场里。在山区里，当日光强烈照射时，地表温度急剧上升，引起地表附近空气密度显著下降，从而由于密度差驱动上升气流流场，进而从半空中吸引高密度空气冲刷下来，也会形成上升-下降交错的空间复杂流场，本发明的飞行器正好可以在此发挥作用。
本发明的低空飞行器具备显著的高稳定性飞行能力，它的出现拓展了低空飞行器对于飞行区域流场复杂程度的适应性，而且它利用了成熟的轴流涵道风扇3设备，通过选定适宜的型号和数量，利用简单的构造实现了高稳定飞行能力，实施起来方便简洁，有着极为广阔的应用空间。
