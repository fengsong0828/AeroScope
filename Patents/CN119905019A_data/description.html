Description

一种低空飞行器空中交通管理方法及系统
技术领域
本发明涉及飞行器空中交通管理技术领域，具体涉及一种低空飞行器空中交通管理方法及系统。
背景技术
低空飞行器空中交通管理是指对在低空空域内(通常指地面至1000米高度范围内)飞行的各种飞行器(如无人机、空中出租车、飞行器等)进行协调、调度、监控和管控的技术和管理方法。随着低空飞行器的使用越来越广泛，特别是在物流、快递、农业、无人机表演等领域，低空飞行器的数量激增，空中交通流量和复杂性也随之增加。因此，低空空域的安全管理和高效调度显得尤为重要。低空飞行器空中交通管理系统通常涉及飞行器的实时定位、飞行路径规划、碰撞预防、空域划分、飞行流量控制以及与其他飞行器和地面控制中心的通信协调等环节。其核心目标是确保在低空空域内，飞行器能够安全、有序地飞行，避免发生碰撞、干扰或其他安全隐患，并最大化空域的利用效率，尤其在无人机表演等场景中，需精确协同多个飞行器，确保表演的顺利与安全进行。
现有技术存在以下不足：
现有技术中，无人机群组在飞行表演时，空中交通管理系统通常依据风速数据预测未来的风速变化，并据此设定固定的飞行速度。然而，由于不同区域的风速存在差异，风速变化可能导致无人机的飞行轨迹偏离，尤其在群组飞行时，部分无人机可能受到不同程度的风速影响。固定的飞行速度无法适应这些变化，因此未能及时调整速度和路径会带来严重后果。
在开放空域中，不同高度和位置的风速差异常见，某些区域可能突发阵风或气流湍流，导致个别无人机的飞行轨迹发生偏离。如果群组中的无人机无法根据风速变化及时调整飞行状态，可能导致群体间相对距离的打破和队形错乱，进而增加碰撞的风险。特别是在群组飞行速度未能适应局部风速时，整体稳定性下降，飞行器无法保持安全间距。若某架无人机因风速变化偏离轨迹，可能与邻近飞行器发生碰撞，造成损害或导致失控。
在所述背景技术部分公开的上述信息仅用于加强对本公开的背景的理解，因此它可以包括不构成对本领域普通技术人员已知的现有技术的信息。
发明内容
本发明的目的是提供一种低空飞行器空中交通管理方法及系统，通过实时监控无人机飞行状态并结合风速、气流等外部扰动，能动态调整偏离轨迹的无人机路径与速度，有效防止因风速变化导致的轨迹偏离，从而保持飞行队形稳定，避免相对距离过近或错乱，减少碰撞风险。同时，深度学习模型对飞行一致性偏差进行智能化评估，识别偏离轨迹的无人机，并在偏差超过阈值时及时调整路径，确保飞行安全，防止因轨迹偏差引发的碰撞，保障群体飞行的稳定与安全，以解决上述背景技术中的问题。
为了实现上述目的，本发明提供如下技术方案：一种低空飞行器空中交通管理方法，包括以下步骤：
在飞行表演开始前，交通管理系统根据预设的飞行计划和表演需求，制定每架无人机的飞行路径、速度、航向和队形；
在飞行表演过程中，实时监测每架无人机的飞行状态参数，为后续的飞行数据分析和调整提供基础数据；
对实时获取的飞行数据进行预处理，确保后续特征提取和分析的准确性和可靠性；
从预处理后的飞行数据中提取出反映当前无人机偏离预定轨迹的关键特征，在检测窗口下，对提取的关键特征进行分析，量化无人机群组的飞行一致性，评估整体飞行状态是否保持预定的队形和协调性；
将量化后的飞行一致性特征输入预先训练好的深度学习模型，智能化评估每架无人机的飞行状态变化；
根据深度学习模型的评估结果，将当前无人机的飞行状态划分为预定轨迹飞行和偏离轨迹飞行；
对于处于预定轨迹飞行状态的无人机，继续以预设的飞行速度进行表演，确保其稳定飞行；
针对被评估为偏离轨迹飞行的无人机，基于深度学习模型的预测结果，动态调整其飞行速度，使其重新回归预定轨迹，避免碰撞事故的发生。

在上述技术方案中，本发明提供的技术效果和优点：
本发明通过实时监控无人机的飞行状态并结合风速数据、气流湍流等外部扰动，能够动态调整偏离轨迹的无人机的飞行路径和速度。这种调整能有效地防止无人机因无法适应风速变化而导致的轨迹偏离，从而避免群体飞行中的相对距离过近或队形错乱现象。通过精确控制无人机的飞行状态，能够保持群组的稳定性，确保各个无人机之间保持安全间距，显著降低因飞行轨迹偏差引发的碰撞风险。
本发明通过深度学习模型对飞行一致性偏差进行智能化评估，可以快速识别出偏离预定轨迹的无人机，并基于偏差调整飞行参数。当飞行一致性偏差指数超过设定阈值时，自动识别出潜在的碰撞风险，并对偏离轨迹的无人机进行精准的路径调整。这一机制确保了及时发现飞行中的异常行为，防止无人机之间因为偏离轨迹而产生碰撞风险，从而保障群体飞行的安全。

通过实时获取无人机的飞行数据并与预设的轨迹进行比较，不断更新飞行一致性偏差指数FlightConsistency，并基于反馈结果修正飞行器的状态，避免因系统滞后而导致调整过度或者调整不足，采用反馈控制策略，确保调整过程中的平滑过渡，公式如下：

这个反馈机制确保了无人机在回归预定轨迹时的精度和稳定性，避免过度调整导致的飞行不稳定，确保无人机逐步回归并稳定在预定轨迹上，减少碰撞风险。
飞行速度调整权重因子αadjust是一个决定飞行速度调整对整体反馈修正值(Ffeedback)贡献的比例系数。它表明在飞行速度与航向角度调整中，飞行速度的调整在修正飞行轨迹时的相对重要性。当αadjust的值较高时，系统会更加依赖飞行速度的调整来实现轨迹修正，特别是在风速变化较大或无人机飞行速度波动较显著时，这样可以通过快速改变飞行速度来迅速恢复到预定轨迹。反之，较低的αadjust表示飞行速度的调整对反馈修正的影响较小，更多地依赖航向角度的修正。
航向角度调整权重因子βadjust决定了航向角度调整对整体反馈修正值Ffeedback)的贡献程度。它反映了在轨迹修正过程中，航向的调整相对于飞行速度的调整的重要性。当βadjust较大时，系统倾向于通过调整航向角度来回归预定轨迹，特别是在飞行路径发生较大弯曲时，系统会优先考虑改变航向角度以恢复飞行方向。较小的βadjust表示飞行器更依赖于飞行速度的调整来纠正轨迹，航向角度的调整作用相对较弱。
针对被评估为偏离轨迹飞行的无人机，基于深度学习模型的预测结果动态调整飞行速度，主要作用是通过精确的速度调控使无人机重新回归预定轨迹，从而避免由于飞行轨迹偏离导致的碰撞事故。无人机在复杂的飞行环境中，可能会受到风速变化、气流扰动、航向调整等多种因素的影响，导致飞行路径发生偏移。当飞行一致性偏差指数FlightConsistency超过预定的参考阈值时，识别出无人机的飞行状态发生异常，需立即采取措施进行修正。此时，基于深度学习模型的预测结果，可以根据飞行器当前的状态信息(如航向角、速度、路径曲率等)和外部环境因素(如风速、气流等)智能预测出调整飞行速度的最佳方案。
深度学习模型通过对历史数据的学习和训练，能够识别复杂的飞行状态模式，并结合实时飞行数据进行动态调整。通过调整飞行速度，可以精确控制无人机的飞行轨迹，避免路径过度偏离或飞行速度过快或过慢导致的稳定性问题。此外，动态调整飞行速度还能够确保群体飞行中其他无人机的相对距离得以维持，避免因单个无人机的偏离而引发整体飞行队形错乱和碰撞风险。该步骤的核心作用是通过对无人机飞行状态的智能评估和实时响应，确保无人机群体能够协调一致地完成飞行表演，确保安全性并避免可能发生的飞行事故。
本发明通过实时监控无人机的飞行状态并结合风速数据、气流湍流等外部扰动，能够动态调整偏离轨迹的无人机的飞行路径和速度。这种调整能有效地防止无人机因无法适应风速变化而导致的轨迹偏离，从而避免群体飞行中的相对距离过近或队形错乱现象。通过精确控制无人机的飞行状态，能够保持群组的稳定性，确保各个无人机之间保持安全间距，显著降低因飞行轨迹偏差引发的碰撞风险。
