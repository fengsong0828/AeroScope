Description

低空飞行器防御装置
技术领域
本实用新型涉及飞行器技术领域，尤其涉及低空飞行器防御装置。
背景技术
飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物，随着飞行器普及，个人飞行器必然是未来发展方向。轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向，飞行器分为3类：航空器、航天器、火箭、导弹和无人机等。
在使用飞行器在城市道路中低空飞行时，其往往可能会因操作不当，而使飞行器撞击到墙壁或是其它障碍物而影响其正常使用，同时当需要对飞行器进行收纳时可将飞行器与防御装置分离，并使防御装置的防撞杆回缩，使其能够更好的收纳。
实用新型内容
本实用新型的目的是为了解决现有技术中存在的缺点，而提出的低空飞行器防御装置。
为了实现上述目的，本实用新型采用了如下技术方案：低空飞行器防御装置，包括飞行器，所述飞行器顶端固定连接有固定座，所述固定座内部开设有安装槽，所述安装槽内部设置有固定块，所述固定块外侧四周均固定连接有螺纹套筒，所述螺纹套筒内部均螺纹连接有螺纹杆，所述螺纹杆另一端均转动连接有防撞杆，所述防撞杆相邻的一端通过连接杆连接。
作为上述技术方案的进一步描述：
所述固定块两侧均开设有两个固定槽，两个所述固定槽内部均滑动连接有限位块，所述限位块另一侧均固定连接有拉杆，所述拉杆外侧均设置有弹簧一。
作为上述技术方案的进一步描述：
所述弹簧一一端均固定连接在推块靠近拉杆的一侧，所述弹簧一另一端均固定连接在安装槽两侧，所述拉杆远离限位块的一端分别固定连接在横杆一侧两端。
作为上述技术方案的进一步描述：
所述螺纹杆靠近螺纹套筒的一端均固定连接有连杆，所述连杆另一端均固定连接有转盘。
作为上述技术方案的进一步描述：
所述套筒内侧壁两侧均开设有滑槽，所述滑槽内部均滑动连接有滑块，所述滑块相互靠近的一侧分别固定连接在推块两侧，所述推块内部均开设有空腔，所述空腔内部均设置有转盘。
作为上述技术方案的进一步描述：
所述飞行器底端四角均固定连接有滑杆，所述滑杆外侧均设置有弹簧二，一侧所述滑杆分别贯穿安装板两端，所述安装板底端两侧均固定连接有固定杆，所述固定杆另一端分别固定连接在底板顶端两侧。
作为上述技术方案的进一步描述：
所述弹簧二一端均固定连接在飞行器底端四角，所述弹簧二另一端分别固定连接在安装板远离底板的一侧两端，所述滑杆远离飞行器的一端均固定连接有限位板。
本实用新型具有如下有益效果：
1、本实用新型中，转动螺纹杆使其在螺纹套筒内部移动，从而使防撞杆所围成的范围大于飞行器，同时在飞行器飞行的过程中利用防撞杆的运转实现对飞行器进行保护，从而防止飞行器在飞行的过程中螺旋桨撞击到墙壁。
2、本实用新型中，当需要对飞行器进行收纳时可拉动拉杆使限位块退出固定槽内部后将固定块从而安槽内部取出，后转动螺纹杆使防撞杆回缩，从而减小其所占用的空间使其便于收纳。

参照图1-5，本实用新型提供的一种实施例：低空飞行器防御装置，包括飞行器1，飞行器1顶端固定连接有固定座14，固定座14内部开设有安装槽17，安装槽17内部设置有固定块4，固定块4外侧四周均固定连接有螺纹套筒5，螺纹套筒5内部均螺纹连接有螺纹杆7，螺纹杆7另一端均转动连接有防撞杆3，防撞杆3相邻的一端通过连接杆6连接，转动螺纹杆7使其在螺纹套筒5内部移动，从而使防撞杆3所围成的范围大于飞行器1，同时在飞行器1飞行的工程中利用防撞杆3的运转实现对飞行器1进行保护，从而防止飞行器1在飞行的过程中螺旋桨撞击到墙壁。
固定块4两侧均开设有两个固定槽20，两个固定槽20内部均滑动连接有限位块18，限位块18另一侧均固定连接有拉杆15，拉杆15外侧均设置有弹簧一16，当需要对飞行器1进行收纳时可拉动拉杆15使限位块18退出固定槽20内部后将固定块4从安装槽17内部取出，后转动螺纹杆7使防撞杆3回缩，从而减小其所占用的空间使其收纳，弹簧一16一端均固定连接在推块9靠近拉杆15的一侧，弹簧一16另一端均固定连接在安装槽17两侧，拉杆15远离限位块18的一端分别固定连接在横杆19一侧两端，将固定块4放入到安装槽17内部，后利用弹簧一16的推力推动限位移动，从而使其伸入到固定槽20内部，从而将固定块4固定在安装槽17内部，螺纹杆7靠近螺纹套筒5的一端均固定连接有连杆13，连杆13另一端均固定连接有转盘12，利用转盘12与连杆13的配合使推块9能够随着螺纹杆7的移动而移动，套筒内侧壁两侧均开设有滑槽8，滑槽8内部均滑动连接有滑块10，滑块10相互靠近的一侧分别固定连接在推块9两侧，推块9内部均开设有空腔11，空腔11内部均设置有转盘12，利用推块9、滑槽8、滑块10的配合实现防止螺纹杆7滑出螺纹套筒5的部分过长，飞行器1底端四角均固定连接有滑杆22，滑杆22外侧均设置有弹簧二24，一侧滑杆22分别贯穿安装板23两端，安装板23底端两侧均固定连接有固定杆21，固定杆21另一端分别固定连接在底板2顶端两侧，利用滑杆22安装板23、底板2的配合实现在飞行器1降落到地面上时对其进行支撑，弹簧二24一端均固定连接在飞行器1底端四角，弹簧二24另一端分别固定连接在安装板23远离底板2的一侧两端，滑杆22远离飞行器1的一端均固定连接有限位板25，利用滑杆22、弹簧二24、安装板23的配合实现减轻飞行器1在降落时所产生的冲击力。
工作原理：将固定块4放入到安装槽17内部，后利用弹簧一16的推力推动限位移动，从而使其伸入到固定槽20内部，同时可转动螺纹杆7使其在螺纹套筒5内部移动，从而推动螺纹杆7上的防撞杆3向外侧延伸，使防撞杆3所围成的范围大于飞行器1，同时在飞行器1飞行的过程中利用防撞杆3的运转实现对飞行器1进行保护，当需要对飞行器1进行收纳时可拉动拉杆15使限位块18退出固定槽20内部后将固定块4从安装槽17内部取出，后转动螺纹杆7使防撞杆3回缩，并且在飞行器1降落时可利用滑杆22、弹簧二24、安装板23的配合实现减轻飞行器1在降落时所产生的冲击力，利用推块9、滑槽8、滑块10的配合实现防止螺纹杆7滑出螺纹套筒5的部分过长。
