Description

一种基于低空飞行器的违章建筑机器视觉监测方法
技术领域
本发明涉及无人机监测与计算机视觉技术领域，特别涉及一种基于低空飞行器的违章建筑机器视觉监测方法。
背景技术
当前城市违章建筑监测主要依赖人工巡查与定期航拍相结合的方式，存在显著的效率瓶颈和技术缺陷。据统计，传统人工巡查的方式平均每平方公里需要投入大约3个工作日，且漏检率较高。现有无人机监测系统普遍采用单一可见光传感器，在复杂光照条件下（如逆光、雾霾）的成像质量就会急剧下降，实测PSNR值常低于25dB。更关键的是，现有算法对小型违建（如屋顶加建、阳台封闭等）的识别率不足60%，这些技术短板直接导致全国年均新增违建查处率长期徘徊在68%左右。
在技术实现层面，现有监测系统面临三大核心挑战：首先，动态稳像技术多依赖单一传感器补偿，如纯IMU方案在强风条件下（风速>10m/s）的补偿残差达±15像素，而基于传统LK光流的方法计算复杂度高达O(n²)，无法满足实时性要求。其次，图像增强与目标检测算法存在割裂，主流方案如SRCNN+ Faster R-CNN的级联架构导致累计误差放大，实测mAP下降12.6个百分点。更重要的是，现有系统缺乏时序分析能力，对施工期违建的识别滞后平均达17.6天。2023年IEEE遥感期刊研究指出，当前最先进的多时相检测算法A^3Net在3个月时间跨度下的变化检测F1-score仅为0.71。这些技术缺陷严重制约了违建治理的时效性，造成全国每年因违建而产生重大的经济损失。
针对基于低空飞行器的违章建筑机器视觉方法，学者陈万卫在其论文《基于无人机摄影测量的建筑违章信息采集研究》中，提出了采用现代遥感影像和二维三维模型数据联动对违章建筑探索提高效率，虽然二三维联动提取违章建筑物矢量数据误差较小，但二维影像上采集建筑物占地面积受房檐影响较大，即使是三维数据，仍然无法消除房檐对面积测量的影响。发明人曾诚等人在专利《一种基于全卷积神经网络的违章建筑自动识别方法》中，实现了自动化识别，但其多步骤的处理流程（包括特征点检测、单应性变换、变化检测和深度学习分类）可能导致整体处理时间较长。
因此，需要一种基于低空飞行器的违章建筑机器视觉方法，能够显著解决航拍监测中图像信噪比低、标注样本不足、及时处理延迟较高等问题。
发明内容
本发明的主要目的在于提供一种基于低空飞行器的违章建筑机器视觉监测方法，可以有效解决背景技术中提到的问题。
为实现上述目的，本发明采取的技术方案为：
一种基于低空飞行器的违章建筑机器视觉监测方法，包括如下步骤：
S1：通过搭载多光谱摄像设备和IMU/BDS模块的低空飞行器采集目标区域视频数据，多光谱摄像设备至少包含可见光与近红外波段传感器；作为本发明的一种优选技术方案：所述S1中，多光谱成像系统采用5镜头同步采集架构，覆盖以下波段：可见光波段（450-650纳米）具备3840×2160像素分辨率，帧率为每秒30帧；近红外波段（700-900纳米）具备1280×720像素分辨率，帧率为每秒30帧；此外，提供可选的偏振光通道，以消除镜面反射的干扰；
S11；作为本发明的一种优选技术方案：所述S1中，IMU模块包括六轴惯性测量单元，陀螺仪零偏稳定性<0.5°/h，采样频率为200Hz；
S12:作为本发明的一种优选技术方案：所述S1中，BDS模块特性为高精度双频RTK，支持SBAS/QZSS增强信号，水平定位精度优于2厘米，垂直定位精度优于5厘米，更新频率可达每秒10次，确保飞行器在复杂环境下的精准定位与导航；
S13:作为本发明的一种优选技术方案：所述S1中,低空飞行器的飞行平台要求：飞行高度应控制在150米至500米范围内，该高度区间可根据实际任务中对地面分辨率的具体需求进行动态调整；飞行速度不得超过12米/秒，以确保航拍图像的纵向与横向重叠率均高于80%；同时，搭载的云台系统需具备±0.01度的抖动补偿精度，从而保障高精度成像的稳定性，这些参数的协同优化有效平衡了作业效率与数据采集质量的要求；
S2：采用多传感器融合的动态稳像预处理方法对原始视频数据进行处理，包括：基于IMU数据的运动补偿，结合SIFT特征点匹配的电子稳像；
S21:作为本发明的一种优选技术方案：所述S2中，动态稳像预处理包括：基于IMU数据的运动补偿，利用六轴IMU（陀螺仪+加速度计）数据，建立无人机运动模型，计算帧间姿态变化：

运动补偿采用扩展卡尔曼滤波(EKF)融合IMU数据与BDS位置信息，估计相机位置变化，并进行仿射变换补偿，消除大尺度运动模糊。

S3：使用基于深度学习的双阶段图像增强技术处理视频帧，依次执行：第一阶段采用U-Net网络进行去模糊处理，第二阶段通过SRGAN网络实现超分辨率重建；
S31:作为本发明中的一种优选技术方案：所述S3中，所述第一阶段U-Net进行去模糊处理，网络架构采用改进的U-Net++结构，在编码器-解码器之间增加密集跳跃连接，增强特征复用；
S32:作为本发明中的一种优选技术方案：所述S3中，所述第二阶段SRGAN超分辨率重建，网络架构采用改进的ESRGAN，生成器基于RRDB(残差密集块)，判别器采用谱归一化PatchGAN；
S4：通过融合无监督与有监督学习的检测算法识别违章建筑，包括：采用自监督预训练构建初始特征提取器，基于Faster R-CNN框架进行有监督微调；
S41:作为本发明中的一种优选技术方案：所述S4中,所述自监督预训练的框架采用MoCo v3,通过视觉变换构建预训练任务，对同一图像施加随机裁剪、旋转（±30°）、色彩抖动生成正样本对，使用ResNet-50作为编码器，动态更新动量队列；特征提取器优化，损失函数采用InfoNCE Loss,最大化正样本相似性；
S42:作为本发明中的一种优选技术方案：所述S4中,所述有监督微调，其网络改进可以通过特征金字塔(FPN)融合多尺度特征，适应不同尺寸违建；还可以用解耦检测头分离分类与回归分支，减少任务冲突；其困难样本挖掘是对负样本按预测置信度排序，保留Top30%作为困难样本；正负样本比例控制在1：3，避免类别不平衡；
S5:结合时序分析和多源数据验证输出检测结果，包括：利用光流法进行跨帧目标关联，将检测结果与GIS地图数据进行空间坐标匹配验证；
S51:作为本发明中的一种优选技术方案：所述S5中，所述时序分析中，跨帧追踪采用Farneback稠密光流算法，计算相邻帧间像素运动场；对检测到的违建目标建立运动轨迹模型，基于卡尔曼滤波预测位置；使用DBSCAN算法对连续N帧（默认N=5）的检测框聚类；剔除瞬时出现的假目标（如飞鸟、云影）；将检测框坐标（像素级）通过透视投影模型转换至WGS84地理坐标系；与国土规划部门的建筑审批数据库比对，标记未备案的违建区域；GIS空间匹配将检测框坐标（像素级）通过透视投影模型转换至WGS84地理坐标系，与国土规划部门的建筑审批数据库比对，标记未备案的违建区域。
与现有技术相比，本发明具有如下有益效果：
本发明通过搭载多光谱摄像设备与IMU/BDS模块的低空飞行器采集数据，采用多传感器融合（卡尔曼滤波、SIFT特征点匹配）的动态稳像预处理、基于U-Net和SRGAN的双阶段图像增强技术、融合MoCov3自监督预训练与FasterR-CNN有监督微调的检测算法，以及光流法跨帧追踪结合GIS数据和建筑审批数据库的时序分析与多源验证；其有益效果为有效解决航拍图像信噪比低、样本不足、处理延迟高的问题，提升违章建筑识别精度与实时性，降低人工成本，扩大监测覆盖范围，并通过三维点云重建提供量化证据，实现对违章建筑的高效监控与精准管理。

所述步骤S3中，使用基于深度学习的双阶段图像增强技术处理视频帧，依次执行：第一阶段采用U-Net网络进行去模糊处理，第二阶段通过SRGAN网络实现超分辨率重建；
第一阶段U-Net进行去模糊处理，网络架构采用改进的U-Net++结构，在编码器-解码器之间增加密集跳跃连接，增强特征复用，通过跳跃连接，使得浅层特征能够直接传递到深层，提高了信息的流通效率和特征的表达能力，在训练过程中，采用均方误差（MSE）作为损失函数，通过反向传播算法优化网络参数，使得输出的去模糊图像与清晰图像之间的误差最小化；
第二阶段SRGAN超分辨率重建，网络架构采用改进的ESRGAN，生成器基于RRDB，判别器采用谱归一化PatchGAN；过引入相对判别器和感知损失，进一步提升重建图像的真实感和细节表现力，确保图像质量满足后续检测任务的高要求。在训练阶段，采用对抗性损失和内容损失的组合作为优化目标，通过交替迭代生成器和判别器，逐步逼近真实的高分辨率图像分布；
所述步骤S4中，通过融合无监督与有监督学习的检测算法识别违章建筑，包括：采用自监督预训练构建初始特征提取器，以Faster R-CNN为框架进行有监督微调；
自监督预训练采用MoCo v3框架，通过视觉变换构建预训练任务，对同一图像施加随机裁剪、翻转（水平或垂直）、色彩抖动等操作生成正样本对。使用ResNet-101作为编码器，以捕获更丰富的特征表示，动态更新动量队列以保持特征表示的时效性和多样性。在预训练阶段，特征提取器不断优化，损失函数采用InfoNCE Loss，旨在最大化正样本之间的相似性，同时最小化负样本之间的相似性，从而增强模型的泛化能力；
Faster R-CNN为框架进行有监督微调，在有监督微调阶段，采用难例挖掘策略，关注那些分类置信度低或边界框回归误差大的样本，通过这些困难样本的梯度回传，使模型更加聚焦于违章建筑的精确检测；
结合时序分析和多源数据验证输出检测结果，包括：利用光流法进行跨帧目标关联，将检测结果与GIS地图数据进行空间坐标匹配验证；
例如，利用Farneback稠密光流算法计算相邻帧间像素运动场，该算法通过比较相邻帧中像素点的灰度值变化，估计像素点的运动矢量，从而实现对目标区域的精确追踪。对于检测到的违建目标，建立运动轨迹模型，基于卡尔曼滤波预测其位置，以平滑运动轨迹并减少噪声干扰。
检测框坐标（像素级）通过透视投影模型转换至WGS84地理坐标系，以便与国土规划部门的建筑审批数据库进行比对。透视投影模型考虑了摄像机的内参和外参，能够准确地将图像坐标转换为地理坐标；通过与建筑审批数据库的比对，标记出未备案的违建区域，实现违章建筑的精准识别；
GIS空间匹配是将检测框坐标与GIS地图数据进行空间坐标匹配验证的过程。通过将检测框坐标转换至WGS84地理坐标系，与GIS地图中的建筑信息进行比对，可以进一步验证检测结果的准确性。同时，GIS空间匹配还可以提供违建区域的具体地理位置信息，为后续的执法行动提供有力支持。
