Description

低空飞行器的接地装置
技术领域
本发明涉及电气技术领域，尤其涉及一种低空飞行器的接地装置。
背景技术
随着低空飞行器电子电气功能发展的日趋自动化与智能化，其系统设计（含电气布线布局、接地控制系统策划等）也逐步向高集成度方向发展，由此而来对飞行器的控制系统电路中各电子电气功能模块的布局及走线要求也愈加严苛，同时也包括接地控制系统的设计，避免因走线布局不合理对电磁兼容（Electromagnetic Compatibility，EMC）、电气连接可靠性等带来不良影响，从而影响飞行器的飞行性能，严重情况下还会干扰飞行器的正常工作。
现有的低空飞行器的接地方式是将飞行器的控制系统电路中所有电子电气模块的地线最后汇集至一个点，然后再与飞行器的外壳接地点相连。此种方式会导致线束走线延长，电气连接可靠性较差，且会增加电磁辐射和电磁抗扰的几率，尤其是辐射发射、传导发射及辐射抗扰、传导抗扰所带来的不良影响。另一方面，现有接线方式，对线束导通性测试及干扰排查过程的故障锁定也会带来不利影响，如所有地线直接汇集至同一接地点不利于采取逐一断开接地线而精准锁定异常点的故障排查方式。
发明内容
本发明提供一种低空飞行器的接地装置，用以提升飞行器的接地的电气连接可靠性，并增加飞行器的电磁抗扰能力。
本发明提供一种低空飞行器的接地装置，包括：多个方形接地载体、多个条形接地载体以及多个柱形接地载体；
各方形接地载体固定设置在飞行器的控制系统电路中的不同区域，所述方形接地载体是包括多个接地插孔的导体，用于接入区域范围内控制系统电路中的接地点；
各柱形接地载体为导体，所述柱形接地载体的第一端与区域范围内控制系统电路中的接地点连接，所述柱形接地载体的第二端与所述方形接地载体的接地插孔连接，各柱形接地载体用于将飞行器的控制系统电路中的所有接地点，接入对应区域中设置的方形接地载体；
各条形接地载体为导体，用于在不同方形接地载体之间进行连接，以使得各方形接地载体之间相互导通；
所述多个条形接地载体中的任一条形接地载体与所述飞行器的外壳接地点连接。
根据本发明提供的一种低空飞行器的接地装置，所述柱形接地载体的第一端为金属连接器，用于基于导线与区域范围内控制系统电路中的接地点连接；
所述柱形接地载体的第二端为接线柱，与所述方形接地载体的接地插孔匹配，用于连接所述方形接地载体的接地插孔。
根据本发明提供的一种低空飞行器的接地装置，所述接线柱包括柱形金属体以及弹性锁紧区域；
所述柱形金属体的圆周设置有一圈弹性锁紧区域，用于与接地插孔对接后的锁紧。
根据本发明提供的一种低空飞行器的接地装置，所述方形接地载体中各接地插孔下方设置有地线标签，所述地线标签用于记载与接地插孔连接的接地点信息。
根据本发明提供的一种低空飞行器的接地装置，所述方形接地载体中各接地插孔是基于在方形金属板上开设无螺纹的通孔或者基于在方形金属板中开设的通孔中安装标准接线柱得到的。
根据本发明提供的一种低空飞行器的接地装置，所述条形接地载体还包括多个紧固点；
各紧固点通过螺钉紧固方式与方形接地载体进行连接。
根据本发明提供的一种低空飞行器的接地装置，所述条形接地载体中紧固点的数量是基于条形接地载体与方形接地载体之间的对接宽度确定的。
根据本发明提供的一种低空飞行器的接地装置，所述条形接地载体为预设厚度的铜箔或预设厚度的铜编织带。
根据本发明提供的一种低空飞行器的接地装置，所述条形接地载体沿飞行器外壳缝隙，粘贴安装在所述飞行器中。
根据本发明提供的一种低空飞行器的接地装置，所述控制系统电路中的不同区域是基于所述控制系统电路中接地点的分布，对所述控制系统电路进行划分得到的。
本发明提供的低空飞行器的接地装置，通过在飞行器的控制系统电路中的不同区域设置方形接地载体，以就近、多点及单点接地为基本设计原则，将众多独立分布的接地点逐步衍生至区域性接地汇集点，提升了电气连接的可靠性。并基于在不同方形接地载体之间进行连接的条形接地载体，将区域性接地汇集点与飞行器外壳接地点连接，形成接地总汇集点，从而建立飞行器的完整的接地系统。将独立分布的接地点进行小范围的分类与汇集，可以有效抑制或者消除EMC所带来的不良隐患，提升飞行器的电磁抗扰能力，同时提升线束导通测试和故障排查的便利性，并进一步提升飞行器电气布线的可装配性和可制造性能力。

具体实施方式
为使本发明的目的、技术方案和优点更加清楚，下面将结合本发明中的附图，对本发明中的技术方案进行清楚、完整地描述，显然，所描述的实施例是本发明一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有作出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
图1是本发明提供的低空飞行器的接地装置的结构示意图，如图1所示，该装置包括多个方形接地载体110、多个条形接地载体120以及多个柱形接地载体130。
对飞行器的控制系统电路所处的范围进行划分，得到多个区域。各方形接地载体110固定设置在飞行器的控制系统电路中的不同区域。即在每一个区域中设置一个方形接地载体110。方形接地载体110可以连接该区域范围内控制系统电路中的接地点。
方形接地载体110用于区域性接地汇集点的建立，可以通过在一定厚度的方形金属板上开设一定数量的接地插孔，接地插孔用于与柱形接地载体相对接（接地线安装在柱形接地载体上），即借助金属板的导电性将众多独立的接地点汇集至一起。
需要说明的是，方形接地载体110中的接地插孔的数量可以根据区域范围中的接地需求数量进行确定。
柱形接地载体130，是用于实现电路中接地点与方形接地载体110中接地插孔的连接过程。
具体的，柱形接地载体130可以是包括两端的柱形金属导体，第一端与控制系统电路中的接地点连接，第二端与方形接地载体的接地插孔连接，以实现将控制系统电路中的接地点与方形接地载体进行连接。
条形接地载体120主要用于建立不同方形接地载体110之间的连接，即通过条形接地载体120，将不同方形接地载体110所形成的区域性接地汇集点再次连接至同一个接地点。
需要说明的是，因低空飞行器的外壳区域一般为非金属材质，所以必须通过条形接地载体将不同的区域性接地汇集点连接至外壳接地点Housing。若用于外壳接地点Housing整体为金属的产品时，条形接地载体120则可取消，因为金属外壳已将所有的区域性接地汇集点连接到了一起。
随着不同电子电气功能模块的配置与增加，飞行器的控制系统面临着布线复杂度上升和EMC性能受影响的双重考验。本发明所涉及的接地装置应用于低空飞行器，可以在优化飞行控制系统EMC性能的基础上，同时进一步提升电气布线的便利性与故障排查的效率。
接地点的架构可以如图2本发明提供的接地架构示意图所示，在布线之初，不同电子电气功能模块所对应的地线均为独立分布的接地点，结合电气功能属性及就近接地原则，若干数量的独立分布接地点首先逐步形成一定数量的区域性接地汇集点，其次区域性接地汇集点又逐步形成更小范围的区域性接地汇集点，并最终与低空飞行器的外壳接地点Housing汇集而形成接地总汇集点，至此则完整构成整个飞行控制系统的接地控制系统。
图3是本发明提供的接地方式示意图。结合图3所示的接地方式示意图和图2所示接地控制系统的架构设计，同步借助接地装置中的不同接地载体，则可有效建立低空飞行器的接地控制系统。从图3可知，接地装置主要包含方形接地载体、条形接地载体和柱形接地载体三大模块。同步结合图2接地架构，独立分布的接地点通过柱形接地载体（可理解为连接器的公头）与方形接地载体（可理解为连接器的母头）建立了导通关系，同时在方形接地载体上形成了区域性的接地汇集点，即n个独立的接地点汇集至一定数量的方形接地载体。在不同区域性接地汇集点之间，通过条形接地载体再一次建立导通关系。如此类推，n个区域性接地汇集点最终全部汇集至一起，并与低空飞行器的外壳接地点Housing相连而形成完整的接地控制系统。
基于上述接地方法，可以通过最短的距离将地线汇集至最近的区域，最大程度地规避EMC的影响同时又提升装配便利性，包括对生产成本带来的有益效果。多点接地，即结合飞行控制系统的接地系统布局，将独立分布的接地点基于就近接地原则，逐步形成多个区域性的接地汇集点，进一步提升EMC性能和干扰排查的便利性；单点接地，即将多点接地形成的区域性接地汇集点最终连接至同一个接地总汇集点，从而形成低空飞行器的完整接地控制系统。
低空飞行器的接地装置中包含的方形接地载体、条形接地载体及柱形接地载体三大部分，其中方形接地载体侧重将独立分布的接地点进行小范围的分类与汇集，从而建立区域性接地汇集点，即根据系统布线规划和导线电气属性，将众多独立分布的地线分阶段地汇集成多个区域性的接地汇集点，规避所有地线全部铺设至最后才进行总汇集，从而对EMC和线束装配等带来不利影响；条形接地载体则用于建立不同方形接地载体之间的连接，即任意两个方形接地载体之间可通过一个条形接地载体进行连接，从而形成共同的接地汇集点；柱形接地载体与方形接地载体以类似连接器的搭配关系进行配合使用，一方面用于地线的便捷连接，另一方面便于干扰或故障排查，即通过逐一断开柱形接地载体可逐步锁定干扰源/故障源。
需要说明的是，以就近、多点及单点接地为基本设计原则，将众多独立分布的接地点逐步衍生至区域性接地汇集点，最后将区域性接地汇集点形成接地总汇集点，从而建立完整的倒三角形接地控制系统。是针对飞行控制系统复杂繁多的接地信号，同时结合各地线的电气功能属性及接地设计原则，建立规范完整的接地控制系统。可以有效抑制/消除EMC所带来的不良隐患的同时，在可制造性设计（Design for Manufacturability，DFM）与可装配性设计（Design for Assembly，DFA）、提质降本增效等方面也可带来一定的有益效果。
根据倒三角形接地控制系统的布线方式，设计方形、条形和柱形三种不同的接地载体，同时通过三大接地载体的有效连接，从而形成完整的接地控制系统。是针对接地控制系统的架构及接地方式的策划，设计不同的接地装置助推规范接地和可靠接地的有效实施。不同接地控制系统的糅合应用，一方面可提升故障排查与锁定的便捷性，另一方面对DFM、DFA及飞行控制器的系统设计优化也有助推作用。
本发明提供的低空飞行器的接地装置，通过在飞行器的控制系统电路中的不同区域设置方形接地载体，以就近、多点及单点接地为基本设计原则，将众多独立分布的接地点逐步衍生至区域性接地汇集点，提升了电气连接的可靠性。并基于在不同方形接地载体之间进行连接的条形接地载体，将区域性接地汇集点与飞行器外壳接地点连接，形成接地总汇集点，从而建立飞行器的完整的接地系统。将独立分布的接地点进行小范围的分类与汇集，可以有效抑制或者消除EMC所带来的不良隐患，提升飞行器的电磁抗扰能力，同时提升线束导通测试和故障排查的便利性，并进一步提升飞行器电气布线的可装配性和可制造性能力。
在一个实施例中，所述柱形接地载体的第一端为金属连接器，用于基于导线与区域范围内控制系统电路中的接地点连接；所述柱形接地载体的第二端为接线柱，与所述方形接地载体的接地插孔匹配，用于连接所述方形接地载体的接地插孔。所述接线柱包括柱形金属体以及弹性锁紧区域；所述柱形金属体的圆周设置有一圈弹性锁紧区域，用于与接地插孔对接后的锁紧。
柱形接地载体，用于与方形接地载体的对接，主要包含柱形金属体、金属连接器及弹性锁紧区域三大部分。其中柱形金属体略小于方形接地载体中的接地插孔，为进一步提升连接可靠性，金属体圆周会设置一圈弹性锁紧区域，既便于柱形接地载体与方形接地载体的对接，同时又提升了锁紧的可靠性，尤其在飞行器飞行过程存在振动的工况下，此种弹性结构设计显得更加重要。
金属连接器可以如图4本发明提供的金属连接器的连接结构示意图所示，具体可以包含两种方式：一种是采取焊接方式将地线装配至柱形接地载体，如图4左图所示的金属连接器，焊接腔体以左图所示的虚线圆圈为界限。另一种是采取螺钉紧固方式，如图4右图所示的地线紧固孔位，通过螺钉方式来装配对应的地线。此两种方式各有利弊，具体可根据线径大小、维护保养等要求综合而定，如地线线径过小，螺钉紧固方式则不适用，此时采取焊接方式对连接可靠性会更有保障，通常情况下，1mm以下线径采取焊接方式。同理，当地线线径达到一定值时，考虑到焊接透锡等因素的控制采取螺钉紧固装配方式则更便捷和可靠。
可以理解的是，基于柱形接地载体，实现了通过最短的距离将地线汇集至最近的区域中的方形接地载体，最大程度地规避EMC的影响同时又提升装配便利性。多点接地，即结合飞行控制系统的接地系统布局，将独立分布的接地点基于就近接地原则，逐步形成多个区域性的接地汇集点，进一步提升EMC性能和干扰排查的便利性。
在一个实施例中，方形接地载体中各接地插孔是基于在方形金属板上开设无螺纹的通孔或者基于在方形金属板中开设的通孔中安装标准接线柱得到的。方形接地载体中各接地插孔下方设置有地线标签，所述地线标签用于记载与接地插孔连接的接地点信息。
地线标签可直接粘贴具有良好粘性的标签，如不干胶标签等，也可直接在接地载体上进行字符丝印，具体可视操作便利性和加工成本而定。接地插孔下方设置的地线标签，可以便于线束导通性检测与故障排查。
相关方法中将所有电子电气模块的地线最后汇集至一个点，对线束导通性测试及干扰排查过程的故障锁定也会带来不利影响，如所有地线直接汇集至同一接地点不利于采取逐一断开接地线而精准锁定异常点的故障排查方式。基于设置地线标签，能够清晰地指示每条接地线，使得维护人员和检修人员在面对复杂的飞行器电气系统时，能够迅速识别出各条接地线的连接情况。这有助于避免误操作或误触，降低安全事故的风险。
方形接地载体示意图可以如图5本发明提供的方形接地载体示意图所示。方形接地载体的大小可根据独立分布接地点在当前规划区域的数量来决定，即取决于n的大小，而紧固点的数量则由方形接地载体的大小/面积来决定，紧固点的位置则根据低空飞行器的整体结构布局和条形接地载体的对接宽度综合而定。紧固点可直接基于在方形金属板开设带有适量螺纹的安装孔，也可以采取铆接螺孔的方式，或者基于在方形金属板中开设的通孔中安装标准接线柱实现，具体则根据金属板厚度、安装要求、螺纹孔加工工艺水平及紧固可靠性共同确定。
接地插孔可采取直接在金属板上开设无螺纹的通孔，此种方式需保证金属板的厚度h能完全满足柱形接地载体对接后的可靠性。基于成本和轻量化设计要求，一般情况下h1=（3~5）mm，避免厚度不够而无法与柱形接地载体完全咬合，其中通孔直径d1=（d2-0.5）mm，d2为柱形接地载体的直径（不含弹性锁紧区域）；二是在通孔基础上安装标准接线柱，这种方式则需确保接线柱内径φ与柱形接地载体相匹配，一般情况要求d2=（φ-0.5）mm，规避地线接触不良等隐患。
在一个实施例中，条形接地载体还包括多个紧固点；各紧固点通过螺钉紧固方式与方形接地载体进行连接。条形接地载体中紧固点的数量是基于条形接地载体与方形接地载体之间的对接宽度确定的。条形接地载体为预设厚度的铜箔或预设厚度的铜编织带。条形接地载体沿飞行器外壳缝隙，粘贴安装在所述飞行器中。
条形接地载体主要用于建立不同方形接地载体之间的连接，即通过条形接地载体，将不同方形接地载体所形成的区域性接地汇集点再次连接至同一个接地点。具体可以如图6本发明提供的条形接地载体示意图所示。
需要说明的是，因低空飞行器的外壳区域为非金属材质，所以必须通过条形接地载体将不同的区域性接地汇集点连接至外壳接地点Housing。当用于外壳整体为金属的产品时，条形接地载体则可取消，因为金属外壳已将所有的区域性接地汇集点连接到了一起。
条形接地载体可选用厚度适中的铜箔或铜编织带，一般要求铜箔或铜编织带的厚度h2=（1~3）mm，基于对飞行控制系统不同电气功能模块电磁干扰信号的屏蔽（侧重辐射发射RE和辐射抗扰RI两种情况），要求具有良好粘接性的铜箔一般沿飞行器外壳缝隙进行粘贴安装，即将缝隙封住，尽可能降低电磁干扰信号的传输能力。
条形接地载体通过螺钉紧固方式与方形接地载体进行连接，其中螺钉紧固点的位置、数量基于条形接地载体与方形接地载体的形状面积共同确定，也即根据二者之间的对接宽度来确定。同时为进一步提升铜箔或铜编织带接地的可靠性，尤其是当条形接地载体超过一定的长度时，可在载体的中间位置适当增加螺钉紧固点。其中方形接地载体的形状相对于圆形、条形或柱形接地载体而言，并非严格意义上的规则方形。
可以理解的是，基于条形接地载体可以实现不同区域接地点进行进一步的汇集，从而实现将飞行器的控制系统电路中所有电子电气模块的地线汇集至飞行器的外壳接地点，从而建立飞行器的完整的接地系统。将独立分布的接地点进行汇集，可以有效抑制或者消除EMC所带来的不良隐患，提升飞行器的电磁抗扰能力，同时提升线束导通测试和故障排查的便利性，并进一步提升飞行器电气布线的可装配性和可制造性能力。
在一个实施例中，所述控制系统电路中的不同区域是基于所述控制系统电路中接地点的分布，对所述控制系统电路进行划分得到的。
可以基于接地点的空间分布，实现划分过程，根据接地点在物理空间中的位置进行划分。相邻或接近的接地点可以归为一个区域，以便于布线和接地汇集点的设置。
具体的，可以根据控制系统电路中接地点的分布，确定所述控制系统电路中包含接地点的范围，然后再根据范围中的不同区域的接地点数量，对控制系统电路进行划分，得到多个不同区域。
例如，将控制系统电路中包含5个相邻接地点的范围作为一个区域。对控制系统电路中所有接地点进行划分，得到控制系统电路中的多个不同区域。
也可以基于电路的功能模块进行划分，如电源模块、信号处理模块、控制模块等。每个功能模块内的接地点可以归为一个区域。
在每个划分好的区域内，选择一个合适的位置作为接地汇集点，在该位置设置方形接地载体。这个位置应该便于导线的连接和汇总，同时避免与其他电气设备的干扰。
根据系统布线规划和导线电气属性，将众多独立分布的地线分阶段地汇集成多个区域性的接地汇集点，连接方形接地载体。这可以确保地线在铺设过程中不会相互干扰，同时提高接地效率。
本发明提供的低空飞行器的接地装置，基于接地点的分布对控制系统电路进行区域划分并建立区域性接地汇集点，以就近、多点及单点接地为基本设计原则，将众多独立分布的接地点逐步衍生至区域性接地汇集点，可以优化地线布局，提升了电气连接的稳定性以及可靠性。将独立分布的接地点进行小范围的分类与汇集，可以有效抑制或者消除EMC所带来的不良隐患，提升飞行器的电磁抗扰能力，同时提升线束导通测试和故障排查的便利性，并进一步提升飞行器电气布线的可装配性和可制造性能力。
