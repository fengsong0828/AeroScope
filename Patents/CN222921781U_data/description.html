Description

一种低空飞行器舱体稳定模块
技术领域
本实用新型涉及货运飞行器相关技术领域，具体是一种低空飞行器舱体稳定模块。
背景技术
飞行器在飞行时，由于其机体、机翼或螺旋桨等会因外部的作用力而改变其飞行状态，如倾斜等。为了调整其飞行状态，往往采用机体中的平衡调整系统。对于大型的飞行器，如直升机，都采用比较复杂的调整控制系统，山中央控制系统进行自动调整。这类控制系统的结构等都比较复杂，不适用于小型低空无人飞行器。
目前，一般的低空飞行器都采用较简单的机械结构进行制造的，如直接通过电机带动螺旋桨旋转而使其爬升，对于在其飞行过程中受到外力影响所有产生的下坠现象都没给予考虑，因此，日前的无人飞行器都很少安装有关的平衡控制装置。而采用结构较复杂的平衡控制装置，将因价格昂贵等原因而被弃用。
实用新型内容
本实用新型的目的在于提供一种低空飞行器舱体稳定模块，以解决上述背景技术中提出的问题。
为实现上述目的，本实用新型提供如下技术方案：
一种低空飞行器舱体稳定模块，包括两组平行设置的横向稳定机构以及一组连接在两组横向稳定机构之间的纵向稳定机构，纵向稳定机构与两组横向稳定机构相互垂直，所述纵向稳定机构包括一组固定杆，固定杆两端固定连接有第一安装架，第一安装架两端固定连接有第二安装架，第二安装架之间同样固定连接有一组固定杆，所述固定杆上滑动连接有配平块，配平块上连接有用于对配平块的位置进行调节的配平组件。
作为本实用新型进一步的方案：所述配平组件包括固定连接在第一安装架与第二安装架上的第一固定架，第一固定架之间固定连接有磁铁杆，磁铁杆上滑动连接有铜管，铜管两端固定连接有推板。
作为本实用新型进一步的方案：所述第一固定架上固定连接有固定筒，固定筒内固定连接有固定块，固定块内滑动连接有伸缩杆，伸缩杆一端穿过固定筒与第一固定架固定连接有触头，触头与推板相互配合，所述第一固定架与固定块之间设置有固定连接在伸缩杆上的挡块，挡块与固定块之间设置有套设在伸缩杆上的弹簧。
作为本实用新型进一步的方案：所述伸缩杆的另一端穿过固定块固定连接有安转座，安装座上固定连接有反射器，所述固定筒底部固定连接有激光测距仪，激光测距仪与反射器相互配合。
作为本实用新型进一步的方案：所述第一安装架与第二安装架上固定连接有第二固定架，第二固定架上固定连接有液压缸，液压缸内滑动连接有液压块，液压块之间固定连接有驱动杆，驱动杆上固定连接有连接架，连接架固定连接在配平块上。
作为本实用新型再进一步的方案：所述液压缸上连接有液压油管的一端，液压油管的另一端连接有液压泵。
与现有技术相比，本实用新型的有益效果是：本实用新型通过配平组件的设置将配平块向另一侧进行推动，从而使配平块向质量较轻的一侧进行推动，从而对舱体重量较轻的一侧的重量进行配平，使舱体的重心保持在舱体的中心处，进而使舱体保持稳定，并且本实用新型还通过横向稳定机构与纵向稳定机构的相互配合对舱体的多个方向的自动配平稳定，提升了设备的实用性。

参阅图1～图3，本实用新型实施例中，一种低空飞行器舱体稳定模块，包括两组平行设置的横向稳定机构以及一组连接在两组横向稳定机构之间的纵向稳定机构，纵向稳定机构与两组横向稳定机构相互垂直，所述纵向稳定机构包括一组固定杆1，固定杆1两端固定连接有第一安装架2，第一安装架2两端固定连接有第二安装架24，第二安装架24之间同样固定连接有一组固定杆1，所述固定杆1上滑动连接有配平块3，配平块3上连接有用于对配平块3的位置进行调节的配平组件，当舱体发生倾斜时，配平块3会在重力的作用下向重量大的一侧进行移动，而此本实用新型通过配平组件的设置将配平块3向另一侧进行推动，从而使配平块3向质量较轻的一侧进行推动，从而对舱体重量较轻的一侧的重量进行配平，使舱体的重心保持在舱体的中心处，进而使舱体保持稳定。
在本实施例的一种情况中，请参阅图1～图3，所述配平组件包括固定连接在第一安装架2与第二安装架24上的第一固定架4，第一固定架4之间固定连接有磁铁杆11，磁铁杆11上滑动连接有铜管12，铜管12两端固定连接有推板13，所述第一固定架4上固定连接有固定筒5，固定筒5内固定连接有固定块6，固定块6内滑动连接有伸缩杆7，伸缩杆7一端穿过固定筒5与第一固定架4固定连接有触头10，触头10与推板13相互配合，所述第一固定架4与固定块6之间设置有固定连接在伸缩杆7上的挡块8，挡块8与固定块6之间设置有套设在伸缩杆7上的弹簧9，所述伸缩杆7的另一端穿过固定块6固定连接有安转座，安装座14上固定连接有反射器15，所述固定筒5底部固定连接有激光测距仪16，激光测距仪16与反射器15相互配合，所述第一安装架2与第二安装架24上固定连接有第二固定架17，第二固定架17上固定连接有液压缸18，液压缸18内滑动连接有液压块19，液压块19之间固定连接有驱动杆20，驱动杆20上固定连接有连接架21，连接架21固定连接在配平块3上，所述液压缸18上连接有液压油管23的一端，液压油管23的另一端连接有液压泵22，当舱体发生倾斜时，铜管12在重力的作用下向重量较大的一侧进行滑动，此时铜管12在磁铁杆11上进行滑动，此时铜管12在滑动过程中产生磁场，此时通过楞次定律可知，铜管12会受到磁场产生的阻力，进而实现铜管12的缓慢滑落，铜管12带动推板13对触头10进行推动，触头10带动伸缩杆7向固定筒5内进行收缩，固定杆1通过挡块8对弹簧9进行压缩，同时伸缩杆7通过安装座14带动反射器15进行移动，此时反射器15与激光测距仪16之间的相互对距离发生改变，此时激光测距仪16控制液压泵22向反射器15与激光测距仪16之间距离减小的一侧的液压缸18内注入液压油，此时液压块19在液压油的推动下带动驱动杆20进行伸出，驱动杆20通过连接架21带动配平块3进行移动，从而实现对舱体的自动配平。
