Description

用于低空无人飞行器的地面目标位置确定方法及装置
技术领域
本发明涉及计算机视觉技术领域，尤其涉及一种用于低空无人飞行器的地面目标位置确定方法、装置及存储介质。
背景技术
在目标追踪、人员救援、物资投送等任务中，低空无人飞行器经常需要获取目标在世界坐标系中的精确位置。但对于山地等绝对高程变化较多的场景，目标的位置通常难以通过单纯的经纬度描述，而是需要带有高程信息的位置描述。
一般而言，低空无人飞行器可携带激光雷达等测距传感器来获取高程信息。然而，测距传感器通常较为笨重且具有较高额功耗，且会导致移动能力和续航里程下降，提高了运行成本。
发明内容
本发明提供一种用于低空无人飞行器的地面目标位置确定方法及装置，用以解决现有技术中系统运行成本过高的技术问题。
第一方面，本发明提供一种用于低空无人飞行器的地面目标位置确定方法，包括如下步骤。
利用低空无人飞行器搭载的光学传感器采集多幅图像，并获取所述光学传感器采集每一图像时的位姿信息；
利用目标检测模型获取所述图像中的目标，并对不同图像中的目标进行匹配，得到包含同一目标的多幅图像；
基于包含同一目标的多幅图像各自对应的所述位姿信息，获取所述同一目标的位置信息。
在一些实施例中，所述利用目标检测模型获取所述图像中的目标，并对不同图像中的目标进行匹配，包括：
利用目标检测模型对所述图像进行目标检测，得到所述图像中的每一目标对应的目标检测框；
获取所述目标检测框中的特征点集合；
基于所述目标检测框的邻域范围和所述目标检测框中的特征点集合，对不同图像中的目标进行匹配。
在一些实施例中，所述获取所述目标检测框中的特征点集合，包括：
获取所述目标检测框对应的图像分块；
对所述图像分块进行特征提取，得到所述图像分块对应的特征点集合；
确定所述目标检测框中的特征点集合为所述图像分块对应的特征点集合。
在一些实施例中，所述基于目标检测框的邻域范围和所述目标检测框中的特征点集合，对不同图像中的目标进行匹配，包括：
确定当前图像中的目标检测框的邻域范围；
将后续图像中的所述邻域范围内存在的目标检测框作为候选匹配框；
对所述候选匹配框中的特征点集合与所述当前图像中的目标检测框中的特征点集合进行匹配。
在一些实施例中，所述方法还包括：
在匹配成功的特征点数量高于预设阈值的情况下，确定所述候选匹配框中的目标与所述当前图像中的目标检测框中的目标为同一目标；和/或
在匹配成功的特征点数量小于或等于预设阈值的情况下，确定所述候选匹配框中的目标与所述当前图像中的目标检测框中的目标不是同一目标。
在一些实施例中，所述基于包含同一目标的多幅图像各自对应的所述位姿信息，获取所述同一目标的位置信息，包括：
基于包含同一目标的多幅图像各自对应的所述位姿信息以及所述同一目标的目标检测框，利用多目视觉法和三角定位法获取所述同一目标在世界坐标系下的位置信息。
在一些实施例中，所述利用低空无人飞行器搭载的光学传感器采集多幅图像，包括：
利用一台低空无人飞行器搭载的光学传感器获取多幅图像；和/或
利用多台低空无人飞行器搭载的光学传感器获取同一时刻对应的多幅图像。
在一些实施例中，所述获取所述光学传感器采集每一图像时的位姿信息，包括：
利用所述低空无人飞行器搭载的定位传感器获取所述光学传感器采集每一图像时在世界坐标系下的位置数据；
利用所述低空无人飞行器搭载的姿态传感器获取所述光学传感器采集每一图像时的姿态数据。

图1是本发明提供的用于低空无人飞行器的地面目标位置确定方法的流程示意图之一，如图1所示，本发明提供一种用于低空无人飞行器的地面目标位置确定方法。该方法包括：
步骤101、利用低空无人飞行器搭载的光学传感器采集多幅图像，并获取所述光学传感器采集每一图像时的位姿信息。
具体地，利用低空无人飞行器搭载或挂载的光学传感器，以固定的帧率采集多幅图像。
低空无人飞行器具备机载计算单元，用于将图像和位姿信息等传输至云端计算机进行处理。
光学传感器可以为可见光相机或热成像相机等成像设备。成像设备应经过单目相机标定，实现图像去畸变处理，并获取内参。
光学传感器采集的图像为对准地面拍摄的图像。
获取图像的同时获取该光学传感器采集每一图像时的位姿信息。位姿信息包括位置信息和姿态信息，该位置信息可以为世界坐标系下的位置坐标，该姿态信息包括俯仰、滚转和偏航角度等。
步骤102、利用目标检测模型获取所述图像中的目标，并对不同图像中的目标进行匹配，得到包含同一目标的多幅图像。
具体地，利用训练好的目标检测模型对图像进行目标识别，输出图像中每一目标的检测框或矩形框，并对不同图像中的目标进行匹配，即进行相似度对比，若目标相似度很高则匹配成功，表明进行匹配的不同图像中的目标是同一目标，遍历完所有的目标后，得到一组或多组图像集，每一图像集为包含同一目标的多幅图像。
目标检测模型可以为YOLO系列或其他目标检测网络模型。
步骤103、基于包含同一目标的多幅图像各自对应的所述位姿信息，获取所述同一目标的位置信息。
具体地，对于每个至少匹配成功一次的目标，统计匹配成功的图像集合，并确定该图像集合中每一图像对应的位姿信息，得到位姿信息集合，基于该位姿信息集合以及该目标的像素位置（即目标检测得到的位置信息），估计该目标的位置信息。

例如，针对目标检测框集合，根据其集合内的每个目标检测框的范围，截取对应区域的图像并提取其子图的特征。例如，设目标检测框集合包含个目标检测框，从其对应的图像中根据每个目标检测框的范围，截取目标子图像，共有n个子图像。之后对每个子图像进行特征提取，获得每个子图像所对应的特征点集合。此步骤的特征提取，可采用结合了改进的FAST角点检测算法和二进制描述子的特征算法（Oriented FAST andRotated BRIEF，ORB）或尺度不变特征转换（Scale-Invariant Feature Transform，SIFT）等算子实施。
