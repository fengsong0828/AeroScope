CN119482930A

Description

低空线轨式3D电网飞行器运行网络系统
技术领域
本发明涉及低空飞行运行系统技术领域，特别是一种低空线轨式3D电网飞行器运行网络系统。
背景技术
低空飞行科技已经逐步进入人类的生活，工作以及各种社会应用。目前市场有了适合各种应用场合的低空飞行器(无人机)，初步形成了在低空飞行经济领域的一些规模效应，为日益发达及相对拥挤的地面交通做出很大的补充。
例如，中国专利CN112141334A公开了一种线轨式机器人电源智能运输飞行器系统，飞行器沿电网飞行并通过电网获取电能。
但是现有的低空飞行运行系统还不够完善，没有形成运行网络。
发明内容
本发明所要解决的技术问题是：现有的低空飞行运行系统还不够完善，没有形成运行网络。
本发明解决其技术问题所采用的技术方案是：一种低空线轨式3D电网飞行器运行网络系统，包括：
第一方向电网，由至少两条第一方向电网线轨构成，第一方向电网线轨都沿水平面的第一方向延伸，并且彼此之间没有交叉点；
第二方向电网，由至少两条第二方向电网线轨构成，第二方向电网线轨都沿水平面的第二方向延伸，并且彼此之间没有交叉点；
其中，第一方向电网线轨和第二方向电网线轨都用于飞行器空中取电，第一方向电网线轨和第二方向电网线轨架设在空中，第一方向电网与第二方向电网在高度方向上下分层设置，第一方向电网线轨和第二方向电网线轨构成具有交叉点的3D电网网格，3D电网网格在水平面上的投影呈网格状。
在一些实施例中，可选地，第一方向电网线轨和第二方向电网线轨都为整段或分段的电网线轨。
在一些实施例中，可选地，第一方向电网线轨和第二方向电网线轨为交流电网线轨或者直流电网线轨。
在一些实施例中，可选地，低空线轨式3D电网飞行器运行网络系统还包括起降平台，所述起降平台位于第一方向电网线轨和第二方向电网线轨附近。
在一些实施例中，可选地，低空线轨式3D电网飞行器运行网络系统还包括通信电缆，第一方向电网线轨和第二方向电网线轨的下方都具有通信电缆，通信电缆与第一方向电网线轨和第二方向电网线轨一起架设在空中。
在一些实施例中，可选地，低空线轨式3D电网飞行器运行网络系统，还包括飞行器，飞行器沿第一方向电网线轨或第二方向电网线轨取电飞行，并在飞行过程中，通过如下方式改变飞行方向，具体为：在第一方向电网线轨和第二方向电网线轨的交叉点处更换至另一方向电网线轨取电，并沿该另一方向电网线轨飞行。
在一些实施例中，可选地，第一方向电网线轨、第二方向电网线轨和通信电缆都通过电网塔架设在空中，在第一方向电网线轨和第二方向电网线轨的下方的通信电缆上都具有等间距设置的色标，用于作为第一方向和第二方向上的飞行参考物，在电网塔的不同高度上具有色标，用于作为高度方向上的飞行参考物。
在一些实施例中，可选地，第一方向和第二方向为相互垂直的两个方向。
在一些实施例中，可选地，低空线轨式3D电网飞行器运行网络系统，还包括起降平台、电网塔监控电子眼、电网塔通信接收发射器和后台管理站，起降平台设置在地面或者大厦楼顶，起降平台配套有供电控制系统、重力感应装置、监控系统、充电系统和通信系统，供电控制系统与重力感应装置、监控系统、充电系统、通信系统连接，用于为它们供电和控制，重力感应装置设置在起降平台上，用于感应飞行器的起降，监控系统用于监控起降平台的位置和工作状态，充电系统用于为飞行器充电，起降平台上具有带反射功能的字母式色标，通过在不同起降平台上设置不同的字母式色标，用于飞行器区分停泊在哪个起降平台，通过同一起降平台上的不同位置的不同的字母式色标，用于飞行器识别停泊在起降平台的那个位置，电网塔监控电子眼、电网塔通信接收发射器设置在电网塔上，通信电缆通过有线方式承接电网塔监控电子眼、电网塔通信接收发射器、起降平台和后台管理站之间的信息连接和互通，电网塔通信接收发射器用于通过无线方式接收和发射数据，并将数据通过通信电缆传输给后台管理站，电网塔监控电子眼用于获取电网塔周围的监控数据，把监控数据传送给后台管理站，后台管理站对接收到的数据分析处理后，做出正确的操作和管理指令，保障整个系统运行的安全可靠性。
在一些实施例中，可选地，电网塔通信接收发射器接收和发射数据包括卫星通讯数据、飞行器实时飞行反馈数据、气象环境数据、起降台反馈数据。
本发明的有益效果是：通过不同高度的第一方向电网和第二方向电网形成井井有条的3D低空运行网络，该3D低空运行网络，建设成本低，为高频次、高密度、长续航、多维度的低空飞行奠定了基础。
整个低空线轨式3D电网飞行器运行网络系统是以后台管理站为中心运作的一个低空交通地图系统，飞行器是最终运行单位，电网线轨是飞行器工作的载体、在通信电缆、起降平台、电网塔监控电子眼、电网塔通信接收发射器的辅助下，整个系统形成了一个闭环管理的3D运行网络，能够最大程度的确保低空飞行的安全、可靠、高效、高密度、高频次。
整个3D低空运行网络虽然还是需要一些电网基建，但是相对于地面公路系统、轨道交通系统建造和维护成本要绝对低很多。

具体实施方式
如图1和2所示，一种低空线轨式3D电网飞行器运行网络系统，包括：
第一方向电网，由至少两条第一方向电网线轨1构成，第一方向电网线轨1都沿水平面的第一方向延伸，并且彼此之间没有交叉点；
第二方向电网，由至少两条第二方向电网线轨2构成，第二方向电网线轨2都沿水平面的第二方向延伸，并且彼此之间没有交叉点；
其中，第一方向电网线轨1和第二方向电网线轨2都用于飞行器空中取电，第一方向电网线轨1和第二方向电网线轨2架设在空中，第一方向电网与第二方向电网在高度方向上下分层设置，第一方向电网线轨1和第二方向电网线轨2构成具有交叉点的3D电网网格，3D电网网格在水平面上的投影呈网格状。
飞行器沿第一方向电网线轨1或第二方向电网线轨2取电飞行，并在飞行过程中，通过如下方式改变飞行方向，具体为：在第一方向电网线轨1和第二方向电网线轨2的交叉点处更换至另一方向电网线轨取电，并沿该另一方向电网线轨飞行。
飞行器一般为无人飞行器，当然可以是有人飞行器，特别是具有自动驾驶功能的有人飞行器。
第一方向电网线轨1和第二方向电网线轨2通过电网塔3架设在空中，根据地面电网政策、低空规划要求和实际地形状况，设计第一方向电网线轨1和第二方向电网线轨2的高度和间距、电网塔3的位置和数量，以及第一方向电网线轨1和第二方向电网线轨2沿第一方向和第二方向延伸的具体走线。优选第一方向电网线轨1和第二方向电网线轨2沿第一方向和第二方向延伸都走直线，第一方向电网的各条第一方向电网线轨1彼此之间都平行设置，第二方向电网的各条第二方向电网线轨2彼此之间也都平行设置，构成矩形的3D电网网格，但是按照实际情况，第一方向电网线轨1和第二方向电网线轨2可能需要沿第一方向和第二方向蜿蜒曲折延伸，各条第一方向电网线轨1和各条第二方向电网线轨2也不完全平行设置，只是大致平行设置。
第一方向电网线轨1和第二方向电网线轨2统称为电网线轨，电网线轨直线延伸，电网塔3也直线排布，尽量设计第一方向电网线轨1的电网塔3高度一致，第二方向电网线轨2的电网塔3也高度一致。即第一方向电网和第二方向电网所处的面与水平面平行，只是两者之间具有高度差。
第一方向电网线轨1和第二方向电网线轨2为交流电网线轨，更具体为单相交流电网线轨，单相交流电网线轨的两根平行的线轨分别是火线4和零线5，根据设计要求可以传输高压电源，也可以传输低压电源。因为电网线轨不适合太长距离的悬空挂设或者为了匹配电网负载，确保电网负载均衡，第一方向电网线轨1和第二方向电网线轨2在延伸一定长度后断开，成为分段的电网线轨。飞行器在连续运行过程中遇到分段处时，可以跳跃飞行过分段处后继续沿电网线轨取电飞行。
第一方向电网线轨1和第二方向电网线轨2还可以是直流电网线轨，根据飞行器的用电属性确定，直流电网线轨的两根平行的线轨分别是正极线和负极线，电压根据设计要求选定。
第一方向和第二方向为相互垂直的两个方向，分别为纵向和横向，第一方向电网线轨1和第二方向电网线轨2的交叉高度是规划设计的一个安全高度，交叉点可以是互相悬空的，也可以位于同一电网塔3的不同高度的塔头上。
低空线轨式3D电网飞行器运行网络系统还包括起降平台7，起降平台7是指给飞行器提供的指定起飞或降落的特殊平台。起降平台7位于第一方向电网线轨1和第二方向电网线轨2附近，优选布置在电网塔3的周围，这样便于各种监控和通信设施的工作与运行。
起降平台7设置在地面或者大厦楼顶，起降平台7配套有供电控制系统、重力感应装置、监控系统、充电系统和通信系统，供电控制系统与重力感应装置、监控系统、充电系统、通信系统连接，用于为它们供电和控制，重力感应装置设置在起降平台7上，用于感应飞行器的起降，监控系统用于监控起降平台7的位置和工作状态，充电系统用于为飞行器充电，通信系统为有线和5G通信系统，监控系统至少包括电子眼监控和位置监控。
低空线轨式3D电网飞行器运行网络系统还包括通信电缆6、电网塔监控电子眼8、电网塔通信接收发射器9和后台管理站。
其中，第一方向电网线轨1和第二方向电网线轨2的下方的稍低的空间位置都具有通信电缆6，通信电缆6与第一方向电网线轨1和第二方向电网线轨2一起架设在空中。通信电缆6具体位于电网线轨的两根线轨之间，并与电网线轨的两根线轨相对平行独立。
通信电缆6不同于电网线轨，不是裸露的电缆，其具有绝缘等级较高的外表绝缘层。由于通信电缆6的刚性强度不大，所以安装时需要被高强度吊缆挂吊。
通信电缆6通过有线方式承接电网塔监控电子眼8、电网塔通信接收发射器9、起降平台7和后台管理站之间的信息连接和互通。
电网塔监控电子眼8分别安装在电网塔3的塔尖、塔腰、塔基部位的不同方向，电网塔监控电子眼8用于获取电网塔3周围的监控数据，把监控数据传送给后台管理站，后台管理站对接收到的数据分析处理后，做出正确的操作和管理指令，保障整个系统运行的安全可靠性。
电网塔通信接收发射器9设置在电网塔3上，具体可安装在电网塔3的塔尖上，可以传输卫星通讯数据、飞行器实时飞行反馈数据、气象环境数据、起降台反馈数据等整个电网运行时的产生的大数据。通过通信电缆6把大数据传输给后台管理站分析处理并且做出正确的操作和管理指令，保障整个低空飞行的安全可靠性。后台管理站实际就是整个系统的运行管理运算中心。
在第一方向电网线轨1和第二方向电网线轨2的下方的通信电缆6上都具有等间距设置的色标，该色标称为线轨色标，用于作为第一方向和第二方向上的飞行参考物，在电网塔3的不同高度上具有色标，该色标称为高度色标，用于作为高度方向上的飞行参考物。
在起降平台7上也具有色标，起降平台7上的色标采用带反射功能的字母式色标，通过在不同起降平台7上设置不同的字母式色标，用于飞行器区分停泊在哪个起降平台7，通过同一起降平台7上的不同位置的不同的字母式色标，用于飞行器识别停泊在起降平台7的那个位置。例如在图1中，中间一个电网塔3附近的起降平台7上的字母式色标为C1和C2，该起降平台7为起降平台C，在周围四角的电网塔3附近的起降平台7上的字母式色标分别为A1和A2、B1和B2、D1和D2、E1和E2，起降平台7分别为起降平台A、起降平台B、起降平台D和起降平台E。如果飞行器在停泊时识别到字母式色标C1，说明飞行器停泊在中间一个电网塔3附近的起降平台C的C1位置。为了显示起降平台7上的字母式色标在图1中绘制的起降平台7为起降平台7的正面。
假设停泊在起降平台A的A1位置的飞行器接受到飞行计划，目的地是起降平台E的E2位置。后台管理站对飞行器发出飞行指令，给出飞行路径。飞行器通过识别电网塔3上的高度色标，沿着电网塔3向上飞行到第二方向电网线轨2上，然后通过识别线轨色标沿第二方向电网线轨2飞到起降平台C附近的电网塔3的位置，然后通过识别高度色标和线轨色标下降至第一方向电网线轨1上，然后沿第一方向电网线轨1飞到起降平台E附近的电网塔3的位置，然后通过识别高度色标和起降平台E的字母式色标E2降落至起降平台E的E2位置，完成一次飞行过程。
具体地，通信电缆6上的色标安装在挂吊通信电缆6的吊缆上，色标为条状，由超轻航空铝型材加工而成，外表涂装强附着力的反射荧光粉，常规有银色和橙色两种颜色，安装时色标按照设计间距等间距安装在吊缆上。电网塔3的不同高度上的色标是指在电网塔3的不同高度位置上涂刷的强附着力的反射荧光粉，常规有银色和橙色两种颜色。起降平台7上的色标也为涂刷在起降平台7上的强附着力的反射荧光粉。当然，通信电缆6、电网塔3和起降平台7上的色标也可以采用其他的色标处理模式，色标颜色也可以根据实际要求定色。色标的长度根据具体应用选定。
本发明的低空线轨式3D电网飞行器运行网络系统以低空飞行的飞行器为飞行或运输单位，3D电网网格为飞行器的电源来源，其中的第一方向电网和第一方向电网有一定绝对的高度差，也就是第一方向电网和第一方向电网不在同一平面，因此，第一方向电网和第一方向电网与起降平台7的垂直起降线路形成了一个飞行器飞行的3D网络。各种低空飞行器可以在后台管理站的操作下，在各种保障系统的服务监控下，在3D网络中完成有序的起降、抓电网线轨充电、单向连续飞行、跳跃电网线轨断点飞行、更换不同方向电网线轨转向飞行、飞行中跳跃避让等各种飞行工作模式。
整个3D电网飞行器运行网络系统为低空载货载人飞行器运输提供了相对安全可靠的运行体系；因为电网直接供电的设计，理论上实现了飞行器无限续航；整个系统拥有有线无线和卫星系统，可同时工作，也可独立工作，可以保障整个运行系统的安全稳定性，系统得到的所有的监测大数据，可以为社会管理系统提供数据补充，与其他飞行航线互补，不影响特殊飞行航线；整个系统虽然还是需要一些电网基建，但是相对于地面公路系统和轨道交通系统，建造和维护成本要绝对低很多。
