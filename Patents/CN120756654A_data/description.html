Description

一种用于低空飞行器抗强风的辅助动力装置
技术领域
本发明涉及低空飞行器技术领域，具体为一种用于低空飞行器抗强风的辅助动力装置。
背景技术
低空区域空气运动复杂，突发风切变机率大、风力等级高(有些达到8级甚至10级以上)、持续时间不确定(几秒钟到几分钟不等)，而在山地、沙漠以及海上等环境下，突发风甚至达到13-16级。
目前，各类低空飞行器现阶段的抗风能力基本达到6级，最多到8级，无法在突发的大风下正常工作，特别是对于低空运人飞行器，在这短短几分钟内，如果不能维持正常姿态，将导致机毁人亡的重大安全事故。
针对上述问题，提出一种用于低空飞行器抗强风的辅助动力装置。
发明内容
本发明的目的在于提供一种用于低空飞行器抗强风的辅助动力装置，解决了背景技术中在不改变飞行器的原有构型的前提下为其提供应对突发大风的辅助动力组件的问题。
为实现上述目的，本发明提供如下技术方案：一种用于低空飞行器抗强风的辅助动力装置，包括；本体；机架，本体的外侧对称固定连接有多组机架；旋转叶，机架的上方转动连接有旋转叶；储罐主体，本体的下方通过螺栓拆卸连接有储罐主体；辅助机构，储罐主体的外侧设有辅助机构，辅助机构包含有真空法兰和拉瓦尔喷管，储罐主体的外侧分布有五组真空法兰，储罐主体的外侧通过真空法兰设有拉瓦尔喷管，五组拉瓦尔喷管分别分布于储罐主体的前后侧、左右侧和底端，储罐主体的内部设有绝缘机构和分层机构。
作为上述技术方案的进一步描述：所述绝缘机构包含有铝箔和玻璃纤维纸，铝箔和玻璃纤维纸依次叠加，储罐主体的内壁固定连接有铝箔。
作为上述技术方案的进一步描述：所述分层机构包含有导气管、多孔介质气化器和相变蓄冷环，储罐主体的内部通过星形支撑架固定连接有导气管，导气管的外侧壁固定连接有相变蓄冷环，导气管的一端与真空法兰固定连接，真空法兰的内部固定连接有多孔介质气化器，多孔介质气化器与拉瓦尔喷管固定连接。
作为上述技术方案的进一步描述：所述储罐主体的外侧固定连接有压电式压力传感器和控制器，控制器为比例积分微分三模仪器。
作为上述技术方案的进一步描述：位于前后方的所述拉瓦尔喷管与储罐主体呈112°分布，位于左右侧的所述拉瓦尔喷管与储罐主体呈105°分布。
与现有技术相比，本发明的有益效果如下：
1、本发明提供的一种用于低空飞行器抗强风的辅助动力装置，通过螺栓将储罐主体安装于本体下方，然后储罐主体内采用液态氮作为动力源，在常温下气化的相变膨胀效应，通过热交换器控制气体释放速率从而产生持续推力来实现抗风能力，避免大风天气对飞行器的影响，且液氮常压储存，避免了高压容器的重量和安全问题。
2、本发明提供的一种用于低空飞行器抗强风的辅助动力装置，通过储罐主体内部设有绝缘机构和分层机构实现对储罐主体的温度保护，且相变蓄冷环可以将多余热量进行吸收，增大使用效率，实现热阻断。

为进一步了解本发明的内容，结合附图对本发明作详细描述。
结合图1和图2，一种用于低空飞行器抗强风的辅助动力装置，包括：本体1；本体1的外侧对称固定连接有多组机架2；机架2的上方转动连接有旋转叶3；本体1的外侧通过机架2和旋转叶3实现现有的低空飞行器功能，为现有的飞行器原有构型；本体1的下方通过螺栓10拆卸连接有储罐主体4；通过螺栓10可以快速地对储罐主体4进行安装拆卸，储罐主体4内部填充有液态氮，储罐主体4的外侧设有辅助机构5，辅助机构5包含有真空法兰501和拉瓦尔喷管502，储罐主体4的外侧分布有五组真空法兰501，真空法兰501为锥面密封形状，从而与导气管701卡合，储罐主体4的外侧通过真空法兰501设有拉瓦尔喷管502，五组拉瓦尔喷管502分别分布于储罐主体4的前后侧、左右侧和底端，拉瓦尔喷管502通过记忆合金卡箍与真空法兰501锁紧固定；储罐主体4的内部设有绝缘机构6和分层机构7；在拉瓦尔喷管502喉部设置纳米疏水涂层，储罐主体4外壁集成PTC加热膜；储罐主体4为不锈钢内胆。
结合图3，储罐主体4的内壁固定连接有铝箔601，铝箔601和玻璃纤维纸602依次叠加，储罐主体4的内部通过星形支撑架704固定连接有导气管701，导气管701的外侧壁固定连接有相变蓄冷环703，导气管701的一端与真空法兰501固定连接，真空法兰501的内部固定连接有多孔介质气化器702，多孔介质气化器702与拉瓦尔喷管502固定连接，储罐主体4内部的液态氮进入到导气管701内，然后进入多孔介质气化器702内部进行反应，然后从拉瓦尔喷管502快速喷出，通过热交换器控制气体释放速率，从而适应不同的风力影响，通过现有的多孔介质气化器702内设有电加热丝，从而让液态氮膨胀，当外界热量对储罐主体4进行干扰时，通过绝缘机构6进行初步干扰，然后内部的相变蓄冷环703进行预热吸收，进行热阻断。
结合图3，储罐主体4的外侧固定连接有压电式压力传感器8和控制器9，控制器9为比例积分微分三模仪器，当外界有强风时，压电式压力传感器8检测到数据然后通过控制器9控制自带的左/右拉瓦尔喷管阀门开启从而实现矢量推力平衡风扰。
工作原理，使用时，通过旋转叶3和机架2实现本体1的飞行，且通过螺栓10实现储罐主体4与本体1的安装，从而适应不同的天气，在强风环境下进行储罐主体4的安装，在遇到强风时，压电式压力传感器8检测到数据然后通过控制器9控制自带的左/右拉瓦尔喷管阀门开启从而实现矢量推力平衡风扰；
液态氮进入导气管701内，然后多孔介质气化器702内的电加热丝实现液态氮的气化，然后经过多孔气质气化器702进入拉瓦尔喷管502内快速喷出，且通过热交换器可以控制气体的释放速率，从而适应不同的风力情况。
