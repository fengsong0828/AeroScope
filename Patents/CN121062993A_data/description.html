Description

一种有线电力低空飞行器
技术领域
本发明涉及有线电力飞行运输设备技术领域，具体为一种有线电力低空飞行器。
背景技术
有线电力低空飞行器是一种通过电缆或导线与地面电源连接、在低空运行的电动飞行器。其核心特点是摆脱传统电池续航限制，通过持续输电实现长时间飞行，同时借助低空环境避开常规航空交通，适用于特定场景的高效作业；该有线飞行器在飞行运输的过程中，由于电源线缆长度较长（通常在200米至230米之间），所以电源线缆的自重较大，对飞行器的负重提升了一定的负担，当飞行器上升的过程中，飞行器上升产生的拉力会对电源线缆产生垂直向下的空气阻力，使得电源线缆产生将飞行器向下方拉扯的拉力，且当飞行器沿水平方向前进或者后退时，电源线缆同样会受到与飞行器方向方向相反的空气阻力，进而对飞行器施加与其飞行方向相反的拉力，使得从而增加飞行器运输过程中的负担，使其难以保持平衡，严重情况甚至会导致飞行器坠落；缺少一种当有线电力飞行器上升或者水平方向飞行时，有效对飞行器移动过程中电源线缆受到的空气阻力进行抵消，以降低电源线缆对飞行器施加的拉扯力，从而确保飞行器平稳飞行的设备。
发明内容
本发明的目的在于提供一种有线电力低空飞行器，以解决上述背景技术中提出的问题。为实现上述目的，本发明提供如下技术方案：一种有线电力低空飞行器，包括飞行器本体，所述飞行器本体的底部设有用于运输货物的载货箱，飞行器本体的一侧设有电源线缆；还包括设于飞行器本体一侧的上升力抵消装置，所述上升力抵消装置包括设于飞行器本体一侧的下降偏转组件，下降偏转组件上设有第一磁铁，下降偏转组件的一侧设有配合组件，配合组件上设有第二磁铁，第一磁铁和第二磁铁处于同一水平高度且二者处于同性相斥的状态，上升力抵消装置的一侧设有水平力抵消装置。
优选的，所述下降偏转组件包括两个对称设于载货箱顶部一侧的偏转板，偏转板设有上部竖直且下部倾斜的偏转槽，两个偏转板之间的一侧设有升降架，两个偏转板之间设有偏转架，偏转架的一端与升降架上下滑动连接且偏转架与升降架的滑动连接处竖直设有第一弹簧，偏转架的两侧与两个偏转槽的上部滑动连接，偏转架的内侧设有安装架，安装架的两侧与偏转架和偏转槽的滑动连接处同轴连接，安装架远离飞行器本体的一侧呈敞开设置。
优选的，所述安装架内侧水平设有第一磁铁，第一磁铁的两侧分别与安装架转动连接且转动连接处设有卷簧，安装架的两侧均设有一个第一斜齿轮，且两个第一斜齿轮分别与第一磁铁和安装架的转动连接处同轴连接，所述偏转架顶部的两侧对称设有水平设置的第一受力板，升降架顶部设有穿线架。
优选的，所述配合组件包括两个对称且水平设于载货箱顶部的第一滑动架，每个第一滑动架顶部均竖直设有一个第二滑动架，每个第二滑动架分别与一个第一滑动架沿水平方向滑动连接且滑动连接处设有第二弹簧，两个第二滑动架之间竖向滑动连接有固定架，固定架与第二滑动架的滑动连接处设有第三弹簧，固定架上设有第二磁铁，电源线缆的一端穿过穿线架的上部和固定架的一端并与二者固定连接。
优选的，所述水平力抵消装置包括设于两个第二滑动架顶部的触发组件，触发组件的一侧设有传动组件，所述触发组件包括对称设于两个第二滑动架顶部的两个连接件，每个连接件靠近飞行器本体的一端均设有一个安装盒，每个安装盒的顶部呈敞开设置，飞行器本体的顶部设有转动架，转动架的一侧转动设有旋转轴，旋转轴穿过两个安装盒的侧端与其活动连接，每个安装盒的内侧均设有一个受力叶片，受力叶片的圆心处与旋转轴同轴连接，受力叶片的上部穿过安装盒顶部敞口处位于安装盒上方。
优选的，所述传动组件包括设于旋转轴两端的驱动齿轮，每个驱动齿轮和第一斜齿轮之间均设有一个侧端与飞行器本体侧端的滑动盒，滑动盒处水平滑动连接有一个齿槽杆，齿槽杆与滑动盒的滑动连接处设有第四弹簧，第四弹簧处于放松状态时齿槽杆位于驱动齿轮远离飞行器本体的一侧，齿槽杆的顶部与驱动齿轮啮合，齿槽杆的底部与第一斜齿轮啮合。
优选的，所述安装架呈敞开设置的一侧的下部设有封闭门，封闭门与安装架的侧端上下滑动连接且滑动连接处设有阻尼垫。
优选的，所述载货箱顶部远离第一磁铁的一端设有配重块。
与现有技术相比，本发明的有益效果：
本发明中，本装置在使用时，在初始阶段，第一磁铁和第二磁铁处于同一水平高度且处于同性相斥的状态，往载货箱内部装好需要运输的货物后，首先控制飞行器本体带着货物上升至指定高度，上升过程中，在空气阻力的影响下，电源线缆会对飞行器本体施加向下的拉力，在上升力抵消装置的作用下，对电源线缆与飞行器本体的连接端施加向上抬起的作用力，使得电源线缆对飞行器本体施加的向下的拉力被抵消掉一部分，从而缓解飞行器本体受到的来自下方的拉力，进而保障飞行器本体上升时的平稳性。
本发明中，当飞行器本体上升至指定高度并往指定区域水平飞行时，此时飞行器本体在前进过程中，由于电源线缆处于飞行器本体的尾部，在空气阻力的作用下，电源线缆对飞行器本体施加与其飞行方向相反的拉力，在传动组件和触发组件的配合下控制第一磁铁和第二磁铁处于异形相吸的状态，从而吸引第二磁铁往第一磁铁的方向靠近，使得电源线缆受到磁吸力靠近飞行器本体，通过磁吸力将电源线缆对飞行器本体施加的拉力进行抵消，从而缓解飞行器本体前进时受到电源线缆对飞行器本体施加的与前进方向相反的拉力；当送货完成后，需要控制飞行器本体返程，该飞行器本体返程无需掉头，沿着来时路线寻迹返航即可，返程时由于飞行器本体移动方向与前进时相反，此时电源线缆在空气阻力的影响下会对飞行器本体施加与返程方向相反的推力，通过触发组件和传动组件控制第一磁铁和第二磁铁重新处于同性相斥的状态，进而对电源线缆施加排斥力，排斥力与电源线缆对飞行器本体施加的推力相互抵消，从而缓解飞行器本体返程时受到电源线缆对飞行器本体施加的与返程方向相反的推力，结合上述原理，实现了当有线电力飞行器水平方向飞行时，有效对飞行器移动过程中电源线缆受到的空气阻力进行抵消，以降低电源线缆对飞行器施加的拉扯力或者推力，从而确保飞行器平稳飞行的效果。

本发明的使用方法和优点：该一种有线电力低空飞行器，工作过程如下：
本装置在使用时，在初始阶段，第一磁铁43和第二磁铁44处于同一水平高度且处于同性相斥的状态，往载货箱2内部装好需要运输的货物后，首先控制飞行器本体1带着货物上升至指定高度，上升过程中，在空气阻力的影响下，电源线缆3会对飞行器本体1施加向下的拉力，此时在空气阻力的作用下两个第一受力板418同步受力，带动偏转架413在升降架412上下降，使得偏转架413的两侧在两个偏转槽411内侧滑动至最底部，滑动的过程中带动第一磁铁43下降且逆时针偏转一定角度，使得第一磁铁43位于第二磁铁44的斜下方，进而利用二者同性相斥的原理对第二磁铁44施加向上抬起的作用力，进而对电源线缆3与飞行器本体1的连接端施加向上抬起的作用力，使得电源线缆3对飞行器本体1施加的向下的拉力被抵消掉一部分，从而缓解飞行器本体1受到的来自下方的拉力，进而保障飞行器本体1上升时的平稳性；
当飞行器本体1上升至指定高度并往指定区域水平飞行时，此时第一受力板418上升复位，第一磁铁43复位至与第二磁铁44平齐且水平的状态，且第一斜齿轮417与齿槽杆523的底部啮合，此时飞行器本体1在前进过程中，由于两个受力叶片515和电源线缆3处于飞行器本体1的尾部，在空气阻力的作用下，电源线缆3对飞行器本体1施加与其飞行方向相反的拉力，此时两个受力叶片515的上部在空气阻力的作用下顺时针旋转，带动旋转轴514转动，进而带动与齿槽杆523啮合的第一斜齿轮417转动180度，且在第四弹簧524的作用下维持住第一斜齿轮417的转动角度，进而带动位于安装架415内侧的第一磁铁43翻转180度，此时第一磁铁43和第二磁铁44处于异形相吸的状态，从而吸引第二磁铁44往第一磁铁43的方向靠近，第二滑动架422在第一滑动架421上滑动，使得电源线缆3受到磁吸力靠近飞行器本体1，通过磁吸力将电源线缆3对飞行器本体1施加的拉力进行抵消，从而缓解飞行器本体1前进时受到电源线缆3对飞行器本体1施加的与前进方向相反的拉力；当送货完成后，需要控制飞行器本体1返程，该飞行器本体1返程无需掉头，沿着来时路线寻迹返航即可，返程时由于飞行器本体1移动方向与前进时相反，此时电源线缆3在空气阻力的影响下会对飞行器本体1施加与返程方向相反的推力，受力叶片515同样受力往逆时针方向进行圆周运动，带动齿槽杆523与第一斜齿轮417脱离啮合，在卷簧416的作用下带动第一磁铁43反向翻转180度，第一磁铁43和第二磁铁44重新处于同性相斥的状态，进而对电源线缆3施加排斥力，排斥力与电源线缆3对飞行器本体1施加的推力相互抵消，从而缓解飞行器本体1返程时受到电源线缆3对飞行器本体1施加的与返程方向相反的推力，结合上述原理，实现了当有线电力飞行器水平方向飞行时，有效对飞行器移动过程中电源线缆3受到的空气阻力进行抵消，以降低电源线缆3对飞行器施加的拉扯力或者推力，从而确保飞行器平稳飞行的效果；
通过在安装架415一侧的底部设置封闭门6，当飞行器本体1闲置时，向上拉动封闭门6对第一磁铁43和第二磁铁44之间进行阻隔，使得第一磁铁43和第二磁铁44之间不再产生作用力，进而延长第一磁铁43和第二磁铁44的使用寿命；
通过在载货箱2顶部的另一侧设置与其余部件重量相同的配重块7，使得飞行器本体1两侧的重量相同，进而避免飞行器本体1运输过程中出现倾斜的状态，从而保持平稳状态进行飞行。
