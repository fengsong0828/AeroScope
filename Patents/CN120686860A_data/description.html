Description

一种跨介质低空无人飞行器自主切换与多机协同控制系统
技术领域
本发明涉及低空飞行器技术领域，特别涉及一种集跨介质自主切换、多源环境感知及分布式协同决策功能于一体的无人飞行器控制系统。
背景技术
随着无人飞行器在城市立体空间监测、水域救援及地下管网巡检等领域的应用深化，复杂介质环境下的可靠切换与多机高效协同需求日益凸显。传统跨介质飞行器系统依赖单一阈值触发机制（如水深>1m）或中心化任务调度，在高动态潮汐、湍流扰动及狭窄地下空间等复杂工况下面临三大技术瓶颈：
（1）感知可靠性不足：异构传感器（毫米波雷达/声呐/LiDAR）在介质切换时数据失效率＞25%（如声呐空中失效、雷达水下失效）；
（2）切换决策僵化：固定阈值法在潮汐波动场景误触发率＞30%；
（3）协同实时性差：中心化调度导致20节点规模下任务响应延迟＞500ms。
迄今为止，针对跨介质无人系统的研究仍存在显著局限：
（1）专利CN114802751A《水空跨介质无人航行器及其控制方法》提出基于水深阈值的推进模式切换方案，该方案未考虑密度梯度（▽ρ）对流体动力的影响，其实施例显示在潮汐流场（密度梯度▽ρ≥1.5kg/m³）中推进效率下降38.7%；
（2）专利CN113064438B《多无人机协同路径规划方法及系统》采用中心化任务分配机制，仿真显示20节点规模下主节点失效崩溃概率达43.6%，且缺乏地下管网拓扑约束；
（3）陈涛等人论文《跨介质飞行器多物理场耦合建模研究》指出磁流体推进器在瞬态切换中易产生空化效应，但未解决专利CN116136559A《一种水空跨介质航行器密封装置》实测涵道密封响应延迟＞100ms引发的动力中断问题。
因此，亟需一种创新性跨介质控制系统，其应具备通过多源时空融合算法（误差抑制至＜5%）解决感知失真问题、基于六维动态决策模型（置信度≥98%）突破切换可靠性瓶颈、构建分布式协同架构（响应延迟≤80ms）实现多机高效协作的能力。以满足城市立体空间作业中对跨介质适应性、狭窄空间通过性及应急响应实时性的严苛需求。

与现有技术相比，本发明具有如下有益效果：
本发明提出了一套创新的三栖协同控制系统，用于解决无人飞行器在跨介质环境下的切换与协同难题。采用可变形涵道结构与磁流体推进器，实现空-水-地动力模式自主重构，并通过接触角>150°的超疏水涂层和模块化减震设计，显著提升介质切换可靠性与结构耐久性；利用联邦卡尔曼滤波时空对齐与SVM动态决策模型，大幅降低感知失真率至<5%及切换误触发率至1.3%，生成高精度六维环境状态矩阵；分布式协同架构与介质自适应路径规划确保多机任务响应延迟≤80ms，并在Φ28cm地下管网实现100%避碰成功率；异构通信网关通过时延<10ms的5G专网和速率2Mbps的蓝绿激光双链路冗余，保障跨介质数据传输完整性。

S52、介质自适应路径规划：
当系统处于空中模式时，根据RRT*-Smart算法，由约束条件湍流代价因子λ=0.1进行介质自适应路径规划；当系统处于水下模式时，根据A*-NS算法，由约束条件Navier-Stokes流体积分进行介质自适应路径规划；当系统处于地下模式时，根据Topo-Dijkstra算法，由约束条件最小转弯半径R≥0.4m进行介质自适应路径规划；
例如：水下救援任务中，领航机（剩余电量>80%）通过声呐定位目标，动态合同网协议在42ms内分配3台无人机形成三角搜索阵型，A-NS算法规划顺暗流路径（节能27%）。

S61、主链路：5G专网（20MHz@3.5GHz，端到端时延<10ms）；
S62、备份链路：
空中切换至水下：由蓝绿激光（532nm，2Mbps）切换至水声通信（50kHz）；
地下切换至空中：使用北斗短报文（传输延迟<5s）；
S63、安全加密系统架构：
采用分层加密策略，在主链路与备份链路部署差异化加密模块，分为5G专网加密层、空-水激光/水声加密层、地-空北斗加密层；
5G专网加密层：集成国密SM4算法的硬件加密芯片（支持20Gbps吞吐量），基于量子密钥分发的动态密钥更新机制（QKD），密钥更新周期≤30秒，通信模组PCB采用16层埋阻设计，关键信号走线实施蛇形屏蔽罩包裹；
空-水激光/水声加密层：采用混沌激光调制加密（CLME）技术，通过半导体激光器注入扰动实现波形混淆，部署改进的OFDM-Hadamard变换加密，每帧数据插入32位动态校验码，水声换能器前置FPGA加密模块（XilinxZynqUltraScale+，工作频率400MHz）；
地-空北斗加密层：应用北斗三号军用级加密通信协议（BD-SAM），采用128位AES-CBC加密+CRC32校验双保险结构，SIM卡槽集成自毁熔丝电路（触发电流≥3A）；
S64、跨介质加密同步机制：
建立基于可信执行环境（TEE）的密钥管理单元，通过硬件安全模块（HSM）存储根密钥，介质切换时触发密钥同步协议，同步时延≤50ms，密钥错误率<10-9；
例如：在渤海跨介质试验中，当飞行器从空中（5G链路）切入水下（激光→水声链路）时：加密控制器检测到介质切换指令（SVM输出P=0.992），触发CLME混沌激光加密初始化，生成初始向量IV=0x7A3E9D（通过水声信道传输），水声接收端FPGA加载256位动态密钥（更新周期2秒），实测数据传输误码率≤10-6，成功抵御每秒5×105次暴力破解攻击。
