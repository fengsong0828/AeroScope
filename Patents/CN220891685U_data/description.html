Description

一种低空空域飞行器监视装置
技术领域
本实用新型涉及监视装置技术领域，具体为一种低空空域飞行器监视装置。
背景技术
民用航空飞行高度一般在6000米以上，低空空域指的是1000米以下的飞行区域，在2009年的全国低空空域管理改革研讨会上，有关方面提出将3000米以下的空域分为3类：管制空域、监视空域、报告空域，空域管理改革工作会议确定真高一千米以下空域将实行分类管理有序放开，随着社会的发展，低空空域的开放势必将带动超万亿元的市场经济规模，以及整个产业链的急速发展黄金期，但是有开放无管制，势必会造成低空空域飞行的混乱，以无人飞机为代表的低空空域飞行器如果没有按照指定航线飞行，或者乱闯禁飞区域，后果将不堪设想，所以有必要对低空空域飞行器进行监视。
如授权公告号为CN115373419B所公开的本发明提供一种超低空飞行器侦察监视方法及装置，方法包括：以预置光学采集设备，在预设时刻采集光学设备位置信息及不少于2批的超低空飞行器方位信息、俯仰信息及距离信息，初测获取超低空飞行器的距离数据，传输光学设备位置信息、超低空飞行器方位信息、俯仰信息及距离数据至数据采集终端，以数据采集终端获取光学设备位置信息、超低空飞行器方位信息、俯仰信息及距离数据，以进行坐标转换预处理以及航迹显示预处理，以得到采集终端预处理信息，以第i部数据采集终端将采集终端预处理信息通过预置通信设备传输至数据融合终端，以对超低空飞行器的航迹进行融合处理；
其虽然实现了本发明解决了侦察连续性较差、精度较低以及监视稳定性较低的技术问题，但是并未解决现有的监视装置在使用时不利于便捷的旋转式控制监视，不利于监视装置便捷的多角度移动控制调节，影响了监视装置探测监视的范围，影响了监视装置调节的范围，大大影响了监视装置转动调节的灵活性。
实用新型内容
本实用新型的目的在于提供一种低空空域飞行器监视装置，以解决上述背景技术中提出监视装置不便于便捷的旋转式控制监视，影响了监视装置探测监视的范围，而且不便于监视装置便捷的多角度移动控制调节，影响了监视装置调节的范围，影响了监视装置转动调节的灵活性的问题。
为实现上述目的，本实用新型提供如下技术方案：一种低空空域飞行器监视装置，包括底座和支撑柱，所述底座的顶端中心位置处安装有支撑柱，所述支撑柱的顶端安装有支撑盘，所述支撑盘的顶端安装有旋转箱，所述旋转箱的外壁上活动安装有支撑臂，所述旋转箱远离支撑臂的一侧外壁上活动安装有限位臂，且支撑臂与限位臂之间安装有C型臂，所述C型臂的外壁上安装有连接臂，所述连接臂的内部安装有监视控制器，所述监视控制器的表面安装有探测雷达，所述探测雷达一侧的监视控制器表面安装有监视器，所述监视器下方的监视控制器表面安装有等间距的三组声音探测器，所述底座上方的支撑柱内部安装有驱动电机，所述驱动电机的输出端安装有驱动轴，所述支撑盘的内部安装有旋转盘，所述驱动轴的顶端与旋转盘相连接。
优选的，所述旋转箱的内部安装有步进电机，所述步进电机的输出端安装有转动轴，且转动轴延伸至旋转箱的外部，所述旋转箱一侧的支撑臂外壁上安装有连接盘，所述转动轴远离步进电机的一端与连接盘相连接，且转动轴延伸至支撑臂的内部。
优选的，所述转动轴一侧的支撑臂外壁上安装有伺服电机，所述伺服电机的输出端安装有活动轴，且活动轴经过支撑臂与C型臂相连接，所述C型臂一侧的限位臂外壁上活动安装有连接轴，且连接轴延伸至C型臂的内部。
优选的，所述连接轴一侧的限位臂外壁上活动安装联动轴，且联动轴延伸至限位臂的内部，并且联动轴与旋转箱相连接。
优选的，所述底座的底端安装有行走车，所述底座的表面安装有等间距的多组锁紧螺栓，且锁紧螺栓延伸至行走车的内部。
优选的，所述旋转箱外壁上安装有控制面板，所述控制面板的输出端与伺服电机、驱动电机、步进电机的输入端电性连接。
与现有技术相比，本实用新型的有益效果是：该监视装置不仅实现了便捷的旋转式控制监视，扩大了监视装置探测监视的范围，而且实现了监视装置便捷的多角度移动控制调节，增加了监视装置调节的范围，提高了监视装置转动调节的灵活性；
(1)通过由驱动电机驱动驱动轴旋转，由驱动轴带动旋转盘旋转，由旋转盘带动支撑盘整体旋转，由支撑盘带动旋转箱转动，由旋转箱带动支撑臂转动，由支撑臂带动C型臂转动，由C型臂带动连接臂转动，由连接臂带动监视控制器转动，由监视控制器带动探测雷达、监视器、声音探测器同步转动，当所述飞行目标靠近禁飞区域时，由监视器与声音探测器反馈至监视控制器，由监视控制器对其发出警报信号，使飞行目标撤离，实现了监视装置便捷的旋转式控制监视，扩大了监视装置探测监视的范围；
(2)通过由步进电机驱动转动轴旋转，由转动轴带动连接盘旋转，由连接盘带动支撑臂转动，由支撑臂带动C型臂进行转动，由C型臂带动连接臂进行转动，由连接臂带动监视控制器进行转动，由监视控制器带动探测雷达、监视器、声音探测器进行转动，来进行位置调节监视，由伺服电机驱动活动轴旋转，由活动轴带动C型臂转动调节，限位臂通过连接轴对C型臂进行限位支撑转动，由连接臂带动监视控制器进行转动高度调节，由监视控制器带动探测雷达、监视器、声音探测器进行高度调节，实现了监视装置便捷的多角度移动控制调节，增加了监视装置调节的范围，提高了监视装置转动调节的灵活性。

具体实施方式
下面将结合本实用新型实施例中的附图，对本实用新型实施例中的技术方案进行清楚、完整地描述。基于本实用新型中的实施例，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他实施例，都属于本实用新型保护的范围。
请参阅图1-5，本实用新型提供的一种实施例：一种低空空域飞行器监视装置，包括底座1和支撑柱2，底座1的顶端中心位置处安装有支撑柱2，支撑柱2的顶端安装有支撑盘3，支撑盘3的顶端安装有旋转箱4，旋转箱4的外壁上活动安装有支撑臂5，旋转箱4远离支撑臂5的一侧外壁上活动安装有限位臂25，且支撑臂5与限位臂25之间安装有C型臂7，C型臂7的外壁上安装有连接臂9，连接臂9的内部安装有监视控制器20，监视控制器20的表面安装有探测雷达16，探测雷达16一侧的监视控制器20表面安装有监视器17，监视器17下方的监视控制器20表面安装有等间距的三组声音探测器18，底座1上方的支撑柱2内部安装有驱动电机10，驱动电机10起到动力输出的作用，驱动电机10的输出端安装有驱动轴11，支撑盘3的内部安装有旋转盘12，驱动轴11的顶端与旋转盘12相连接；
使用时，外接电源，首先通过操作控制面板24打开驱动电机10，在底座1的支撑下，由驱动电机10驱动驱动轴11旋转，由驱动轴11带动旋转盘12旋转，由旋转盘12带动支撑盘3整体旋转，由支撑盘3带动旋转箱4转动，由旋转箱4带动支撑臂5转动，由支撑臂5带动C型臂7转动，由C型臂7带动连接臂9转动，由连接臂9带动监视控制器20转动，由监视控制器20带动探测雷达16、监视器17、声音探测器18同步转动，监视控制器20型号为广域雷达微波检测器DTAM D10类似装置，探测雷达16型号为346式雷达类似民用装置，监视器17型号为海康威视ECU-3XXX高清摄像头，声音探测器18型号为NA-39A声测系统，探测雷达16、监视器17、声音探测器18接收来自北斗卫星的信号，其与终端设备信号连接，由探测雷达16探测1000米以下的低空空域飞行目标，并获取的飞行目标的三维坐标信息反馈至监视控制器20，由监视控制器20将飞行目标的信息传输至监视器17与声音探测器18，由监视器17与声音探测器18来对飞行目标区域进行探测监视，当飞行目标靠近禁飞区域时，由监视器17与声音探测器18反馈至监视控制器20，由监视控制器20对其发出警报信号，使飞行目标撤离，实现了监视装置便捷的旋转式控制监视，扩大了监视装置探测监视的范围；
旋转箱4的内部安装有步进电机13，步进电机13起到动力输出的作用，步进电机13的输出端安装有转动轴14，且转动轴14延伸至旋转箱4的外部，旋转箱4一侧的支撑臂5外壁上安装有连接盘15，转动轴14远离步进电机13的一端与连接盘15相连接，且转动轴14延伸至支撑臂5的内部，转动轴14一侧的支撑臂5外壁上安装有伺服电机6，伺服电机6起到动力输出的作用；
伺服电机6的输出端安装有活动轴21，且活动轴21经过支撑臂5与C型臂7相连接，C型臂7一侧的限位臂25外壁上活动安装有连接轴8，且连接轴8延伸至C型臂7的内部，连接轴8一侧的限位臂25外壁上活动安装联动轴22，且联动轴22延伸至限位臂25的内部，并且联动轴22与旋转箱4相连接，底座1的底端安装有行走车23，底座1的表面安装有等间距的多组锁紧螺栓19，且锁紧螺栓19延伸至行走车23的内部，旋转箱4外壁上安装有控制面板24，控制面板24的输出端与伺服电机6、驱动电机10、步进电机13的输入端电性连接；
使用时，通过操作控制面板24打开步进电机13，由步进电机13驱动转动轴14旋转，由转动轴14带动连接盘15旋转，由连接盘15带动支撑臂5转动，在限位臂25的限位支撑下，由支撑臂5带动C型臂7进行转动，限位臂25通过联动轴22对支撑臂5进行限位支撑，由C型臂7带动连接臂9进行转动，由连接臂9带动监视控制器20进行转动，由监视控制器20带动探测雷达16、监视器17、声音探测器18进行转动，来进行位置调节监视，同时操作控制面板24打开伺服电机6，在支撑臂5的支撑下，由伺服电机6驱动活动轴21旋转，由活动轴21带动C型臂7转动调节，限位臂25通过连接轴8对C型臂7进行限位支撑转动，由C型臂7带动连接臂9进行转动高度调节，由连接臂9带动监视控制器20进行转动高度调节，由监视控制器20带动探测雷达16、监视器17、声音探测器18进行高度调节，来进行高度调节监视，同时可以推动行走车23，带动监视装置进行行走式监视，实现了监视装置便捷的多角度移动控制调节，增加了监视装置调节的范围，提高了监视装置转动调节的灵活性。
