Description

一种低空飞行器的航路变更方法
技术领域
本发明属于低空交通领域，特别涉及一种低空飞行器的航路变更方法。
背景技术
随着城市空中交通(UAM)和物流无人机的大规模应用，低空空域内无人机运行密度呈指数级增长。面对城市复杂低空环境下规模化的异构航空器运行，在现有无人机自主避障技术，空域资源分配，无人机自动化管控技术缺陷下，规模化飞行器常态化安全运行是目前需要研究的重大问题，低空公共航路可以约束海量的飞行器的飞行方向和路径，使飞行器在规定的飞行路径内有序运行，因此航路网络的科学布局以及公共航路的构建尤为重要。对于规模化飞行器运行过程中任务实时变更，本模型的提出对于维持航路内规模化飞行器高速安全运行具有较高价值意义。
在对低空交通场景下的航路网络与低空航路构建方面，目前的相关研究较为缺乏，传统的航向变更占用较大的空域且耗时较长，此外传统无人机飞行缺乏速度过渡机制，使得无人机整体运行效率低下，难以满足低空交通场景下高密度高速无人机安全运行需求。
发明内容
为了解决上述背景技术提到的现有问题，本发明提出了一种低空飞行器的航路变更方法。
本发明所述的一种低空飞行器的航路变更方法，步骤如下：
基于干线航路网络及支线航路网络构建的航路模型；
干线航路网络采用分层式三维航路结构；支线航路网络采用换向功能子网络及航速跃迁功能子网络构成；
航路变更方法包括航向调整及航速跃迁：
当低空飞行器得到航向调整指令后，低空飞行器进入换向功能子网络中的换向功能航路爬升、下降、转向，使低空飞行器完成航向变更；
当低空飞行器得到航速跃迁指令后，低空飞行器进入航速跃迁功能子网络中的航速跃迁功能航路，在分层式三维航路结构的各层之间进行航速变更。
本发明所述的低空飞行器的航路变更方法，所述换向功能航路采用螺旋结构航路；螺旋曲线切线与干线航路网络中的干线航路轴线夹角30°≤α≤60°，螺旋半径随高度层变化调整表述式如下：

其中K值由航速和航路周围环境决定；
航速跃迁功能航路采用螺旋结构航路，螺旋曲线切线与干线航路网络中的干线航路轴线夹角30°≤α≤60°，螺旋半径随速度变化调整表述式如下：

其中λ为加速系数；
所述的分层式三维航路结构中各层航速基准值为：V0+ΔV、航向约束为θ+Δθ。
本发明所述的低空飞行器的航路变更方法，换向功能子网络的螺旋结构或航速跃迁航路子网络的螺旋结构中螺旋结构航路均采用顺时针与逆时针转向。
本发明所述的低空飞行器的航路变更方法，所述干线航路网络结构中设置低空飞行器的加速准备区，螺旋加速区，缓冲区；在加速准备区，螺旋加速区、缓冲区中飞行的低空飞行器之间设定水平安全间隔、垂直安全间隔。

有益效果
低空航路不同于传统航路的最大特点在于，其飞行器的高密度高速安全运行需求。基于这一特征，本发明创造性提出了一种低空飞行器的航路变更方法，能够满足规模化无人机在三维分层航路内高速运行过程中，面对任务变更或飞行状态调整时，具有较强的适应性，飞行器不仅可以实时变更飞行状态，还可以使原航路内飞行器高速运行的干扰降到最低。

本发明提出了一种低空飞行器的航路变更方法，包括以下步骤：
(1)航路模型由干线航路网络和支线航路网络构成，所述支线航路网络包括换向功能子网络和航速跃迁功能子网络；
(2)所述干线航路网络采用分层式三维航路结构，各层航速设置有航速基准值(V0+ΔV)和航向约束(θ+Δθ)，相邻航路层之间保持安全间隔(Hmin)；
(3)所述换向功能子网络采用螺旋结构，其螺旋曲线切线与干线航路轴线夹角30°≤α≤60°，螺旋半径随高度层变化按ΔR＝K·h的关系调整，其中K值由航速和航路周围环境决定。
(4)所述航速跃迁航路子网络采用螺旋结构，其螺旋曲线切线与干线航路轴线夹角30°≤α≤60°，螺旋半径随速度变化调整ΔR＝λ·ΔV/V0的关系调整，其中λ为加速系数。
进一步地，所述的支线功能航路网络可根据城市地理状况与无人机飞行变更需求分布于干线航路网络两侧，螺旋结构支线航路可分别采用顺时针与逆时针转向；
优选地，当无人机原航路正常飞行时收到任务变更通知，实现在最小干扰航路内其他无人机的航路高度层变更，应设置加速准备区，螺旋加速区，缓冲区；

实施例一：
某城市核心区低空航路网络采取“东单西双”的原则，采用东西向分层的三维航路结构。干线航路由三层组成，分别为：
低速层L1：(60-120m，航速基准值为10m/s,航向东向)；
反向层L2：(120-180m，航速基准值为10m/s,航向西向)；
高速层L3：(180-240m，航速基准值为30m/s,航向东向)；
相邻层间垂直间隔Hmin＝60m,支线功能航路分布于干线两侧，换向功能子网络采用初始螺旋半径为150m，螺旋曲线切线与干线夹角45°；
案例1(航速跃迁)：
L1层一架配送无人机运行速度10m/s，东向，接收到系统命令，需要减少配送时间，提前到达配送目的地，该无人机需紧急提速至L3层，安全系数γ为1.4，响应时间为2s，制动加速度为3m/s2，定位误差容限ΔS为5m，水平安全间距为35.7m,无人机确认前方空域无冲突后进入准备区，加速系数λ为1.5，螺旋距参数P为80m，螺旋曲线长度为612m。完成加速后无人机即将航路合并进入缓冲合并区，航路合并余量Δ为3m，缓冲区长度为56.34m，无人机在缓冲区内调整航向角，平稳进入L3高速层航路。
案列2(航向调整)
L1层一架巡检无人机运行速度10m/s，东向，在接到任务后需要调转方向，需要进入换向功能支线航路，螺旋半径为72m，无人机切入逆时针螺旋支路，通过72m半径螺旋路径爬升并反转航向180°，取空气密度ρ为1.225kg/m3，最大爬升率vz为5m/s，湍流补偿系数η为1.1，垂直间隔控制在51.8m，L2层航路并行航路数3，航速基准值为10m/s，无人机等效长度Lu为8m，水平安全间隔Dh为35.7m，容量评估为0.69架次/s。
