Description

一种ETC结合低空飞行器的停车缴费系统及其方法
技术领域
本发明属于停车收费技术领域，具体涉及一种ETC结合低空飞行器的停车缴费系统及其方法。
背景技术
电子收费系统(ETC,Electronic Toll Collecting System)作为一种现代化的收费方式，已广泛应用于高速公路及桥梁的收费管理中。该系统通过车辆上安装的车载单元(OBU,On Board Unit)与收费站车道上的路边单元(RSU,Road Side Unit)之间的无线通信和信息交换，实现了车辆的不停车收费。ETC系统主要由车辆自动识别系统、中心管理系统和其他辅助设施构成。其中，车辆自动识别系统包括车载单元、路边单元以及环路感应器等组件，车载单元存储有车辆的识别信息，通常安装于车辆前挡风玻璃上，而路边单元则安装于收费站旁，环路感应器则埋设在车道地面下。当车辆经过收费站时，环路感应器首先感知到车辆的存在，随后路边单元发出询问信号，车载单元响应并进行双向通信和数据交换，中心管理系统则根据获取的车辆识别信息与数据库中的信息对比，进而控制管理系统的相应动作，如从预付款项账户中扣除过路费。随着ETC技术的不断发展，其应用范围日益广泛，然而，传统的ETC系统中RSU通常只能固定设置在路侧，主要应用于高速公路收费系统，对于其他场景的适应性存在局限性。
为了扩展ETC技术的应用范围，公告号CN202976217U的中国专利文献提出了一种基于ETC技术的智能停车场管理系统。该系统包括服务器单元、入口单元和出口单元，入口和出口单元均配备有ETC读头，这些读头通过控制器与管理计算机相连，而入口和出口单元的管理计算机又通过交换机与服务器相连。此外，入口单元还配备了票箱机、语音提示机、显示屏、入口摄像机、道闸和地感等设备，用于处理未安装ETC电子标签的外来车辆。出口单元则配备了相应的语音提示机、显示屏、出口摄像机、道闸和地感。这种智能停车场管理系统能够提供高效、智能、安全的停车管理服务。然而，该系统仍存在一些不足之处：(1)该停车场管理系统主要适用于封闭的停车场环境，可以在出入口处设置龙门架以承载ETC感应设备，从而实现ETC自动感应收费。对于开放式的路侧停车场景，由于路边单元的感应范围有限，难以实现ETC停车收费；(2)由于当前交通系统中ETC车辆与非ETC车辆混行的情况普遍存在，单纯依靠ETC收费系统难以对所有车辆进行路侧停车收费，往往需要人工介入粘贴收缴单；(3)由于停车收费系统通常是根据车辆进入泊位至离开泊位的时间进行结算，如果用户驶离泊位后忘记缴费，会导致停车欠费，进而影响企业的资金回流。
综上所述，尽管ETC技术在高速公路收费和封闭停车场管理中展现出了高效便捷的优势，但在面对开放式的路侧停车场景时，现有的ETC收费系统及其扩展应用仍存在诸多局限性。特别是在混合交通流、开放式停车环境以及用户缴费习惯多样性等复杂场景下，现有技术难以全面满足停车收费管理的需求。因此，针对现有技术中存在的问题，亟需提供一种ETC结合低空飞行器的停车缴费系统及其方法，以实现对所有车辆的自动识别和高效收费，同时提升用户的使用体验和企业的运营效率。
发明内容
针对相关技术中的问题，本发明提出一种ETC结合低空飞行器的停车缴费系统及其方法，以克服现有相关技术所存在的上述技术问题。
本发明的技术方案是这样实现的：一种ETC结合低空飞行器的停车缴费系统，用于对路边泊位内的停泊车辆进行计时收费；所述路边泊位的一侧至少设有立柱，所述立柱的上部设有泊车计时装置、飞行器缴费装置和控制总成；所述控制总成分别与泊车计时装置、飞行器缴费装置通信连接；
所述泊车计时装置包括监控模块和计时模块；所述监控模块用于监控驶入路边泊位内的停泊车辆，并调动所述计时模块对该停泊车辆单独停车计时；所述计时模块依据停车时长内置有多个计时阈值，当停车时长达到计时阈值时，所述泊车计时装置反馈给所述控制总成，所述控制总成向所述飞行器缴费装置发送缴费指令；
所述飞行器缴费装置包括起落平台和无人机；所述起落平台设于所述立柱上，用于为所述无人机提供停靠空间，以及充电供能；所述无人机包括无人机本体，以及设于无人机本体上的路边模块、缴费打印模块和机械爪；当所述飞行器缴费装置接收到所述缴费指令后，所述无人机飞行至待缴费的停泊车辆处，无人机上的路边模块尝试与停泊车辆进行信息交互，并发出扣费指令；
所述无人机根据停泊车辆是否响应扣费指令，调动所述缴费打印模块打印对应的缴费单据，并通过所述机械爪将所述缴费单据，夹持在停泊车辆的挡风玻璃与雨刷器之间。
本发明通过无人机实现远程扣费和信息交互，无需依赖固定的ETC感应设备，从而适用于各种开放式路侧停车场景；此外，通过无人机在停车时长达到计时阈值后即进行缴费提醒和扣费操作，有效避免了用户忘记缴费的情况。
作为以上方案的进一步改进，若停泊车辆响应扣费指令，则所述停泊车辆为ETC车辆；所述ETC车辆上至少设有车载模块，所述车载模块接收所述路边模块发射的微波信号后被激活，向路边模块反馈车辆信息；所述路边模块根据所述车辆信息，关联停泊车辆的ETC账户并发出扣费指令，车载单元响应扣费；所述缴费打印模块打印的缴费单据上附有二维码，所述二维码关联有停泊车辆的扣费信息；
若停泊车辆未响应扣费指令，则所述停泊车辆为非ETC车辆；所述缴费打印模块打印的缴费单据上附有二维码，所述二维码关联有停泊车辆的欠费信息。
作为以上方案的进一步改进，若停泊车辆为ETC车辆，则当停车时长达到下一个计时阈值时，所述无人机飞行至待停泊车辆，执行扣费操作；缴费单据上二维码关联的扣费信息实时更新；
若停泊车辆为非ETC车辆，则当停车时长达到下一个计时阈值时，无人机不执行扣费操作；缴费单据上二维码关联的欠费信息实时更新。
作为以上方案的进一步改进，所述机械爪包括安装座，所述安装座的一端固定在所述无人机本体上，另一端设有多节依次连接的转动臂，各节转动臂设有独立的驱动源。
作为以上方案的进一步改进，末节转动臂上设有所述缴费打印模块，所述缴费打印模块设有出单口，所述缴费单据从所述出单口输出；沿所述出单口延伸有两个平板，两个平板之间具有供所述缴费单据穿行的输送通道；两个平板延伸的末端对应连接有上夹板和下夹板，其中，所述上夹板由驱动单元驱动，上夹板绕其连接处转动以使两个夹板相抵或相离，对应关闭或打开所述输送通道；
当缴费打印模块打印对应的缴费单据后，机械爪的所述上夹板和下夹板初始抵接，机械爪的上夹板和下夹板共同插入挡风玻璃与雨刷器之间的间隙；所述上夹板由驱动单元驱动，打开输送通道，且抬起雨刷器；此时，缴费单据自所述输送通道输出，机械爪抽离，使所述缴费单据由所述雨刷器抵压在挡风玻璃上。
作为以上方案的进一步改进，两个平板之间设有多个转动滚轴，所述缴费单据由所述转动滚轴动力输送。
作为以上方案的进一步改进，所述监控模块设有图像摄取单元，用于摄取停泊车辆的图像，并从所述图像中获取停泊车辆的车牌号；所述计时模块根据停泊车辆的车牌号建立或归入对应的计时档案，所述计时档案包括停泊车辆的车牌号、泊位号、泊车开始时间、泊车结束时间，以及历史停车记录。
作为以上方案的进一步改进，所述立柱的上部还设有照明模块、太阳能板和储能电池；所述太阳能板与储能电池连接，所述储能电池分别与所述照明模块、起落平台电连接；所述起落平台上设有无线充电模块，并通过所述无线充电模块为所述无人机充电供能。
作为以上方案的进一步改进，所述无人机上还设有摄像模块，所述摄像模块配置为拍摄路边泊位内的停泊车辆，进行取证。
一种ETC结合低空飞行器的停车缴费方法，应用在如上所述的一种ETC结合低空飞行器的停车缴费系统上，包括以下步骤：
S1：停泊车辆驶入路边泊位，所述监控模块对停泊车辆进行监控，并调动所述计时模块对该停泊车辆单独停车计时；所述计时模块依据停车时长内置有多个计时阈值，当停车时长达到计时阈值时，所述泊车计时装置反馈给所述控制总成，所述控制总成向所述飞行器缴费装置发送缴费指令；
S2：当所述飞行器缴费装置接收到所述缴费指令后，所述无人机飞行至待缴费的停泊车辆处，无人机上的路边模块尝试与停泊车辆进行信息交互，并发出扣费指令；
S3：所述无人机根据停泊车辆是否响应扣费指令，调动所述缴费打印模块打印对应的缴费单据，并通过所述机械爪将所述缴费单据，夹持在停泊车辆的挡风玻璃与雨刷器之间。
本发明的有益效果：
(1)解决开放式路侧停车场景ETC收费难题：传统的停车场管理系统主要依赖封闭的停车场环境和龙门架上的ETC感应设备实现自动收费。然而，在开放式的路侧停车场景中，由于路边模块的感应范围有限，ETC自动收费系统难以有效覆盖。本发明的ETC结合低空飞行器的停车缴费系统，通过在路边泊位设置立柱，并在立柱上部署泊车计时装置、飞行器缴费装置和控制总成，有效解决了这一难题。该系统能够精准监控停泊车辆，并通过无人机实现远程扣费和信息交互，无需依赖固定的ETC感应设备，从而适用于各种开放式路侧停车场景，提高了收费系统的灵活性和覆盖率。
(2)实现ETC车辆与非ETC车辆的混合收费：当前交通系统中，ETC车辆与非ETC车辆混行的情况普遍存在。传统的ETC收费系统难以对所有车辆进行有效收费，往往需要人工介入粘贴收缴单，这不仅增加了人力成本，也降低了收费效率。本发明的系统通过无人机上的路边模块与停泊车辆进行信息交互，并发出扣费指令。对于ETC车辆，只要其车载设备支持信息交互，即可实现自动扣费。对于不支持信息交互的非ETC车辆，无人机还可以打印缴费单据并夹持在车辆挡风玻璃与雨刷器之间，实现了对所有停泊车辆的混合收费，大大提高了收费系统的通用性和效率。
(3)解决停车欠费问题，提升资金回流：传统的停车收费系统通常是根据车辆进入泊位至离开泊位的时间进行结算。然而，如果用户驶离泊位后忘记缴费，会导致停车欠费，进而影响企业的资金回流。本发明的系统通过无人机在停车时长达到计时阈值后即进行缴费提醒和扣费操作，有效避免了用户忘记缴费的情况。同时，无人机打印的缴费单据也作为缴费凭证，方便用户核对和补缴费用。这种即时扣费和凭证打印的方式，不仅减少了停车欠费现象，还提升了企业的资金回流速度和运营效率。

下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅是本发明的一部分实施例，而不是全部的实施例。基于本发明中的实施例，本领域普通技术人员在没有作出创造性劳动前提下所获得的所有其他实施例，都属于本发明保护的范围。
如图1-图5所示，一种ETC结合低空飞行器的停车缴费系统，用于对路边泊位内的停泊车辆进行计时收费；所述路边泊位的一侧至少设有立柱1，所述立柱1的上部设有泊车计时装置11、飞行器缴费装置2和控制总成(图中未绘出)；所述控制总成分别与泊车计时装置11、飞行器缴费装置2通信连接；
所述泊车计时装置11包括监控模块和计时模块；所述监控模块用于监控驶入路边泊位内的停泊车辆，并调动所述计时模块对该停泊车辆单独停车计时；所述计时模块依据停车时长内置有多个计时阈值，当停车时长达到首个计时阈值时，所述泊车计时装置11反馈给所述控制总成，所述控制总成向所述飞行器缴费装置2发送缴费指令；
所述飞行器缴费装置2包括起落平台21和无人机22；所述起落平台21设于所述立柱1上，用于为所述无人机22提供停靠空间，以及充电供能；所述无人机22包括无人机本体23，以及设于无人机本体23上的路边模块24(即RSU)、缴费打印模块25和机械爪26；当所述飞行器缴费装置2接收到所述缴费指令后，所述无人机22飞行至待缴费的停泊车辆处，无人机22上的路边模块24尝试与停泊车辆进行信息交互，并发出扣费指令；
所述无人机22根据停泊车辆是否响应扣费指令，调动所述缴费打印模块25打印对应的缴费单据，并通过所述机械爪26将所述缴费单据，夹持在停泊车辆的挡风玻璃3与雨刷器4之间。
在本实施例中，若停泊车辆响应扣费指令，则所述停泊车辆为ETC车辆；所述ETC车辆上至少设有车载模块(即OBU)，所述车载模块接收所述路边模块24发射的微波信号后被激活，向路边模块24反馈车辆信息；所述路边模块24根据所述车辆信息，关联停泊车辆的ETC账户并发出扣费指令，车载单元响应扣费；所述车辆信息包括停泊车辆的车牌号及ETC账户。
所述缴费打印模块25打印的缴费单据上附有二维码，所述二维码关联有停泊车辆的扣费信息；若停泊车辆未响应扣费指令，则所述停泊车辆为非ETC车辆；非ETC车辆可能未配置车载模块，或者OBU未启用、OBU非法拆卸、OBU挂失、卡挂失、卡禁用、余额不足等情况。所述缴费打印模块25打印的缴费单据上附有二维码，所述二维码关联有停泊车辆的欠费信息。通过车载模块与路边模块24的微波信号交互，ETC车辆能够迅速响应扣费指令，完成费用的自动扣除，为ETC用户提供了更为便捷、高效的停车缴费体验。当停泊车辆未响应扣费指令时，系统能够自动判定其为非ETC车辆，并通过缴费打印模块25打印出含有欠费信息的二维码缴费单据。车主只需扫描二维码，即可轻松完成缴费，避免了人工介入的繁琐和不便。
传统的ETC收费系统难以对所有车辆进行有效收费，往往需要人工介入粘贴收缴单，这不仅增加了人力成本，也降低了收费效率。本发明的系统通过无人机22上的路边模块24与停泊车辆进行信息交互，并发出扣费指令。对于ETC车辆，只要其车载设备支持信息交互，即可实现自动扣费。对于不支持信息交互的非ETC车辆，无人机22还可以打印缴费单据并夹持在车辆挡风玻璃3与雨刷器4之间，实现了对所有停泊车辆的混合收费，大大提高了收费系统的通用性和效率。
在本实施例中，若停泊车辆为ETC车辆，则当停车时长达到下一个计时阈值时，所述无人机22飞行至待停泊车辆，执行扣费操作；缴费单据上二维码关联的扣费信息实时更新；若停泊车辆为非ETC车辆，则当停车时长达到下一个计时阈值时，无人机22不执行扣费操作；缴费单据上二维码关联的欠费信息实时更新。本实施例通过无人机22在停车时长达到计时阈值后即进行缴费提醒和扣费操作，有效避免了用户忘记缴费的情况。同时，无人机22打印的缴费单据也作为缴费凭证，方便用户核对和补缴费用。这种即时扣费和凭证打印的方式，不仅减少了停车欠费现象，还提升了企业的资金回流速度和运营效率。

在本实施例中，末节转动臂262上设有所述缴费打印模块25，所述缴费打印模块25设有出单口251，所述缴费单据从所述出单口251输出；沿所述出单口251延伸有两个平板27，两个平板27之间具有供所述缴费单据穿行的输送通道271；两个平板27延伸的末端对应连接有上夹板281和下夹板282，其中，所述上夹板281由驱动单元283驱动，上夹板281绕其连接处转动以使两个夹板相抵或相离，对应关闭或打开所述输送通道271；所述驱动单元283为电动推杆，所述电动推杆的座体与平板27相固接，电动推杆的杆头与上夹板281传动连接。在本实施例中，两个平板27之间设有多个转动滚轴272，所述缴费单据由所述转动滚轴272动力输送。
当缴费打印模块25打印对应的缴费单据后，机械爪26的所述上夹板281和下夹板282初始抵接，机械爪26的上夹板281和下夹板282共同插入挡风玻璃3与雨刷器4之间的间隙；所述上夹板281由驱动单元283驱动，打开输送通道271，且抬起雨刷器4；此时，缴费单据自所述输送通道271输出，机械爪26抽离，使所述缴费单据由所述雨刷器4抵压在挡风玻璃3上。
需要说明的是，传统的停车收费系统通常是根据车辆进入泊位至离开泊位的时间进行结算。然而，如果用户驶离泊位后忘记缴费，会导致停车欠费，进而影响企业的资金回流。为提高资金回流效率，有必要放置缴费单据以达到提醒功能。一般需要人工介入放置所述缴费单据，本停车缴费系统通过无人机22搭载机械爪26实现缴费单据的自动投放，极大地提高了投放效率和准确性。相较于人工放置，无人机22能够迅速抵达指定位置，避免了因人力不足或交通拥堵而导致的单据投放延迟，同时也减少了人为错误的可能性。其次，机械爪26的设计精巧且功能强大。通过多节依次连接的转动臂262和独立的驱动源263，机械爪26能够灵活调整姿态，准确地将缴费单据投放到车辆挡风玻璃3与雨刷器4之间的间隙中。这种设计不仅确保了单据的稳固放置，还避免了因放置位置不当而导致的单据丢失或损坏。此外，缴费打印模块25与机械爪26的协同工作也大大提升了用户体验。缴费单据在打印完成后，能够立即通过机械爪26投放到车辆上，无需人工干预。同时，上夹板281和下夹板282的设计使得单据能够平稳地输出并固定在挡风玻璃3上，既方便车主查看，又避免了单据被风吹落或丢失的风险。

通过上述方案，在具体应用中，一种ETC结合低空飞行器的停车缴费方法，包括以下步骤：
S1：停泊车辆驶入路边泊位，所述监控模块对停泊车辆进行监控，并调动所述计时模块对该停泊车辆单独停车计时；所述计时模块依据停车时长内置有多个计时阈值，当停车时长达到计时阈值时，所述泊车计时装置11反馈给所述控制总成，所述控制总成向所述飞行器缴费装置2发送缴费指令；
S2：当所述飞行器缴费装置2接收到所述缴费指令后，所述无人机22飞行至待缴费的停泊车辆处，无人机22上的路边模块24尝试与停泊车辆进行信息交互，并发出扣费指令；
S3：所述无人机22根据停泊车辆是否响应扣费指令，调动所述缴费打印模块25打印对应的缴费单据，并通过所述机械爪26将所述缴费单据，夹持在停泊车辆的挡风玻璃3与雨刷器4之间。
本停车缴费系统通过无人机22实现远程扣费和信息交互，无需依赖固定的ETC感应设备，从而适用于各种开放式路侧停车场景；此外，通过无人机22在停车时长达到计时阈值后即进行缴费提醒和扣费操作，有效避免了用户忘记缴费的情况。
