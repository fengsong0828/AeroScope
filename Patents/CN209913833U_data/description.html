Description

低空飞行器防御系统
技术领域
本申请涉及低空飞行器(包括小型固定翼飞机、旋翼机、直升机、无人机、飞行汽车、飞行货箱等在真高3000米以下的飞行目标)防御技术领域，尤其涉及一种低空飞行器防御系统。
背景技术
随着低空飞行器的不断普及，低空飞行器给人们日常生活带来了较多的便利。但是，由于低空飞行器具有体积小、速度快、隐蔽性强等特点，较易被不法分子利用于毒品走私、投掷危险物、非法拍摄等，给社会安全防范带来了巨大的挑战。
因此，如何保证各区域不受小型低空飞行器的干扰和侵扰，以维护社会安全，成为亟待解决的难题。
发明内容
本申请提出一种低空飞行器防御系统，以实现当低空飞行器进入第二区域后，由干扰装置对低空飞行器进行多方位干扰，使低空飞行器无法定位、通信，从而迫使低空飞行器降落或者返航。
本申请一方面实施例提出了一种低空飞行器防御系统，包括：
探测雷达，所述探测雷达用于对第一区域内进行扫描以探测所述第一区域内出现的低空飞行器，并获取所述低空飞行器的位置信息，其中，所述第一区域包括至少一个第二区域；
视频监视装置，所述视频监视装置用于根据所述低空飞行器的位置信息对所述低空飞行器进行跟踪，并通过显示器显示所述低空飞行器的视频图像；
干扰装置，所述干扰装置用于在所述低空飞行器的位置信息处于所述第二区域内时，通过发射天线阵列向所述第二区域发送多个工作频段的干扰信号，所述干扰信号用于对所述低空飞行器的通信链路进行干扰；
监控中心平台，所述监控中心平台与所述探测雷达、所述视频监控装置和所述干扰装置相互通信，所述监控中心平台用于在所述低空飞行器的位置信息处于所述第二区域内时生成报警信息，并根据所述报警信息生成第一控制指令，并将所述第一控制指令发送至所述干扰装置，其中，所述第一控制指令用于控制所述干扰装置的开启和关闭。
本申请实施例的低空飞行器防御系统，通过探测雷达对第一区域内进行扫描以探测第一区域内出现的低空飞行器，并获取低空飞行器的位置信息，通过视频监视装置根据低空飞行器的位置信息对低空飞行器进行跟踪，并通过显示器显示低空飞行器的视频图像，并通过干扰装置在低空飞行器的位置信息处于第二区域内时，通过发射天线阵列向第二区域发送多个工作频段的干扰信号，其中，干扰信号用于对低空飞行器的通信链路进行干扰，以及通过监控中心平台在低空飞行器的位置信息处于第二区域内时生成报警信息，并根据报警信息生成第一控制指令，并将第一控制指令发送至干扰装置，其中，第一控制指令用于控制干扰装置的开启和关闭。由此，当低空飞行器进入第二区域后，由干扰装置对低空飞行器进行多方位干扰，使低空飞行器无法定位、通信，从而迫使低空飞行器降落或者返航。

如图1所示，该低空飞行器防御系统可以包括：探测雷达110、视频监视装置120、干扰装置130以及监控中心平台140。
其中，探测雷达110，用于对第一区域内进行扫描以探测第一区域内出现的低空飞行器，并获取低空飞行器的位置信息，其中，第一区域包括至少一个第二区域。
本申请实施例中，低空飞行器可以为小型固定旋翼机、直升机、无人机、飞行汽车、飞行货箱等在真高3000米以下的飞行目标，其中，无人机可以为小型固定翼无人机、多旋翼无人机、“蜂群”无人机等，对此不作限制。
本申请实施例中，第一区域为预先设置的，可以根据用户的实际需求以及具体应用场景进行设置，比如，该第一区域可以为办公区域、居民的活动区域、人流密集区域等等，对此不作限制。举例而言，当该低空飞行器防御系统应用于人群密集的车站时，第一区域可以为包含整个车站以及车站周边预设范围内的区域。
本申请实施例中，探测雷达110的个数为至少一个，当探测雷达110的个数为一个时，该探测雷达110可以位于第一区域的中心位置，方位覆盖范围为360度，扫描区域覆盖第一区域，比如探测范围可以为500米至10千米，即探测距离可以为500米至10千米。当探测雷达110的个数为多个时，多个探测雷达110的设置位置不作限制，但是，多个探测雷达110的扫描区域需覆盖第一区域。
本申请实施例中，探测雷达110用于对第一区域内的低空飞行器进行探测，得到低空飞行器的位置信息。需要说明的是，探测雷达110还可以获取低空飞行器的飞行状态信息，其中，飞行状态信息包括低空飞行器与探测雷达110之间的距离，以及低空飞行器飞行的方位角和俯仰角。
作为一种可能的实现方式，由于Ku波段的地面干扰很小，并且Ku波段频率高，一般在12.5-18GHz之间，不易受微波辐射干扰，大大地降低了探测雷达110对接收环境的要求，因此，本申请中，探测雷达110的工作频段可以为Ku波段，或者，探测雷达110的工作频段还可以为X波段，X波段是指频率在8-12GHz的无线电波波段。
视频监视装置120，用于根据低空飞行器的位置信息对低空飞行器进行跟踪，并通过显示器显示低空飞行器的视频图像。
本申请实施例中，探测雷达110在获取低空飞行器的位置信息之后，视频监视装置120可以根据低空飞行器的位置信息，对低空飞行器进行实时跟踪，并通过显示器显示低空飞行器的视频图像，以使相关工作人员通过该视频图像，实时观察该低空飞行器。
作为一种可能的实现方式，该视频监视装置120可以为光电跟踪仪，包括摄像装置、激光照明组件、万向变速室外云台、多功能解码器和显示器，由该光电跟踪仪根据低空飞行器的位置信息对低空飞行器进行跟踪，并通过显示器显示低空飞行器的视频图像。即本申请中，可以通过光电跟踪仪捕获、跟踪、锁定低空飞行器。
其中，光电跟踪仪的方位转动角的范围为0-360°，光电跟踪仪的工作频段可以为红外频段，光电跟踪仪的搜索范围可以为50米至3千米。
干扰装置130，用于在低空飞行器的位置信息处于第二区域内时，通过发射天线阵列向第二区域发送多个工作频段的干扰信号，干扰信号用于对低空飞行器的通信链路进行干扰。
本申请实施例中，至少一个第二区域也为预先设置的，比如至少一个第二区域可以设置在第一区域的四周内侧。
本申请实施例中，干扰装置130指的是发射或转发电子干扰信号的设备，该干扰装置100可以为固定式干扰装置，或者还可以为手持式干扰装置，比如手持式干扰枪，本申请对此并不做限制。其中，干扰装置130的干扰距离不小于2千米。
本申请实施例中，当低空飞行器的位置信息处于第二区域内时，可以通过干扰装置130对该低空飞行器进行干扰，通过发射天线阵列向第二区域发送多个工作频段的干扰信号，其中，干扰信号用于对低空飞行器的通信链路进行干扰，使低空飞行器无法定位、通信、失去控制，从而驱使低空飞行器驶出第二区域或促使低空飞行器迫降。
本申请实施例中，为了实现对低空飞行器进行多方位干扰以及定向干扰，干扰装置130可以产生不同频率的电子干扰信号。具体地，该干扰装置130可以通过发射天线阵列向第二区域发送多个工作频段的干扰信号，以对低空飞行器的通信链路进行干扰，使低空飞行器无法定位、通信，从而迫使低空飞行器降落或者返航。
其中，多个工作频段可以根据需求进行设置，比如可以为北斗卫星导航系统B1频段、卫星定位系统(Global Positioning System，简称GPS)频段(比如为民用GPS频段，如车载导航仪的工作频段、行驶记录仪的工作频段)、无线保真(Wireless Fidelity，简称WiFi)频段。
作为一种可能的实现方式，为了实现对低空飞行器进行全方位干扰以及定向干扰，多个工作频段可以覆盖全频段，比如，多个频段可以包括：810-1300MHz、1520-1580MHz、2400-2500MHz、5725-5850MHz、5600-5940MHz。
监控中心平台140，与探测雷达110、视频监控装置120和干扰装置130相互通信，监控中心平台140用于在低空飞行器的位置信息处于第二区域内时生成报警信息，并根据报警信息生成第一控制指令，并将第一控制指令发送至干扰装置130，其中，第一控制指令用于控制干扰装置130的开启和关闭。
本申请实施例中，当监控平台140根据探测雷达110获取的低空飞行器的位置信息，确定该低空飞行器位于第二区域时，可以生成报警信息，以提醒相关工作人员，其中，报警信息可以为声、光、文字等形式。同时，还可以根据报警信息生成第一控制指令，并将该第一控制指令发送至干扰装置130。相应的，当干扰装置130接收到该第一控制指令后，可以根据该第一控制指令开启设备，对低空飞行器进行多方位干扰。
或者，该第一控制指令还可以用于控制干扰装置130发射干扰信号，对低空飞行器进行干扰，当干扰装置130接收到该第一控制指令后，可以直接根据该第一控制指令，向第二区域发送多个工作频段的干扰信号。
本申请实施例的低空飞行器防御系统，通过探测雷达对第一区域内进行扫描以探测第一区域内出现的低空飞行器，并获取低空飞行器的位置信息，通过视频监视装置根据低空飞行器的位置信息对低空飞行器进行跟踪，并通过显示器显示低空飞行器的视频图像，并通过干扰装置在低空飞行器的位置信息处于第二区域内时，通过发射天线阵列向第二区域发送多个工作频段的干扰信号，其中，干扰信号用于对低空飞行器的通信链路进行干扰，以及通过监控中心平台在低空飞行器的位置信息处于第二区域内时生成报警信息，并根据报警信息生成第一控制指令，并将第一控制指令发送至干扰装置，其中，第一控制指令用于控制干扰装置的开启和关闭。由此，当低空飞行器进入第二区域后，由干扰装置对低空飞行器进行多方位干扰，使低空飞行器无法定位、通信，从而迫使低空飞行器降落或者返航。
可以理解的是，实际应用时，进入第一区域的飞行目标可能并非低空飞行器，例如，小鸟、风筝等飞行物体，此时，探测雷达110能够检测到该飞行目标，但是，并不能确定该飞行目标是否为低空飞行器，因此，本申请中，为了提升低空飞行器防御的准确性，需要对探测雷达110探测到的飞行目标进行识别，以确定该飞行目标是否为低空飞行器。
作为一种可能的实现方式，可以由人工对该飞行目标进行识别。具体地，当探测雷达110探测并获取到飞行目标的位置信息后，可以将飞行目标的位置信息发送至监控中心平台140，监控中心平台140可以将飞行目标的位置信息发送至视频监视装置120，以使视频监视装置120根据所述低空飞行器的位置信息对所述低空飞行器进行跟踪。同时，视频监视装置120还可以通过显示器显示低空飞行器的视频图像，相关工作人员通过该视频监视装置120显示的视频图像，可以确定进入第一区域的飞行目标是否为低空飞行器，若是，则通知监控中心平台140飞行目标为低空飞行器，当监控中心平台140根据低空飞行器的位置信息，确定该低空飞行器处于第二区域内时，可以向干扰装置发送第一控制指令，以对低空飞行器进行干扰。
作为另一种可能的实现方式，还可以基于图像识别技术，确定该飞行目标是否为低空飞行器。具体地，监控中心平台140可以从视频监视装置120获取视频图像，对视频图像进行图像检测，以确定视频图像中是否包含低空飞行器，例如，可以基于深度学习的图像识别技术，对视频图像进行检测，比如可以获取训练图像，对训练图像中的低空飞行器进行标注，利用标注后的训练图像对识别模型进行训练，根据训练后的识别模型对视频图像进行识别，可以确定视频图像中是否包含低空飞行器，若是，则在根据低空飞行器的位置信息，确定该低空飞行器处于第二区域内时，可以向干扰装置发送第一控制指令，以对低空飞行器进行干扰。
需要说明的是，当确定飞行目标并非低空飞行器时，可以不对该飞行目标进行防御，并通过探测雷达110继续探测出现在第一区域内的飞行目标，并通过上述方法，识别该飞行目标是否为低空飞行器，直到识别该飞行目标为低空飞行器时，可以通过该低空飞行器防御系统，对低空飞行器进行防御。
作为一种可能的实现方式，第一区域中还可以包括第三区域，例如，参见图2，第三区域21可以设置在第一区域20的中心位置，至少一个第二区域22沿第三区域21的四周设置，第三区域21的安全等级高于第二区域22的安全等级。比如，可以将第一区域的中心位置作为危险区域，即该危险区域为第三区域，并在第三区域的外周设置四个干扰区域，作为第二区域。
举例而言，当该低空飞行器防御系统应用于人群密集的车站时，第一区域可以为包含整个车站以及车站周边预设范围内的区域，第三区域可以为整个车站区域，第二区域可以为车站的周边区域。
可以理解的是，当低空飞行器处于第二区域时，若干扰装置130对低空飞行器进行全方位干扰，未能迫使低空飞行器降落或者返航，此时，低空飞行器可能飞行至第三区域，由于第三区域的安全等级较高，因此，可以对低空飞行器进行激光打击，由激光对低空飞行器机体或关键部位进行损伤，从而使低空飞行器坠毁或丧失飞行能力，以防止低空飞行器对第三区域进行非法侦测、恐袭等，可以有效提高第三区域的安全系数，确保第三区域的安全有序。
具体地，参见图3，在图1所示实施例的基础上，该低空飞行器防御系统还可以包括：激光发射器150。
其中，激光发射器150，用于在低空飞行器的位置信息处于第三区域内时，发射激光束以击落低空飞行器。
本申请实施例中，当低空飞行器的位置信息处于第三区域内时，可以通过激光发射器150发射激光束，以击落该低空飞行器，即可以通过激光发射器150对低空飞行器进行激光打击，由激光对低空飞行器机体或关键部位进行损伤，从而使低空飞行器坠毁或丧失飞行能力，以击落该低空飞行器。
其中，激光发射器150的工作波段可以为1微米，干扰距离不小于2千米，损伤距离不小于500米。比如，在低空飞行器的飞行高度大于2千米时，激光发射器150发射的激光束能够干扰低空飞行器的飞行，而在低空飞行器的飞行高度小于500米时，激光发射器150发射的激光束能够对低空飞行器机体或关键部位进行损伤，从而使低空飞行器坠毁或丧失飞行能力，以击落该低空飞行器。
监控中心平台140，还用于与激光发射器150相互通信，在低空飞行器的位置信息处于第三区域时，生成第二控制指令，并将第二控制指令发送至激光发射器150，其中，第二控制指令，用于控制激光发射器150的开启和关闭。

或者，该第二控制指令还可以用于控制激光发射器150向低空飞行器发射激光束，当激光发射器150接收到该第二控制指令后，可以根据该第二控制指令，对低空飞行器进行激光打击。
