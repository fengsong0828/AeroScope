Description

一种基于飞行需求的低空飞行器公共航路网规划方法
技术领域
本申请涉及低空航路规划技术领域，尤其涉及一种基于飞行需求的低空飞行器公共航路网规划方法。
背景技术
低空公共数字航路是指通过数字化建模的方式为低空飞行器在低空空域生成的结构化导航路径，其设计的核心目标是实现多类型飞行器的有序运行。现有技术中，低空航路规划主要基于空域网格化分层方法。
现有方法存在显著局限性：一方面，空域分层规则与任务类型脱节，未针对载物、载人、应急任务分别设计独立空域层及优先级策略。在未来高密度低空飞行器运行的情况下，载人和载物飞行器飞行速度不同，如果在同一高度中混合飞行，容易降低飞行效率，也增加了碰撞风险。同时，载人飞行器噪声大，载物飞行器噪声小，两者适合在不同高度的空域飞行，以降低飞行器噪声对居民的影响。另一方面，航路规划缺乏对实际起降点通行能力与飞行需求量的综合考虑。距离较远且飞行需求量较大的起降点之间更适合建立一条飞行高度较高的航路，类似于高速公路，在该航路中，飞行器可以更快的飞行速度飞行，以提升整体低空航路网的效率。
发明内容
本申请实施例的主要目的在于提出一种基于飞行需求的低空飞行器公共航路网规划方法，以至少解决一个现有技术存在的问题。
为实现上述目的，本申请实施例的一方面提出了一种基于飞行需求的低空飞行器公共航路网规划方法，所述方法包括以下步骤：
将空域根据高度划分为载物层、载人层、应急层；
将载物层根据高度划分为多个子载物层；
根据载物飞行需求量和起降点通行能力，计算每两个起降点之间的载物航路需求容量指数；
按照从高到低的顺序规划子载物层内的航路；
在每个子载物层内，计算起降点对的航路优先级，根据起降点对的航路优先级从大到小遍历起降点对，如果各个起降点对未在任何一个子载物层中存在已规划的航路，且对应起降点对的连线未与当前子载物层其他航路相交，则在对应起降点对之间规划一条航路；
利用与规划载物层相同的方法规划载人层的航路，对应急层则不进行航路规划。
