Description

一种低空飞行器安全管理系统及方法
技术领域
本发明涉及飞行安全技术领域，尤其涉及一种低空飞行器安全管理系统及方法。
背景技术
随着低空经济的发展，低空飞行器的应用场景日益广泛，广泛应用于物流配送、载人飞行、旅游消费、应急救援等多种场景中；然而，传统的低空飞行器监管方案存在诸多安全问题，如地面网络覆盖不足、数据传输延迟、防碰撞能力不足等问题，严重影响低空飞行器的飞行安全。
因此，本发明提供一种低空飞行器安全管理系统及方法。
发明内容
本发明提供一种低空飞行器安全管理系统及方法，通过对低空飞行器进行飞行预安全管理，检测低空飞行器的设备状态以及评估低空飞行器的飞行计划；构建低空飞行器的飞行安全管理模型，根据当前飞行安全影响因素以及低空飞行器的飞行数据进行分析评估以及分析预警，获取低空飞行器的当前飞行风险值，对低空飞行器进行实时安全管理；当低空飞行器完成飞行计划时，获取飞行数据进行数据归档分析，优化低空飞行器的未来飞行计划；实现了对低空飞行器的飞行安全管理。
本发明提供一种低空飞行器安全管理系统，包括：
飞行预管理模块，用于对低空飞行器进行飞行预安全管理，检测低空飞行器的设备状态以及评估低空飞行器的飞行计划；
飞行管理模块，用于构建低空飞行器的飞行安全管理模型，根据当前飞行安全影响因素以及低空飞行器的飞行数据进行分析评估以及分析预警，获取低空飞行器的当前飞行风险值，对低空飞行器进行实时安全管理；
飞行后管理模块，用于当低空飞行器完成飞行计划时，获取飞行数据进行数据归档分析，优化低空飞行器的未来飞行计划。
根据本发明提供的一种低空飞行器安全管理系统，飞行预管理模块，包括：
飞行预安全管理单元，用于当低空飞行器在当前时刻存在飞行计划时，标记为预飞行低空飞行器，对低空飞行器进行飞行预安全管理，其中，飞行预安全管理包括：设备状态检测安全管理以及飞行计划评估安全管理；
设备状态检测单元，用于对预飞行低空飞行器的设备状态进行检测，当判定设备状态处于正常运行状态时，判定预飞行低空飞行器的设备状态检测安全管理通过，其中，设备状态包括：正常运行状态、异常运行状态以及故障状态；
飞行计划评估单元，用于获取预飞行低空飞行器的飞行计划并进行飞行计划评估，当飞行计划评估结果显示飞行计划合格时，允许预飞行低空飞行器执行起飞操作，其中，飞行计划包括：预设飞行路线、预设飞行高度、预设飞行区域以及预设飞行许可。
根据本发明提供的一种低空飞行器安全管理系统，飞行计划评估单元，包括：
飞行区域评估块，用于对预飞行低空飞行器的预设飞行区域进行评估，获取历史飞行区域，若预飞行低空飞行器的预设飞行区域与历史飞行区域重合，则判定预飞行低空飞行器的预设飞行区域已知，飞行区域评估合格；
若预飞行低空飞行器的预设飞行区域与历史飞行区域不重合，则判定预飞行低空飞行器的预设飞行区域未知，获取未知飞行区域的可能风险因素进行飞行区域评估，其中，可能风险因素包括：飞行障碍物风险因素以及飞行禁区风险因素；
根据预飞行低空飞行器的预设飞行许可确定飞行禁区风险因素；
根据飞行高度以及飞行路线对未知飞行区域进行评估；
飞行高度评估块，用于获取预设飞行区域的当前天气数据，获取预飞行低空飞行器的可飞行高度范围，对预设飞行高度进行评估；
当预设飞行高度处于可飞行高度范围时，判定飞行高度评估合格；
当预设飞行高度不处于可飞行高度范围时，判定飞行高度评估不合格，根据可飞行高度范围对预设飞行高度进行校正；
飞行路线评估块，用于对预飞行低空飞行器的预设飞行路线进行评估，当预设飞行路线避开飞行障碍物风险因素以及避开飞行禁区风险因素时，判定飞行路线评估合格；
当飞行区域评估合格、飞行高度评估合格以及飞行路线评估合格时，判定飞行计划评估结果显示飞行计划合格。
根据本发明提供的一种低空飞行器安全管理系统，飞行管理模块，包括：
数据监测单元，用于当预飞行低空飞行器执行起飞操作时，根据预部署传感器以及定位设备对低空飞行器进行实时监测，获取低空飞行器的飞行数据，其中，低空飞行器的飞行数据包括：飞行环境数据、飞行状态数据以及飞行计划数据；
数据处理单元，用于对低空飞行器的飞行数据进行数据处理，剔除噪声数据以及异常数据；
对剔除噪声数据以及异常数据后的飞行数据进行数据特征提取，获取低空飞行器的飞行数据特征。
根据本发明提供的一种低空飞行器安全管理系统，飞行管理模块，包括：
模型构建单元，用于根据历史飞行数据构建低空飞行器的飞行安全管理模型；
影响因素单元，用于根据历史飞行数据获取飞行安全影响因素，根据低空飞行器的飞行数据特征对飞行安全影响因素进行因素选择，确定当前飞行安全影响因素；
飞行安全评估单元，用于根据当前飞行安全影响因素构建飞行安全评估指标体系，根据历史飞行数据确定飞行安全评估指标的影响权值；
将低空飞行器的飞行数据输入低空飞行器的飞行安全管理模型中，根据飞行安全评估指标体系以及影响权值确定低空飞行器的当前飞行初始安全值；
根据历史飞行数据对飞行安全值进行范围划分，获取低飞行安全范围、中飞行安全范围以及高飞行安全范围；
根据低空飞行器的飞行路线中的飞行节点对飞行区域进行划分，获取飞行子区域；
当监测到飞行子区域对应的当前飞行初始安全值为高飞行安全范围时，评估低空飞行器处于安全飞行状态中，无需进行飞行安全管理；
当监测到飞行子区域对应的当前飞行初始安全值为低飞行安全范围时，评估低空飞行器处于危险飞行状态中，即刻更改飞行子区域中的飞行路线，对低空飞行器进行飞行安全管理；
当监测到飞行子区域对应的当前飞行初始安全值为中飞行安全范围时，若飞行子区域为已知飞行区域，则获取飞行子区域中定位设备的定位节点，根据低空飞行器的当前飞行位置选择相距最近的定位节点对低空飞行器进行分析评估；
若飞行子区域为未知飞行区域，则获取与未知飞行区域相邻的已知相邻飞行区域中定位设备的定位节点，从各相邻的已知飞行区域中选取与低空飞行器的当前飞行位置相距最近的定位节点对低空飞行器进行分析评估；
根据定位节点与低空飞行器的相距距离确定数据传输延迟；
根据低空飞行器的飞行数据标定低空飞行器的避障范围并划分，当避障范围内出现障碍物时，根据定位节点获取障碍物在避障范围内的分布情况，其中，避障范围包括：低风险避障范围、中风险避障范围以及高风险避障范围；
设定各避障范围内的障碍物对低空飞行器的飞行影响权值，结合所述数据传输延迟对低空飞行器的当前飞行初始安全值进行加权，获取低空飞行器的当前飞行风险值；

根据当前飞行风险值所处飞行安全范围对低空飞行器进行飞行安全管理；
飞行安全预警单元，用于根据历史预警数据以及飞行风险值，设定飞行风险值预警阈值，若当前飞行风险值不高于低飞行风险值预警阈值时，判定低空飞行器当前处于安全飞行状态中，无需进行飞行安全管理，不生成飞行风险预警数据，对低空飞行器根据预设预警时段进行飞行安全预警判定；
若当前飞行风险值不低于高飞行风险值预警阈值时，判定低空飞行器当前处于危险飞行状态中，即刻进行飞行安全管理并同步生成飞行风险预警数据；
若当前飞行风险值处于低飞行风险值预警阈值与高飞行风险值预警阈值之间，则不生成飞行风险预警数据，对低空飞行器实时进行飞行安全预警判定；
安全管理单元，用于根据低空飞行器的当前飞行风险值以及飞行风险预警数据对低空飞行器进行实时安全管理。
根据本发明提供的一种低空飞行器安全管理系统，安全管理单元，包括：
自动安全管理块，用于根据低空飞行器的当前飞行风险值以及飞行风险预警数据生成安全控制指令并下发至低空飞行器，对低空飞行器进行实时安全管理，其中，安全控制指令包括：调整飞行区域控制指令、调整飞行高度控制指令、调整飞行路线控制指令；
手动安全管理块，用于根据低空飞行器的当前飞行风险值以及飞行风险预警数据进行可视化展示，允许用户根据当前飞行风险值以及飞行风险预警数据对低空飞行器进行手动控制。
根据本发明提供的一种低空飞行器安全管理系统，飞行后管理模块，包括：
飞行完成判定单元，用于当低空飞行器处于飞行路线终点且监测到低空飞行器停止运行时，判定低空飞行器完成飞行计划；
数据归档单元，用于当低空飞行器完成飞行计划时，获取飞行数据并根据飞行时段进行数据归档；
飞行优化单元，用于当数据归档结果显示低空飞行器的实际飞行计划与预设飞行计划不一致时，获取偏差飞行计划，对低空飞行器的未来飞行计划进行飞行优化。
本发明提供一种低空飞行器安全管理方法，包括：
步骤1：对低空飞行器进行飞行预安全管理，检测低空飞行器的设备状态以及评估低空飞行器的飞行计划；

本发明实施例提供一种低空飞行器安全管理系统，飞行预管理模块，包括：
飞行预安全管理单元，用于当低空飞行器在当前时刻存在飞行计划时，标记为预飞行低空飞行器，对低空飞行器进行飞行预安全管理，其中，飞行预安全管理包括：设备状态检测安全管理以及飞行计划评估安全管理；
设备状态检测单元，用于对预飞行低空飞行器的设备状态进行检测，当判定设备状态处于正常运行状态时，判定预飞行低空飞行器的设备状态检测安全管理通过，其中，设备状态包括：正常运行状态、异常运行状态以及故障状态；
