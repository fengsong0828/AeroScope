Description

一种适用于低空飞行器的封严结构
技术领域
本发明属于航空宇航发动机密封领域，尤其涉及一种适用于低空飞行器的封严结构。
背景技术
随着国家大力推进低空经济的发展，无人机下游场景不断增加，无人机应用环境多元化、复杂化。对于一千米以下的航空器，封严所面对的挑战重点从空气转变为水，低空环境下，液体水以及水汽对于封严结构的冲击成为泄露的重要因素，而航空器的效率严重依赖于通过动叶径向间隙、叶根燕尾榫头、静叶根部篦齿气体的泄漏量，面对低空经济对低空航空器的长航时、低能耗的要求，在原有的封严结构上必须有针对化的创新性设计以解决液体水以及水汽的问题。
对比诸多封严结构，篦齿封严具有低成本、结构可靠和维护简单等易于推广使用的优点，更加适合发展低空经济所需求的可大量生产。篦齿封严性能受限于旋转件与静止壁面的径向间隙，理想情况下篦齿零间隙可以避免性能损失，但受制于旋转件与静止件工作过程中发生的振动、热变形和蠕变，以及篦齿封严自身的制造公差以及装配过程所需保留的公差使得使用过程中必须需要保留一定的间隙。如何提升传统封严结构的密封性能成为重要课题，一种方法是改变篦齿结构的流道来改善泄露，比如弯曲流道、交错篦齿封严以及交错篦齿封严的形状参数变量对于泄露性能的影响，另一种则是设置流体屏障，以较小的几何变化来减小泄露。但目前仍没有针对于低空航空器所涉及的封严结构。
发明内容
本发明所要解决的技术问题是针对上述现有技术的不足，提供一种针对低空航空器液体水以及水汽吸收问题，有效扰乱流场以及吸收水的适用于低空飞行器的封严结构。
为实现上述技术目的，本发明采取的技术方案为：
一种适用于低空飞行器的封严结构，包括转轴,转轴外同轴套设有封严衬套，转轴上设置有多个轴向排布的封严篦齿，封严篦齿的外周面与封严衬套内环面间隙配合，封严衬套上设置有旋转件以及凸起件，旋转件位于封严篦齿与封严衬套之间间隙的前方，旋转件能通过旋转将流向封严篦齿与封严衬套之间间隙的水打散，使其改变流向，凸起件位于旋转件后方，同时也处于封严篦齿与封严衬套之间间隙的前方，凸起件向封严篦齿与封严衬套之间间隙处凸出，用于遮挡水流向封严篦齿与封严衬套之间间隙。
为优化上述技术方案，采取的具体措施还包括：
上述的封严衬套内环面在旋转件以及凸起件之间设置有吸水材料，吸水材料用于吸收溅射至封严衬套内环面上的水。
上述的凸起件为弧形件，凸起件的弯曲方向为：凸起件下端向前弯曲。
上述的旋转件包括螺纹管、扇面、底盘、螺纹柱和连接杆，其中，扇面和底盘位于封严衬套下方，连接杆位于封严衬套上方，封严衬套上开设有上下贯穿封严衬套的穿孔，螺纹管固定在穿孔中，扇面位于封严篦齿与封严衬套之间间隙的前方，扇面的扇轴与螺纹柱下端固定连接，螺纹柱可旋转安装在螺纹管中，螺纹柱上端与连接杆的一端周向可转动、轴向定位连接，连接杆另一端与底盘连接，底盘位于吸水材料下方，当扇面在受到水流由前向后冲击时，扇面转动能使螺纹柱在螺纹管中正转上移，连接杆上移拉动底盘挤压吸水材料，使吸水材料脱水，当扇面受到的水流由前向后冲击减小时，扇面、底盘和和连接杆自重使螺纹柱在螺纹管中反转下移，底盘放松吸水材料。
上述的吸水材料为海绵。
上述的螺纹管为内外均设置有螺纹的双向螺纹管，穿孔为螺纹孔，螺纹管的外侧螺纹与穿孔的螺纹相配合，使螺纹管与穿孔密封连接。
上述的底盘为网格状刚性材料制作。
上述的扇面由若干个风扇叶片等弧度连接在扇轴上组成。
上述的凸起件为径向弯度处于45°与50°之间的刚性材料。
上述的最前两个封严篦齿与封严衬套之间间隙前设置旋转件，其中，仅最前一个旋转件设置有螺纹管、扇面、底盘、螺纹柱和连接杆，第二个旋转件仅设置扇面，不设置螺纹管、底盘、螺纹柱和连接杆。
本发明的有益效果是：本发明含有衬套加装结构的篦齿封严结构为了吸收低空时通道内水分，旋转件在通道入口处破水扰乱水流，凸起件使得扰乱后水流回流被渗水面上海绵吸水材料充分吸收，为进一步吸收通道内水分，旋转件通过下端扇面在破水的同时吸收水流动能，使得旋转件通过双向螺纹孔向上旋转，使得网格底盘同时向上提升，挤压海绵吸水材料，能够重新恢复材料吸水性能。在入口篦齿结构和第二个篦齿间加装旋转件下端扇面以及渗水面和凸起件，一方面吸收入口处海绵吸水材料挤压后水分，一方面吸收通道内剩余水分。通过反复吸水保持篦齿结构封严效果，减少了结构的泄露量。

本发明的一种含有衬套加装结构的篦齿封严结构，包括转轴1,转轴1外同轴套设有封严衬套3，转轴1上设置有多个轴向排布的封严篦齿2，封严篦齿2的外周面与封严衬套3内环面间隙配合，封严衬套3上设置旋转件4以及凸起件5，通过旋转件4以及凸起件5的破水效果配合吸水材料6保持篦齿封严结构良好的密封性能。
实施例中，入口处封严篦齿旋转件4下端扇面42以破水减缓水流速度，扰乱水流冲击，便于水分通过渗水面7被吸水材料6充分吸收，扇面42同时吸收破水时的水流动能使得旋转件通过穿孔31向上旋转，底盘43也随着旋转件4向上旋转同时上升，与封严衬套3挤压吸水材料6，使得吸水材料6重新恢复吸水性能，完成排水后随着重力作用向下旋转继续吸水。
实施例中，扇面42为刚性材料，得以充分吸收水流动能。底盘43为网格状刚性材料，便于水分通过渗水面被吸收。
实施例中，凸起件5为径向弯度处于45°与50°之间的刚性材料，使得水流回流回渗水面被吸水材料6充分吸收。
实施例中，于入口封严篦齿2与后一个封严篦齿2间加装扇面42，吸水材料6以及后方凸起件5，扇面42破水扰乱流场，凸起件5阻挡通道内水流便于吸水材料6吸收通道内剩余水分。
本发明的衬套上加装结构的篦齿封严结构在低空飞行下有较大水流冲击环境中扰乱水流流场，减弱水流冲击对于篦齿封严结构泄漏量影响，同时利用水流冲击动能，达到反复吸收通道内水分，有效保持篦齿封严结构的密封性能，削弱了水流流场动能，减小了篦齿封严结构的泄漏量。
