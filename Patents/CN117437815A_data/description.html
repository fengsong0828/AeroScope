Description

一种基于车路协同下的低空飞行器避障系统和方法
技术领域
本发明涉及飞行器避障领域，具体涉及一种基于车路协同下的低空飞行器避障系统和方法。
背景技术
飞行器的应用在未来将有机会成为城市交通运输工具的重要组成部分，由于其不受地形和地面道路条件的约束，具有灵活、高效便捷的特性，能够快速跨越障碍，抵达目的地。在低空飞行时，为实现对周边环境的感知，需要安装摄像头和雷达对周边的环境进行识别，但大部分无人飞行器飞行都是因为对周边障碍物识别错误或者不准确导致坠机，且大雾和黑夜等能见度低的天气，无人飞行器对周边环境的感知能力进一步降低，无人飞行器基本无法执行飞行任务。
现有实现低空飞行器避障的方法主要通过个感知传感器来实现障碍物的感知，通常包括超声波、红外、雷达等主动式传感器，也有基于视觉传感器的被动传感器，门春雷通过采用双目相机视觉图像的方法，识别障碍物的位置和深度信息，再通过对空间内的障碍物和预定路径生成排斥势场和吸引势场，有效的规避障碍物。马经纬通过将双目摄像机和毫米波雷达进行融合，采用推帚立体视觉算法和图像数据融合的方法标记障碍物的位置，以实时躲避障碍物。D·S·马特洛夫通过从飞行器向多个方向发射红外信号，通过对接收回传信号实现对障碍物的识别，以达到检测周边障碍物的目的。
上述方法通过传感器对周边障碍物的识别，要求对传感器进行准确的标定，同时在高动态环境下具备对成像数据实时解析的能力，在空间距离测算方面则需要双目相机或雷达进行测距，一方面对设备硬件性能提出了较高的要求，另一方面对障碍物检测的准确性提出较大挑战。
近年来，随着V2X无线短程通信技术的发展，在车辆安全辅助驾驶方面取得了重大进步，其有低延迟和高可靠性，以确保信息的及时准确的传递，有效解决不具有感知能力的车辆获取视野盲区的重要信息，同理在无人飞行器中同样可以用于解决不具备感知能力或者感知能力异常时对障碍物的识别，以保障飞行器能够正常安全飞行。
因此在此背景下提出了一种基于车路协同环境下的低空飞行器避障方法，实现飞行器在低空飞行时安全运行，不仅能够降低飞行器对环境感知传感器的需求，降低产品成本，同时也能提高在异常天气环境下使用的稳定性。
发明内容
根据背景技术提出的问题，本发明提供一种基于车路协同下的低空飞行器避障系统和方法，通过车路协同V2X通信技术可以有效提高低空飞行器对障碍物的识别能力，提高飞行器避障能力，减少因避障不及时而导致的坠机。接下来对本发明做进一步地阐述。
一种基于车路协同下的低空飞行器避障方法，包括以下步骤：
S1，障碍物预警信息构造，采集障碍物信息，根据不同的障碍物特性，采用不同的采集策略对所采集的障碍物信息实现障碍物的信息表达；表达后在路侧设备RSU中配置障碍物信息，以及构造障碍物的影响区域信息，并实现障碍物信息的广播；
S2，障碍物预警，机载设备采集自身相关数据，判断障碍物类型表达方式，构建斥力场Ureq(q)，当同时收到多个障碍物信息时，得到每个障碍物的斥力场Ureq(q)，通过斥力场的叠加得到飞行器飞行周边环境的斥力完整斥力场环境，当前U(q)斥力场值大于0时将产生障碍物预警，并将所有预警的障碍物的位置和影响范围构造成预警信息发送至用户端；
S3，周边飞行器预警，当周边出现其他飞行器时，获取到周边飞行器的运动状态和位置信息，并将周边飞行器的位置信息实时传送至用户端，通过计算本飞行器与周边飞行器的位置信息，当前飞行器之间的距离满足飞行器安全预警的阈值R1，并且飞行器之间的高程在预警飞行器预警范围内时，触发周边飞行器预警，并将所有的触发周边飞行器预警的飞行器发送至用户端；当飞行器之间距离小于R2时，飞行器将可能发生安全碰撞，此时飞行器将根据背景飞行器的位置进行计算采用自动避障的方式触发避障，自动避障的方法详将步骤S4；
S4，飞行器自动避障，当触发安全碰撞预警时，飞行器距离障碍物较近，通过计算飞行器与障碍之间的位置关系，得到飞行器的避障策略。
一种基于车路协同下的低空飞行器避障系统，包括：机载终端OBU(OnboardUnit)、路侧终端RSU(RoadSideUnit)、用户飞行控制器和飞行器控制模块。
所述的机载终端OBU，包括：飞行器状态信息采集模块、机载终端V2X无线通信模块和障碍物预警模块。其中：所述的飞行器状态信息采集模块，用于实时获取自身运动状态信息，包括经纬度、高程、速度，角速度，加速度和航向角等；所述的机载终端V2X无线通信模块，用于采集和发送自身的飞行器运动状态至周边其他飞行器，同时接收路侧广播出来的障碍物信息；所述的障碍物预警模块，用于计算当飞行器靠近障碍物时，触发障碍物预警。
所述的路侧终端RSU，包括：路侧障碍物信息配置模块，路侧终端V2X无线通信模块。其中：所述的路侧障碍物信息配置模块，用于用户配置障碍物信息，障碍物信息可通过障碍物经纬度、高程来表达障碍物区域范围；所述的V2X无线通信模块用于将障碍物信息广播至周边的飞行器机载终端。
所述的用户飞行控制器，包括：飞行控制模块和飞行预警展示模块。其中：所述的飞行控制模块，用于用户下发飞行控制器指令，操控飞行器飞行；所述的飞行器预警展示模块，用于展示飞行器展示周边障碍物信息，和相关障碍物预警信息。
所述的飞行器控制模块，包括：飞行控制模块接收并执行控制指令，障碍物预警控制模块。其中：所述的飞行控制模块用于接收用户飞行控制指令，将控制指令转换为飞行策略；所述的障碍物预警控制模块，将OBU产生的预警信息计算得到的预警执行策略，下发至飞行器，飞行器将控制策略转换为飞行策略。
有益效果：与现有技术相比，本发明系统可通过飞行器和路侧设备通信实现飞行器对周边障碍物信息的感知，脱离了传统的雷达或摄像头等方式对周边环境的感知，障碍物预警的信息可通过PC5直连通信的方式进行广播发送，也可通过UU网络通信的方式实现更大范围内的障碍物信息传递，实现了飞行器对周边障碍物环境的感知，并通过在用户端对周边障碍物和周边飞行器位置虚拟展示，实现了对周边障碍物预警，用户通过对周边障碍物的感知，可实飞行器的安全飞行和有效避障。

S2，障碍物预警，具体包括以下步骤：
S201，机载设备采集自身相关数据，包括飞行器的经纬度、高程、速度、角速度、加速度和航向角等信息。
S202，判断障碍物类型表达方式，分为点域的方式或者为点高的方式：如果为点域的方式，则需要通过障碍物的点域范围进行通行合成处理，得到障碍物的点域作用空间范围，并进入步骤S203，生成封闭区域内的斥力场模型，最近点通过距离筛选和平面几何计算得到飞行器与障碍物面的垂线距离，作为斥力模型影响距离的判断依据；如果为点高的方式，则需要通过获取在同高程范围内的障碍物影响半径Rn作为最大影响距离。

S5，预警警告和用户反馈
用户端收到预警障碍物信息时，在用户端高亮障碍物的位置和影响范围，并根据障碍物的所在的方位和障碍物出现的斥力场大小给与不同等级障碍物预警，通过障碍物的预警信息可及时帮助飞行器操控人员实现飞行器安全飞行并到达目的地；当用户收到周边飞行器预警信息时，通过用户端展示周边飞行器实时位置以外，对周边存在安全预警的飞行器进行危险预警展示，以帮助用户实现对周边存在风险的飞行器进行规避。
本发明装置可通过飞行器和路侧设备通信实现飞行器对周边障碍物信息的感知，脱离了传统的雷达或摄像头等方式对周边环境的感知，障碍物预警的信息可通过PC5直连通信的方式进行广播发送，也可通过UU网络通信的方式实现更大范围内的障碍物信息传递，实现了飞行器对周边障碍物环境的感知，并通过在用户端对周边障碍物和周边飞行器位置虚拟展示，实现了对周边障碍物预警，用户通过对周边障碍物的感知，可实飞行器的安全飞行和有效避障。
本方法在车路协同环境下通过采集障碍物关键信息，并通过路侧设备构造障碍物信息，将障碍物信息通过PC5或UU的方式广播至周边飞行器，当周边飞行器收到相应的障碍物信息时，通过飞行器对障碍物的位置大小进行分析，并建立人工势场法模型得到周边障碍物的位置大小，以及当触发障碍物预警时，给与飞行器控制者提供相应的障碍物预警提醒；同时，当遇到周边有其他飞行器时，通过PC5实时通信获取到周边飞行器的位置，当出现飞行器安全碰撞预警时，给与飞行器控制者提供相应的飞行器安全预警提醒，辅助提高飞行器飞行安全。相比传统的方案，本方案通过采集障碍物关键信息，并通过路侧设备对障碍物信息的表达和构建，广播至周边飞行器，周边飞行器通过接收路侧广播出来的障碍物信息，将障碍物信息进行回传和预警信息提醒。在障碍物的信息表达和识别上与上述方法完全不同，且本方法同样不需要进行深度学习处理和神经网络的训练，极大的降低设备的性能要求。
