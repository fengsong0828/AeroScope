Description

一种可载人的低空飞行器
技术领域
本实用新型具体涉及一种可载人的低空飞行器，属于玩具出行工具领域。
背景技术
现有的飞行器在飞行时控制复杂，因此，我们提出一种可载人的低空飞行器。
实用新型内容
本实用新型要解决的技术问题克服现有的缺陷，提供一种可载人的低空飞行器，通过在底板前侧设置测距仪和蜂鸣警报器，以及底板内部链接蓄电池的电量检测器，当底板前侧距离障碍物的距离低于设置值，则测距仪将信息传递到控制器，控制器控制蜂鸣警报器发出警报，当蓄电池电量低于设定电量，则将信息传递到控制器，控制器控制蜂鸣警报器发出警报，可有效避免发生事故，该底板顶部设置了控制台，控制台两侧均设置了踏板，该踏板内部前后均设置了压力传感器，两只脚踩上去后，身体的摇晃改变脚的重力倾向，控制器根据每个压力传感器传递的信息计算出人体的倾斜方向，生成飞行指令传送到飞行控制器，飞行控制器控制各个电动机功率来实现设备的飞行方向，可以有效解决背景技术中的问题。
为了解决上述技术问题，本实用新型提供了如下的技术方案：
本实用新型提供一种可载人的低空飞行器，包括底板、侧板、护板和控制台，所述侧板设置于底板两侧，所述侧板前侧和后侧均设有螺旋桨，所述螺旋桨内设有电动机，所述侧板中部设有太阳能电池板，所述护板设置于底板顶部两侧，所述控制台设置于底板顶面中部，所述控制台两侧均设有踏板，所述踏板内部从前往后依次设有两个压力传感器，所述底板前侧从左往右依次设有充电口、测距仪和蜂鸣警报器，所述底板内部设有蓄电池，所述蓄电池一侧设有太阳能控制器，所述蓄电池另一侧设有充电芯片，所述蓄电池顶部从左往右依次设有控制器、飞行控制器和电量检测器。
优选的，所述控制台两侧的踏板上设置的四组压力传感器均通过电路与控制器电性连接，所述底板顶部设置的两个侧板顶部分别设有停止键和启动键，所述停止键和启动键均通过电路与控制器电性连接。
优选的，所述太阳能电池板通过电路与太阳能控制器电性连接，所述太阳能控制器通过电路与蓄电池电性连接，所述充电口通过电路与充电芯片电性连接，所述充电芯片通过电路与蓄电池电性连接，所述充电芯片通过电路分别与控制器和飞行控制器电性连接，所述蓄电池通过电路与电量检测器电性连接，所述电量检测器通过电路与控制器电性连接。
优选的，所述测距仪通过电路与控制器电性连接，所述蜂鸣警报器通过电路与控制器电性连接。
优选的，所述螺旋桨内设置的电动机通过电路与飞行控制器电性连接，所述飞行控制器通过电路与控制器电性连接。
本实用新型所达到的有益效果是：通过在底板前侧设置测距仪和蜂鸣警报器，以及底板内部链接蓄电池的电量检测器，当底板前侧距离障碍物的距离低于设置值，则测距仪将信息传递到控制器，控制器控制蜂鸣警报器发出警报，当蓄电池电量低于设定电量，则将信息传递到控制器，控制器控制蜂鸣警报器发出警报，可有效避免发生事故，该底板顶部设置了控制台，控制台两侧均设置了踏板，该踏板内部前后均设置了压力传感器，两只脚踩上去后，身体的摇晃改变脚的重力倾向，控制器根据每个压力传感器传递的信息计算出人体的倾斜方向，生成飞行指令传送到飞行控制器，飞行控制器控制各个电动机功率来实现设备的飞行方向。

具体实施方式
以下结合附图对本实用新型的优选实施例进行说明，应当理解，此处所描述的优选实施例仅用于说明和解释本实用新型，并不用于限定本实用新型。
实施例：请参阅图1-3，本实用新型一种可载人的低空飞行器，包括底板1、侧板2、护板3和控制台4，侧板2设置于底板1两侧，侧板2前侧和后侧均设有螺旋桨6，螺旋桨6内设有电动机17，侧板2中部设有太阳能电池板5，护板3设置于底板1顶部两侧，控制台4设置于底板1顶面中部，控制台4两侧均设有踏板18，踏板18内部从前往后依次设有两个压力传感器11，底板1前侧从左往右依次设有充电口10、测距仪8和蜂鸣警报器7，底板1内部设有蓄电池12，蓄电池12一侧设有太阳能控制器16，蓄电池12另一侧设有充电芯片15，蓄电池12顶部从左往右依次设有控制器13、飞行控制器14和电量检测器9。
进一步，控制台4两侧的踏板18上设置的四组压力传感器11均通过电路与控制器13电性连接，底板1顶部设置的两个侧板2顶部分别设有停止键20和启动键19，停止键20和启动键19均通过电路与控制器13电性连接，该飞行器的控制直接由踏板18、停止键20和启动键19控制，其中，踏板18内设置的压力传感器可实时检测人体脚底的压力，并将压力值传送到控制器13，控制器13可根据各压力传感器11之间的差值计算出人体的倾斜方向和倾斜力度，以此为基础向飞行控制器14发送飞行指令，实现用人体的倾斜控制飞行器的飞行方向。
进一步，太阳能电池板5通过电路与太阳能控制器16电性连接，太阳能控制器16通过电路与蓄电池12电性连接，充电口10通过电路与充电芯片15电性连接，充电芯片15通过电路与蓄电池12电性连接，充电芯片15通过电路分别与控制器13和飞行控制器14电性连接，蓄电池12通过电路与电量检测器9电性连接，电量检测器9通过电路与控制器13电性连接，太阳能电池板5可不间断的为蓄电池12充电，延长设备的续航能力，当蓄电池12内的电量不足时，电量检测器9即可将检测信息传送到控制器13，由控制器13控制蜂鸣警报器7发出警报。
进一步，测距仪8通过电路与控制器13电性连接，蜂鸣警报器7通过电路与控制器13电性连接，当飞行器前侧距离障碍物一定距离时，测距仪8即可向控制器13发送信息，控制器13即刻控制蜂鸣警报器7发出警报，其中，测距仪8可为红外线测距仪或者声波测距仪。
进一步，螺旋桨6内设置的电动机17通过电路与飞行控制器14电性连接，飞行控制器14通过电路与控制器13电性连接，其中，飞行控制器14可选用普通的四旋翼飞行控制器，控制器13为CPU，内置运行程序，可计算出各个压力传感器11之间的差值，并以此计算出人体的倾斜角度。
使用时：通过在底板前侧设置测距仪和蜂鸣警报器，以及底板内部链接蓄电池的电量检测器，当底板前侧距离障碍物的距离低于设置值，则测距仪将信息传递到控制器，控制器控制蜂鸣警报器发出警报，当蓄电池电量低于设定电量，则将信息传递到控制器，控制器控制蜂鸣警报器发出警报，可有效避免发生事故，该底板顶部设置了控制台，控制台两侧均设置了踏板，该踏板内部前后均设置了压力传感器，两只脚踩上去后，身体的摇晃改变脚的重力倾向，控制器根据每个压力传感器传递的信息计算出人体的倾斜方向，生成飞行指令传送到飞行控制器，飞行控制器控制各个电动机功率来实现设备的飞行方向，通过设置启动键和停止键，可控制设备的启停，该载人飞行器，控制简便，适宜推广使用。
