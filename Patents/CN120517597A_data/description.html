Description

载人低空飞行器座椅及飞行设备
技术领域
本申请涉及飞行器领域，具体涉及载人低空飞行器座椅及飞行设备。
背景技术
当前一些飞行器的座椅在面临应急着陆等动态冲击工况时，乘员的身体受到的冲击过大，一些已知的座椅对乘员的防坠保护能力较差。
发明内容
本申请的目的在于克服上述技术不足，提出一种载人低空飞行器座椅及飞行设备，解决已知技术中座椅存在防坠保护能力较差的技术问题。
为达到上述技术目的，本申请采取了以下技术方案：
第一方面，本申请提供了一种载人低空飞行器座椅，包括座椅骨架、座椅主体、缓冲组件、第一调节件以及安全带组件。所述座椅主体设于所述座椅骨架，所述座椅主体具有相背设置的乘坐侧及背侧。所述缓冲组件设于所述座椅主体，并位于所述乘坐侧。所述第一调节件一端连接所述座椅骨架，所述第一调节件的另一端连接所述座椅主体，所述第一调节件用于调节所述座椅主体沿第一方向相对所述座椅骨架的位置。所述安全带组件的一端连接所述第一调节件，所述安全带组件的另一端绕过所述座椅主体，并连接所述背侧。
在一些实施例中，所述座椅主体包括座板，所述座板设有安装槽，所述安装槽沿背离所述座椅骨架的方向敞开设置，所述缓冲组件设于所述安装槽，且所述缓冲组件与所述安装槽的槽侧面间隔设置。
在一些实施例中，所述缓冲组件包括第一缓冲件及第二缓冲件，所述第一缓冲件具有背离所述座椅骨架的第一顶面，所述第二缓冲件具有背离所述座椅骨架的第二顶面，所述第二缓冲件位于所述第一缓冲件沿第二方向背离所述背侧的一侧，所述第二方向与所述第一方向垂直或斜交，所述第一顶面位于所述第二顶面沿第一方向背离所述座椅骨架的一侧。
在一些实施例中，所述座椅骨架开设第一滑槽，所述第一滑槽沿所述第一方向延伸，所述第一调节件的一端沿所述第一方向可滑动地设于所述第一滑槽，所述第一调节件的另一端连接所述座椅主体，所述第一调节件连接所述安全带组件。其中，所述载人低空飞行器座椅还包括第一滚动件，所述第一滚动件设于所述第一调节件，所述第一滚动件与所述第一滑槽的槽表面滚动配合；和/或，所述载人低空飞行器座椅还包括第一减摩件，所述第一减摩件设于所述第一调节件的表面，所述第一减摩件与所述第一滑槽的槽表面接触。
在一些实施例中，所述第一滑槽形成有第一敞口，所述座椅骨架在所述第一敞口处形成有凸出设置的第一止挡部，所述第一调节件沿第三方向的一侧穿过所述第一止挡部，并连接所述座椅主体。所述载人低空飞行器座椅还包括所述第一滚动件及所述第一减摩件，所述第一滚动件设于所述第一调节件的底面，所述第一滚动件与所述第一滑槽的槽底面滚动配合；所述第一减摩件设于所述第一调节件的侧面，所述第一减摩件与所述第一滑槽的槽侧面以及所述第一止挡部接触。
在一些实施例中，所述载人低空飞行器座椅还包括第二调节件，所述第二调节件的一端连接所述座椅骨架，所述第二调节件的另一端用于连接机架，所述机架设有导轨，所述导轨形成有沿第二方向延伸的第二滑槽，所述第二调节件一端伸入所述第二滑槽内，所述第二调节件另一端连接所述座椅骨架。其中，所述载人低空飞行器座椅还包括第二滚动件，所述第二滚动件连接所述第二调节件，所述第二滚动件位于所述第二调节件朝向所述第二滑槽的槽底面的一侧，所述第二滚动件与所述第二滑槽的槽底面滚动配合；和/或，所述载人低空飞行器座椅还包括第二减摩件，所述第二减摩件设于所述第二调节件的表面，所述第二减摩件用于接触所述第二滑槽的槽表面。
在一些实施例中，所述第二滑槽形成有第二敞口，所述机架在所述第二敞口处形成有凸出设置的第二止挡部，所述第二调节件沿第一方向的一侧穿过所述第二止挡部，并连接所述座椅骨架。所述载人低空飞行器座椅还包括所述第二滚动件及所述第二减摩件，所述第二滚动件设于所述第二调节件的底面，所述第二滚动件与所述第二滑槽的槽底面滚动配合；所述第二减摩件设于所述第二调节件的侧面，所述第二减摩件与所述第二滑槽的槽侧面以及所述第二止挡部接触。
在一些实施例中，所述安全带组件包括锁扣件及腰带，所述锁扣件位于所述乘坐侧，所述腰带的一端连接所述锁扣件，所述腰带的另一端连接所述第一调节件。
在一些实施例中，所述座椅主体包括座板、靠背以及调角器，所述缓冲组件设于所述座板，所述靠背通过所述调角器可转动地连接所述座板。
第二方面，本申请还提供了一种飞行设备，包括机架及前述的载人低空飞行器座椅。所述载人低空飞行器座椅的座椅骨架连接所述机架。
与已知技术相比，本申请提供的载人低空飞行器座椅，通过安全带组件连接第一调节件，使安全带组件承载的至少一部分人体载荷可以通过第一调节件传导至座椅骨架，从而分散座椅主体承载的人体载荷，提高人体载荷的传递效率，进而优化载人低空飞行器座椅的整体承载能力，并提高对乘员的保护效果。同时，缓冲组件可以对传导至座椅主体的人体载荷起到缓冲作用，并能够吸收在飞行设备应急坠撞过程中的冲击载荷，以进一步对乘员起到保护作用。因此，本申请的载人低空飞行器座椅可以通过安全带组件分散乘员的人体载荷，再通过缓冲组件对乘员与座椅主体之间的人体载荷起到缓冲吸收作用，大幅提高了对乘员的防坠保护能力。

具体实施方式
为了使本申请的目的、技术方案及优点更加清楚明白，以下结合附图及实施例，对本申请进行进一步详细说明。应当理解，此处所描述的具体实施例仅仅用以解释本申请，并不用于限定本申请。
为了解决已知飞行器上的座椅安全性不高的技术问题，本申请提供了一种载人低空飞行器座椅及飞行设备，能够实现提高座椅安全性。
需要说明的是，本申请的载人低空飞行器座椅用于但不限于飞行设备等，为了方便说明，在本申请中，仅以载人低空飞行器座椅应用于飞行设备为例进行说明，而载人低空飞行器座椅应用于其他类型的乘用设备中的原理与应用于飞行设备中的原理实质相同，在此不一一赘述。其中，飞行设备可以是低空飞行器，例如电动垂直起降飞行器。可以理解的是，本申请的载人低空飞行器座椅也可以应用于其他空域的飞行场景。
