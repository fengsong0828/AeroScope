Description

一种低空飞行器综合服务站
技术领域
本发明涉及一种公共交通系统，特别是一种利用飞行汽车作为交通工具，在城市上空运行，在各交通枢纽、高速服务站、地铁站等大型公共交通枢纽站或者服务区设置的综合服务站，属于现代交通设施领域。
背景技术
城市公共交通系统是目前国内研究的热点，城市土地资源有限，我国城市公交逐渐由地面向地下发展，但随着人口和机动车数量的不断增长，地面交通拥堵频繁出现，出行效率严重受到影响，地面交通系统的发展遇到严峻的挑战。然而全国乃至全球低空领域的利用率极少，开发利用低空领域成为交通系统的新方向。随着飞行汽车的研发，空中飞行器越来越多，因此需要建立一套适合飞行汽车运行的低空立体交通系统。而低空飞行器综合服务站是为飞行器起飞和降落提供服务的综合场所。目前已有的公共交通服务站的设施有地铁站、机场、高速收费站等，这些都因为功能的单一性和地点的特殊性并不能为飞行汽车的运行提供服务。且机场的建造成本高，专用性太强，不能满足飞行汽车平民化使用。现有的专利中有基于高速公路服务站的飞行器综合服务站，该发明创造因为是基于高速公路服务站的，因此局限性太强，不方便使用，且对于服务站的功能描述并不清晰，随着科学技术的发展，飞行汽车将如同普通汽车一样进入千家万户，人们就不能仅仅局限于高速公路上的综合服务站，人们希望飞行汽车可以在城市任何一个地方遨游。
发明内容
本发明的目的是设计一种特殊的服务站，该服务站在现有高速服务站、公交车站、地铁站等交通枢纽服务站的基础上新增一些特定的模块，不仅能够满足现有的交通工具的运行和使用，同时也能满足未来飞行汽车的使用，使飞行汽车能够通过该综合服务站实现起降导航等及其他各项服务功能。
本发明所述低空飞行器综合服务站包括供飞行汽车使用的专用出口、专用入口、起降场地、能量补给、导航基站、通用服务区、综合服务大厅， 在有低空航路网分布的在公路或者河流的两边均设置有若干个综合服务站，单侧综合服务站不仅可供飞行汽车起飞，也能提供降落服务。每个综合服务站设有供飞行汽车驶入的专用入口和驶出的专用出口，出口和入口通过辅道与道路或者河流码头相连，综合服务站的专用入口可控制进入的飞行汽车数量。
所述通用服务区设置在综合服务站内的前部，通用服务区是指为飞行汽车提供能量补给、维修服务及停车休息服务的地方。
与通用服务区相连的是综合服务大厅，综合服务大厅不仅能够为驾驶员提供餐饮、缴费和休息等服务，同时，在综合服务大厅将设置有大型显示屏，实时显示天空中飞行汽车的数量和运行状态，以及飞行汽车是否允许起飞降落等状况。如遇恶劣天气或天空中有紧急任务在执行，该显示屏将提醒广大驾驶员朋友此时飞行汽车禁止驶入低空航道，请耐心等待。
在综合服务大厅的后面设置有供飞行汽车起飞和降落的起落场，专起落场前头设置专用入口，后头设置专用出口，专用入口只允许飞行汽车进入，起落场专门供飞行汽车起飞和降落。普通汽车不可以进入。小型起落场设置有若干个起飞跑道，每个起飞跑道设置有信号指示，红灯表示该起飞跑道不能使用或者有别的飞行汽车即将使用该跑道进行降落，绿灯表示该跑道目前空闲可以使用，跑道使用信息由导航基站根据飞行汽车的位置反馈给控制中心，跑道的使用情况由低空交通管理控制中心统一调度，飞行汽车需要降落与起飞时必须严格遵守起飞场地的信号灯指示，否则将造成空中交通事故。
另外在综合服务站中设置有导航基站，实现了高精度统一的控制网，使得在任何地方都可以精确定位到飞行汽车的位置。本发明中所设置的基站对飞行汽车进行定位、巡航，同时将进入空中快道的飞行汽车的数量、交通状态等信息反馈回来，为飞行汽车的空中调度、管控等提供参考依据。
所述通用服务区可以拓展功能，比如底盘租借管理服务，随着目前科学技术的发展，电动飞行汽车在动力与负载的匹配上有着矛盾的方面，一方面要求电动飞行汽车的动力大，载荷才大，但是另一方面动力增加后电池的重量也相应增加，实际载荷增加较少，因此可以考虑底盘自卸式电动飞行汽车，该发明方案所述低空飞行器综合服务站可以提供底盘租借和管理服务，方便电动飞行汽车运行。
本发明所述低空飞行器综合服务站，不仅能够给普通汽车提供服务，特别地是要为飞行汽车运行提供服务，一般来讲，该低空飞行器综合服务站主要分布在高速公路、码头、河流、地铁、城市公路的两侧，服务站的设置取决于航路网的布局，本发明所述服务站的地址仅供参考，对于航路网沿高速路设置的低空快道，其综合服务站可以设置在高速服务站内，而城市路面综合服务站的选址主要考虑城市主要交通集散点（如工业区、居住区、行政中心，商业中心、体育中心、文化中心、车站等客流集散点，货场、码头、仓库等货流集散点）。该综合服务站的功能类似于高速公路的服务站，但是又区别于高速服务站。高速服务站主要是为汽车提供加油服务，为乘客提供休息、餐饮服务，占地面积较小，功能单一。本发明中的综合服务站涵盖了高速现有服务站的所有功能，并在此基础上增加充电桩、飞行汽车导航基站、飞行汽车起降场地等。
本发明所述综合服务站是在现有高速服务站的基础上进行了改善，不仅可以满足现有的车辆对服务站的需求，同时还能满足将来飞行汽车对服务站的需求，该发明中所述的综合服务站布局合理，功能完善，成本较低，是未来发展的新趋势。

具体实施方式
参见附图1，一种低空飞行器综合服务站，包括供飞行汽车使用的专用出口、专用入口、起降场地、通用服务区、综合服务大厅，其特征在于，所述综合服务大厅设置在综合服务站内的中部，在综合服务大厅的前面设有通用服务区，在综合服务大厅的后面设有供飞行汽车起飞和降落的起落场地，在综合服务大厅的两端各设有与起落场连接的飞行汽车专用入口及飞行汽专用出口；
所述专用入口只允许飞行汽车进入并控制进入的飞行汽车数量；
在综合服务站中设置有导航基站，飞行汽车接收导航基站信号，在跑道上进行起飞和/或降落；导航基站对飞行汽车进行定位、巡航，并实时反馈进入空中快道的飞行汽车的数量、交通状态信息；
在综合服务站中设置有能量补给装置；
所述通用服务区设有底盘租借管理服务机构。
一种低空飞行器综合服务站的运行流程如图2所示：当飞行汽车抵达低空飞行器综合服务站时，在服务区的专用入口处等待，当服务站内飞行汽车数量过多，达到服务站的服务极限时，飞行汽车不允许驶入该服务，当服务站空闲时，飞行汽车允许驶入服务站。进入服务站后，飞行汽车的驾驶员可以根据自己的需要选择所需的服务，该综合服务站提供三种服务：飞行器的维修或者能量补给、起飞降落服务、停车休息服务；驾驶员需要根据自身的需要驶入相应的服务区域。若飞行汽车需要通过该服务站起飞，则在起飞前驾驶员需要在服务中心输入此次飞行的目的地，目的地确定后，服务中心会对该飞行汽车进行一个规划，并将此规划发送到飞行汽车的移动信号接收器上，这时驾驶员就可以进入起飞跑道起飞。而对于不需要起飞的飞行汽车，完成自己所需的服务后通过该服务站的出口驶出该服务站。
本发明所述低空飞行器综合服务站，不仅能够给普通汽车提供服务，特别地是要为飞行汽车运行提供服务，一般来讲，该低空飞行器综合服务站主要分布在高速公路、码头、河流、地铁、城市公路的两侧，服务站的设置取决于航路网的布局，本发明所述服务站的地址仅供参考，对于航路网沿高速路设置的低空快道，其综合服务站可以设置在高速服务站内，而城市路面综合服务站的选址主要考虑城市主要交通集散点（如工业区、居住区、行政中心，商业中心、体育中心、文化中心、车站等客流集散点，货场、码头、仓库等货流集散点）。该综合服务站的功能类似于高速公路的服务站，但是又区别于高速服务站。高速服务站主要是为汽车提供加油服务，为乘客提供休息、餐饮服务，占地面积较小，功能单一。本发明中的综合服务站涵盖了高速现有服务站的所有功能，并在此基础上增加充电桩、飞行汽车导航基站、飞行汽车起降场地等。
