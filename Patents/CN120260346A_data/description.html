Description

一种低空飞行器交通管理系统
技术领域
本发明涉及飞行器技术领域，特别涉及一种低空飞行器交通管理系统。
背景技术
低空经济是指以低空空域(通常指100米到3000米的高度空间)为活动范围，以无人机、飞行器、通用航空器等为载体，通过技术创新和模式创新形成的综合性经济形态。随着低空经济快速发展，低空飞行器(特别是无人飞行器)数量呈现指数级增长态势。为保障空域安全，管理部门正逐步建立完善低空交通管理法规体系，这对低空交通的智能化监测与管理提出了迫切需求。现有的低空飞行管控系统主要聚焦于非法飞行器的侦测与反制，普遍缺乏对合法飞行器的交通管理功能，既无法实现航路交通的智能指挥，又难以依据法规开展交通执法，这种功能缺失导致低空航路面临合法飞行器交通拥堵疏导无策、违规行为处置无据等管理困境。
发明内容
本发明要解决的技术问题是提供一种能对合法飞行器进行执法管理的低空交通管理系统。
为解决上述技术问题，本发明提供一种低空飞行器交通管理系统，包括监管平台和巡管无人机，所述巡管无人机搭载有通信模块和摄像模块，通过通信模块通信连接所述监管平台和所在空域的低空飞行器；所述巡管无人机利用通信模块接收所在空域的低空飞行器的标识信息和飞行信息，并利用摄像模块拍摄所在空域的低空飞行器的视频数据，依据预设的低空飞行交通规则，根据所述标识信息、所述飞行信息和所述视频数据生成执法指令发给所在空域的低空飞行器进行执法管理；所述巡管无人机若判断自身不能对所在空域的低空飞行器进行执法管理，则将所述标识信息、所述飞行信息和所述视频数据回传给监管平台，所述监管平台按照管理人员的操作，依据预设的低空飞行交通规则生成执法指令发给巡管无人机，所述巡管无人机将所述执法指令发给所在空域的低空飞行器进行执法管理。
进一步地，所述巡管无人机有多个，多个巡管无人机可分别在不同空域进行飞行巡管。
进一步地，所述巡管无人机判断自身不能对所在空域的低空飞行器进行执法管理，系根据所述标识信息、所述飞行信息和所述视频数据来判断。
进一步地，所述巡管无人机包括中心计算模块，具体利用所述中心计算模块来判断自身不能对所在空域的低空飞行器进行执法管理，还利用所述中心计算模块生成执法指令。
进一步地，所述监管平台包括计算机和供管理人员操作的输入设备，具体利用所述计算机来按照管理人员的操作生成执法指令。
进一步地，所述通信模块是定义了A2X通信协议的5G移动通信模块，所述巡管无人机的通信模块通过A2X通信协议的广播信道向所在空域的低空飞行器发出执法指令。
进一步地，包括网络基站，所述网络基站包括5G基站和5G核心网，所述巡管无人机的通信模块通过所述网络基站通信连接所述监管平台。
进一步地，所述巡管无人机搭载有数据存储模块，将低空飞行器的标识信息、飞行信息和视频数据存储在所述数据存储模块中。
进一步地，所述巡管无人机搭载有用于驱离所在空域的低空飞行器或飞鸟的监管工具。
进一步地，所述监管工具包括无线电干扰设备、声波驱离器和/或无人机捕捉网。
本发明具有以下有益效果：巡管无人机依据预设的低空飞行交通规则，根据低空飞行器的标识信息、飞行信息和视频数据生成执法指令发给所在空域的低空飞行器进行执法管理，如此利用巡管无人机直接向低空飞行器发出执法指令，无需通过网络基站中转就能实现对合法低空飞行器进行高效的现场执法；巡管无人机若判断自身不能对所在空域的低空飞行器进行执法管理，则将低空飞行器的标识信息、飞行信息和视频数据回传给监管平台，监管平台按照管理人员的操作，依据预设的低空飞行交通规则生成执法指令发给巡管无人机，巡管无人机将执法指令发给所在空域的低空飞行器进行执法管理，如此利用监管平台按照管理人员的操作实现对合法低空飞行器进行精准执法。

具体实施方式
以下结合具体实施方式对本发明创造作进一步详细说明。
本实施例提供一种低空飞行器交通管理系统，如图1所示，低空飞行器交通管理系统包括监管平台、网络基站和巡管无人机，该网络基站包括5G基站和5G核心网，其中：5G基站是第五代移动通信技术(5G)的核心基础设施，负责无线信号的发射、接收和处理，为用户提供高速率、低时延、大连接的通信服务；5G核心网是在移动通信网络中，连接无线接入网与外部网络(如互联网、其他运营商网络)的核心基础设施。
本实施例中，巡管无人机有多个，多个巡管无人机可分别在不同空域进行飞行巡管。如图2所示，巡管无人机搭载有通信模块、摄像模块、监管工具、中心计算模块、指令播送模块、飞行数据采集模块和监管平台通信模块，摄像模块包括摄像头和视频数据采集模块，监管工具为能对低空飞行器实施管理措施的管理措施实施模块。该通信模块是定义了A2X通信协议的5G移动通信模块，A2X通信协议应用于无人机领域可以实现无人机与周围环境(其他无人机、基础设施、网络等)的实时信息交互和智能协同，以提升无人机交通安全、效率和自动飞行能力。低空飞行器搭载有5G移动通信模块。巡管无人机通过通信模块通信连接监管平台和所在空域的低空飞行器，且监管平台通信连接巡管无人机所在空域的低空飞行器，各个低空飞行器之间通信连接。其中，巡管无人机的通信模块通过网络基站构建的5G移动通信网通信连接监管平台，通过A2X通信协议的广播信道通信连接所在空域的低空飞行器，而监管平台通过网络基站通信连接巡管无人机所在空域的低空飞行器，各个低空飞行器之间通过A2X通信协议的广播信道通信连接。具体地，巡管无人机与5G基站之间、巡管无人机所在空域的低空飞行器与5G基站之间分别通过5G移动通信网络的无线空中接口Uu接口101进行通信连接，巡管无人机与其所在空域的低空飞行器之间、各个低空飞行器之间分别通过5G移动通信网络A2X通信协议的无线空中接口PC5接口102进行通信连接，5G基站与5G核心网之间、5G核心网与监管平台之间通过IP网络接口103进行通信连接。
巡管无人机搭载的监管工具包括无线电干扰设备、声波驱离器和无人机捕捉网，巡管无人机可通过无线电干扰设备向所在空域的低空飞行器发射定向射频信号，干扰低空飞行器的通信链路(如GPS、遥控信号)，迫使返航或降落，从而驱离所在空域的低空飞行器，或通过无人机捕捉网捕捉非法的低空飞行器，巡管无人机还可通过声波驱离器发射高频声波或特定声压，干扰飞鸟听觉系统(对鸟类有效)，从而驱离所在空域的飞鸟。
巡管无人机在飞行巡管时若有低空飞行器进入其空域，则通过其搭载的指令播送模块、通信模块的A2X通信协议的广播信道向所在空域的低空飞行器发出报告指令，要求所在空域的低空飞行器报告其标识信息和飞行信息，低空飞行器搭载有指令处理模块，在通过接收到巡管无人机报告指令之后，指令处理模块通过5G移动通信模块向巡管无人机发送表示信息和飞行信息。巡管无人机利用通信模块和飞行数据采集模块接收所在空域的低空飞行器的标识信息和飞行信息，并利用摄像模块拍摄所在空域的低空飞行器的视频数据，再将低空飞行器的标识信息、飞行信息和视频数据存入巡管无人机搭载的数据存储模块。中心计算模块根据数据存储模块中的低空飞行器的标识信息、飞行信息和视频数据研判巡管无人机所在空域的低空交通状况和低空飞行器的合法性，再根据低空交通状况判断巡管无人机自身能否对所在空域的低空飞行器进行执法管理。具体地，若巡管无人机所在空域的低空飞行器数量较少(例如不超过10个)，则中心计算模块根据低空飞行器的标识信息、飞行信息和视频数据研判出巡管无人机所在空域的低空交通状况简单，故判断巡管无人机自身能对所在空域的低空飞行器进行执法管理；若巡管无人机所在空域的低空飞行器数量较多(例如超过10个)，则中心计算模块根据低空飞行器的标识信息、飞行信息和视频数据研判出巡管无人机所在空域的低空交通状况复杂，故判断巡管无人机自身不能对所在空域的低空飞行器进行执法管理。
在判断出巡管无人机自身能对所在空域的低空飞行器进行执法管理的情况下，巡管无人机的中心计算模块依据预设的低空飞行交通规则，根据低空飞行器的标识信息、飞行信息和视频数据生成执法指令，通过其指令播送模块、通信模块的A2X通信协议的广播信道将执法指令发给所在空域的低空飞行器进行执法管理，督促合法低空飞行器遵守低空飞行交通规则、及时疏导交通拥塞，利用监管工具警告和反制非法侵入航线的非法低空飞行器，保证低空飞行空间的有序畅通。如此利用巡管无人机直接向低空飞行器发出执法指令，无需通过网络基站中转就能实现对合法低空飞行器进行高效的现场执法。
在判断出巡管无人机自身不能对所在空域的低空飞行器进行执法管理的情况下，巡管无人机的中心计算模块通过通信模块将数据存储模块中的低空飞行器的标识信息、飞行信息和视频数据经网络基站回传给监管平台，监管平台包括计算机、IP通信模块和供管理人员操作的输入设备，计算机作为指令处理模块，管理人员通过监管平台获知低空飞行器的标识信息、飞行信息和视频数据之后，可在监管平台的输入设备进行管理操作，监管平台利用计算机来按照管理人员的操作，依据预设的低空飞行交通规则生成执法指令，然后将该执法指令通过网络基站发给巡管无人机，巡管无人机通过其通信模块的A2X通信协议的广播信道将执法指令发给所在空域的低空飞行器进行执法管理，督促合法低空飞行器遵守低空飞行交通规则、及时疏导交通拥塞，利用监管工具警告和反制非法侵入航线的非法低空飞行器，保证低空飞行空间的有序畅通。如此利用监管平台按照管理人员的操作实现对合法低空飞行器进行精准执法。
本实施例中，由于监管平台通过网络基站通信连接巡管无人机所在空域的低空飞行器，因此在监管平台利用计算机来按照管理人员的操作，依据预设的低空飞行交通规则生成执法指令之后，可将该执法指令直接通过网络基站发给巡管无人机所在空域的低空飞行器进行执法管理。
