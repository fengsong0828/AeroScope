CN1326748C

Description

有线电力低空飞行器
技术领域
本发明涉及一种有线电力低空飞行器。
背景技术
当今世界飞行器种类繁多。但是，迄今为止，国际上用电作动力的飞行器还未出现。
发明内容
本发明所要解决的技术问题在于避免上述现有技术中的不足，而提出一种采用可移动输电装置和电力飞行器相连接而构成的一种新的交通工具——有线电力低空飞行器，电力飞行器携带可移动输电线沿微型轨道及电力线飞行，能得到源源不断的电力供应，故能实现沿微型轨道低空飞行。
本发明所提供的技术方案是：一种有线电力低空飞行器，由可移动输电装置和电力飞行器相连接构成，该可移动输电装置为支架桥输电装置，电力飞行器是由电动机作发动机的飞行器；所述支架桥输电装置是由微型轨道支架连接桥、铺在桥面上的微型轨道及电力线、行驶在轨道上的藏轮式微型输电车、连接飞行器与输电车之间的钢丝绳输电线和与输电车相连接的电路接通器以及飞行指路灯构成；其中：
藏轮式微型输电车是由车厢、车厢横轴、车轮、藏轮杆、藏轮轴和藏轮构成；
微型轨道支架连接桥是由钢筋混凝土桥墩、钢筋混凝土横梁、微型钢筋混凝土多眼桥、斜撑式桥梁支柱、斜撑式横梁支柱和斜拉支柱构成；
微型轨道及电力线是由藏轮专用轨和装在轨道间的电力线构成；
钢丝绳输电线是由钢丝绳和两条电力线交织组成；
电路接通器是由接通器杆及其所连接的接通器托和接通器托两端下面所装铜制小圆柱构成；
飞行指路灯是沿微型轨道支架连接桥两侧装一排红灯和一排绿灯。
进一步地，所述可移动输电装置为钢丝绳输电装置，该钢丝绳输电装置是由两岸的钢筋混凝土高塔、高塔顶部的钢筋混凝土横梁、牵引用钢丝绳、绝缘杆、钢丝绳电力线、电路接通器、细钢丝绳输电线和拉紧用铁环及钢丝绳构成；
钢丝绳电力线是由钢丝绳和电力线交织构成；
电路接通器是在3米长的绝缘杆两端各拴几枚套在钢丝绳电力上的铜环构成；
细钢丝绳输电线是由细钢丝绳和两条电力线交织组成。
进一步地，所述藏轮式微型输电车有牵引型和电力机车型。
进一步地，所述电力飞行器有直升电力飞行器、螺旋桨电力飞行器、螺旋桨式直升电力飞行器、飞艇式电力飞行器和轮船式电力飞行器。
进一步地，所述直升电力飞行器是由电动机作发动机的直升飞机；螺旋桨电力飞行器是由电动机作发动机的螺旋桨飞机；螺旋桨式直升电力飞行器是螺旋桨电力飞行器和直升电力飞行器的结合体；飞艇式电力飞行器是在螺旋桨电力飞行器上方装一个氢气囊或氦气囊；轮船式电力飞行器是外形近似轮船、上面开口、没有起落架的螺旋桨电力飞行器。
进一步地，所述牵引型输电车是由电力飞行器所牵引的藏轮式微型输电车；电力机车型输电车，是在藏轮式微型输电车中装电力机车的微型化动力设备，其操纵设备装在飞行器上。
进一步地，所述可移动输电装置的钢丝绳输电线和细钢丝绳输电线，其中钢丝绳和细钢丝绳的上端均固定在卷线轴上，钢丝绳下端连接输电车，细钢丝绳下端连接电路接通器的绝缘杆；电力线上端均与电动机接通，下端均与电路接通器接通。
进一步地，所述电力飞行器上装电路报警电铃，电铃开关拉绳连接在钢丝绳输电线和细钢丝绳输电线的危险段；藏轮式微型输电车上装电视监控设备。
进一步地，所述轮船式电力飞行器必须有专用码头，陆用电力飞行器必须有起降场，起降场有多条微型轨道及电力线，微型轨道两端由轮式可移动支架桥支撑，变更轨道使用轨道变更与连接高架车。
本发明有如下优点：更安全的交通工具，不用燃油，即使出了事故也不会爆炸和燃烧，速度快，安全，快捷，价廉的交通工具的社会效益是显而易见的。

具体实施方式
下面结合来详细说明本发明。
一种有线电力低空飞行器，由可移动输电装置和电力飞行器相连接构成，该可移动输电装置为支架桥输电装置，电力飞行器是由电动机作发动机的飞行器；所述支架桥输电装置是由微型轨道支架连接桥、铺在桥面上的微型轨道及电力线、行驶在轨道上的藏轮式微型输电车、连接飞行器与输电车之间的钢丝绳输电线和与输电车相连接的电路接通器以及飞行指路灯构成；其中：
藏轮式微型输电车如图1所示，是由车厢1，横轴2，车轮3，藏轮杆4，藏轮轴5，藏轮6组成的藏轮式微型输电车相连接而成，在配上相应的藏轮专用轨7，藏轮专用轨7比普通道轨高，轨面外侧宽，外侧下面能装下一个小轮，故称藏轮；
微型轨道支架连接桥是由钢筋混凝土桥墩、钢筋混凝土横梁、微型钢筋混凝土多眼桥、斜撑式桥梁支柱、斜撑式横梁支柱和斜拉支柱构成；
微型轨道及电力线是由藏轮专用轨和装在轨道间的电力线构成；
钢丝绳输电线是由钢丝绳和电力线交织组成；
电路接通器是由接通器杆及其所连接的接通器托和接通器托两端下面所装铜制小圆柱构成；
飞行指路灯是沿微型轨道支架连接桥两侧装一排红灯和一排绿灯。
另外，藏轮式微型输电车有牵引型和电力机车型两种。牵引型是由飞行器牵引的藏轮式微型输电车；电力机车型是在藏轮式微型输电车中装电力机车的微型化动力设备，其操纵设备装在飞行器上。当电力飞行器时速达到300公里，所牵引的藏轮式微型输电车若仍能在微型轨道上正常行驶，电力机车型输电车就没有用处了。否则，电力机车型用于高速飞行，牵引型用于一般速度飞行。
微型轨道支架连接桥是由深埋地下的钢筋混凝土桥墩、架在桥墩之上并和桥墩浇注为一体钢筋混凝土横梁、架在横梁之上的微型钢筋混凝土多眼桥、斜撑式桥梁支柱、斜撑式横梁支柱以及斜拉支柱构成。支架桥高度应居于高压线和其他明线之上。
钢丝绳上端固定在电力飞行器上的以压缩弹簧为轴心的卷线轴上，电力线上端与电动机接通，然后将钢丝绳输电线的大部分卷在卷线轴上，再将钢丝绳下端连接输电车，电力线下端与电路接通器接通。电路接通器是由固定在输电车后车厢上的两根接通器杆以及两接通器杆下端所连接的绝缘接通器托和接通器托两端下面所装的铜制小圆柱构成，两铜制小圆柱置于两条电力线的上面。电力飞行器必须沿轨道飞行，为了安全，要沿微型轨道支架桥两侧装一排红灯和一排绿灯，称为飞行指路灯。不论黑夜还是白天，只要有电力飞器沿轨道飞行，均要打开指路灯。输电车装电视监控设备，监控钢丝绳输电线是否偏离红绿灯线。有了电视监控，即使是大雾天也不会影响飞行。飞行器装电路报警电铃，开关拉绳连接在钢丝绳输电线的危险段。为了飞行安全，上行和下行不能在同一高度飞行，而且上下行均必须沿轨道右侧飞行，以免上下行飞行器相撞。如果仍不能确保安全，则应建造上行和下行各自独立的而且要相隔一定距离的支架桥输电装置。电力飞行器左翼尖端装定滑轮，钢丝绳输电线经由定滑轮再连接和牵引藏轮式微型输电车。当电力飞行器沿轨道右侧飞行时，所牵引的输电车及钢丝绳输电线就是一条可移动输电线，能将电源源不断地输往飞行器，可保飞行器沿轨道持续飞行。
钢丝绳输电装置以钢丝绳为依托，适用于水面。它是由建在水面两岸的钢筋混凝土高塔、浇注在高塔顶部的钢筋混凝土横梁、连接在横梁一侧的两条牵引用钢丝绳、两条牵引用钢丝绳所连接的3米多长的绝缘杆、绝缘杆两端所连接的钢丝绳电力线、套在两根钢丝绳电力线上的电路接通器、连接飞行器与电路接通器之间细钢丝绳输电线以及牵引用钢丝绳电力线拉紧装置构成。钢丝绳电力线是由钢丝绳和电力线交织组成。电路接通器是一根3米多长的绝缘杆，两端连接套在钢丝绳电力线上的几枚铜环构成。细钢丝绳输电线由细钢丝绳和两条电力线交织组成。细钢丝绳上端固定在电力飞行器的以压缩弹簧为轴心的卷线轴上，两条电力线与电动机接通，然后将细钢丝绳输电线的大部分卷在卷线轴上，再将细钢丝绳下端连接电路接通器的绝缘杆，两电力线下端与电路接通器接通。拉紧装置是套在牵引用钢丝绳上的铁环，铁环拴拉紧用钢丝绳。
钢丝绳输电装置跨度不可能很大，因为跨度太大了，钢丝绳电力线自身的重力将把钢丝绳电力线自身拉断。但在较窄的水面，如长江黄河两岸，上海崇明之间，舟山与宁波附近海岸之间、青岛黄岛之间、蓬莱长岛之间是可以建造的，若采取某些措施使其跨度加大，6000米以内的水面是有建造的可能性的。
具体做法是：取两条等长的牵引用钢丝绳，分别固定在高塔横梁的一侧，两条钢丝绳的高度和宽度相距均约2米，再取两条等长的钢丝绳电力线，分别与近3米长的绝缘杆的两端连接，绝缘杆两端再与两条牵引用钢丝绳连接，然后用同样的方法，将牵引用钢丝绳，绝缘杆和钢丝绳电力线相连接并用机械拉紧之后固定于对岸高塔横梁的同侧，第二步将拉紧装置中的拉紧用钢丝绳用机械拉紧固定于两高塔的中下部，在拉紧过程中可用多架直升机将钢丝绳电力线托起。拉紧之后两条钢丝绳电力线应该等长，才能确保两电力线不会因风力影响而相交。做成之后，钢丝绳电力线下垂最低点要达到一定的高度，避免被轮船桅杆撞坏，虽然陆用电力飞行器可以飞越水面，但万一出现故障却不能在水面降落，应该使用轮船式电力飞行器，这就需要建造飞行器专用码头。
假定水面宽6000米，钢丝绳电力线按3500米计算，两端的牵引用钢丝绳长度需要1600米。三者加起来为7600米。由于钢丝绳电力线的实际跨度不足3500米。距两岸码头均有一千几百米的距离。只有借助电力飞行器上的细钢丝绳输电线，才有可能飞临两岸的专用码头，因此，细钢丝绳输电线的长度必须达到要求。当电力飞行器沿钢丝绳电力线飞行，所牵引的电路接通器在两条钢丝绳电力线上向前滑行，此时细钢丝绳输电线和它所连接的电路接通器就是一条可移动输电线，能将电源源不断的输往飞行器。当电路接通器滑行至钢丝绳电力线的尽头，电路接通器不能再往前滑行，此时飞行器继续飞行，细钢丝绳输电线从飞行器中被逐步拉出，直至飞抵专用码头，为了这3500米长的钢丝绳电力线不至被自身的重力拉断，在制作钢丝绳时要将受重力最大的两端做得粗一些，以下逐步变细，这样的钢丝绳跨度可以大一些，至于能否被拉断，要对使用的钢丝绳进行计算，认为可行再建造。
电力飞行器是用电动机作发动机的飞行器，大约有下列五种：一是直升电力飞行器，是用电动机作发动机的直升飞机：二是螺旋桨电力飞行器，是用电动机作发动机的螺旋桨飞机，三是螺旋桨式直升电力飞行器，是螺旋桨电力飞行器与直升电力飞行器的结合体，既装螺旋桨又装直升起降与推进装置，易于起降，一旦发生故障易于着陆；四是飞艇式电力飞行器，是在螺旋桨电力飞行器上方装一个氢气囊或氦气囊，适合货运，五是轮船式电力飞行器，其外型近似轮船，上面开口，没有起落架的螺旋桨电力飞行器。电力飞行器均应装蓄电池，以应急需。
陆用电力飞行器必须建起降场。起降场应有多条微型轨道及电力线。两条微型轨道及电力线的两端即微型轨道及电力线的接头处，不是由固定的桥墩支撑，而是由轮式可移动支架桥支撑。变更轨道使用微型轨道变更与连接高架车。
当有线电力低空飞行器牵引藏轮式微型输电车沿微型轨道飞行时，钢丝绳输电线及藏轮式输电车的重力大于压缩弹簧的弹力，钢丝绳输电线被逐步拉出，当电力飞行器达到正常飞行高度，此时重力和弹力应处于平衡状态，钢丝绳输电线不再被拉出，电力飞行器沿轨道正常飞行。若电力飞行器继续升高，飞行器对输电车的拉力加大，输电车的前轮3将被抬起，藏轮6同时被抬起并与上方的轨道7接触，藏轮6将背靠轨道7旋转，此时输电车的重力加大，钢丝绳输电线继续被拉出，当拉至危险段时，电铃开关拉绳将打开开关，电铃立即报警。这时电力飞行器应立即下降，一切才能恢复正常。若电力飞行器偏离微型轨道，输电车车轮3的侧面与藏轮专轨7之间的摩擦力加大，重力加上摩擦力大于压缩弹簧的弹力，钢丝绳输电线同样将被拉出，至危险段时电铃报警。此时电力飞行器应立即靠近微型轨道飞行，一切才能恢复正常。如果电力飞行器继续升高或继续偏离微型轨道，电力飞行器必将把钢丝绳输电线拉断，电路立即中断。此时电力飞行器只能借助蓄电池的电力立即着陆。
有线电力低空飞行器作为一种新的交通工具若能成功，与飞机，火车，汽车相比，有以下优势：第一，有线电力低空飞行器有条件成为比飞机火车汽车更安全的交通工具，特别是更安全的客运工具。人是世间最宝贵的财富，交通安全与否是第一位的，有线电力低空飞行器不用燃油，即使出了事故也不会爆炸和燃烧，这一点决定了它比飞机汽车安全。由于在低空飞行，上不会和高空飞行的飞机相撞，下不会和地面上的火车，汽车，行人相撞，由于必须沿微型轨道飞行，也不会和建筑物、山头相撞，也由于在低空飞行，万一出了事故，甚至摔在地上，和飞机从高空摔下相比死伤要少得多，有线电力飞行器虽然要沿微型轨道飞行，但仍有一定的飞行自由度，不会像火车汽车遇紧急情况无处可躲，飞行器可以立即转向，可以拉断钢丝绳输电线紧急着陆，所在一般不会发生碰撞和追尾事故，但是，也存着安全隐患，即输电装置易于遭受人为的破坏。预防方法之一，是在浇注桥墩和桥梁支柱时，将金属尖状物、刃状物或碎玻璃等浇注在桥墩上半部和桥梁支柱上并露出表面，使人难以攀登。第二，它的基础设施投资少，见效快，特别是占用耕地甚少，除起降场需要征地外，支架桥输电装置可沿公路或铁路建造，不需要征地，这点是火车汽车难以比拟的，第三，飞行速度快，不但能大大超过火车，汽车的速度，而且可以达到并超过高速铁路的速度，当然无法和喷气式客机相比，但是，由于起降场可建在近郊乘客赴起降场比赴机场用时少。因此，1000公里以内，乘有线电力低空飞行器可能比乘飞机更快捷。第四，有线电力低空气飞行器基本上不受天气变化和自然灾害的影响，无论是风霜雨雪和大雾天，均能照常飞行，即便发生了洪灾，洪水可以冲垮甚至淹没公路和铁路，却不能冲垮微型轨道，这点是飞机，火车，汽车所不及的，第五，有线电力低空飞行器有条件成为比飞机、火车汽车更便易的客运工具，由于所用能源价格低廉，由于占用资金少，由于基础设施是微型的，易于维护，所需员工少等诸多因素，经营成本是相当低的，即便票价高于火车汽车，但由于速度快，比乘火车汽车所花的时间成本少，加上时间成本综合计算，电力飞行器的票价应该更便宜。第六，社会效益显著，安全，快捷，价廉的交通工具的社会效益是显而易见的，能促进国民经济的发展。
