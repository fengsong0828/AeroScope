Description

具有散热结构的驱动系统和低空飞行器
技术领域
本实用新型涉及低空飞行设备技术领域，具体而言，涉及一种具有散热结构的驱动系统和低空飞行器。
背景技术
随着低空经济的热度越来越高，电动垂直起降飞行器(eVTOL)因其安全、智能、经济、环保等特点而广受大众的青睐，电动垂直起降飞行器(eVTOL)使得人和货物能够以无缝且经济的方式在城市的低空快速流动以及灵活作业，有利于快速开发城市的低空空域资源，使得电动垂直起降飞行器(eVTOL)逐渐成为城市空中交通的主流发展方向。
现有的电动垂直起降飞行器(eVTOL)主要采用电机直驱，即，通过外转子带动螺旋桨旋转，从而驱动电动垂直起降飞行器(eVTOL)进行作业，这种采用电机直驱方式的电动垂直起降飞行器(eVTOL)存在以下技术问题：电机的定子和/或电机内部的轴承没有润滑油的润滑和冷却，导致轴承长期使用过程中，轴承只能通过消耗自身的润滑脂来实现润滑，由于轴承自身的润滑脂的消耗导致轴承的使用寿命降低，严重影响电动垂直起降飞行器(eVTOL)的正常作业，存在飞行失效的风险。
现有的部分电动垂直起降飞行器(eVTOL)通过电机直驱，容易导致电机长时间在低转速区间范围内工作，无法发挥电机的高转速特性，不利于驱动系统的驱动效率，同时还需要较大的电机尺寸。
现有的部分电动垂直起降飞行器(eVTOL)通过设备内部的润滑油来实现对定子和轴承的润滑和冷却，但是，上述经过换热后的润滑油往往处于高温状态，无法有效带走定子和/或轴承上的热量。
实用新型内容
本实用新型的主要目的在于提供一种具有散热结构的驱动系统和低空飞行器，以解决现有技术中的部分电动垂直起降飞行器(eVTOL)通过设备内部的润滑油来实现对定子和轴承的润滑和冷却，但是，上述经过换热后的润滑油往往处于高温状态，无法有效带走定子和/或轴承上的热量的问题。
为了实现上述目的，根据本实用新型的一个方面，提供了一种有散热结构的驱动系统，用于与低空飞行器的螺旋桨驱动连接，驱动系统包括电机壳体、电机组件、行星组件、泵体结构和散热结构，电机壳体具有容纳腔以及与容纳腔连通的第一避让过孔，电机壳体具有通油油道；电机组件设置在容纳腔内，且电机组件包括电机驱动轴和太阳轮，太阳轮套设在电机驱动轴的外周侧，容纳腔与太阳轮相对的腔壁面处具有齿圈结构；行星组件包括行星架和行星轮，行星架的轴向一端的至少部分罩设在电机驱动轴的外周侧，行星架的轴向另一端穿出第一避让过孔以用于与螺旋桨驱动连接；行星轮可转动地设置在行星架上，并同时与太阳轮、齿圈结构均外啮合配合；泵体结构设置在电机壳体背离第一避让过孔的一端，泵体结构具有相连通的储油腔和回油腔，储油腔与通油油道的油道入口连通；其中，位于第一避让过孔内的行星架的部分外周面与第一避让过孔的部分孔壁面之间形成第一环形空间，第一环形空间用于容纳第一轴承结构，通油油道的第一油道出口与第一环形空间连通，以对第一轴承结构进行润滑和冷却，第一环形空间与回油腔连通；和/或，电机组件还包括定子结构，容纳腔具有第一子环腔和第二子容纳腔，定子结构位于第一子环腔内，至少行星轮和齿圈结构位于第二子容纳腔内，且第二子容纳腔与第一子环腔连通，通油油道的第二油道出口与第二子容纳腔连通，以对位于第二子容纳腔内的行星轮、齿圈结构进行润滑和冷却，以及对位于第一子环腔内的定子结构进行冷却，第一子环腔与回油腔连通；散热结构设置在泵体结构背离电机壳体的轴向一端，以用于对储油腔和/或回油腔内的润滑油进行散热。
进一步地，散热结构与泵体结构可拆卸地连接。
进一步地，电机壳体与电机组件同心设置；和/或，电机组件与泵体结构同心设置；和/或，散热结构与泵体结构同心设置。
进一步地，散热结构包括散热壳体，散热壳体具有容纳槽，且容纳槽的槽口朝向泵体结构一侧，以使容纳槽的槽壁面与泵体结构的转子泵壳体背离电机壳体一端的端部表面之间形成安装空间；驱动系统还包括电机控制器，电机控制器设置在安装空间内，并与电机组件控制连接。
进一步地，散热壳体背离泵体结构一端的端面上具有第一安装区域和第二安装区域，且第二安装区域位于第一安装区域的外周侧；散热结构还包括风扇和散热翅片，其中，风扇可转动地设置在第一安装区域处；散热翅片为多个，多个散热翅片呈放射状设置在第二安装区域处。
进一步地，风扇的扇叶的数量小于散热翅片的数量。
进一步地，散热翅片在散热壳体的径向上的长度大于风扇的扇叶在散热壳体的径向上的长度。
进一步地，行星架包括行星架本体，行星架本体在远离电机组件的方向上包括依次连接的第一行星轴段、第二行星轴段、第三行星轴段、第四行星轴段；第一行星轴段的外直径D1、第二行星轴段的外直径D2、第三行星轴段的外直径D3、第四行星轴段的外直径D4之间满足：D1＝D4＜D3＜D2，行星轮可转动地设置在第二行星轴段处。
进一步地，第二行星轴段具有多个行星齿轮轴，多个行星齿轮轴绕第二行星轴段的周向间隔设置，行星轮为多个，多个行星轮与多个行星齿轮轴一一对应，各行星轮均套设在对应的各行星齿轮轴的外周侧；行星齿轮轴的外周面与行星轮的内周面之间形成第三环形空间，驱动系统还包括第三轴承结构，第三轴承结构位于第三环形空间内。
进一步地，电机壳体包括筒体结构、顶部盖板、底部盖板和分腔结构，筒体结构的轴向两端均呈敞口状，筒体结构具有容纳腔；顶部盖板盖设在筒体结构的轴向第一端，且顶部盖板的轴向第一端具有第一避让过孔；底部盖板盖设在筒体结构的轴向第二端；其中，筒体结构的筒壁面、顶部盖板朝向筒体结构一侧的顶部壁面、底部盖板朝向筒体结构一侧的底部壁面围成容纳腔，筒壁面上凸出设置有支撑凸台；分腔结构搭接在支撑凸台上，以将容纳腔分成第一子容纳腔和第二子容纳腔，分腔结构具有第三避让过孔，电机驱动轴的第一端位于第一子容纳腔内，电机驱动轴的第二端穿过第一避让过孔并伸入第二子容纳腔内设置；第二子容纳腔的腔壁面上具有齿圈结构，第一行星轴段、第二行星轴段均位于第二子容纳腔内，第三行星轴段、第四行星轴段的部分均位于第一避让过孔处；第三行星轴段的外周面与第一避让过孔的孔壁面之间形成第一环形空间，驱动系统还包括第一轴承结构，第一轴承结构位于第一环形空间内；第四行星轴段的部分外周面与第一避让过孔的孔壁面之间夹设有油封结构，且第四行星轴段远离第三行星轴段的端部伸出第一避让过孔设置。
进一步地，底部盖板背离第一避让过孔的一端开设有第二避让过孔，第二避让过孔的外周处围设有环形凸台，以在环形凸台的内圈和第二避让过孔的孔壁面之间形成环形支撑面，环形支撑面的内圈和第二避让过孔的孔壁面之间形成避让沉台；电机驱动轴背离行星组件一端的外周面与环形凸台的内周面之间形成第二环形空间；驱动系统还包括第二轴承结构，第二轴承结构位于第二环形空间内，并支撑设置在环形支撑面上。
进一步地，在电机壳体的径向上，第一子容纳腔由外向内依次包括相连通的第一子环腔、第二子环腔、中心腔；电机组件包括电机驱动轴、转子结构、环形挡油板和定子结构，其中，电机驱动轴的部分可转动地设置在中心腔内，电机驱动轴套设有太阳轮的轴段伸入第二子容纳腔内设置；转子结构套设在电机驱动轴的外周侧，并位于第二子环腔内；环形挡油板套设在转子结构的外周侧，并位于第二子环腔与第一子环腔的分界处；定子结构套设在环形挡油板的外周侧，定子结构位于第一子环腔内。
进一步地，行星架本体具有贯通孔，贯通孔至少贯通行星架本体朝向电机组件一端的端面，以使电机驱动轴具有太阳轮的一端伸入贯通孔内，且用于安装行星轮的环形区域与贯通孔连通；电机驱动轴具有第四避让过孔，第四避让过孔贯通电机驱动轴的轴向两端；行星架还包括贯通轴，贯通轴的第一端伸入贯通孔内设置，贯通轴的第二端穿过第四避让过孔、第二避让过孔并与泵体结构驱动连接。
进一步地，泵体结构包括转子泵壳体，转子泵壳体具有储油腔和回油腔，转子泵壳体设置在电机壳体背离第一避让过孔的一端，转子泵壳体背离电机壳体的一端具有第六避让过孔；第一安装区域与电机控制器的第七避让过孔相对的位置处开设有第八避让过孔；贯通轴的第二端依次穿过第六避让过孔、第七避让过孔、第八避让过孔并与风扇驱动连接。
贯通轴电机壳体的外周面设置有多个第一散热片，多个第一散热片呈放射状绕电机壳体的周向间隔设置；和/或，泵体结构的转子泵壳体的外周面设置有多个第二散热片，多个第二散热片呈放射状绕转子泵壳体的周向间隔设置。
根据本实用新型的另一方面，提供了一种低空飞行器，包括驱动系统和螺旋桨，驱动系统与螺旋桨驱动连接，驱动系统为上述的驱动系统。
应用本实用新型的技术方案，提供了一种具有行星组件的驱动系统，驱动系统包括电机壳体、电机组件和行星组件，其中，电机壳体具有容纳腔以及与容纳腔连通的第一避让过孔；电机组件设置在容纳腔内，且电机组件包括电机驱动轴和太阳轮，太阳轮套设在电机驱动轴的外周侧，容纳腔与太阳轮相对的腔壁面处具有齿圈结构；行星组件包括行星架和行星轮，其中，行星架的轴向一端的至少部分罩设在电机驱动轴的外周侧，行星架的轴向另一端穿出第一避让过孔以用于与螺旋桨驱动连接；行星轮可转动地设置在行星架上，并同时与太阳轮、齿圈结构均外啮合配合。这样，通过行星架与螺旋桨驱动连接，在电机组件的基础上，增设行星组件以作为减速装置，能够显著提升驱动系统的驱动效率，充分利用电机驱动轴的高转速特性，有利于整个驱动系统的小型化设计。
此外，本申请通过设置散热结构，散热结构起到了对储油腔和/或回油腔内的润滑油进行散热的目的，从而确保储油腔和/或回油腔内的润滑油能够始终保持预设的温度，以确保其的润滑和换热效果。
