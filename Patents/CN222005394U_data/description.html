Description

一种具有低空飞行警示构件的低空飞行器
技术领域
本实用新型涉及无人机技术领域，具体为一种具有低空飞行警示构件的低空飞行器。
背景技术
无人飞行器在夜间飞行时需满足一些要求，比如需要提供警示灯，为空中其他飞行物和底面人员提供警示或作为无人飞行器的运动标记，警示灯一般通过固定频率的闪烁来实现警示，其发出的光线能够覆盖的范围越大越好。
目前，无人飞行器的警示灯分为集成式和拆装式两类，集成式的警示灯预装在无人飞行器上，虽然便于控制无人飞行器的体积和重量，但警示灯不能依据需求进行定制；而拆装式警示灯常采用抱箍安装在无人机的脚架上，由于抱箍采用螺栓进行紧固，安装警示灯需要频繁紧固螺栓导致安装不方便，且需要一根延长线对警示灯供电，延长线压影响无人飞行器的稳定性。
实用新型内容
本实用新型的目的在于提供一种具有低空飞行警示构件的低空飞行器，以解决上述背景技术中提出的问题。
为实现上述目的，本实用新型提供如下技术方案：一种具有低空飞行警示构件的低空飞行器，包括飞行器，所述飞行器下表面安装有装配座，所述装配座内部安装有四个接电头，装配座使得飞行器具备安装基座，便于选择合适的警示灯；
警示灯，所述警示灯上表面设有挡板，所述挡板上设有螺接帽，所述螺接帽与装配座螺接，所述螺接帽上表面设有第一通电环和第二通电环，且第一通电环和第二通电环与警示灯电性连接，所述第一通电环和第二通电环与对应的接电头接触；
通过第一通电环和第二通电环与接电头接触供电的方式，可以免去延长线，避免延长线影响飞行器的稳定性；
伸缩弹簧杆，所述伸缩弹簧杆下端与挡板转动连接，所述装配座表面设有插接孔，所述伸缩弹簧杆上端插进插接孔内部，可以避免警示灯转动，确保第一通电环和第二通电环与接电头接触良好。
进一步的，所述装配座内壁设有管壳，所述接电头滑动连接在管壳内部，所述接电头内部与管壳之间设有弹簧，通过弹簧将接电头推向第一通电环和第二通电环，避免出现接触不良的问题。
进一步的，所述警示灯的外壳表面设有防撞网，且防撞网罩住警示灯，对警示灯起到防护作用，将警示灯拆下来进行携带时，防止警示灯意外摔坏。
与现有技术相比，本实用新型的有益效果是：
(1)通过装配座使得警示灯以螺接的方式与飞行器连接，可以拆装警示灯，也可以更换与装配座匹配的警示灯，可以根据需求选择警示灯。
(2)警示灯通过第一通电环和第二通电环与接电头接触进行供电，无需在警示灯和飞行器之间使用延长线供电，不仅省去延长线，也可以避免延长线影响飞行器的稳定性。
(3)通过伸缩弹簧杆与装配座的插接限制警示灯转动，避免飞行器在飞行的过程中警示灯松动，预防警示灯坠落，提高飞行器的安全性。

实施例：
请参阅图1-5，本实用新型提供一种技术方案：一种具有低空飞行警示构件的低空飞行器，包括飞行器1，所述飞行器1下表面安装有装配座2，所述装配座2内部安装有四个接电头17，四个接电头17与飞行器1电性连接，使得接电头17可以为警示灯5供电，并使得警示灯5被飞行器1的遥控器控制；
警示灯5，所述警示灯5上表面设有挡板3，所述挡板3上设有螺接帽16，所述螺接帽16与装配座2螺接，所述螺接帽16上表面设有第一通电环10和第二通电环11，且第一通电环10和第二通电环11与警示灯5电性连接，所述第一通电环10和第二通电环11与对应的接电头17接触；
挡板3与装配座2抵接后，可以起到防尘作用，避免颗粒物进入到装配座2内部，警示灯5通过螺接帽16与装配座2螺接，实现警示灯5的可拆卸功能，每两个接电头17与第一通电环10和第二通电环11接触，相较于两个接电头17，确保供电不易出现接触不良的问题，实现飞行器1对警示灯5的供电。
伸缩弹簧杆7，所述伸缩弹簧杆7下端与挡板3转动连接，所述装配座2表面设有插接孔9，所述伸缩弹簧杆7上端插进插接孔9内部，可以限制挡板3和螺接帽16转动，预防警示灯5松动。
在本实施例中，如图3所示，所述装配座2内壁设有管壳12，所述接电头17滑动连接在管壳12内部，所述接电头17内部与管壳12之间设有弹簧13，弹簧13使得接电头17具有伸缩能力，当第一通电环10和第二通电环11抵住接电头17时，弹簧13推动接电头17稳定的与第一通电环10和第二通电环11接触。
在本实施例中，如图3所示，所述管壳12表面设有限位槽14，所述接电头17侧面设有凸块15，且凸块15与限位槽14滑动连接，限位槽14卡住凸块15后，防止接电头17从管壳12内脱落。
在本实施例中，如图2所示，所述警示灯5的外壳表面设有防撞网4，且防撞网4罩住警示灯5，防撞网4对警示灯5起到防摔作用，预防警示灯5摔坏。
在本实施例中，如图2所示，所述挡板3侧面设有孔板6，所述孔板6内部插入有销钉，所述伸缩弹簧杆7套在销钉表面，实现伸缩弹簧杆7与挡板3的转动连接，在伸缩弹簧杆7与插接孔9分离的状态下，不会影响警示灯5的拧动。
在本实施例中，如图2所示，所述伸缩弹簧杆7表面插接有拨动杆8，方便压缩伸缩弹簧杆7与插接孔9分离，拆卸警示灯5较为方便。
具体的，使用时，把警示灯5与飞行器1连接在一起时，螺接帽16与装配座2对齐后再拧动，直到挡板3抵住装配座2，此时，第一通电环10和第二通电环11与对应的接电头17接触，伸缩弹簧杆7与插接孔9对齐，转动伸缩弹簧杆7使其插进插接孔9内部，完成警示灯5与飞行器1的装配，飞行器1的电力通过接电头17、第一通电环10、第二通电环11传输给警示灯5，使得警示灯5被点亮；
在警示灯5的挡板3和螺接帽16与装配座2匹配的情况下，可以对警示灯5进行多种设计，便于依据需求选择警示灯5的类型，使用更灵活，警示灯5处于飞行器1下表面，不会过多影响飞行器1的中心，提高飞行器1飞行转向的稳定性。
