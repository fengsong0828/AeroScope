Description

一种低空飞行器全方位监管系统
技术领域
本发明涉及运输管理技术领域，尤其涉及一种低空飞行器全方位监管系统。
背景技术
随着无人机和低空飞行器的广泛应用，低空空域的飞行器数量迅速增长，对飞行器的实时监管和安全管理需求愈发突出。然而，传统监管系统通常侧重于高空航空交通管理，缺乏对低空飞行器的精细化管理手段。当前部分低空监管技术通过单一参数监测飞行器，但难以实现对多维数据的全面监控。个体异常行为的精确识别和群体协同风险的有效分析存在不足。此外，大多数现有系统的调整策略仅限于静态预设，缺乏根据实时飞行行为动态调整的能力。
因此，本发明提供一种低空飞行器全方位监管系统。
发明内容
本发明提供一种低空飞行器全方位监管系统，通过构建全方位监管系统的监管网络，实时监测全方位监管系统的飞行数据以及环境数据，确定所有低空飞行器的独立飞行行为，确定所有低空飞行器的群体飞行行为，以确定调整策略并进行管理，可以提升通信稳定性与效率，实现精细化、智能化的个体与群体风险的协调管理，提升了飞行器的自主决策能力和监管系统的智能化水平，提升低空空域的安全性和运行效率，满足复杂空域的实时监管需求，增强空域安全性与适应性。
本发明提供一种低空飞行器全方位监管系统，包括：
监测模块：构建全方位监管系统的监管网络，实时监测全方位监管系统中所有低空飞行器在指定时间周期内的飞行数据以及环境数据；
独立模块：对全方位监管系统中每个低空飞行器的子飞行数据以及子环境数据进行独立分析，确定所有低空飞行器的独立飞行行为；
群体模块：对全方位监管系统中所有低空飞行器的飞行数据、环境数据以及独立飞行行为进行综合分析，确定所有低空飞行器的群体飞行行为；
管理模块：基于所有低空飞行器的独立飞行行为以及群体飞行行为，确定调整策略并进行管理。
根据本发明提供的一种低空飞行器全方位监管系统，所述监测模块，包括：
确定单元：确定全方位监管系统中所有低空飞行器的监管范围以及通信需求；
第一区域单元：对监管范围进行网格划分，确定全方位监管系统的第一区域，其中，第一区域包括多个子区域；
第一类别单元：基于全方位监管系统的通信需求，对第一区域中的所有子区域进行分类，确定每个子区域的第一类别，其中，第一类别分别为重点区域、边缘区域、基站区域以及高空区域；
部署单元：对是重点区域的所有子区域进行基于5G基站的密集部署，对是边缘区域的所有子区域进行边缘计算节点部署，对是基站区域的所有子区域进行基于5G的疏散部署，对是高空区域的所有子区域进行高空通信部署；
网络单元：基于第一区域中所有子区域的部署结果，构建全方位监管系统的监管网络。
根据本发明提供的一种低空飞行器全方位监管系统，所述监测模块，还包括：
子飞行数据单元：基于低空飞行器的第一传感器组实时监测全方位监管系统中每个低空飞行器在指定时间周期内的子飞行数据；
飞行数据单元：基于所有低空飞行器的子飞行数据，确定全方位监管系统在指定时间周期内的飞行数据；
子环境数据单元：基于低空飞行器的第二传感器组实时监测全方位监管系统中每个低空飞行器在指定时间周期内的子环境数据；
环境数据单元：基于所有低空飞行器的子环境数据，确定全方位监管系统在指定时间周期内的环境数据。

独立分析单元：基于每个低空飞行器的子区域集合中所有子区域部署的边缘计算节点以及低空飞行器的端侧处理器对低空飞行器预处理后的子运行数据和子环境数据进行独立分析，确定每个低空飞行器的第一子独立飞行行为或第二子独立飞行行为；

飞行合规结果以及飞行安全结果子单元：基于飞行状态结果为正常飞行的每个低空飞行器的飞行基准模型的输出结果，确定正常飞行的每个低空飞行器的飞行合规结果以及飞行安全结果，其中，飞行合规结果包括合规飞行以及违规飞行，飞行安全结果包括安全飞行以及风险飞行；
违规飞行器子单元：确定飞行状态结果为正常飞行且飞行合规结果为违规飞行的每个低空飞行器为违规飞行器，并发出违规飞行警报；
风险飞行器子单元：确定飞行状态结果为正常飞行且飞行安全结果为风险飞行的每个低空飞行器为风险飞行器，并发出风险飞行警报；
第一子独立飞行行为子单元：基于未发出警报的每个低空飞行器的飞行状态结果、飞行合规结果以及飞行安全结果，确定每个低空飞行器的第一子独立飞行行为；
第二子独立飞行行为子单元：基于发出异常飞行警报、违规飞行警报或风险飞行警报的任一警报的每个低空飞行器的飞行状态结果、飞行合规结果、飞行安全结果以及发出的警报，确定每个低空飞行器的第二子独立飞行行为。

实施例2：
本发明实施例提供一种低空飞行器全方位监管系统，所述监测模块，包括：
确定单元：确定全方位监管系统中所有低空飞行器的监管范围以及通信需求；
第一区域单元：对监管范围进行网格划分，确定全方位监管系统的第一区域，其中，第一区域包括多个子区域；
第一类别单元：基于全方位监管系统的通信需求，对第一区域中的所有子区域进行分类，确定每个子区域的第一类别，其中，第一类别分别为重点区域、边缘区域、基站区域以及高空区域；
部署单元：对是重点区域的所有子区域进行基于5G基站的密集部署，对是边缘区域的所有子区域进行边缘计算节点部署，对是基站区域的所有子区域进行基于5G的疏散部署，对是高空区域的所有子区域进行高空通信部署；
网络单元：基于第一区域中所有子区域的部署结果，构建全方位监管系统的监管网络。
该实施例中，确定所有低空飞行器在全方位监管系统中的活动区域，明确监管覆盖范围，并分析飞行器在不同区域内的通信需求，如数据传输带宽、实时性要求等。
该实施例中，将整个监管范围划分为网格化的第一区域，以便实现分布式部署。
该实施例中，基于通信需求，将子区域分类为四种类型：重点区域：通信需求高、飞行器密集的区域；边缘区域：处于监管网络边界，通信较弱的区域；基站区域：已有5G基站覆盖的区域；高空区域：飞行器在较高空域活动，需要专门通信支持的区域。
该实施例中，对是重点区域的所有子区域进行密集部署5G基站，覆盖主要低空飞行空域，确保5G基站支持超低时延通信（1ms以下），以满足飞行器的实时数据上传需求。
该实施例中，对是边缘区域的所有子区域部署边缘计算节点，处理飞行器数据分析与任务调度，减少云端处理延迟，利用边缘节点进行数据预处理，如飞行轨迹预测、异常行为检测等。
该实施例中，基站区域表示5G和高空基站能直接覆盖的子区域，通信性能佳，对是基站区域的所有子区域疏散部署，以满足飞行器的实时数据上传需求。
该实施例中，对是高空区域的所有子区域部署高空通信设备（如5G气球、无人机基站）以补充5G地面网络。
该实施例中，基于各子区域的通信设备部署结果，构建覆盖全区域的多层次监管网络，保证飞行器在所有区域内的稳定通信与监管。
该实施例中，监管网络是利用5G的高带宽低时延特性和卫星互联网的全球覆盖优势构建的混合网络，既可提供高速网络接入，又保证在地面和高空的无缝连接。
上述技术方案的有益效果：对第一区域进行差异化部署，构建全方位监管系统的监管网络，优化了监管系统的网络覆盖和资源配置，提升了通信稳定性与效率，实现了精细化、智能化的低空监管网络，满足复杂空域的实时监管需求。

实施例5：
本发明实施例提供一种低空飞行器全方位监管系统，所述独立分析单元，包括：

飞行状态子单元：将每个低空飞行器的运行向量以及环境向量输入到对应飞行基准模型中，基于飞行基准模型的输出结果，判断每个低空飞行器的飞行状态结果，其中，飞行状态结果包括正常飞行以及异常飞行；
异常飞行器子单元：确定飞行状态结果为异常飞行的每个低空飞行器为异常飞行器，并发出异常飞行警报；
飞行合规结果以及飞行安全结果子单元：基于飞行状态结果为正常飞行的每个低空飞行器的飞行基准模型的输出结果，确定正常飞行的每个低空飞行器的飞行合规结果以及飞行安全结果，其中，飞行合规结果包括合规飞行以及违规飞行，飞行安全结果包括安全飞行以及风险飞行；
违规飞行器子单元：确定飞行状态结果为正常飞行且飞行合规结果为违规飞行的每个低空飞行器为违规飞行器，并发出违规飞行警报；
风险飞行器子单元：确定飞行状态结果为正常飞行且飞行安全结果为风险飞行的每个低空飞行器为风险飞行器，并发出风险飞行警报；
第一子独立飞行行为子单元：基于未发出警报的每个低空飞行器的飞行状态结果、飞行合规结果以及飞行安全结果，确定每个低空飞行器的第一子独立飞行行为；
第二子独立飞行行为子单元：基于发出异常飞行警报、违规飞行警报或风险飞行警报的任一警报的每个低空飞行器的飞行状态结果、飞行合规结果、飞行安全结果以及发出的警报，确定每个低空飞行器的第二子独立飞行行为。
该实施例中，通过边缘计算节点和飞行器自身的端侧处理器，对低空飞行器的子运行数据和子环境数据进行特征提取，分别生成描述飞行器状态的运行向量和描述环境状况的环境向量，为后续分析提供标准化的数值表达。
该实施例中，将运行向量和环境向量输入到预设的飞行基准模型（基于历史数据和规则训练的模型）中，判断飞行器是否处于正常飞行或异常飞行状态。
该实施例中，对飞行状态结果判定为异常飞行的飞行器，立即标记为异常飞行器，并发出预警，通知监管系统采取干预措施。
该实施例中，针对正常飞行的飞行器，分析其运行是否符合法规要求，输出合规飞行或违规飞行结果，违规飞行可能涉及飞行高度、航线偏离等问题。

该实施例中，将正常飞行但判定为违规飞行的飞行器标记为违规飞行器，并发出警报，提示监管系统关注。
该实施例中，将正常飞行但判定为风险飞行的飞行器标记为风险飞行器，并发出预警，提示其潜在的安全隐患。

该实施例中，对触发异常飞行、违规飞行或风险飞行警报的飞行器，综合其状态、合规性、安全性及警报内容，生成第二子独立飞行行为，用于进一步跟踪和管理。
上述技术方案的有益效果：确定每个低空飞行器的第一子独立飞行行为或第二子独立飞行行为，可以提升飞行器状态评估和行为管理的自动化与智能化水平，显著增强低空飞行器的安全性与监管效率。
