Description

一种基于改进的快速扩展随机树的固定翼飞行器的低空轨迹 规划方法
技术领域
本发明属于固定翼飞行器低空轨迹规划技术领域，尤其涉及一种基于改进的快速扩展随机树的固定翼飞行器的低空轨迹规划方法。
背景技术
对于无人机低空飞行来讲，由于与地面相撞的风险增加，在设计低空飞行机动轨迹时，不但要考虑飞行任务的完成，还需要将飞行的安全性、可操作性和飞机本身的性能约束加以考虑，对于飞机的机动轨迹而言，需要根据战场的环境，确定飞行的高度、速度、方位，随着战场环境的变化调整机动轨迹，面对低空场景，利用山脊地形因素实现躲避地方雷达或快速脱离敌方导弹的机动轨迹的实现，能够确保飞机满足近地防撞等安全性需求。
在固定翼无人机低空复杂地形的轨迹规划中，现有基于传统RRT(快速扩展随机树)的算法主要存在两个问题：探索效率不足：在复杂地形约束下，算法探索可行路径的过程效率较低。路径冗余度高：生成的路径往往包含过多不必要的节点和转折，导致路径不够简洁。同时，针对低空场景中需满足固定翼无人机动力学约束(如最小转弯半径和最大爬升角)的3D-Dubins路径规划，现有方法在确保路径曲线光滑性和在复杂威胁环境中实现完全避障方面仍存在挑战。
因此，本发明旨在解决如何在低空复杂地形环境中，高效地生成满足固定翼无人机动力学约束(最小转弯半径、最大爬升角)且路径冗余度低、光滑可行，并能完全避开威胁区域的飞行轨迹。
发明内容
为解决上述技术问题，本发明提出了一种基于改进的快速扩展随机树的固定翼飞行器的低空轨迹规划方法，以解决上述现有技术存在的问题。
为实现上述目的，本发明提供了一种基于改进的快速扩展随机树的固定翼飞行器的低空轨迹规划方法，包括：
获取固定翼飞机低空运动的节点与规划空间，基于定义的节点初始化两棵随机生成树；
在所述规划空间内对两棵随机生成树进行协同扩展，直至两棵随机生成树之间的距离满足阈值要求，获得可行路径点集；
采用贪心算法对所述可行路径点集进行优化后连接获得2D路径；
对低空运动场景的高度进行级别划分，根据高度级别选取纵向Dubins连接策略将2D路径映射至三维空间，获得固定翼飞行器的最终低空轨迹并验证是否进入威胁区，若进入威胁区，则通过重复以上步骤重新进行路径规划。
可选的，节点类型包括起始节点、目标节点、扩展节点；所述起始节点属性包括起始点坐标、起始点三维坐标位置和起始点航向；目标节点属性包括目标点坐标、目标点三维坐标位置和目标点航向；所述规划空间为固定翼飞机低空运动的三维空间；所述三维空间包括雷达威胁区和山脉威胁区。
可选的，基于起始节点与目标节点分别初始化随机生成树。
可选的，获得可行路径点集的过程包括：
根据随机生成树当前节点的地形复杂度选择生成树扩展策略，基于生成树扩展策略在所述规划空间内对两棵随机生成树进行协同扩展，并在探索过程中基于节点碰撞障碍区的次数动态调整探索步长，直至两棵随机生成树之间的距离满足阈值要求，获得可行路径点集；其中，所述生成树扩展策略包括交替连接策略、人工势场法探索策略与随机探索策略。
可选的，所述交替连接策略包括：两棵随机生成树交替探索扩展，交替扩展过程中，将当前树的最新扩展节点作为另一棵树的参考扩展方向，通过地形复杂度函数动态计算交替频率，当两棵随机生成树的最近节点间距满足阈值要求且路径无障碍时，直接连接两树并终止扩展。
可选的，所述人工势场法探索策略包括：为目标节点建立引力势场函数，为障碍物建立斥力势场函数，引入距离矫正因子消除目标点附近的斥力震荡；计算当前节点所受的引力向量、斥力向量及合成向量，归一化后确定扩展方向；其中，斥力势场的作用范围限定为障碍物安全阈值距离内。

本发明还提供了一种计算机装置，包括：存储器、处理器以存储在存储器上并可在处理器上运行的计算机程序，所述处理器执行所述计算机程序以实现上述方法的步骤。
与现有技术相比，本发明具有如下优点和技术效果：
本发明对RRT算法进行了改进，融合了人工势场法、交替连接和自适应探索步长三种策略，能够高效生成规避山脉和雷达等威胁区的路径点集，显著加速了低空突防轨迹的规划过程。进一步引入的贪心算法有效精简了路径点集，减少了节点数量。
在获得精简路径点集的基础上，针对低空突防任务，细分为低、中、高三种飞行高度场景。固定翼无人机根据不同场景和自身运动学约束，选取相应的机动策略。通过三维Dubins路径算法连接路径点，最终生成一条光滑且满足运动学约束的突防轨迹。该轨迹可包含爬升、俯冲、盘旋上升/下降、左/右转弯等典型机动动作，确保无人机安全抵达目标点。
本发明不仅显著提升了RRT算法的效率(减少冗余节点、加速探索)，而且针对低空突防中光滑机动轨迹的生成，提出了基于高度进一步细分的机动策略选择方法，对固定翼无人机低空威胁规避的轨迹设计领域具有重要参考价值。

实施例一
如图1-2所示，本实施例中提供一种基于改进的快速扩展随机树的固定翼飞行器的低空轨迹规划方法，能够有效避开雷达、山脉等威胁区，同时满足飞机的动力学约束，包括以下步骤：
步骤1：定义节点类型及规划空间。节点包含起始节点、目标点。规划空间指固定翼飞机低空运动的三维空间，低空是指飞行高度不大于2000m的飞行高度。
步骤2：初始化两棵随机生成树，第一棵树从起始节点开始扩展，第二棵树从目标节点开始扩展。
步骤3：根据当前节点的地形复杂度来选取生成树扩展策略，共分为1.交替连接策略，2.人工势场法探索策略，3.随机探索策略。在探索过程中根据节点碰撞障碍区次数自适应调整探索步长。
步骤4：判断两棵树之间的距离是否小于给定阈值，如果满足，则迭代结束；如果不满足，则返回步骤3进一步进行迭代。
步骤5：减少得到的可行路点集合中路点数量。通过贪心优化算法得到可行的路点集，新得到的路点集中包含更少的可行路点。
步骤6：将离散路点集连接起来。在水平平面用6种2D-Dubins曲线连接。
步骤7：将2D-Dubins曲线映射到3D空间中，即为2D-Dubins曲线添加高度信息。在添加高度信息的过程中，进一步将低空场景细分为低/中/高三种不同高度场景，依据最大爬升率选择不同的纵向Dubins连接策略。
步骤8：判断得到的光滑曲线是否进入威胁区，如果进入威胁区，则重新进行规划并返回步骤1。

在获得精简路径点集的基础上，针对低空突防任务，细分为低、中、高三种飞行高度场景。固定翼无人机根据不同场景和自身运动学约束，选取相应的机动策略。通过三维Dubins路径算法连接路径点，最终生成一条光滑且满足运动学约束的突防轨迹。该轨迹可包含爬升、俯冲、盘旋上升/下降、左/右转弯等典型机动动作，确保无人机安全抵达目标点。
本发明不仅显著提升了RRT算法的效率(减少冗余节点、加速探索)，而且针对低空突防中光滑机动轨迹的生成，提出了基于高度进一步细分的机动策略选择方法，对固定翼无人机低空威胁规避的轨迹设计领域具有重要参考价值。
