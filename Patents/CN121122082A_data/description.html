Description

一种低空飞行器监视方法、装置、设备及存储介质
技术领域
本发明涉及低空交通管理技术领域，特别涉及一种低空飞行器监视方法、装置、设备及存储介质。
背景技术
随着低空飞行器的广泛应用，特别是在多运营商、多飞行器、多任务的环境下，有效的低空交通管理系统的需求日益增加。
现有的航空监视系统主要有ADS-B系统、雷达和光学监控系统。其中，ADS-B系统用于高空飞行的飞行状态播报，ADS-B播报的飞行数据源自飞行器机载传感器，在卫星导航信号受阻时ADS-B播报的飞行器位置精度退化，利用ADS-B对低空飞行器进行监视时需要飞行器配合，并且，现有的ADS-B系统不能用于探测、监视非合作飞行器；雷达和光学监控系统，在监视低空的、飞行速度慢的、小型化的飞行器时仍可能失去目标，同时不能提供飞行器的身份信息；同时低空的环境复杂，例如城市建筑物遮挡、多种干扰源，都导致低空空域监视的可靠性不佳，影响低空飞行活动的快速发展和多样化的发展。
发明内容
本发明为解决上述技术问题，提供一种低空飞行器监视方法、装置、设备及存储介质。
为实现上述目的，本发明提供一种基于Ku波段探测雷达和ADS-B融合的低空飞行器监视方法，包括：
获取飞行器对应的ADS-B报文信号，所述ADS-B报文信号包括飞行器的身份信息、位置与速度信息；
获取飞行器对应的Ku波段雷达探测信号，所述Ku波段雷达探测信号包括Ku波段雷达探测到飞行器的位置与速度信息，处理获得Ku波段雷达探测到飞行器的位置与速度信息；
采用多速率交互式多模型粒子滤波算法对所述Ku波段探测到飞行器的位置与速度信息和所述ADS-B报文信号中的位置与速度信息进行匹配融合，
当所述Ku波段探测到飞行器的位置与速度信息和所述ADS-B报文信号中的位置与速度信息匹配，输出融合后的飞行器的位置与速度信息；
当所述Ku波段探测到飞行器的位置与速度信息和所述ADS-B报文信号中的位置与速度信息不匹配，则利用所述Ku波段探测雷达信号输出飞行器的位置与速度信息并输出飞行器异常信号。
可选的，利用高斯混合概率假设密度滤波器处理Ku波段雷达探测信息，获取飞行器的位置与速度信息，包括：
获取Ku波段探测雷达信号；
将获取的所述Ku波段雷达探测信号中的有关飞行器的位置与速度信息进行预处理，获得初始Ku波段雷达探测到飞行器的位置与速度信息以及目标点；
将所述初始Ku波段雷达探测到飞行器的位置与速度信息利用高斯混合概率假设密度滤波器进行处理，最终获得Ku波段雷达探测到飞行器的位置与速度信息。

所述数据传输处理系统包括数据融合处理模块，所述数据传输处理系统接收飞行器对应的ADS-B报文对应的飞行器的位置与速度信息和Ku波段雷达探测到飞行器的位置与速度信息，利用数据融合处理模块采用多速率交互式多模型粒子滤波算法对Ku波段探测到飞行器的位置与速度信息和ADS-B报文信号中的位置与速度信息进行匹配融合，当Ku波段探测到飞行器的位置与速度信息和ADS-B报文信号中的位置与速度信息匹配，输出融合后的飞行器的位置与速度信息并输出飞行器正常信号；当Ku波段探测到飞行器的位置与速度信息和ADS-B报文信号中的位置与速度信息不匹配，则利用Ku波段探测雷达信号输出飞行器的位置与速度信息并输出飞行器异常信号。

综上所述，本发明的优点及有益效果为：
本发明提供一种低空飞行器监视方法、系统、设备及存储介质。低空飞行器监视方法，采用ADS-B和Ku波段探测雷达共同工作，利用ADS-B获取飞行器对应的身份信息、位置与速度信息，利用Ku波段探测雷达获取飞行器对应的位置与速度信息，两种不同的信息来源形成互补，提升监视系统的可靠性；将Ku波段探测雷达和ADS-B获取的位置与速度信息采用多速率交互式多模型粒子滤波算法进行融合，将不同来源的低空飞行器的位置与速度信息进行处理，实现对空域中飞行器状态更精准的估计，显著提高了监测数据的可靠性和实用性，避免了现有技术中，仅采用Ku波段探测雷达或者ADS-B对飞行器进行监控，导致信息来源单一，监视的可靠性不足的问题，同时实现对监视区域的多信息余度监测，进一步提升对飞行器监视的可靠性。

具体的，本发明采用ADS-B和Ku波段探测雷达共同工作，利用ADS-B获取飞行器对应的身份信息、位置与速度信息，利用Ku波段探测雷达获取飞行器对应的位置与速度信息，两种不同的信息来源形成互补，提升监视系统的可靠性；将Ku波段探测雷达和ADS-B获取的位置与速度信息采用多速率交互式多模型粒子滤波算法进行融合，将不同来源的低空飞行器的位置与速度信息进行处理，实现对空域中飞行器状态更精准的估计，显著提高了监测数据的可靠性和实用性。
执行步骤S10，获取飞行器对应的ADS-B报文信号，所述ADS-B报文信号包括飞行器的身份信息、位置与速度信息。
在本发明实施例中，所述飞行器为无人机。
在其他实施例中，所述飞行器为载人飞机、直升飞机或者其他适合的飞行设备。
在本发明实施例中，ADS-B为机载广播式自动相关监视技术(AutomaticDependent Surveillance-Broadcast，ADS-B)，为一种开放式广播技术。
在本发明实施例中，通过ADS-B接收器接收飞行器广播的ADS-B报文，通过ADS-B内部的处理器对接收的所述ADS-B报文进行信号解调、数据解码获得所述ADS-B报文信号，通过ADS-B OUT发射器输出解码后的所述ADS-B报文信号。
通过接收广播发来的飞行器对应的ADS-B报文信号，获取飞行器的位置与速度信息，其中，所述飞行器的位置包括飞行器当前的经度、纬度、高度其中的一种或几种。
在本发明实施例中，所述ADS-B报文信号还包括飞行器姿态信息、航线信息、天气信息等其中的一种或多种。
执行步骤S20，获取飞行器对应的Ku波段雷达探测信号，所述Ku波段雷达探测信号包括Ku波段雷达探测到飞行器的位置与速度信息，处理获得Ku波段雷达探测到飞行器的位置与速度信息。
在本发明实施例中，所述Ku波段雷达探测信号的获取基于电磁波的反射原理，通过发送短脉冲信号、接收目标回波信号、分析回波信号的频率变化获取Ku波段雷达探测信号并实现对监视区域的探测、监视。

步骤S211，获取Ku波段探测雷达信号。
在本发明实施例中，通过Ku波段探测雷达设备获取所述Ku波段雷达探测信号。
步骤S212，将获取的所述Ku波段雷达探测信号中的有关飞行器的位置与速度信息进行预处理，获得初始Ku波段雷达探测到飞行器的位置与速度信息以及目标点。
在本发明实施例中，对所述Ku波段雷达探测信号中的有关飞行器的位置与速度信息进行的预处理包括：数字下变频、脉冲压缩（Pulse Compression，PC）、动目标监测（MTD）、恒虚警监测（CFAR）、聚类（Clustering）其中的一种或几种。

交互式多模型算法复杂度较低，在模型有效的情况下能够高效准确地融合信息产生更精确地飞行器状态估计。
在本发明实施例中，利用多速率交互式多模型粒子滤波算法中的协方差矩阵判断信息匹配效果，当该矩阵的迹大于指定阈值时，视作ADS-B报文信号与Ku波段雷达探测信息不匹配，否则视作匹配。
本发明还提供一种基于Ku波段探测雷达和ADS-B融合的低空飞行器监视系统，包括：
飞行监视信息收集系统、数据传输处理系统，所述飞行监视信息收集系统、数据传输处理系统相连；
所述飞行监视信息收集系统包括ADS-B地面接收模块和Ku波段雷达探测模块，利用ADS-B地面接收模块获取飞行器对应的ADS-B报文信号，所述ADS-B报文信号包括飞行器的身份信息、位置与速度信息；利用Ku波段雷达探测模块获取飞行器对应的Ku波段雷达探测信号，所述Ku波段雷达探测信号包括Ku波段雷达探测到飞行器的位置与速度信息；
所述数据传输处理系统包括数据融合处理模块，所述数据传输处理系统接收飞行器对应的ADS-B报文对应的飞行器的位置与速度信息和Ku波段雷达探测到飞行器的位置与速度信息，利用数据融合处理模块采用多速率交互式多模型粒子滤波算法对Ku波段探测雷达信号和ADS-B报文信号中的位置与速度信息进行匹配融合，当Ku波段探测雷达信号和ADS-B报文信号匹配，输出融合后的飞行器的位置与速度信息并输出飞行器正常信号；当Ku波段探测雷达信号和ADS-B报文信号不匹配，则利用Ku波段探测雷达信号输出飞行器的位置与速度信息并输出飞行器异常信号。
在本发明实施例中，所述飞行监视信息收集系统通过ADS-B地面接收模块和Ku波段雷达探测模块收集监视信息，并向所述数据传输处理系统输出所述监视信息，所述监视信息包括ADS-B报文信号、Ku波段雷达探测信号。
通过采用所述ADS-B地面接收模块和所述Ku波段雷达探测模块共同工作，实现对监视区域的多信息余度监视，且两种不同的信息来源形成互补，提升监视系统的可靠性。
在本发明实施例中，所述ADS-B地面接收模块包括：ADS-B接收器、ADS-B内部的处理器、ADS-B OUT发射器，所述ADS-B接收器、所述ADS-B内部的处理器和所述ADS-B OUT发射器依次相连；
通过所述ADS-B接收器接收飞行器广播的ADS-B报文，通过ADS-B内部的处理器对接收的所述ADS-B报文进行信号解调、数据解码获得所述ADS-B报文信号，通过所述ADS-BOUT发射器将解码后的所述ADS-B报文信号发送到所述数据传输处理系统。
通过接收广播发来的飞行器对应的ADS-B报文信号，获取飞行器的位置与速度信息，其中，所述飞行器的位置信息包括飞行器当前的经度、纬度、高度。
在本发明实施例中，所述ADS-B报文信号还包括飞行器姿态信息、航线信息、天气信息等其中的一种或多种。
在本发明实施例中，所述Ku波段雷达探测模块包括所述Ku波段探测雷达设备，所述Ku波段探测雷达设备基于电磁波的反射原理，通过发送短脉冲信号、接收目标回波信号、分析回波信号的频率变化获取所述Ku波段探测雷达信号，进而实现对监视区域的飞行器的探测、监视。
在本发明实施例中，所述数据传输处理系统还包括多功能接入设备、外部数据接口、内部数据接口；
所述内部数据接口的一端与所述飞行监视信息收集系统相连，另一端与所述数据融合处理模块的一端相连，用于接收所述飞行监视信息收集系统输出的所述监视信息，并向所述数据融合处理模块输出所述监视信息；
所述数据融合处理模块的另一端与所述多功能接入设备的一端、所述内部数据接口的一端相连，所述数据融合处理模块接收所述飞行监视信息收集系统输出的所述监视信息并将所述监视信息进行匹配融合，并输出对应的融合结果，至所述多功能接入设备和所述内部数据接口，其中，所述融合结果包括飞行器的位置与速度信息、飞行器正常/异常信号；
所述多功能接入设备的另一端用于输出融合结果；所述内部数据接口的另一端用于输出除所述融合结果以外的其他监视信息。
本发明通过所述内部数据接口、所述外部数据接口和所述多功能接入设备为所述飞行监视信息收集系统与所述数据融合处理模块之间提供高速的数据链接；通过所述数据融合处理模块将Ku波段探测雷达和ADS-B获取的位置与速度信息采用多速率交互式多模型粒子滤波算法进行融合，将不同来源的低空飞行器的位置与速度信息进行处理，实现对空域中飞行器状态更精准的估计，显著提高了监测数据的可靠性和实用性。
