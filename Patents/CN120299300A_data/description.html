Description

低空飞行器运行轨迹管理方法及系统
技术领域
本发明涉及通用航空技术领域，尤其涉及一种低空飞行器运行轨迹管理方法及系统。
背景技术
目前通航领域中的低空飞行器指的是能够在地面以下至一定高度范围内（通常是几米到几千米）进行飞行的航空器，包括轻型运动飞机、无人机等，可应用于农业、物流、监测、救援等领域。
低空飞行器的运行轨迹可通过全球定位系统GPS、惯性导航系统、视觉导航、地面控制站等技术获取，然而，若低空飞行器的应用场景比较复杂，目标运行轨迹复杂多变，以目标运行轨迹为标准对低空飞行器的实时运行轨迹进行管控必然会存在一定误差，难以确保低空飞行器的可靠飞行和安全保障。
发明内容
本发明提供一种低空飞行器运行轨迹管理方法及系统，有效保障低空飞行器运行轨迹的正确性并提升运行控制精度。
本发明提供一种低空飞行器运行轨迹管理方法，包括：
获取低空飞行器在预设场景的目标运行轨迹；
基于即时定位与地图构建技术以及所述目标运行轨迹的形状特性，将所述目标运行轨迹转换为具有规则特征的轨迹地图，规则特征基于形状特性确定；
基于预设的检测精度需求，对所述轨迹地图进行采样，得到至少一目标轨迹点；
获取实时运行轨迹点，基于所述目标运行轨迹的形状特性对应的指标对比所述目标轨迹点和所述实时轨迹点，判断所述低空飞行器的实时运行轨迹是否偏移所述目标运行轨迹，得到运行轨迹检测结果。
作为一个实施例，所述基于即时定位与地图构建技术以及所述目标运行轨迹的形状特性，将所述目标运行轨迹转换为具有规则特征的轨迹地图，包括：
基于所述目标运行轨迹的形状特性，确定所述目标运行轨迹对应的运行轨迹单元以及所述运行轨迹单元之间的排列关系；
基于所述即时定位与地图构建技术对所述运行轨迹单元进行缩放并按照所述排列关系生成所述轨迹地图。
作为一个实施例，所述目标运行轨迹的形状特性对应的指标包括轨迹半径和轨迹位移，对应的，所述获取实时运行轨迹点，基于所述目标运行轨迹的形状特性对应的指标对比所述目标轨迹点和所述实时轨迹点，包括：
获取实时轨迹点，根据所述实时轨迹点，确定实时轨迹半径和实时轨迹位移，根据所述目标轨迹点，确定目标轨迹半径和目标轨迹位移；
分别对比目标轨迹半径与实时轨迹半径以及目标轨迹位移与实时轨迹位移。
作为一个实施例，所述分别对比目标轨迹半径与实时轨迹半径以及目标轨迹位移与实时轨迹位移，包括：
判断目标轨迹半径与实时轨迹半径的差值是否满足第一阈值；
若目标轨迹半径与实时轨迹半径的差值不满足第一阈值，判定所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹；
若目标轨迹半径与实时轨迹半径的差值满足第一阈值，依次判断实时轨迹位移的位移方向和位移值是否与目标轨迹位移的位移方向和位移值一致，若位移方向和位移值均一致，判定所述低空飞行器的实时运行轨迹未偏移所述目标运行轨迹，否则，判定所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹。
作为一个实施例，还包括：
若所述运行轨迹检测结果表征所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹，生成轨迹修正指令并将所述轨迹修正指令发送至所述低空飞行器。
作为一个实施例，若所述运行轨迹检测结果表征所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹，则生成轨迹修正指令，包括：
若所述运行轨迹检测结果表征所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹，提升所述检测精度需求；
基于提升后的检测精度需求，分别对所述目标运行轨迹和所述实时运行轨迹进行采样和对比，确定导致轨迹偏移的异常轨迹点；
根据所述异常轨迹点，生成轨迹修正指令。

图1是本发明提供的低空飞行器运行轨迹管理方法的流程示意图之一，如图1所示，本发明提供一种低空飞行器运行轨迹管理方法，包括如下步骤。
步骤S100，获取低空飞行器在预设场景的目标运行轨迹。预设场景指的是低空飞行器的实际运行轨迹与目标运行轨迹的吻合度要求比较高的场景，比如飞行表演、应急救援等。
步骤S200，基于即时定位与地图构建技术以及所述目标运行轨迹的形状特性，将所述目标运行轨迹转换为具有规则特征的轨迹地图，规则基于形状特性确定。以预设场景为飞行表演为例进行说明，低空飞行器用于多机开花、俯冲、翻滚等特技表演，其目标运行轨迹具有不规则性和多变性等特点，通过对目标运行轨迹进行拆分和量化，转换为具有规则特征的轨迹地图。
步骤S300，基于预设的检测精度需求，对所述轨迹地图进行采样，得到至少一目标轨迹点。检测精度需求高，对轨迹地图采样的频率越高，得到的目标轨迹点越多，采样周期越短。
步骤S400，获取实时运行轨迹点，基于所述目标运行轨迹的形状特性对应的指标对比所述目标轨迹点和所述实时轨迹点，判断所述低空飞行器的实时运行轨迹是否偏移所述目标运行轨迹，得到运行轨迹检测结果。
可以理解的是，本发明通过即时定位与地图构建技术以及目标运行轨迹的形状特性，将目标运行轨迹转换为具有规则特征的轨迹地图，再从轨迹地图上采集目标轨迹点与低空飞行器的实时运行轨迹进行对比，能够避免因目标运行轨迹复杂多变的形状特性导致的误差，提高了运行轨迹管控精度和准确性，保障低空飞行器运行轨迹的正确性并提升运行控制精度。
在上述实施例的基础上，作为一个可选的实施例，所述基于即时定位与地图构建技术以及所述目标运行轨迹的形状特性，将所述目标运行轨迹转换为具有规则特征的轨迹地图，包括步骤S210-步骤S220。
步骤S210，基于所述目标运行轨迹的形状特性，确定所述目标运行轨迹对应的运行轨迹单元以及所述运行轨迹单元之间的排列关系。
通过对目标运行轨迹的研究，将目标运行轨迹中出现数量最多至少一个规则形状作为运行轨迹单元，比如用于多机开花、俯冲、翻滚等特技表演的低空飞行器的目标运行轨迹，具体包含弧形、圆形、八字形等规则形状。
步骤S220，基于所述即时定位与地图构建技术对所述运行轨迹单元进行缩放并按照所述排列关系生成所述轨迹地图。
如图2所示，本发明实施例以低空飞行器的目标运行轨迹为八字形为例，将目标运行轨迹拆分为两个圆形的运行轨迹单元，两个圆形左右排列，圆心在同一条水平线上，以左侧圆形的一点作为轨迹起始点和轨迹结束点，圆形周边的箭头为轨迹运行方向。
如图3a-图3c所示，圆形运行轨迹单元的关键参数包括参考圆心O、轨迹半径R、轨迹角度θ和轨迹位移S，在其他实施例中，圆形运行轨迹单元的轨迹半径大小、圆心位置、轨迹角度大小以及轨迹距离长短等参数可根据实际需求进行设置。
即时定位与地图构建技术（Simultaneous Localization And Mapping，SLAM）是一种允许机器人或其他自主体在未知环境中同时构建地图和定位自身的技术。
本发明实施例可调用SLAM技术的API接口，利用SLAM技术将运行轨迹单元进行缩放。设定轨迹地图的缩放比为k，则k＜1，即图3b所示轨迹半径为实际运行轨迹半径的k倍，轨迹角度θ因不受k的影响，缩放前后均保持一致，轨迹位移S则结合R和θ共同计算得出，如图2所示，运行轨迹沿顺时针方向开始，设定顺时针方向的参数为正值，逆时针方向则为负值，即与运行轨迹方向相同的参数为正值，与运行轨迹方向相反的参数为负值。
理想情况下，低空飞行器应按照目标运行轨迹即图2中的两个圆周轨迹进行匀速飞行，为确保低空飞行器运行轨迹的正确性，在固定周期内定时从轨迹地图中采样得到目标轨迹点，每隔一个固定时间进行一次采样。
设C圆为单个圆形运行轨迹的实际周长，也即完成一个圆周全程飞行的轨迹位移，其计算公式如下：
C圆=2πR/k （1）
其中，k为轨迹地图的缩放比，R为轨迹地图的圆周半径，与低空飞行器目标运行轨迹半径相差1/k倍。
轨迹角度的计算公式如下：
θ=360°/n （2）
其中，n为采样次数，将圆形运行轨迹平分为n等份。
轨迹位移的计算公式如下：
S=θ·πR/180 （3）
其中，S为轨迹地图上两个相邻目标轨迹点之间的轨迹位移，与低空飞行器的目标轨迹位移相差k倍。
综合上式可知，得到如下公式：
S目标=2πR/n （4）
S实际=1/k·2πR/n （5）
其中，S目标为轨迹地图上低空飞行器的目标轨迹位移，S实际为低空飞行器在理想情况下的实际轨迹位移。
如图4所示，将圆周运行轨迹平分为n等份，从轨迹起始点开始，目标轨迹位移依次为S0、S1、S2、S3……Sn，取前4个目标轨迹点S0-S3为应用实例，理论上，S0=S1=S2=S3=S目标=2πR/n。
当低空飞行器沿目标轨迹匀速飞行时，即以固定的半径、恒定的速度进行飞行，则相邻两段轨迹位移的差值应完全相等，基于此判定，可依据轨迹半径R、轨迹位移差值△S两大指标来制定对低空飞行器的管控方法，即通过判定R和△S的变化情况来进一步推算低空飞行器实际运行轨迹与目标运行轨迹之间的符合度。
可以理解的是，本发明根据所述目标运行轨迹的形状特性，将所述目标运行轨迹拆分为若干运行轨迹单元，降低目标运行轨迹的复杂度，提升低空飞行器运行轨迹的检测和异常判定的准确性。
在上述实施例的基础上，作为一个可选的实施例，所述目标运行轨迹的形状特性对应的指标包括轨迹半径和轨迹位移，对应的，所述获取实时运行轨迹点，基于所述目标运行轨迹的形状特性对应的指标对比所述目标轨迹点和所述实时轨迹点，包括步骤S410-步骤S420。
步骤S410，获取实时轨迹点，根据所述实时轨迹点，确定实时轨迹半径和实时轨迹位移，根据所述目标轨迹点，确定目标轨迹半径和目标轨迹位移。
步骤S420，分别对比目标轨迹半径与实时轨迹半径以及目标轨迹位移与实时轨迹位移。
可选的，所述分别对比目标轨迹半径与实时轨迹半径以及目标轨迹位移与实时轨迹位移，包括如下步骤：
步骤S421，判断目标轨迹半径与实时轨迹半径的差值是否满足第一阈值；
步骤S422，若目标轨迹半径与实时轨迹半径的差值不满足第一阈值，判定所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹；
步骤S423，若目标轨迹半径与实时轨迹半径的差值满足第一阈值，依次判断实时轨迹位移的位移方向和位移值是否与目标轨迹位移的位移方向和位移值一致，若位移方向和位移值均一致，判定所述低空飞行器的实时运行轨迹未偏移所述目标运行轨迹，否则，判定所述低空飞行器的实时运行轨迹偏移所述目标运行轨迹。
在步骤S410中，实时轨迹点可通过全球定位系统GPS、惯性导航系统、视觉导航、地面控制站等技术获取，不再赘述。
如图5所示，假设将圆形运行轨迹划分为8份，为方便查看，图中只展示了上半圆周，分别对应（S0-S3）区域的4段目标轨迹位移。以S0段轨迹位移为例，A区域所对应的实线轨迹为预设轨迹，通过镜像可得出，B区域所对应的虚线轨迹与实线轨迹相同，即低空飞行器以恒定速度沿虚线轨迹飞行，其对应的实时轨迹位移与目标轨迹位移相等，而且所对应的轨迹角度也一致，由此可知，若只关注轨迹位移的大小，则肯定会存在误判风险。同理，同一圆周的其他区域段也存在同样的误判隐患。实际管控过程则依据采样精度的要求进行采样点的划分与设定，即采样精度要求越高，则对应的采样区域越多，同理采样点也会越多，如此一来对运行轨迹的管控也会更精准。
基于上述情况，为进一步提升检测准确率，在步骤S420中，在判定轨迹位移的基础上，必须同步结合轨迹半径进行综合判定，进一步的将轨迹半径的判定应放在第一步，其次才是轨迹位移的检测，最后则结合前两步的检测结果判定实时运行轨迹是否偏移所述目标运行轨迹。
R0-R4为轨迹地图上的目标轨迹半径，因轨迹地图缩放比为k，低空飞行器在理想情况下的实际轨迹半径为R实际=R/k，将R0-R4代入上式，将实际轨迹半径与实时轨迹半径进行对比。若实际轨迹半径和实时轨迹半径的差值小于或等于第一阈值，表示低空飞行器当前并未偏离目标运行轨迹，若大于第一阈值，可将实际轨迹点记录为异常轨迹点，便于后续对实时运行轨迹进行实时修正与调控。

低空飞行器运行轨迹管理系统控制程序初始化后将进入运行轨迹检测环节，结合预设的目标运行轨迹及对应的指标参数，调用SLAM功能生成轨迹地图。
进入轨迹检测环节，首先进行轨迹半径检测，对比分析每个采样周期的实际轨迹半径（由目标轨迹半径放大得到）与实时轨迹半径的差值，若差值满足检测要求，则正常进入下一流程，不满足则结合检测结果及时修正运行轨迹，同时记录检测数据与检测结果（含正常、异常两种情况），为运行轨迹的可追溯性和轨迹修复奠定基础，进一步提升低空飞行器的运行可靠性与安全性。
轨迹位移检测包含位移方向判定和位移大小判定两大方面，首先基于目标运行轨迹及运行方向判定轨迹位移方向的正确性，判定结果正常则执行流向位移大小判定流程，异常则结合检测数据及时修正运行方向。
位移大小判定流程指的是，根据运行轨迹的等份划分及采样周期设定，逐一进行分段检测，当检测结果正常时，则判定实时运行轨迹满足目标运行轨迹的运行要求，反之则结合管控精度要求对异常段做进一步的检测周期细分，结合检测原理和控制流程对当前区域细分检测段所对应的位移进行逐一检测，同时对异常点位进行实时修正，有效保障低空飞行器运行轨迹的正确性并提升运行控制精度。
