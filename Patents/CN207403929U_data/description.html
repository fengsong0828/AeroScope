Description

一种低空飞行器降落用的减震底盘
技术领域
本实用新型涉及低空飞行器领域，具体为一种低空飞行器降落用的减震底盘。
背景技术
飞行器是任何由人类制造、能飞离地面、在空间飞行并由人来控制的提供给个人使用的飞行物。低空飞行是距地面100m到1000m之间的飞行，适用于训练、伞降、空投、侦察、强击和农林等作业。随着飞行器普及，个人飞行器必然是未来发展方向。轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向。
低空飞行根据其降落方式分为垂直降落和俯冲式降落，现有的垂直降落常采用支架组件进行支撑着地，这种结构避震性能差，对机身振动冲击大，容易损伤机身部件，俯冲式降落常采用行走轮配合起落杆进行降落，这种结构机头容易出现点头现象，严重时出现飞行器着地不稳翻滚，造成严重事故，而且对起落杆结构强度要求高，维护维修成本高。
实用新型内容
针对以上问题，本实用新型提供了一种低空飞行器降落用的减震底盘，结构简单，飞行器降落减震性能好，安全稳定性好，可以有效解决背景技术中的问题。
为实现上述目的，本实用新型提供如下技术方案：一种低空飞行器降落用的减震底盘，包括底板、承接板以及安装在底板和承接板之间的缓冲装置和防低头装置，底板和承接板均为轴对称板件结构，且上下平行布置安装，底板和承接板之间至少设置有三组缓冲装置，且缓冲装置呈轴对称结构布置安装，且对称轴与底板和承接板的对称轴共面平行，底板和承接板位于飞行器前进方向一体化设置有延伸板，防低头装置安装在底板和承接板上的两组延伸板之间，底板的下表面设置有行走轮。
作为本实用新型一种优选的技术方案，缓冲装置包括主连杆、副连杆和固定块，三组所述固定块呈一字型排列固定在承接板的下表面，且相邻两个固定块之间间距相同，相邻两组固定块之间固定设置有滑杆，滑杆上滑动套接有滑块，主连杆和副连杆交叉呈X型并且主连杆和副连杆的上端通过销轴活动连接滑杆，主连杆和副连杆的下端通过销轴活动连接在底板的上表面。
作为本实用新型一种优选的技术方案，滑块的两侧与固定块之间设置有蓄能弹簧，蓄能弹簧套接在滑杆上，主连杆的上端和副连杆的下端之间通过销轴活动连接设置有液压弹簧避震器，副连杆的上端和主连杆的下端之间也通过销轴活动连接设置有液压弹簧避震器。
作为本实用新型一种优选的技术方案，主连杆的中部设置有通槽，副连杆穿过通槽呈X型交叉与主连杆连接。
作为本实用新型一种优选的技术方案，防低头装置包括液压弹簧避震器和簧板，液压弹簧避震器上下两端通过销轴转动连接在上下两组延伸板之间，簧板呈半环状结构，且两端分别焊接连接在两下两组延伸板之间。
与现有技术相比，本实用新型的有益效果是：该种减震底盘在飞行器进行降落时，缓冲装置吸收降落时产生的冲击，保护机身不受损伤，而且稳定性好，防低头装置避免在进行俯冲式降落时机头点头现象，保持飞行器降落稳定性，均适用于两种降落方式，而且结构简单，易于维护维修。

实施例：
请参阅图1-5，本实用新型提供一种技术方案：一种低空飞行器降落用的减震底盘，包括底板1、承接板2以及安装在底板1和承接板2之间的缓冲装置3和防低头装置，底板1和承接板2均为轴对称板件结构，且上下平行布置安装，底板1和承接板2之间至少设置有三组缓冲装置3，且缓冲装置3呈轴对称结构布置安装，且对称轴与底板1和承接板2的对称轴共面平行，底板1和承接板2位于飞行器前进方向一体化设置有延伸板6，防低头装置安装在底板1和承接板2上的两组延伸板6之间，底板1的下表面设置有行走轮。本实施例中所述的减震底盘整体结构不局限于附图中的三角形结构，也可以是圆形，方形等轴对称结构，这样便于根据飞行器的形状结构设计相应的减震底盘，更好的提高飞行器降落时的稳定性，底板1上的延伸板6长度长于承接板2上的延伸板6这样增加飞行器着地时重心落入的面积，避免出现点头过度出现事故，稳定性高，防低头装置和缓冲装置3吸收振动冲击，保护机身。
本实施例中，缓冲装置3包括主连杆7、副连杆8和固定块10，三组所述固定块10呈一字型排列固定在承接板2的下表面，且相邻两个固定块10之间间距相同，相邻两组固定块10之间固定设置有滑杆9，滑杆9上滑动套接有滑块12，主连杆7和副连杆8交叉呈X型并且主连杆7和副连杆8的上端通过销轴活动连接滑杆9，主连杆7和副连杆8的下端通过销轴活动连接在底板1的上表面，滑块12的两侧与固定块10之间设置有蓄能弹簧11，蓄能弹簧11套接在滑杆9上，主连杆7的上端和副连杆8的下端之间通过销轴活动连接设置有液压弹簧避震器4，副连杆8的上端和主连杆7的下端之间也通过销轴活动连接设置有液压弹簧避震器4，主连杆7的中部设置有通槽71，副连杆8穿过通槽71呈X型交叉与主连杆7连接。多个缓冲装置3按照轴对称布置，有效吸收来自飞行器整体的冲击，吸收能量均匀，稳定好，而且缓冲装置3结构简单，便于维护维修。
本实施例中，防低头装置包括液压弹簧避震器4和簧板5，液压弹簧避震器4上下两端通过销轴转动连接在上下两组延伸板6之间，簧板5呈半环状结构，且两端分别焊接连接在两下两组延伸板6之间。在进行俯冲式降落时机头冲击较大，因此会出现点头现象，液压弹簧避震器4和簧板5配合吸收冲击能量，避免点头过度保证降落的安全性。
本实用新型的工作原理和优点：将该减震底盘安装在飞行器的下部，在进行垂直降落时，缓冲装置3受到飞行器的动能挤压，此时两组滑块12互相远离，滑块12两侧的蓄能弹簧11进行拉伸、收缩吸收能量，同时主连杆7和副连杆8交叉交增大，趋于弧形平行，两个液压弹簧避震器4进一步吸收能量，降低冲击，对称机身进行保护，不受冲击损伤，当进行俯冲式降落时，缓冲装置3进行上述吸能缓冲作用，同时机头在进行下压时，两组延伸板6之间的液压弹簧避震器4和簧板5吸收能量，并起到支撑作用，避免机头点头过度，保证安全，适用于两种降落方式，而且结构简单，易于维护维修。
