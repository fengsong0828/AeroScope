Description

一种城区低空无人飞行器战术冲突探测化解路径优化方法
技术领域
本发明涉及无人飞行器飞行安全技术领域，尤其涉及一种城区低空无人飞行器战术冲突探测化解路径优化方法。
背景技术
本部分提供的仅仅是与本公开相关的背景信息,其并不必然是现有技术。
近年来在全国多个大型城市相继获得低空飞行试点，使得我国城市交通往立体化方向发展迈出重要的一步，为城市空中交通(Urban Air Mobility,UAM)的发展提供了政策支持。由于城区空域的不断开放，无人飞行器(又称“无人机”)广泛应用于城市中的物流、航拍、交通检测等多类工作，其数量更是成指数倍增长。飞行器数量的不断增加势必导致城区内可用空域资源逐渐减少，飞行过程中难免发生飞行冲突，增大了空中碰撞事故发生的可能性，进而容易导致无人机坠毁，危及地面行人安全。
为了应对多架低空飞行器在同一有限空域内产生的飞行冲突，大量的科研学者开始着手研究飞行冲突探测与化解(Conflict Detection and Resolution,CD&R)技术以确保飞行器面临冲突威胁后可以自主实现冲突化解。通常是每两架无人机之间判断是否发生冲突，若发生冲突则采取冲突化解措施。无人机飞行冲突探测化解的方法，大致分为两种：“战略级的全局航迹规划”以及“战术级的局部冲突探测与解脱”，而CD&R通常指代后者，要求无人机可以在复杂多变环境中实时完成高效的冲突探测与化解。当无人机自主飞行不受操控时，必须给其赋予一个安全飞行策略，以防止无人机与各种固定和移动的障碍物发生碰撞。此外，在同一个有限的城市空域内存在大量无人机时，规避移动障碍物可能会导致飞行平稳性差，导致飞行方向上产生波动或振荡，即为“抖动”。需要提前考虑发生冲突甚至产生碰撞的风险程度，进而决定是否需要采取紧急的冲突化解措施。或可以在传统冲突化解方法基础上优化改进，得到更加平滑的安全飞行路径。同时尽可能在实现冲突化解的基础上缩短无人机飞行路径长度，减少空域资源的浪费。
因此，为确保无人机在复杂的城市环境中能够安全、稳定地飞行，研究出一种高效的飞行冲突探测化解算法显得尤为重要。目前针对无人机飞行的冲突探测化解的方法包括但不限于动态窗口法、人工势场法、速度障碍法和几何优化方法，尤其速度障碍法凭借着算法模型相对其他算法模型简单易于实现，得到学者的广泛应用。但在无人飞行器飞行过程中采用传统速度障碍法及其变体(互惠速度障碍法)会存在飞行抖动频发问题，且无法保证飞行冲突化解的飞行路径较短，没有考虑路径长度优化等问题，值得学者对其进一步优化。
发明内容
发明目的：针对现有技术的不足，提供一种基于CCR-GDO-RVO(ConsiderCollision Risk-Gradient Descent Optimization-Reciprocal Velocity Obstacle)的城区低空无人飞行器战术冲突探测化解路径优化方法。解决针对无人飞行器采取速度障碍法在飞行过程容易产生剧烈抖动且没有考虑路径长度优化等问题。实现城区低空空域无人机飞行冲突探测和化解。

步骤4：基于RVO算法通过进一步通过考虑碰撞风险的互惠速度障碍(ConsiderCollision Risk-Reciprocal Velocity Obstacle,CCR-RVO)冲突探测算法减小冲突发生区域，设置飞行安全保护区半径调节函数，调节保护区半径大小，根据保护调节后的保护区半径调整无人机飞行速度，调整无人机飞行冲突化解速度。
步骤4-1：根据两架无人机相对位置和速度来考虑两架无人机的碰撞风险，为无人机实时选择适宜的保护区半径。
根据保护区半径的实时变化来增大无人机每一时刻可选避障速度矢量范围。无人机之间的碰撞风险程度可以依据渐变色的碰撞锥区域评估，区域颜色随着无人机A与无人机B距离逐渐减小，碰撞锥区域的颜色逐渐由白到灰再变黑，此时对于两架无人机来说碰撞风险程度逐渐增高，发生碰撞的可能性也随之增大。
设置保护区半径变化比例系数k，且0<k≤1，通过缩小下一飞行时刻的传统碰撞锥中的保护区半径，进而缩小RCC区域，此时生成的RVO区域即为考虑碰撞风险的互惠速度障碍区域；从而增大无人机可供选择的速度区域。
同理如果相对距离较近，则很快会发生碰撞，则下一飞行时刻扩大保护区半径，进而扩大RCC，缩小无人机可供选择的速度区域；当相对距离减小接近到保护区半径时，保护区半径仍为保持不变。

采用上述技术方案所产生的有益效果在于：本发明提供的一种基于CCR-GDO-RVO的城区低空无人飞行器战术冲突探测化解算法，通过考虑每架无人机飞行速度方向及大小，判断是否存在飞行冲突，存在飞行冲突是否需要采取紧急避碰措施，对考虑碰撞风险后的速度进行二次优化减小每架无人机飞行路径长度，保障无人机更为安全平稳飞行，并提高飞行效率，占用更少空域资源。
本发明的原理是：
无人机搭载ADS-B接收机获取ADS-B报文，经报文解析可以获得无人机的飞行经纬度、高度和速度等信息。通过地心地固坐标转换和东-北-天坐标转换得到无人机相对地面固定参考点的三维坐标位置。构造无人机球形保护区，构建碰撞锥模型，通过位置和相对速度判断是否存在飞行冲突。通过碰撞锥模型和飞行速度得到互惠速度障碍区域，选择该区域外的速度可实现冲突化解。考虑碰撞风险程度，赋予一个调节函数对互惠速度障碍区域的无人机保护区半径进行实时调整，得到考虑碰撞风险的互惠速度障碍区域，选择该区域外的一个无人机接近最优的飞行速度作为冲突化解速度。利用梯度下降算法原理思路来优化上一步得到的冲突化解速度中的无人机飞行冲突化解速度。无飞行冲突时，无人机按照计算得到的最优飞行速度方向和大小飞向目标点，避免无人机偏离预定航线。

根据碰撞风险程度赋予调节函数来实时调整无人机保护区半径，扩大无人机飞行可选区域，进而选择更优飞行速度，减少由传统速度障碍法及其变体(互惠速度障碍法)导致的飞行抖动频发问题，且无法保证飞行冲突化解的飞行路径较短，没有考虑路径长度优化等问题。
根据梯度下降算法原理思路，对飞行速度进一步优化，在保证冲突化解的前提下，缩短无人机飞行路径，对无人机飞行路径进行优化，减少空域资源的浪费。
无人机无需冲突化解时，会根据当前位置和目标点位置重新规划最优飞行速度方向和大小飞向目标点，避免无人机偏离预定航线。
采用上述技术方案所产生的有益效果在于：本发明提供的一种城区低空无人飞行器战术冲突探测化解路径优化方法，通过考虑每架无人机飞行速度方向及大小，判断是否存在飞行冲突，存在飞行冲突是否需要采取紧急避碰措施，对考虑碰撞风险后的速度进行二次优化减小每架无人机飞行路径长度，保障无人机更为安全平稳飞行，并提高飞行效率，占用更少空域资源。

步骤5：进一步优化无人机飞行速度以减小无人机从起始点到目标点的飞行路径长度。
梯度下降算法(Gradient Descent Optimization,GDO)是一种用于优化目标函数的迭代方法，广泛应用于机器学习、深度学习和最优化问题中。其核心思想是通过不断调整参数，使得目标函数的值逐渐减小，从而找到局部或全局的最小值。
根据梯度下降算法(Gradient Descent Optimization,GDO)原理思路，在步骤4的基础上对互惠速度障碍法实现无人机冲突化解飞行路径进行优化。
在实际应用中，优化飞行路径长度自然重要，但优化路径长度和碰撞避免之间存在一定的权衡。过于追求路径长度的优化可能导致碰撞风险增大，而过于注重避免碰撞可能会导致路径长度变长。因此，在探测到冲突后，需要根据实际场景来调整梯度下降算法和RVO算法之间的权重，需要在优化的目标函数中引入一定的碰撞代价，以平衡两者之间的关系，从而整体目标函数应具有如下形式：
J(P)＝αL(P)+βC(P)；
其中，J(P)为整体优化目标函数，α和β分别为权重系数，用于平衡路径长度和碰撞代价，L(P)为当前时刻无人机飞行路径的总长度，C(P)为无人机的碰撞代价函数，P为无人机位置点，随时间和速度变化。
