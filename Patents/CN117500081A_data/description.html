Description

面向异质飞行器混合运行的低空智联网无线资源调度方法
技术领域
本发明属于低空智联网技术领域，具体涉及一种面向异质飞行器混合运行的低空智联网无线资源调度方法，以保证网络整体内的资源利用率与各子网络的负载均衡。
背景技术
目前我国低空空域尚未形成统一的航空通信网络，通用航空和无人机产业的快速发展亟需构建高效的低空通信、监测和信息服务体系，即低空智联网。同时，低空智联网的建设依托通信卫星网络、移动公共网络及航空专用网络等多种异构网络，各类网络差异较大，很容易出现网络资源分配不公平和各类网络负载差异显著等问题，造成网络资源利用率降低，网络整体性能下降。
发明内容
因此本发明提出低空智联网网络架构和网络中无线资源调度方法，实现复杂异构网络场景下的高效资源分配。
面向异质飞行器混合运行的低空智联网无线资源调度方法，包括以下步骤：
首先，构建基于SDN的低空智联网网络架构
SDN的低空智联网网络体系架构，包括业务层、控制层和资源层三层组成，控住层将卫星网络、航空专网和移动公网接入地面核心网，同时将业务管理功能下放到网络边缘，将异构网络的带宽资源抽象成统一的虚拟资源池，并细化网络节点的虚拟网络功能，从而实现低空智联网的高效管理；
其中，业务层为通用航空器及地面管控系统提供飞行控制、导航定位、感知与避撞和空中交通管理等应用服务。
资源层将低空智联网中通信卫星网络、航空专用网络、移动公共网络异构网络的资源抽象成虚拟资源池，方便统一管理与调度。
控制层由接入控制、资源管理和管理中心三部分组成。接入控制的功能是辅助用户业务决策选择合适的网络并控制接入，包括业务分类、选择决策和网络接入模块。资源管理负责对网络的无线资源进行管理与分配，主要包括资源监测、调度决策和资源调度模块。管理中心由应用管理、数据管理、故障管理和移动性管理组成。应用管理负责周期性收集、统计和分析应用数据；故障管理对网络故障进行统计分析，并生成故障处理策略，下发给故障管理节点；数据管理充当将各子网络的业务数据接入至地面核心网的网关，并负责所有业务数据的聚合、分析、处理、分发和存储；移动性管理的功能是负责用户和接入节点间的相对关系的鉴权、管理和维护，负责记录、更新和存储用户和接入节点位置信息。
