Description

一种低空飞行器交通枢纽及其运行方法
技术领域
本发明涉及一种低空飞行器交通枢纽及其运行方法。
背景技术
低空经济交通工具主要包括直升机、无人机和飞行汽车三大类：城市空中交通（UAM）是低空经济的主体产业，载人垂直起降器则是UAM 的主流形式。低空基建是发展低空经济的基石，未来人们出行不仅仅依赖于地铁车站、公交枢纽、机场等传统的城市交通枢纽，还可以依靠用于接纳低空载人垂直起降器的交通枢纽设施。目前专门适用于载人垂直起降器的城市交通枢纽站建筑则尚无应用。
随着低空经济与智能物流技术的快速发展，无人机配送凭借其灵活性强、时效性高、地形适应能力突出的优势，逐步成为城市及偏远地区物流体系的重要组成部分。然而，现有物流配送中心的设计理念仍主要围绕传统地面运输工具（如货车、叉车等）展开，其功能布局及智能化配套系统难以满足低空无人机规模化、高频次、全天候作业的需求。
发明内容
本发明的目的在于克服现有技术中存在的上述不足，而提供一种结构设计合理的低空飞行器交通枢纽及其运行方法，提供了专门适用于载人垂直起降器和货运无人机的城市交通枢纽，满足载人垂直起降器和货运无人机的作业需求。
本发明解决上述问题所采用的技术方案是：一种低空飞行器交通枢纽，包括客运中心和物流中心；所述的客运中心设置有降落区、外环滑行通道、降落停机区、乘客进站区、维保专用通道、起飞停机区、乘客候机区、起飞区和维保区；外环滑行通道分别与降落区、降落停机区、维保专用通道、起飞停机区和起飞区相连；降落停机区与乘客进站区相连；维保区与维保专用通道相连；起飞停机区与乘客候机区相连；所述的物流中心设置有作业区、降落平台和起飞平台，作业区设置有进货作业区、货物接收区、分拣加工区、货物出发区、充电区和出货作业区；降落平台与货物接收区相连；货物接收区与进货作业区相连，进货作业区设置有进货舱门、智能机械臂和货物传送装置，该货物传送装置与分拣加工区相连；货物出发区与出货作业区相连，出货作业区设置有出货舱门、智能机械臂和货物传送装置，该货物传送装置与分拣加工区相连；充电区与降落平台、货物接收区和货物出发区相连；货物出发区与起飞平台相连。
本发明所述的客运中心还设置有维保专用电梯，维保区通过维保专用电梯与维保专用通道相连。
本发明所述的降落区和起飞区分别设置于客运中心的两端；外环滑行通道设置于客运中心的最外围；所述的维保专用通道和维保专用电梯设置于降落停机区和起飞停机区之间。
本发明所述的客运中心为矩形平面布局或者圆形平面布局。
本发明所述的客运中心以单独建设，或者建设在已有建筑物或者传统交通站上。
本发明所述的降落平台和起飞平台分别设置于物流中心的两端；作业区位于物流中心的中心区域，其中货物接收区靠近降落平台，货物出发区靠近起飞平台，充电区则分布于作业区的外侧。
本发明所述的物流中心底部不带裙房时，在物流中心顶部设置货物出发区和货物接收区，两区域下设置进货作业区和出货作业区传送货物，进货作业区和出货作业区的货物传送装置均设置有竖直段和水平段，水平段位于竖直段的上方，当货物在底部被分拣加工完毕后，经由竖直段运送至水平段，再由水平段被分发至各个出货舱门的下方，最后由智能机械臂进行装货操作；而当货运无人机卸货时，货物经进货舱门进入进货作业区，经由水平段运送至竖直段，再经竖直段传送至底部的分拣加工区。
本发明所述的心物流中心的底部带有外扩的裙房时，在物流中心顶部设置货物出发区和货物接收区，两区域下设置进货作业区和出货作业区传送货物，进货作业区和出货作业区的货物传送装置均设置有竖直段，分拣加工区设置于外扩的裙房内；当货物在底部被分拣加工完毕后，经由竖直段被分发至各个出货舱门的下方，最后由智能机械臂进行装货操作；而当货运无人机卸货时，货物经进货舱门进入进货作业区，经竖直段传送至底部的分拣加工区。
本发明所述的客运中心和物流中心采用模块化布置，单独建设或者多模块组合建设。
一种低空飞行器交通枢纽的运行方法，包括客运中心的运行方法和物流中心的运行方法；
（一）所述的客运中心的运行方法包括以下步骤：
a)首先载人垂直起降器在降落区降落；
b)然后通过外环滑行通道滑行至降落停机区；
c)乘客下机后进入乘客进站区后离去；
d)载人垂直起降器由降落停机区回到外环滑行通道；若载人垂直起降器有维保充电需求，则滑行至维保专用通道，然后进入维保区；若载人垂直起降器无维保充电需求，则在外环滑行通道上继续向前滑行至起飞停机区等待乘客上机；
e)维保充电完成后的载人垂直起降器进入维保专用通道，然后进入外环滑行通道，再滑行至起飞停机区等待乘客上机；
f)乘客由乘客候机区登机；
g)载人垂直起降器由起飞停机区回到外环滑行通道，并滑行至起飞区离去；
h)至此，载人垂直起降器的降落、乘客换乘、维保、起飞离去的过程完成。
（二）所述的物流中心的运行方法包括以下步骤：
a)货运无人机临近物流中心，向其发送进场信号；
b)物流中心接收信号，同步获取货运无人机的状态数据，分配货运无人机至相应的降落平台；
c)降落后的货运无人机如果携带货物需要接收，则物流中心分配其至货物接收区，并同时向货物接收区发出预备接收指令；
d)货运无人机行至货物接收区后，该货物接收区对货运无人机进行身份识别，解锁进货舱门，并由进货作业区的智能机械臂卸载货物至货物传送装置，空载的货运无人机向物流中心返回状态信息；
e) 卸载后的货物被进货作业区的货物传送装置运送至分拣加工区，待分拣加工完毕后，重新打包编码，其信息被输入物流中心中；
f)物流中心同时可接受由传统方式投送的物流包裹，物流包裹在分拣加工区分拣加工完毕后，重新打包编码，然后与上述步骤e）中的货物统一受物流中心监控和调配；
g)物流中心实时监控各分拣加工区的货物存储情况，将货物通过货物传送装置运送至出货作业区，等待货运无人机装货向外配送；
h)对于空机降落、卸载货物后的无人机，物流中心实时监控其状态信息；再根据步骤g）中的待配送货物的状态信息，物流中心分配能满足续航要求的货运无人机至相应的货物出发区；
i)对于电量不足的货运无人机，物流中心分配其运行至充电区进行充电；
j)货运无人机运行至充电区后完成充电，向物流中心反馈状态信息，返回至上述步骤h）流程，等待分配任务指令；
k)电量充足的货运无人机接收到物流中心的任务指令后，行至货物出发区，该货物出发区对货运无人机进行身份识别后，解锁出货舱门，出货作业区的智能机械臂向货运无人机装载货物，满载的货运无人机向物流中心返回信息等待起飞指令；
l)物流中心为满载的货运无人机分配起飞平台，货运无人机就位，向物流中心反馈状态，等待起飞指令；
m)物流中心向货运无人机下达起飞指令，货运无人机起飞。

本发明与现有技术相比，具有以下优点和效果：
1)本发明客运中心能够实现低空载人垂直起降器的起降、乘客换乘和维修保养，能够实现人机分流、单向交通流以提高交通效率；
2)本发明客运中心适用于多种场景：可置于已有建筑物的屋面上；可设置于公路、轨道等上方，不影响下部交通；可与传统枢纽站建筑结合在一起建造，实现乘客无缝衔接。
3)本发明物流中心能够深度融合低空无人机的作业特性与智能物流系统的管控需求，实现空域资源动态优化、多设备协同调度、全流程自动化作业，从而为低空物流经济规模化发展提供核心基础设施支撑，未来推广应用价值极高。

具体实施方式
下面结合附图并通过实施例对本发明作进一步的详细说明，以下实施例是对本发明的解释而本发明并不局限于以下实施例。
本实施例的低空飞行器交通枢纽，包括客运中心A和物流中心B。
一、客运中心A设置有降落区A1、外环滑行通道A2、降落停机区A3、乘客进站区A4、维保专用通道A5、维保专用电梯A6、起飞停机区A7、乘客候机区A8、起飞区A9和维保区A13。
外环滑行通道A2分别与降落区A1、降落停机区A3、维保专用通道A5、起飞停机区A7和起飞区A9相连；降落停机区A3与乘客进站区A4相连；维保区A13通过维保专用电梯A6与维保专用通道A5相连；起飞停机区A7与乘客候机区A8相连。
降落区A1和起飞区A9应分别设置于客运中心的两端，以实现单向交通流向。外环滑行通道A2设置于客运中心的最外围，并且为单向设计。乘客进站区A4和乘客候机区A8为单向流，分别仅供乘客下机后离去和等候登机。维保专用通道A5和维保专用电梯A6，设置于降落停机区A3和起飞停机区A7之间，以保证有维保需求时，载人垂直起降器A16内无乘客；同时，二者均为往返单向设计，以提高交通流效率。
本发明实施例客运中心A的运行方法，包括以下步骤：
a)首先载人垂直起降器A16在降落区A1降落；
b)然后通过外环滑行通道A2滑行至降落停机区A3；
c)乘客下机后进入乘客进站区A4后离去；
d)载人垂直起降器A16由降落停机区A3回到外环滑行通道A2；若载人垂直起降器A16有维保充电需求，则滑行至维保专用通道A5，由维保专用电梯A6将其运送至维保区A13；若载人垂直起降器A16无维保充电需求，则在外环滑行通道A2上继续向前滑行至起飞停机区A7等待乘客上机；
e)维保充电完成后的载人垂直起降器A16，通过维保专用电梯A6运送至维保专用通道A5，然后进入外环滑行通道A2，再滑行至起飞停机区A7等待乘客上机；
f)乘客由乘客候机区A8登机；
g)载人垂直起降器A16由起飞停机区A7回到外环滑行通道A2，并滑行至起飞区A9离去；
h)至此，载人垂直起降器A16的降落、乘客换乘、维保、起飞离去的过程完成。

二、物流中心B设置有作业区、降落平台B3和起飞平台B12，作业区设置有进货作业区B2、货物接收区B5、分拣加工区B8、货物出发区B9、充电区B10和出货作业区B14。货运无人机B1在平面的降落、卸货、充电、装货以及起飞路线为纵向单向流线设计，不可逆行，以使得交通效率最大化。
降落平台B3与货物接收区B5通过无人机运行通道B16相连；货物接收区B5与进货作业区B2相连，进货作业区B2设置有进货舱门B6、智能机械臂B7和货物传送装置B13，该货物传送装置B13与分拣加工区B8相连；货物出发区B9与出货作业区B14相连，出货作业区B14设置有出货舱门B11、智能机械臂B7和货物传送装置B13，该货物传送装置B13与分拣加工区B8相连；充电区B10与降落平台B3、货物接收区B5和货物出发区B9通过无人机运行通道B16相连；货物出发区B9与起飞平台B12通过无人机运行通道B16相连。
物流中心B设置一个控制系统，能够对无人机和物流货物的状态分别进行实时监控，并根据它们的状态信息，智能地调度货物在不同分区间的传送、无人机的降落起飞、智能开关舱门、装卸载货物、无人机的充电和临时停靠等。
降落平台B3和起飞平台B12分别设置于物流中心B两端。作业区位于物流中心B中心区域，其中货物接收区B5靠近降落平台B3、货物出发区B9靠近起飞平台B12。充电区B10则分布于作业区的外侧，由无人机运行通道B16所分隔。这种布置的优点是，无人机降落、起飞、卸货、充电停放、载货等流程之间相互独立，可以串联依次进行，也可按实际情况单独运行某个流程，使运行效率大为提高。
进货舱门B6和出货舱门B11均水平布置，在闭合状态下，与地面标高保持一致，使货运无人机B1能够在上面自由运行；智能机械臂B7位于舱门的正下方，且悬挂于物流中心的受力结构上，用于承担货物B4的装载与卸载工作。
本发明实施例物流中心B的运行方法，包括以下步骤：
a)货运无人机B1临近物流中心B，向其发送进场信号；
b)物流中心B接收信号，同步获取货运无人机B1的定位、电量、载货情况等状态数据，分配货运无人机B1至相应的降落平台B3；
c)根据上述步骤b），降落后的货运无人机B1如果携带货物B4需要接收，则物流中心B的控制系统分配其至货物接收区B5的停机位，并同时向货物接收区B5发出预备接收指令；
d)根据上述步骤c），货运无人机B1行至货物接收区B5后，该货物接收区B5对货运无人机B1进行身份识别，自动解锁进货舱门B6，并由进货作业区B2的智能机械臂B7卸载货物B4至货物传送装置B13，空载的货运无人机B1向物流中心B返回状态信息；
e)根据上述步骤d), 卸载后的货物B4被进货作业区B2的货物传送装置B13运送至物流中心B底部的分拣加工区B8，待分拣加工完毕后，重新打包编码，其信息被输入物流中心B的控制系统中；
f)物流中心B同时也可接受由传统方式投送的物流包裹，物流包裹在分拣加工区B8分拣加工完毕后，重新打包编码，然后与上述e）中所描述的货物B4统一受物流中心B监控和调配；
g)物流中心B实时监控各分拣加工区B8的货物B4存储情况，将货物B4通过货物传送装置B13运送至出货作业区B14，等待货运无人机B1装货向外配送；
h)根据上述步骤b）和d），对于空机降落、卸载货物后的无人机，物流中心B实时监控其储备电量等状态信息；再根据g）中的待配送货物B4的状态信息，物流中心B分配能满足续航要求的货运无人机B1至相应的货物出发区B9；
i)根据上述步骤h),对于电量不足的货运无人机B1，物流中心B分配其运行至充电区B10的停机位进行充电；
j)根据上述步骤i），货运无人机B1运行至充电区B10后完成充电，向物流中心B反馈状态信息，返回至上述h）流程，等待分配任务指令；
k)根据上述步骤h), 电量充足的货运无人机B1接收到物流中心B的任务指令后，行至货物出发区B9的停机位，该货物出发区B9对货运无人机B1进行身份识别后，自动解锁出货舱门B11，出货作业区B14的智能机械臂B7向货运无人机B1装载货物B4，满载的货运无人机B1向物流中心B返回信息等待起飞指令；同时出货舱门B11关闭，向物流中心B传送最新状态信息；
l)根据上述步骤k），物流中心B为满载的货运无人机B1分配起飞平台B12，货运无人机B1就位，向物流中心B反馈状态，等待起飞指令；
m)根据上述步骤l），物流中心B向货运无人机B1下达起飞指令，货运无人机B1起飞。至此，完成一个完整的工作循环。
本实施的物流中心B采用三维空间布局模式，即底部分拣加工作业+顶部货物接收、配送模式，通过垂直运输对货物进行竖向传送。该模式的优势在于：能够充分利用物流中心的竖向空间，节约用地；较高的无人机作业平台使得无人机的降落、起飞均在离地面较高的位置，一方面减少无人机对周围环境的干扰，另一方面能有效节约能源。具体的，上述所描述的物流中心B的竖向布局如图10和图11所示。图10为物流中心B的底部不带裙房的情形，在物流中心B顶部设置货物出发区B9和货物接收区B5，两区域下设置进货作业区B2和出货作业区B14传送货物，进货作业区B2和出货作业区B14的货物传送装置B13均设置有竖直段和水平段，水平段位于竖直段的上方，当货物B4在底部被分拣加工完毕后，经由竖直段运送至水平段，再由水平段被分发至各个出货舱门B11的下方，最后由智能机械臂B7进行装货操作；而当货运无人机B1卸货时，货物B4经进货舱门B6进入进货作业区B2，经由水平段运送至竖直段，再经竖直段传送至底部的分拣加工区B8。图11为物流中心B的底部带有外扩的裙房B15的情形，在物流中心B顶部设置货物出发区B9和货物接收区B5，两区域下设置进货作业区B2和出货作业区B14传送货物，进货作业区B2和出货作业区B14的货物传送装置B13均设置有竖直段，分拣加工区B8可置于外扩的裙房B15内，当货物B4在底部被分拣加工完毕后，经由竖直段被分发至各个出货舱门B11的下方，最后由智能机械臂B7进行装货操作；而当货运无人机B1卸货时，货物B4经进货舱门B6进入进货作业区B2，经竖直段传送至底部的分拣加工区B8。
本实施的物流中心B可为多个，进行模块化布置, 单独建设或者多模块组合建设，既可单独应用于支线或末端等较小型的物流配送中心，也可以多个模块组合，应用于大型的干线物流中心。参见图12和图13，充电区B10和作业区间隔布置，二者之间为无人机运行通道B16。物流中心B的底部区域，也可以设置外扩的裙房15的大底盘，和传统的物流业务形态相结合，满足多层次的物流需求；货物B4在底层分拣加工区B8完成后，可由货物传送装置B13传送至上部多个高度的无人机作业平台，该布置极大的提高了土地面积利用率，更高的起飞降落位置也节约了无人机能源消耗。
