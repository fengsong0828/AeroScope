Description

一种通信覆盖增强的低空飞行器路径规划方法
技术领域
本发明涉及通信技术领域以及路径规划技术领域，尤其涉及一种通信覆盖增强的低空飞行器路径规划方法。
背景技术
近年来，随着低空经济的快速发展，低空飞行器因其机动性、灵活性以及在多种场景中按需提供服务的能力而受到广泛关注。在无线通信系统的背景下，低空飞行器被视为提升覆盖范围、系统容量与通信可靠性的重要技术。当低空飞行器作为空中用户时，通常需要依托地面蜂窝网络以获得可靠的通信连接。与传统地面用户相比，低空飞行器通常在更高的空域中飞行，从而与地面基站之间形成以视距为主导的信道。一方面，这种特性使低空飞行器即使在远距离情况下也能够建立稳定链路，从而提升其连接机会；另一方面，强视距传播同时也导致低空飞行器更易受到大量非关联基站的同频干扰，给通信性能保障带来了严峻挑战。因此，在低空经济蓬勃发展的背景下，如何进行有效的干扰抑制已成为蜂窝网络中低空飞行器通信中的关键问题。
然而，当下的低空飞行器通信性能往往受限于固定天线结构所带来的空间自由度不足，从而限制了干扰抑制能力，导致低空飞行器的轨迹规划结果不理想。故在此提出一种通信覆盖增强的低空飞行器路径规划方法。该方法在低空飞行器上装备可移动天线阵列，通过允许天线在有限区域内灵活移动，提供额外的空间自由度和波束赋形的灵活性，从而实现更有效的干扰抑制，以在确保全程可靠通信性能的同时，最小化低空飞行器的任务完成时间。
发明内容
鉴于上述的分析，本发明旨在公开了一种通信覆盖增强的低空飞行器路径规划方法；解决低空飞行器轨迹规划受限于固定天线结构的问题。
本发明公开的一种通信覆盖增强的低空飞行器路径规划方法，包括：
对装备有可移动天线的低空飞行器与至少一个基站组成的通信系统进行建模；得到包含飞行器轨迹、天线位置、波束赋形向量和基站关联的联合状态模型；
建立模型的离散化优化问题；所述问题以信干噪比、飞行器最大速度、天线移动范围和速度、最小天线间距、基站关联为约束，以最小化飞行器从起点到终点的任务完成时间为优化目标；
采用选择性一致代价搜索算法在离散网格上迭代扩展节点，每次扩展时联合优化波束赋形、天线位置和基站关联，保留满足信干噪比且潜在任务时间较小的节点，至终点输出最优路径；
飞行器按最优路径飞行并实时调整天线位置和波束赋形，保持链路质量。
