Description

一种可拆卸低空飞行器透明件
技术领域
本发明属于低空飞行产品技术领域，具体涉及一种可拆卸低空飞行器透明件。
背景技术
现有的低空飞行器透明前挡方案基本采用结构，工艺复杂，容易出现缺胶漏胶现象，且清胶的难度大，整个节拍耗时较长；在恶劣的环境试验下，容易出现脱胶风险，对驾驶员、乘客存在较大的安全隐患，并且通过打胶方案形成的低空飞行器透明件是固定死的状态，后期拆解及维修也比较麻烦，因此有必要开发出一种新的低空飞行器透明件。
发明内容
本发明所要解决的技术问题是针对现有技术的现状，而提供一种安装简单，方便拆卸，应力小，大幅的降低生产工时成本，提升产品的生产效率及合格率的可拆卸低空飞行器透明件。
本发明解决上述技术问题所采用的技术方案为：一种可拆卸低空飞行器透明件，其特征在于，包括有透明件、密封条以及机身，所述的密封条位于透明件和机身之间并用于透明件和机身之间的安装密封，所述的透明件与机身之间具有可拆卸连接结构，透明件通过可拆卸连接结构与机身连接安装。
在上述的一种可拆卸低空飞行器透明件中，所述的透明件和机身之间的可拆卸连接结构为螺栓连接或螺钉连接。
作为其中一种方式，在上述的一种可拆卸低空飞行器透明件中，所述的透明件包括有本体，本体为PC材料制成，所述的本体内嵌有铜套，铜套的中间为固定孔，所述的固定孔同时穿过本体、密封条以及机身，可拆卸连接结构为螺栓和螺母，所述的螺栓穿过固定孔且一端部与螺母连接固定。
在上述的一种可拆卸低空飞行器透明件中，所述的透明件为双色注塑形成，其中第一色为PC，第二色为PUR，所述的本体外侧面上双色注塑形成有PUR介质体。这里通过PUR介质体可以来提高透明件的韧性及耐热性。
作为第二种方式，在上述的一种可拆卸低空飞行器透明件中，所述的透明件为三色注塑形成，透明件包括有本体，透明件的第一色为PC，第二色为PUR，第三色为PC/ABS，所述的本体外侧面上注塑由第二色形成的PUR介质体，其中PC/ABS上具有通孔，可拆卸连接结构为螺栓和螺母，所述的螺栓穿过通孔且一端部与螺母连接固定。
作为第三种方式，在上述的一种可拆卸低空飞行器透明件中，所述的透明件为三色注塑形成，透明件包括有本体，其中透明件的第一色为PC，第二色为PUR，第三色为PC/ABS，其中本体的外侧面通过膜内反应成型PUR介质体；在本体的背面注塑PC/ABS的嵌件块，可拆卸连接结构为螺钉和螺母，在嵌件块上具有盲孔，螺母位于盲孔处，螺钉穿在盲孔上与螺母固定。
作为第四种方式，在上述的一种可拆卸低空飞行器透明件中，所述的透明件为双色注塑形成，透明件包括有本体，其中透明件的第一色为PC，第二色为PUR，其中本体的外侧面通过膜内反应成型PUR介质体，可拆卸连接结构为螺栓和螺母，所述的第一色为PC上嵌有螺栓，螺母与螺栓固定。
在上述的一种可拆卸低空飞行器透明件中，密封条其中一部分设置在于透明件的底部，在透明件的最外侧遮蔽涂层，侧边间隙打密封胶。
与现有技术相比，本发明的优点在于采用密封圈螺栓安装结构，代替打胶结构，安装简单，应力小，大幅的降低生产工时成本，提升产品的生产效率及合格率，密封条结构也提高透明件与机身配合间隙及外观效果，也便于后期拆解及维修。

具体实施方式
以下是本发明的具体实施例并结合附图，对本发明的技术方案作进一步的描述，但本发明并不限于这些实施例。
图中，透明件100；PC101；PUR102；PC/ABS103；密封条200；机身300；本体400；铜套500；螺栓600；螺母700；螺钉800。
实施例1
如图1所示，本可拆卸低空飞行器透明件，包括有透明件100、密封条200以及机身300，密封条200位于透明件100和机身300之间并用于透明件100和机身300之间的安装密封，密封条200也提高透明件100与机身300配合间隙及外观效果，这里为了方便进行安装，透明件100与机身300之间具有可拆卸连接结构，透明件100通过可拆卸连接结构与机身300连接安装，这里采用可拆卸的连接结构，代替打胶结构，安装简单，应力小，大幅的降低生产工时成本，提升产品的生产效率及合格率，具体来说，在本实施例中，透明件100和机身300之间的可拆卸连接结构为螺栓连接，在本实施例中，透明件100包括有本体400，本体400为PC材料制成，为了方便进行安装，本体400内嵌有铜套500，铜套500的中间为固定孔，固定孔同时穿过本体400、密封条200以及机身300，可拆卸连接结构为螺栓600和螺母700，螺栓600穿过固定孔且一端部与螺母700连接固定，其中透明件100为双色注塑形成，其中第一色为PC101，第二色为PUR102，本体400外侧面上双色注塑形成有PUR介质体，这里通过PUR介质体可以来提高透明件100的韧性及耐热性，同时螺栓600与螺母700连接代替打胶结构，安装简单，应力小。
实施例2
如图2所示，本实施例大部分的结构与实施例1相同，主要区别在于透明件100为三色注塑形成，透明件100包括有本体400，透明件100的第一色为PC101，第二色为PUR102，第三色为PC/ABS103，所述的本体400外侧面上注塑由第二色形成的PUR介质体，其中PC/ABS103上具有通孔，可拆卸连接结构为螺栓600和螺母700，螺栓600穿过通孔且一端部与螺母700连接固定，这里通过PC/ABS103上的通孔实现安装，可以减少本体400的应力及银纹的风险；最后在螺栓600处进行刮腻子喷漆，达到外观要求。
实施例3
透明件100为三色注塑形成，透明件100包括有本体400，其中透明件100的第一色为PC101，第二色为PUR102，第三色为PC/ABS103，其中本体400的外侧面通过膜内反应成型PUR介质体；第三色为PC/ABS103位于第一色为PC101底部，在本体400的背面注塑PC/ABS103的嵌件块，透明件100和机身300之间的可拆卸连接结构为螺钉800和螺母700，在嵌件块上具有盲孔，螺母700位于盲孔处，螺钉800穿在盲孔上与螺母700固定，这里本实施例中将螺母700作为嵌件块，此方案可以避免打通孔引起应力集中，影响使用寿命，也可以减少透明件100边缘的黑边喷漆处理，同时也保证外观的效果。
实施例4
本实施例中，透明件100为双色注塑形成，透明件100包括有本体400，其中透明件100的第一色为PC101，第二色为PUR102，其中本体400的外侧面通过膜内反应成型PUR介质体，可拆卸连接结构为螺栓600和螺母700，第一色为PC101上嵌有螺栓600，螺母700与螺栓600固定，密封条200其中一部分设置在于透明件100的底部，在透明件100的最外侧遮蔽涂层，侧边间隙打密封胶，这种方案采用3层密封结构，密封性更可靠，且底部采用密封圈，更方便后续的拆卸及维修。
