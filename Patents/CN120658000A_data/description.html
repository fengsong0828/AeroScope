Description

一种具有散热结构的低空飞行器节能电机
技术领域
本发明涉及节能电机技术领域，尤其涉及一种具有散热结构的低空飞行器节能电机。
背景技术
随着电动垂直起降飞行器等低空飞行器的快速发展，电机作为其核心动力部件，面临着前所未有的性能要求。尤其是在高功率密度(如300W/cm2)工作环境下，传统风冷散热技术逐渐暴露出导热系数低(通常＜50W/m·K)、散热效率不足、温升过高等问题。铝制外壳在长时间高负荷运行下，温升可达80℃以上，这不仅影响电机效率，还可能引发磁钢退磁、绝缘失效、寿命缩短等严重安全隐患，为此，我们提出了一种具有散热结构的低空飞行器节能电机来解决上述问题。
发明内容
本发明的目的是为了解决现有技术中存在的缺点，而提出的一种具有散热结构的低空飞行器节能电机。
为了实现上述目的，本发明采用了如下技术方案：
一种具有散热结构的低空飞行器节能电机，包括保护壳，所述保护壳内固定有承载板，所述承载板的上端安装有电机本体，所述承载板的上端一周固定有固定杆，所述固定杆上活动连接有夹持机构，所述夹持机构的一端抵触在电机本体的一侧，所述承载板的下端固定有固定管，所述固定管内安装有吸热结构，所述吸热结构的下端设有散热风扇，且散热风扇固定在固定管内的侧壁上。
优选地，所述夹持机构包括贯穿设置在固定杆上的移动环，所述移动环的一侧固定有弧形夹持板，所述弧形夹持板的一侧固定有吸热块，所述吸热块的一端抵触在电机本体的一侧。
优选地，所述移动环的一侧贯穿设有螺柱，所述固定杆的一侧沿垂直方向等间距设有多个螺纹盲孔，所述螺柱的一端螺纹拧接在其中一个螺纹盲孔内，所述螺柱的一侧固定有连接块。
优选地，所述承载板上设有第一开口，所述电机本体的下端卡接在第一开口内，所述第一开口和固定管连通。
优选地，所述吸热结构包括固定在固定管内的垫板，所述垫板的上端扣接有第一过滤网，所述第一过滤网的上端扣接有吸热板，所述吸热板的上端扣接有第二过滤网。
优选地，所述垫板、第一过滤网和吸热板的两侧均设有插孔，所述第二过滤网的下端两侧均固定有插杆，同一侧的一个插杆贯穿同一侧的多个插孔。
优选地，所述保护壳的下端设有第三开口，所述第三开口内安装有第三过滤网。
优选地，所述电机本体上安装有输出轴，所述输出轴贯穿保护壳的侧壁并延伸至保护壳的上端，所述输出轴的上端固定有桨叶。
优选地，所述吸热块采用复合相变材料制成。
在本发明中：
1、预设夹持机构：根据电机本体尺寸，将四个夹持机构均匀设置在电机四周，通过调节螺柱使弧形夹持板紧密贴合电机外壳，吸热块接触到位；
2、开始运行：电机启动后，运行过程中产生的热量通过接触面迅速传递至吸热块与吸热板中；
3、相变吸热：吸热块内部的复合相变材料在达到设定温度后开始吸热相变，延缓温度上升速度，形成“热缓冲”效应；
4、热传导与强制对流：吸热板将底部热量传导至固定管内，散热风扇启动，将热量从第三开口排出，形成强制对流散热路径；
5、多层过滤防护：各层过滤网既保证了空气流通效率，又有效阻挡灰尘杂质，防止堵塞与污染。
本发明实现“被动加主动”双重散热模式：初期通过相变材料吸热降低热负荷，后期通过风扇强制散热排出热量，在高温或高负荷运行状态下，相变材料的“热缓冲”作用可降低电机瞬间温升，避免因过热导致的功率下降或保护性停机，整体系统设计显著提升了电机的热稳定性与运行效率，进而延长飞行器续航时间，达到节能目的。
本发明适用于无人机、垂直起降飞行器(eVTOL)、城市空中交通(UAM)等新型低空飞行平台，在高温、高湿、多尘等恶劣环境下仍能保持稳定运行，适用于消防、救援、巡检、物流等多种任务场景，可拓展应用于电动航空器动力系统、车载电机系统等高热管理需求领域。
本发明具有以下好处：
1、吸热块采用相变材料，能够在特定温度范围内大量吸热而不显著升温，实现“热能缓冲”，相比传统金属材料，具有更高的热容量与可控热响应能力；
2、上部夹持吸热、底部吸热板传导、侧部风扇强制对流，构建了“立体散热”系统，多点接触设计避免局部过热，提升整体热均匀性；
3、各层吸热结构通过插杆与插孔连接，便于拆卸与更换，过滤网设计兼顾散热与防护，延长系统使用寿命；
4、移动环与螺柱调节结构，可适配不同尺寸电机，提升通用性与适应性；
综上所述，本发明突破了传统风冷散热的技术瓶颈，实现了高效、稳定、安全的热管理方案，其在低空飞行器电机领域的应用，不仅解决了当前高功率密度下的散热难题，也为未来高性能电动推进系统的发展提供了坚实基础，能够在特定温度范围内大量吸热而不显著升温，实现“热能缓冲”，相比传统金属材料，具有更高的热容量与可控热响应能力。

具体实施方式
下面将结合本发明实施例中的附图，对本发明实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本发明一部分实施例，而不是全部的实施例。
参照图1-6，一种具有散热结构的低空飞行器节能电机，包括保护壳10，保护壳不仅起到防护作用，还为电机提供结构支撑与气流导向。承载板4采用高强度轻质合金材料(如航空铝材)，其上设有第一开口8，便于电机本体11的稳固嵌入，同时实现与下方吸热结构的热传导通路，这种上下贯通的设计，使得热量传递路径更短，热传导效率更高；
保护壳10内固定有承载板4，承载板4的上端安装有电机本体11，承载板4的上端一周固定有固定杆2，固定杆2上活动连接有夹持机构，该夹持机构采用模块化设计，支持快速安装与更换，适用于不同尺寸的电机本体11；
夹持机构的一端抵触在电机本体11的一侧，承载板4的下端固定有固定管5，固定管5内安装有吸热结构，吸热结构的下端设有散热风扇19，且散热风扇19固定在固定管5内的侧壁上，散热风扇19为高效直流无刷风扇，直流无刷风扇，具有以下特点：
低功耗、低噪音、长寿命；
智能调速控制：通过温度反馈模块实现风扇转速自动调节；
防尘防水设计：适用于多尘、潮湿环境；
适用于低空飞行器的复杂环境，第三开口23的位置设计在保护壳下部，形成底部进风、顶部排风的自然对流与强制对流相结合的散热路径，提升整体散热效率，结合飞行器飞行时的气流效应，第三开口23还可作为气动辅助散热口，进一步增强散热能力；
夹持机构包括贯穿设置在固定杆2上的移动环3，移动环3的一侧固定有弧形夹持板1，弧形夹持板1的一侧固定有吸热块12，吸热块12内部填充复合相变材料(如石蜡+金属泡沫复合材料)，具有高热容与良好的导热性能，当电机运行产生热量，吸热块迅速吸收热量并发生相变(如从固态变为液态)，实现“热缓冲”效应，延缓电机温度骤升，提升电机运行稳定性，相变材料的选择可根据电机工作温度区间定制，实现最佳热管理效果；
吸热块12的一端抵触在电机本体11的一侧，弧形夹持板1的曲率设计与电机外壳轮廓匹配，增强夹持稳定性与热接触面积；
移动环3的一侧贯穿设有螺柱7，固定杆2的一侧沿垂直方向等间距设有多个螺纹盲孔6，螺柱7的一端螺纹拧接在其中一个螺纹盲孔6内，螺柱7的一侧固定有连接块22，通过调节螺柱7与多个螺纹盲孔6的配合，可实现对电机外壳的精准贴合，确保吸热块12与电机表面之间无间隙接触，提高热传导效率；
承载板4上设有第一开口8，电机本体11的下端卡接在第一开口8内，第一开口8和固定管5连通，风扇控制电路中设有温度反馈控制模块，可以根据电机温度自动调节风扇转速，实现节能与降噪双重优化；
承载板与保护壳之间采用硅胶垫+螺栓复合固定方式，既保证结构强度，又具备一定的减震缓冲效果，防止飞行振动导致电机松动；
吸热结构包括固定在固定管5内的垫板13，垫板13的上端扣接有第一过滤网14，第一过滤网14的上端扣接有吸热板15，吸热板15的上端扣接有第二过滤网16，吸热板15采用高导热合金材料(铜铝复合板)，其表面可进行微通道设计，增加热传导面积，垫板13起到支撑与隔热作用，防止底部热量反向传导至飞行器结构，插杆17与插孔20的插接方式便于拆卸与更换，方便后期维护与清洁，多层过滤网设计不仅提高空气流通效率，还有效防止灰尘和杂质进入核心散热系统，延长设备使用寿命；
垫板13、第一过滤网14和吸热板15的两侧均设有插孔20，第二过滤网16的下端两侧均固定有插杆17，同一侧的一个插杆17贯穿同一侧的多个插孔20；第一过滤网14与第二过滤网16采用不锈钢网加活性炭复合结构；
第三过滤网21采用蜂窝状多孔金属网，并可选配静电除尘层或纳米光催化涂层，以吸附PM2.5颗粒、有害气体等污染物；
保护壳10的下端设有第三开口23，第三开口23内安装有第三过滤网21，过滤网采用多孔金属网或蜂窝结构滤材，兼具高通透性与高过滤效率，可选配静电除尘层或活性碳吸附层，用于吸附飞行过程中吸入的细颗粒物或有害气体，定期清理或更换过滤网可维持系统长期高效运行，尤其适用于城市低空飞行环境，防止灰尘积累影响散热性能；
电机本体11上安装有输出轴，输出轴贯穿保护壳10的侧壁并延伸至保护壳10的上端，输出轴的上端固定有桨叶9，输出轴采用高精度轴承支撑，减少运行摩擦与热量产生，桨叶9采用碳纤维复合材料，具备高强度、轻量化与低风阻特性，提升飞行器整体效率，输出轴与桨叶的连接结构应具备动平衡设计，防止运行振动造成结构疲劳与热分布不均；
吸热块12采用复合相变材料制成，复合相变材料(石蜡+金属泡沫复合材料)，具备高热容与良好导热性，在电机启动初期迅速吸收热量，实现“热缓冲”效应，防止温度骤升：
具有以下好处：
高热容：在电机启动初期迅速吸收热量；
可控相变温度：可根据电机工作温度定制相变材料(如添加石墨烯提升导热系数)；
缓冲温升：防止电机因温度骤升导致性能下降；
模块化更换：便于后期维护与材料升级。
