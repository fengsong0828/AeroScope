Description

一种低空飞行器金属机壳钻孔攻丝一体设备
技术领域
本发明涉及飞行器外壳加工设备技术领域，具体为一种低空飞行器金属机壳钻孔攻丝一体设备。
背景技术
低空飞行器是指那些在较低高度进行飞行的航空器，低空飞行器可以用来监控作物健康、土壤状况、灌溉需求等，帮助农民提高产量并减少资源浪费；同时低空飞行器可用于监测森林火灾、海洋污染、野生动物保护等情况，为环境保护工作提供及时的数据支持，在现代社会中扮演着越来越重要的角色，飞行器的金属外壳为飞行器提供了必要的结构强度，确保其在飞行过程中能够承受各种力的作用，如气动压力、振动和冲击，在金属机壳加工的过程中需要使用设备在机壳上进行钻孔和攻丝，以满足后期的装配需求，现有用于低空飞行器金属机壳的钻孔攻丝设备基本可以满足日常的使用需求，但是设备的装夹结构简单，进行钻孔攻丝之前需要搭配人工进行装夹固定，同时受到机壳外形的限制，装夹的角度需要在现场进行调试，影响了设备的实用性；同时为了防止钻孔的碎屑在攻丝过程中对产品造成划伤，需要在两步工序之间进行碎屑的清理，增大了工作人员的劳动强度；且钻孔攻丝输出端上安装的钻头和丝锥需要在加工过程中进行更换，影响了设备的加工效率；因此设计一种低空飞行器金属机壳钻孔攻丝一体设备是很有必要的。
发明内容
本发明的目的在于提供一种低空飞行器金属机壳钻孔攻丝一体设备，以解决上述背景技术中提出的问题。
为实现上述目的，本发明提供如下技术方案：一种低空飞行器金属机壳钻孔攻丝一体设备，包括加工平台、定位机构、自适应装夹机构、底部支撑、清洁机构、位置调节机构和钻孔攻丝机构，所述加工平台的底部均匀设置有定位机构中的定位支架，定位支架中设置有定位气缸，定位气缸的一侧固定连接有自适应装夹机构中的旋转电机，旋转电机固定滑动块的底部，且旋转电机的输出端贯穿滑动块固定连接在下压气缸的底部，下压气缸的输出端固定连接有下压支架，下压支架中滑动连接有支撑球，支撑球的底端设置有顶压支架，加工平台的底部对称设置有清洁机构中的第一支架，第一支架上设置有第一气缸，第一气缸固定在第二支架的底部，第二支架上设置有第二气缸，第二气缸固定在连接支架的底部一侧上，连接支架上对称开设的凹槽中嵌入安装有第三气缸；加工平台的底部四角对称设置有位置调节机构中的升降电机，升降电机的输出端固定连接有升降丝杆，升降丝杆上设置有连接块，连接块对称固定在升降架上，升降架上滑动连接有滑移支架，且滑移支架上滑动连接有钻孔攻丝机构中的连接板。
作为本发明的进一步技术方案，所述定位机构由定位支架、定位气缸、侧气缸和定位柱组成，侧气缸固定在旋转电机的一侧上，侧气缸的输出端固定连接有定位柱。
作为本发明的进一步技术方案，所述自适应装夹机构由安装环、旋转电机、滑动块、下压气缸、下压支架、支撑球、顶压支架、适应气缸和顶推架组成，滑动块滑动连接在加工平台之间。
作为本发明的进一步技术方案，所述下压支架上设置有安装环，安装环上均匀开设的凹槽中固定套接有适应气缸适应气缸的输出端固定连接有顶推架。
作为本发明的进一步技术方案，所述清洁机构由第一支架、第一气缸、第二支架、第二气缸、连接支架、第三气缸、第三支架、传动电机和清洁轮组成，第二支架滑动连接在底部支撑上。
作为本发明的进一步技术方案，所述第三气缸的输出端固定连接有第三支架，且第三支架固定连接传动电机的输出端，传动电机嵌入安装在清洁轮两侧开设的凹槽中。
作为本发明的进一步技术方案，所述位置调节机构由升降电机、升降丝杆、连接块、升降架、滑移气缸、滑移支架、顶部气缸和升降气缸组成，连接块通过内部嵌入安装的滚珠螺母与升降丝杆相互配合连接。
作为本发明的进一步技术方案，所述升降架中对称安装的导轨上设置有滑移气缸，滑移气缸对称固定在滑移支架上，滑移支架中滑动连接有顶部气缸。
作为本发明的进一步技术方案，所述钻孔攻丝机构由连接板、连接台、转动架、输出电机、中心气缸、底部气缸、外部气缸和装夹气缸组成，连接板对称安装在滑移气缸的两侧，连接板的一侧固定连接有升降气缸，升降气缸的输出端固定在连接台上。
作为本发明的进一步技术方案，所述连接台的顶部中心处固定连接有输出电机，且输出电机的输出端贯穿转动架固定连接在转动架上，转动架顶部开设的凹槽中固定连接有中心气缸，转动架侧壁上均匀开设的凹槽底部固定连接有底部气缸，底部气缸的输出端固定连接有外部气缸，外部气缸的输出端固定连接有装夹气缸。
与现有技术相比，本发明的有益效果是：该一种低空飞行器金属机壳钻孔攻丝一体设备，利用设置的定位气缸带动旋转电机、滑动块和侧气缸沿着加工平台滑动，调节侧气缸及定位柱的位置，同时侧气缸带动定位柱上移，通过两侧均匀分布的定位柱对加工平台上放置的机壳进行初步定位，接着适应气缸通过顶推架调整顶压支架的朝向角度，配合下压气缸带动下压支架进行下压，将顶推架顶压在机壳上完成固定的过程，装夹固定的过程无需人工参与，同时便于调节顶压支架的下压角度，可以适配不同曲面结构的机壳加工过程，提高了设备的实用性；利用设置的第一气缸带动第二支架沿着第一支架滑动，同时第二气缸带动连接支架沿着第一支架滑动，来调整第三支架的水平位置，同时由第三气缸调节第三支架的高度位置，配合传动电机带动清洁轮的转动对机壳的表面进行清理，扫除钻孔过程中残留的碎屑，降低了工作人员的劳动强度；利用均匀分布在转动架上的外部气缸对装夹气缸进行支撑，装夹气缸用于固定不同规格的钻头和丝锥进行固定，进行钻孔过攻丝时连接对应规格的外部气缸带动装夹气缸向转动架中心移动，同时底部气缸带动外部气缸下移，钻头和丝锥移动至转动架中心处时，通过底部气缸带动外部气缸上移，接着中心气缸对钻头和丝锥进行固定，同时装夹气缸解除对钻头和丝锥的固定，随后底部气缸和外部气缸带动装夹气缸归位，归位后输出电机通过转动架带动钻头和丝锥转动配合位置调节机构调整加工位置对在机壳上进行钻孔或攻丝操作，更换刀具时通过底部气缸带动外部气缸调节装夹气缸的位置，对中心处的刀具进行装夹，同时中心气缸解除对刀具的控制，然后对应规格的外部气缸带动装夹气缸向转动架中心移动，同时底部气缸带动外部气缸下移，钻头和丝锥移动至转动架中心处时，通过底部气缸带动外部气缸上移，接着中心气缸对钻头和丝锥进行固定，同时装夹气缸解除对钻头和丝锥的固定，随后底部气缸和外部气缸带动装夹气缸归位，自动完成换刀过程，保障了设备的加工效率。

请参阅附图1-附图7，本发明提供的一种实施例：一种低空飞行器金属机壳钻孔攻丝一体设备，包括加工平台1、定位机构2、自适应装夹机构3、底部支撑4、清洁机构5、位置调节机构6和钻孔攻丝机构7，加工平台1的底部均匀设置有定位机构2中的定位支架21，定位支架21中设置有定位气缸22，定位气缸22的一侧固定连接有自适应装夹机构3中的旋转电机31，旋转电机31固定滑动块32的底部，且旋转电机31的输出端贯穿滑动块32固定连接在下压气缸34的底部，下压气缸34的输出端固定连接有下压支架35，下压支架35中滑动连接有支撑球36，支撑球36的底端设置有顶压支架37，加工平台1的底部对称设置有清洁机构5中的第一支架51，第一支架51上设置有第一气缸52，第一气缸52固定在第二支架53的底部，第二支架53上设置有第二气缸54，第二气缸54固定在连接支架55的底部一侧上，连接支架55上对称开设的凹槽中嵌入安装有第三气缸56；加工平台1的底部四角对称设置有位置调节机构6中的升降电机61，升降电机61的输出端固定连接有升降丝杆62，升降丝杆62上设置有连接块63，连接块63对称固定在升降架64上，升降架64上滑动连接有滑移支架66，且滑移支架66上滑动连接有钻孔攻丝机构7中的连接板71；定位机构2由定位支架21、定位气缸22、侧气缸23和定位柱24组成，侧气缸23固定在旋转电机31的一侧上，侧气缸23的输出端固定连接有定位柱24；自适应装夹机构3由安装环30、旋转电机31、滑动块32、下压气缸34、下压支架35、支撑球36、顶压支架37、适应气缸38和顶推架39组成，滑动块32滑动连接在加工平台1之间；下压支架35上设置有安装环30，安装环30上均匀开设的凹槽中固定套接有适应气缸38适应气缸38的输出端固定连接有顶推架39；清洁机构5由第一支架51、第一气缸52、第二支架53、第二气缸54、连接支架55、第三气缸56、第三支架57、传动电机58和清洁轮59组成，第二支架53滑动连接在底部支撑4上；第三气缸56的输出端固定连接有第三支架57，且第三支架57固定连接传动电机58的输出端，传动电机58嵌入安装在清洁轮59两侧开设的凹槽中；位置调节机构6由升降电机61、升降丝杆62、连接块63、升降架64、滑移气缸65、滑移支架66、顶部气缸67和升降气缸69组成，连接块63通过内部嵌入安装的滚珠螺母与升降丝杆62相互配合连接；升降架64中对称安装的导轨上设置有滑移气缸65，滑移气缸65对称固定在滑移支架66上，滑移支架66中滑动连接有顶部气缸67；钻孔攻丝机构7由连接板71、连接台72、转动架73、输出电机74、中心气缸75、底部气缸76、外部气缸77和装夹气缸78组成，连接板71对称安装在滑移气缸65的两侧，连接板71的一侧固定连接有升降气缸69，升降气缸69的输出端固定在连接台72上；连接台72的顶部中心处固定连接有输出电机74，且输出电机74的输出端贯穿转动架73固定连接在转动架73上，转动架73顶部开设的凹槽中固定连接有中心气缸75，转动架73侧壁上均匀开设的凹槽底部固定连接有底部气缸76，底部气缸76的输出端固定连接有外部气缸77，外部气缸77的输出端固定连接有装夹气缸78，底部气缸76用于调节装夹气缸78上非工作状态刀具的高度位置，避免对工作过程中转动的钻头和丝锥产生影响。
工作原理：使用时，利用设置的定位气缸22带动旋转电机31、滑动块32和侧气缸23沿着加工平台1滑动，调节侧气缸23及定位柱24的位置，同时侧气缸23带动定位柱24上移，通过两侧均匀分布的定位柱24对加工平台1上放置的机壳进行初步定位，接着适应气缸38通过顶推架39调整顶压支架37的朝向角度，配合下压气缸34带动下压支架35进行下压，将顶推架39顶压在机壳上完成固定的过程，装夹固定的过程无需人工参与，同时便于调节顶压支架37的下压角度，可以适配不同曲面结构的机壳加工过程，提高了设备的实用性；利用设置的第一气缸52带动第二支架53沿着第一支架51滑动，同时第二气缸54带动连接支架55沿着第一支架51滑动，来调整第三支架57的水平位置，同时由第三气缸56调节第三支架57的高度位置，配合传动电机58带动清洁轮59的转动对机壳的表面进行清理，扫除钻孔过程中残留的碎屑，降低了工作人员的劳动强度；利用均匀分布在转动架73上的外部气缸77对装夹气缸78进行支撑，装夹气缸78用于固定不同规格的钻头和丝锥进行固定，进行钻孔过攻丝时连接对应规格的外部气缸77带动装夹气缸78向转动架73中心移动，同时底部气缸76带动外部气缸77下移，钻头和丝锥移动至转动架73中心处时，通过底部气缸76带动外部气缸77上移，接着中心气缸75对钻头和丝锥进行固定，同时装夹气缸78解除对钻头和丝锥的固定，随后底部气缸76和外部气缸77带动装夹气缸78归位，归位后输出电机74通过转动架73带动钻头和丝锥转动配合位置调节机构6调整加工位置对在机壳上进行钻孔或攻丝操作，更换刀具时通过底部气缸76带动外部气缸77调节装夹气缸78的位置，对中心处的刀具进行装夹，同时中心气缸75解除对刀具的控制，然后对应规格的外部气缸77带动装夹气缸78向转动架73中心移动，同时底部气缸76带动外部气缸77下移，钻头和丝锥移动至转动架73中心处时，通过底部气缸76带动外部气缸77上移，接着中心气缸75对钻头和丝锥进行固定，同时装夹气缸78解除对钻头和丝锥的固定，随后底部气缸76和外部气缸77带动装夹气缸78归位，自动完成换刀过程，保障了设备的加工效率。
