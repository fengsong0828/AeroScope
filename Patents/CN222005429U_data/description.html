Description

一种低空飞行器防碰撞结构
技术领域
本实用新型涉及低空飞行器技术领域，具体为一种低空飞行器防碰撞结构。
背景技术
无人驾驶飞机简称“无人机”，是利用无线电遥控设备和自备的程序控制装置操纵的不载人飞机，或者由车载计算机完全地或间歇地自主地操作，而无人机一般分为无人固定翼飞机与无人多旋翼飞行器。
现有的，若无人机出现故障时，而无人机便会出现跌落的现象，无人机上的部件受到强烈撞击会损坏，若没有防跌落保护会带来严重的经济损失，因此大部分无人机都安装有防撞装置；但是，目前的无人机防撞装置只对无人机的螺旋桨进行保护，当无人机在空中飞行时受到天气、飞鸟以及气流等各种原因，会使得无人机在集群飞行时出现晃动，进而容易导致多组无人机发生碰撞，而发生碰撞的无人机便会向下掉落，而无人机落地时由于惯性会对机体造成损伤，让整个无人机内部的元件或者无人机的外壳进行摔坏，进而不便于对无人机进行保护。鉴于此，我们推出一种低空飞行器防碰撞结构。
实用新型内容
本实用新型的目的在于提供一种低空飞行器防碰撞结构，以解决上述背景技术中提出的问题。
为实现上述目的，本实用新型提供如下技术方案：一种低空飞行器防碰撞结构，包括：飞行器、第一凹型架和定位板；
所述飞行器的底部设置有L型连接架，L型连接架与飞行器之间为固定连接，所述L型连接架的数量为两组，两组所述L型连接架之间设置有监控器；
所述第一凹型架设置在L型连接架的底部，且第一凹型架位于监控器的底部，第一凹型架与L型连接架之间为焊接设置，而第一凹型架可以对监控器进行保护；
所述定位板设置在第一凹型架的底部，所述定位板的数量为两组，两组定位板与第一凹型架之间为焊接设置；
所述飞行器的侧边设置有防碰机构，通过所述防碰机构的第二凹型架在第三凹型架的表面滑动，以使第二凹型架挤压第一气囊形成对飞行器的保护。
优选的，所述防碰机构包括第三凹型架一侧连接的插接块，所述插接块插接在飞行器的侧边，飞行器的侧边开设有用于插接插接块的插接孔，所述第三凹型架与插接块固定连接，所述第二凹型架的内侧与第一气囊的一侧接触，第二凹型架与第一气囊之间为固定连接，所述第三凹型架的上下端连接有限位块，所述第二凹型架的内部开设有用于滑接限位块的限位槽，而第三凹型架表面的限位块会滑接在限位槽的内部，而限位槽的设置可以对第二凹型架的移动进行限位让第二凹型架始终处于水平状态的左右移动。
优选的，所述插接块的表面开设有螺孔，螺孔与插接块之间为一体成型设置，所述飞行器的表面连接有螺杆，且螺杆贯穿螺孔后延伸至飞行器的底部，而螺孔的内螺纹与螺杆的外螺纹相匹配旋合，进而可以让螺杆贯穿飞行器和插接块。
优选的，所述第一凹型架的底部连接有弹簧，弹簧与第一凹型架之间为固定连接，所述弹簧的底部连接有升降板，且升降板滑接在两组定位板内侧，所述升降板的底部连接有支撑柱，支撑柱与升降板之间为焊接设置，所述支撑柱的底部连接有防撞板，而防撞板的底部设置有防护垫，而防护垫也可对防撞板进行保护，所述第一凹型架与升降板之间连接有第二气囊，所述第二气囊的一侧连接有第一进气管，所述第二气囊的另一侧连接有第一出气管。
优选的，所述定位板的内部开设有用于滑接升降板的滑槽，滑槽与定位板之间为一体成型设置，而滑槽的设置可以对升降板的移动进行限位，让升降板始终处于竖直状态的升降运动。
优选的，所述飞行器的表面连接有螺旋桨，螺旋桨设置在飞行器的四角处。
优选的，所述第一气囊的一侧连接有第二进气管，所述第一气囊的另一侧连接有第二出气管，且第二进气管与第二出气管贯穿第三凹型架。
与现有技术相比，本实用新型的有益效果是：本实用新型通过飞行器最底部的防撞板会与地面接触，而这时防撞板会带动支撑柱和升降板向第一凹型架的底部进行移动，以使升降板会对弹簧造成挤压，而升降板也会对第二气囊进行挤压，让第二气囊内部的气体从第一出气管的内部跑出，从而也可让第二气囊对飞行器得到缓冲，从而可以对飞行器的底部进行保护，配合弹簧和第二气囊的共同使用，从而可以对飞行器的底部进行保护，避免垂直掉落对飞行器造成损坏；
飞行器会向一侧进行倾斜，而第二凹型架会与地面接触，第二凹型架会向飞行器的侧边移动，以使第二凹型架对第一气囊造成挤压，并且第三凹型架表面的限位块会滑接在限位槽的内部，而限位槽的设置可以对第二凹型架的移动进行限位让第二凹型架始终处于水平状态的左右移动，而第二凹型架、第一气囊和第三凹型架的共同配合使用，可以对飞行器的侧边进行保护，避免让飞行器发生倾斜时对飞行器造成损坏，进而可以有效的对飞行器进行保护。

请参阅图1-3，本实用新型提供一种技术方案：一种低空飞行器防碰撞结构，包括：飞行器1，所述飞行器1的底部设置有L型连接架6，所述L型连接架6的数量为两组，两组所述L型连接架6之间设置有监控器8；

所述防碰机构包括第三凹型架15一侧连接的插接块13，所述插接块13插接在飞行器1的侧边，所述第三凹型架15与插接块13固定连接，所述第二凹型架3的内侧与第一气囊4的一侧接触(并连接)，所述第三凹型架15内侧与第一气囊4的另一侧接触(并连接)，第三凹型架15的上下端连接有限位块17，所述第二凹型架3的内部开设有用于滑接限位块17的限位槽16。
所述插接块13的表面开设有螺孔14，螺孔14与插接块13之间为一体成型设置，所述飞行器1的表面连接有螺杆5，且螺杆5贯穿螺孔14后延伸至飞行器1的底部，而螺孔14的内螺纹与螺杆5的外螺纹相匹配旋合，进而可以让螺杆5贯穿飞行器1和插接块13。
所述第一凹型架7的底部连接有弹簧18，弹簧18与第一凹型架7之间为固定连接，所述弹簧18的底部连接有升降板19，且升降板19滑接在两组定位板10内侧，所述升降板19的底部连接有支撑柱12，支撑柱12与升降板19之间为焊接设置，所述支撑柱12的底部连接有防撞板11，而防撞板11的底部设置有防护垫，而防护垫也可对防撞板11进行保护，所述第一凹型架7与升降板19之间连接有第二气囊22，所述第二气囊22的一侧连接有第一进气管23，所述第二气囊22的另一侧连接有第一出气管24。

所述飞行器1的表面连接有螺旋桨2，螺旋桨2设置在飞行器1的四角处。
所述第一气囊4的一侧连接有第二进气管20，所述第一气囊4的另一侧连接有第二出气管21，且第二进气管20与第二出气管21贯穿第三凹型架15。
具体的，使用时，在飞行器1使用的过程中，如飞行器1出现电量不足时或者误碰的状态下，飞行器1便会向下掉落，而飞行器1出现垂直掉落时，飞行器1最底部的防撞板11会与地面接触，而这时防撞板11会带动支撑柱12和升降板19向第一凹型架7的底部进行移动，以使升降板19会对弹簧18造成挤压，让弹簧18压缩，而升降板19也会对第二气囊22进行挤压，让第二气囊22内部的气体从第一出气管24的内部跑出，从而也可让第二气囊22对飞行器1得到缓冲，从而可以对飞行器1的底部进行保护，配合弹簧18和第二气囊22的共同使用，可以避免垂直掉落对飞行器1造成损坏，而升降板19的侧边滑接在滑槽9的内部，以使滑槽9对升降板19的移动进行限位，让升降板19始终处于竖直状态的升降运动，而升降板19取消对第二气囊22进行挤压后，第二气囊22便会利用第一进气管23让外界的气体进入至第二气囊22的内部，让第二气囊22鼓起恢复至初始的状态；
而掉落后，飞行器1会向一侧进行倾斜，而第二凹型架3会与地面接触，第二凹型架3会向飞行器1的侧边移动，以使第二凹型架3对第一气囊4造成挤压，而第一气囊4压缩，第一气囊4内部的气体便会从第二出气管21中排出，并且第三凹型架15表面的限位块17会滑接在限位槽16的内部，而限位槽16的设置可以对第二凹型架3的移动进行限位让第二凹型架3始终处于水平状态的左右移动，而第二凹型架3、第一气囊4和第三凹型架15的共同配合使用，可以对飞行器1的侧边进行保护，避免让飞行器1发生倾斜时对飞行器1造成损坏，进而可以有效的对飞行器1进行保护；
而第二凹型架3取消对第一气囊4进行挤压后，第一气囊4便会利用第二进气管20将外界空气吸入至第一气囊4的内部，让第一气囊4恢复至初始的状态。
