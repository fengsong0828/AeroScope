Description

一种低空飞行器专用降落伞安全装置
技术领域
本发明涉及降落伞安全技术领域，尤其涉及一种低空飞行器专用降落伞安全装置。
背景技术
飞行器降落伞由伞布、伞绳、引导伞和伞布表面的导流孔等部件与结构组成，低空飞行器降落伞需要实现低高度下飞行器的安全着落，而传统的高空降落伞需要较大的高度与空间进行展开，因此传统的高空降落伞并不适用于低空飞行器。
在低空飞行器发生坠落等事故时，现有的飞行器降落伞通过气流带动其引导伞张开，引导伞拉动降落伞并使其展开，使伞布和伞绳逐渐被气流拉出，降落伞逐段被拉出并展开，这就导致飞行器的降落伞展开速度慢，并且降落伞在未展开时其伞绳堆叠摆放，而降落伞展开时容易受气流影响发生偏移和转动，从而导致堆叠的伞绳相互缠绕，使降落伞的展开速度受到影响，进而因为降落伞展开不及时而造成事故。
发明内容
为了克服飞行器的降落伞展开速度慢的缺点，本发明提供了一种低空飞行器专用降落伞安全装置。
为解决上述的技术问题，本发明采用以下技术方案：一种低空飞行器专用降落伞安全装置，包括有外壳，所述外壳内安装有高度感应器，所述外壳内固接有周向均匀分布的伞绳壳体，周向均匀分布的所述伞绳壳体共同通过伞绳固接有位于所述外壳内的降落伞，所述外壳上设置有对称分布的腔体，所述外壳的腔体内转动连接有转轴，所述转轴上固接有第一转动叶扇，所述转轴上固接并绕设有第一柔性绳，所述第一柔性绳穿过所述外壳的内壁，所述第一柔性绳绕过滑轮组并固接有重块，所述外壳内固接有对称分布的电磁块，所述外壳内的电磁块与相邻所述重块磁吸配合，所述重块与所述降落伞挤压配合，所述外壳上安装有用于调节对称分布所述第一转动叶扇转动状态的启动组件，所述外壳内固接有对称分布的限位架，所述限位架与相邻所述重块滑动配合。
更进一步的是，所述启动组件包括有对称分布的滑架，对称分布的所述滑架均滑动连接于所述外壳上远离所述伞绳壳体的一侧，所述滑架内滑动连接有电控门，所述电控门上固接有第一液压伸缩杆，所述第一液压伸缩杆的伸缩端固接有与相邻所述电控门滑动连接的滑动壳体，所述外壳固接有对称分布的第二液压伸缩杆，所述第一液压伸缩杆通过导管与相邻所述第二液压伸缩杆连通，对称分布的所述第二液压伸缩杆分别位于所述外壳上相邻的腔体处，所述外壳固接有对称分布且分别与其上相邻腔体封堵配合的矩形壳体，所述矩形壳体内滑动连接有与所述外壳上相邻腔体封堵配合的滑动挡板，所述滑动挡板与相邻所述第二液压伸缩杆的伸缩端固接，所述降落伞固接有位于对称分布的所述滑动壳体之间的引导伞。
更进一步的是，所述滑架靠近所述伞绳壳体的一侧固接有滑动架，所述滑动架与所述外壳滑动连接，对称分布的所述滑动架的对向侧均固接有挡风板，对称分布的所述滑动架之间固接有触发绳，所述滑动架与所述外壳之间固接有弹性件。
更进一步的是，所述外壳的底部固接有贯穿其的切割壳体，所述切割壳体的下侧固接并连通有集气罩，所述切割壳体内滑动连接有滑动切刀，所述滑动切刀与所述切割壳体之间固接有弹性件，所述触发绳穿过所述切割壳体，所述滑动切刀用于对所述触发绳进行切割。
更进一步的是，还包括有对称分布的第二转动叶扇，对称分布的所述第二转动叶扇均通过转动轴转动连接于所述外壳上，周向均匀分布的所述伞绳壳体上共同固接有固定盘，所述伞绳壳体的伞绳穿过所述固定盘，所述固定盘远离所述伞绳壳体的一侧转动连接有滑槽架，所述第二转动叶扇的转动轴与所述滑槽架之间依次通过锥齿轮组和直齿轮组传动，所述滑槽架内滑动连接有周向均匀分布的滑块，所述滑块固接有穿过所述滑槽架和所述固定盘的第二柔性绳，所述第二柔性绳远离相邻所述滑块的一端固接有转动环，所述第二柔性绳的转动环转动连接有环形切刀，所述环形切刀位于所述固定盘内，所述环形切刀用于对相邻所述伞绳壳体的伞绳进行切割。
更进一步的是，所述环形切刀上固接有传动齿轮，所述固定盘内靠近相邻所述传动齿轮的位置固接有齿条，所述传动齿轮与相邻所述齿条啮合。
更进一步的是，所述外壳固接有对称分布的导流壳体，所述第二转动叶扇位于相邻所述导流壳体内，所述导流壳体位于所述外壳上靠近相邻所述滑动架的一侧。
更进一步的是，对称分布的所述导流壳体的背向侧均转动连接有转动盖，所述外壳上固接有对称分布的第一气压伸缩杆，所述第一气压伸缩杆的伸缩端与相邻所述电控门挤压配合，所述导流壳体上固接有通过导管与相邻所述第一气压伸缩杆连通的第二气压伸缩杆，所述第二气压伸缩杆的伸缩端与相邻所述转动盖固接。
更进一步的是，还包括有固定壳体，所述固定壳体固接于所述外壳的上侧，所述固定壳体内滑动连接有滑动盘，所述滑动盘和所述固定壳体之间固接有弹性件，所述降落伞通过导流孔安装有收紧绳，所述滑动盘与所述收紧绳固接。
更进一步的是，所述外壳的上侧固接有U形切刀，所述U形切刀用于对所述收紧绳进行切割。
与现有技术相比，本发明具有如下优点：1、本发明通过重块夹紧并带动降落伞抛出，模拟人手抛出降落伞的动作，加快降落伞打开的速度，从而增加低空飞行器坠落时的缓冲力度，进而更好的保护飞行器与人体安全；
2、本发明通过气流改变第一转动叶扇的转速，使飞行器的坠落速度与重块抛出降落伞的速度成正比，使飞行器高速坠落时降落伞能够以更快的速度打开，进而更好的保护飞行器与人体安全；
3、本发明通过滑动切刀切断触发绳，使电控门能够自动打开，避免飞行器突然坠落导致驾驶人员手动打开不及时，从而导致低空飞行器坠落时降落伞不能及时打开，进而影响飞行器与驾驶人员的安全；
4、本发明通过快速割断伞绳壳体的伞绳，使飞行器与降落伞分离，当飞行器侧面来风时，降低飞行器在临近地面时的风阻，进而防止飞行器受气流影响发生倾斜甚至翻转。

具体实施方式
下面结合附图对本发明的技术方案作进一步说明。
现有的降落伞通过气流带动其引导伞进行展开，使得降落伞逐段被拉出并展开，这就导致飞行器的降落伞展开速度慢，并且降落伞在未展开时其伞绳堆叠摆放，若降落伞逐段展开会增加伞绳在拉出过程中相互缠绕的风险。
实施例1：一种低空飞行器专用降落伞安全装置，如图1-图3所示，包括有外壳1，外壳1安装有控制面板，其中控制面板未在图中展示，外壳1内安装有与控制面板电连接的高度感应器，外壳1的左侧通过螺栓与低空飞行器螺纹连接，外壳1内固接有周向均匀分布的若干个伞绳壳体101，周向均匀分布的伞绳壳体101共同通过伞绳固接有位于外壳1内的降落伞102，伞绳堆放在相邻伞绳壳体101内，当降落伞102展开时，降落伞102将伞绳壳体101内的伞绳抽出，外壳1上设置有上下对称分布的两个腔体，外壳1腔体的左侧设置有开口且为飞行器的迎风侧，外壳1的腔体内转动连接有转轴103，转轴103上固接有第一转动叶扇104，转轴103上固接并绕设有第一柔性绳105，第一柔性绳105穿过外壳1的内壁，第一柔性绳105绕过滑轮组106并固接有重块107，外壳1内固接有对称分布且均与控制面板电连接的电磁块，外壳1内的电磁块与相邻重块107磁吸配合，其中滑轮组106由两个定滑轮组成，其中一个定滑轮位于外壳1的内壁中，另一个定滑轮位于外壳1的右侧，重块107与降落伞102挤压配合，用于在第一柔性绳105带动重块107移动时，重块107沿着相邻限位架3的左侧斜面移动并逐渐对降落伞102进行挤压，外壳1上安装有用于调节对称分布第一转动叶扇104转动状态的启动组件2，外壳1内固接有上下对称分布的两组限位架3，每组限位架3分别由前后对称分布的两个限位架3组成，限位架3的左右两侧均设置有用于导向相邻重块107的斜面，限位架3与相邻重块107滑动配合，用于使重块107沿着相邻限位架3移动并挤压降落伞102。
如图4与图5所示，启动组件2包括有前后对称分布且均与控制面板电连接的电控门201，外壳1右侧的上部固接有前后对称分布的滑架2011，电控门201滑动连接于相邻滑架2011内，电控门201上固接有第一液压伸缩杆202，第一液压伸缩杆202与控制面板电连接，第一液压伸缩杆202内的液体为液压油，第一液压伸缩杆202的伸缩端固接有与相邻电控门201滑动连接的滑动壳体203，外壳1固接有上下对称分布的第二液压伸缩杆204，第二液压伸缩杆204内的液体同样为液压油，第一液压伸缩杆202通过导管与相邻第二液压伸缩杆204连通，对称分布的第二液压伸缩杆204分别位于外壳1上相邻的腔体处，外壳1固接有上下对称分布且分别与其上相邻腔体左侧的开口封堵配合的矩形壳体205，用于使左侧气流通过外壳1上相邻腔体开口的上板部分进入其内，从而带动第一转动叶扇104转动，矩形壳体205内滑动连接有与外壳1上相邻腔体封堵配合的滑动挡板206，滑动挡板206上下移动用于控制外壳1上相邻腔体开口的开启状态，以上侧的滑动挡板206为例，当上侧的滑动挡板206向上移动至极限位置时，外壳1上相邻腔体的开口封闭，滑动挡板206与相邻第二液压伸缩杆204的伸缩端固接，降落伞102固接有位于对称分布的滑动壳体203之间的引导伞207，滑动壳体203的开启状态用于控制引导伞207的抛出。
如图3-图6所示，滑架2011的左侧固接有滑动架401，滑动架401与外壳1滑动连接，前后对称分布的滑动架401的对向侧均固接有挡风板402，对称分布的两个滑动架401之间固接有触发绳403，滑动架401与外壳1之间固接有弹性件，其中弹性件为压簧且初始为压缩蓄力状态，挡风板402近似为C形，用于阻挡左侧的气流对弹性件的冲击，外壳1的底部固接有贯穿其的切割壳体404，切割壳体404的下侧固接并连通有集气罩405，集气罩405的孔径自上而下逐渐增大，用于收集下侧气流并进行加压，切割壳体404内滑动连接有用于切断触发绳403的滑动切刀406，滑动切刀406与切割壳体404之间固接有弹性件，其中弹性件为压簧，触发绳403穿过切割壳体404，滑动切刀406用于对触发绳403进行切割，当滑动切刀406受下侧气流挤压向上移动时，滑动切刀406将触发绳403切断。
当驾驶人员需要使用本装置对低空飞行器（下述中简称飞行器）进行保护时，驾驶人员通过螺栓将外壳1螺纹连接在飞行器后侧的上部，在使用过程中若飞行器发生故障导致坠落时，高度感应器首先监测到高度发生骤变并通过控制面板发出指令，使第一液压伸缩杆202的伸出端收缩并带动相邻滑动壳体203移动，滑动壳体203失去对引导伞207的阻挡，引导伞207受气流影响向飞行器坠落的相反方向运动，与此同时驾驶人员通过控制面板开启两个电控门201，使两个电控门201分别沿着相邻滑架2011背向移动，电控门201失去对降落伞102的阻挡，降落伞102受气流影响向飞行器坠落的相反方向运动，使伞绳壳体101内的伞绳逐渐被抽出，降落伞102逐渐打开，减缓飞行器的坠落速度。
在第一液压伸缩杆202的伸出端收缩的过程中，第一液压伸缩杆202通过导管将其内部液体挤压至相邻第二液压伸缩杆204内，使第二液压伸缩杆204的伸缩端收缩，第二液压伸缩杆204的伸缩端带动相邻滑动挡板206移动至相邻矩形壳体205内，外壳1的两个腔体不再封闭（外壳1的两个腔体初始时被相邻滑动挡板206和相邻矩形壳体205阻挡封闭），第一转动叶扇104与外部气流接触而发生转动，伴随着飞行器的坠落，第一转动叶扇104带动相邻转轴103转动，转轴103带动其上相邻的第一柔性绳105绕其转动，第一柔性绳105通过相邻滑轮组106带动相邻重块107移动，使得降落伞102在逐渐打开的过程中，上下两侧的重块107夹紧降落伞102并向电控门201的方向移动，重块107在移动过程中首先沿着相邻限位架3的左侧斜面移动，使两个重块107逐渐相向移动并夹紧降落伞102，然后重块107沿着相邻限位架3的右侧斜面移动，使两个重块107逐渐背向移动并松开降落伞102，模拟人手抛出降落伞102的动作，加快降落伞102打开的速度，从而增加低空飞行器坠落时的缓冲力度，进而更好的保护飞行器与人体安全。
在两个重块107带动降落伞102移动的过程中，飞行器的坠落速度越快则气流带动第一转动叶扇104转动的速度越快，进而使得第一柔性绳105带动重块107移动的速度越快，这就使降落伞102向外抛出的速度增快，总之，飞行器的坠落速度与重块107移动并抛出降落伞102的速度成正比，使飞行器高速坠落时降落伞102能够以更快的速度打开，进而更好的保护飞行器与人体安全。
在飞行器坠落的过程中，若驾驶人员反应不及时，未能及时开启电控门201时，低空飞行器向下坠落，气流通过集气罩405进入切割壳体404，气流推动滑动切刀406向上移动，此时滑动切刀406带动其弹性件压缩，滑动切刀406挤压并切断触发绳403，滑动架401的弹性件弹出（滑动架401的弹性件初始状态为压缩蓄力），使两个滑动架401均向其背向侧移动，滑动架401带动相邻挡风板402移动，滑动架401通过相邻滑架2011带动相邻电控门201移动，使电控门201自动打开，避免飞行器突然坠落导致驾驶人员手动打开不及时，从而导致低空飞行器坠落时降落伞102不能及时打开，进而影响飞行器与驾驶人员的安全。
当驾驶人员在使用过降落伞102后，驾驶人员对降落伞102进行检修、回收和复位，并手动复位重块107，然后驾驶人员通过控制面板控制第一液压伸缩杆202复位，随后关闭第一液压伸缩杆202，最后对本装置进行维护，以便于下一次的使用。
由于飞行器和降落伞在坠落至临近地面时，飞行器的速度有所减缓，这就使降落伞的伞面所受的气流冲击力减小，若此时降落伞和飞行器的侧面袭来强风，则会导致降落伞发生偏移，从而带动飞行器偏斜，这就会导致飞行器在着陆时产生侧翻甚至是翻滚，进而威胁驾驶人员的生命安全。

当降落伞102打开，且飞行器坠落至临近地面后，高度感应器感应到高度变化并控制电控门201再次背向移动（上述实施例中电控门201第一次背向移动时，电控门201移动至与相邻第一气压伸缩杆802的伸缩端接触），此时电控门201移动并挤压相邻第一气压伸缩杆802的伸缩端，第一气压伸缩杆802的伸缩端收缩并通过导管将其内部气体挤压至相邻第二气压伸缩杆803内，第二气压伸缩杆803的伸缩端带动相邻第二气压伸缩杆803转动，使导流壳体7打开，若此时飞行器侧面气流吹过，以前侧来风为例，气流带动前侧导流壳体7内的第二转动叶扇501，前侧的第二转动叶扇501依次通过其转动轴、相邻锥齿轮组和相邻直齿轮组带动滑槽架502转动，使周向分布的滑块504受离心力作用向远离固定盘503圆心的方向移动，滑块504通过相邻第二柔性绳5041拉动其转动环移动，从而使环形切刀505移动并对相邻伞绳壳体101的伞绳进行切割。
在环形切刀505移动并对相邻伞绳壳体101的伞绳进行切割的过程中，环形切刀505带动相邻传动齿轮601移动，传动齿轮601沿着相邻齿条602移动，使得传动齿轮601在移动过程中带动相邻环形切刀505转动，环形切刀505转动并移动对相邻伞绳壳体101的伞绳进行切割，使降落伞102与伞绳壳体101脱离，减少飞行器在临近地面时受侧面气流影响导致的偏移，通过快速割断伞绳壳体101的伞绳，使飞行器与降落伞102分离，从而降低飞行器在临近地面时的风阻，进而防止气流对飞行器着陆的影响。
若飞行器坠落速度快时，降落伞的伞面承受的冲击力较大，容易导致伞面破裂，而伞面的破裂处会和导流孔的作用类似，破裂处对降落伞内侧面的气流进行导流，但伞面的破裂处不在降落伞的中央，这就会导致降落伞内气流分布不均，从而导致降落伞与飞行器产生偏斜，进而影响飞行器着陆时的安全性。
实施例3：在实施例2的基础上，如图1与图11-图13所示，还包括有固定壳体901，固定壳体901固接于外壳1的上侧，固定壳体901内滑动连接有滑动盘902，滑动盘902和固定壳体901之间固接有弹性件，其中弹性件为压簧，降落伞102通过导流孔安装有收紧绳903，降落伞102靠近其导流孔的位置设置有具有弹性的空心环，收紧绳903的右端穿入降落伞102的空心环内，收紧绳903进行拉紧时会带动空心环收缩发生褶皱，用于模拟衣服松紧带的收口，当收紧绳903向左拉紧时，降落伞102上空心环回缩并带动其导流孔的孔径减小，当收紧绳903向右放松时，降落伞102上空心环张大并带动其导流孔的孔径增大，滑动盘902与收紧绳903固接，外壳1的上侧固接有U形切刀10，U形切刀10用于对收紧绳903进行切割，当收紧绳903向上移动时，U形切刀10对收紧绳903进行切割。
当降落伞102打开后，气流冲击降落伞102的内表面并沿其导流孔流出，当飞行器的坠落速度较快或降落伞102的内表面气流冲击力较大时，气流同时挤压滑动盘902移动，使滑动盘902的弹性件压缩，滑动盘902带动收紧绳903移动，使收紧绳903放松（收紧绳903在降落伞102未打开时，收紧绳903穿过两个电控门201并位于外壳1内），气流挤压降落伞102的伞面，随后气流沿着降落伞102的伞面流动至其导流孔处，使降落伞102的导流孔受气流挤压并向周向位置撑开，使得降落伞102的导流孔面积逐渐增大并带动收紧绳903逐渐收紧，使降落伞102导流孔的面积与滑动盘902受到的气流冲击力度呈正比，避免气流冲击力较大时导流孔气流流通不及时导致降落伞102破损，从而使气流从降落伞102的破损处逃逸，进而导致降落伞102内的气流流动不平衡，使降落伞102与飞行器在坠落过程中产生偏移，甚至导致飞行器发生翻转。
当环形切刀505将伞绳壳体101的伞绳切断后，降落伞102受惯性作用在切断的瞬间向飞行器坠落的相反方向移动，使收紧绳903向上移动并挤压U形切刀10，U形切刀10对收紧绳903进行切断，使飞行器与降落伞102分离，从而降低飞行器在临近地面时的风阻，进而防止气流对飞行器着陆的影响。
