Description

一种基于低空网格化服务平台的飞行器实时监控装置及其使 用方法
技术领域
本发明涉及飞行器监控技术领域，具体为一种基于低空网格化服务平台的飞行器实时监控装置及其使用方法。
背景技术
低空网格化服务平台是用于管理和监控低空空域的系统，主要涉及低空飞行器的操作，通常包括空域管理、数据共享、预警系统、数据分析等功能，低空网格化服务平台结合飞行器进行实时监控，能够提供高效、灵活、全面的监控解决方案；
在使用飞行器进行实时监控时，通常需要选择适合的传感器和设备，如高清摄像头、热成像仪、激光雷达等，根据具体监控需求进行配置，飞行器在进行监控作业时，需要长时间的在户外飞行，而由于空气质量越来越差，空气中的灰尘、雾霾等杂质越来越多，在飞行器飞行的过程中，杂质容易附着在摄像头的表面，影响摄像的清晰度，使得工作人员查看到的监控画面模糊，影响对实时状况的判断。
发明内容
本发明的目的在于提供一种基于低空网格化服务平台的飞行器实时监控装置及其使用方法，具备可调节监控角度、除尘的优点，解决了由于空气质量越来越差，空气中的灰尘、雾霾等杂质越来越多，在飞行器飞行的过程中，杂质容易附着在摄像头的表面，影响摄像的清晰度，使得工作人员查看到的监控画面模糊，影响对实时状况判断的问题。
为实现上述目的，本发明提供如下技术方案：一种基于低空网格化服务平台的飞行器实时监控装置及其使用方法，包括飞行器本体，所述飞行器本体的底部设置有固定板，所述固定板两侧的前端和后端均固定连接有支腿，所述固定板的底部设置有活动板，所述活动板的底部设置有高清摄像头本体，所述高清摄像头本体的表面设置有调节机构，所述活动板的底部设置有清理机构，所述调节机构包括支撑片，所述支撑片位于活动板底部的两侧，并与活动板固定连接，所述高清摄像头本体的表面套设有固定套，所述固定套的两侧均固定连接有转动柱，所述转动柱远离固定套的一端与支撑片的一侧通过轴承活动连接，所述清理机构包括安装槽，所述安装槽开设于活动板的底部，所述安装槽的内腔可拆卸连接有拆卸板，所述拆卸板的底部固定连接有清理垫。
优选的，右侧所述支撑片的右侧固定连接有第一电机，所述第一电机的输出轴贯穿右侧所述支撑片，并与右侧所述转动柱的右端传动连接，所述固定板的顶部固定连接有第二电机，所述第二电机的输出轴贯穿至固定板的底部，并与活动板的顶部传动连接，所述第二电机的输出轴与固定板的内腔通过轴承活动连接。
优选的，所述第二电机输出轴的表面且位于固定板的底部通过轴承活动连接有支撑环，所述支撑环的顶部与固定板的底部固定连接，所述飞行器本体底部的两侧均固定连接有安装框，所述固定板顶部的两侧均固定连接有插入板，所述插入板的顶部延伸至安装框的内腔，并与安装框的内腔可拆卸来连接，所述安装框内腔的背面固定连接有螺母，所述安装框的正面设置有定位螺栓，所述定位螺栓的后端延伸至螺母的内腔，并与螺母的内腔螺纹连接。
优选的，所述安装槽内腔的两侧均开设有定位槽，所述拆卸板的两侧均固定连接有定位块，所述定位块的一侧延伸至定位槽的内腔，并与定位槽的内腔滑动连接，所述拆卸板的底部开设有辅助槽。
优选的，两个所述定位槽内腔相远离的一侧均开设有固定槽，所述固定槽的内腔滑动连接有卡块，所述卡块的一侧固定连接有弹簧，所述弹簧远离卡块的一端与固定槽的内壁固定连接。
优选的，所述定位块远离拆卸板的一侧开设有滑槽，所述卡块的一端延伸至滑槽的内腔，并与滑槽的内腔活动连接，所述滑槽的内腔开设有卡槽，所述卡块与卡槽的内腔卡合。
优选的，其使用方法步骤如下：
A、在飞行器本体带动高清摄像头本体飞行、高清摄像头本体对低空状况实时监控的过程中，需要调节高清摄像头本体的监控角度时，启动第一电机，第一电机在运行时，能够配合固定套和转动柱带动高清摄像头本体转动，从而对高清摄像头本体的监控角度进行调节，需要调节高清摄像头本体的监控方向时，启动第二电机，第二电机的输出轴带动活动板转动，活动板再配合支撑片、固定套和转动柱带动高清摄像头本体转动，使高清摄像头本体的监控方向能够进行调节，通过拆卸固定板能够将其从飞行器本体的底部拆除，以便对其表面的多个元件进行检修，在对固定板进行拆卸时，首先转动定位螺栓，使定位螺栓的后端从螺母的内腔移出，移出后即可将其从安装框的内腔抽出，抽出后定位螺栓对插入板的定位效果消失，此时即可将插入板从安装框的内腔抽出，进而达到对固定板的拆卸目的；
B、当工作人员观察到监控画面变得模糊、需要对高清摄像头本体的表面进行清理时，同样启动第一电机，第一电机配合固定套和转动柱带动高清摄像头本体转动，使高清摄像头本体做完整的圆周运动，当高清摄像头本体转动到最上方时，其表面能够与清理垫的底部摩擦，从而被清理垫擦去附着在表面的杂质，提高摄像画面的清晰度，需要对清理垫进行拆卸清洗时，首先使用指甲扣住辅助槽，并向外拉动拆卸板，此时卡块在卡槽的内腔会受到因拉力而产生的压力，压力导致卡槽的内壁对卡块进行挤压，使其向固定槽的内腔滑动，当卡块完全从卡槽的内腔滑出后，即可将拆卸板从安装槽的内腔抽出，以便对清理垫进行清洗。
与现有技术相比，本发明的有益效果如下：
本发明通过设置飞行器本体，用于对底部的元件进行支撑，同时能够带动其飞行，以便提高高清摄像头本体的监控范围，通过设置固定板和支腿，固定板用于对表面的各元件进行支撑，支腿用于对装置整体在下落时进行支撑，通过设置高清摄像头本体，用于对低空状况进行监控，通过设置调节机构，能够调节高清摄像头本体的监控方向和角度，从而进一步的增大高清摄像头本体的监控范围，通过设置清理机构，能够在高清摄像头本体的表面附着有杂质时，对杂质进行清理，从而确保高清摄像头本体摄像画面的清晰度，使工作人员能够更清晰的看到监控画面，防止因画面模糊而做出错误判断。
