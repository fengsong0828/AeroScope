Description

一种基于网格的社区低空飞行器快速锁定方法
技术领域
本发明涉及一种基于网格的社区低空飞行器快速锁定方法。属于航空航天技术领域。
背景技术
随着社会的发展，科技的进步，低空飞行器的数量和种类不断增多。如无人机、私人小型飞机等。低空飞行器的发展与使用，一方面革新了相关行业的工作模式。如利用无人机发快递，因空域中不存在“堵车”，从而提高了快递的发送效率。还如，利用私人小型飞机代替婚车，给人以较为前卫和惊喜的感觉。但另一方面，低空飞行器的应用也带来了很多社会问题。如目前很多民用无人机的遥控范围有限，当无人机超越遥控范围时，就会发生失控现象。无人机就会游离于空域中，直至电量耗尽而坠落。还如，我国2015年中国民航局将无人机驾驶人员的资质管理权授予了中国航空器拥有者及驾驶员协会（中国AOPA），要求7公斤及以上的无人机驾驶需要考取执照（民用无人驾驶航空器系统驾驶员合格证），但从目前来看，无照飞行的人员较多，这就造成了一定的安全隐患。
鉴于上述问题，实现对低空飞行器，特别是对社区飞行器进行管理和控制成为研究课题。而此课题的关键技术就是对低空飞行器，特别是对社区范围内的低空飞行器进行快速锁定和跟踪。目前，对于低空飞行器的扫描技术较多。如申请号为201610395176的发明专利提出了一种飞行监视服务的方法及系统。此技术应用飞机自带的黑盒子向基地提供信息数据，可以看出，此方法不适用于对不明飞行器的锁定与跟踪。还如申请号为2015101900420的发明专利提出了一种低空预警探测机动车系统。此系统搭载了雷达和摄像跟踪系统，具有低空飞行器的扫描与跟踪功能。但此系统的飞行器捕捉与锁定需要人为干预。另外，需要强调的是，此类系统通过直接把雷达获取的空域目标三维信息实时传送给光电摄像头，因雷达扫描生成数据，及数据传送给光电摄像头，摄像头再进行基于机械云台的实时调整过程中的时延不容忽视，还有机械云台的调整到位也需要有一定的时间。故容易造成目标跟踪不稳定的情况。故此类系统还存在很大的智能化提升空间，来解决目标的稳定跟踪和快速捕捉问题。
发明内容
针对现有技术中存在的问题，本发明提出了一种基于网格的社区低空飞行器快速锁定方法。解决了社区小范围内的低空飞行器的快速锁定与跟踪问题。
为了实现上述目的，本发明采用如下技术方案：
一种基于网格的社区低空飞行器快速锁定方法的主要思想是：通过对社区以车载站为原点的小范围空域进行网格划分，划分出一系列网格空间。通过雷达扫描获取低空飞行器的三维数据，进而进行三维数据向网格空间的映射。然后，把信息传送给云台摄像头。云台摄像头指向所确定的网格区域，对飞行器进行视频采集。服务器实时接收雷达获取的低空飞行器数据，并不断实现三维数据向网格空间的映射。假如飞行器映射的网格空间发生了变化，则向摄像头传输新的网格信息，云台摄像头再进行调整，指向新的网格区域。此思想的优越性就在于，通过判断飞行器所在的网格空间是否发生改变而确定是否调整摄像头的视频采集方位，而不是根据飞机的动态实时进行摄像头的调整。此方法可有效解决信息的传输延时与摄像头机械调整的不同步等问题，实现了对飞行器的高效、稳定锁定。
一种基于网格的社区低空飞行器快速锁定方法，包括如下步骤：
S101、搭建实现此发明方法的硬件系统
硬件系统主要包括车载平台、车载雷达系统、摄像头系统、数据处理及应用服务器等。车载平台的功能是搭载车载雷达设备、摄像头系统设备、数据处理及应用服务器等设备，车载雷达系统的功能是对低空领域的飞行器进行扫描，并获取低空飞行器的相关数据信息。摄像头系统的功能是能接收控制命令，并实现对视频的采集和显示。数据处理及应用服务器的功能是实现对各种算法的应用和实现。
S102、划分三维网格空间
假设车载平台的经纬度及高度分别为x、y、z，则三维网格空间的划分方法为：第一步：构建坐标系。以车载平台为原点O，Y轴与经线平行，北极方向为Y轴正方向。X轴与纬线平行，正东方向为X轴的正方向。X轴Y轴确定的平面的法线方向为Z轴正方向，且符合右手法则。第二步：相关参数设定。用户可以根据需要设定所构建的三维网格的长、宽、高参数，假设为a、b、c。进而用户根据所在社区的范围大小设定所构建的三维区域空间的长、宽、高，假设为A、B、C。且A、B、C分别为a、b、c的偶数倍。第三步：空域网格形式化定义。构建一个二维网格数组W[N][6]，N的值为A*B*C/a*b*c，N的值标识了把空域划分为N个网格。每个网格有六个参数，参数意义的定义为W[N][0]标识此网格的X轴左界限，W[N][1]标识此网格的X轴右界限，W[N][2]标识此网格的Y轴左界限，W[N][3]标识此网格的Y轴右界限，W[N][4]标识此网格的Z轴左界限，W[N][5]标识此网格的Z轴右界限。第四步：进行空域的网格划分。第一环节：令变量k=0，i=0，j=0。变量t从0到N/4-1，W[t][0]=i*a，W[t][1]=(i+1)*a，W[t][2]=j*b，W[t][3]=(j+1)*b，W[t][4]=k*c，W[t][5]=(k+1)*c。若(i+1)*a不等于A/2，则i=i+1。若(i+1)*a等于A/2，且(j+1)*b不等于B/2，则i=0，j=j+1。若(i+1)*a等于A/2，且(j+1)*b等于B/2，则i=0，j=0，k=k+1。第二环节：令变量k=0，i=0，j=0。变量t从N/4到N/2-1，W[t][0]=i*a，W[t][1]=(i+1)*a，W[t][3]=-j*b，W[t][2]=-(j+1)*b，W[t][4]=k*c，W[t][5]=(k+1)*c。若(i+1)*a不等于A/2，则i=i+1。若(i+1)*a等于A/2，且(j+1)*b不等于B/2，则i=0，j=j+1。若(i+1)*a等于A/2，且(j+1)*b等于B/2，则i=0，j=0，k=k+1。第三环节：令变量k=0，i=0，j=0。变量t从N/2到3N/4-1，W[t][1]=-i*a，W[t][0]=-(i+1)*a，W[t][2]=j*b，W[t][3]=(j+1)*b，W[t][4]=k*c，W[t][5]=(k+1)*c。若(i+1)*a不等于A/2，则i=i+1。若(i+1)*a等于A/2，且(j+1)*b不等于B/2，则i=0，j=j+1。若(i+1)*a等于A/2，且(j+1)*b等于B/2，则i=0，j=0，k=k+1。第四环节：令变量k=0，i=0，j=0。变量t从3N/4到N-1，W[t][1]=-i*a，W[t][0]=-(i+1)*a，W[t][3]=-j*b，W[t][2]=-(j+1)*b，W[t][4]=k*c，W[t][5]=(k+1)*c。若(i+1)*a不等于A/2，则i=i+1。若(i+1)*a等于A/2，且(j+1)*b不等于B/2，则i=0，j=j+1。若(i+1)*a等于A/2，且(j+1)*b等于B/2，则i=0，j=0，k=k+1。
S103、低空飞行器三维信息向三维网格空间映射。
通过带有GPS定位系统的低空雷达扫描系统，可以获取社区空域范围内的飞行器目标的多维信息，这里仅需要飞行器的经度、维度、高度三维信息。假设目标的经度、维度、高度三维信息对应的值分别为x、y、z。则低空飞行器三维信息向三维网格空间映射的具体方法为：遍历空域网格划分后的二维数组W[N][6]。令i从0到N-1，若{{ (W[i][0]>x并且W[i][1]<x)或者W[i][0]==x或者W[i][1]==x}并且{若(W[i][2]>y并且W[i][3]<y)或者W[i][2]==y或者W[i][3]==y}并且{ (W[i][4]>z并且W[i][5]<z)或者W[i][4]==z或者W[i][5]==z}}，锁定此时的i值，并退出。则获得结果为：此飞行器处在网格i中。
S104、云台摄像头的目标锁定
这里首先对云台摄像头的初始位置进行定义。云台底座与摄像头两个固定立面的边缘交接线有四个，取任意两条线，假设这两条线的中点分别为a，b，则直线ab为摄像头底座的底座轴线。规定摄像头只能在底座轴线的一侧转动。从底座轴线的中心两侧各取两个点。在摄像头镜头左侧的一点定义为i点，在摄像头右侧的一点定义为j点。从点i到点j的直线方向定义为底座轴线的正方向。定义摄像头的镜头中心点k与尾部平面中心点s的连线为摄像头的镜头中心线，从点k到点s的直线方向为镜头中心线的正方向。定义云台的初始状态为：底座轴线与三维网格空间的X轴平行，且底座轴线的正方向与X轴正方向一致。镜头中心线与X轴Y轴确定的平面的夹角为0度。云台摄像头的目标锁定方法为：第一步，获取飞行器的网络空间中心点。假设网格空间为i，假设网格空间i的中心点为x，y，z。则x= (W[i][0]+W[i][1])/2，y= (W[i][2]+W[i][3])/2，z= (W[i][4]+W[i][5])/2。第二步，第一环节：假如x，y，z的值都大于0，则令角度e=90-arctg(y/x),则调整云台底座轴线正方向与X轴正方向的夹角为e度，且云台底座轴线正方向与Y轴负方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第二环节：假如y，z的值都大于0，且x的值小于0。则令角度e=90-arctg(y/-x),则调整云台底座轴线正方向与X轴正方向的夹角为e度，且云台底座轴线正方向与Y轴正方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第三环节：假如x，z的值都大于0，且y的值小于0。，则令角度e=90-arctg(-y/x),则调整云台底座轴线正方向与X轴负方向的夹角为e度，且云台底座轴线正方向与Y轴负方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第四环节：假如z的值大于0，且x、y的值都小于0。则令角度e=90-arctg(y/x),则调整云台底座轴线正方向与X轴负方向的夹角为e度，且云台底座轴线正方向与Y轴正方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。
S105、云台摄像头的目标跟踪
云台摄像头的飞行器目标跟踪步骤是：第一步：通过S103对低空飞行器三维信息向三维网格空间映射，确定了网格空间信息为i。第二步，通过S104进行云台摄像头的目标锁定。第三步，服务器实时接收雷达获取的低空飞行器数据，并不断实现三维数据向网格空间的映射。假如飞行器映射的网格空间发生的变化，如当前摄像头实际锁定网格空间i，而此时计算的网格空间为j，且j不等于i，则向摄像头传输新的网格信息。第四步：云台摄像头再根据传来的信息进行调整，指向新的网格区域。
本发明具有如下优点：
本发明给出了一种基于网格的社区低空飞行器快速锁定方法。此发明的优越性就在于，通过判断飞行器所在的网格空间是否发生改变而确定是否调整摄像头的视频采集方位，而不是根据飞机的动态实时进行摄像头的调整。此方法可有效解决信息的传输延时与摄像头机械调整的不同步等问题，实现了对飞行器的高效、稳定锁定。另外，通过对社区小范围空域进行网格划分，为实现上述功能的优越性提供了技术支持，一般空间长、宽、高均不大于2公里，每个网格的长、宽、高均不小于20米，不大于50米。

S103、低空飞行器三维信息向三维网格空间映射。
通过带有GPS定位系统的低空雷达扫描系统，可以获取社区空域范围内的飞行器目标的多维信息，这里仅需要飞行器的经度、维度、高度三维信息。假设目标的经度、维度、高度三维信息对应的值分别为x、y、z。则低空飞行器三维信息向三维网格空间映射的具体方法为：遍历空域网格划分后的二维数组W[N][6]。令i从0到N-1，若{{ (W[i][0]>x并且W[i][1]<x)或者W[i][0]==x或者W[i][1]==x}并且{若(W[i][2]>y并且W[i][3]<y)或者W[i][2]==y或者W[i][3]==y}并且{ (W[i][4]>z并且W[i][5]<z)或者W[i][4]==z或者W[i][5]==z}}，锁定此时的i值，并退出。则获得结果为：此飞行器处在网格i中。
S104、云台摄像头的目标锁定
图4给出了摄像头及云台部分的截面图。由图4可知，云台底座与摄像头两个固定立面的边缘交接线有四个，取任意两条线，假设这两条线的中点分别为a，b，则直线ab为摄像头底座的底座轴线。规定摄像头只能在底座轴线的一侧转动。从底座轴线的中心两侧各取两个点。在摄像头镜头左侧的一点定义为i点，在摄像头右侧的一点定义为j点。从点i到点j的直线方向定义为底座轴线的正方向。定义摄像头的镜头中心点k与尾部平面中心点s的连线为摄像头的镜头中心线，从点k到点s的直线方向为镜头中心线的正方向。图5为云台底座轴线及摄像头镜头中心线示意图。定义云台的初始状态为：底座轴线与三维网格空间的X轴平行，且底座轴线的正方向与X轴正方向一致。镜头中心线与X轴Y轴确定的平面的夹角为0度。云台摄像头的目标锁定方法为：第一步，获取飞行器的网络空间中心点。假设网格空间为i，假设网格空间i的中心点为x，y，z。则x= (W[i][0]+W[i][1])/2，y= (W[i][2]+W[i][3])/2，z= (W[i][4]+W[i][5])/2。第二步，第一环节：假如x，y，z的值都大于0，则令角度e=90-arctg(y/x),则调整云台底座轴线正方向与X轴正方向的夹角为e度，且云台底座轴线正方向与Y轴负方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第二环节：假如y，z的值都大于0，且x的值小于0。则令角度e=90-arctg(y/-x),则调整云台底座轴线正方向与X轴正方向的夹角为e度，且云台底座轴线正方向与Y轴正方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第三环节：假如x，z的值都大于0，且y的值小于0。，则令角度e=90-arctg(-y/x),则调整云台底座轴线正方向与X轴负方向的夹角为e度，且云台底座轴线正方向与Y轴负方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。第四环节：假如z的值大于0，且x、y的值都小于0。则令角度e=90-arctg(y/x),则调整云台底座轴线正方向与X轴负方向的夹角为e度，且云台底座轴线正方向与Y轴正方向的夹角为90-e度。令角度r=，调整摄像头镜头中心线与X轴Y轴确定的平面的夹角为r度。图6为目标锁定示意图。
S105、云台摄像头的目标跟踪
云台摄像头的飞行器目标跟踪步骤是：第一步：通过对低空飞行器三维信息向三维网格空间映射，确定了网格空间信息为i。第二步，通过进行云台摄像头的目标锁定。第三步，服务器实时接收雷达获取的低空飞行器数据，并不断实现三维数据向网格空间的映射。假如飞行器映射的网格空间发生的变化，如当前摄像头实际锁定网格空间i，而此时计算的网格空间为j，且j不等于i，则向摄像头传输新的网格信息。第四步：云台摄像头再根据传来的信息进行调整，指向新的网格区域。
