Description

一种适用于中低空飞行器航拍测量的图像分辨率校准装置
技术领域
本发明属于飞行器航拍测量的技术领域，更具体的，本发明涉及一种适用于中低空飞行器航拍图像校准的技术领域。
背景技术
无人机航拍摄影是以无人驾驶飞机作为空中平台, 以机载遥感设备, 如高分辨率CCD 数码相机、轻型光学相机、红外扫描仪，激光扫描仪、磁测仪等获取信息，用计算机对图像信息进行处理，并按照一定精度要求制作成图像。航拍摄影可用于野外微地貌研究、林地考察、考古、文物修缮等领域。航拍时一般选用如道路交叉点等像片上明显地物点，用测角交会、测距导线、等外水准、高程导线等普通测量方法测定其平面坐标和高程。
对航拍图像进行复原加工是航拍的一项重要工作，但是用计算机对图像信息进行处理，并按照一定精度要求制作成图像就需要对图形拍摄精度有一个准确的认识，如果没有参照很容易因为修订比例的问题造成图像失真。
发明内容
针对现有技术中存在的用计算机对图像信息进行处理，修订比例不准确造成图像失真的问题，本发明提供了一种适用于中低空飞行器航拍测量的图像分辨率校准装置，与现有技术相比，本发明装置使用方便，结构简单，利用航拍飞行器先行对图像矫正装置进行抓取，记录能够识别的最低分辨率并将其一并传给处理图形的计算机，计算机处理航拍图像时利用前面的测试数据准确对图形进行校准，在航拍领域具有广泛的实用性。
为了解决上述问题，本发明采用的技术方案如下：
本发明提供的一种适用于中低空飞行器航拍测量的图像分辨率校准装置，包括承载板和支撑机构，支撑机构上设有承载板，承载板上设有分辨率标记板、信号标志和分割槽，分辨率标记板设有若干个，每个分辨率标记板上设有与其他分辨率标记板不同的刻度线，相邻分辨率标记板之间设有分割槽，分辨率标记板中部设有信号标志，分辨率标记板和信号标志对应设置。

本发明中，连接杆之间设有第三支撑板，第三支撑板上设有水平仪。

本发明中，第二承载板上设有驱虫盒，第一承载板中部设有通气孔，驱虫盒和通气孔对应设置。
本发明中，信号标志采用不同颜色的斑点。
本发明中，信号标志为不同颜色的发光二极管，发光二极管和电源电性连接。
本发明中，风扇设有两组。
本发明中，支撑管上设有若干定位孔，连接杆内设有定位销槽，定位销槽内设置定位销，定位销和定位销槽之间通过弹簧连接。
本发明中，支腿下设有尖头。
本发明采用的第一承载板、第二承载板、支撑管、连接杆、支腿、分辨率标记板、信号标志、风扇、控制器、驱虫盒、支撑杆、电源、定位销和水平仪均可通过公共渠道购买，控制器采用金贝 TR-A6高速引闪器，无线发射模块采用金贝 TR-A6高速引闪器的2.4G无线远程控制器。
使用本发明提供的技术方案可以达到如下有益效果：
本发明提供的一种适用于中低空飞行器航拍测量的图像分辨率校准装置，简单可靠，能够解决不同航拍高度或者航拍设备抓取的图像比例存在差别，利用计算机按照相同方法处理后图像失真的问题，实现了每一个设备每一个拍摄环境单独校准分辨率，保证了图像的准确和清楚，在航拍领域具有广泛的实用性。

实施例二：本发明图像分辨率校准装置的适用
在适用本发明提供的一种适用于中低空飞行器航拍测量的图像分辨率校准装置时，在航拍飞行器即将航拍的区域起始位置安装本发明装置，将支腿5尖头16插入地下，利用水平仪15和支腿5调整本发明放置角度，在驱虫盒11里放置驱虫材料，驱赶蚊虫等减少分辨率标记板6造成污染影响准确性，利用风扇9吹掉落在分辨率标记板6上面的浮灰和树叶等，航拍飞机经过对其发出型号，利用航拍飞机拍摄信号标志7，将能够识别的分辨率标记与信号标志7对应，方便图像传回后根据信号标志7获得最低分辨率的分辨率标记值，获得校准值，后期飞行器拍摄完所有物体后，利用校准值对拍摄到的物体进行矫正。
