Description

一种低空飞行器专用紧急防护套装及使用方法
技术领域
本发明涉及保护服技术领域，特别涉及一种低空飞行器专用紧急防护套装及使用方法。
背景技术
随着飞行器普及，个人飞行器必然是未来发展方向。轻巧，灵活的低空，低速飞行器将是个人飞行器的发展方向。
然而，低空飞行器如果失控，由于在空中飞行。驾驶员或者乘客无法跟地面车俩内部人员一样迅速打开车门逃生，地面车辆的安全设备一般为车内安全气囊及安全带，在低空飞行器中，这些装置无法完善的保护驾驶员以及乘客。因此设计一款低空飞行器的防护套装是必要的。
发明内容
有鉴于上述现有技术中的缺陷，本发明的目之一是提供一种低空飞行器专用紧急防护套装，其技术方案如下：
一种低空飞行器专用紧急防护套装，包括：
快穿马甲，所述快穿马甲用于穿戴在使用者的上身，且所述快穿马甲的前胸部设置有前安全气囊背包，该前安全气囊背包内设置有前气囊，所述快穿马甲的后背部设置有后安全气囊背包，该后安全气囊背包内设置有后气囊，所述快穿马甲上设置有气囊保护帽收纳装置，该气囊保护帽收纳装置内设置有气囊保护帽；
肘护具，所述肘护具用于穿戴在使用者的肘部，且数量为两个，所述肘护具上设置有两个分别向前和向后射出的肘气囊；
膝护具，所述膝护具用于穿戴在使用者的膝部，且数量为两个，所述膝护具上设置有两个分别向前和向后射出的膝气囊；
所述前气囊、后气囊、气囊保护帽、肘气囊和膝气囊各自对应有炸药仓，各气囊接通到对应的炸药仓内，所述炸药仓内设置有爆炸产生气体的炸药，所述炸药仓内设置有用于点燃所述炸药的点火器。
进一步，所述前安全气囊背包的气囊射出口朝右，所述前气囊从所述前安全气囊背包的气囊射出口射出；所述后安全气囊背包的气囊射出口朝左，所述后气囊从所述后安全气囊背包的气囊射出口射出。
进一步，所述肘护具包括肘护尼龙套，该肘护尼龙套上肘弯部和外肘部对应位置均设置有肘护内护壳，所述肘护内护壳外侧还设置有肘护外护壳，两个所述肘护外护壳连接为一体。
进一步，所述膝护具包括膝护尼龙套，该膝护尼龙套上膝弯部和膝盖部对应位置均设置有膝护内护壳，所述膝护内护壳外侧还设置有膝护外护壳，两个所述膝护外护壳连接为一体。
进一步，所述点火器为电子点火器。
进一步，所述点火器对应设置有控制点火的控制芯片，且该控制芯片上连接有蓝牙模块，所述快穿马甲的前胸位置设置有点火触发开关，且该点火触发开关上也设置有蓝牙模块，且与所述控制芯片上的蓝牙模块产生通信以发送点火信号。
进一步，所述肘护内护壳和膝护内护壳均采用凯夫拉纤维板。
进一步，所述气囊保护帽收纳装置设置在所述快穿马甲后背部的上部位置，且所述气囊保护帽收纳装置上的气囊射出口朝上。
本发明的目的之二是提供上述低空飞行器专用紧急防护套装的使用方法，技术方案为：触发所述点火触发开关，该点火触发开关通过所述蓝牙模块发送点火信号到所述前安全气囊背包、后安全气囊背包、气囊保护帽收纳装置、肘护具和膝护具处的各个所述控制芯片，控制芯片驱动所述点火器点火，引燃所述炸药，炸药燃烧产生大量气体，并撑开各个气囊。
有益效果：本发明构思新颖、设计合理，且便于使用，本发明全新研发设计了一种套装，包括快穿马甲、肘护具和膝护具三部分，并通过炸药爆炸产生气体，撑开气囊对人体形成保护，适用于低空飞行棋的配合使用。

具体实施方式
下面将结合本申请实施例中的附图，对本申请实施例中的技术方案进行清楚、完整地描述，显然，所描述的实施例仅仅是本申请一部分实施例，而不是全部的实施例。基于本申请中的实施例，本领域普通技术人员在没有作出创造性劳动前提下所获得的所有其他实施例，都属于本申请保护的范围。
如图1-5所示，一种低空飞行器专用紧急防护套装，包括：
快穿马甲1，快穿马甲1用于穿戴在使用者的上身，且快穿马甲1的前胸部设置有前安全气囊背包2，该前安全气囊背包2内设置有前气囊9，快穿马甲1的后背部设置有后安全气囊背包3，该后安全气囊背包3内设置有后气囊10，快穿马甲1上设置有气囊保护帽收纳装置4，该气囊保护帽收纳装置4内设置有气囊保护帽11；
肘护具5，肘护具5用于穿戴在使用者的肘部，且数量为两个，肘护具5上设置有两个分别向前和向后射出的肘气囊7；
膝护具6，膝护具6用于穿戴在使用者的膝部，且数量为两个，膝护具6上设置有两个分别向前和向后射出的膝气囊8；
前气囊9、后气囊10、气囊保护帽11、肘气囊7和膝气囊8各自对应有炸药仓12，各气囊接通到对应的炸药仓12内，炸药仓12内设置有爆炸产生气体的炸药13，炸药仓12内设置有用于点燃炸药13的点火器14。
本发明的基础方案中，在具体使用时，将快穿马甲1穿戴在上身，两个肘护具5分别穿戴在使用者的肘部，两个膝护具6分别穿戴在使用者的膝部。在起到保护作用时，通过点燃炸药仓12内的炸药13，炸药13燃烧产生大量的气态，撑开前气囊9、后气囊10、气囊保护帽11、肘气囊7和膝气囊8，并分别位于人体不同的位置，起到保护作用。
较佳地，前安全气囊背包2的气囊射出口朝右，前气囊9从前安全气囊背包2的气囊射出口射出；后安全气囊背包3的气囊射出口朝左，后气囊10从后安全气囊背包3的气囊射出口射出。本优选方案中，给出了一种前气囊9的具体射出方向形式，并提供相应的装置结构，以使得在射出前气囊9是朝左侧射出的，对人体左侧位置形成保护。同理，也给出了一种后气囊10的具体射出方向形式，并提供相应的装置结构，以使得在射出后气囊10是朝右侧射出的，对人体右侧位置形成保护。
较佳地，肘护具5包括肘护尼龙套5a，该肘护尼龙套5a上肘弯部和外肘部对应位置均设置有肘护内护壳5b，肘护内护壳5b外侧还设置有肘护外护壳5c，两个肘护外护壳5c连接为一体。本优选方案中，给出了一种肘护具5的具体结构形式，其中，肘护尼龙套5a用来套在人体的肘部，并直接通过设置肘护内护壳5b来直接保护到人体的肘部，同时，再设置肘护外护壳5c进一步进行保护。
较佳地，膝护具6包括膝护尼龙套6a，该膝护尼龙套6a上膝弯部和膝盖部对应位置均设置有膝护内护壳6b，膝护内护壳6b外侧还设置有膝护外护壳6c，两个膝护外护壳6c连接为一体。本优选方案中，给出了一种膝护具6的具体结构形式，其中，膝护尼龙套6a用来套在人体的膝部，并直接通过设置膝膝护内护壳6b来直接保护到人体的膝盖位置，同时，再设置膝护外护壳6c进一步进行保护。
较佳地，点火器14为电子点火器。本优选的方案，给出了一种具体的打火器形式。
较佳地，点火器14对应设置有控制点火的控制芯片，且该控制芯片上连接有蓝牙模块，快穿马甲1的前胸位置设置有点火触发开关，且该点火触发开关上也设置有蓝牙模块，且与控制芯片上的蓝牙模块产生通信以发送点火信号。本优选方案中，给出了一种点火器14在是电子打火器的情况下，采用遥控点火的方案形式，且具体是通过蓝牙通信。
较佳地，肘护内护壳5b和膝护内护壳6b均采用凯夫拉纤维板。本优选方案中，给出了一种肘护内护壳5b和膝护内护壳6b的具体材料形式。
较佳地，气囊保护帽收纳装置4设置在快穿马甲1后背部的上部位置，且气囊保护帽收纳装置4上的气囊射出口朝上。本优选方案中，给出了一种气囊保护帽收纳装置4的气囊射出结构形式。
结合以上实施例，在另一实施例中给出低空飞行器专用紧急防护套装的使用方法，其具体如下：
触发点火触发开关，该点火触发开关通过蓝牙模块发送点火信号到前安全气囊背包2、后安全气囊背包3、气囊保护帽收纳装置4、肘护具5和膝护具6处的各个控制芯片，控制芯片驱动点火器14点火，引燃炸药13，炸药13燃烧产生大量气体，并撑开各个气囊。
在具体实施时，前气囊9和后气囊10撑开后的高度可以取为1.6米-1.8米，以适应大部分人的身高。
