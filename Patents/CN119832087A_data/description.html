Description

一种用于低空飞行器的数据感知智能处理方法
技术领域
本发明涉及数据处理技术领域，尤其涉及一种用于低空飞行器的数据感知智能处理方法。
背景技术
在当今的航空领域，低空飞行器的应用日益广泛，包括无人机、直升机、轻型飞机等。随着技术的不断发展，对低空飞行器的性能、安全性和智能化要求也越来越高。
随着无人机的迅猛发展，低空空域管控面临新的挑战和机遇，各国纷纷加大研发投入。
中国专利公开号：CN109739265B，公开了一种空中作业飞行器系统，包括：至少一个伴随飞行器，用于获取作业飞行器视野盲区内的补充环境信息；与伴随飞行器建立有数据连接的数据处理器，用于根据补充环境信息生成作业指令；与数据处理器建立有数据连接的作业飞行器，用于根据作业指令对作业对象执行空中作业或躲避障碍物；由此可见，上述技术方案存在以下问题：无法精准确定无人机的实际运行状况，无法在确定飞行器的运行异常时及时对飞行器的运行参数进行调节，导致数据处理效率低。
发明内容
为此，本发明提供一种用于低空飞行器的数据感知智能处理方法，用以克服现有技术中无法精准确定无人机的实际运行状况，无法在确定飞行器的运行异常时及时对飞行器的运行参数进行调节，导致数据处理效率低的问题。
为实现上述目的，本发明提供一种用于低空飞行器的数据感知智能处理方法，包括：
S1，控制飞行器周期性获取图像信息，在图像信息中建立坐标系，标定图像信息中的若干特征点，获取各特征点对应的坐标；
S2，基于各特征点在当前周期的坐标与在前一周期的坐标确定各特征点在两相邻周期中的变化距离；
S3，对各变化距离进行清洗；
S4，基于完成清洗的各变化距离确定平均映射参量，基于获取各特征点的坐标确定角度离散参量；
S5，基于平均映射参量和角度离散参量确定飞行器的运行是否合格，包括：
判定飞行器的运行不合格，并调节飞行器的前驱螺旋桨和后驱螺旋桨的转速比，或基于映射差异参量确定针对飞行器的处理方式，其中处理方式为调节所述S3的数据清洗标准或将飞行器各螺旋桨的转速调节至对应值；
或，判定飞行器的运行合格，并控制飞行器持续使用当前的运行参数运行。
进一步地，在所述S3，对各变化距离进行清洗的过程，包括：
若变化距离大于预设变化距离，则将单个变化距离筛除；
若变化距离小于等于预设变化距离，则将单个变化距离保留；
在所述S4中，基于完成清洗的各变化距离确定平均映射参量的过程，包括：
逐一确定各变化距离映射在坐标系Y轴上的长度，得到该变化距离对应的映射距离；
求解各映射距离的平均值，得到平均映射参量；
在所述S5中，基于平均映射参量确定飞行器的运行是否合格的过程，包括：
若平均映射参量小于等于第一预设平均映射参量，则判定飞行器的运行不合格，并基于平均映射参量将飞行器的前驱螺旋桨和后驱螺旋桨的转速比调节至对应值；
若平均映射参量小于等于第二预设平均映射参量且大于第一预设平均映射参量，则判定飞行器的运行合格；
若平均映射参量小于等于第三预设平均映射参量且大于第二预设平均映射参量，则基于角度离散参量确定飞行器的运行是否合格；
若平均映射参量大于第三预设平均映射参量，则判定飞行器的运行不合格，并基于映射差异参量确定针对飞行器的处理方式。
进一步地，基于平均映射参量将飞行器的后驱螺旋桨和前驱螺旋桨的转速比调节至对应值，其中：
基于平均映射参量确定的转速比的增加幅度与平均映射参量成反比。
进一步地，基于角度离散参量确定飞行器的运行是否合格的过程，包括：
获取特征点在当前周期的坐标与在前一周期的对应的坐标在同一坐标系内的连线与Y周的夹角，得到对应特征点的角度参量；
求解各特征点的角度参量的方差，得到角度离散参量；
若角度离散参量小于等于预设角度离散参量，则判定飞行器的运行不合格，并基于映射差异参量确定针对飞行器的处理方式；
若角度离散参量大于预设角度离散参量，则基于角度离散参量将飞行器各螺旋桨的转速调节至对应值。
进一步地，基于角度离散参量将飞行器各螺旋桨的转速调节至对应值，其中：
基于角度离散参量确定的飞行器各螺旋桨的转速的增加幅度与角度离散参量成正比。
进一步地，基于映射差异参量确定针对飞行器的处理方式的过程，包括：
将平均映射参量与第三预设平均映射参量的差值记为映射差异参量；
若映射差异参量小于等于第一预设映射差异参量，则基于飞行器在当前周期的高度与上一周期的高度的差值将第一预设平均映射参量、第二预设平均映射参量和第三预设平均映射参量调节至对应值；
若映射差异参量小于等于第二预设映射差异参量且大于第一预设映射差异参量，则基于所述S3中筛除的数据的数量确定针对飞行器的处理方式；
若映射差异参量大于第二预设映射差异参量，则基于各所述变化距离的方差将预设变化距离调节至对应值。
进一步地，基于飞行器在当前周期的高度与上一周期的高度的差值将第一预设平均映射参量、第二预设平均映射参量和第三预设平均映射参量调节至对应值，其中:
基于高度差值确定的第一预设平均映射参量、第二预设平均映射参量和第三预设平均映射参量的降低幅度与高度差值成正比。
进一步地，基于所述S3中筛除的数据的数量确定针对飞行器的处理方式的过程，包括：
若数据数量小于等于预设数据数量，则角度离散参量将飞行器各螺旋桨的转速调节至对应值；
若数据数量大于预设数据数量，则基于各所述变化距离的方差将预设变化距离调节至对应值。
进一步地，基于各所述变化距离的方差将预设变化距离调节至对应值，其中：
基于各所述变化距离的方差确定的预设变化距离的降低幅度与各所述变化距离的方差成正比。
与现有技术相比，本发明的有益效果在于，针对飞行器的数据进行流程化处理，获取各特征点的坐标信息，通过各个特征点的坐标信息变化情况确定飞行器的移动情况，以作为数据依据精准确定飞行器的实际运行状况，并在确定飞行器的运行异常时及时对飞行器的运行参数进行调节，提高了数据处理效率。
进一步地，对各变化距离进行清洗筛分，去除异常值，以提高数据的可靠性；确定平均映射参量，平均映射参量表征了飞行器在当前指令飞行速度下的实际飞行情况，在平均映射参量小于等于第一预设平均映射参量时，此情况下飞行器的移速过低，此时调节前后螺旋桨的转速比，以使后面转速高于前面转速，让飞行器向前倾斜，从而改变各螺旋桨与垂直方向的角度，以使各螺旋桨产生后推力；在平均映射参量小于等于第三预设平均映射参量且大于第二预设平均映射参量时，基于角度离散参量确定飞行器的运行是否合格，在角度离散参量大于预设角度离散参量时，此时各角度的离散大，此情况下飞行高度降低，且飞行器运行速度过快，此时飞行器存在坠落风险，统一增加各螺旋桨转速以保证无人机运行安全，进一步提高了数据处理效率。
进一步地，在确定飞行器的运行异常时，基于映射差异参量确定针对飞行器的处理方式，在映射差异参量小于等于第一预设映射差异参量时，此时差值小，由于飞行器飞行高度低于预设值，此时修正判定基准以重新持续监测飞行器的运行状况；在映射差异参量大于第二预设映射差异参量时，此时由于获取的数据量过低导致数据分析出现误差，此情况下调节数据清洗标准以增加数据分析量提高数据的分析精度；在映射差异参量小于等于第二预设映射差异参量且大于第一预设映射差异参量时，此时结合筛除的数据的数量综合确定针对飞行器的处理方式，在数据数量小于等于预设数据数量时，此时用于分析的数据量充足，此情况下由于飞行器坠落导致飞行器急速下降，此时针对各螺旋桨的转速进行调节，在保证飞行器精准运行的同时，进一步提高了数据处理效率。
